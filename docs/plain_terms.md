# 訳語の平易化リスト

対訳の校訂・生成で, 日常では使わない文語漢語・古語表現を平易語へ統一する際の
参照リスト. 置換語・保持する技術用語・機械置換の落とし穴をまとめる. 校訂で
新たな置換例や注意点が見つかったら, このファイルに追記して維持する.

## 古語表現 -> 平易

- 〜せん (意志) -> 〜しよう (例: 出息せん -> 出息しよう, 調御せん -> 調御しよう)
- われ…せん -> 私は…しよう (本文の他箇所の文体に合わせる)
- 〜せんがため -> 〜するため (例: 調御せんがため -> 調御するため)
- 〜せしめ (使役の古語) -> 〜させ (例: 生起せしめる -> 生起させる)
- 〜らしめ (使役の古語) -> 文脈で 3 分し訳す:
  - X ならしめる (断定なり + しむ) -> X にする (寂静ならしめる -> 寂静にする,
    軽安ならしめ -> 軽安にし, 円満ならしめる -> 円満にする,
    飽くことなからしめる -> 飽きさせない)
  - 動詞の使役 V-らしめ -> V-(さ)せ (入らしめて -> 入らせて, 造らしめよ ->
    造らせよ, 至らしめて -> 至らせて)
  - たらしめる (断定たり + しむ) -> 〜とする (たらしめること -> とすること,
    既知ならしめること -> 既知にすること)
  - 注意: X ならしめつつ が直前の「静め, 滅させ」等と並ぶ箇所 (寂静にしつつ) は
    同義反復にならないか確認する
- 〜べし -> 〜べきである (例: 繋ぐべし -> 繋ぐべきである)
- 坐す (文語) -> 坐る (nisīdati. 語釈の訳「坐る」に合わせる. 結跏趺坐 等の
  名詞の 坐 は保持)
- 〜たる (断定たり, 連体) -> 同格・同一性は 〜である, 役割・機能は 〜となる で
  訳し分ける
  - である: 一切知者たる仏 -> 一切知者である仏, 最上の果たる阿羅漢果 ->
    最上の果である阿羅漢果, 捨遣たる随観 -> 捨遣である随観, 寂静たること ->
    寂静であること, 沙門たること -> 沙門であること
  - となる: 所依たる教法 -> 依りどころとなる教法, 頂点たるもの ->
    頂点となるもの, 地盤たる阿蘭若 -> 地盤となる阿蘭若
  - 形容動詞タリ活用は別扱い: 騒然たる -> 騒がしい
  - 語釈中の「～たる」(bhūta 等の訳) も「～である」に統一
  - 動詞は対象外: 当たる, わたる (三世にわたる) 等

## 保持 (変更しない)

- 増上戒 / 増上心 / 増上慧 (三学。増上は単独では使わずこの合成語のみ)
- 瑜伽行者, 生起
- 確立した仏教技術用語: 随観, 所縁, 業処, 寂静, 近行, 安止, 無礙解, 念処,
  寂止, 軽安, 遍知, 寂滅, 等持, 数息, 結跏趺坐, 威儀, 阿闍梨, 焦燥, 円満
- 技術用語の構成要素は止息→静まる の例外: 厭離止息智 (nibbidāpaṭippassaddhiñāṇa)
  は四観智の定型名なので止息のまま保持
- 別語に注意: parapattiya の「他に依存する」は paṭibaddha (繋属→結びついた) と
  別語なので変えない

## 機械置換の落とし穴 (重要)

漢語 + する は活用が一定だが, 和語動詞は活用が変わるため機械置換で崩れる. 置換後は
必ず diff 全行 + 活用形の列挙 (grep -oE で全形を確認) で検証する.

- 転起した -> ×起こりた -> ○起こった
- 転起させるべき -> ×起こしるべき -> ○起こすべき
- 転起させつつ、起こしつつ -> 重複 -> ○起こしつつ
- 不転起 -> ×不働き -> ○働きの停止
- 止息する (自動) -> 静まる / 止息しつつ (他動) -> 静めつつ を取り違えない
- 勝妙である -> すぐれている (×すぐれである), 勝妙なる -> すぐれている (×すぐれたる)

### 自己重複 (新->旧 が併記されている箇所)

第一稿が旧語に新語を補足併記している場合, 一律置換で同語が重複する. 例えば
現起 に現前を併記した「現起（現前）」「現起、現前すること」「現起した、現前した」を
現起->現前 すると「現前（現前）」等になる. patis_1_03 で約 28 箇所発生した.

- 対策: 置換後に「新語（…新語…）」「新語、新語」「新語・新語」「新語〔…新語…〕」を
  grep し, 重複を除くか括弧内を別語 (例: 立ち現れる/対象) に言い換える
- 対象が二義をもつ語 (upaṭṭhāna = 対象義/作用義) は括弧注記で訳し分けを保つ

## 未対応の候補

- 代名詞「われ」: 該当なし (言われ・問われ等の受身形のみ。代名詞は せん 校訂時に
  「私は」へ変換済み)
- 〜にして / 〜がごとく / 〜しようと欲する は仏典訳の文体として保留 (要判断)

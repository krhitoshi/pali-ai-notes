# yathattha の訳語「真実の目的」の根拠

Paṭisambhidāmagga に現れる yathattha (yathā + attha) を「真実の目的」と訳す
根拠と, 検討の経緯をまとめる. 該当箇所は patisambhidamagga/patis_1_03.md の
§161(2) (yathā の語義解説段), §162(1) (Desitā 句), §162(2) (Pabhāseti 句) と,
註 (patisambhidamagga/patis_attha_1_03.md) の対応箇所.

## 訳語

- yathattha (主格) = **真実の目的** (§161(2)「十の真実の目的」)
- yathatthaṃ (副詞的対格, §162(2)) = **真実のとおりに**
- 各項の X-attho (attadamathattho 等) = **X という目的** (attha = DoP 第 1 義
  aim, purpose)

## 根拠

1. **X-attho の attha は「修習のねらい・成果」**: 註は各項を修習の完成時に
   実際に起こる事態として釈す (attadamathattho = 阿羅漢道の刹那に荒々しさが
   消えること, samathattho = 清涼となること). 「意義」では significance に
   寄り伝わらないため「目的」を採る
2. **yathattho は実質述語**: §162(2) で現等覚 (abhisambujjhati) の対象に立つ.
   「世間を照らす」根拠が「語義を覚ったから」では通らない. また単なる語義釈
   なら標準定型 (`*Yathāti attadamathaṭṭho` / `kenaṭṭhena` 型) が使えたのに,
   本文は yathattha という複合語を立てて主格で等置している
3. **「真実の」は註の語釈による**: `Yathatthāti yathāsabhāvatthā`(あるがままの
   本性どおりの目的). 註によれば偈の yathā は「説かれたとおり (pakāra)」と
   「あるがまま (sabhāva)」の ekasesa で, 十項は後者に当たる. 語形成上も
   yathā 派生語 = 真実の (yathāva / yāthāva. Main: 真の・実の・真実の, PTS:
   同義語 taccha・bhūta・aviparīta) は正常. **この解釈が註依存であることは
   §161(2) の訳注に明記してある**
4. 主張内容: 教説と事実の一致. 「この修習は自己を調御する」等の教説の言明が,
   説かれたとおり実際に成り立つ (aviparīta = 顛倒なし)

## 別解 (訳注に記載)

Ñāṇamoli (The Path of Discrimination, PTS 1982) は「yathā という語の十の意味」
の列挙と解する ("there are ten meanings of 'according as'"). X-attha を語義に
使う用例は本文にもあり (kenaṭṭhena buddho §162(1), niyyānāvaraṇaṭṭhena
§153(2)), この読みは排除できない. 本訳は上記 2 の形式上の理由と §162(2) の
通りやすさで実質解を採り, 別解として §161(2) の訳注に記録した.

## 経緯 (旧訳の問題点と検討過程)

- 第一稿の「如実義」は外部裏付けのない訳語で, 如実 = yathābhūta (§158(5)
  如実智見) と衝突していた. 辞書 (Main / Concise / PTS) に yathattha の見出し
  はなく, DoP は y- 未刊, Paṭisambhidāmagga に漢訳もない. 2026/06/29 に問題を
  指摘して保留
- 2026/07/03 の attha 改訂で暫定「如実の意義」を経て, 同日「真実の目的」で
  確定し本文へ適用
- 「そのとおりの意義」案は, 羅列中で「そのとおり」の先行詞が宙に浮き,
  括弧 (yathā) を付けても原語表示にしかならないため退けた
- 「ありのまま／あるがままの義」案は註 sabhāva に沿うが, 羅列の訳語としては
  「真実の」がより読める日本語になるため統合した

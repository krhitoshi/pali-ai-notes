
## 218 (1)

218. Tattha araññagatoti ‘‘araññanti nikkhamitvā bahi indakhīlā sabbametaṃ arañña’’nti (vibha. 529) ca, ‘‘āraññakaṃ nāma senāsanaṃ pañcadhanusatikaṃ pacchima’’nti (pārā. 654) ca evaṃ vuttalakkhaṇesu araññesu yaṃkiñci pavivekasukhaṃ araññaṃ gato. Rukkhamūlagatoti rukkhasamīpaṃ gato. Suññāgāragatoti suññaṃ vivittokāsaṃ gato. Ettha ca ṭhapetvā araññañca rukkhamūlañca avasesasattavidhasenāsanagatopi suññāgāragatoti vattuṃ vaṭṭati.

Evamassa ututtayānukūlaṃ dhātucariyānukūlañca ānāpānassatibhāvanānurūpaṃ senāsanaṃ upadisitvā alīnānuddhaccapakkhikaṃ santaṃ iriyāpathaṃ upadisanto nisīdatīti āha. Athassa nisajjāya daḷhabhāvaṃ assāsapassāsānaṃ pavattanasukhataṃ ārammaṇapariggahūpāyañca dassento pallaṅkaṃ ābhujitvātiādimāha. Tattha pallaṅkanti samantato ūrubaddhāsanaṃ. Ābhujitvāti bandhitvā.

1. Tattha araññagato ti

→「ここで『araññagata(森に赴いた者)』(という語)について(説く)。」

2. ‘‘araññan ti nikkhamitvā bahi indakhīlā sabbam etaṃ arañña’’n ti (vibha. 529) ca,

* bahi indakhīlā = 「インダキーラ(城門の門柱・境界標)より外は」

* ti (vibha. 529) ca = 「と(『分別論』529)(説かれ)、また」

→「『森とは、(村の)門柱(indakhīla)の外に出たならば、それすべてが森である』と(『分別論』に説かれ)、」

3. ‘‘āraññakaṃ nāma senāsanaṃ pañcadhanusatikaṃ pacchima’’n ti (pārā. 654) ca

6. Suññāgāragato ti suññaṃ vivittokāsaṃ gato.

7. Ettha ca ṭhapetvā araññañca rukkhamūlañca avasesasattavidhasenāsanagato pi suññāgāragato ti vattuṃ vaṭṭati.

  * ṭhapetvā = 「置いて・除いて」

8. Evam assa ututtayānukūlaṃ dhātucariyānukūlañca ānāpānassatibhāvanānurūpaṃ senāsanaṃ upadisitvā alīnānuddhaccapakkhikaṃ santaṃ iriyāpathaṃ upadisanto nisīdatī ti āha.

  * utu = 「時節・季節」/ ttaya = 「三つ」/ anukūla = 「適した・順応した」
* dhātu-cariyā-anukūlañ ca = 「界(気質)と行(性向)とに適した」
  * dhātu = 「界・要素(体質)」/ cariyā = 「行・性向(気質)」

* senāsanaṃ upadisitvā = 「臥坐処を指示して」

  * alīna = 「沈滞しない・退縮しない」/ anuddhacca = 「掉挙せぬ・浮わつかぬ」/ pakkhika = 「～の側に属する」
* santaṃ iriyāpathaṃ = 「寂静な威儀(姿勢)を」

* nisīdati ti āha = 「『坐る』と(世尊は)説いた」

→「このように、(世尊は)彼に対し、三つの時節に適し、体質と性向とに適し、入出息念の修習にふさわしい臥坐処を示したうえで、沈滞にも掉挙にも偏らない寂静な威儀を示しつつ『坐る』と説いたのである。」

9. Atha assa nisajjāya daḷhabhāvaṃ assāsapassāsānaṃ pavattanasukhataṃ ārammaṇapariggahūpāyañca dassento pallaṅkaṃ ābhujitvā ti-ādim āha.

* nisajjāya daḷhabhāvaṃ = 「坐の堅固さ(安定性)を」
  * nisajjā = 「坐ること」/ daḷha-bhāva = 「堅固な状態」
* assāsa-passāsānaṃ pavattana-sukhataṃ = 「出息と入息の転起(働き)が容易であることを」
  * assāsa-passāsa = 「出入息」/ pavattana = 「転起・運行」/ sukhatā = 「容易さ・楽なること」
* ārammaṇa-pariggaha-upāyañ ca = 「および所縁(対象)を把握する方法を」

10. Tattha pallaṅkan ti samantato ūrubaddhāsanaṃ.

  * samantato = 「周囲に・すっかり」/ ūru = 「腿(もも)」/ baddha = 「結んだ・組んだ」/ āsana = 「坐・座法」

→「そこで『結跏趺坐』とは、四方より腿を組んだ坐法(の意である)。」

→「『ābhujitvā(組んで)』とは、(腿を)結んで(組んで)、という意味である。」

## 218 (2)

1. Ujuṃ kāyaṃ paṇidhāyā ti uparimasarīraṃ ujukaṃ ṭhapetvā.

* uparima-sarīraṃ ujukaṃ ṭhapetvā = 「上半身をまっすぐに据えて(の意である)」
  * uparima = 「上の」/ sarīra = 「身体」/ ujuka = 「まっすぐな」/ ṭhapetvā = 「置いて・据えて」

→「『身体をまっすぐに保って』とは、上半身をまっすぐに据えて(の意である)。」

  * paṇamati = 「曲がる・屈する」

4. Athassa yā tesaṃ paṇamanappaccayā khaṇe khaṇe vedanā uppajjeyyuṃ, tā na uppajjanti.

* yā tesaṃ paṇamana-paccayā = 「それら(皮肉腱)の屈曲を縁として(生じうる)ところの」
  * paṇamana = 「曲がること」/ paccaya = 「縁・条件」

* vedanā uppajjeyyuṃ = 「諸々の(苦)受が生じるであろう(ところの)」
  * vedanā = 「受(感受)」/ uppajjeyyuṃ = 「生じるであろう(願望・可能)」
* tā na uppajjanti = 「それら(の受)は生じない」

→「さてその者には、それら(皮膚・肉・腱)の屈曲を縁として刹那刹那に生じるであろう諸々の(苦)受、それらが生じない。」

5. Tāsu anuppajjamānāsu cittaṃ ekaggaṃ hoti, kammaṭṭhānaṃ na paripatati, vuddhiṃ phātiṃ upagacchati.

* tāsu anuppajjamānāsu = 「それら(の受)が生じないとき(絶対処格)」
  * anuppajjamāna = 「生じない(現在分詞)」
* cittaṃ ekaggaṃ hoti = 「心は一境性となる(統一される)」
  * ekagga = 「一境性の・専一の」
* kammaṭṭhānaṃ na paripatati = 「業処(瞑想の所縁)は退落しない(崩れ落ちない)」
  * kammaṭṭhāna = 「業処・瞑想主題」/ paripatati = 「落ちる・退失する」

  * vuddhi = 「増大・成長」/ phāti = 「繁栄・増進」/ upagacchati = 「至る・近づく」

→「それら(の受)が生じないとき、心は一境性となり、業処は退落せず、増大と繁栄(向上)とに至る。」

6. Parimukhaṃ satiṃ upaṭṭhapetvā ti kammaṭṭhānābhimukhaṃ satiṃ ṭhapayitvā.

* parimukhaṃ satiṃ upaṭṭhapetvā ti = 「『面前に念を現起させて(parimukhaṃ satiṃ upaṭṭhapetvā)』とは」

7. Atha vā parī ti pariggahaṭṭho.

* parī ti = 「『pari(遍)』とは」
* pariggaha-aṭṭho = 「遍く把握するという義(意味)である」

10. Tena vuccati ‘‘parimukhaṃ sati’’n ti evaṃ paṭisambhidāyaṃ (paṭi. ma. 1.164) vutta-nayenā pettha attho daṭṭhabbo.

  * vutta-naya = 「説かれた仕方・趣旨」/ daṭṭhabba = 「見られるべき・理解されるべき」

→「それゆえに『遍く出離の念(parimukhā sati)』と、このように『無礙解道』(1.164)に説かれた方法によっても、ここでの意味は理解されるべきである。」

11. Tatrāyaṃ saṅkhepo, pariggahita-niyyānaṃ satiṃ katvā ti.

  * saṅkhepa = 「要約・略説」

  * pariggahita = 「把握された・遍取された(過去分詞)」/ niyyāna = 「出離」/ katvā = 「なして・作して」

→「そこでの要約はこうである──『遍く把握された出離をもつ念をなして』(という趣旨である)。」

## 219 (1)

219.Sosatova assasati sato passasatīti so bhikkhu evaṃ nisīditvā evañca satiṃ upaṭṭhapetvā taṃ satiṃ avijahanto sato eva assasati sato passasati, satokārī hotīti vuttaṃ hoti. Idāni yehākārehi satokārī hoti, te dassetuṃ dīghaṃ vā assasantotiādimāha. Vuttañhetaṃ paṭisambhidāyaṃ ‘‘so satova assasati sato passasatī’’ti etasseva vibhaṅge –

‘‘Bāttiṃsāya ākārehi sato kārī hoti. Dīghaṃ assāsavasena cittassa ekaggataṃ avikkhepaṃ pajānato sati upaṭṭhitā hoti. Tāya satiyā tena ñāṇena sato kārī hoti. Dīghaṃ passāsavasena…pe… paṭinissaggānupassī assāsavasena. Paṭinissaggānupassī passāsavasena cittassa ekaggataṃ avikkhepaṃ pajānato sati upaṭṭhitā hoti. Tāya satiyā tena ñāṇena sato kārī hotī’’ti (paṭi. ma. 1.165).

### 対訳

1. So sato va assasati sato passasatī ti （釈される句＝レンマ）

* so = 「彼は」（指示代名詞・主格）
* sato = 「念ある者として、気づきをもって」（sarati の過去分詞 sata の主格、形容詞的用法）
* va = eva 「まさに、こそ」（強調）
* assasati = 「入息する（息を吸う）」（√sas, 三・単・現）
* sato = 「念ある者として」
* passasati = 「出息する（息を吐く）」
* ti = iti（引用標識）

→ 「『彼は念をもって入息し、念をもって出息する』とは――」

2. so bhikkhu evaṃ nisīditvā evañca satiṃ upaṭṭhapetvā taṃ satiṃ avijahanto sato eva assasati sato passasati, satokārī hotī ti vuttaṃ hoti.

* so bhikkhu = 「その比丘は」
* evaṃ = 「このように」
* nisīditvā = 「坐して（坐った後に）」（nisīdati の連続体）
* evañca = evaṃ + ca 「そしてこのように」
* satiṃ = 「念を」（対格）
* upaṭṭhapetvā = 「現前させて、確立して（の後に）」（upaṭṭhāti の使役・連続体）

* avijahanto = 「捨て去ることなく」（a + vijahati の現在分詞）

→ 「その比丘は、このように坐し、このように念を確立して、その念を捨てることなく、まさに念をもって入息し、念をもって出息する、すなわち『念をなす者となる』と説かれているのである。」

* te = 「それら（の行相）を」（対格・複数）
* dassetuṃ = 「示すために」（不定詞）
* dīghaṃ vā assasanto = 「あるいは長く入息しつつ」（dīghaṃ＝長く・副詞的対格、vā＝あるいは、assasanto＝入息する者・現在分詞）
* ti ādim = iti + ādiṃ 「～等（で始まる文）を」
* āha = 「(世尊は)説かれた」

→ 「いまや、いかなる行相によって『念をなす者』となるのか、それらを示すために、『あるいは長く入息しつつ』等(の句)を説かれた。」

* – = （以下に引用が続く）

3. Sakalassa passāsakāyassa ādimajjhapariyosānaṃ viditaṃ karonto pākaṭaṃ karonto passasissāmī ti sikkhati.

  * sakalassa passāsakāyassa = 「出息という身体(息の集まり)の全体の」(属格)
  * ādi-majjha-pariyosānaṃ = 「初め・中間・終わりを」
  * viditaṃ karonto pākaṭaṃ karonto = 「知られたものとなし、明瞭なものとなしつつ」
  * passasissāmī ti sikkhati = 「『私は出息しよう』と学ぶ」

→ 「出息という身体(息の全体)の、初め・中間・終わりを知られたものとなし、明瞭なものとなしつつ、『私は出息しよう』と学ぶのである。」

4. Evaṃ viditaṃ karonto pākaṭaṃ karonto ñāṇasampayuttacittena assasati ceva passasati ca.

  * evaṃ = 「このように」
  * viditaṃ karonto pākaṭaṃ karonto = 「知られたものとなし、明瞭なものとなしつつ」
  * ñāṇa-sampayutta-cittena = 「智と相応した心によって」(ñāṇa=智, sampayutta=相応した, citta=心)
  * assasati ceva passasati ca = 「入息もし、また出息もする」

→ 「このように(初・中・終を)知られたものとなし、明瞭なものとなしつつ、智と相応した心によって、入息もし、また出息もするのである。」

5. Tasmā ‘‘assasissāmi passasissāmī’’ti sikkhatī ti vuccati.

  * tasmā = 「それゆえに」
  * assasissāmi passasissāmī ti = 「『私は入息しよう、出息しよう』と」
  * sikkhati = 「学ぶ」
  * ti vuccati = 「と言われる」

→ 「それゆえに、『私は入息しよう、出息しよう』と学ぶ、と説かれるのである。」

6. Ekassa hi bhikkhuno cuṇṇavicuṇṇavisaṭe assāsakāye passāsakāye vā ādi pākaṭo hoti, na majjhapariyosānaṃ.

  * ekassa hi bhikkhuno = 「実に、ある一人の比丘にとっては」
  * cuṇṇa-vicuṇṇa-visaṭe = 「粉々に砕け散った(息の身体において)」(cuṇṇa=粉砕した, vicuṇṇa=さらに細かく砕けた, visaṭa=拡散した。処格)
  * assāsakāye passāsakāye vā = 「入息の身体において、あるいは出息の身体において」(処格)
  * ādi pākaṭo hoti = 「初めが明瞭になる」
  * na majjha-pariyosānaṃ = 「中間と終わりは(明瞭になら)ない」

→ 「実に、ある一人の比丘にとっては、粉々に砕け散った入息の身体あるいは出息の身体において、初めは明瞭になるが、中間と終わりは(明瞭になら)ない。」

7. So ādimeva pariggahetuṃ sakkoti, majjhapariyosāne kilamati.

  * so = 「彼は」
  * ādim eva = 「ただ初めだけを」
  * pariggahetuṃ sakkoti = 「把捉することができる」
  * majjha-pariyosāne = 「中間と終わりにおいては」(処格)
  * kilamati = 「疲弊する、苦労する」

→ 「彼はただ初めだけを把捉することができ、中間と終わりにおいては苦労する。」

8. Ekassa majjhaṃ pākaṭaṃ hoti, na ādipariyosānaṃ.

  * ekassa = 「ある者にとっては」
  * majjhaṃ pākaṭaṃ hoti = 「中間が明瞭になる」
  * na ādi-pariyosānaṃ = 「初めと終わりは(明瞭になら)ない」

→ 「ある者にとっては中間が明瞭になるが、初めと終わりは(明瞭になら)ない。」

9. Ekassa pariyosānaṃ pākaṭaṃ hoti, na ādimajjhaṃ.

  * ekassa = 「ある者にとっては」
  * pariyosānaṃ pākaṭaṃ hoti = 「終わりが明瞭になる」
  * na ādi-majjhaṃ = 「初めと中間は(明瞭になら)ない」

→ 「ある者にとっては終わりが明瞭になるが、初めと中間は(明瞭になら)ない。」

10. So pariyosānaṃyeva pariggahetuṃ sakkoti, ādimajjhe kilamati.

  * so = 「彼は」
  * pariyosānaṃ yeva = 「ただ終わりだけを」
  * pariggahetuṃ sakkoti = 「把捉することができる」
  * ādi-majjhe = 「初めと中間においては」(処格)
  * kilamati = 「苦労する」

→ 「彼はただ終わりだけを把捉することができ、初めと中間においては苦労する。」

11. Ekassa sabbampi pākaṭaṃ hoti, so sabbampi pariggahetuṃ sakkoti, na katthaci kilamati, tādisena bhavitabban ti dassento āha – ‘‘sabbakāyapaṭisaṃvedī assasissāmī ti …pe… passasissāmī ti sikkhatī’’ti.

  * ekassa = 「ある者にとっては」
  * sabbam pi pākaṭaṃ hoti = 「すべてもまた明瞭になる」
  * so sabbam pi pariggahetuṃ sakkoti = 「彼はすべてをも把捉することができる」
  * na katthaci kilamati = 「どこにおいても苦労しない」
  * tādisena bhavitabban ti = 「そのような者であるべきである、と」(bhavitabba=あるべき)
  * dassento āha = 「示しつつ説いた」(dassento=示しつつ, āha=説いた・言った)
  * sabbakāyapaṭisaṃvedī …pe… sikkhatī ti = 「『全身を感受しつつ……と学ぶ』と」

→ 「ある者にとってはすべてもまた明瞭になり、彼はすべてをも把捉することができ、どこにおいても苦労しない。(行者は)そのような者であるべきである、と示しつつ、(世尊は)『全身を感受しつつ、私は入息しよう……出息しよう、と学ぶ』と説かれたのである。」

12. Tattha sikkhatī ti evaṃ ghaṭati vāyamati.

  * tattha = 「そこにおいて、その(文)中の」
  * sikkhatī ti = 「『学ぶ』とは」
  * evaṃ = 「このように」
  * ghaṭati = 「努める」
  * vāyamati = 「精進する、励む」

→ 「そこにおいて『学ぶ』とは、このように努め、精進する(ことである)。」

13. Yo vā tathābhūtassa saṃvaro, ayamettha adhisīlasikkhā.

  * yo vā = 「あるいは～であるところの(ものは)」
  * tathābhūtassa = 「そのようになった者の」
  * saṃvaro = 「律儀、防護」
  * ayam ettha = 「これがここでは」
  * adhisīla-sikkhā = 「増上戒学」

→ 「あるいは、そのようになった者の律儀、これがここでは増上戒学である。」

14. Yo tathābhūtassa samādhi, ayaṃ adhicittasikkhā.

  * yo = 「～であるところの(ものは)」
  * tathābhūtassa samādhi = 「そのようになった者の定」
  * ayaṃ = 「これが」
  * adhicitta-sikkhā = 「増上心学」

→ 「そのようになった者の定、これが増上心学である。」

15. Yā tathābhūtassa paññā, ayaṃ adhipaññāsikkhā ti

  * yā = 「～であるところの(ものは)」(女性)
  * tathābhūtassa paññā = 「そのようになった者の慧」
  * ayaṃ = 「これが」
  * adhipaññā-sikkhā ti = 「増上慧学である、と」

→ 「そのようになった者の慧、これが増上慧学である、と(いうように)。」

16. imā tisso sikkhāyo tasmiṃ ārammaṇe tāya satiyā tena manasikārena sikkhati āsevati bhāveti bahulīkarotī ti evamettha attho daṭṭhabbo.

  * imā tisso sikkhāyo = 「これら三学を」
  * tasmiṃ ārammaṇe = 「その所縁において」(処格)
  * tāya satiyā = 「その念によって」(具格)
  * tena manasikārena = 「その作意によって」(具格)
  * sikkhati = 「学(修学)し」
  * āsevati = 「習修し、親近し」
  * bhāveti = 「修習し」
  * bahulīkaroti = 「多く修し、頻繁に行う」
  * ti = 「と」
  * evam ettha attho daṭṭhabbo = 「このようにここでは意味が見られるべきである」

→ 「これら三学を、その所縁において、その念によって、その作意によって、学修し、習修し、修習し、多く修するのである、と――このようにここでは意味が理解されるべきである。」

1. Tattha yasmā purimanaye kevalaṃ assasitabbaṃ passasitabbam eva, na ca aññaṃ kiñci kātabbaṃ.

## 220 (3)

Seyyathāpi purisassa dhāvitvā, pabbatā vā orohitvā, mahābhāraṃ vā sīsato oropetvā ṭhitassa oḷārikā assāsapassāsā honti, nāsikā nappahoti, mukhena assasantopi passasantopi tiṭṭhati. ### 対訳

1. Seyyathāpi purisassa dhāvitvā, pabbatā vā orohitvā, mahābhāraṃ vā sīsato oropetvā ṭhitassa oḷārikā assāsapassāsā honti, nāsikā nappahoti, mukhena assasantopi passasantopi tiṭṭhati.

  * seyyathāpi = 「あたかも～のごとく」(譬喩の導入)
  * purisassa = 「人にとって」(属格)
  * dhāvitvā = 「走って(後に)」(dhāvati の連続体)
  * pabbatā vā orohitvā = 「あるいは山から降りて」(pabbata=山, orohati=降りる)
  * mahābhāraṃ vā sīsato oropetvā = 「あるいは重い荷を頭から下ろして」(mahābhāra=重荷, sīsa=頭, oropeti=下ろす)
  * ṭhitassa = 「立っている者にとって」(属格)
  * oḷārikā assāsapassāsā honti = 「粗大な入息・出息となる」
  * nāsikā nappahoti = 「鼻(鼻息)では足りない」
  * mukhena assasanto pi passasanto pi tiṭṭhati = 「口によって入息し、また出息しつつとどまる」

→ 「あたかも、人が走った後に、あるいは山から降りて、あるいは重い荷を頭から下ろして立っているとき、(その人の)入息・出息は粗大なものとなり、鼻(鼻息)では足りず、口によって入息し、また出息しつつとどまるように。」

2. Yadā panesa taṃ parissamaṃ vinodetvā nhatvā ca pivitvā ca allasāṭakaṃ hadaye katvā sītāya chāyāya nipanno hoti, athassa te assāsapassāsā sukhumā honti ‘‘atthi nu kho natthī’’ti vicetabbatākārappattā,

  * yadā pana esa = 「しかし、この(人)が～する時には」
  * taṃ parissamaṃ vinodetvā = 「その疲労を取り除いて」(parissama=疲労, vinodeti=除去する)
  * nhatvā ca pivitvā ca = 「沐浴し、また(水を)飲んで」(nhāyati=沐浴する, pivati=飲む)
  * allasāṭakaṃ hadaye katvā = 「濡れた布を胸(心臓のあたり)に当てて」(alla=濡れた, sāṭaka=布, hadaya=胸・心臓)
  * sītāya chāyāya nipanno hoti = 「涼しい木陰に横たわっている」(sīta=涼しい, chāyā=陰, nipanna=横たわった)
  * atha assa = 「そのとき彼の」
  * te assāsapassāsā sukhumā honti = 「その入息・出息は微細なものとなる」
  * atthi nu kho natthī ti = 「『有るのか、無いのか』と」
  * vicetabbatā-ākāra-ppattā = 「探究されるべき様相に達したものとなる」

→ 「しかし、この(人)がその疲労を取り除き、沐浴し、また(水を)飲み、濡れた布を胸に当てて涼しい木陰に横たわるとき、そのとき彼の入息・出息は微細なものとなり、『(息は)有るのか、無いのか』と探究されるべき様相に達したものとなる。」

3. evameva imassa bhikkhuno pubbe apariggahitakāle kāyo ca …pe… vicetabbatākārappattā honti.

  * evam eva = 「まさにそのように」
  * imassa bhikkhuno = 「この比丘にとって」(属格)
  * pubbe apariggahita-kāle = 「以前、(息が)まだ把捉されていない時には」
  * kāyo ca …pe… = 「身も……(以下、前段に準ずる中略)」
  * vicetabbatā-ākāra-ppattā honti = 「探究されるべき様相に達したものとなる」

→ 「まさにそのように、この比丘にとっても、以前、(息が)まだ把捉されていない時には、身も〔心も焦燥を伴い粗大であり……(中略)……それら(身と心)が静まった時には入息・出息は微細なものとなって〕『有るのか、無いのか』と探究されるべき様相に達したものとなるのである。」

4. Taṃ kissa hetu?

  * taṃ = 「それは」
  * kissa hetu = 「何の理由によってか」(kissa=何の, hetu=原因)

→ 「それは何の理由によってか。」

5. Tathā hissa pubbe apariggahitakāle ‘‘oḷārikoḷārike kāyasaṅkhāre passambhemī’’ti ābhogasamannāhāramanasikārapaccavekkhaṇā natthi, pariggahitakāle pana atthi.

  * tathā hi = 「なぜなら、まことに」
  * assa = 「彼に」(属格)
  * pubbe apariggahita-kāle = 「以前、把捉されていない時には」
  * oḷārik'oḷārike kāyasaṅkhāre = 「粗大なる粗大な身行を」(oḷārika の重複、強意。「ことに粗大な身行」。処格/対格的)
  * passambhemī ti = 「『(私は)安息させよう』と」(passambheti の一人称)
  * ābhoga-samannāhāra-manasikāra-paccavekkhaṇā = 「専注・(心の)集中・作意・観察」(ābhoga=傾注・専注, samannāhāra=綜合的把握・集中, manasikāra=作意, paccavekkhaṇā=観察・省察)
  * natthi = 「存在しない」
  * pariggahita-kāle pana atthi = 「しかし把捉された時には存在する」

→ 「なぜなら、彼には、以前、(息が)まだ把捉されていない時には、『ことに粗大な身行を安息させよう』という専注・集中・作意・観察がないが、しかし(息が)把捉された時にはそれがあるからである。」

6. Tenassa apariggahitakālato pariggahitakāle kāyasaṅkhāro sukhumo hoti.

  * tena = 「それゆえに」
  * assa = 「彼にとって」(属格)
  * apariggahita-kālato = 「(息が)把捉されていない時よりも」(処格起点。比較)
  * pariggahita-kāle = 「把捉された時には」(処格)
  * kāyasaṅkhāro sukhumo hoti = 「身行は微細なものとなる」

→ 「それゆえに、彼にとって、(息が)まだ把捉されていない時に比べて、把捉された時には身行は微細なものとなるのである。」

7. Tenāhu porāṇā –

  * tena = 「それゆえに」
  * āhu = 「(彼らは)言った」(複数)
  * porāṇā = 「古人たち、古徳は」

→ 「それゆえに、古徳(古人たち)は(次のように)説いた――」

8. ‘‘Sāraddhe kāye citte ca, adhimattaṃ pavattati;
Asāraddhamhi kāyamhi, sukhumaṃ sampavattatī’’ti. (pārā. aṭṭha. 2.165);

  * sāraddhe kāye citte ca = 「身と心が焦燥している(激している)とき」(sāraddha=焦燥した・興奮した・激した。処格・絶対句)
  * adhimattaṃ pavattati = 「(身行・息は)過度に転起する」(adhimatta=過度の・甚だしい, pavattati=起こる・働く)
  * asāraddhamhi kāyamhi = 「身が焦燥していないとき」(a-sāraddha=焦燥していない。処格)
  * sukhumaṃ sampavattati = 「微細なものとして転起する」(sukhuma=微細な, sampavattati=よく起こる・働く)
  * ti = 「と」(引用)
  * (pārā. aṭṭha. 2.165) = 「(『律・波羅夷註』2.165)」(典拠表示)

→ 「『身と心が焦燥しているとき、(身行は)過度に転起し、身が焦燥していないとき、(身行は)微細なものとして転起する』と。(『律蔵・波羅夷註』2.165)」

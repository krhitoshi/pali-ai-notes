# vism_08 校訂履歴

vism_08.md (Visuddhimagga 第8章 安般念) の校訂記録.

### 2026/06/28 assāsa / passāsa の方向統一

- 内容: assāsa / assasati =「出息（外へ出る息）」, passāsa / passasati =「入息（内へ入る息）」に統一
- 対象: §215, §219, §220, §221（第一稿は assāsa を「入息」と訳していた）
- 不変: §216, §217, §218, §222-§238（元から方針どおり）
- 根拠: 本文 §219(2)「Assāso ti bahi nikkhamanavāto ... vinayaṭṭhakathāyaṃ vuttaṃ」（律註は assāsa=外へ出る息）。CLAUDE.md「翻訳の方針」に準拠
- 校訂: Claude Opus 4.8 + MCP（pali-dict / cscd-text）。詳細は vism_08_v1_check.md

### 2026/06/28 ānāpāna / ānāpānassati の訳語統一

- 内容: ānāpānassati の訳を「出入息念」に統一（旧訳の音写「安般念」と「入出息念」を置換）
- 単独 ānāpāna も「出入息」に統一（旧訳の音写「安般」を置換。例: ānāpāna 第四禅 = 出入息の第四禅）
- 対象: 複合語も含む（ānāpānassatisamādhi=出入息念三昧, Ānāpānassatikathā=出入息念論, ānāpānassatikammaṭṭhāna=出入息念の業処）
- 方針: assāsa=出息 / passāsa=入息 と整合させ, 音写でなく訳語に統一
- 校訂: Claude Opus 4.8

### 2026/06/28 不確かな語源の削除

- 内容: vīmaṃsati（§226）の語源「vi-√mṛś 由来」を削除（vīmaṃsati は √man の desiderative で vi-√mṛś は誤り）
- 方針: 確実でない語源・語根の分解は無理に書かず削除する
- 校訂: Claude Opus 4.8

### 2026/06/28 文語漢語を平易語に置換

- 内容: 日常で使わない漢語を平易な訳語に統一
  - 摂受→会得, 殊勝・勝妙→すぐれた, 把捉・遍取→把握, 顕示→明示, 遮遣→排除, 滲入→混ぜ物, 所作→務め, 厭逆→厭わしい, 懊悩→苦しみ, 希求→願い, 善男子→良家の子, 転起→働き, 現起→現前, 止息→静める, 通達→洞察
- 保持: 増上戒・増上心・増上慧, 瑜伽行者, 生起, および確立した仏教技術用語（随観・所縁・業処・寂静・近行・安止・無礙解・念処・軽安・遍知・寂滅・等持・数息・結跏趺坐・威儀・阿闍梨 など）
- 校訂: Claude Opus 4.8

### 2026/06/28 古語表現を平易語に置換

- 〜せん（意志）→ 〜しよう（出息せん→出息しよう 等）。われ…せん → 私は…しよう
- 〜せんがため → 〜するため（調御せんがため→調御するため）
- 〜せしめ（使役の古語）→ 〜させ（生起せしめる→生起させる 等, 25 件）
- 〜べし → 〜べきである（繋ぐべし→繋ぐべきである 等, 5 件）
- 代名詞「われ」は該当なし（言われ・問われ等の受身形のみ）
- 校訂: Claude Opus 4.8

### 2026/06/29 使役の古語「らしめ」を平易語に置換

- 内容: 使役の古語「〜らしめ」を文脈で 3 分して平易語に統一（21 行）
  - X ならしめる（断定なり + しむ）→ X にする: 寂静ならしめる→寂静にする（§48）,
    軽安ならしめ→軽安にし（§419, §422）, 寂静ならしめつつ→寂静にしつつ
    （§1900, §1915, §1918, §2285, §2288）, 円満ならしめる→円満にする
    （§5803, §5805, §5806, §5808, §5812 計 6 箇所）,
    飽くことなからしめる→飽きさせない（§299, §302）
  - 動詞の使役 V-らしめ → V-(さ)せ: 入らしめて→入らせて（§591, §596）,
    造らしめよ→造らせよ（§755）, 至らしめて→至らせて（§4838）
  - たらしめる（断定たり + しむ）→ 〜とする: たらしめること→とすること
    （§5818, §5822）, 既知ならしめること→既知にすること（§5818）
- 併修: 同一行に残っていた「生ぜしめない」（せしめ系）→「生じさせない」（§299）
- 不変: 「所縁とすること」（§2445 等）は元から平易表現で対象外
- 落とし穴: 「寂静にしつつ」は直前の「静め, 滅させ」と並ぶが別動作のため同義反復に
  ならない（§1918, §2288 で確認）
- 校訂: Claude Opus 4.8

### 2026/06/29 「所依たる」を平易語に置換

- 内容: 漢語「所依」(sannissaya) を「依りどころ」に, 断定の古語「たる」を「となる」に
  統一し,「所依たる」->「依りどころとなる」とした（6 行）
  - §239: 所依たる念 -> 依りどころとなる念
  - §483: 依りどころ（所依）たる教法 -> 依りどころとなる教法（併記の (所依) も解消）
  - §484: sannissaya =「所依・依拠・基盤」->「依りどころ・基盤」
  - §485: bhūta の語釈「～たる・～となった」->「～である・～となった」,
    sannissaya-bhūta「所依たる」->「依りどころとなる」
  - §489, §494: 〔所依たる〕在り方 -> 〔依りどころとなる〕在り方
  - §494: 人の所依たる教法 -> 人の依りどころとなる教法
- 区別: 所縁 (保持リスト) とは別語. 所依は保持対象でないため平易化
- 範囲外: 他の「たる」(頂点たる・一切知者たる・最上の果たる・地盤たる・
  寂静たること 等) は今回未着手
- 校訂: Claude Opus 4.8

### 2026/06/29 残りの断定の古語「たる」を平易語に置換

- 内容: 断定の古語「〜たる」を文脈で訳し分けて平易語に統一（11 行）
  - 同格・同一性 -> 〜である: 沙門たること->沙門であること（§502）,
    一切知者たる仏->一切知者である仏（§683, §699）, 最上の果たる阿羅漢果->
    最上の果である阿羅漢果（§710, §712）, 捨遣たる随観->捨遣である随観（§5648）,
    寂静たること->寂静であること（§5760, §5764）
  - 役割・機能 -> 〜となる: 頂点たるもの->頂点となるもの（§682, §699）,
    地盤たる阿蘭若->地盤となる阿蘭若（§866）
  - 形容動詞タリ活用 -> 騒然たる->騒がしい（§693）
  - 語釈中の「～たる」(bhūta 等)->「～である」（§682, §697）
- 対象外: 動詞「当たる」(§4206),「わたる」(§4957) は古語の断定でないため不変
- 校訂: Claude Opus 4.8

### 2026/06/29 kūṭa の訳語を DoP に合わせて修正

- 内容: §216 の譬喩 (荒れた牛で乱れた心を喩える箇所) の kūṭa を
  「気が荒い・あばれる・性悪な」に統一（6 行）
  - kūṭagoṇa: 曲(くせ)ある牛（強情な牡牛）-> 気の荒い牡牛（§547, §554）
  - kūṭadhenu: 曲(くせ)ある／強情な牝牛 -> 気の荒い牝牛（§561, §562）
  - kūṭavaccha: 強情な仔牛 -> 気の荒い仔牛（§561, §565）
  - 語釈 kūṭa: 「曲がった・狡猾な・強情な・偽りの」->「(牛馬について)
    気が荒い・あばれる・性悪な」（本義はよこしまな・偽りの）（§548）
- 根拠: DoP kūṭa2 (mfn.) は false, fraudulent; lying; wild, vicious.
  DoP は kūṭagoṇa = "a savage ox", kūṭadhenu = "a vicious cow" を立項し,
  典拠に本箇所 (Vism 268,34 / 269,1) を挙げる
- 注意: ここの「wild」は野生 (feral) ではなく気性が荒い意. この牛は飼い牛で,
  dametukāmo (調御しようと欲する) 対象. 語源も kūṭa「よこしま・罠」由来で
  自然界の野生とは別系統
- 校訂: Claude Opus 4.8 + MCP (DoP)

### 2026/06/29 naya の漢訳「理趣」を「方法」に統一

- 内容: naya の訳語のうち漢訳寄りの「理趣」を「方法」に置換（8 行）
  - §150 修習の方軌（…理趣）-> 修習の方法（訳文 §152/§155/§162 の
    「修習の方軌」も方法に統一）, §210/§212 この理趣 -> この方法,
    §1597 この方法(理趣) -> この方法, §2156/§2157 理趣 -> 方法,
    §2227/§2232 説かれた理趣 -> 説かれた方法
- 根拠: DoP naya 1 = way, way of proceeding; (right) method; method of
  interpretation; inference. 本章の用例 (Vism 274,17 等) も sense 1.
  「理趣」は道理・趣旨寄りの漢訳で naya の直接的語義からずれる
- 範囲外: 「仕方・方式・要領・趣旨」(§1088, §1855, §2913, §2996, §3070 等) は
  既に平易な訳で意味も通るため今回は不変. 全面的に「方法」へ統一するかは保留
- 校訂: Claude Opus 4.8 + MCP (DoP)

### 2026/06/29 asecanaka の語釈「灌ぎ混ぜ/灌ぎ加え」を平易語に

- 内容: §319-332 の asecanaka 語釈の難解な逐語訳を平易化（4 行）
  - 灌ぎ混ぜ (secana) -> 注ぎ足し, 灌ぎ加え/混和 (anāsitta) -> 注ぎ足し
  - §322 asecanaka 語釈, §325/§326 secana, §327 anāsittaka, §332 訳文
- 根拠: DoP asecanaka = "never causing surfeit; (註: not requiring anything
  added)". 註釈 Mp III 315「甘くするため別の味を注ぎ足していない (anāsittaka)」,

- 技術ラベル「不雑」も平易化: 不雑 -> 混ぜ物を要しない／要せず（§321, §325,

- 校訂: Claude Opus 4.8 + MCP (DoP)

### 2026/06/29 文語の連体・断定「なる」(形容動詞ナリ) を平易語に

- 内容: 形容動詞ナリ活用連体形・断定の「なる」を平易な「な/である/の」に置換
  - 形容動詞 -> な: 寂静なる（§36, §252）, 劣悪なる（§386, §388）,
    粗大なる/微細なる -> 粗大さ/微細さ・粗大な（§1924, §1944, §2035）,
    楽なる触（§4472, §4479）, 楽なること->楽であること（§964）,
    無常（なるもの）->無常なもの（§5503, §5505, §5518, §5521）,
    不放逸なる（§6057, §6068）
  - 断定: 安楽なる住なる -> 安楽な住である（§321, §323, §332, §371, §374）
  - 漢語+なる -> の: 不善なる諸法->不善の諸法（§392, §395）,
    内なる/外なる風->内の/外の風（§1272, §1275, §1280, §1288）
  - 悪なる不善の諸法 -> 悪しき不善の諸法（pāpaka。§45, §50, §56）
  - 「という」の意のなる: いわゆる豹なるもの -> いわゆる豹（§833, §843）
- 範囲外 (現代でも生きる慣用のため保持): いかなる (15), 聖なる (2),
  大いなる (12)
- 校訂: Claude Opus 4.8

### 2026/06/29 sukha-samphassa の訳「楽な触」を平易に

- 内容: 楽な触 -> 心地よい感触（§4472, §4479）
- 理由: ここの samphassa は相 (nimitta) の触れ心地の描写 (兜羅綿のよう等)
  で, 技術用語の「触 (phassa)」でなく「感触」の意. 「楽な触」は触を単独名詞に
  使い不自然だった
- 根拠: PED sukha-samphassa = pleasant to touch. (DoP は s- が未刊行で未収録)
- 校訂: Claude Opus 4.8

### 2026/07/01 漢語「依拠」を平易語に置換

- 内容: nissāya (ni-√si 連続体・後置詞) 系の訳語「依拠」を文脈で訳し分けて平易化
  (4 箇所). 「依って」は 依る/因る/拠る と紛らわしいため採らない
  - §690 padaṭṭhāna 語釈: 直接の依拠 -> 直接のよりどころ (名詞. 所依->依りどころ と揃える)
  - §793 nissāya の対格支配の注記: ～に依拠して -> ～を頼りにして
  - §803 訳文: 山の険所に依拠して身を潜め -> 山の険所を頼りに身を潜め (手段)
  - §3027 paṭibaddha 語釈: 結びつけられた, 依拠した -> 結びつけられた, 依存した
- 根拠: nissāya は多義 (Main/PTS/村上: によって・頼りにして・にもとづいて・の故に).
  文脈依存のため一律置換せず, 手段は「頼りに」, 名詞は「よりどころ」, paṭibaddha の
  語釈は「依存」に分けた. 同語根 sannissaya は既に 所依->依りどころ に平易化済み
- 範囲: まず vism_08 のみ. patis_a_1_03 等 他ファイルの「依拠」(nissāya/nissita) は未着手
- 校訂: Claude Opus 4.8 + MCP (pali-dict)

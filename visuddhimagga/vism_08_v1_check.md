## 指摘

### 高

1. assāsa / passāsa の方向がファイル内で不統一 (最重要)
   - 誤りは §215, §219, §220, §221 に集中 (assāsa / assasati を「入息」と訳出).
     §216-218, §222-238 は「出息」または中立 (複合語のみ) で方針どおり.
     同一ファイルで逆向きが混在していた
   - 節ごとの状況:

     | 節 | 第一稿の assasati / assāsa | 方針 (出息) |
     |---|---|---|
     | §215(6)-(12) | 入息 | 違反 → 要修正 |
     | §217(2)(4) | 出息 | 準拠 (基準形) |
     | §219(1)(3)(4) の動詞 | 入息 | 違反 → 要修正 |
     | §219(2) 名詞定義訳 | 外へ出る息 (出息) | 準拠 |
     | §220, §221 | 入息 | 違反 → 要修正 |
     | §222, §223 | 出入息(複合)のみ/なし | 準拠 |
     | §224-230 | 出息 | 準拠 |
     | §231-238 | 出息 | 準拠 |

   - §219 は内部矛盾: 名詞定義 (§219(2) 項2-3「Assāso = 外へ出ていく風」) は正しいのに,
     同節の動詞 assasati や §219(1)(3)(4) は「入息」と逆に訳している
   - 根拠: 本文 §219(2) 自身「Assāso ti bahi nikkhamanavāto. Passāso ti anto
     pavisanavāto ti vinayaṭṭhakathāyaṃ vuttaṃ」(assāsa=外へ出る息), 「Suttantaṭṭha-
     kathāsu pana uppaṭipāṭiyā āgataṃ」(経註は逆順). DoP/PTS も Vism 272 を
     「assāsa=outgoing, passāsa=ingoing」と引く
   - 提案: §215, §219, §220, §221 の assāsa/assasati→出息, passāsa/passasati→入息
     に統一 (他の節は修正不要)

2. 方向の注記の取り違え 2 箇所
   - §215(1) 行83「〔本伝統では assasati＝入息、passasati＝出息〕」
   - §221(6) 行2636「assāsa = 入息, passāsa = 出息 (Visuddhimagga・経註の慣例とは逆)」
   - 指摘: いずれも Vism が採る向き (律註順 assāsa=出息) を取り違えている
   - 提案: 「Vism は律註順で assāsa=出息 / passāsa=入息 を採る (経註は逆)」に訂正

### 中

3. vīmaṃsati の語源 (§226 行3663)
   - 第一稿: 「(vi-√mṛś 由来、意欲動詞 vīmaṃsati)」
   - 指摘: 語根が誤り. vīmaṃsati は √man の desiderative (PTS / Concise)。
     「意欲動詞 (desiderative)」の指摘は正しいが √mṛś は √man に直す
     (√mṛś は sammasati = saṃ+√mṛś の方)

4. vātūpaladdhi (§221(4) 行2486)
   - 第一稿: 「風(気息)の獲得」
   - 指摘: upaladdhi = 存在・現起・知覚 (DoP「the being found, existence, perception」,
     Paṭis-a 517「upaladdhī ti viññāṇaṃ」, 複合 vāt'-upaladdhi も DoP に登録)。
     「獲得」は誤り
   - 提案: 「(気息の) 現起・知覚」に

5. sāraddha の訳が不統一 (§227 と §224 / §228)
   - §227 (偈): 「焦燥した・興奮した・激した」(正)
   - §224(1) 行3406/3422: 「こわばり、動揺」, §228: 「こわばり」と弱い
   - 指摘: saṃraddha = 激情の・興奮した (Main), violent/angry, passaddha の対義 (PTS)。
     「焦燥・興奮・ざわつき」が原義. §227 に揃えて統一する
   - 注: §228 の kāyadaratha (daratha = 懊悩) は別語で「身の懊悩・こわばり」で可

6. pītisīsa (§234(2) 項7)
   - 第一稿: 「喜びを主題(首目)として受が説かれている」
   - 指摘: mahāṭīkā は「pīti-apadesena taṃsampayuttā vedanā vuttā, na pītī」
     (喜びの名目のもとに, 相応する受が説かれた, 喜び自体ではない)。
     「主題として」だと喜び自体に読めるので「喜びの名目のもとに相応する受」と註記

### 低

7. paṭicodeti (§229 項8-9): 「責める・励ます・叱咤すべき」の「励ます」は辞書にない
   (DoP「reprove, admonish」, 本箇所 Vism 283,30 を引用)。「励ます」は削除候補
8. asecanaka (§237 項2): 「混じり気のない (それ自体で満足を与える)」の括弧内は
   字義 (DoP「飽きさせない」, 註 anāsittaka) からやや踏み込んだ補足。任意
9. ādisamannāhāra (§216(2) 項3): 「作意」は註で samannāhāra ≒ manasikāra の用例が
   あり妥当だが, 辞書字義 (存念/注意) との差は註記推奨
10. assāsapassāsa の複合訳が「出入息」「入出息」で混在 (§218 等)。語順上「出入息」が
    方針整合的。慣用の ānāpānassati =「入出息念」とは別語として扱う
11. sūcipāsavedhana (§230 項7) [要確認]: 「針孔をあける錐」。vedhana=穿つもので妥当だが
    「針の眼に糸を通すもの」とも取りうる。註に語釈なく断定不可

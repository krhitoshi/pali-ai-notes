### 対訳

8. Ānāpānassatisuttaṃ

* ānāpāna = 「出入息」（āna「出息」+ apāna「入息」）

* sati = 「念」

* suttaṃ = 「経」

→「出入息念経」

1. Evaṃ me sutaṃ –

* evaṃ = 「このように」

* me = 「私によって」

* sutaṃ = 「聞かれた（suṇāti の過去分詞）」

→「このように私は聞いた。」

2. ekaṃ samayaṃ bhagavā sāvatthiyaṃ viharati pubbārāme migāramātupāsāde

* ekaṃ samayaṃ = 「あるとき」

* bhagavā = 「世尊は」

* sāvatthiyaṃ = 「サーヴァッティー（舎衛城）に」

* viharati = 「住んでいた」

* pubbārāme = 「東園（プッバーラーマ）の」

  * pubba-ārāma = 「東の園」

* migāramātupāsāde = 「ミガーラマーター（鹿子母）の高楼に」

  * migāra-mātu-pāsāda = 「ミガーラの母（＝ヴィサーカー）の高楼」

→「あるとき、世尊はサーヴァッティーの東園、ミガーラマーターの高楼に住んでいた。」

3. sambahulehi abhiññātehi abhiññātehi therehi sāvakehi saddhiṃ –

* sambahulehi = 「多くの」

* abhiññātehi abhiññātehi = 「よく知られた、名高い（それぞれに名高い）」

  * 繰り返しは「それぞれによく知られた」という強調・分配の表現

* therehi = 「長老たちである」

* sāvakehi = 「弟子たちと」

* saddhiṃ = 「共に」

→「多くの、それぞれに名高い長老の弟子たちと共に――」

4. āyasmatā ca sāriputtena āyasmatā ca mahāmoggallānena āyasmatā ca mahākassapena āyasmatā ca mahākaccāyanena āyasmatā ca mahākoṭṭhikena āyasmatā ca mahākappinena āyasmatā ca mahācundena āyasmatā ca anuruddhena āyasmatā ca revatena āyasmatā ca ānandena,

* āyasmatā ca = 「尊者～と（āyasmant の具格）」

* sāriputtena = 「サーリプッタ（舎利弗）」

* mahāmoggallānena = 「マハーモッガッラーナ（大目連）」

* mahākassapena = 「マハーカッサパ（大迦葉）」

* mahākaccāyanena = 「マハーカッチャーヤナ（大迦旃延）」

* mahākoṭṭhikena = 「マハーコッティカ（大拘絺羅）」

* mahākappinena = 「マハーカッピナ（大劫賓那）」

* mahācundena = 「マハーチュンダ（大純陀）」

* anuruddhena = 「アヌルッダ（阿那律）」

* revatena = 「レーヴァタ（離婆多）」

* ānandena = 「アーナンダ（阿難）」

→「すなわち、尊者サーリプッタ、尊者マハーモッガッラーナ、尊者マハーカッサパ、尊者マハーカッチャーヤナ、尊者マハーコッティカ、尊者マハーカッピナ、尊者マハーチュンダ、尊者アヌルッダ、尊者レーヴァタ、尊者アーナンダと、」

5. aññehi ca abhiññātehi abhiññātehi therehi sāvakehi saddhiṃ.

* aññehi ca = 「そのほかの」

* abhiññātehi abhiññātehi = 「それぞれに名高い」

* therehi sāvakehi saddhiṃ = 「長老の弟子たちと共に」

→「そのほかの、それぞれに名高い長老の弟子たちと共に（住んでいた）。」

6. Tena kho pana samayena therā bhikkhū nave bhikkhū ovadanti anusāsanti.

* tena kho pana samayena = 「さて、そのとき」

* therā bhikkhū = 「長老の比丘たちは」

* nave bhikkhū = 「新参の比丘たちを」

  * nava = 「新しい、新参の」

* ovadanti = 「教誡していた」

* anusāsanti = 「教導していた」

→「さて、そのとき、長老の比丘たちは新参の比丘たちを教誡し、教導していた。」

7. Appekacce therā bhikkhū dasapi bhikkhū ovadanti anusāsanti,

* appekacce = 「ある者たちは（api + ekacce）」

* therā bhikkhū = 「長老の比丘たちは」

* dasapi bhikkhū = 「十人の比丘たちをも」

* ovadanti anusāsanti = 「教誡し、教導していた」

→「ある長老の比丘たちは十人の比丘たちを教誡し、教導し、」

8. appekacce therā bhikkhū vīsampi bhikkhū ovadanti anusāsanti,

* vīsampi = 「二十人をも（vīsaṃ + pi）」

→「ある長老の比丘たちは二十人の比丘たちを教誡し、教導し、」

9. appekacce therā bhikkhū tiṃsampi bhikkhū ovadanti anusāsanti,

* tiṃsampi = 「三十人をも（tiṃsaṃ + pi）」

→「ある長老の比丘たちは三十人の比丘たちを教誡し、教導し、」

10. appekacce therā bhikkhū cattārīsampi bhikkhū ovadanti anusāsanti.

* cattārīsampi = 「四十人をも（cattārīsaṃ + pi）」

→「ある長老の比丘たちは四十人の比丘たちを教誡し、教導していた。」

11. Te ca navā bhikkhū therehi bhikkhūhi ovadiyamānā anusāsiyamānā uḷāraṃ pubbenāparaṃ visesaṃ jānanti.

* te ca navā bhikkhū = 「そして、それらの新参の比丘たちは」

* therehi bhikkhūhi = 「長老の比丘たちによって」

* ovadiyamānā = 「教誡されつつ（ovadati の受動現在分詞）」

* anusāsiyamānā = 「教導されつつ（anusāsati の受動現在分詞）」

* uḷāraṃ = 「大いなる、すぐれた」

* pubbenāparaṃ = 「以前よりもさらに後の、順次の（pubbena aparaṃ）」

  * 「前の状態と比べて後にはより高い」という段階的向上を表す

* visesaṃ = 「殊別、卓越（ここでは修行上の向上・達成）」

* jānanti = 「知る、体得する」

  * 異読: pajānanti（了知する）、sañjānanti（認知する）

→「そして、それらの新参の比丘たちは、長老の比丘たちに教誡され、教導されて、以前にもましてすぐれた大いなる向上を体得していた。」

1. Tena kho pana samayena bhagavā tadahuposathe pannarase pavāraṇāya puṇṇāya puṇṇamāya rattiyā bhikkhusaṅghaparivuto abbhokāse nisinno hoti.

* tena kho pana samayena = 「さてその時」

* bhagavā = 「世尊は」

* tadahuposathe = 「その布薩の日に」（tad-ahu-uposatha）

  * tad-ahu = 「その日」

  * uposatha = 「布薩（半月ごとの集会日）」

* pannarase = 「十五日の」

* pavāraṇāya = 「自恣の（日である）」

  * pavāraṇā = 「自恣（雨安居の終わりに互いに過失の指摘を招く行事）」

* puṇṇāya puṇṇamāya rattiyā = 「満ちた満月の夜に」

  * puṇṇā = 「満ちた（pūrati の過去分詞）」

  * puṇṇamā = 「満月」

* bhikkhusaṅghaparivuto = 「比丘僧団に囲まれて」

  * parivuta = 「囲まれた（pari + vṛ の過去分詞）」

* abbhokāse = 「野外に、露地に」

* nisinno hoti = 「坐っていた」

→「さてその時、世尊は、その十五日の布薩、自恣の日、満ちた満月の夜に、比丘僧団に囲まれて野外に坐っていた。」

2. Atha kho bhagavā tuṇhībhūtaṃ tuṇhībhūtaṃ bhikkhusaṅghaṃ anuviloketvā bhikkhū āmantesi –

* atha kho = 「そこで」

* tuṇhībhūtaṃ tuṇhībhūtaṃ = 「まったく沈黙している」

  * tuṇhībhūta = 「沈黙した」（繰り返しは強調）

* bhikkhusaṅghaṃ = 「比丘僧団を」

* anuviloketvā = 「見渡して」（anu + vi + lok の連続体）

* bhikkhū āmantesi = 「比丘たちに呼びかけた」

→「そこで世尊は、まったく沈黙している比丘僧団を見渡して、比丘たちに呼びかけた。」

3. ‘‘āraddhosmi, bhikkhave, imāya paṭipadāya; āraddhacittosmi, bhikkhave, imāya paṭipadāya.

  * āraddha = 「満足した、喜んだ」

* bhikkhave = 「比丘たちよ」

* imāya paṭipadāya = 「この実践に（よって）」

  * paṭipadā = 「実践、行道」

* āraddhacitto asmi = 「私は心が満足している」

→「比丘たちよ、私はこの実践に満足している。比丘たちよ、私はこの実践に心が満足している。」

4. Tasmātiha, bhikkhave, bhiyyosomattāya vīriyaṃ ārabhatha appattassa pattiyā, anadhigatassa adhigamāya, asacchikatassa sacchikiriyāya.

* tasmātiha = 「それゆえここに」（tasmā + iha）

* bhiyyosomattāya = 「さらに一層」

* vīriyaṃ ārabhatha = 「精進を起こしなさい」

  * ārabhatha = 「起こしなさい（ārabhati の命令法二人称複数）」

* appattassa pattiyā = 「まだ得ていないものを得るために」

  * appatta = 「未得の（a + patta）」

  * patti = 「獲得」

* anadhigatassa adhigamāya = 「まだ到達していないものに到達するために」

  * anadhigata = 「未到達の」

  * adhigama = 「到達、証得」

* asacchikatassa sacchikiriyāya = 「まだ実証していないものを実証するために」

  * asacchikata = 「未実証の」

  * sacchikiriyā = 「実証、作証」

→「それゆえここに、比丘たちよ、まだ得ていないものを得るために、まだ到達していないものに到達するために、まだ実証していないものを実証するために、さらに一層、精進を起こしなさい。」

8. Te ca kho therā bhikkhū bhiyyosomattāya nave bhikkhū ovadanti anusāsanti.

→「その法を智慧によって考察し、精査し、遍く思察することに至る彼には、退縮しない精進が開始されている。」

7. Yasmiṃ samaye, bhikkhave, bhikkhuno taṃ dhammaṃ paññāya pavicinato pavicayato parivīmaṃsaṃ āpajjato āraddhaṃ hoti vīriyaṃ asallīnaṃ, vīriyasambojjhaṅgo tasmiṃ samaye bhikkhuno āraddho hoti, vīriyasambojjhaṅgaṃ tasmiṃ samaye bhikkhu bhāveti, vīriyasambojjhaṅgo tasmiṃ samaye bhikkhuno bhāvanāpāripūriṃ gacchati.

* vīriya-sambojjhaṅgo = 「精進覚支が」

→「比丘たちよ、その法を智慧によって考察し、精査し、遍く思察することに至る比丘に、退縮しない精進が開始されているその時、その時、比丘には精進覚支が開始されており、その時、比丘は精進覚支を修習し、その時、比丘の精進覚支は修習の完成に至る。」

8. Āraddhavīriyassa uppajjati pīti nirāmisā.

* āraddha-vīriyassa = 「精進を開始した者に」

* uppajjati = 「生じる（ud + √pad）」

* pīti = 「喜が」

* nirāmisā = 「物質的な報酬によらない（nis + āmisa）」

  * āmisa = 「財、物質的なもの、世俗的な味わい」

→「精進を開始した者には、物質的なものによらない喜が生じる。」

9. Yasmiṃ samaye, bhikkhave, bhikkhuno āraddhavīriyassa uppajjati pīti nirāmisā, pītisambojjhaṅgo tasmiṃ samaye bhikkhuno āraddho hoti, pītisambojjhaṅgaṃ tasmiṃ samaye bhikkhu bhāveti, pītisambojjhaṅgo tasmiṃ samaye bhikkhuno bhāvanāpāripūriṃ gacchati.

* pīti-sambojjhaṅgo = 「喜覚支が」

→「比丘たちよ、精進を開始した比丘に物質的なものによらない喜が生じるその時、その時、比丘には喜覚支が開始されており、その時、比丘は喜覚支を修習し、その時、比丘の喜覚支は修習の完成に至る。」

10. Pītimanassa kāyopi passambhati, cittampi passambhati.

* pīti-manassa = 「喜の意をもつ者にとって」

  * pīti-mana = 「喜ある意（心）」

* kāyo pi = 「身体も」

* passambhati = 「軽安になる、鎮まる（pa + √śrambh）」

* cittam pi = 「心も」

→「喜の意をもつ者は、身体も軽安になり、心も軽安になる。」

11. Yasmiṃ samaye, bhikkhave, bhikkhuno pītimanassa kāyopi passambhati, cittampi passambhati, passaddhisambojjhaṅgo tasmiṃ samaye bhikkhuno āraddho hoti, passaddhisambojjhaṅgaṃ tasmiṃ samaye bhikkhu bhāveti, passaddhisambojjhaṅgo tasmiṃ samaye bhikkhuno bhāvanāpāripūriṃ gacchati.

* passaddhi-sambojjhaṅgo = 「軽安覚支が」

  * passaddhi = 「軽安（passambhati の名詞形）」

→「比丘たちよ、喜の意をもつ比丘の身体も軽安になり、心も軽安になるその時、その時、比丘には軽安覚支が開始されており、その時、比丘は軽安覚支を修習し、その時、比丘の軽安覚支は修習の完成に至る。」

12. Passaddhakāyassa sukhino cittaṃ samādhiyati.

* passaddha-kāyassa = 「身体が軽安になった者にとって」

* sukhino = 「楽ある者にとって（sukhin の属格）」

* cittaṃ = 「心が」

* samādhiyati = 「集中する、定まる（saṃ + ā + √dhā の受動）」

→「身体が軽安になり、楽ある者は、心が定まる。」

13. Yasmiṃ samaye, bhikkhave, bhikkhuno passaddhakāyassa sukhino cittaṃ samādhiyati, samādhisambojjhaṅgo tasmiṃ samaye bhikkhuno āraddho hoti, samādhisambojjhaṅgaṃ tasmiṃ samaye bhikkhu bhāveti, samādhisambojjhaṅgo tasmiṃ samaye bhikkhuno bhāvanāpāripūriṃ gacchati.

* samādhi-sambojjhaṅgo = 「定覚支が」

→「比丘たちよ、身体が軽安になり楽ある比丘の心が定まるその時、その時、比丘には定覚支が開始されており、その時、比丘は定覚支を修習し、その時、比丘の定覚支は修習の完成に至る。」


# 118

118. Ākāsadhātuyaṃ ākāsagatanti ākāsabhāvaṃ gataṃ. Upādinnantiādinnaṃ gahitaṃ parāmaṭṭhaṃ, sarīraṭṭhakanti attho. Kaṇṇacchiddanti maṃsalohitādīhi asamphuṭṭhakaṇṇavivaraṃ. Nāsacchiddādīsupi eseva nayo. Yenacāti yena chiddena. Ajjhoharatīti anto paveseti, jivhābandhanato hi yāva udarapaṭalā manussānaṃ vidatthicaturaṅgulaṃ chiddaṭṭhānaṃ hoti. Taṃ sandhāyetaṃ vuttaṃ. Yattha cāti yasmiṃ okāse. Santiṭṭhatīti patiṭṭhāti. Manussānañhi mahantaṃ paṭaparissāvanamattañca udarapaṭalaṃ nāma hoti. Taṃ sandhāyetaṃ vuttaṃ. Adhobhāgaṃ nikkhamatīti yena heṭṭhā nikkhamati. Dvattiṃsahatthamattaṃ ekavīsatiyā ṭhānesu vaṅkaṃ antaṃ nāma hoti. Taṃ sandhāyetaṃ vuttaṃ. Yaṃ vā panaññampīti iminā sukhumasukhumaṃ cammamaṃsādiantaragatañceva lomakūpabhāvena ca ṭhitaṃ ākāsaṃ dasseti. Sesametthāpi pathavīdhātuādīsu vuttanayeneva veditabbaṃ.

## 対訳

1. Ākāsadhātuyaṃ ākāsagatanti ākāsabhāvaṃ gataṃ.

* ākāsadhātuyaṃ = 「空間界（ākāsadhātu）において」
* ākāsagataṃ ti = 「『空間に属するもの（ākāsagata）』とは」
* ākāsabhāvaṃ gataṃ = 「空間の性質に至ったものである」

→「空間界の項における『空間に属するもの』とは、空間の性質に至ったもののことである。」


2. Upādinnantiādinnaṃ gahitaṃ parāmaṭṭhaṃ, sarīraṭṭhakanti attho.

* upādinnaṃ ti = 「『執受されたもの（upādinna）』とは」
* ādinnaṃ = 「取られたもの」
* gahitaṃ = 「捕捉されたもの」
* parāmaṭṭhaṃ = 「執着されたもの（であり）」
* sarīraṭṭhakaṃ = 「身体に留まるもの」
* ti attho = 「という意味である」

→「『執受されたもの』とは、取られ、捕捉され、執着されたもの、すなわち身体に留まるもの、という意味である。」


3. Kaṇṇacchiddanti maṃsalohitādīhi asamphuṭṭhakaṇṇavivaraṃ.

* kaṇṇacchiddaṃ ti = 「『耳の穴（kaṇṇacchidda）』とは」
* maṃsalohitādīhi = 「肉や血などによって」
* asamphuṭṭha = 「触れられていない（満たされていない）」
* kaṇṇavivaraṃ = 「耳の開口部（空間）である」

→「『耳の穴』とは、肉や血などで満たされていない耳の開口部のことである。」


4. Nāsacchiddādīsupi eseva nayo.

* nāsacchiddādīsu pi = 「鼻の穴（nāsacchidda）などにおいても」
* eseva nayo（eso eva nayo）= 「これと全く同じ理屈である」

→「鼻の穴などにおいても、これと全く同じ理屈である。」


5. Yenacāti yena chiddena.

* yena ca ti = 「『また、それを通って（yena ca）』とは」
* yena chiddena = 「その穴を通って（という意味である）」

→「『また、それを通って』とは、その穴を通って、という意味である。」


6. Ajjhoharatīti anto paveseti, jivhābandhanato hi yāva udarapaṭalā manussānaṃ vidatthicaturaṅgulaṃ chiddaṭṭhānaṃ hoti.

* ajjhoharatī ti = 「『飲み込む（ajjhoharati）』とは」
* anto paveseti = 「内側へと入れることである」
* jivhābandhanato hi = 「というのも、舌の根（舌の繋がっている所）から」
* yāva udarapaṭalā = 「胃袋に至るまで」
* manussānaṃ = 「人間には」
* vidatthicaturaṅgulaṃ = 「一張り（vidatthi：親指から小指を広げた長さ）と四指幅（caturaṅgula）の」
* chiddaṭṭhānaṃ = 「穴の場所（空間）が」
* hoti = 「ある（からである）」

→「『飲み込む』とは、内側へと入れることである。というのも、人間には舌の根から胃袋に至るまで、一張りと四指幅の大きさの空間があるからである。」


7. Taṃ sandhāyetaṃ vuttaṃ.

* taṃ sandhāya = 「それを指して（意図して）」
* etaṃ vuttaṃ = 「これは言われた」

→「それを指して、このように言われたのである。」


8. Yattha cāti yasmiṃ okāse.

* yattha ca ti = 「『また、そこに（yattha ca）』とは」
* yasmiṃ okāse = 「その空間（場所）に」

→「『また、そこに』とは、その空間（場所）に、という意味である。」


9. Santiṭṭhatīti patiṭṭhāti.

* santiṭṭhatī ti = 「『留まる（santiṭṭhati）』とは」
* patiṭṭhāti = 「安住する（留まる）ことである」

→「『留まる』とは、安住することである。」


10. Manussānañhi mahantaṃ paṭaparissāvanamattañca udarapaṭalaṃ nāma hoti.

* manussānañ hi = 「実に、人間には」
* mahantaṃ = 「大きな」
* paṭaparissāvanamattañ ca = 「布の濾水器（水漉し）ほどの大きさの」
* udarapaṭalaṃ nāma = 「胃袋というものが」
* hoti = 「ある」

→「実に人間には、大きな布の濾水器ほどの大きさの、胃袋というものがある。」


11. Taṃ sandhāyetaṃ vuttaṃ.

* taṃ sandhāya = 「それを指して」
* etaṃ vuttaṃ = 「これは言われた」

→「それを指して、このように言われたのである。」


12. Adhobhāgaṃ nikkhamatīti yena heṭṭhā nikkhamati.

* adhobhāgaṃ nikkhamatī ti = 「『下部へと排出する』とは」
* yena = 「それを通って」
* heṭṭhā = 「下へと」
* nikkhamati = 「出ていくことである」

→「『下部へと排出する』とは、それ（空間）を通って下へと出ていくことである。」


13. Dvattiṃsahatthamattaṃ ekavīsatiyā ṭhānesu vaṅkaṃ antaṃ nāma hoti.

* dvattiṃsahatthamattaṃ = 「三十二ハッタ（腕尺、肘から指先までの長さ）ほどの大きさで」
* ekavīsatiyā ṭhānesu = 「二十一箇所で」
* vaṅkaṃ = 「曲がっている」
* antaṃ nāma = 「腸というものが」
* hoti = 「ある」

→「三十二腕尺ほどの大きさで、二十一箇所で曲がっている、腸というものがある。」


14. Taṃ sandhāyetaṃ vuttaṃ.

* taṃ sandhāya etaṃ vuttaṃ = 「それを指して、これは言われた」

→「それを指して、このように言われたのである。」


15. Yaṃ vā panaññampīti iminā sukhumasukhumaṃ cammamaṃsādiantaragatañceva lomakūpabhāvena ca ṭhitaṃ ākāsaṃ dasseti.

* yaṃ vā pan' aññaṃ pi ti = 「『あるいは、また他のいかなる（空間界）であれ』とは」
* iminā = 「これによって」
* sukhumasukhumaṃ = 「極めて微細な」
* cammamaṃsādiantaragatañ c' eva = 「皮や肉などの間に至ったもの（存在する空間）や」
* lomakūpabhāvena ca = 「毛穴の状態で」
* ṭhitaṃ ākāsaṃ = 「存在する空間を」
* dasseti = 「示している」

→「『あるいは、また他のいかなるものであれ』という言葉によって、皮や肉などの間に存在する極めて微細な空間や、毛穴の状態で存在する空間を示している。」


16. Sesametthāpi pathavīdhātuādīsu vuttanayeneva veditabbaṃ.

* sesam ettha api = 「ここにおける残りの部分も」
* pathavīdhātuādīsu = 「地界などにおいて」
* vuttanayen' eva = 「言われた理屈（方法）と全く同じように」
* veditabbaṃ = 「知られるべきである」

→「ここにおける残りの部分も、地界などの項で説かれたのと全く同じ理屈で理解されるべきである。」

# 119

119. Idānissa tādibhāvalakkhaṇaṃ ācikkhanto pathavīsamantiādimāha. Iṭṭhāniṭṭhesu hi arajjanto adussanto tādī nāma hoti. Manāpāmanāpāti ettha aṭṭha lobhasahagatacittasampayuttā manāpā nāma, dve domanassacittasampayuttā amanāpā nāma. Cittaṃ na pariyādāya ṭhassantīti ete phassā uppajjitvā tava cittaṃ antomuṭṭhigataṃ karonto viya pariyādāya gahetvā ṭhātuṃ na sakkhissanti ‘‘ahaṃ sobhāmi, mayhaṃ vaṇṇāyatanaṃ pasanna’’nti puna attabhāvaṃ nissāya chandarāgo nuppajjissati. Gūthagatantiādīsu gūthameva gūthagataṃ. Evaṃ sabbattha.

Na katthaci patiṭṭhitoti pathavīpabbatarukkhādīsu ekasmimpi na patiṭṭhito, yadi hi pathaviyaṃ patiṭṭhito bhaveyya, pathaviyā bhijjamānāya saheva bhijjeyya, pabbate patamāne saheva pateyya, rukkhe chijjamāne saheva chijjeyya.

## 対訳
1. Idānissa tādibhāvalakkhaṇaṃ ācikkhanto pathavīsamantiādimāha.

* Idānissa (idāni + assa) = 「今、彼の」
* tādibhāvalakkhaṇaṃ = 「如実なる者（不動の者）のあり方の特徴を」
* ācikkhanto = 「示しつつ（教えつつ）」
* pathavīsamantiādimāha (pathavī-samaṃ + ti + ādiṃ + āha) = 「『大地に等しく』等と語った」

→ 「今、彼の如実なる者（不動の者）のあり方の特徴を示しつつ、『大地に等しく』等と語ったのである。」

2. Iṭṭhāniṭṭhesu hi arajjanto adussanto tādī nāma hoti.

* iṭṭhāniṭṭhesu (iṭṭha + aniṭṭhesu) = 「望ましいものと望ましくないものにおいて」
* hi = 「実に」
* arajjanto = 「執着することなく」
* adussanto = 「嫌悪することなく」
* tādī nāma = 「『如実なる者』と」
* hoti = 「なる」

→ 「実に、望ましいものと望ましくないものにおいて執着することなく、嫌悪することなき者が、『如実なる者』となるのである。」

3. Manāpāmanāpāti ettha aṭṭha lobhasahagatacittasampayuttā manāpā nāma, dve domanassacittasampayuttā amanāpā nāma.

* manāpāmanāpāti = 「『好ましいものと好ましくないもの』とは」
* ettha = 「ここにおいて」
* aṭṭha = 「8つの」
* lobhasahagatacittasampayuttā = 「貪欲を伴う心と結びついたものが」
* manāpā nāma = 「『好ましいもの』という」
* dve = 「2つの」
* domanassacittasampayuttā = 「憂いを伴う心と結びついたものが」
* amanāpā nāma = 「『好ましくないもの』という」

→ 「『好ましいものと好ましくないもの』とは、ここにおいて、8つの貪欲を伴う心と結びついたものが『好ましいもの』であり、2つの憂いを伴う心と結びついたものが『好ましくないもの』である。」

4. Cittaṃ na pariyādāya ṭhassantīti ete phassā uppajjitvā tava cittaṃ antomuṭṭhigataṃ karonto viya pariyādāya gahetvā ṭhātuṃ na sakkhissanti

* cittaṃ = 「心を」
* na pariyādāya = 「圧倒することなく（支配することなく）」
* ṭhassantīti = 「留まるであろう、とは」
* ete phassā = 「これらの触（接触）が」
* uppajjitvā = 「生じて」
* tava = 「汝の」
* cittaṃ = 「心を」
* antomuṭṭhigataṃ = 「握りこぶしの中に入ったもの（掌握された状態）に」
* karonto viya = 「するかのように」
* pariyādāya = 「圧倒して」
* gahetvā = 「掴んで」
* ṭhātuṃ = 「留まることは」
* na sakkhissanti = 「できないであろう」

→ 「『心を圧倒（支配）して留まることはない』とは、これらの接触が生じて、汝の心を掌握するかのように圧倒し、掴んで留まることはできないであろう（という意味である）。」

5. ‘‘ahaṃ sobhāmi, mayhaṃ vaṇṇāyatanaṃ pasanna’’nti puna attabhāvaṃ nissāya chandarāgo nuppajjissati.

* ahaṃ = 「私は」
* sobhāmi = 「美しい（輝いている）」
* mayhaṃ = 「私の」
* vaṇṇāyatanaṃ = 「色の処（容姿）が」
* pasannanti = 「澄んでいる（鮮やかである）、と」
* puna = 「再び」
* attabhāvaṃ = 「自己の存在（身体）を」
* nissāya = 「依り所として」
* chandarāgo = 「欲貪が」
* nuppajjissati (na + uppajjissati) = 「生じないであろう」

→ 「『私は美しい、私の容姿は鮮やかである』として、再び自己の存在（身体）を依り所とした欲貪が生じることはないであろう。」

6. Gūthagatantiādīsu gūthameva gūthagataṃ. Evaṃ sabbattha.

* gūthagatantiādīsu = 「『糞になったもの』等において」
* gūthameva = 「糞そのものが」
* gūthagataṃ = 「『糞になったもの』である」
* evaṃ = 「このように」
* sabbattha = 「すべてにおいて」

→ 「『糞になったもの』等（の表現）においては、糞そのものが『糞になったもの』である。すべてにおいてこのよう（な解釈）である。」

7. Na katthaci patiṭṭhitoti pathavīpabbatarukkhādīsu ekasmimpi na patiṭṭhito,

* na katthaci = 「どこにも〜ない」
* patiṭṭhitoti = 「依存している、と（は）」
* pathavīpabbatarukkhādīsu = 「大地、山、樹木などのうち」
* ekasmimpi = 「一つにさえ」
* na patiṭṭhito = 「依存していない」

→ 「『どこにも依存していない』とは、大地、山、樹木などのうち、一つにさえ依存していない（ということである）。」

8. yadi hi pathaviyaṃ patiṭṭhito bhaveyya, pathaviyā bhijjamānāya saheva bhijjeyya,

* yadi hi = 「というのも、もし」
* pathaviyaṃ = 「大地に」
* patiṭṭhito bhaveyya = 「依存していたならば」
* pathaviyā = 「大地が」
* bhijjamānāya = 「壊れる時に」
* saheva = 「一緒に」
* bhijjeyya = 「壊れるであろう」

→ 「というのも、もし大地に依存していたならば、大地が崩壊する時に一緒に壊れるであろうから。」

9. pabbate patamāne saheva pateyya, rukkhe chijjamāne saheva chijjeyya.

* pabbate = 「山が」
* patamāne = 「崩れる時に」
* saheva = 「一緒に」
* pateyya = 「崩れる（落ちる）であろう」
* rukkhe = 「木が」
* chijjamāne = 「切られる時に」
* saheva = 「一緒に」
* chijjeyya = 「切られるであろう」

→ 「山が崩れる時には一緒に崩れるであろうし、木が切られる時には一緒に切られるであろう。」

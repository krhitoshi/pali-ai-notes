8. Ānāpānassatisuttavaṇṇanā

* ānāpānassati-sutta-vaṇṇanā = 「出入息念経の註釈」

  * ānāpāna = 「出入息」（āna「出息」+ apāna「入息」）

  * sati = 「念」

  * vaṇṇanā = 「註釈、解説」

→「8. 出入息念経の註釈」

1. 144. Evaṃ me sutan ti ānāpānassatisuttaṃ.

* evaṃ me sutaṃ = 「このように私は聞いた」

* ti = 「～とは（引用の標識）」

* ānāpānassatisuttaṃ = 「出入息念経（である）」

→「『このように私は聞いた』（で始まるのは）出入息念経である。」

2. Tattha aññehi cā ti ṭhapetvā pāḷiyaṃ āgate dasa there aññehipi abhiññātehi bahūhi sāvakehi saddhiṃ.

* tattha = 「そこ（その経）において」

* aññehi ca = 「そして他の（長老たち）と」（経文からの引用句）

* ṭhapetvā = 「～を除いて（ṭhapeti の連続体）」

* pāḷiyaṃ āgate = 「聖典（経文）に出ている」

  * pāḷiyaṃ = 「聖典において（処格）」

  * āgate = 「出てきた、挙げられた（対格複数）」

* dasa there = 「十人の長老たちを」

* aññehi pi = 「他の～とも」

* abhiññātehi = 「よく知られた、著名な」

* bahūhi sāvakehi saddhiṃ = 「多くの弟子たちとともに」

→「そこで『そして他の（長老たち）と』とは、経文に挙げられている十人の長老たちを除いて、他の著名な多くの弟子たちとともに、という意味である。」

3. Tadā kira mahā bhikkhusaṅgho ahosi aparicchinnagaṇano.

* tadā = 「そのとき」

* kira = 「～という、伝えるところでは」

* mahā bhikkhusaṅgho = 「大きな比丘サンガ」

* ahosi = 「あった（hoti のアオリスト）」

* aparicchinnagaṇano = 「数が限定されない、数えきれない」

  * a-paricchinna = 「限定されない（paricchindati の過去分詞の否定）」

  * gaṇana = 「数、計算」

→「そのとき、比丘サンガは数えきれないほど大きかったという。」

4. Ovadanti anusāsantī ti āmisasaṅgahena dhammasaṅgahena cā ti dvīhi saṅgahehi saṅgaṇhitvā kammaṭṭhānovādānusāsanīhi ovadanti ca anusāsanti ca.

* ovadanti anusāsanti = 「教誡し、教授する」（経文からの引用句）

* āmisasaṅgahena = 「財（物質）による援助によって」

  * āmisa = 「財、物質的なもの」

  * saṅgaha = 「援助、世話、まとめ取ること」

* dhammasaṅgahena ca = 「そして法による援助によって」

* ti dvīhi saṅgahehi = 「という二つの援助によって」

* saṅgaṇhitvā = 「助けて、世話をして（saṅgaṇhāti の連続体）」

* kammaṭṭhānovādānusāsanīhi = 「業処についての教誡と教授によって」

  * kammaṭṭhāna = 「業処（瞑想の対象・修習法）」

  * anusāsanī = 「教授」

* ovadanti ca anusāsanti ca = 「教誡し、また教授する」

→「『教誡し、教授する』とは、財による援助と法による援助という二つの援助によって助けたうえで、業処についての教誡と教授によって教誡し、また教授する（という意味である）。」

5. Te cā ti cakāro āgamasandhimattaṃ.

* te ca = 「そして彼らは」（経文からの引用句）

  * matta = 「～にすぎない、単なる」

→「『te ca（そして彼らは）』における ca の語は、単に文をつなぐためのものにすぎない。」

6. Uḷāraṃ pubbenāparaṃ visesaṃ jānantī ti sīlaparipūraṇādito pubbavisesato uḷārataraṃ aparaṃ kasiṇaparikammādivisesaṃ jānantī ti attho.

### 対訳

1. Āraddhoti tuṭṭho.

* āraddho = 「満足した」

* tuṭṭho = 「喜んだ、満足した（tussati の過去分詞）」

→「『āraddho（満足した）』とは、喜んでいる（という意味である）。」

2. Appattassa pattiyāti appattassa arahattassa pāpuṇanatthaṃ.

* appattassa = 「未だ得ていないものの」

* pattiyā = 「獲得のために」

* appattassa arahattassa = 「未だ得ていない阿羅漢果に」

* pāpuṇanatthaṃ = 「到達するために」（pāpuṇana-atthaṃ）

→「『未得のものの獲得のために』とは、未だ得ていない阿羅漢果に到達するために（という意味である）。」

3. Sesapadadvayepi ayameva attho.

* sesa-pada-dvaye pi = 「残りの二句においても」

* ayam eva attho = 「まさにこれと同じ意味である」

→「残りの二句においても、まさにこれと同じ意味である。」

4. Komudiṃ cātumāsininti pacchimakattikacātumāsapuṇṇamaṃ.

* komudiṃ cātumāsiniṃ = 「コームディー・チャートゥマーシニー（の満月）を」

* pacchima-kattika-cātumāsa-puṇṇamaṃ = 「（雨期）四か月の最後であるカッティカ月の満月を」

→「『コームディー・チャートゥマーシニーを』とは、四か月（の雨期）の最後のカッティカ月の満月を（いう）。」

5. Sā hi kumudānaṃ atthitāya komudī, catunnaṃ vassikānaṃ māsānaṃ pariyosānattā cātumāsinīti vuccati.

* sā hi = 「実にそれ（その満月）は」

* kumudānaṃ atthitāya = 「白睡蓮（クムダ）が存在することから」

* komudī = 「コームディー（白睡蓮の月）」

* catunnaṃ vassikānaṃ māsānaṃ = 「雨期の四か月の」

* pariyosānattā = 「終わりであることから」

* cātumāsinī ti vuccati = 「『チャートゥマーシニー（四月の）』と言われる」

→「実にそれは、白睡蓮が存在することから『コームディー』と（言われ）、雨期の四か月の終わりであることから『チャートゥマーシニー』と言われる。」

6. Āgamessāmīti udikkhissāmi, ajja apavāretvā yāva sā āgacchati, tāva katthaci agantvā idheva vasissāmīti attho.

* āgamessāmi = 「私は待とう」

* udikkhissāmi = 「私は待ち望もう、見守っていよう」

* ajja apavāretvā = 「今日は自恣を行わずに」

* yāva sā āgacchati = 「それ（その満月）が来るまで」

* tāva = 「その間」

* katthaci agantvā = 「どこへも行かずに」

* idheva vasissāmi = 「ここにだけ住もう」

* ti attho = 「という意味である」

→「『私は待とう』とは、待ち望もう（という意味であり）、今日は自恣を行わず、その（満月）が来るまで、その間どこへも行かずにここにだけ住もう、という意味である。」

7. Iti bhikkhūnaṃ pavāraṇasaṅgahaṃ anujānanto evamāha.

* iti = 「このように」

* bhikkhūnaṃ = 「比丘たちに」

* pavāraṇa-saṅgahaṃ = 「自恣の延期（の便宜）を」

  * pavāraṇā = 「自恣（雨安居明けに互いに過ちの指摘を請う行事）」

  * saṅgaha = 「まとめ、便宜。ここでは自恣を延期して安居期間を延長する優遇措置」

* anujānanto = 「許可しながら」

* evam āha = 「このように言った」

→「このように（世尊は）比丘たちに自恣の延期を許可して、このように言われたのである。」

8. Pavāraṇasaṅgaho nāma ñattidutiyena kammena diyyati

* pavāraṇasaṅgaho nāma = 「自恣の延期というものは」

* ñattidutiyena kammena = 「白二羯磨によって」

  * ñatti = 「白（提案の表明）」、dutiya = 「第二」。一度の白と一度の羯磨語による僧団の議決

* diyyati = 「与えられる」

→「自恣の延期というものは、白二羯磨によって与えられる。」

9. kassa panesa diyyati, kassa na diyyatīti.

* kassa pana esa diyyati = 「では、これは誰に与えられるのか」

* kassa na diyyati = 「誰に与えられないのか」

→「では、これは誰に与えられ、誰に与えられないのか（と問うならば）。」

10. Akārakassa tāva bālaputhujjanassa na diyyati, tathā āraddhavipassakassa ceva ariyasāvakassa ca.

* akārakassa = 「（修行を）行わない者には」

* tāva = 「まず」

* bālaputhujjanassa = 「愚かな凡夫には」

* na diyyati = 「与えられない」

* tathā = 「同様に」

* āraddhavipassakassa c'eva = 「観（ヴィパッサナー）をすでに確立した者にも」

  * āraddha = 「開始した、確立した」、vipassaka = 「観を行う者」

* ariyasāvakassa ca = 「聖なる弟子にも（与えられない）」

→「まず、（修行を）行わない愚かな凡夫には与えられない。同様に、観をすでに確立した者にも、聖なる弟子にも（与えられない）。」

11. Yassa pana samatho vā taruṇo hoti vipassanā vā, tassa diyyati.

* yassa pana = 「しかし、～である者には」

* samatho vā ... vipassanā vā = 「止であれ観であれ」

* taruṇo hoti = 「未熟（若い状態）である」

* tassa diyyati = 「その者に与えられる」

→「しかし、止であれ観であれ未熟である者には、与えられる。」

12. Bhagavāpi tadā bhikkhūnaṃ cittācāraṃ parivīmaṃsanto samathavipassanānaṃ taruṇabhāvaṃ ñatvā –

* bhagavā pi = 「世尊もまた」

* tadā = 「その時」

* bhikkhūnaṃ cittācāraṃ = 「比丘たちの心の動きを」

* parivīmaṃsanto = 「よく考察して」

* samathavipassanānaṃ taruṇabhāvaṃ = 「止と観が未熟な状態であることを」

* ñatvā = 「知って」

→「世尊もまたその時、比丘たちの心の動きをよく考察し、止と観が未熟な状態であることを知って――」

13. ‘‘mayi ajja pavārente disāsu vassaṃvuṭṭhā bhikkhū idha osarissanti.

* mayi ajja pavārente = 「私が今日自恣を行うなら（絶対処格）」

* disāsu = 「諸方で」

* vassaṃvuṭṭhā bhikkhū = 「雨安居を過ごした比丘たちが」

* idha osarissanti = 「ここに集まって来るだろう」

→「『私が今日自恣を行うなら、諸方で雨安居を過ごした比丘たちがここに集まって来るだろう。」

14. Tato ime bhikkhū vuḍḍhatarehi bhikkhūhi senāsane gahite visesaṃ nibbattetuṃ na sakkhissanti.

* tato = 「そうすると」

* ime bhikkhū = 「これらの比丘たちは」

* vuḍḍhatarehi bhikkhūhi senāsane gahite = 「より年長の比丘たちによって坐臥処が取られると（絶対処格）」

* visesaṃ nibbattetuṃ = 「特別な成果（上位の証得）を生み出すことが」

* na sakkhissanti = 「できないだろう」

→「そうすると、これらの比丘たちは、より年長の比丘たちに坐臥処を取られて、特別な成果を生み出すことができないだろう。」

15. Sacepi cārikaṃ pakkamissāmi, imesaṃ vasanaṭṭhānaṃ dullabhameva bhavissati.

* sace pi = 「たとえ～としても」

* cārikaṃ pakkamissāmi = 「私が遊行に出発する」

* imesaṃ = 「この者たちにとって」

* vasanaṭṭhānaṃ = 「住む場所は」

* dullabham eva bhavissati = 「得がたいものになるだろう」

→「たとえ私が遊行に出発したとしても、この者たちにとって住む場所は得がたいものになるだろう。」

16. Mayi pana apavārente bhikkhūpi imaṃ sāvatthiṃ na osarissanti, ahampi cārikaṃ na pakkamissāmi,

* mayi pana apavārente = 「しかし私が自恣を行わないなら（絶対処格）」

* bhikkhū pi = 「比丘たちも」

* imaṃ sāvatthiṃ na osarissanti = 「このサーヴァッティーに集まって来ないだろう」

* aham pi = 「私も」

* cārikaṃ na pakkamissāmi = 「遊行に出発しない」

→「しかし私が自恣を行わないなら、比丘たちもこのサーヴァッティーに集まって来ないだろうし、私も遊行に出発しない。」

17. evaṃ imesaṃ bhikkhūnaṃ vasanaṭṭhānaṃ apalibuddhaṃ bhavissati.

* evaṃ = 「このようにして」

* imesaṃ bhikkhūnaṃ = 「これらの比丘たちの」

* vasanaṭṭhānaṃ = 「住む場所は」

* apalibuddhaṃ bhavissati = 「妨げのない（占有されない）ものとなるだろう」

→「このようにして、これらの比丘たちの住む場所は妨げのないものとなるだろう。」

18. Te attano attano vasanaṭṭhāne phāsu viharantā samathavipassanā thāmajātā katvā visesaṃ nibbattetuṃ sakkhissantī’’ti

* te = 「彼らは」

* attano attano vasanaṭṭhāne = 「それぞれ自分の住む場所で」

* phāsu viharantā = 「快適に住みながら」

* samathavipassanā thāmajātā katvā = 「止と観を力強いものにして」

  * thāmajāta = 「力（thāma）が生じた、力強くなった」

* visesaṃ nibbattetuṃ sakkhissanti = 「特別な成果を生み出すことができるだろう」

* ti = 「と（考えて）」

→「彼らはそれぞれ自分の住む場所で快適に住みながら、止と観を力強いものにして、特別な成果を生み出すことができるだろう』と（考えて）――」

19. so taṃdivasaṃ apavāretvā kattikapuṇṇamāyaṃ pavāressāmīti bhikkhūnaṃ pavāraṇasaṅgahaṃ anujāni.

* so = 「彼（世尊）は」

* taṃdivasaṃ apavāretvā = 「その日は自恣を行わず」

* kattikapuṇṇamāyaṃ pavāressāmī ti = 「『カッティカ月の満月に自恣を行おう』と」

* bhikkhūnaṃ pavāraṇasaṅgahaṃ anujāni = 「比丘たちに自恣の延期を許可した」

→「彼（世尊）はその日は自恣を行わず、『カッティカ月の満月に自恣を行おう』と、比丘たちに自恣の延期を許可したのである。」

20. Pavāraṇasaṅgahasmiñhi laddhe yassa nissayapaṭipannassa ācariyupajjhāyā pakkamanti,

* pavāraṇasaṅgahasmiṃ hi laddhe = 「実に、自恣の延期が得られたとき（絶対処格）」

* yassa nissayapaṭipannassa = 「依止に住している者の」

  * nissaya = 「依止（新学比丘が師のもとで指導を受けて住むこと）」

## 149 (2)

Nāhaṃ, bhikkhave, muṭṭhassatissa asampajānassāti ettha ayamadhippāyo – yasmā cittapaṭisaṃvedī assasissāmītiādinā nayena pavatto bhikkhu kiñcāpi assāsapassāsanimittaṃ ārammaṇaṃ karoti, tassa pana cittassa ārammaṇe satiñca sampajaññañca upaṭṭhapetvā pavattanato citte cittānupassīyeva nāmesa hoti.


# 2. Aggivacchasuttavaṇṇanā

## 187

187.Evaṃme sutanti aggivacchasuttaṃ. Tattha na kho ahanti paṭhamavāre nāhaṃ sassatadiṭṭhikoti vadati, dutiye nāhaṃ ucchedadiṭṭhikoti. Evaṃ antānantikādivasena sabbavāresu paṭikkhepo veditabbo. Hoti ca na ca hotīti ayaṃ panettha ekaccasassatavādo. Neva hoti na na hotīti ayaṃ amarāvikkhepoti veditabbo.

### 対訳

1. 2. Aggivacchasuttavaṇṇanā

* aggivacchasutta = 「アッジヴァッチャ経」
* vaṇṇanā = 「註釈」

→ 「2. アッジヴァッチャ経註釈」

2. 187. Evaṃme sutanti aggivacchasuttaṃ.

* evaṃ me sutaṃ ti = 「『このように私は聞いた』という（言葉で始まる）」
* aggivacchasuttaṃ = 「アッジヴァッチャ経は（以下のように註釈される）」

→ 「187. 『このように私は聞いた』というアッジヴァッチャ経（の註釈である）。」

3. Tattha na kho ahanti paṭhamavāre nāhaṃ sassatadiṭṭhikoti vadati,

* tattha = 「そこ（その経）において」
* na kho ahaṃ ti = 「『私は実に（そのような見解を持た）ない』と」
* paṭhamavāre = 「最初の回（最初の問いと答え）において」
* nāhaṃ = 「私は～ではない（na + ahaṃ）」
* sassata-diṭṭhiko ti = 「常見（世界は永遠であるという見解）を持つ者である、と」
* vadati = 「（世尊は）語っておられる」

→ 「そこにおいて、『私は実に～ない』と、最初の回では『私は常見（永遠論）を持つ者ではない』と語り、」

4. dutiye nāhaṃ ucchedadiṭṭhikoti.

* dutiye = 「第二の回においては」
* nāhaṃ = 「私は～ではない」
* uccheda-diṭṭhiko ti = 「断見（世界は断滅するという見解）を持つ者である、と（語っておられる）」

→ 「第二の回においては『私は断見（断滅論）を持つ者ではない』と（語っておられる）。」

5. Evaṃ antānantikādivasena sabbavāresu paṭikkhepo veditabbo.

* evaṃ = 「このように」
* anta-anantika-ādi-vasena = 「有限や無限など（の問い）の観点によって」
* sabba-vāresu = 「すべての回（の問い）において」
* paṭikkhepo = 「拒否（否定）が」
* veditabbo = 「知られるべきである、理解されるべきである」

→ 「このように、有限や無限などの観点によって、すべての回において否定（がなされていること）が理解されるべきである。」

6. Hoti ca na ca hotīti ayaṃ panettha ekaccasassatavādo.

* hoti ca na ca hoti ti = 「『存在し、かつ存在しない』という」
* ayaṃ = 「これは」
* panettha = 「しかし、ここにおいて（pana + ettha）」
* ekacca-sassata-vādo = 「一部永遠論（一分常見）の主張である」

→ 「しかし、ここにおいて『存在し、かつ存在しない』というのは一部永遠論（一分常見）である。」

7. Neva hoti na na hotīti ayaṃ amarāvikkhepoti veditabbo.

* neva hoti na na hoti ti = 「『存在するでもなく、存在しないでもない』という」
* ayaṃ = 「これは」
* amarā-vikkhepo ti = 「鰻のうねり（のようなごまかし）論（不死矯乱論）である、と」
* veditabbo = 「知られるべきである」

→ 「『存在するでもなく、存在しないでもない』というのは、鰻のうねり論（ごまかしの論理）であると理解されるべきである。」

## 189 (1)

189.Sadukkhanti kilesadukkhena ceva vipākadukkhena ca sadukkhaṃ. Savighātanti tesaṃyeva dvinnaṃ vasena saupaghātakaṃ. Saupāyāsanti tesaṃyeva vasena saupāyāsaṃ. Sapariḷāhanti tesaṃyeva vasena sapariḷāhaṃ.

### 対訳
1. 189. Sadukkhanti kilesadukkhena ceva vipākadukkhena ca sadukkhaṃ.

* sadukkhanti = sa-dukkhaṃ + iti 「『苦を伴う』とは」
* kilesa-dukkhena = 「煩悩の苦によって」
* ca eva ... ca = ceva ... ca 「〜と、まさに〜と（によって）」
* vipāka-dukkhena = 「異熟（果報）の苦によって」
* sa-dukkhaṃ = 「苦を伴う（ということである）」

→ 189. 「『苦を伴う』とは、煩悩の苦と、異熟（果報）の苦とによって、苦を伴うということである。

2. Savighātanti tesaṃyeva dvinnaṃ vasena saupaghātakaṃ.

* savighātanti = sa-vighātaṃ + iti 「『苦悩を伴う』とは」
* tesaṃ yeva = 「まさにそれらの」
* dvinnaṃ = 「二つの（苦の）」
* vasena = 「〜によって、ゆえに」
* sa-upaghātakaṃ = 「損害（打ちひしがれること）を伴う（ということである）」

→ 「『苦悩を伴う』とは、まさにそれら二つの（苦の）ゆえに、損害（苦悩）を伴うということである。

3. Saupāyāsanti tesaṃyeva vasena saupāyāsaṃ.

* saupāyāsanti = sa-upāyāsaṃ + iti 「『絶望を伴う』とは」
* tesaṃ yeva vasena = 「まさにそれら（二つの苦）のゆえに」
* sa-upāyāsaṃ = 「絶望（煩悶）を伴う（ということである）」

→ 「『絶望を伴う』とは、まさにそれら（二つの苦）のゆえに、絶望を伴うということである。

4. Sapariḷāhanti tesaṃyeva vasena sapariḷāhaṃ.

* sapariḷāhanti = sa-pariḷāhaṃ + iti 「『焦燥を伴う』とは」
* tesaṃ yeva vasena = 「まさにそれら（二つの苦）のゆえに」
* sa-pariḷāhaṃ = 「焦燥を伴う（ということである）」

→ 「『焦燥を伴う』とは、まさにそれら（二つの苦）のゆえに、焦燥を伴うということである。」

## 189 (2)

Idāni tāni vibhajitvā dassento sabbaahaṃkāra-mamaṃkāra-mānānusayānanti āha. Ettha hi ahaṃkāro diṭṭhi, mamaṃkāro taṇhā, mānānusayo māno. Anupādā vimuttoti catūhi upādānehi kañci dhammaṃ anupādiyitvā vimutto.

### 対訳
1. Kiñci diṭṭhigatanti kāci ekā diṭṭhipi ruccitvā khamāpetvā gahitā atthīti pucchati.

* kiñci diṭṭhigatanti = kiñci diṭṭhigataṃ + iti 「『何らかの見解が』とは」
* kāci = 「いかなる」
* ekā = 「一つの」
* diṭṭhipi = diṭṭhi + pi 「見解であっても」
* ruccitvā = 「好ましく思って」
* khamāpetvā = 「受け入れて（承認して）」
* gahitā = 「執着された（把握された）ものが」
* atthīti = atthi + iti 「ありますか、と」
* pucchati = 「（彼は）尋ねている」

→ 「『何らかの見解が』とは、いかなる一つの見解であっても、好ましく思い、受け入れて、執着しているものがありますか、と尋ねているのである。

2. Apanītanti nīhaṭaṃ apaviddhaṃ.

* apanītanti = apanītaṃ + iti 「『取り除かれた』とは」
* nīhaṭaṃ = 「排除された」
* apaviddhaṃ = 「捨て去られた（ということである）」

→ 「『取り除かれた』とは、排除され、捨て去られたということである。

3. Diṭṭhanti paññāya diṭṭhaṃ.

* diṭṭhanti = diṭṭhaṃ + iti 「『見られた』とは」
* paññāya = 「智慧によって」
* diṭṭhaṃ = 「見られた（ということである）」

→ 「『見られた』とは、智慧によって見られたということである。

4. Tasmāti yasmā pañcannaṃ khandhānaṃ udayavayaṃ addasa, tasmā.

* tasmāti = tasmā + iti 「『それゆえに』とは」
* yasmā = 「〜だから」
* pañcannaṃ = 「五つの」
* khandhānaṃ = 「蘊の」
* udayavayaṃ = 「生起と消滅を」
* addasa = 「（彼が）見た」
* tasmā = 「だからである」

→ 「『それゆえに』とは、五蘊の生起と消滅を見たから、それゆえに（ということである）。

5. Sabbamaññitānanti sabbesaṃ tiṇṇampi taṇhādiṭṭhimānamaññitānaṃ.

* sabbamaññitānanti = sabba-maññitānaṃ + iti 「『すべての思量の』とは」
* sabbesaṃ = 「すべての」
* tiṇṇampi = tiṇṇaṃ + pi 「三つの」
* taṇhādiṭṭhimānamaññitānaṃ = taṇhā-diṭṭhi-māna-maññitānaṃ 「渇愛・悪見・慢という思量の（ことである）」

→ 「『すべての思量の』とは、すべての三つの、渇愛・悪見・慢という思量のことである。

6. Mathitānanti tesaṃyeva vevacanaṃ.

* mathitānanti = mathitānaṃ + iti 「『動揺の』とは」
* tesaṃyeva = tesaṃ + yeva 「まさにそれらの」
* vevacanaṃ = 「同義語である」

→ 「『動揺の』とは、まさにそれら（の思量）の同義語である。

7. Idāni tāni vibhajitvā dassento sabbaahaṃkāra-mamaṃkāra-mānānusayānanti āha.

* idāni = 「今」
* tāni = 「それらを」
* vibhajitvā = 「分析して（分けて）」
* dassento = 「示しながら」
* sabbaahaṃkāramamaṃkāramānānusayānanti = sabba-ahaṃkāra-mamaṃkāra-māna-anusayānaṃ + iti 「『すべての我執・我所執・慢の随眠の』と」
* āha = 「（世尊は）説いた」

→ 「今、それらを分析して示しながら、『すべての我執・我所執・慢の随眠の』と説いたのである。

8. Ettha hi ahaṃkāro diṭṭhi, mamaṃkāro taṇhā, mānānusayo māno.

* ettha = 「ここで」
* hi = 「実に」
* ahaṃkāro = 「我執とは」
* diṭṭhi = 「悪見（のことである）」
* mamaṃkāro = 「我所執とは」
* taṇhā = 「渇愛（のことである）」
* mānānusayo = 「慢の随眠とは」
* māno = 「慢（のことである）」

→ 「実にここでは、我執とは悪見のことであり、我所執とは渇愛のことであり、慢の随眠とは慢のことである。

9. Anupādā vimuttoti catūhi upādānehi kañci dhammaṃ anupādiyitvā vimutto.

* anupādā vimuttoti = anupādā vimutto + iti 「『執着することなく解脱している』とは」
* catūhi = 「四つの」
* upādānehi = 「取（執着）によって」
* kañci = 「いかなる」
* dhammaṃ = 「法をも」
* anupādiyitvā = 「執着することなく（an-upādiyitvā）」
* vimutto = 「解脱している（ということである）」

→ 「『執着することなく解脱している』とは、四つの取（執着）によっていかなる法をも執着することなく解脱している、ということである。」

## 190

## 192

### 対訳

1. 192. Yena rūpenāti yena rūpena sattasaṅkhātaṃ tathāgataṃ rūpīti paññāpeyya.

* 192. Yena rūpenā ti = 「『どのような色によって』とは」
* yena rūpena = 「どのような色によって」
* satta-saṅkhātaṃ = 「衆生（有情）と呼ばれる」
* tathāgataṃ = 「如来を」
* rūpī ti = 「『色をもつ者』と」
* paññāpeyya = 「施設するか（ということである）」

→ 「192. 『どのような色によって』とは、どのような色によって、衆生（有情）と呼ばれる如来を『色をもつ者』と施設するか、ということである。」

2. Gambhīroti guṇagambhīro.

* Gambhīro ti = 「『深い』とは」
* guṇa-gambhīro = 「徳（性質）において深い、ということである」

→ 「『深い』とは、徳において深い、ということである。」

3. Appameyyoti pamāṇaṃ gaṇhituṃ na sakkuṇeyyo.

* Appameyyo ti = 「『測り知れない』とは」
* pamāṇaṃ = 「量を」
* gaṇhituṃ = 「把握することは」
* na sakkuṇeyyo = 「できない、ということである」

→ 「『測り知れない』とは、量を把握することはできない、ということである。」

4. Duppariyogāḷhoti duogāho dujjāno.

* Duppariyogāḷho ti = 「『探求し難い』とは」
* du-ogāho = 「見極め難い（深く入り込み難い）」
* du-jjāno = 「知り難い、ということである」

→ 「『探求し難い』とは、見極め難く、知り難い、ということである。」

5. Seyyathāpi mahāsamuddoti yathā mahāsamuddo gambhīro appameyyo dujjāno, evameva khīṇāsavopi.

* Seyyathāpi mahāsamuddo ti = 「『あたかも大海のように』とは」
* yathā = 「～のように」
* mahāsamuddo = 「大海が」
* gambhīro = 「深く」
* appameyyo = 「測り知れず」
* dujjāno = 「知り難い（ように）」
* evaṃ eva = 「まさにそのように」
* khīṇāsavo pi = 「漏尽者（煩悩が尽きた者＝阿羅漢）も（そうである）、ということである」

→ 「『あたかも大海のように』とは、大海が深く、測り知れず、知り難いように、まさにそのように漏尽者もそうである、ということである。」

6. Taṃ ārabbha upapajjatītiādi sabbaṃ na yujjati.

* Taṃ = 「その者（漏尽者）に」
* ārabbha = 「関して」
* upapajjatī ti ādi = 「『生じる』等といったことは」
* sabbaṃ = 「すべて」
* na yujjati = 「当てはまらない（妥当しない）」

→ 「その者に関して『生じる』等といったことは、すべて当てはまらない。」

7. Kathaṃ? Yathā parinibbutaṃ aggiṃ ārabbha puratthimaṃ disaṃ gatotiādi sabbaṃ na yujjati, evaṃ.

* Kathaṃ = 「どのように（当てはまらないの）か？」
* Yathā = 「～のように」
* parinibbutaṃ = 「吹き消された（消滅した）」
* aggiṃ = 「火に」
* ārabbha = 「関して」
* puratthimaṃ = 「東の」
* disaṃ = 「方角へ」
* gato ti ādi = 「『行った』等といったことは」
* sabbaṃ = 「すべて」
* na yujjati = 「当てはまらない」
* evaṃ = 「そのように（当てはまらない）、ということである」

→ 「どのようにか？ 吹き消された火に関して『東の方角へ行った』等といったことがすべて当てはまらないように、そのように（当てはまらない）、ということである。」

8. Aniccatāti aniccatāya.

* Aniccatā ti = 「『無常性により（aniccatā）』とは」
* aniccatāya = 「（具格の意味での）無常性によって、という意味である」

→ 「『無常性により』とは、無常性によって（という意味の具格である）。」

9. Sāre patiṭṭhitanti lokuttaradhammasāre patiṭṭhitaṃ.

* Sāre patiṭṭhitan ti = 「『心材に確立している』とは」
* lokuttara-dhamma-sāre = 「出世間の法という心材に」
* patiṭṭhitaṃ = 「確立している、ということである」

→ 「『心材に確立している』とは、出世間の法という心材に確立している、ということである。」

10. Sesaṃ sabbattha uttānamevāti.

* Sesaṃ = 「残りの部分は」
* sabbattha = 「どこにおいても（すべて）」
* uttānaṃ eva = 「明瞭である」
* ti = 「ということである」

→ 「残りの部分は、すべて明瞭である。」

11. Papañcasūdaniyā majjhimanikāyaṭṭhakathāya

* Papañca-sūdaniyā = 「『パパンチャスーダニー』（という名の）」
* majjhima-nikāya-aṭṭhakathāya = 「マッジマ・ニカーヤ（中部）の註釈書の」

→ 「『パパンチャスーダニー』という、マッジマ・ニカーヤ（中部）の註釈書における」

12. Aggivacchasuttavaṇṇanā niṭṭhitā.

* Aggi-vaccha-sutta-vaṇṇanā = 「アッギ・ヴァッチャ経の解説が」
* niṭṭhitā = 「終わった」

→ 「アッギ・ヴァッチャ経の解説が終わった。」

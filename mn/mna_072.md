
# 2. Aggivacchasuttavaṇṇanā

## 187

187.Evaṃme sutanti aggivacchasuttaṃ. Tattha na kho ahanti paṭhamavāre nāhaṃ sassatadiṭṭhikoti vadati, dutiye nāhaṃ ucchedadiṭṭhikoti. Evaṃ antānantikādivasena sabbavāresu paṭikkhepo veditabbo. Hoti ca na ca hotīti ayaṃ panettha ekaccasassatavādo. Neva hoti na na hotīti ayaṃ amarāvikkhepoti veditabbo.

### 対訳

1. 2. Aggivacchasuttavaṇṇanā

* aggivacchasutta = 「アッジヴァッチャ経」
* vaṇṇanā = 「註釈」

→ 「2. アッジヴァッチャ経註釈」

2. 187. Evaṃme sutanti aggivacchasuttaṃ.

* evaṃ me sutaṃ ti = 「『このように私は聞いた』という（言葉で始まる）」
* aggivacchasuttaṃ = 「アッジヴァッチャ経は（以下のように註釈される）」

→ 「187. 『このように私は聞いた』というアッジヴァッチャ経（の註釈である）。」

3. Tattha na kho ahanti paṭhamavāre nāhaṃ sassatadiṭṭhikoti vadati,

* tattha = 「そこ（その経）において」
* na kho ahaṃ ti = 「『私は実に（そのような見解を持た）ない』と」
* paṭhamavāre = 「最初の回（最初の問いと答え）において」
* nāhaṃ = 「私は～ではない（na + ahaṃ）」
* sassata-diṭṭhiko ti = 「常見（世界は永遠であるという見解）を持つ者である、と」
* vadati = 「（世尊は）語っておられる」

→ 「そこにおいて、『私は実に～ない』と、最初の回では『私は常見（永遠論）を持つ者ではない』と語り、」

4. dutiye nāhaṃ ucchedadiṭṭhikoti.

* dutiye = 「第二の回においては」
* nāhaṃ = 「私は～ではない」
* uccheda-diṭṭhiko ti = 「断見（世界は断滅するという見解）を持つ者である、と（語っておられる）」

→ 「第二の回においては『私は断見（断滅論）を持つ者ではない』と（語っておられる）。」

5. Evaṃ antānantikādivasena sabbavāresu paṭikkhepo veditabbo.

* evaṃ = 「このように」
* anta-anantika-ādi-vasena = 「有限や無限など（の問い）の観点によって」
* sabba-vāresu = 「すべての回（の問い）において」
* paṭikkhepo = 「拒否（否定）が」
* veditabbo = 「知られるべきである、理解されるべきである」

→ 「このように、有限や無限などの観点によって、すべての回において否定（がなされていること）が理解されるべきである。」

6. Hoti ca na ca hotīti ayaṃ panettha ekaccasassatavādo.

* hoti ca na ca hoti ti = 「『存在し、かつ存在しない』という」
* ayaṃ = 「これは」
* panettha = 「しかし、ここにおいて（pana + ettha）」
* ekacca-sassata-vādo = 「一部永遠論（一分常見）の主張である」

→ 「しかし、ここにおいて『存在し、かつ存在しない』というのは一部永遠論（一分常見）である。」

7. Neva hoti na na hotīti ayaṃ amarāvikkhepoti veditabbo.

* neva hoti na na hoti ti = 「『存在するでもなく、存在しないでもない』という」
* ayaṃ = 「これは」
* amarā-vikkhepo ti = 「鰻のうねり（のようなごまかし）論（不死矯乱論）である、と」
* veditabbo = 「知られるべきである」

→ 「『存在するでもなく、存在しないでもない』というのは、鰻のうねり論（ごまかしの論理）であると理解されるべきである。」

## 189

189.Sadukkhanti kilesadukkhena ceva vipākadukkhena ca sadukkhaṃ. Savighātanti tesaṃyeva dvinnaṃ vasena saupaghātakaṃ. Saupāyāsanti tesaṃyeva vasena saupāyāsaṃ. Sapariḷāhanti tesaṃyeva vasena sapariḷāhaṃ.

### 対訳
1. 189. Sadukkhanti kilesadukkhena ceva vipākadukkhena ca sadukkhaṃ.

* sadukkhanti = sa-dukkhaṃ + iti 「『苦を伴う』とは」
* kilesa-dukkhena = 「煩悩の苦によって」
* ca eva ... ca = ceva ... ca 「〜と、まさに〜と（によって）」
* vipāka-dukkhena = 「異熟（果報）の苦によって」
* sa-dukkhaṃ = 「苦を伴う（ということである）」

→ 189. 「『苦を伴う』とは、煩悩の苦と、異熟（果報）の苦とによって、苦を伴うということである。

2. Savighātanti tesaṃyeva dvinnaṃ vasena saupaghātakaṃ.

* savighātanti = sa-vighātaṃ + iti 「『苦悩を伴う』とは」
* tesaṃ yeva = 「まさにそれらの」
* dvinnaṃ = 「二つの（苦の）」
* vasena = 「〜によって、ゆえに」
* sa-upaghātakaṃ = 「損害（打ちひしがれること）を伴う（ということである）」

→ 「『苦悩を伴う』とは、まさにそれら二つの（苦の）ゆえに、損害（苦悩）を伴うということである。

3. Saupāyāsanti tesaṃyeva vasena saupāyāsaṃ.

* saupāyāsanti = sa-upāyāsaṃ + iti 「『絶望を伴う』とは」
* tesaṃ yeva vasena = 「まさにそれら（二つの苦）のゆえに」
* sa-upāyāsaṃ = 「絶望（煩悶）を伴う（ということである）」

→ 「『絶望を伴う』とは、まさにそれら（二つの苦）のゆえに、絶望を伴うということである。

4. Sapariḷāhanti tesaṃyeva vasena sapariḷāhaṃ.

* sapariḷāhanti = sa-pariḷāhaṃ + iti 「『焦燥を伴う』とは」
* tesaṃ yeva vasena = 「まさにそれら（二つの苦）のゆえに」
* sa-pariḷāhaṃ = 「焦燥を伴う（ということである）」

→ 「『焦燥を伴う』とは、まさにそれら（二つの苦）のゆえに、焦燥を伴うということである。」

## 190

190.Na upetīti na yujjati. Ettha ca ‘‘na upapajjatī’’ti idaṃ anujānitabbaṃ siyā. Yasmā pana evaṃ vutte so paribbājako ucchedaṃ gaṇheyya, upapajjatīti pana sassatameva, upapajjati ca na ca upapajjatīti ekaccasassataṃ, neva upapajjati na na upapajjatīti amarāvikkhepaṃ, tasmā bhagavā – ‘‘ayaṃ appatiṭṭho anālambo hotu, sukhapavesanaṭṭhānaṃ mā labhatū’’ti ananuññāya ṭhatvā anuññampi paṭikkhipi. Alanti samatthaṃ pariyattaṃ . Dhammoti paccayākāradhammo. Aññatrayogenāti aññattha payogena. Aññatrācariyakenāti paccayākāraṃ ajānantānaṃ aññesaṃ ācariyānaṃ santike vasantena.

### 対訳

1. Na upetīti na yujjati.

* na upeti = 「『na upeti（当てはまらない）』」
* ti = 「というのは」
* na yujjati = 「妥当しない（結びつかない）（という意味である）」

→ 「『当てはまらない』とは、妥当しないという意味である。」

2. Ettha ca ‘‘na upapajjatī’’ti idaṃ anujānitabbaṃ siyā.

* ettha = 「ここで」
* ca = 「そして」
* na upapajjatī = 「『生まれない』」
* ti = 「という」
* idaṃ = 「このことは」
* anujānitabbaṃ = 「認められるべき」
* siyā = 「であったかもしれない」

→ 「そしてここでは、『生まれない』というこのことは認められてもよかったかもしれない。」

3. Yasmā pana evaṃ vutte so paribbājako ucchedaṃ gaṇheyya, upapajjatīti pana sassatameva, upapajjati ca na ca upapajjatīti ekaccasassataṃ, neva upapajjati na na upapajjatīti amarāvikkhepaṃ,

* yasmā = 「なぜなら～だからである」
* pana = 「しかし」
* evaṃ = 「このように」
* vutte = 「言われた場合」
* so = 「その」
* paribbājako = 「遊行者は」
* ucchedaṃ = 「断滅論を」
* gaṇheyya = 「取る（執着する）かもしれない」
* upapajjatī = 「『生まれる』」
* ti = 「と（言われた場合）」
* pana = 「は」
* sassatameva = 「まさに常見（永遠論）を（取るかもしれない）」
* upapajjati ca na ca upapajjatī = 「『生まれ、かつ生まれない』」
* ti = 「と（言われた場合は）」
* ekaccasassataṃ = 「一部常見（一分無常一分常論）を（取るかもしれない）」
* neva upapajjati na na upapajjatī = 「『生まれるのでもなく、生まれないのでもない』」
* ti = 「と（言われた場合は）」
* amarāvikkhepaṃ = 「曖昧なごまかしの論（鰻のようにつかみどころのない混乱論）を（取るかもしれない）」

→ 「しかし、もしそのように言われたならば、その遊行者は断滅論を取るかもしれず、『生まれる』と言われたならばまさに常見を、『生まれ、かつ生まれない』と言われたならば一部常見を、『生まれるのでもなく、生まれないのでもない』と言われたならば曖昧なごまかしの論を取るかもしれないからである。」

4. tasmā bhagavā – ‘‘ayaṃ appatiṭṭho anālambo hotu, sukhapavesanaṭṭhānaṃ mā labhatū’’ti ananuññāya ṭhatvā anuññampi paṭikkhipi.

* tasmā = 「それゆえに」
* bhagavā = 「世尊は」
* ayaṃ = 「この者が」
* appatiṭṭho = 「足場（拠り所）のない者と」
* anālambo = 「つかまる所（対象）のない者と」
* hotu = 「なれ」
* sukhapavesanaṭṭhānaṃ = 「安楽に入り込める場所（見解の逃げ場）を」
* mā labhatū = 「得るな」
* ti = 「と（考えて）」
* ananuññāya = 「許可しないこと（認めないこと）に」
* ṭhatvā = 「留まって（立脚して）」
* anuññampi (anuññaṃ + pi) = 「許可（認めること）さえも」
* paṭikkhipi = 「拒否したのである」

→ 「それゆえに世尊は、『この者が足場なき者、つかまる所なき者となれ、容易に入り込める見解の場所を得ることがないように』と考えて、認めない立場に立脚し、認めることさえも拒否したのである。」

5. Alanti samatthaṃ pariyattaṃ .

* alaṃ = 「『alaṃ（十分である、無理はない）』」
* ti = 「とは」
* samatthaṃ = 「適切である」
* pariyattaṃ = 「十分である（という意味である）」

→ 「『alaṃ』とは、適切である、十分であるという意味である。」

6. Dhammoti paccayākāradhammo.

* dhammo = 「『dhammo（法）』」
* ti = 「とは」
* paccayākāradhammo = 「縁起の法（条件の現れとしての法）のことである」

→ 「『法』とは、縁起の法のことである。」

7. Aññatrayogenāti aññattha payogena.

* aññatrayogena = 「『aññatrayogena（異なる実践をする）』」
* ti = 「とは」
* aññattha = 「他のところで（異なった）」
* payogena = 「実践（努力）をもつことによって（という意味である）」

→ 「『aññatrayogena』とは、異なった実践をもつことによって、という意味である。」

8. Aññatrācariyakenāti paccayākāraṃ ajānantānaṃ aññesaṃ ācariyānaṃ santike vasantena.

* aññatrācariyakena = 「『aññatrācariyakena（異なる師に従う）』」
* ti = 「とは」
* paccayākāraṃ = 「縁起を」
* ajānantānaṃ = 「知らない」
* aññesaṃ = 「他の」
* ācariyānaṃ = 「師たちの」
* santike = 「もとに」
* vasantena = 「住む（従う）ことによって（という意味である）」

→ 「『aññatrācariyakena』とは、縁起を知らない他の師たちのもとに住むことによって、という意味である。」

## 191

191.Tena hi vacchāti yasmā tvaṃ sammohamāpādinti vadasi, tasmā taṃyevettha paṭipucchissāmi. Anāhāronibbutoti appaccayo nibbuto.

### 対訳

1. 191. Tena hi vacchāti yasmā tvaṃ sammohamāpādinti vadasi, tasmā taṃyevettha paṭipucchissāmi.

* Tena hi vacchā ti = 「『それならば、ヴァッチャよ』とは」
* yasmā = 「～だから」
* tvaṃ = 「あなたが」
* sammohamāpādinti（sammohaṃ + āpādiṃ + ti） = 「『私は混乱に陥った』と」
* vadasi = 「言う」
* tasmā = 「それゆえ」
* taṃyevettha（taṃ + yeva + ettha） = 「これについて、まさにあなたに」
* paṭipucchissāmi = 「逆に尋ねよう（という意味である）」

→「191. 『それならば、ヴァッチャよ』とは、あなたが『私は混乱に陥った』と言うので、それゆえ『これについて、まさにあなたに逆に尋ねよう』という意味である。」

2. Anāhāronibbutoti appaccayo nibbuto.

* Anāhāronibbuto ti（anāhāro + nibbuto + ti） = 「『食（燃料）がなくなり消滅した』とは」
* appaccayo = 「条件（縁）がなくなったことで」
* nibbuto = 「消滅した（ということである）」

→「『食（燃料）がなくなり消滅した』とは、条件（縁）がなくなったことで消滅した、ということである。」

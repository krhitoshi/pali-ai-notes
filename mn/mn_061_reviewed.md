# 107

107. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā rājagahe viharati veḷuvane kalandakanivāpe. Tena kho pana samayena āyasmā rāhulo ambalaṭṭhikāyaṃ viharati. Atha kho bhagavā sāyanhasamayaṃ paṭisallānā vuṭṭhito yena ambalaṭṭhikā yenāyasmā rāhulo tenupasaṅkami. Addasā kho āyasmā rāhulo bhagavantaṃ dūratova āgacchantaṃ. Disvāna āsanaṃ paññāpesi, udakañca pādānaṃ. Nisīdi bhagavā paññatte āsane. Nisajja pāde pakkhālesi. Āyasmāpi kho rāhulo bhagavantaṃ abhivādetvā ekamantaṃ nisīdi.

## 対訳

1. 2. Bhikkhuvaggo

  * bhikkhu = 「比丘」
  * vaggo = 「品（章）」

→ 「2. 比丘品」

2. 3. Ambalaṭṭhikarāhulovādasuttaṃ

  * ambalaṭṭhika = 「アンバラッティカ（地名・王舎城近くの休憩所）」
  * rāhula = 「ラーフラ（人名、世尊の子）」
  * ovāda = 「教誡」
  * suttaṃ = 「経」

→ 「3. アンバラッティカにおけるラーフラへの教誡経」

3. 107. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā rājagahe viharati veḷuvane kalandakanivāpe.

  * evaṃ = 「このように」
  * me = 「私によって」
  * sutaṃ = 「聞かれた」
  * ekaṃ samayaṃ = 「ある時」
  * bhagavā = 「世尊は」
  * rājagahe = 「王舎城（ラージャガハ）に」
  * viharati = 「滞在しておられる」
  * veḷuvane = 「竹林（ヴェールヴァナ）に」  [註: veḷuvana は園の名、竹で囲まれていた — Mūlapaṇṇāsa-aṭṭh.]
  * kalandaka-nivāpe = 「カランダカ（栗鼠）の餌場に」  [註: nivāpa = 撒餌・餌場 — PTS/Main]

→ 「107. このように私は聞いた。ある時、世尊は王舎城の竹林、栗鼠の餌場に滞在しておられた。」

4. Tena kho pana samayena āyasmā rāhulo ambalaṭṭhikāyaṃ viharati.

  * tena kho pana samayena = 「さて、その時」
  * āyasmā rāhulo = 「具寿ラーフラは」
  * ambalaṭṭhikāyaṃ = 「アンバラッティカに」
  * viharati = 「滞在している」

→ 「さて、その時、具寿ラーフラはアンバラッティカに滞在していた。」

5. Atha kho bhagavā sāyanhasamayaṃ paṭisallānā vuṭṭhito yena ambalaṭṭhikā yenāyasmā rāhulo tenupasaṅkami.

  * atha kho = 「そして」
  * bhagavā = 「世尊は」
  * sāyanha-samayaṃ = 「夕方に」
  * paṭisallānā = 「独坐（禅思）から」  [註: paṭisallāna = 瞑想のための独坐・遠離 — PTS]
  * vuṭṭhito = 「立ち上がり」
  * yena ambalaṭṭhikā = 「アンバラッティカのある所、」
  * yena āyasmā rāhulo = 「具寿ラーフラのいる所へ」
  * tena upasaṅkami = 「そこへ近づかれた」

→ 「そして夕方、世尊は独坐（瞑想）から立ち上がり、アンバラッティカの具寿ラーフラのいる所へと赴かれた。」

6. Addasā kho āyasmā rāhulo bhagavantaṃ dūratova āgacchantaṃ.

  * addasā kho = 「実に見た」
  * āyasmā rāhulo = 「具寿ラーフラは」
  * bhagavantaṃ = 「世尊を」
  * dūrato eva = 「遠くから」
  * āgacchantaṃ = 「来られるのを」

→ 「具寿ラーフラは遠くから来られる世尊のお姿を見た。」

7. Disvāna āsanaṃ paññāpesi, udakañca pādānaṃ.

  * disvāna = 「見て」
  * āsanaṃ = 「座席を」
  * paññāpesi = 「設けた」  [註: paññāpeti = 設ける・敷く・用意する — PTS/DoP]
  * udakañca (udakaṃ + ca) = 「そして水を」
  * pādānaṃ = 「足のための」

→ 「見て、座席と、足を洗うための水を用意した。」

8. Nisīdi bhagavā paññatte āsane.

  * nisīdi = 「座られた」
  * bhagavā = 「世尊は」
  * paññatte = 「設けられた」  [註: paññatta は paññāpeti の過去分詞]
  * āsane = 「座席に」

→ 「世尊は用意された座席に座られた。」

9. Nisajja pāde pakkhālesi.

  * nisajja = 「座って」
  * pāde = 「両足を」
  * pakkhālesi = 「洗われた」  [註: pakkhāleti = 洗う・浄める — PTS]

→ 「座って、両足を洗われた。」

10. Āyasmāpi kho rāhulo bhagavantaṃ abhivādetvā ekamantaṃ nisīdi.

  * āyasmā pi kho rāhulo = 「具寿ラーフラもまた」
  * bhagavantaṃ = 「世尊に」
  * abhivādetvā = 「礼拝して」  [註: abhivādeti = 敬礼する・礼拝する — PTS/Main]
  * ekamantaṃ = 「一方に（傍らに）」
  * nisīdi = 「座った」

→ 「具寿ラーフラもまた世尊に礼拝し、傍らに座った。」

## 意訳

107. このように私は聞いた。ある時、世尊は王舎城の竹林、栗鼠の餌場に滞在しておられた。
さて、その時、具寿ラーフラはアンバラッティカに滞在していた。
そして夕方、世尊は独坐（瞑想）から立ち上がり、アンバラッティカの具寿ラーフラのいる所へと赴かれた。
具寿ラーフラは遠くから来られる世尊のお姿を見た。
見て、座席と足を洗うための水を用意した。
世尊は用意された座席に座られた。
座ってから、両足を洗われた。
具寿ラーフラもまた世尊に礼拝し、傍らに座った。

## 検証メモ

- 原文照合: CSCD 本文 §107 (Majjhimapaṇṇāsapāḷi) と一致。文字のドリフトなし
- 辞書確認:
  - nivāpa = 撒餌・餌場 (PTS/Main)。「栗鼠の餌場」で可
  - paṭisallāna = 瞑想のための独坐・遠離 (PTS)。「瞑想」より「独坐（禅思）」が原語に近い
  - ovāda = 教誡 (DoP: admonishing, instruction)
  - paññāpeti = 設ける・敷く・用意する (PTS/DoP)
  - pakkhāleti = 洗う・浄める (PTS)
  - abhivādeti = 敬礼する・礼拝する (PTS/Main)
- 註釈: veḷuvana は園の名で竹に囲まれていた (Mūlapaṇṇāsa-aṭṭh.)
- 修正点: 第一稿の訳に内容上の誤りなし。意訳の「瞑想から」を「独坐（瞑想）から」に調整
- 要確認: なし

# 108

108. Atha kho bhagavā parittaṃ udakāvasesaṃ udakādhāne ṭhapetvā āyasmantaṃ rāhulaṃ āmantesi – ‘‘passasi no tvaṃ, rāhula, imaṃ parittaṃ udakāvasesaṃ udakādhāne ṭhapita’’nti? …pe… Seyyathāpi, rāhula, rañño nāgo īsādanto urūḷhavā abhijāto saṅgāmāvacaro saṅgāmagato … rakkhateva soṇḍaṃ. …pe… Tasmātiha te, rāhula, ‘hassāpi na musā bhaṇissāmī’ti – evañhi te, rāhula, sikkhitabbaṃ.

(原文全体は mn_061.md #108 を参照。byte 単位で CSCD §108 と照合済み)

## 対訳

1. Atha kho bhagavā parittaṃ udakāvasesaṃ udakādhāne ṭhapetvā āyasmantaṃ rāhulaṃ āmantesi – ‘‘passasi no tvaṃ, rāhula, imaṃ parittaṃ udakāvasesaṃ udakādhāne ṭhapita’’nti? ‘‘Evaṃ, bhante’’.

* atha kho = 「そこで、さて」
* bhagavā = 「世尊は」
* parittaṃ = 「わずかな」
* udakāvasesaṃ = 「水の残りを」
* udakādhāne = 「水器に」  [註: udakādhāna は辞書見出しになし。udaka（水）+ ādhāna（容れ物・受け）からなる複合語]
* ṭhapetvā = 「残して、置いて」
* āmantesi = 「語りかけた」
* passasi no = 「見ているか」
* imaṃ ... ṭhapitaṃ ti = 「これが残されているのを」
* evaṃ, bhante = 「はい、世尊よ」

→ 「そこで世尊は、水器にわずかに水の残りを残して、尊者ラーフラに語りかけた。『ラーフラよ、あなたはこのわずかな水の残りが水器に残されているのを見ているか？』『はい、世尊よ。』」

2. ‘‘Evaṃ parittakaṃ kho, rāhula, tesaṃ sāmaññaṃ yesaṃ natthi sampajānamusāvāde lajjā’’ti.

* evaṃ = 「このように」
* parittakaṃ = 「わずかなものである」
* sāmaññaṃ = 「沙門性（出家者としての境地）」  [註: sāmañña = samaṇaship。四沙門果 (sāmaññaphala) に連なる語 — PTS/Main]
* yesaṃ natthi = 「～のない者たちの」
* sampajāna-musā-vāde = 「故意の妄語（意識的に嘘をつくこと）に対して」
* lajjā = 「恥じらい」

→ 「『ラーフラよ、故意に嘘をつくことに恥じらいのない者たちの沙門性は、実にこのようにわずかなものである。』」

3. Atha kho bhagavā parittaṃ udakāvasesaṃ chaḍḍetvā … ‘‘passasi no tvaṃ, rāhula, parittaṃ udakāvasesaṃ chaḍḍita’’nti? ‘‘Evaṃ, bhante’’.

* chaḍḍetvā = 「捨てて」
* chaḍḍitaṃ ti = 「捨てられたのを」

→ 「そこで世尊は、そのわずかな水の残りを捨てて、尊者ラーフラに語りかけた。『ラーフラよ、あなたはこのわずかな水の残りが捨てられたのを見ているか？』『はい、世尊よ。』」

4. ‘‘Evaṃ chaḍḍitaṃ kho, rāhula, tesaṃ sāmaññaṃ yesaṃ natthi sampajānamusāvāde lajjā’’ti.

* chaḍḍitaṃ = 「捨てられたものである」

→ 「『ラーフラよ、故意に嘘をつくことに恥じらいのない者たちの沙門性は、実にこのように捨てられたものである。』」

5. Atha kho bhagavā taṃ udakādhānaṃ nikkujjitvā … ‘‘passasi no tvaṃ, rāhula, imaṃ udakādhānaṃ nikkujjita’’nti? ‘‘Evaṃ, bhante’’.

* nikkujjitvā = 「伏せて（裏返して）」
* nikkujjitaṃ ti = 「伏せられたのを」

→ 「そこで世尊は、その水器を伏せて、尊者ラーフラに語りかけた。『ラーフラよ、あなたはこの水器が伏せられたのを見ているか？』『はい、世尊よ。』」

6. ‘‘Evaṃ nikkujjitaṃ kho, rāhula, tesaṃ sāmaññaṃ yesaṃ natthi sampajānamusāvāde lajjā’’ti.

* nikkujjitaṃ = 「伏せられたものである」  [要確認: 第一稿は「伏せられたもの（台無しにされたもの）」とするが, nikkujjita の原義は「伏せる・ひっくり返す」。「台無し」は文脈解釈なので対訳では原義に留め, 解釈は意訳に回す]

→ 「『ラーフラよ、故意に嘘をつくことに恥じらいのない者たちの沙門性は、実にこのように伏せられたものである。』」

7. Atha kho bhagavā taṃ udakādhānaṃ ukkujjitvā … ‘‘passasi no tvaṃ, rāhula, imaṃ udakādhānaṃ rittaṃ tuccha’’nti? ‘‘Evaṃ, bhante’’.

* ukkujjitvā = 「仰向けにして（元に戻して）」
* rittaṃ = 「空であるのを」
* tucchaṃ ti = 「空虚であるのを」

→ 「そこで世尊は、その水器を仰向けにして、尊者ラーフラに語りかけた。『ラーフラよ、あなたはこの水器が空で空虚であるのを見ているか？』『はい、世尊よ。』」

8. ‘‘Evaṃ rittaṃ tucchaṃ kho, rāhula, tesaṃ sāmaññaṃ yesaṃ natthi sampajānamusāvāde lajjā’’ti.

* rittaṃ tucchaṃ = 「空で空虚である」

→ 「『ラーフラよ、故意に嘘をつくことに恥じらいのない者たちの沙門性は、実にこのように空で空虚なものである。』」

9. Seyyathāpi, rāhula, rañño nāgo īsādanto urūḷhavā abhijāto saṅgāmāvacaro saṅgāmagato purimehipi pādehi kammaṃ karoti … naṅguṭṭhenapi kammaṃ karoti; rakkhateva soṇḍaṃ.

* seyyathāpi = 「たとえば～のように」
* rañño nāgo = 「王の象が」
* īsādanto = 「車の長柄のような（太く長い）牙を持ち」  [註: īsā = 車・犂の轅（ながえ）。īsā-danta は註でも「ratha-īsā-sadisa-danta（戦車の轅のような牙）」と説明 — PTS, M I 414]
* urūḷhavā = 「たくましく（巨大で）」  [註: urūḷhava (= ubbūḷhava, 異読 sī.pī.) = 壮大な・強大な (large, strong)。Ja は ubbāhana-samattha（担い得る力）と注 — PTS, M I 414]
* abhijāto = 「血統の優れた（良家の生まれの）」  [註: abhijāta = 良家の生まれ。PTS/DoP が本句 nāgo abhijāto を M I 414 で引く]
* saṅgāmāvacaro = 「戦場に慣れた」  [註: saṅgāma（戦） + avacara（慣れ親しんだ）]
* saṅgāmagato = 「戦場に赴いて」
* purimehi pi pādehi kammaṃ karoti = 「前足でも働きをする（戦う）」
* naṅguṭṭhena pi = 「尻尾でも」
* rakkhati eva soṇḍaṃ = 「鼻だけは守る」  [註: soṇḍā = 象の鼻 (trunk) — PTS, M I 415]

→ 「『たとえばラーフラよ、車の長柄のような牙を持つ、たくましく、血統の優れた、戦場に慣れた王の象が、戦場に赴いて、前足でも戦い…尻尾でも戦うが、鼻だけは守っているとする。』」

10. Tattha hatthārohassa evaṃ hoti – ‘ayaṃ kho rañño nāgo … rakkhateva soṇḍaṃ. Apariccattaṃ kho rañño nāgassa jīvita’nti.

* hatthārohassa = 「象使いに」  [註: hatthāroha = 象使い (elephant driver) — Concise]
* evaṃ hoti = 「このように（考えが）浮かぶ」
* apariccattaṃ = 「（まだ）捨てられていない」
* rañño nāgassa jīvitaṃ ti = 「王の象の命は」

→ 「『その場合、象使いにはこう思われる。「この王の象は…前足でも戦い…尻尾でも戦うが、鼻だけは守っている。王の象はまだ自分の命を捨てていない」と。』」

11. Yato kho, rāhula, rañño nāgo … naṅguṭṭhenapi kammaṃ karoti, soṇḍāyapi kammaṃ karoti, tattha hatthārohassa evaṃ hoti –

* yato kho = 「しかし、～の時」
* soṇḍāya pi kammaṃ karoti = 「鼻でも戦う」

→ 「『しかしラーフラよ、この王の象が、戦場に赴き…尻尾でも戦い、そして鼻でも戦う時、その場合、象使いにはこう思われる。』」

12. ‘ayaṃ kho rañño nāgo … naṅguṭṭhenapi kammaṃ karoti, soṇḍāyapi kammaṃ karoti.

* soṇḍāya pi kammaṃ karoti = 「鼻でも戦う」

→ 「『「この王の象は…尻尾でも戦い、そして鼻でも戦っている。」』」

13. Pariccattaṃ kho rañño nāgassa jīvitaṃ. Natthi dāni kiñci rañño nāgassa akaraṇīya’nti.

* pariccattaṃ = 「（完全に）捨てられた」  [註: pariccajati（捨てる）の過去分詞]
* natthi dāni kiñci = 「今やいかなる～もない」
* akaraṇīyaṃ ti = 「為し得ないことは」

→ 「『「今や王の象は自分の命を完全に捨てきった。この王の象にとって、もはや為し得ないことは何もない」と。』」

14. Evameva kho, rāhula, yassa kassaci sampajānamusāvāde natthi lajjā, nāhaṃ tassa kiñci pāpaṃ akaraṇīyanti vadāmi.

* evam eva kho = 「全くこれと同じように」
* yassa kassaci = 「誰であれその人に」
* na ahaṃ ... vadāmi = 「私は～ないと説く」
* kiñci pāpaṃ akaraṇīyaṃ ti = 「いかなる悪行であれ為し得ないものは（ない）」

→ 「『全くこれと同じように、ラーフラよ、誰であれ意識的に嘘をつくことに恥じらいのない者にとって、もはや為し得ない悪行は何もない、と私は説く。』」

15. Tasmātiha te, rāhula, ‘hassāpi na musā bhaṇissāmī’ti – evañhi te, rāhula, sikkhitabbaṃ.

* tasmā iha = 「それゆえに、ここで」
* hassā pi = 「冗談からでも」  [註: hassa = 笑い・戯れ (jest)]
* na musā bhaṇissāmi ti = 「嘘を言わないぞと」
* sikkhitabbaṃ = 「学ばれるべきである（訓練されるべきである）」

→ 「『それゆえに、ラーフラよ、「冗談からであっても決して嘘をつかない」と、実にこのようにあなたは学ぶべきである。』」

## 意訳

そこで世尊は、水器にわずかに水を残したまま、尊者ラーフラに語りかけられた。
「ラーフラよ、この水器にわずかに水が残されているのが見えるか？」
「はい、世尊よ。」
「ラーフラよ、故意に嘘をつくことに恥じらいのない者の沙門性とは、このようにわずかなものなのだ。」

次に世尊は、そのわずかな水を捨てて、尊者ラーフラに語りかけられた。
「ラーフラよ、このわずかな水が捨てられたのが見えるか？」
「はい、世尊よ。」
「ラーフラよ、故意に嘘をつくことに恥じらいのない者の沙門性とは、このように捨てられたものなのだ。」

次に世尊は、その水器を伏せて、尊者ラーフラに語りかけられた。
「ラーフラよ、この水器が伏せられたのが見えるか？」
「はい、世尊よ。」
「ラーフラよ、故意に嘘をつくことに恥じらいのない者の沙門性とは、このように伏せられた（台無しにされた）ものなのだ。」

次に世尊は、その水器を元通り上に向けて、尊者ラーフラに語りかけられた。
「ラーフラよ、この水器が空っぽで中身がないのが見えるか？」
「はい、世尊よ。」
「ラーフラよ、故意に嘘をつくことに恥じらいのない者の沙門性とは、このように空っぽで中身のないものなのだ。

ラーフラよ、たとえば車の長柄のように太い牙を持つ、たくましく血統の優れた王の象がいたとしよう。戦場に慣れたその象が戦場に赴き、前足や後ろ足、体の前部や後部、頭、耳、牙、尻尾のすべてを使って戦いながらも、最も急所である鼻だけは守っているとする。
その場合、象使いはこう思うだろう。『この象は体中を使って戦っているが、鼻だけは守っている。この象はまだ自分の命を惜しんでいるのだ』と。

しかしラーフラよ、いざその王の象が、体中のあらゆる部分だけでなく、ついにその鼻までも使って戦い始めたとする。
その場合、象使いはこう思うだろう。『この象は体中を使い、ついに鼻までも使って戦っている。この象はもはや自分の命を完全に捨てきったのだ。今やこの象にできないことは何もない』と。

ラーフラよ、全くこれと同じことなのだ。誰であれ、故意に嘘をつくことに少しの恥じらいも持たない者には、もはや犯すことのできない悪行は何一つとして残されていない、と私は説く。

それゆえに、ラーフラよ、『たとえ冗談であっても決して嘘はつかない』と、あなたはこのように自らを訓練し、学ばなければならない。」

## 意訳

109. 「ラーフラよ、どう思うか。鏡の目的は何であるか？」
「尊師よ、観察（反省）のためです。」
「ラーフラよ、まさにそのように、繰り返しよく観察して身体による行為を行うべきであり、繰り返しよく観察して言葉による行為を行うべきであり、繰り返しよく観察して心による行為を行うべきである。

ラーフラよ、そなたが身体による行為を行おうとする時には、その身体による行為についてこのように観察しなければならない。『私が行おうとしているこの身体による行為は、自らを害することになるだろうか。他者を害することになるだろうか。自他両方を害することになるだろうか。これは不善な行為であり、苦しみをもたらし、苦しみの報いを生むだろうか』と。
もし観察して、『私が行おうとしているこの行為は、自他を害する不善な行為であり、苦しみをもたらす』と知るならば、ラーフラよ、そなたはそのような行為を決して行ってはならない。
しかしもし観察して、『私が行おうとしているこの行為は、誰も害さない善なる行為であり、安楽をもたらす』と知るならば、そなたはそのような行為を行うべきである。

ラーフラよ、そなたがまさに身体による行為を行っている最中にも、その行為について観察しなければならない。『私が今しているこの行為は、自他を害する不善な行為であり、苦しみをもたらしているだろうか』と。
もし観察して、『この行為は自他を害する不善な行為である』と知るならば、そなたはその行為を直ちにやめなければならない。
しかしもし観察して、『この行為は誰も害さない善なる行為である』と知るならば、そなたはその行為をそのまま続けるべきである。

ラーフラよ、そなたが身体による行為を行った後にも、その行為について観察しなければならない。『私が行ったこの行為は、自他を害する不善な行為であり、苦しみをもたらしただろうか』と。
もし観察して、『この行為は自他を害する不善な行為だった』と知るならば、ラーフラよ、そなたはそのような行為を、師や知恵ある修行仲間に告白し、包み隠さず明らかにしなければならない。そして将来に向けた防護（律儀）としなければならない。
しかしもし観察して、『この行為は誰も害さない善なる行為だった』と知るならば、ラーフラよ、そなたはまさにその喜びと歓喜をもって、昼も夜も善き教えを学びながら過ごすべきである。」

- 修正点: なし。当初 sasakkaṃ を PTSD の字義で「できる限り」に修正したが, aṭṭhakathā
  (ekaṃseneva na kātabbaṃ) により第一稿「決して/確実に」が妥当と確認し撤回した
- 註釈確認 (Majjhimapaṇṇāsa-aṭṭh. §109):
  - sasakkaṃ = ekaṃsena (絶対に)
  - anupadajjeyyāsi = upatthambheyyāsi punappunaṃ kareyyāsi (支え繰り返しなす =「続ける」)
  - paṭisaṃhareyyāsi = nivatteyyāsi mā kareyyāsi (退け, なすな =「やめる」)

→ 「この意業は不善であり、苦をもたらすものであり、苦の果報をもたらすものである』と。（その時は）ラーフラよ、あなたはそのような意業を中止すべきである。

16. Sace pana tvaṃ, rāhula, paccavekkhamāno evaṃ jāneyyāsi –

→ 「しかしラーフラよ、もしあなたが観察して、このように知るならば、

17. ‘yaṃ kho ahaṃ idaṃ manasā kammaṃ karomi idaṃ me manokammaṃ nevattabyābādhāyapi saṃvattati, na parabyābādhāyapi saṃvattati, na ubhayabyābādhāyapi saṃvattati –

→ 「『私が行っているこの意による業、私のこの意業は、決して自らを害することにも繋がらず、他者を害することにも繋がらず、両者を害することにも繋がっていない。

18. kusalaṃ idaṃ manokammaṃ sukhudrayaṃ sukhavipāka’nti, anupadajjeyyāsi tvaṃ, rāhula, evarūpaṃ manokammaṃ.

* anupadajjeyyāsi = 「継続すべきである」  [註: aṭṭhakathā は「upatthambheyyāsi punappunaṃ kareyyāsi（支え, 繰り返しなすべし）」と注釈 — Majjhimapaṇṇāsa-aṭṭh. §109。「続ける」で確定]

→ 「この意業は善であり、楽をもたらすものであり、楽の果報をもたらすものである』と。（その時は）ラーフラよ、あなたはそのような意業を継続すべきである。

19. Katvāpi te, rāhula, manasā kammaṃ tadeva te manokammaṃ paccavekkhitabbaṃ –

* katvā pi = 「行った後にも」

→ 「ラーフラよ、意による業を行った後にも、まさにその意業を、あなたは（次のように）観察すべきである。

20. ‘yaṃ nu kho ahaṃ idaṃ manasā kammaṃ akāsiṃ idaṃ me manokammaṃ attabyābādhāyapi saṃvattati [saṃvatti (sī. pī.)], parabyābādhāyapi saṃvattati, ubhayabyābādhāyapi saṃvattati –

* akāsiṃ = 「私は行った」
* [saṃvatti] = 「（異読）繋がった」

→ 「『私が行ったこの意による業、私のこの意業は、自らを害することにも繋がり、他者を害することにも繋がり、両者を害することにも繋がっただろうか。

21. akusalaṃ idaṃ manokammaṃ dukkhudrayaṃ dukkhavipāka’nti?

→ 「この意業は不善であり、苦をもたらすものであり、苦の果報をもたらすものだっただろうか』と。

22. Sace kho tvaṃ, rāhula, paccavekkhamāno evaṃ jāneyyāsi –

* sace kho = 「実に、もし」

→ 「ラーフラよ、もしあなたが観察して、このように知るならば、

23. ‘yaṃ kho ahaṃ idaṃ manasā kammaṃ akāsiṃ idaṃ me manokammaṃ attabyābādhāyapi saṃvattati, parabyābādhāyapi saṃvattati, ubhayabyābādhāyapi saṃvattati –

→ 「『私が行ったこの意による業、私のこの意業は、自らを害することにも繋がり、他者を害することにも繋がり、両者を害することにも繋がった。

24. akusalaṃ idaṃ manokammaṃ dukkhudrayaṃ dukkhavipāka’nti, evarūpaṃ pana te, rāhula, manokammaṃ aṭṭīyitabbaṃ harāyitabbaṃ jigucchitabbaṃ; aṭṭīyitvā harāyitvā jigucchitvā āyatiṃ saṃvaraṃ āpajjitabbaṃ.

* aṭṭīyitabbaṃ = 「悩むべきである、思い煩うべきである」  [註: aṭṭīyati = 悩む・困惑する。aṭṭiyati-harāyati-jigucchati は定型 (D I 213) — PTS]
* harāyitabbaṃ = 「恥じるべきである」  [註: harāyati = 慚愧する (hiri の denom.) — PTS]
* jigucchitabbaṃ = 「厭離すべきである」  [註: jigucchati = 厭う・嫌悪する (gup の desid.) — PTS]
* aṭṭīyitvā harāyitvā jigucchitvā = 「悩み、恥じ、厭離して」
* āyatiṃ saṃvaraṃ āpajjitabbaṃ = 「将来の防護（律儀）をなすべきである」
* [異読] evarūpe / manokamme = 「（処格）そのような意業において」 (sī. pī.)

→ 「この意業は不善であり、苦をもたらすものであり、苦の果報をもたらすものであった』と。（その時は）ラーフラよ、あなたはそのような意業を悩み、恥じ、厭離すべきである。悩み、恥じ、厭離して、将来に向けた防護（律儀）をなすべきである。

25. Sace pana tvaṃ, rāhula, paccavekkhamāno evaṃ jāneyyāsi –

→ 「しかしラーフラよ、もしあなたが観察して、このように知るならば、

26. ‘yaṃ kho ahaṃ idaṃ manasā kammaṃ akāsiṃ idaṃ me manokammaṃ nevattabyābādhāyapi saṃvattati, na parabyābādhāyapi saṃvattati, na ubhayabyābādhāyapi saṃvattati –

→ 「『私が行ったこの意による業、私のこの意業は、決して自らを害することにも繋がらず、他者を害することにも繋がらず、両者を害することにも繋がらなかった。

27. kusalaṃ idaṃ manokammaṃ sukhudrayaṃ sukhavipāka’nti, teneva tvaṃ, rāhula, pītipāmojjena vihareyyāsi ahorattānusikkhī kusalesu dhammesu.

* teneva (tena eva) = 「まさにそれによって」
* pītipāmojjena = 「喜悦と歓喜によって」
* vihareyyāsi = 「留まるべきである、過ごすべきである」
* ahorattānusikkhī (aho-ratta-anusikkhī) = 「昼も夜も随学しつつ」
* kusalesu dhammesu = 「善法において」

→ 「この意業は善であり、楽をもたらすものであり、楽の果報をもたらすものであった』と。（その時は）ラーフラよ、あなたはまさにその喜悦と歓喜をもって、昼も夜も善法を随学しつつ過ごすべきである。」

## 検証メモ

- 原文照合: CSCD 本文 §111 と一致。文字のドリフトなし
- 辞書確認: aṭṭīyati = 悩む, harāyati = 慚愧, jigucchati = 厭悪 (定型 aṭṭiyati-harāyati-jigucchati, D I 213)。byābādha / udraya / paṭisaṃharati / saṃvara は #109 と共通
- 異読: 項目 24 に evarūpe / manokamme (処格) の異読あり (sī. pī.)。本文は対格 evarūpaṃ … manokammaṃ
- 修正点: なし (sasakkaṃ は第一稿どおり。#109 の註釈確認を参照)
- 註釈確認: anupadajjeyyāsi = 「続ける」(upatthambheyyāsi punappunaṃ kareyyāsi, aṭṭh. §109)

# 112

112. ‘‘Ye hi keci, rāhula, atītamaddhānaṃ samaṇā vā brāhmaṇā vā kāyakammaṃ parisodhesuṃ, vacīkammaṃ parisodhesuṃ, manokammaṃ parisodhesuṃ … Idamavoca bhagavā. Attamano āyasmā rāhulo bhagavato bhāsitaṃ abhinandīti. Ambalaṭṭhikarāhulovādasuttaṃ niṭṭhitaṃ paṭhamaṃ.

(原文全体は mn_061.md #112 を参照。byte 単位で CSCD §112 と照合済み)

## 対訳

1. ‘‘Ye hi keci, rāhula, atītamaddhānaṃ samaṇā vā brāhmaṇā vā kāyakammaṃ parisodhesuṃ, vacīkammaṃ parisodhesuṃ, manokammaṃ parisodhesuṃ,

* ye hi keci = 「誰であれ、いかなる者であれ」
* atītam-addhānaṃ = 「過去の時代において」
* samaṇā vā brāhmaṇā vā = 「沙門たちであれ、バラモンたちであれ」
* kāyakammaṃ / vacīkammaṃ / manokammaṃ = 「身業 / 語業 / 意業を」
* parisodhesuṃ = 「清らかにした」  [註: parisodheti = 清くする・浄化する (pari + śudh の caus.) — PTS]

→ 「ラーフラよ、過去の時代において身業を清らかにし、語業を清らかにし、意業を清らかにしたいかなる沙門やバラモンであれ、」

2. sabbe te evamevaṃ paccavekkhitvā paccavekkhitvā kāyakammaṃ parisodhesuṃ … manokammaṃ parisodhesuṃ.

* sabbe te = 「彼らはみな」
* evamevaṃ = 「まさにこのように」
* paccavekkhitvā paccavekkhitvā = 「繰り返し省察して」

→ 「彼らはみな、まさにこのように繰り返し省察して身業を清らかにし、語業を清らかにし、意業を清らかにしたのである。」

3. Yepi hi keci, rāhula, anāgatamaddhānaṃ samaṇā vā brāhmaṇā vā … parisodhessanti,

* anāgatam-addhānaṃ = 「未来の時代において」
* parisodhessanti = 「清らかにするであろう」

→ 「ラーフラよ、未来の時代において身業・語業・意業を清らかにするであろういかなる沙門やバラモンであれ、」

4. sabbe te evamevaṃ paccavekkhitvā paccavekkhitvā … parisodhessanti.

→ 「彼らはみな、まさにこのように繰り返し省察して身業・語業・意業を清らかにするであろう。」

5. Yepi hi keci, rāhula, etarahi samaṇā vā brāhmaṇā vā … parisodhenti,

* etarahi = 「現在において」
* parisodhenti = 「清らかにする」

→ 「ラーフラよ、現在において身業・語業・意業を清らかにするいかなる沙門やバラモンであれ、」

6. sabbe te evamevaṃ paccavekkhitvā paccavekkhitvā … parisodhenti.

→ 「彼らはみな、まさにこのように繰り返し省察して身業・語業・意業を清らかにするのである。」

7. Tasmātiha, rāhula, ‘paccavekkhitvā paccavekkhitvā kāyakammaṃ parisodhessāmi … manokammaṃ parisodhessāmī’ti – evañhi te, rāhula, sikkhitabba’’nti.

* tasmā iha = 「それゆえに」
* parisodhessāmi = 「清らかにしよう」
* sikkhitabbaṃ ti = 「学ばれるべきである、と」

→ 「それゆえにラーフラよ、『繰り返し省察して身業を清らかにしよう、語業を清らかにしよう、意業を清らかにしよう』と、ラーフラよ、あなたはまさにこのように学ぶべきである。」

8. Idamavoca bhagavā. Attamano āyasmā rāhulo bhagavato bhāsitaṃ abhinandīti.

* idam avoca bhagavā = 「世尊はこれを語られた」
* attamano = 「歓喜して、心満ち足りて」  [註: attamana = 心喜んだ・満足した (delighted) — PTS/Main]
* bhāsitaṃ = 「語られたこと（説法）を」
* abhinandī ti = 「喜んで受け入れた、と」

→ 「世尊はこれを語られた。具寿ラーフラは歓喜して、世尊の説法を喜んで受け入れた。」

9. Ambalaṭṭhikarāhulovādasuttaṃ niṭṭhitaṃ paṭhamaṃ.

* niṭṭhitaṃ = 「完結した」
* paṭhamaṃ = 「第一の（経）」

→ 「アンバラッティカ・ラーフラ教誡経（第一）が完結した。」

## 検証メモ

- 原文照合: CSCD 本文 §112 と一致。文字のドリフトなし
- 辞書確認: parisodheti = 清くする・浄化する (PTS), attamana = 心喜んだ・満足した (PTS/Main)
- 修正点: なし
- 要確認: なし

## 校閲サマリ (MN 61 全体)

- 結論: 訳語そのものの実質的な変更は nikkujjita (#108) の 1 箇所のみ。第一稿
  (Gemini) は精度が高く, 校閲の主な成果は訳の確定・出典付け・原文保証であった
- 原文照合: #107-#112 すべて CSCD 本文と一致。文字のドリフトなし
- sasakkaṃ (#109 項目 8, #110 項目 3, #111 項目 6): aṭṭhakathā が「ekaṃseneva na
  kātabbaṃ（絶対になすべきでない）」と注釈 (Majjhimapaṇṇāsa-aṭṭh. §109)。第一稿の
  「決して/確実に」が妥当。当初 PTSD の字義 (sa+sakka = できる限り) のみで「できる限り」
  に修正したが, 註釈確認のうえ撤回した
- 解釈の混入:
  - nikkujjita (#108 項目 6): 「台無し」は文脈解釈。対訳は原義「伏せられた」に留めた
- 註釈で確定した語 (Majjhimapaṇṇāsa-aṭṭh. §109):
  - anupadajjeyyāsi = upatthambheyyāsi punappunaṃ kareyyāsi (支え繰り返しなす =「続ける」)
  - paṭisaṃhareyyāsi = nivatteyyāsi mā kareyyāsi (退け, なすな =「やめる」)
- 教訓: 辞書の字義と慣用句の意味は一致しないことがある。idiom は aṭṭhakathā の語釈を
  必ず確認する。sasakkaṃ は PTSD の字義のみに依拠した当初判断が誤りだった

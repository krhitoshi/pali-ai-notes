
# 7. Cātumasuttavaṇṇanā

## 157

157.Yathāupanissayenāti yo yo upanissayo yathāupanissayo, tena yathāupanissayena sammāpayogena. Patiṭṭhahissanti sāsane patiṭṭhaṃ paṭilabhissanti. Vasanaṭṭhānānīti vassaggādivasena vasanaṭṭhānāni. Saṇṭhāpayamānāti suvibhattabhāvena ṭhapentā.

### 対訳

1. 157. Yathāupanissayenāti yo yo upanissayo yathāupanissayo, tena yathāupanissayena sammāpayogena.

* yathā-upanissayenā ti = 「『それぞれの縁（素質）に応じて』とは」
* yo yo = 「各々の」
* upanissayo = 「縁（素質）が」
* yathā-upanissayo = 「応じた縁であり」
* tena yathā-upanissayena = 「その応じた縁による」
* sammā-payogena = 「正しい実践（努力）によって（という意味である）」

→「157. 『それぞれの縁に応じて』とは、各々の縁が応じた縁であり、その応じた縁に応じた正しい実践によって、という意味である。」

2. Patiṭṭhahissanti sāsane patiṭṭhaṃ paṭilabhissanti.

* patiṭṭhahissanti = 「『確立するであろう』とは」
* sāsane = 「教え（仏法）において」
* patiṭṭhaṃ = 「確立（安住）を」
* paṭilabhissanti = 「得るであろう（という意味である）」

→「『確立するであろう』とは、教えにおいて確立を得るであろう、という意味である。」

3. Vasanaṭṭhānānīti vassaggādivasena vasanaṭṭhānāni.

* vasana-ṭṭhānānī ti = 「『住む場所を（尋ねて）』とは」
* vassa-agga-ādi-vasena = 「受戒年数（戒臘）の順位などに応じて」
* vasana-ṭṭhānāni = 「住む場所を（という意味である）」

→「『住む場所を』とは、受戒年数の順位などに応じた住む場所を、という意味である。」

4. Saṇṭhāpayamānāti suvibhattabhāvena ṭhapentā.

* saṇṭhāpayamānā ti = 「『配置しながら』とは」
* su-vibhatta-bhāvena = 「きちんと区分けされた状態で（整然と）」
* ṭhapentā = 「置きながら（という意味である）」

→「『配置しながら』とは、整然と置きながら、という意味である。」

5. Avinibbhogasaddanti vinibhuñjitvā gahetuṃ asakkuṇeyyasaddaṃ.

* avinibbhoga-saddan ti = 「『区別できない騒音』とは」
* vinibhuñjitvā = 「区別して（分離して）」
* gahetuṃ = 「捉える（聞き取る）ことが」
* asakkuṇeyya-saddaṃ = 「できない音（という意味である）」

→「『区別できない騒音』とは、聞き分けることができない音、という意味である。」

6. Vacīghosopi hi bahūhi ekaccaṃ pavattito ṭhānato ca dūrataro kevalaṃ mahānigghoso eva hutvā sotapathamāgacchati.

* vacī-ghoso pi = 「声であっても」
* hi = 「実に」
* bahūhi = 「多くの人々によって」
* ekaccaṃ (ekajjaṃ) = 「一斉に（同時に）」
* pavattito = 「発せられ」
* ṭhānato ca = 「その場所から」
* dūra-taro = 「より遠く離れると」
* kevalaṃ = 「ただ」
* mahā-nigghoso eva = 「大きな轟音としてのみ」
* hutvā = 「なって」
* sota-pathaṃ = 「耳の領域（聴覚）に」
* āgacchati = 「入ってくる」

→「実に声であっても、多くの人々によって一斉に発せられ、その場所から遠く離れると、ただ大きな轟音となって耳に達するからである。」

7. Macchavilopeti macche vilumpitvā viya gahaṇe, macchānaṃ vā nayane.

* maccha-vilope ti = 「『魚を奪い合う時のように』とは」
* macche = 「魚を」
* vilumpitvā viya = 「まるで奪い取るようにして」
* gahaṇe = 「捕まえる時に」
* macchānaṃ vā = 「あるいは魚の」
* nayane = 「持ち去る（運び去る）時に（という意味である）」

→「『魚を奪い合う時のように』とは、まるで魚を奪い取るようにして捕まえる時、あるいは魚を運び去る時のことである。」

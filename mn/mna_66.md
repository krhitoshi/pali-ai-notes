# 6. Laṭukikopamasuttavaṇṇanā

## 148

148.Evaṃme sutanti laṭukikopamasuttaṃ. Tattha yena so vanasaṇḍoti ayampi mahāudāyitthero bhagavatā saddhiṃyeva piṇḍāya pavisitvā saddhiṃ paṭikkami. Tasmā yena so bhagavatā upasaṅkamanto vanasaṇḍo tenupasaṅkamīti veditabbo. Apahattāti apahārako. Upahattāti upahārako. Paṭisallānā vuṭṭhitoti phalasamāpattito vuṭṭhito.

### 対訳

1. 148. Evaṃ me sutan ti laṭukikopamasuttaṃ.

* Evaṃ = 「このように」
* me = 「私によって」
* sutaṃ = 「聞かれた」
* ti = 「と（始まるのは）」
* laṭukikopamasuttaṃ = 「ウズラの比喩の経（である）」

→ 「148. 『このように私は聞いた』と（始まるのは）ウズラの比喩の経である。」

2. Tattha yena so vanasaṇḍoti

* Tattha = 「そこにおいて（経文の中で）」
* yena so vanasaṇḍo = 「その林がある所へ」
* ti = 「とは」

→ 「その中で『その林がある所へ』とは、」

3. ayampi mahāudāyitthero bhagavatā saddhiṃyeva piṇḍāya pavisitvā saddhiṃ paṭikkami.

* ayaṃ pi = 「この」
* mahāudāyitthero = 「マハーウダーイー（大ウダーイー）長老も」
* bhagavatā = 「世尊と」
* saddhiṃ yeva = 「まさに共に」
* piṇḍāya = 「托鉢のために」
* pavisitvā = 「入って」
* saddhiṃ = 「共に」
* paṭikkami = 「戻った」

→ 「このマハーウダーイー長老も、世尊とまさに一緒に托鉢に入り、一緒に戻ったのである。」

4. Tasmā yena so bhagavatā upasaṅkamanto vanasaṇḍo tenupasaṅkamīti veditabbo.

* Tasmā = 「それゆえに」
* yena = 「～のある所へ」
* so = 「その」
* bhagavatā = 「世尊が」
* upasaṅkamanto = 「近づかれた」
* vanasaṇḍo = 「林が（ある所へ）」
* tena = 「そこへ」
* upasaṅkami = 「近づいた」
* ti = 「と」
* veditabbo = 「知られるべきである、理解すべきである」

→ 「それゆえに、世尊が向かわれたその林のある所へ（長老も）向かった、と理解すべきである。」

5. Apahattāti apahārako.

* Apahattā = 「除去する者（apaharati の行為者名詞）」
* ti = 「とは」
* apahārako = 「取り去る者、除去する者（という意味である）」

→ 「『apahattā』とは、除去する者という意味である。」

6. Upahattāti upahārako.

* Upahattā = 「もたらす者（upaharati の行為者名詞）」
* ti = 「とは」
* upahārako = 「もたらす者、与える者（という意味である）」

→ 「『upahattā』とは、もたらす者という意味である。」

7. Paṭisallānā vuṭṭhitoti phalasamāpattito vuṭṭhito.

* Paṭisallānā = 「独座瞑想から」
* vuṭṭhito = 「立ち上がって」
* ti = 「とは」
* phala-samāpattito = 「果等至（果定）から」
* vuṭṭhito = 「出定して（という意味である）」

→ 「『独座から立ち上がり』とは、果等至（聖者の果の瞑想境地）から出定して、という意味である。」

# 4. Dīghanakhasuttaṃ

## 201 (1)

201. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā rājagahe viharati gijjhakūṭe pabbate sūkarakhatāyaṃ. Atha kho dīghanakho paribbājako yena bhagavā tenupasaṅkami; upasaṅkamitvā bhagavatā saddhiṃ sammodi. Sammodanīyaṃ kathaṃ sāraṇīyaṃ vītisāretvā ekamantaṃ aṭṭhāsi. Ekamantaṃ ṭhito kho dīghanakho paribbājako bhagavantaṃ etadavoca – ‘‘ahañhi, bho gotama, evaṃvādī evaṃdiṭṭhi – ‘sabbaṃ me nakkhamatī’’’ti. ‘‘Yāpi kho te esā, aggivessana, diṭṭhi – ‘sabbaṃ me nakkhamatī’ti, esāpi te diṭṭhi nakkhamatī’’ti? ‘‘Esā ce [esāpi (ka.)] me, bho gotama, diṭṭhi khameyya, taṃpassa tādisameva, taṃpassa tādisamevā’’ti. ‘‘Ato kho te, aggivessana, bahū hi bahutarā lokasmiṃ ye evamāhaṃsu – ‘taṃpassa tādisameva, taṃpassa tādisamevā’ti. Te tañceva diṭṭhiṃ nappajahanti aññañca diṭṭhiṃ upādiyanti. Ato kho te, aggivessana, tanū hi tanutarā lokasmiṃ ye evamāhaṃsu – ‘taṃpassa tādisameva, taṃpassa tādisamevā’ti.

→ 「その時、尊者サーリプッタは世尊の背後に立ち、世尊を扇いでいた。」

2. Atha kho āyasmato sāriputtassa etadahosi –

* Atha kho = そこで
* āyasmato sāriputtassa = 尊者サーリプッタに
* etadahosi (etad + ahosi) = このような考えが起こった

→ 「そこで、尊者サーリプッタにこのような考えが起こった。」

3. ‘‘tesaṃ tesaṃ kira no bhagavā dhammānaṃ abhiññā pahānamāha,

* tesaṃ tesaṃ ... dhammānaṃ = それら、それらの事象（諸法）の
* kira = 実に（伝聞や確信を表す）
* no = われわれに
* bhagavā = 世尊は
* abhiññā = 殊勝な智によって（了知して）
* pahānam = 断捨（捨離）を
* āha = 説かれる

→ 「『世尊は実に、それら様々な事象（諸法）を了知して、われわれにその断捨を説かれる。』

4. tesaṃ tesaṃ kira no sugato dhammānaṃ abhiññā paṭinissaggamāhā’’ti.

* tesaṃ tesaṃ ... dhammānaṃ = それら、それらの事象（諸法）の
* kira = 実に
* no = われわれに
* sugato = 善逝は
* abhiññā = 殊勝な智によって（了知して）
* paṭinissaggam = 放棄を
* āhā ti = 説かれる、と。

→ 「『善逝は実に、それら様々な事象（諸法）を了知して、われわれにその放棄を説かれる』と。」

5. Iti hidaṃ āyasmato sāriputtassa paṭisañcikkhato anupādāya āsavehi cittaṃ vimucci.

* Iti = このように
* hidaṃ (hi + idaṃ) = 実にこれを
* āyasmato sāriputtassa = 尊者サーリプッタが
* paṭisañcikkhato = 観察していると（熟考していると）
* anupādāya = 執着することなく
* āsavehi = 漏（煩悩）から
* cittaṃ = 心が
* vimucci = 解脱した

→ 「このように尊者サーリプッタが観察していると、執着を離れ、煩悩（漏）から心が解脱した。」

6. Dīghanakhassa pana paribbājakassa virajaṃ vītamalaṃ dhammacakkhuṃ udapādi –

* Dīghanakhassa pana paribbājakassa = 一方、遊行者ディーガナカには
* virajaṃ = 塵のない
* vītamalaṃ = 垢を離れた
* dhammacakkhuṃ = 法眼（真理の眼）が
* udapādi = 生じた

→ 「一方、遊行者ディーガナカには、塵のない、垢を離れた法眼が生じた。」

7. ‘‘yaṃ kiñci samudayadhammaṃ sabbaṃ taṃ nirodhadhamma’’nti.

* yaṃ kiñci = いかなるものであれ
* samudayadhammaṃ = 生起する性質のものは
* sabbaṃ taṃ = そのすべては
* nirodhadhammaṃ = 滅尽する性質のものである
* ti = と。

→ 「すなわち『いかなるものであれ、生起する性質のものは、すべて滅尽する性質のものである』と。」

8. Atha kho dīghanakho paribbājako diṭṭhadhammo pattadhammo viditadhammo pariyogāḷhadhammo

* Atha kho = そこで
* dīghanakho paribbājako = 遊行者ディーガナカは
* diṭṭhadhammo = 法を見た者となり
* pattadhammo = 法に到達した者となり
* viditadhammo = 法を理解した者となり
* pariyogāḷhadhammo = 法を極めた者となり

→ 「そこで、遊行者ディーガナカは、法を見、法に到達し、法を理解し、法を極め、」

9. tiṇṇavicikiccho vigatakathaṃkatho vesārajjappatto aparappaccayo satthusāsane bhagavantaṃ etadavoca –

* tiṇṇavicikiccho = 疑いを度り超え
* vigatakathaṃkatho = 惑いを離れ
* vesārajjappatto = 無畏（自信）に達し
* aparappaccayo = 他者に依存することなく（他を縁とせず）
* satthusāsane = 師の教えにおいて
* bhagavantaṃ = 世尊に
* etadavoca (etad + avoca) = このように言った

→ 「疑いを度り超え、惑いを離れ、無畏に達し、師の教えにおいて他に依存することのない者となり、世尊にこのように言った。」

10. ‘‘abhikkantaṃ, bho gotama, abhikkantaṃ, bho gotama!

* abhikkantaṃ = 素晴らしいことです
* bho gotama = 尊きゴータマよ

→ 「素晴らしいことです、尊きゴータマよ。素晴らしいことです、尊きゴータマよ。」

11. Seyyathāpi, bho gotama, nikkujjitaṃ vā ukkujjeyya, paṭicchannaṃ vā vivareyya, mūḷhassa vā maggaṃ ācikkheyya, andhakāre vā telapajjotaṃ dhāreyya – cakkhumanto rūpāni dakkhantīti –

* Seyyathāpi = あたかも～のように
* bho gotama = 尊きゴータマよ
* nikkujjitaṃ vā ukkujjeyya = 伏せられたものを仰向けにするかのように
* paṭicchannaṃ vā vivareyya = 覆われたものを開いて見せるかのように
* mūḷhassa vā maggaṃ ācikkheyya = 迷っている者に道を教えるかのように
* andhakāre vā telapajjotaṃ dhāreyya = 暗闇に油灯をかかげるかのように
* cakkhumanto = 眼ある者たちが
* rūpāni = 形色（色）を
* dakkhantī ti = 見るだろう、と

→ 「尊きゴータマよ、あたかも伏せられたものを仰向けにし、覆われたものを開き、道に迷った者に道を教え、あるいは暗闇の中に『眼ある者は色（形）を見るであろう』と油灯を掲げるかのように、」

12. evameva kho bhotā gotamena anekapariyāyena dhammo pakāsito.

* evameva = まさにそのように
* kho = 実に
* bhotā gotamena = 尊きゴータマによって
* anekapariyāyena = 様々な方法で（無数の方法で）
* dhammo pakāsito = 法が明らかにされた

→ 「まさにそのように、尊きゴータマによって様々な方法で法が明らかにされました。」

13. Esāhaṃ bhavantaṃ gotamaṃ saraṇaṃ gacchāmi dhammañca bhikkhusaṅghañca.

* Esāhaṃ (Eso ahaṃ) = この私は
* bhavantaṃ gotamaṃ = 尊きゴータマに
* saraṇaṃ gacchāmi = 帰依します
* dhammañca bhikkhusaṅghañca = 法と、そして比丘僧伽（僧）にも

→ 「この私は、尊きゴータマに帰依します。また、法と比丘僧伽に帰依します。」

14. Upāsakaṃ maṃ bhavaṃ gotamo dhāretu ajjatagge pāṇupetaṃ saraṇaṃ gata’’nti.

* Upāsakaṃ = 在家信者として
* maṃ = 私を
* bhavaṃ gotamo = 尊きゴータマは
* dhāretu = 記憶してください（受け入れてください）
* ajjatagge = 今日以降
* pāṇupetaṃ = 命ある限り
* saraṇaṃ gataṃ = 帰依した者として
* ti = と。

→ 「尊きゴータマは、私を、今日より命ある限り帰依した在家信者としてお受け入れください。」

15. Dīghanakhasuttaṃ niṭṭhitaṃ catutthaṃ.

* Dīghanakhasuttaṃ = ディーガナカ経（長爪経）
* niṭṭhitaṃ = 終わる
* catutthaṃ = 第四の

→ 「ディーガナカ経（第四経）が終了した。」


# 2. Aṭṭhakanāgarasuttaṃ

## 17

17. Evaṃ me sutaṃ – ekaṃ samayaṃ āyasmā ānando vesāliyaṃ viharati beluvagāmake [veḷuvagāmake (syā. kaṃ. ka.)]. Tena kho pana samayena dasamo gahapati aṭṭhakanāgaro pāṭaliputtaṃ anuppatto hoti kenacideva karaṇīyena. Atha kho dasamo gahapati aṭṭhakanāgaro yena kukkuṭārāmo yena aññataro bhikkhu tenupasaṅkami; upasaṅkamitvā taṃ bhikkhuṃ abhivādetvā ekamantaṃ nisīdi. Ekamantaṃ nisinno kho dasamo gahapati aṭṭhakanāgaro taṃ bhikkhuṃ etadavoca – ‘‘kahaṃ nu kho, bhante, āyasmā ānando etarahi viharati? 5. Ekamantaṃ nisinno kho dasamo gahapati aṭṭhakanāgaro taṃ bhikkhuṃ etadavoca – ‘‘kahaṃ nu kho, bhante, āyasmā ānando etarahi viharati? Dassanakāmā hi mayaṃ taṃ āyasmantaṃ ānanda’’nti.

* kahaṃ nu kho = 「いったいどこに」

* bhante = 「尊師よ(呼びかけ)」

* viharati = 「住んでいるのですか」

→「一方に坐って、アッタカ市の住人ダサマ居士はその比丘にこう言った――『尊師よ、尊者アーナンダはいまどこに住んでいるのですか。私たちはその尊者アーナンダにお会いしたいのです』と。」

6. ‘‘Eso, gahapati, āyasmā ānando vesāliyaṃ viharati beluvagāmake’’ti.

7. Atha kho dasamo gahapati aṭṭhakanāgaro pāṭaliputte taṃ karaṇīyaṃ tīretvā yena vesālī yena beluvagāmako yenāyasmā ānando tenupasaṅkami; upasaṅkamitvā āyasmantaṃ ānandaṃ abhivādetvā ekamantaṃ nisīdi.

* pāṭaliputte = 「パータリプッタで(処格)」

* taṃ karaṇīyaṃ tīretvā = 「その用事を済ませて」

  * tīretvā = 「終えて、済ませて(tīreti の連続体)」

* yena vesālī yena beluvagāmako yenāyasmā ānando tenupasaṅkami = 「ヴェーサーリーの、ベールヴァ村の、尊者アーナンダのいるところへ近づいた」

* upasaṅkamitvā = 「近づいて」

* āyasmantaṃ ānandaṃ abhivādetvā = 「尊者アーナンダに礼拝して」

→「そこでアッタカ市の住人ダサマ居士は、パータリプッタでその用事を済ませてから、ヴェーサーリーのベールヴァ村の、尊者アーナンダのいるところへ近づいた。近づいて尊者アーナンダに礼拝し、一方に坐った。」

1. 18. Ekamantaṃ nisinno kho dasamo gahapati aṭṭhakanāgaro āyasmantaṃ ānandaṃ etadavoca –

* ekamantaṃ = 「一方に、傍らに」

* nisinno = 「坐った（nisīdati の過去分詞）」

* kho = 「実に（強調の不変化詞）」

* dasamo gahapati aṭṭhakanāgaro = 「アッタカ市の居士ダサマは」

  * dasama = 「ダサマ（人名）」

  * gahapati = 「居士、資産家」

  * aṭṭhakanāgara = 「アッタカ市の人」

* āyasmantaṃ ānandaṃ = 「尊者アーナンダに」

* etadavoca = 「このように言った」（etad + avoca）

  * avoca = 「言った（vacati のアオリスト）」

→「一方に坐ったアッタカ市の居士ダサマは、尊者アーナンダにこのように言った。」

2. ‘‘atthi nu kho, bhante ānanda, tena bhagavatā jānatā passatā arahatā sammāsambuddhena ekadhammo akkhāto

* atthi nu kho = 「いったい～はあるでしょうか」

  * nu = 「～か（疑問の不変化詞）」

* bhante ānanda = 「尊者アーナンダよ」

* tena bhagavatā = 「かの世尊によって」

* jānatā passatā = 「知る方、見る方（である）」

  * jānat = 「知る者（jānāti の現在分詞、具格）」

  * passat = 「見る者（passati の現在分詞、具格）」

* arahatā sammāsambuddhena = 「阿羅漢であり正等覚者である（世尊によって）」

* ekadhammo = 「一つの法が」

* akkhāto = 「説かれた（akkhāti の過去分詞）」

→「尊者アーナンダよ、いったい、かの知る方・見る方・阿羅漢・正等覚者である世尊によって、一つの法が説かれているでしょうか。」

3. yattha bhikkhuno appamattassa ātāpino pahitattassa viharato avimuttañceva cittaṃ vimuccati,

* yattha = 「そこにおいて（その法において）」

* bhikkhuno ... viharato = 「比丘が～住むとき（属格独立句的に：住む比丘にとって）」

* appamattassa = 「不放逸で」

* ātāpino = 「熱心で」

* pahitattassa = 「自ら励む（者にとって）」

  * pahita-atta = 「専心した自己をもつ」

* avimuttañceva cittaṃ vimuccati = 「まだ解脱していない心が解脱し」

  * avimutta = 「解脱していない」

  * ca eva = 「そしてまさに」

  * vimuccati = 「解脱する」

→「その法において、不放逸で、熱心で、自ら励んで住む比丘には、まだ解脱していない心が解脱し、」

4. aparikkhīṇā ca āsavā parikkhayaṃ gacchanti,

* aparikkhīṇā = 「まだ滅尽していない」

  * a-parikkhīṇa = 「滅尽した（parikkhīyati の過去分詞）の否定」

* ca = 「そして」

* āsavā = 「諸々の漏（煩悩）が」

* parikkhayaṃ gacchanti = 「滅尽に至る」

  * parikkhaya = 「滅尽」

  * gacchanti = 「行く、至る」

→「まだ滅尽していない諸々の漏が滅尽に至り、」

5. ananuppattañca anuttaraṃ yogakkhemaṃ anupāpuṇātī’’ti?

* ananuppattañca = 「そしてまだ到達していない」

  * an-anuppatta = 「到達した（anupāpuṇāti の過去分詞）の否定」

* anuttaraṃ = 「無上の」

* yogakkhemaṃ = 「軛安穏（束縛からの安穏、涅槃）に」

  * yoga = 「軛、束縛」

  * khema = 「安穏」

* anupāpuṇāti = 「到達する」

* ti = 「～と（引用の終わり、疑問文）」

→「まだ到達していない無上の軛安穏に到達する（ような一つの法が説かれているでしょうか）」と。

6. ‘‘Atthi kho, gahapati, tena bhagavatā jānatā passatā arahatā sammāsambuddhena ekadhammo akkhāto

### 対訳

1. 19. ‘‘Idha, gahapati, bhikkhu vivicceva kāmehi vivicca akusalehi dhammehi savitakkaṃ savicāraṃ vivekajaṃ pītisukhaṃ paṭhamaṃ jhānaṃ upasampajja viharati.

* idha = 「ここに、この（教えの）なかで」

* gahapati = 「居士よ、家長よ」（呼格）

* bhikkhu = 「比丘は」

* vivicca eva kāmehi = 「もろもろの欲からまさに離れて」

  * vivicca = 「離れて」（viviccati の連続体）

* vivicca akusalehi dhammehi = 「もろもろの不善法から離れて」

* savitakkaṃ = 「尋をともなう」

* savicāraṃ = 「伺をともなう」

* vivekajaṃ = 「遠離から生じた」（viveka-ja）

* pītisukhaṃ = 「喜と楽のある」

* paṭhamaṃ jhānaṃ = 「第一禅を」

* upasampajja viharati = 「成就して住む」

  * upasampajja = 「到達して、成就して」（連続体）

→「居士よ、ここに比丘は、もろもろの欲から離れ、もろもろの不善法から離れて、尋をともない、伺をともない、遠離から生じた喜と楽のある第一禅を成就して住みます。」

2. So iti paṭisañcikkhati – ‘idampi paṭhamaṃ jhānaṃ abhisaṅkhataṃ abhisañcetayitaṃ.

* so = 「彼は」

* iti paṭisañcikkhati = 「このように深く考察する」

  * paṭisañcikkhati = 「深く考察する、省察する」

* idampi paṭhamaṃ jhānaṃ = 「この第一禅もまた」

* abhisaṅkhataṃ = 「形成されたものである」（abhisaṅkharoti の過去分詞）

* abhisañcetayitaṃ = 「思念によって作り出されたものである」（abhisañceteti の過去分詞）

→「彼はこのように深く考察します――『この第一禅もまた、形成されたもの、思念によって作り出されたものである。』」

3. Yaṃ kho pana kiñci abhisaṅkhataṃ abhisañcetayitaṃ tadaniccaṃ nirodhadhamma’nti pajānāti.

* yaṃ kho pana kiñci = 「そして、およそ何であれ～であるもの」

* abhisaṅkhataṃ abhisañcetayitaṃ = 「形成されたもの、思念によって作り出されたもの」

* tad aniccaṃ = 「それは無常であり」

* nirodhadhammaṃ = 「滅する性質のものである」

  * nirodha-dhamma = 「滅を法（性質）とするもの」

* ti pajānāti = 「～と了知する」

→「『およそ何であれ、形成されたもの、思念によって作り出されたものは、無常であり、滅する性質のものである』と了知します。」

4. So tattha ṭhito āsavānaṃ khayaṃ pāpuṇāti.

* so = 「彼は」

* tattha ṭhito = 「そこ（その考察・その法）に立って」

  * ṭhita = 「立った、とどまった」（tiṭṭhati の過去分詞）

* āsavānaṃ khayaṃ = 「もろもろの漏の滅尽に」

  * āsava = 「漏、煩悩の流れ」

  * khaya = 「滅尽、尽きること」

* pāpuṇāti = 「到達する」

→「彼はそこに立って、もろもろの漏の滅尽に到達します。」

5. No ce āsavānaṃ khayaṃ pāpuṇāti, teneva dhammarāgena tāya dhammanandiyā pañcannaṃ orambhāgiyānaṃ saṃyojanānaṃ parikkhayā opapātiko hoti tattha parinibbāyī anāvattidhammo tasmā lokā.

* no ce = 「もし～しないならば」

* āsavānaṃ khayaṃ pāpuṇāti = 「もろもろの漏の滅尽に到達する」

* teneva dhammarāgena = 「まさにその法への貪愛によって」

  * dhamma-rāga = 「法（＝止観・禅）への貪愛、法への愛着」

* tāya dhammanandiyā = 「その法への歓喜によって」

  * dhamma-nandī = 「法への歓喜、法を喜ぶこと」

* pañcannaṃ orambhāgiyānaṃ saṃyojanānaṃ parikkhayā = 「五つの下分結の完全な滅尽によって」

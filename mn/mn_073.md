* bhagavato pāde sirasā vandati = 「世尊の御足に頭をつけて礼拝します」
* evañca vadeti = 「そして、このように言っています」

→ 『大徳よ、比丘ヴァッチャゴッタが世尊の御足に頭をつけて礼拝し、このように言っています。』

8. pariciṇṇo me bhagavā, pariciṇṇo me sugato’’’ti.

* pariciṇṇo = 「仕えられた、奉仕された（＝私は十分に仕え終えました）」
* me = 「私によって」
* bhagavā = 「世尊は」
* pariciṇṇo = 「仕えられた」
* me = 「私によって」
* sugato = 「善逝は」
* ti = 「と」

→ 『世尊は私によって仕えられました。善逝は私によって仕えられました』と。」

9. ‘‘Evamāvuso’’ti kho te bhikkhū āyasmato vacchagottassa paccassosuṃ.

* evamāvuso ti (evaṃ āvuso ti) = 「承知しました、友よ、と」
* kho te bhikkhū = 「それらの比丘たちは」
* āyasmato vacchagottassa = 「具寿ヴァッチャゴッタに」
* paccassosuṃ = 「同意して答えた」

→ 「それらの比丘たちは『承知しました、友よ』と具寿ヴァッチャゴッタに答えた。」

10. Atha kho te bhikkhū yena bhagavā tenupasaṅkamiṃsu; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdiṃsu.

* atha kho = 「そして」
* te bhikkhū = 「それらの比丘たちは」
* yena bhagavā tena = 「世尊のおられるところへ」
* upasaṅkamiṃsu = 「近づいた」
* upasaṅkamitvā = 「近づいて」
* bhagavantaṃ = 「世尊に」
* abhivādetvā = 「礼拝して」
* ekamantaṃ nisīdiṃsu = 「傍らに座った」

→ 「そして、それらの比丘たちは世尊のおられるところへ近づいた。近づいて、世尊に礼拝し、傍らに座った。」

11. Ekamantaṃ nisinnā kho te bhikkhū bhagavantaṃ etadavocuṃ –

* ekamantaṃ nisinnā kho = 「傍らに座って」
* te bhikkhū = 「それらの比丘たちは」
* bhagavantaṃ = 「世尊に」
* etadavocuṃ (etaṃ avocuṃ) = 「このように言った」

→ 「傍らに座り、それらの比丘たちは世尊にこのように言った。」

12. ‘‘āyasmā, bhante, vacchagotto bhagavato pāde sirasā vandati, evañca vadeti –

* āyasmā vacchagotto = 「具寿ヴァッチャゴッタが」
* bhante = 「大徳よ」
* bhagavato pāde sirasā vandati = 「世尊の御足に頭をつけて礼拝します」
* evañca vadeti = 「そして、このように言っています」

→ 「『大徳よ、具寿ヴァッチャゴッタが世尊の御足に頭をつけて礼拝し、このように言っています。』

13. ‘pariciṇṇo me bhagavā, pariciṇṇo me sugato’’’ti.

* pariciṇṇo me bhagavā = 「『世尊は私によって仕えられました」
* pariciṇṇo me sugato ti = 「善逝は私によって仕えられました』と」

→ 『世尊は私によって仕えられました。善逝は私によって仕えられました』と。」

14. ‘‘Pubbeva me, bhikkhave, vacchagotto bhikkhu cetasā ceto paricca vidito –

* pubbeva (pubbe eva) = 「以前からすでに」
* me = 「私には」
* bhikkhave = 「比丘たちよ」
* vacchagotto bhikkhu = 「比丘ヴァッチャゴッタについては」
* cetasā ceto paricca = 「彼の心を私の心で見極めて」
* vidito = 「知られていた」

→ 「『比丘たちよ、比丘ヴァッチャゴッタについては、彼の心を私の心で見極め、以前からすでに私には知られていた。』

15. ‘tevijjo vacchagotto bhikkhu mahiddhiko mahānubhāvo’ti.

* tevijjo = 「三明を備えた者」
* vacchagotto bhikkhu = 「比丘ヴァッチャゴッタは」
* mahiddhiko = 「大いなる神通力を持つ者」
* mahānubhāvo ti = 「大いなる威力を持つ者である、と」

→ 『比丘ヴァッチャゴッタは三明を備え、大いなる神通力と大いなる威力を持つ者である』と。

16. Devatāpi me etamatthaṃ ārocesuṃ –

* devatā pi = 「天神たちもまた」
* me = 「私に」
* etamatthaṃ (etaṃ atthaṃ) = 「この事柄を」
* ārocesuṃ = 「知らせた」

→ 「天神たちもまた、私にこの事を知らせてきた。

17. ‘tevijjo, bhante, vacchagotto bhikkhu mahiddhiko mahānubhāvo’’’ti.

* tevijjo = 「三明を備えた者」
* bhante = 「大徳よ」
* vacchagotto bhikkhu = 「比丘ヴァッチャゴッタは」
* mahiddhiko = 「大いなる神通力を持つ者」
* mahānubhāvo ti = 「大いなる威力を持つ者である、と」

→ 『大徳よ、比丘ヴァッチャゴッタは三明を備え、大いなる神通力と大いなる威力を持つ者です』と。」

18. Idamavoca bhagavā.

* idaṃ = 「これを」
* avoca = 「語った」
* bhagavā = 「世尊は」

→ 「世尊はこれを語られた。」

19. Attamanā te bhikkhū bhagavato bhāsitaṃ abhinandunti.

* attamanā = 「歓喜して（心を満たされて）」
* te bhikkhū = 「それらの比丘たちは」
* bhagavato bhāsitaṃ = 「世尊の語られたことを」
* abhinandunti = 「喜んで受け入れた」

→ 「歓喜して、それらの比丘たちは世尊の語られたことを喜んで受け入れた。」

20. Mahāvacchasuttaṃ niṭṭhitaṃ tatiyaṃ.

* mahāvacchasuttaṃ = 「摩訶ヴァッチャ経が」
* niṭṭhitaṃ = 「終わった」
* tatiyaṃ = 「第三の（経）」

→ 「第三の経である、摩訶ヴァッチャ経が終わった。」

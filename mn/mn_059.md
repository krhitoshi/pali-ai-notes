
# 9. Bahuvedanīyasuttaṃ

## 88

88. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā sāvatthiyaṃ viharati jetavane anāthapiṇḍikassa ārāme. Atha kho pañcakaṅgo thapati yenāyasmā udāyī tenupasaṅkami; upasaṅkamitvā āyasmantaṃ udāyiṃ abhivādetvā ekamantaṃ nisīdi. Ekamantaṃ nisinno kho pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘kati nu kho, bhante udāyi, vedanā vuttā bhagavatā’’ti? ‘‘Tisso kho, thapati [gahapati (syā. kaṃ. pī.)], vedanā vuttā bhagavatā. Sukhā vedanā, dukkhā vedanā, adukkhamasukhā vedanā – imā kho, thapati, tisso vedanā vuttā bhagavatā’’ti. Evaṃ vutte, pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘na kho, bhante udāyi, tisso vedanā vuttā bhagavatā; dve vedanā vuttā bhagavatā – sukhā vedanā, dukkhā vedanā. Yāyaṃ, bhante, adukkhamasukhā vedanā santasmiṃ esā paṇīte sukhe vuttā bhagavatā’’ti. Dutiyampi kho āyasmā udāyī pañcakaṅgaṃ thapatiṃ etadavoca – ‘‘na kho, gahapati, dve vedanā vuttā bhagavatā; tisso vedanā vuttā bhagavatā. Sukhā vedanā, dukkhā vedanā, adukkhamasukhā vedanā – imā kho, thapati, tisso vedanā vuttā bhagavatā’’ti. Dutiyampi kho pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘na kho, bhante udāyi, tisso vedanā vuttā bhagavatā; dve vedanā vuttā bhagavatā – sukhā vedanā, dukkhā vedanā. Yāyaṃ, bhante , adukkhamasukhā vedanā santasmiṃ esā paṇīte sukhe vuttā bhagavatā’’ti. Tatiyampi kho āyasmā udāyī pañcakaṅgaṃ thapatiṃ etadavoca – ‘‘na kho, thapati, dve vedanā vuttā bhagavatā; tisso vedanā vuttā bhagavatā. Sukhā vedanā, dukkhā vedanā, adukkhamasukhā vedanā – imā kho, thapati, tisso vedanā vuttā bhagavatā’’ti. Tatiyampi kho pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘na kho, bhante udāyi, tisso vedanā vuttā bhagavatā, dve vedanā vuttā bhagavatā – sukhā vedanā, dukkhā vedanā. Yāyaṃ, bhante, adukkhamasukhā vedanā santasmiṃ esā paṇīte sukhe vuttā bhagavatā’’ti. Neva kho sakkhi āyasmā udāyī pañcakaṅgaṃ thapatiṃ saññāpetuṃ na panāsakkhi pañcakaṅgo thapati āyasmantaṃ udāyiṃ saññāpetuṃ.

### 対訳

1. Bahuvedanīyasuttaṃ

* bahu = 「多くの」
* vedanīya = 「感受されるべきもの、受に関する」(vedanā に関連)
* suttaṃ = 「経」

→「多受経(多くの受についての経)」

2. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā sāvatthiyaṃ viharati jetavane anāthapiṇḍikassa ārāme.

* evaṃ me sutaṃ = 「このように私は聞いた」
* ekaṃ samayaṃ = 「ある時」
* bhagavā = 「世尊は」
* sāvatthiyaṃ viharati = 「サーヴァッティーに滞在していた」
* jetavane = 「ジェータ林の」
* anāthapiṇḍikassa ārāme = 「アナータピンディカの園において」

→「このように私は聞いた。ある時、世尊はサーヴァッティーのジェータ林、アナータピンディカの園に滞在していた。」

3. Atha kho pañcakaṅgo thapati yenāyasmā udāyī tenupasaṅkami; upasaṅkamitvā āyasmantaṃ udāyiṃ abhivādetvā ekamantaṃ nisīdi.

* atha kho = 「さて」
* pañcakaṅgo thapati = 「棟梁パンチャカンガは」
  * thapati = 「棟梁、建築師」
* yena āyasmā udāyī tena upasaṅkami = 「尊者ウダーイーのいるところへ近づいた」
* upasaṅkamitvā = 「近づいて」
* āyasmantaṃ udāyiṃ abhivādetvā = 「尊者ウダーイーに敬礼して」
* ekamantaṃ nisīdi = 「一方に坐った」

→「さて、棟梁パンチャカンガは尊者ウダーイーのもとへ近づいた。近づいて尊者ウダーイーに敬礼して、一方に坐った。」

4. Ekamantaṃ nisinno kho pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘kati nu kho, bhante udāyi, vedanā vuttā bhagavatā’’ti?

* ekamantaṃ nisinno = 「一方に坐った(状態で)」
* etadavoca = 「このように言った」
* kati nu kho = 「いったいいくつの」
* bhante udāyi = 「尊者ウダーイーよ」
* vedanā vuttā bhagavatā = 「受が世尊によって説かれたのですか」
  * vuttā = 「説かれた(vadati の過去分詞・複数)」

→「一方に坐った棟梁パンチャカンガは、尊者ウダーイーにこう言った――『尊者ウダーイーよ、世尊によって、いったいいくつの受が説かれたのですか』と。」

5. ‘‘Tisso kho, thapati, vedanā vuttā bhagavatā. Sukhā vedanā, dukkhā vedanā, adukkhamasukhā vedanā – imā kho, thapati, tisso vedanā vuttā bhagavatā’’ti.

* tisso = 「三つの(女性形)」
* thapati = 「棟梁よ」(異読: gahapati「居士よ」)
* sukhā vedanā = 「楽受」
* dukkhā vedanā = 「苦受」
* adukkhamasukhā vedanā = 「不苦不楽受」
  * a-dukkha-m-asukha = 「苦でもなく楽でもない」
* imā ... tisso vedanā = 「これら三つの受が」

→「『棟梁よ、世尊によって三つの受が説かれました。楽受、苦受、不苦不楽受――棟梁よ、これら三つの受が世尊によって説かれました』と。」

6. Evaṃ vutte, pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘na kho, bhante udāyi, tisso vedanā vuttā bhagavatā; dve vedanā vuttā bhagavatā – sukhā vedanā, dukkhā vedanā.

* evaṃ vutte = 「このように言われたとき」
* na kho ... tisso vedanā vuttā = 「三つの受が説かれたのではありません」
* dve = 「二つの」

→「このように言われたとき、棟梁パンチャカンガは尊者ウダーイーにこう言った――『尊者ウダーイーよ、世尊によって三つの受が説かれたのではありません。二つの受が説かれました――楽受と苦受です。』」

7. Yāyaṃ, bhante, adukkhamasukhā vedanā santasmiṃ esā paṇīte sukhe vuttā bhagavatā’’ti.

* yā ayaṃ ... adukkhamasukhā vedanā = 「この不苦不楽受というものは」
* santasmiṃ ... paṇīte sukhe = 「寂静なすぐれた楽の中に(含めて)」
  * santa = 「寂静な」
  * paṇīta = 「すぐれた」
  * sukhe = 「楽の中に(処格)」
* esā ... vuttā bhagavatā = 「これは世尊によって説かれました」

→「『尊者よ、この不苦不楽受というものは、世尊によって寂静なすぐれた楽の中に(含めて)説かれたのです』と。」

8. Dutiyampi kho āyasmā udāyī pañcakaṅgaṃ thapatiṃ etadavoca – ‘‘na kho, gahapati, dve vedanā vuttā bhagavatā; tisso vedanā vuttā bhagavatā. Sukhā vedanā, dukkhā vedanā, adukkhamasukhā vedanā – imā kho, thapati, tisso vedanā vuttā bhagavatā’’ti.

* dutiyampi = 「二度目にも」
* gahapati = 「居士よ」(本文ではここのみ gahapati。thapati の異読か) (要確認)

→「二度目にも尊者ウダーイーは棟梁パンチャカンガにこう言った――『居士よ、世尊によって二つの受が説かれたのではありません。三つの受が説かれました。楽受、苦受、不苦不楽受――棟梁よ、これら三つの受が世尊によって説かれました』と。」

9. Dutiyampi kho pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘na kho, bhante udāyi, tisso vedanā vuttā bhagavatā; dve vedanā vuttā bhagavatā – sukhā vedanā, dukkhā vedanā. Yāyaṃ, bhante, adukkhamasukhā vedanā santasmiṃ esā paṇīte sukhe vuttā bhagavatā’’ti.

(語彙は 6・7 と同じ)

→「二度目にも棟梁パンチャカンガは尊者ウダーイーにこう言った――『尊者ウダーイーよ、世尊によって三つの受が説かれたのではありません。二つの受が説かれました――楽受と苦受です。尊者よ、この不苦不楽受というものは、世尊によって寂静なすぐれた楽の中に(含めて)説かれたのです』と。」

10. Tatiyampi kho āyasmā udāyī pañcakaṅgaṃ thapatiṃ etadavoca – ‘‘na kho, thapati, dve vedanā vuttā bhagavatā; tisso vedanā vuttā bhagavatā. Sukhā vedanā, dukkhā vedanā, adukkhamasukhā vedanā – imā kho, thapati, tisso vedanā vuttā bhagavatā’’ti.

* tatiyampi = 「三度目にも」

→「三度目にも尊者ウダーイーは棟梁パンチャカンガにこう言った――『棟梁よ、世尊によって二つの受が説かれたのではありません。三つの受が説かれました。楽受、苦受、不苦不楽受――棟梁よ、これら三つの受が世尊によって説かれました』と。」

11. Tatiyampi kho pañcakaṅgo thapati āyasmantaṃ udāyiṃ etadavoca – ‘‘na kho, bhante udāyi, tisso vedanā vuttā bhagavatā, dve vedanā vuttā bhagavatā – sukhā vedanā, dukkhā vedanā. Yāyaṃ, bhante, adukkhamasukhā vedanā santasmiṃ esā paṇīte sukhe vuttā bhagavatā’’ti.

(語彙は 6・7 と同じ)

→「三度目にも棟梁パンチャカンガは尊者ウダーイーにこう言った――『尊者ウダーイーよ、世尊によって三つの受が説かれたのではありません。二つの受が説かれました――楽受と苦受です。尊者よ、この不苦不楽受というものは、世尊によって寂静なすぐれた楽の中に(含めて)説かれたのです』と。」

12. Neva kho sakkhi āyasmā udāyī pañcakaṅgaṃ thapatiṃ saññāpetuṃ na panāsakkhi pañcakaṅgo thapati āyasmantaṃ udāyiṃ saññāpetuṃ.

* neva ... sakkhi = 「できなかった」
  * sakkhi = 「できた(sakkoti のアオリスト)」
* saññāpetuṃ = 「納得させること(sañjānāti の使役の不定詞)」
* na pana asakkhi = 「また~もできなかった」

→「尊者ウダーイーは棟梁パンチャカンガを納得させることができず、棟梁パンチャカンガもまた尊者ウダーイーを納得させることができなかった。」

### 対訳

1. Assosi kho āyasmā ānando āyasmato udāyissa pañcakaṅgena thapatinā saddhiṃ imaṃ kathāsallāpaṃ.

* assosi = 「聞いた（suṇāti の過去形・アオリスト）」

* kho = 「実に（強調の不変化詞）」

* āyasmā ānando = 「尊者アーナンダは」

* āyasmato udāyissa = 「尊者ウダーインの（属格）」

* pañcakaṅgena thapatinā saddhiṃ = 「大工パンチャカンガとの（具格 + saddhiṃ = ～と共に）」

  * thapati = 「大工、棟梁」

* imaṃ kathāsallāpaṃ = 「この対話・話し合いを」

  * kathā = 「話」 + sallāpa = 「会話」

→ 「尊者アーナンダは、尊者ウダーインと大工パンチャカンガとのこの対話を聞いた。」

2. Atha kho āyasmā ānando yena bhagavā tenupasaṅkami;

* atha kho = 「そこで」

* yena bhagavā tena upasaṅkami = 「世尊のいる所へ近づいた（定型句：yena ～ tena + upasaṅkamati）」

→ 「そこで尊者アーナンダは、世尊のいる所へ近づいた。」

3. upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdi.

* upasaṅkamitvā = 「近づいて（連続体）」

* bhagavantaṃ abhivādetvā = 「世尊に敬礼して」

* ekamantaṃ nisīdi = 「一方に坐った」

→ 「近づいて世尊に敬礼して、一方に坐った。」

4. Ekamantaṃ nisinno kho āyasmā ānando yāvatako ahosi āyasmato udāyissa pañcakaṅgena thapatinā saddhiṃ kathāsallāpo taṃ sabbaṃ bhagavato ārocesi.

* ekamantaṃ nisinno = 「一方に坐った（者として）」

* yāvatako ahosi = 「～があった限りの、その全部の」

  * yāvataka = 「～の限りの（関係詞的形容詞）」

* kathāsallāpo = 「対話が（主格）」

* taṃ sabbaṃ = 「そのすべてを」

* bhagavato ārocesi = 「世尊に告げた」

  * āroceti = 「告げる、報告する」

→ 「一方に坐った尊者アーナンダは、尊者ウダーインと大工パンチャカンガとの間にあった対話のすべてを、世尊に告げた。」

5. Evaṃ vutte, bhagavā āyasmantaṃ ānandaṃ etadavoca –

* evaṃ vutte = 「このように言われたとき（絶対処格）」

* etad avoca = 「このことを言った」

→ 「このように言われたとき、世尊は尊者アーナンダにこう言った。」

6. ‘‘santaññeva kho, ānanda, pariyāyaṃ pañcakaṅgo thapati udāyissa nābbhanumodi,

* santaṃ eva = 「実際に存在するものであるのに（sant「存在する」の対格 + eva「まさに」）」

* pariyāyaṃ = 「方法、説き方、道理を（対格）」

  * pariyāya = 「教説の様式、言い方、観点」

* pañcakaṅgo thapati = 「大工パンチャカンガは」

* udāyissa = 「ウダーインの（言った説き方を）」

* na abbhanumodi = 「承認しなかった、同意しなかった」

  * abbhanumodati = abhi + anu + modati 「随喜する、承認する」

→ 「アーナンダよ、実際に存在する説き方であるのに、大工パンチャカンガはウダーインの（その説き方を）承認しなかった。」

7. santaññeva ca pana pariyāyaṃ udāyī pañcakaṅgassa thapatissa nābbhanumodi.

* ca pana = 「また一方で」

* udāyī = 「ウダーインは」

* pañcakaṅgassa thapatissa = 「大工パンチャカンガの（言った説き方を）」

→ 「また一方で、実際に存在する説き方であるのに、ウダーインも大工パンチャカンガの（その説き方を）承認しなかった。」

8. Dvepānanda, vedanā vuttā mayā pariyāyena,

* dve pi = 「二つもまた」

* ānanda = 「アーナンダよ（呼格）」

* vedanā = 「受（感受）が」

* vuttā mayā = 「私によって説かれた（vutta = vadati の過去分詞）」

* pariyāyena = 「一つの説き方によって（具格）」

→ 「アーナンダよ、私は一つの説き方によって二受をも説いた。」

9. tissopi vedanā vuttā mayā pariyāyena, pañcapi vedanā vuttā mayā pariyāyena, chapi vedanā vuttā mayā pariyāyena,

* tisso pi = 「三つもまた（tisso = ti の女性形）」

* pañca pi = 「五つもまた」

* cha pi = 「六つもまた」

→ 「一つの説き方によって三受をも説き、一つの説き方によって五受をも説き、一つの説き方によって六受をも説いた。」

10. aṭṭhārasapi vedanā vuttā mayā pariyāyena, chattiṃsapi vedanā vuttā mayā pariyāyena, aṭṭhasatampi vedanā vuttā mayā pariyāyena.

* aṭṭhārasa pi = 「十八もまた」

* chattiṃsa pi = 「三十六もまた」

* aṭṭhasataṃ pi = 「百八もまた（aṭṭha「八」+ sata「百」= 百八）」

→ 「一つの説き方によって十八受をも説き、一つの説き方によって三十六受をも説き、一つの説き方によって百八受をも説いた。」

11. Evaṃ pariyāyadesito kho, ānanda, mayā dhammo.

* evaṃ = 「このように」

* pariyāya-desito = 「説き方（観点）によって説かれた」

  * desita = 「説かれた（deseti の過去分詞）」

* dhammo = 「法は」

→ 「アーナンダよ、このように私によって法は（さまざまな）説き方によって説かれている。」

12. Evaṃ pariyāyadesite kho, ānanda, mayā dhamme ye aññamaññassa subhāsitaṃ sulapitaṃ na samanujānissanti na samanumaññissanti na samanumodissanti tesametaṃ pāṭikaṅkhaṃ –

* evaṃ pariyāyadesite mayā dhamme = 「このように私によって説き方によって説かれた法において（絶対処格）」

* ye = 「～する者たちは（関係代名詞）」

* aññamaññassa = 「お互いの」

* subhāsitaṃ = 「よく語られたことを（su + bhāsita）」

* sulapitaṃ = 「よく話されたことを（su + lapita）」

* na samanujānissanti = 「認めないであろう」

  * samanujānāti = saṃ + anu + jānāti 「承認する、認可する」

* na samanumaññissanti = 「同意しないであろう」

  * samanumaññati = saṃ + anu + maññati 「賛同する」

* na samanumodissanti = 「随喜しないであろう」

  * samanumodati = saṃ + anu + modati 「共に喜ぶ、随喜する」

* tesaṃ etaṃ pāṭikaṅkhaṃ = 「彼らにはこのことが予期される」

  * pāṭikaṅkha = 「予期されるべきこと、当然予想されること」

→ 「アーナンダよ、このように私によって法が説き方によって説かれているとき、お互いのよく語られたこと・よく話されたことを認めず、同意せず、随喜しない者たち、彼らにはこのことが予期される。すなわち――」

13. bhaṇḍanajātā kalahajātā vivādāpannā aññamaññaṃ mukhasattīhi vitudantā viharissanti.

* bhaṇḍana-jātā = 「口論を生じた者たちとなり」

11. Idhānanda, bhikkhu sabbaso rūpasaññānaṃ samatikkamā, paṭighasaññānaṃ atthaṅgamā, nānattasaññānaṃ amanasikārā ‘ananto ākāso’ti ākāsānañcāyatanaṃ upasampajja viharati.

* sabbaso = 「完全に、あらゆる点で」
* rūpasaññānaṃ = 「色想の（rūpa = 色・物質、saññā = 想）」
* samatikkamā = 「超越により（samatikkama = 超えること）」
* paṭighasaññānaṃ = 「有対想の（paṭigha = 対礙・感官の抵触）」
* atthaṅgamā = 「消滅により（atthaṅgama = 滅没）」
* nānattasaññānaṃ = 「種々想の（nānatta = 多様性）」
* amanasikārā = 「作意しないことにより（a + manasikāra = 注意を向けること）」
* ananto ākāso ti = 「『虚空は無辺である』と」
* ākāsānañcāyatanaṃ = 「空無辺処に（ākāsa = 虚空、ānañca = 無辺、āyatana = 処）」
* upasampajja viharati = 「到達して住する」

→ 「アーナンダよ、ここに比丘は、完全に色想を超越し、有対想を消滅させ、種々想を作意しないことにより、『虚空は無辺である』と空無辺処に到達して住する。」

12. Idaṃ kho, ānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca.

→ 「アーナンダよ、これが、この楽よりもさらにすぐれ、さらに精妙な他の楽である。」

→ 「アーナンダよ、もしこのように言う者がいたら……（中略）……」

→ 「アーナンダよ、もしこのように言う者がいたら……（中略）……」

* natthi kiñci ti = 「『何もない』と（na atthi = ない、kiñci = 何か）」

* nevasaññānāsaññāyatanaṃ = 「非想非非想処に（na eva saññā = 想があるのでもなく、na asaññā = 想がないのでもない）」

→ 「アーナンダよ、ここに比丘は、完全に無所有処を超越して、非想非非想処に到達して住する。」

24. Idaṃ kho, ānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca.

→ 「アーナンダよ、これが、この楽よりもさらにすぐれ、さらに精妙な他の楽である。」

## 90 (3)

‘‘Yo kho, ānanda, evaṃ vadeyya – ‘etaparamaṃ sattā sukhaṃ somanassaṃ paṭisaṃvedentī’ti, idamassa nānujānāmi. Taṃ kissa hetu? Atthānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca. Katamañcānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca? Idhānanda, bhikkhu sabbaso nevasaññānāsaññāyatanaṃ samatikkamma saññāvedayitanirodhaṃ upasampajja viharati. Idaṃ kho, ānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca.

### 対訳

1. ‘‘Yo kho, ānanda, evaṃ vadeyya –

* yo = 「～する者が（関係代名詞）」
* kho = 「実に（強調の不変化詞）」
* ānanda = 「アーナンダよ（呼格）」
* evaṃ = 「このように」
* vadeyya = 「言うだろう（vadati の願望法）」

→ 「アーナンダよ、もし誰かがこのように言うとしよう。」

2. ‘etaparamaṃ sattā sukhaṃ somanassaṃ paṭisaṃvedentī’ti,

* eta-paramaṃ = 「これを最高とする」
  * eta = 「これ」
  * paramaṃ = 「最高の、究極の」
* sattā = 「衆生たちは（複数・主格）」
* sukhaṃ = 「楽を」
* somanassaṃ = 「喜（心の喜び）を」
* paṭisaṃvedenti = 「感受する、経験する」
* ti = 「～と（引用）」

→ 「『衆生たちはこれを最高の楽・喜として感受する』と。」

3. idamassa nānujānāmi.

* idaṃ = 「これを」
* assa = 「彼に（その者に）」
* na anujānāmi = 「私は認めない」
  * anujānāti = 「承認する、許可する」

→ 「私はその者にこれ（この主張）を認めない。」

4. Taṃ kissa hetu?

* taṃ = 「それは」
* kissa hetu = 「何の理由によってか」
  * kissa = 「何の（kiṃ の属格）」
  * hetu = 「理由、原因」

→ 「それはなぜか？」

5. Atthānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca.

* atthi = 「ある、存在する」
* ānanda = 「アーナンダよ」
* etamhā sukhā = 「この楽よりも（奪格）」
* aññaṃ sukhaṃ = 「別の楽が」
* abhikkantataraṃ ca = 「よりすぐれた（abhikkanta の比較級）」
* paṇītataraṃ ca = 「より勝妙な→よりすぐれて精妙な（paṇīta の比較級）」

→ 「アーナンダよ、この楽よりもさらにすぐれ、さらに精妙な別の楽があるからである。」

6. Katamañcānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca?

* katamaṃ ca = 「では、どのようなものが」
* ānanda = 「アーナンダよ」
* etamhā sukhā = 「この楽よりも」
* aññaṃ sukhaṃ = 「別の楽が」
* abhikkantataraṃ ca paṇītataraṃ ca = 「よりすぐれ、より精妙な」

→ 「では、アーナンダよ、この楽よりもさらにすぐれ、さらに精妙な別の楽とはどのようなものか？」

7. Idhānanda, bhikkhu sabbaso nevasaññānāsaññāyatanaṃ samatikkamma saññāvedayitanirodhaṃ upasampajja viharati.

* idha = 「ここに（この教えにおいて）」
* ānanda = 「アーナンダよ」
* bhikkhu = 「比丘が」
* sabbaso = 「完全に、あまねく」
* nevasaññānāsaññāyatanaṃ = 「非想非非想処を」
  * na eva saññā = 「想があるのでもなく」
  * na asaññā = 「想がないのでもない」
  * āyatanaṃ = 「処（領域）」
* samatikkamma = 「超えて（samatikkamati の連続体）」
* saññāvedayitanirodhaṃ = 「想受滅を」
  * saññā = 「想」
  * vedayita = 「感受されたもの（受）」
  * nirodhaṃ = 「滅」
* upasampajja = 「達して（upasampajjati の連続体）」
* viharati = 「住する」

→ 「アーナンダよ、ここに比丘が、非想非非想処を完全に超えて、想受滅に達して住する。」

8. Idaṃ kho, ānanda, etamhā sukhā aññaṃ sukhaṃ abhikkantatarañca paṇītatarañca.

* idaṃ kho = 「これこそが」
* ānanda = 「アーナンダよ」
* etamhā sukhā = 「この楽よりも」
* aññaṃ sukhaṃ = 「別の楽」
* abhikkantataraṃ ca paṇītataraṃ ca = 「よりすぐれ、より精妙な」

→ 「アーナンダよ、これこそが、この楽よりもさらにすぐれ、さらに精妙な別の楽である。」

91. ‘‘Ṭhānaṃ kho panetaṃ, ānanda, vijjati yaṃ aññatitthiyā paribbājakā evaṃ vadeyyuṃ – ‘saññāvedayitanirodhaṃ samaṇo gotamo āha; tañca sukhasmiṃ paññapeti. Tayidaṃ kiṃsu, tayidaṃ kathaṃsū’ti? Evaṃvādino, ānanda, aññatitthiyā paribbājakā evamassu vacanīyā – ‘na kho, āvuso, bhagavā sukhaṃyeva vedanaṃ sandhāya sukhasmiṃ paññapeti; api ca, āvuso, yattha yattha sukhaṃ upalabbhati yahiṃ yahiṃ taṃ taṃ tathāgato sukhasmiṃ paññapetī’’’ti.

### 対訳

1. Ṭhānaṃ kho panetaṃ, ānanda, vijjati

* ṭhānaṃ = 「道理、可能性、場合」

* kho pana = 「また実に」

* etaṃ = 「この（こと）」

* ānanda = 「アーナンダよ（呼格）」

* vijjati = 「見出される、あり得る（vid の受動形）」

→ 「アーナンダよ、またこういうことがあり得る。」

2. yaṃ aññatitthiyā paribbājakā evaṃ vadeyyuṃ –

* yaṃ = 「すなわち～ということ（関係代名詞）」

* añña-titthiyā = 「他の宗派の、外道の」

  * añña = 「他の」

  * titthiya = 「宗派の者、外道」

* paribbājakā = 「遍歴行者たち」

* evaṃ = 「このように」

* vadeyyuṃ = 「言うかもしれない（vadati の願望法・三人称複数）」

3. ‘saññāvedayitanirodhaṃ samaṇo gotamo āha; tañca sukhasmiṃ paññapeti.

* saññā-vedayita-nirodhaṃ = 「想受滅を」

  * saññā = 「想」

  * vedayita = 「感受されたもの、受」

  * nirodha = 「滅」

* samaṇo gotamo = 「沙門ゴータマは」

* āha = 「説いた（言った）」

* tañca = taṃ ca = 「そしてそれを」

* sukhasmiṃ = 「楽の中に、楽として（処格）」

* paññapeti = 「設定する、説き示す（paññāpeti）」

→ 「『沙門ゴータマは想受滅を説いた。そしてそれを楽として説き示す。』」

4. Tayidaṃ kiṃsu, tayidaṃ kathaṃsū’ti?

* tayidaṃ = taṃ idaṃ = 「そのこれは、それはいったい」

* kiṃsu = 「何であるのか（kiṃ + 疑問の強調辞 su）」

* kathaṃsu = 「どうしてなのか（kathaṃ + su）」

* ti = 「～と（引用の終わり）」

→ 「『それはいったい何なのか、それはいったいどうしてなのか』と。」

5. Evaṃvādino, ānanda, aññatitthiyā paribbājakā evamassu vacanīyā –

* evaṃ-vādino = 「このように語る者たちは」

  * vādin = 「語る者」

* ānanda = 「アーナンダよ」

* aññatitthiyā paribbājakā = 「他派の遍歴行者たちは」

* evam assu vacanīyā = 「このように言われるべきである」

  * assu = 「～であるべき（atthi の願望法・三人称複数）」

  * vacanīyā = 「言われるべき（vacati の未来受動分詞）」

→ 「アーナンダよ、このように語る他派の遍歴行者たちには、このように言うべきである。」

6. ‘na kho, āvuso, bhagavā sukhaṃyeva vedanaṃ sandhāya sukhasmiṃ paññapeti;

* na kho = 「実に～ではない」

* āvuso = 「友よ（呼格）」

* bhagavā = 「世尊は」

* sukhaṃ yeva vedanaṃ = 「楽なる受だけを」

  * sukhaṃ = 「楽の」

  * yeva = eva = 「～だけ、まさに」

  * vedanaṃ = 「受を」

* sandhāya = 「～に関して、～を指して（saṃ + dhā の連続体）」

* sukhasmiṃ paññapeti = 「楽として説き示す」

→ 「『友よ、世尊は楽なる受だけを指して楽として説き示すのではない。』」

7. api ca, āvuso, yattha yattha sukhaṃ upalabbhati yahiṃ yahiṃ taṃ taṃ tathāgato sukhasmiṃ paññapetī’’’ti.

* upalabbhati = 「見出される、得られる（upa + labh の受動形）」

* avoca = 「語った（vacati のアオリスト）」

9. Attamano āyasmā ānando bhagavato bhāsitaṃ abhinandīti.

10. Bahuvedanīyasuttaṃ niṭṭhitaṃ navamaṃ.

  * bahu = 「多くの」

  * vedanīya = 「感受されるべきもの、受」

  * sutta = 「経」

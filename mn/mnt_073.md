
## 198

198.Abhiññā vā kāraṇanti yañhi taṃ tatra tatra sakkhibhabbatāsaṅkhātaṃ iddhividhapaccanubhavanādi, tassa abhiññā kāraṇaṃ. Atha iddhividhapaccanubhavanādi abhiññā, evaṃ sati abhiññāpādakajjhānaṃ kāraṇaṃ. Avasāne chaṭṭhābhiññāya pana arahattaṃ. Ettha ca yasmā paṭhamasutte āsavakkhayo adhippeto, āsavā khīṇā eva, na puna khepetabbā, tasmā tattha ‘‘yāvadevā’’ti na vuttaṃ. Idha phalasamāpatti adhippetā, sā ca punappunaṃ samāpajjīyati, tasmā ‘‘yāvadevā’’ti vuttaṃ. Tato eva hi ‘‘arahattaṃ vā kāraṇa’’nti vuttaṃ. Tañhi ‘‘kudāssu nāmāhaṃ tadāyatanaṃ upasampajja viharissāmi, yadariyā etarahi upasampajja viharantī’’ti (ma. ni. 1.465; 3.307) anuttaresu vimokkhesu pihaṃ upaṭṭhapetvā abhiññā nibbattentassa kāraṇaṃ, tayidaṃ sabbasādhāraṇaṃ na hotīti sādhāraṇavasena naṃ dassento ‘‘arahattassa vipassanā vā’’ti āha.

### 対訳

1. 198. Abhiññā vā kāraṇanti yañhi taṃ tatra tatra sakkhibhabbatāsaṅkhātaṃ iddhividhapaccanubhavanādi, tassa abhiññā kāraṇaṃ.

* Abhiññā vā kāraṇan ti = 「あるいは神通が原因である」とは
* yaṃ hi taṃ = 実にその
* tatra tatra = それぞれの対象における
* sakkhibhabbatāsaṅkhātaṃ = 実現（証作）能力と呼ばれる
* iddhividhapaccanubhavanādi = 神通の種類の経験などが（主格）
* tassa = それの（属格）
* abhiññā = 神通が（主格）
* kāraṇaṃ = 原因である（ということである）

→ 「198. 『あるいは神通が原因である』とは、実にそのそれぞれの対象における実現能力と呼ばれる『神通の種類の経験』などにとって、神通が原因である（ということである）。」

2. Atha iddhividhapaccanubhavanādi abhiññā, evaṃ sati abhiññāpādakajjhānaṃ kāraṇaṃ.

* atha = しかしながら、あるいは
* iddhividhapaccanubhavanādi = 神通の種類の経験などが
* abhiññā = 神通（そのもの）であるならば
* evaṃ sati = そうであるなら（処格絶対格）
* abhiññāpādakajjhānaṃ = 神通の基礎となる禅那が
* kāraṇaṃ = 原因である

→ 「しかしながら、『神通の種類の経験』などが神通（そのもの）であるならば、そうであるなら、神通の基礎となる禅那が原因である。」

3. Avasāne chaṭṭhābhiññāya pana arahattaṃ.

* avasāne = 最後の箇所においては（処格）
* chaṭṭhābhiññāya = 第六の神通（漏尽通）にとっては（与格・属格）
* pana = 一方、しかし
* arahattaṃ = 阿羅漢位が（原因である）

→ 「しかし最後の箇所において、第六の神通（漏尽通）にとっては、阿羅漢位が（原因である）。」

4. Ettha ca yasmā paṭhamasutte āsavakkhayo adhippeto, āsavā khīṇā eva, na puna khepetabbā, tasmā tattha ‘‘yāvadevā’’ti na vuttaṃ.

* ettha ca = そしてここで
* yasmā = 〜であるから
* paṭhamasutte = 最初の経において
* āsavakkhayo = 漏尽が
* adhippeto = 意図されている（過去分詞）
* āsavā = 諸々の漏（煩悩）は
* khīṇā eva = まさに尽きており
* na puna khepetabbā = 再び尽くされるべきではない（khepetabba）
* tasmā = それゆえに
* tattha = そこでは
* yāvadevā ti = 「望むなら（yāvadeva）」とは
* na vuttaṃ = 説かれていない

→ 「そしてここで、最初の経においては漏尽が意図されているため、諸々の漏はまさに尽きており、再び尽くされるべきではないから、それゆえにそこでは『望むなら』とは説かれていない。」

5. Idha phalasamāpatti adhippetā, sā ca punappunaṃ samāpajjīyati, tasmā ‘‘yāvadevā’’ti vuttaṃ.

* idha = ここ（この経）では
* phalasamāpatti = 果等至（阿羅漢果の定）が
* adhippetā = 意図されている
* sā ca = そしてそれは
* punappunaṃ = 何度も、再三にわたり
* samāpajjīyati = 入定される、等至される（受動態）
* tasmā = それゆえに
* yāvadevā ti vuttaṃ = 「望むなら」と説かれている

→ 「ここ（この経）では果等至が意図されており、そしてそれは何度も入定されるものであるから、それゆえに『望むなら』と説かれているのである。」

6. Tato eva hi ‘‘arahattaṃ vā kāraṇa’’nti vuttaṃ.

* tato eva hi = まさにそれゆえに
* arahattaṃ vā kāraṇan ti = 「あるいは阿羅漢位が原因である」と
* vuttaṃ = 説かれている

→ 「まさにそれゆえに、『あるいは阿羅漢位が原因である』と説かれている。」

7. Tañhi ‘‘kudāssu nāmāhaṃ tadāyatanaṃ upasampajja viharissāmi, yadariyā etarahi upasampajja viharantī’’ti (ma. ni. 1.465; 3.307) anuttaresu vimokkhesu pihaṃ upaṭṭhapetvā abhiññā nibbattentassa kāraṇaṃ,

* tañ hi = なぜならそれ（阿羅漢位）は
* kudāssu nāma ahaṃ = 「いつ私は（kudā + su + nāma + ahaṃ）」
* tadāyatanaṃ = その基盤（境地）に
* upasampajja viharissāmi = 到達して住するのだろうか
* yadariyā = 「聖者たちが（yaṃ + ariyā）」
* etarahi = 今、現在
* upasampajja viharantī ti = 到達して住している（境地に）」と
* anuttaresu vimokkhesu = 無上の解脱に対して（処格）
* pihaṃ upaṭṭhapetvā = 希求（憧れ）を起こして
* abhiññā nibbattentassa = 神通を生じさせる者の（属格）
* kāraṇaṃ = 原因（となるからである）

→ 「なぜならそれは、『聖者たちが今到達して住している境地に、私はいつ到達して住するのだろうか』と無上の解脱に対して希求を起こし、神通を生じさせる者にとっての原因（となるからである）。」

8. tayidaṃ sabbasādhāraṇaṃ na hotīti sādhāraṇavasena naṃ dassento ‘‘arahattassa vipassanā vā’’ti āha.

* tayidaṃ = しかしそれは（taṃ idaṃ）
* sabbasādhāraṇaṃ = 全て（の者）に共通するものでは
* na hoti ti = ない、と（考えて）
* sādhāraṇavasena = 共通するものとして
* naṃ = それを
* dassento = 示す者として（＝註釈者は示すために）
* arahattassa vipassanā vā ti = 「あるいは阿羅漢位のためのヴィパッサナーが（原因である）」と
* āha = 言ったのである

→ 「しかしそれは全て（の修行者）に共通するものではないと考え、共通するものとしてそれを示すために、（註釈者は）『あるいは阿羅漢位のためのヴィパッサナーが（原因である）』と言ったのである。」

## 200

200.Paricaranti nāma vippakatabrahmacariyavāsattā. Pariciṇṇo hoti sāvakena nāma satthu dhamme kattabbā paricariyā sammadeva niṭṭhāpitattā. Tenāha ‘‘iti…pe… thero evamāhā’’ti. Tesaṃ guṇānanti tesaṃ asekkhaguṇānaṃ. Sesaṃ suviññeyyameva.

Mahāvacchasuttavaṇṇanāya līnatthappakāsanā samattā.

### 対訳

1. 200. Paricaranti nāma vippakatabrahmacariyavāsattā.

* 200. = 「段落番号 200」
* paricaranti nāma = 「『仕えている』とは」
* vippakatabrahmacariyavāsattā (vippakata-brahmacariya-vāsa-ttā) = 「梵行の実践が未完成であるからである」

→ 「200. 『仕えている（最中である）』とは、梵行の実践が未完成であるからである。」

2. Pariciṇṇo hoti sāvakena nāma satthu dhamme kattabbā paricariyā sammadeva niṭṭhāpitattā.

* pariciṇṇo hoti = 「『仕えられた（完了した）状態となる』とは」
* sāvakena nāma = 「声聞によって」
* satthu = 「師の」
* dhamme = 「法において」
* kattabbā = 「なされるべき」
* paricariyā = 「奉仕が」
* sammadeva (sammā eva) = 「完全に」
* niṭṭhāpitattā = 「終えられたからである」

→ 「声聞によって『仕えられた状態となる』とは、師の法においてなされるべき奉仕が完全に終えられたからである。」

3. Tenāha ‘‘iti…pe… thero evamāhā’’ti.

* tenāha (tena āha) = 「それゆえに、（註釈者は）言った」
* iti = 「このように」
* pe = 「…省略…」
* thero = 「長老は」
* evamāhā ti (evaṃ āha ti) = 「このように言った、と」

→ 「それゆえに『このように…（省略）…長老はこのように言った』と（註釈者は）言ったのである。」

4. Tesaṃ guṇānanti tesaṃ asekkhaguṇānaṃ.

* tesaṃ guṇānan ti = 「『それらの功徳を』とは」
* tesaṃ = 「それらの」
* asekkhaguṇānaṃ (asekkha-guṇānaṃ) = 「無学（阿羅漢）の功徳のことである」

→ 「『それらの功徳を』とは、それらの無学の功徳のことである。」

5. Sesaṃ suviññeyyameva.

* sesaṃ = 「残りの部分は」
* suviññeyyameva (suviññeyyaṃ eva) = 「まさしく容易に理解できるものである」

→ 「残りの部分は、まさしく容易に理解できるものである。」

6. Mahāvacchasuttavaṇṇanāya līnatthappakāsanā samattā.

* mahāvacchasuttavaṇṇanāya = 「マハーヴァッチャ経の解説（註釈）に対する」
* līnatthappakāsanā = 「隠れた意味の開明（復註『リーナッタッパカーサナー』）が」
* samattā = 「完了した」

→ 「マハーヴァッチャ経の解説に対する『隠れた意味の開明（復註）』が終わった。」

# 7. Cātumasuttavaṇṇanā

## 157 (1)

157.Evaṃme sutanti cātumasuttaṃ. Tattha cātumāyanti evaṃnāmake gāme. Pañcamattāni bhikkhusatānīti adhunā pabbajitānaṃ bhikkhūnaṃ pañca satāni. Therā kira cintesuṃ – ‘‘ime kulaputtā dasabalaṃ adisvāva pabbajitā, etesaṃ bhagavantaṃ dassessāma, bhagavato santike dhammaṃ sutvā attano attano yathāupanissayena patiṭṭhahissantī’’ti. Tasmā te bhikkhū gahetvā āgatā. Paṭisammodamānāti ‘‘kaccāvuso, khamanīya’’ntiādiṃ paṭisanthārakathaṃ kurumānā. Senāsanāni paññāpayamānāti attano attano ācariyupajjhāyānaṃ vasanaṭṭhānāni pucchitvā dvāravātapānāni vivaritvā mañcapīṭhakaṭasārakādīni nīharitvā papphoṭetvā yathāṭṭhāne saṇṭhāpayamānā. Pattacīvarāni paṭisāmayamānāti, bhante, idaṃ me pattaṃ ṭhapetha, idaṃ cīvaraṃ, idaṃ thālakaṃ, idaṃ udakatumbaṃ, imaṃ kattarayaṭṭhinti evaṃ samaṇaparikkhāre saṅgopayamānā.

6. evamevaṃ imāni cattāri bhayāni bhāyitvā sāsane apabbajantassāpi imaṃ sāsanaṃ nissāya ānisaṃso natthi, taṇhāpipāsāya pipāsito ca hoti kilesarajena saṃkiliṭṭhacitto ca.

* evamevaṃ = 「まさにそのように」
* imāni cattāri bhayāni = 「これら4つの恐怖を」
* bhāyitvā = 「恐れて」
* sāsane = 「教えにおいて」
* apabbajantassāpi = 「出家しない者にも」
* imaṃ sāsanaṃ nissāya = 「この教えに由来する」
* ānisaṃso natthi = 「利益はなく」
* taṇhāpipāsāya = 「渇愛の渇きによって」
* pipāsito ca hoti = 「喉が渇いたままとなり」
* kilesarajena = 「煩悩の塵によって」
* saṃkiliṭṭhacitto ca = 「心が汚れたままとなる」

→ 「まさにそのように、これら4つの恐怖を恐れてこの教えに出家しない者にも、この教えによる利益はなく、渇愛の渇きに苦しみ、煩悩の塵で心が汚れたままとなる。」

7. Yathā pana imāni cattāri bhayāni abhāyitvā udakaṃ orohantassa vuttappakāro ānisaṃso hoti, evaṃ imāni abhāyitvā sāsane pabbajitassāpi vuttappakāro ānisaṃso hoti.

* yathā pana = 「しかし、ちょうど～のように」
* imāni cattāri bhayāni = 「これら4つの恐怖を」
* abhāyitvā = 「恐れることなく」
* udakaṃ orohantassa = 「水に入る者には」
* vuttappakāro = 「説かれたような種類の（上述の）」
* ānisaṃso hoti = 「利益がある（ように）」
* evaṃ = 「そのように」
* imāni abhāyitvā = 「これらを恐れることなく」
* sāsane pabbajitassāpi = 「教えに出家した者にも」
* vuttappakāro ānisaṃso hoti = 「上述の利益がある」

→ 「しかし、ちょうどこれら4つの恐怖を恐れることなく水に入る者には上述の利益があるように、これらを恐れることなく教えに出家した者にも上述の利益がある。」

8. Thero panāha – ‘‘cattāri bhayāni bhāyitvā udakaṃ anotaranto sotaṃ chinditvā paratīraṃ pāpuṇituṃ na sakkoti, abhāyitvā otaranto sakkoti, evamevaṃ bhāyitvā sāsane apabbajantopi taṇhāsotaṃ chinditvā nibbānapāraṃ daṭṭhuṃ na sakkoti, abhāyitvā pabbajanto pana sakkotī’’ti.

* thero panāha (thero pana āha) = 「しかし、長老は言った」
* cattāri bhayāni = 「4つの恐怖を」
* bhāyitvā = 「恐れて」
* udakaṃ anotaranto = 「水に入らない者は」
* sotaṃ chinditvā = 「流れを断ち切って」
* paratīraṃ pāpuṇituṃ = 「対岸に到達することは」
* na sakkoti = 「できない（が）」
* abhāyitvā otaranto = 「恐れることなく入る者は」
* sakkoti = 「できる」
* evamevaṃ = 「まさにそのように」
* bhāyitvā = 「恐れて」
* sāsane apabbajantopi = 「教えに出家しない者も」
* taṇhāsotaṃ chinditvā = 「渇愛の流れを断ち切って」
* nibbānapāraṃ = 「涅槃の彼岸を」
* daṭṭhuṃ = 「見ることは」
* na sakkoti = 「できない（が）」
* abhāyitvā = 「恐れることなく」
* pabbajanto pana = 「出家する者は」
* sakkotī ti = 「できる、と」

→ 「しかし、長老は言った。『4つの恐怖を恐れて水に入らない者は、流れを断ち切って対岸に到達することはできないが、恐れることなく入る者は到達できる。まさにそのように、恐れて教えに出家しない者も、渇愛の流れを断ち切って涅槃の彼岸を見ることはできないが、恐れることなく出家する者は見ることができるのである』と。」

9. Sesaṃ sabbattha uttānameva.

* sesaṃ = 「残りの部分は」
* sabbattha = 「すべてにおいて」
* uttānameva (uttānaṃ eva) = 「明白である」

→ 「残りの部分はすべて明白である。」

10. Ayaṃ pana desanā neyyapuggalassa vasena niṭṭhāpitāti.

* ayaṃ pana desanā = 「そして、この説法は」
* neyyapuggalassa = 「所導の者（教導されて悟るべき人物）の」
* vasena = 「力によって（対象として）」
* niṭṭhāpitā ti = 「完結されたのである、と」

→ 「そして、この説法は所導の者を対象として完結されたのである。」

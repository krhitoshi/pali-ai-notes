# 5. Māgaṇḍiyasuttaṃ

## 207 (1)

207. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā kurūsu viharati kammāsadhammaṃ nāma kurūnaṃ nigamo, bhāradvājagottassa brāhmaṇassa agyāgāre tiṇasanthārake [tiṇasantharake (sī. syā. kaṃ. pī.)]. Atha kho bhagavā pubbaṇhasamayaṃ nivāsetvā pattacīvaramādāya kammāsadhammaṃ piṇḍāya pāvisi. Kammāsadhammaṃ piṇḍāya caritvā pacchābhattaṃ piṇḍapātapaṭikkanto yena aññataro vanasaṇḍo tenupasaṅkami divāvihārāya. Taṃ vanasaṇḍaṃ ajjhogāhetvā aññatarasmiṃ rukkhamūle divāvihāraṃ nisīdi. Atha kho māgaṇḍiyo [māgandiyo (sī. pī.)] paribbājako jaṅghāvihāraṃ anucaṅkamamāno anuvicaramāno yena bhāradvājagottassa brāhmaṇassa agyāgāraṃ tenupasaṅkami. Addasā kho māgaṇḍiyo paribbājako bhāradvājagottassa brāhmaṇassa agyāgāre tiṇasanthārakaṃ paññattaṃ. Disvāna bhāradvājagottaṃ brāhmaṇaṃ etadavoca – ‘‘kassa nvayaṃ bhoto bhāradvājassa agyāgāre tiṇasanthārako paññatto, samaṇaseyyānurūpaṃ [samaṇaseyyārūpaṃ (sī. pī.)] maññe’’ti? ‘‘Atthi, bho māgaṇḍiya, samaṇo gotamo sakyaputto sakyakulā pabbajito. Taṃ kho pana bhavantaṃ gotamaṃ evaṃ kalyāṇo kittisaddo abbhuggato – ‘itipi so bhagavā arahaṃ sammāsambuddho vijjācaraṇasampanno sugato lokavidū anuttaro purisadammasārathi satthā devamanussānaṃ buddho bhagavā’ti. Tassesā bhoto gotamassa seyyā paññattā’’ti.

* kāma-upasaṃhitehi = 「欲を伴う」
* rajanīyehi = 「貪欲をそそる（魅力的な）」

→ ここに、ある者が、望ましく、愛すべき、心にかなう、好ましく、欲を伴い、貪欲をそそる、目で認識される対象（色）によって、かつて楽しんだことがあるとしよう。

3. so aparena samayena rūpānaṃyeva samudayañca atthaṅgamañca assādañca ādīnavañca nissaraṇañca yathābhūtaṃ viditvā

* so = 「彼は」
* aparena samayena = 「後に（別の時に）」
* rūpānaṃ yeva = 「まさにその色の」
* samudayañ ca = 「生起と」
* atthaṅgamañ ca = 「滅尽と」
* assādañ ca = 「魅力（味の良さ）と」
* ādīnavañ ca = 「患難（危険）と」
* nissaraṇañ ca = 「出離を」
* yathā-bhūtaṃ = 「あるがままに」
* viditvā = 「知って」

→ 彼は後に、まさにその色の生起と滅尽と魅力と患難と出離をあるがままに知って、

4. rūpataṇhaṃ pahāya rūpapariḷāhaṃ paṭivinodetvā vigatapipāso ajjhattaṃ vūpasantacitto vihareyya.

* rūpa-taṇhaṃ = 「色に対する渇愛を」
* pahāya = 「断ち切って」
* rūpa-pariḷāhaṃ = 「色に対する煩悶（熱）を」
* paṭivinodetvā = 「除去して」
* vigata-pipāso = 「渇き（渇愛）が離れた者として」
* ajjhattaṃ = 「内面的に」
* vūpasanta-citto = 「心が寂静に達した者として」
* vihareyya = 「留まる（過ごす）であろう」

→ 色に対する渇愛を断ち切り、色に対する煩悶を除去して、渇きを離れ、内面的に心が寂静に達して過ごすとしよう。

5. Imassa pana te, māgaṇḍiya, kimassa vacanīya’’’nti?

* imassa = 「この者について」
* pana = 「ところで」
* te = 「あなたにとって（あなたによって）」
* māgaṇḍiya = 「マガンディヤよ」
* kiṃ assa vacanīyaṃ = 「何が言われるべきであろうか」
* ti = 「と（引用の終了）」

→ マガンディヤよ、ところで、この者についてあなたは何を言うべきであろうか（何を非難できようか）」と。

6. ‘‘Na kiñci, bho gotama’’.

* na kiñci = 「何も（ない）」
* bho gotama = 「尊きゴータマよ」

→ 「何もありません、尊きゴータマよ」。

7. ‘‘Taṃ kiṃ maññasi, māgaṇḍiya – ‘idhekacco sotaviññeyyehi saddehi…pe… ghānaviññeyyehi gandhehi… jivhāviññeyyehi rasehi… kāyaviññeyyehi phoṭṭhabbehi paricāritapubbo assa iṭṭhehi kantehi manāpehi piyarūpehi kāmūpasaṃhitehi rajanīyehi,

* Taṃ kiṃ maññasi, māgaṇḍiya = 「マガンディヤよ、それをどう思うか。」
* idhekacco = 「ここに、ある者が」
* sota-viññeyyehi saddehi = 「耳で認識されるべき音（声）によって」
* …pe… = 「…（中略）…」
* ghāna-viññeyyehi gandhehi = 「鼻で認識されるべき匂い（香）によって」
* jivhā-viññeyyehi rasehi = 「舌で認識されるべき味によって」
* kāya-viññeyyehi phoṭṭhabbehi = 「身体で認識されるべき触覚対象によって」
* paricārita-pubbo assa = 「かつて楽しんだことがあるとしよう」
* iṭṭhehi kantehi manāpehi piyarūpehi kāmūpasaṃhitehi rajanīyehi = 「望ましく、愛すべき、心にかなう、好ましく、欲を伴い、貪欲をそそる（対象によって）」

→ 「マガンディヤよ、それをどう思うか。ここに、ある者が、望ましく、愛すべき、心にかなう、好ましく、欲を伴い、貪欲をそそる、耳で認識される音で…（中略）…鼻で認識される匂いで…舌で認識される味で…身体で認識される触覚対象によって、かつて楽しんだことがあるとしよう。

8. so aparena samayena phoṭṭhabbānaṃyeva samudayañca atthaṅgamañca assādañca ādīnavañca nissaraṇañca yathābhūtaṃ viditvā phoṭṭhabbataṇhaṃ pahāya phoṭṭhabbapariḷāhaṃ paṭivinodetvā vigatapipāso ajjhattaṃ vūpasantacitto vihareyya.

* so aparena samayena = 「彼は後に」
* phoṭṭhabbānaṃ yeva = 「まさにその触覚対象の」
* samudayañ-ca atthaṅgamañ-ca assādañ-ca ādīnavañ-ca nissaraṇañ-ca yathābhūtaṃ viditvā = 「生起と滅尽と魅力と患難と出離をあるがままに知って」
* phoṭṭhabba-taṇhaṃ pahāya = 「触覚対象に対する渇愛を断ち切り」
* phoṭṭhabba-pariḷāhaṃ paṭivinodetvā = 「触覚対象に対する煩悶を除去して」
* vigata-pipāso = 「渇きを離れ」
* ajjhattaṃ vūpasanta-citto vihareyya = 「内面的に心が寂静に達して過ごすとしよう。」

→ 彼は後に、まさにその触覚対象の生起と滅尽と魅力と患難と出離をあるがままに知って、触覚対象に対する渇愛を断ち切り、触覚対象に対する煩悶を除去して、渇きを離れ、内面的に心が寂静に達して過ごすとしよう。

9. Imassa pana te, māgaṇḍiya, kimassa vacanīya’’’nti? ‘‘Na kiñci, bho gotama’’.

* Imassa pana te, māgaṇḍiya, kimassa vacanīyaṃ = 「マガンディヤよ、ところで、この者についてあなたは何を言うべきであろうか」
* ti = 「と（引用の終了）」
* Na kiñci, bho gotama = 「何もありません、尊きゴータマよ」。

→ マガンディヤよ、ところで、この者についてあなたは何を言うべきであろうか」と。 「何もありません、尊きゴータマよ」。

# 5. Māgaṇḍiyasuttaṃ

## 207 (1)

207. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā kurūsu viharati kammāsadhammaṃ nāma kurūnaṃ nigamo, bhāradvājagottassa brāhmaṇassa agyāgāre tiṇasanthārake [tiṇasantharake (sī. syā. kaṃ. pī.)]. Atha kho bhagavā pubbaṇhasamayaṃ nivāsetvā pattacīvaramādāya kammāsadhammaṃ piṇḍāya pāvisi. Kammāsadhammaṃ piṇḍāya caritvā pacchābhattaṃ piṇḍapātapaṭikkanto yena aññataro vanasaṇḍo tenupasaṅkami divāvihārāya. Taṃ vanasaṇḍaṃ ajjhogāhetvā aññatarasmiṃ rukkhamūle divāvihāraṃ nisīdi. Atha kho māgaṇḍiyo [māgandiyo (sī. pī.)] paribbājako jaṅghāvihāraṃ anucaṅkamamāno anuvicaramāno yena bhāradvājagottassa brāhmaṇassa agyāgāraṃ tenupasaṅkami. Addasā kho māgaṇḍiyo paribbājako bhāradvājagottassa brāhmaṇassa agyāgāre tiṇasanthārakaṃ paññattaṃ. Disvāna bhāradvājagottaṃ brāhmaṇaṃ etadavoca – ‘‘kassa nvayaṃ bhoto bhāradvājassa agyāgāre tiṇasanthārako paññatto, samaṇaseyyānurūpaṃ [samaṇaseyyārūpaṃ (sī. pī.)] maññe’’ti? ‘‘Atthi, bho māgaṇḍiya, samaṇo gotamo sakyaputto sakyakulā pabbajito. Taṃ kho pana bhavantaṃ gotamaṃ evaṃ kalyāṇo kittisaddo abbhuggato – ‘itipi so bhagavā arahaṃ sammāsambuddho vijjācaraṇasampanno sugato lokavidū anuttaro purisadammasārathi satthā devamanussānaṃ buddho bhagavā’ti. Tassesā bhoto gotamassa seyyā paññattā’’ti.

→ 「このように説かれたとき、遊行者マーガンディヤは世尊にこう言った。」

2. ‘‘abhikkantaṃ, bho gotama, abhikkantaṃ, bho gotama!

* abhikkantaṃ = 「素晴らしいです、見事です」
* bho = 「尊き」
* gotama = 「ゴータマよ」

→ 「『素晴らしい、尊きゴータマよ。素晴らしい、尊きゴータマよ！

3. Seyyathāpi, bho gotama, nikkujjitaṃ vā ukkujjeyya, paṭicchannaṃ vā vivareyya, mūḷhassa vā maggaṃ ācikkheyya, andhakāre vā telapajjotaṃ dhāreyya – cakkhumanto rūpāni dakkhantīti;

* seyyathāpi = 「あたかも～のように」
* bho gotama = 「尊きゴータマよ」
* nikkujjitaṃ = 「伏せられたものを」
* vā = 「あるいは」
* ukkujjeyya = 「仰向けにする（上を向ける）ように」
* paṭicchannaṃ = 「隠されたものを」
* vivareyya = 「現す（開く）ように」
* mūḷhassa = 「迷った者に」
* maggaṃ = 「道を」
* ācikkheyya = 「教えるように」
* andhakāre = 「暗闇に」
* telapajjotaṃ = 「油灯を」
* dhāreyya = 「掲げるように」
* cakkhumanto = 「眼ある者たちは」
* rūpāni = 「形（色）を」
* dakkhanti = 「見るだろう」
* ti = 「と」

→ 「尊きゴータマよ、あたかも、伏せられたものを仰向けにするように、隠されたものを現すように、迷った者に道を教えるように、あるいは『眼ある者たちは形を見るだろう』と暗闇に油灯を掲げるように、

4. evamevaṃ bhotā gotamena anekapariyāyena dhammo pakāsito.

* evamevaṃ = 「全くそのように」
* bhotā = 「尊き」
* gotamena = 「ゴータマによって」
* anekapariyāyena = 「様々な方法で（幾多の法門によって）」
* dhammo = 「法が」
* pakāsito = 「明らかにされた」

→ 「全くそのように、尊きゴータマによって様々な方法で法が明らかにされました。

5. Esāhaṃ bhavantaṃ gotamaṃ saraṇaṃ gacchāmi dhammañca bhikkhusaṅghañca.

* esāhaṃ (eso + ahaṃ) = 「いま私は」
* bhavantaṃ = 「尊き」
* gotamaṃ = 「ゴータマに」
* saraṇaṃ = 「帰依処として」
* gacchāmi = 「行きます（帰依します）」
* dhammañca (dhammaṃ + ca) = 「法と」
* bhikkhusaṅghañca (bhikkhusaṅghaṃ + ca) = 「比丘僧伽に」

→ 「私はいま、尊きゴータマに帰依します。また法と、比丘僧伽に帰依します。

6. Labheyyāhaṃ bhoto gotamassa santike pabbajjaṃ, labheyyaṃ upasampada’’nti.

* labheyyāhaṃ (labheyyaṃ + ahaṃ) = 「私は得られますように」
* bhoto = 「尊き」
* gotamassa = 「ゴータマの」
* santike = 「御許（おんもと）で」
* pabbajjaṃ = 「出家を」
* labheyyaṃ = 「得られますように」
* upasampadaṃ = 「具足戒を（受戒を）」
* ti = 「と」

→ 「私は、尊きゴータマの御許で出家を得られますように、具足戒を得られますように』と。」

7. ‘‘Yo kho, māgaṇḍiya, aññatitthiyapubbo imasmiṃ dhammavinaye ākaṅkhati pabbajjaṃ, ākaṅkhati upasampadaṃ,

* yo = 「いかなる（者が）」
* kho = 「実に」
* māgaṇḍiya = 「マーガンディヤよ」
* aññatitthiyapubbo = 「以前に他の外道であった（者）」
* imasmiṃ = 「この」
* dhammavinaye = 「法と律において」
* ākaṅkhati = 「望む（ならば）」
* pabbajjaṃ = 「出家を」
* upasampadaṃ = 「具足戒を」

→ 「『マーガンディヤよ、以前に他の外道であった者が、この法と律において出家を望み、具足戒を望むならば、

8. so cattāro māse parivasati;

* so = 「その者は」
* cattāro = 「４つの」
* māse = 「月を（４ヶ月間）」
* parivasati = 「別住する（見習い期間を過ごす）」

→ 「その者は４ヶ月間別住する。

9. catunnaṃ māsānaṃ accayena āraddhacittā bhikkhū pabbājenti, upasampādenti bhikkhubhāvāya.

* catunnaṃ = 「４つの」
* māsānaṃ = 「月の」
* accayena = 「経過によって（過ぎた後に）」
* āraddhacittā = 「満足した心を持つ（納得した）」
* bhikkhū = 「比丘たちが」
* pabbājenti = 「出家させる」
* upasampādenti = 「具足戒を授ける」
* bhikkhubhāvāya = 「比丘の状態（身分）へと」

→ 「４ヶ月が経過した後に、比丘たちが心に満足したならば、出家させ、比丘の身分へと具足戒を授ける。

10. Api ca mettha puggalavemattatā viditā’’ti.

* api ca = 「しかしながら」
* mettha (me + ettha) = 「私には・ここにおいて（この点において）」
* puggalavemattatā = 「人の違い（個人の差異）が」
* viditā = 「知られている（分かっている）」
* ti = 「と」

→ 「しかしながら、この点において私には個人の違いが分かっている』と。」

## 222 (2

 ‘‘Sace, bhante, aññatitthiyapubbā imasmiṃ dhammavinaye ākaṅkhantā pabbajjaṃ, ākaṅkhantā upasampadaṃ cattāro māse parivasanti, catunnaṃ māsānaṃ accayena āraddhacittā bhikkhū pabbājenti upasampādenti bhikkhubhāvāya; ahaṃ cattāri vassāni parivasissāmi, catunnaṃ vassānaṃ accayena āraddhacittā bhikkhū pabbājentu, upasampādentu bhikkhubhāvāyā’’ti . Alattha kho māgaṇḍiyo paribbājako bhagavato santike pabbajjaṃ, alattha upasampadaṃ. Acirūpasampanno kho panāyasmā māgaṇḍiyo eko vūpakaṭṭho appamatto ātāpī pahitatto viharanto nacirasseva – yassatthāya kulaputtā sammadeva agārasmā anagāriyaṃ pabbajanti tadanuttaraṃ – brahmacariyapariyosānaṃ diṭṭheva dhamme sayaṃ abhiññā sacchikatvā upasampajja vihāsi. ‘Khīṇā jāti, vusitaṃ brahmacariyaṃ, kataṃ karaṇīyaṃ, nāparaṃ itthattāyā’ti abbhaññāsi. Aññataro kho panāyasmā māgaṇḍiyo arahataṃ ahosīti.

Māgaṇḍiyasuttaṃ niṭṭhitaṃ pañcamaṃ.

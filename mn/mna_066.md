# 6. Laṭukikopamasuttavaṇṇanā

## 148

148.Evaṃme sutanti laṭukikopamasuttaṃ. Tattha yena so vanasaṇḍoti ayampi mahāudāyitthero bhagavatā saddhiṃyeva piṇḍāya pavisitvā saddhiṃ paṭikkami. Tasmā yena so bhagavatā upasaṅkamanto vanasaṇḍo tenupasaṅkamīti veditabbo. Apahattāti apahārako. Upahattāti upahārako. Paṭisallānā vuṭṭhitoti phalasamāpattito vuṭṭhito.

1. 148. Evaṃ me sutan ti laṭukikopamasuttaṃ.

* yena so vanasaṇḍo = 「その林がある所へ」
* ti = 「とは」

→ 「その中で『その林がある所へ』とは、」

→ 「それゆえに、世尊が向かわれたその林のある所へ（長老も）向かった、と理解すべきである。」

* Paṭisallānā = 「独座瞑想から」

* phala-samāpattito = 「果等至（果定）から」
* vuṭṭhito = 「出定して（という意味である）」

→ 「『独座から立ち上がり』とは、果等至（聖者の果の瞑想境地）から出定して、という意味である。」

1. 149. Yaṃbhagavāti yasmiṃ samaye bhagavā.

* na = 「～ではない」

## 149 (2)

Puna bhūtapubbanti iminā ratti vikālabhojane ādīnavaṃ dasseti. Tattha andhakāratimisāyanti bahalandhakāre. Māṇavehīti corehi. Katakammehīti katacorakammehi. Corā kira katakammā yaṃ nesaṃ devataṃ āyācitvā kammaṃ nipphannaṃ, tassa upahāratthāya manusse māretvā galalohitādīni gaṇhanti. Te aññesu manussesu māriyamānesu kolāhalā uppajjissanti, pabbajitaṃ pariyesanto nāma natthīti maññamānā bhikkhū gahetvā mārenti. Taṃ sandhāyetaṃ vuttaṃ. Akatakammehīti aṭavito gāmaṃ āgamanakāle kammanipphannatthaṃ puretaraṃ balikammaṃ kātukāmehi. Asaddhammena nimantetīti ‘‘ehi bhikkhu ajjekarattiṃ idheva bhuñjitvā idha vasitvā sampattiṃ anubhavitvā sve gamissasī’’ti methunadhammena nimanteti.

Puna bhūtapubbanti iminā attanā diṭṭhakāraṇaṃ katheti. Vijjantarikāyāti vijjuvijjotanakkhaṇe . Vissaramakāsīti mahāsaddamakāsi. Abhummeti bhū’ti vaḍḍhi, abhū’ti avaḍḍhi, vināso mayhanti attho. Pisāco vata manti pisāco maṃ khādituṃ āgato vata. Ātumārī mātumārīti ettha ātūti pitā, mātūti mātā. → 「『pisāco vata maṃ（まったく私に悪鬼が）』とは、悪鬼が私を食うために来たのだ、ということである。」

15. Ātumārī mātumārīti ettha ātūti pitā, mātūti mātā.

* ātumārī mātumārīti = 「『ātumārī mātumārī（父は死ね、母は死ね）』について」
* ettha = 「ここで」
* ātūti = 「『ātu』とは」
* pitā = 「父（のこと）」
* mātūti = 「『mātu』とは」
* mātā = 「母のことである」

→ 「『ātumārī mātumārī（父は死ね、母は死ね）』について、ここで『ātu』とは父であり、『mātu』とは母のことである。」

16. Idaṃ vuttaṃ hoti – yassa pitā vā mātā vā atthi, taṃ mātāpitaro amhākaṃ puttakoti yathā tathā vā uppādetvā yaṃkiñci khādanīyabhojanīyaṃ datvā ekasmiṃ ṭhāne sayāpenti.

* idaṃ = 「これが」
* vuttaṃ hoti = 「語られたことである（すなわち～という意味である）」
* yassa = 「誰であれ、その人に」
* pitā vā mātā vā = 「父や母が」
* atthi = 「いるならば」
* taṃ = 「その者を」
* mātāpitaro = 「父母は」
* amhākaṃ = 「我々の」
* puttako = 「息子だ」
* 'ti = 「と」
* yathā tathā vā = 「どのようにしてであれ」
* uppādetvā = 「（食べ物を）手に入れて」
* yaṃkiñci = 「何らかの」
* khādanīyabhojanīyaṃ = 「噛む食べ物や柔らかい食べ物を」
* datvā = 「与えて」
* ekasmiṃ ṭhāne = 「ある場所（一つの所）に」
* sayāpenti = 「寝かせるものである」

→ 「これは次のような意味である。誰であれ、その人に父母がいるならば、父母は『我々の息子だ』と、何とかして食べ物を手に入れて、何らかの噛む食べ物や柔らかい食べ物を与え、一つの場所に寝かせるものである。

17. So evaṃ rattiṃ piṇḍāya na carati.

* so = 「彼は（そのような人は）」
* evaṃ = 「このように」
* rattiṃ = 「夜に」
* piṇḍāya = 「托鉢のために」
* na carati = 「歩き回らない」

→ 「そのような人は、このように夜に托鉢のために歩き回ることはない。

18. Tuyhaṃ pana mātāpitaro matā maññe, tena evaṃ carasīti.

* tuyhaṃ = 「あなたの」
* pana = 「しかし」
* mātāpitaro = 「父母は」
* matā = 「死んでいる」
* maññe = 「と私は思う」
* tena = 「だから」
* evaṃ = 「このように」
* carasi = 「歩き回っているのだ」
* 'ti = 「と」

→ 「しかしあなたの父母は死んでいるのだと思う、だからこのように歩き回っているのだ』と。」

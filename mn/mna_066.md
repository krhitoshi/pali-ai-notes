# 6. Laṭukikopamasuttavaṇṇanā

## 148

148.Evaṃme sutanti laṭukikopamasuttaṃ. Tattha yena so vanasaṇḍoti ayampi mahāudāyitthero bhagavatā saddhiṃyeva piṇḍāya pavisitvā saddhiṃ paṭikkami. Tasmā yena so bhagavatā upasaṅkamanto vanasaṇḍo tenupasaṅkamīti veditabbo. Apahattāti apahārako. Upahattāti upahārako. Paṭisallānā vuṭṭhitoti phalasamāpattito vuṭṭhito.

→ 「その中で『その林がある所へ』とは、」

→ 「『独座から立ち上がり』とは、果等至（聖者の果の瞑想境地）から出定して、という意味である。」

1. 149. Yaṃbhagavāti yasmiṃ samaye bhagavā.

## 149 (2)

Puna bhūtapubbanti iminā ratti vikālabhojane ādīnavaṃ dasseti. Tattha andhakāratimisāyanti bahalandhakāre. Māṇavehīti corehi. Katakammehīti katacorakammehi. Corā kira katakammā yaṃ nesaṃ devataṃ āyācitvā kammaṃ nipphannaṃ, tassa upahāratthāya manusse māretvā galalohitādīni gaṇhanti. Te aññesu manussesu māriyamānesu kolāhalā uppajjissanti, pabbajitaṃ pariyesanto nāma natthīti maññamānā bhikkhū gahetvā mārenti. Taṃ sandhāyetaṃ vuttaṃ. Akatakammehīti aṭavito gāmaṃ āgamanakāle kammanipphannatthaṃ puretaraṃ balikammaṃ kātukāmehi. Asaddhammena nimantetīti ‘‘ehi bhikkhu ajjekarattiṃ idheva bhuñjitvā idha vasitvā sampattiṃ anubhavitvā sve gamissasī’’ti methunadhammena nimanteti.

Puna bhūtapubbanti iminā attanā diṭṭhakāraṇaṃ katheti. Vijjantarikāyāti vijjuvijjotanakkhaṇe . Vissaramakāsīti mahāsaddamakāsi. Abhummeti bhū’ti vaḍḍhi, abhū’ti avaḍḍhi, vināso mayhanti attho. Pisāco vata manti pisāco maṃ khādituṃ āgato vata. Ātumārī mātumārīti ettha ātūti pitā, mātūti mātā. →「『草や蔓や木材などの資材は集めるのが困難である。どうすればいいだろうか』と思い悩むのである。」

6. Athassa taṃ agārakaṃ vejayantapāsādo viya upaṭṭhāti.

* atha = 「すると」
* assa = 「彼にとって」
* taṃ agārakaṃ = 「その小さな家が」
* vejayantapāsādo viya = 「（インドラ神の）ヴェージャヤンタ宮殿のように」
* upaṭṭhāti = 「現れる（思える）」

→「すると彼には、その小さな家がヴェージャヤンタ宮殿のように立派に思えるのである。」

7. Athassa khaṭopikaṃ oloketvā –

* atha = 「また」

* oloketvā = 「見て（次のように考える）」

8. ‘‘mayi gate imaṃ visaṅkharitvā uddhanālātaṃ karissanti,

* mayi gate = 「私が去った（出家した）なら」
* imaṃ visaṅkharitvā = 「これを解体して」
* uddhanālātaṃ = 「かまどの薪に」
* karissanti = 「してしまうだろう」

→「『私が去ったなら、これを解体してかまどの薪にしてしまうだろう。』」

9. puna aṭṭanipādavilīvādīni laddhabbāni honti, kinti karissāmī’’ti cinteti.

* puna = 「再び（戻ってきたときには）」
* aṭṭani-pāda-vilīvādīni = 「寝台の枠、脚、竹ひごなどを」
* laddhabbāni honti = 「手に入れなければならない」
* kinti karissāmī ti cinteti = 「どうすればいいだろうか、と考える」

→「『再び戻ってきたときには、寝台の枠、脚、竹ひごなどを手に入れなければならない。どうすればいいだろうか』と考える。」

10. Athassa sā sirisayanaṃ viya upaṭṭhāti.

* atha assa = 「すると彼にとって」
* sā = 「それ（寝台）が」
* sirisayanaṃ viya = 「豪華な寝台（吉祥なる寝台）のように」
* upaṭṭhāti = 「現れる」

→「すると彼には、それが豪華な寝台のように思えるのである。」

11. Tato dhaññakumbhiṃ oloketvā –

* dhaññakumbhiṃ = 「穀物の壺を」
* oloketvā = 「見て（次のように考える）」

12. ‘‘mayi gate ayaṃ gharaṇī imaṃ dhaññaṃ tena tena saddhiṃ bhuñjissati.

* mayi gate = 「私が去ったなら」
* ayaṃ gharaṇī = 「この主婦（妻）は」
* imaṃ dhaññaṃ = 「この穀物を」
* tena tena saddhiṃ = 「あちこちの男と（適当な誰かと）一緒に」
* bhuñjissati = 「食べてしまうだろう」

→「『私が去ったなら、この妻はこの穀物をあちこちの男と一緒に食べてしまうだろう。』」

13. Puna āgatena jīvitavutti nāma laddhabbā hoti, kinti karissāmī’’ti cinteti.

* puna āgatena = 「再び戻ってきた者（私）によって」
* jīvitavutti nāma = 「生活の糧が」
* laddhabbā hoti = 「手に入れられなければならない」
* kinti karissāmī ti cinteti = 「どうすればいいだろうか、と考える」

→「『再び戻ってきたときには、生活の糧を手に入れなければならない。どうすればいいだろうか』と考える。」

14. Athassa sā aḍḍhateḷasāni koṭṭhāgārasatāni viya upaṭṭhāti.

* atha assa = 「すると彼にとって」

15. Tato mātugāmaṃ oloketvā –

16. ‘‘mayi gate imaṃ hatthigopako vā assagopako vā yo koci palobhessati,

17. puna āgatena bhattapācikā nāma laddhabbā hoti, kinti karissāmī’’ti cinteti.

18. Athassa sā rūpinī devī viya upaṭṭhāti.

19. Idaṃ sandhāya ‘‘so na sakkuṇeyyā’’tiādi vuttaṃ.

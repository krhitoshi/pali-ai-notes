## 185

185.Evaṃme sutanti tevijjavacchasuttaṃ. Tattha ekapuṇḍarīketi puṇḍarīko vuccati setambarukkho, so tasmiṃ ārāme eko puṇḍarīko atthīti ekapuṇḍarīko. Etadahosīti tattha pavisitukāmatāya ahosi. Cirassaṃ kho, bhanteti pakatiyā āgatapubbataṃ upādāya. Dhammassa cānudhammanti idha sabbaññutaññāṇaṃ dhammo nāma, mahājanassa byākaraṇaṃ anudhammo nāma. Sesaṃ jīvakasutte (ma. ni. 2.51 ādayo) vuttanayameva. Na me teti ananuññāya ṭhatvā anuññampi paṭikkhipati. ‘‘Sabbaññū sabbadassāvī aparisesaṃ ñāṇadassanaṃ paṭijānātī’’ti hi idaṃ anujānitabbaṃ siyā, – ‘‘carato ca me…pe… paccupaṭṭhita’’nti idaṃ pana nānujānitabbaṃ. Sabbaññutaññāṇena hi āvajjitvā pajānāti. Tasmā ananuññāya ṭhatvā anuññampi paṭikkhipanto evamāha.

1. 185. Evaṃme sutanti tevijjavacchasuttaṃ.

* evaṃ me sutaṃ ti = 「このように私は聞いた」と（始まる経は）
* tevijjavacchasuttaṃ = 「三明ヴァッチャ経」である

→ 「185. 『このように私は聞いた』というのは、三明ヴァッチャ経（の註釈）である。」

2. Tattha ekapuṇḍarīketi puṇḍarīko vuccati setambarukkho, so tasmiṃ ārāme eko puṇḍarīko atthīti ekapuṇḍarīko.

* tattha = 「その（経文）において」
* ekapuṇḍarīketi = 「エーカプンダリーカとは」
* puṇḍarīko vuccati = 「プンダリーカは～と言われる」
* setamba-rukkho = 「白マンゴーの木（と）」
* so = 「それが」
* tasmiṃ ārāme = 「その園に」
* eko = 「一本の」
* puṇḍarīko = 「プンダリーカ（白マンゴー）が」
* atthīti (atthi iti) = 「あるからと」
* ekapuṇḍarīko = 「エーカプンダリーカ（一白マンゴー）と呼ばれる」

→ 「その経文における『エーカプンダリーカ』について、プンダリーカとは白マンゴーの木のことを言う。その園に一本の白マンゴーの木があることから『エーカプンダリーカ』と呼ばれる。」

3. Etadahosīti tattha pavisitukāmatāya ahosi.

* etad ahosi ti = 「『これ（次のような思い）が起こった』とは」
* tattha = 「そこに（その場所へ）」
* pavisitu-kāmatāya = 「入りたかったから」
* ahosi = 「生じた（のである）」

→ 「『次のような思いが起こった』とは、そこに（エーカプンダリーカの園へ）入りたかったために生じたのである。」

4. Cirassaṃ kho, bhanteti pakatiyā āgatapubbataṃ upādāya.

* cirassaṃ kho, bhante ti = 「『尊者よ、実に久しく』とは」
* pakatiyā = 「普段は（以前には）」
* āgata-pubbataṃ = 「来たことがあったこと」
* upādāya = 「に基づいて（言っている）」

→ 「『尊者よ、実に久しく』とは、以前には（世尊がここに）来たことがあったことに基づいて言っている。」

5. Dhammassa cānudhammanti idha sabbaññutaññāṇaṃ dhammo nāma, mahājanassa byākaraṇaṃ anudhammo nāma.

* dhammassa ca anudhammaṃ ti = 「『法に随順する法を』とは」
* idha = 「ここでは」
* sabbaññuta-ñāṇaṃ = 「一切智の智が」
* dhammo nāma = 「法という」
* mahājanassa = 「多くの人々の（語る）」
* byākaraṇaṃ = 「記別（解説、語ること）が」
* anudhammo nāma = 「随順法という」

→ 「『法に随順する法を』とは、ここでは一切智の智が法であり、多くの人々が語ることが随順法である。」

6. Sesaṃ jīvakasutte (ma. ni. 2.51 ādayo) vuttanayameva.

* sesaṃ = 「残りの部分は」
* jīvakasutte = 「ジーヴァカ経（マッジマ・ニカーヤなど）において」
* vutta-nayam eva = 「説かれた道理（方法）の通りである」

→ 「残りの部分はジーヴァカ経で説かれた道理の通りである。」

7. Na me teti ananuññāya ṭhatvā anuññampi paṭikkhipati.

* na me te ti = 「『彼らは私の（言った通りに語る者たち）ではない』とは」
* ananuññāya = 「認容されない（認められない）部分に」
* ṭhatvā = 「立脚して」
* anuññaṃ pi = 「認容される（認められる）部分をも」
* paṭikkhipati = 「拒絶している（否定している）」

→ 「『彼らは私の～ではない』とは、認容されない部分に立脚して、認容される部分をも拒絶しているのである。」

8. ‘‘Sabbaññū sabbadassāvī aparisesaṃ ñāṇadassanaṃ paṭijānātī’’ti hi idaṃ anujānitabbaṃ siyā, –

* Sabbaññū sabbadassāvī aparisesaṃ ñāṇadassanaṃ paṭijānātī ti = 「『一切智者であり一切見者であって、余すところのない智見を自称している』とは」
* hi = 「なぜなら」
* idaṃ = 「これ（この部分は）」
* anujānitabbaṃ = 「認容されるべき（認められるべき）もので」
* siyā = 「あるかもしれないが」

→ 「なぜなら、『一切智者であり一切見者であって、余すところのない智見を自称している』という、この部分は認容されるべきかもしれないが、」

9. ‘‘carato ca me…pe… paccupaṭṭhita’’nti idaṃ pana nānujānitabbaṃ.

* carato ca me...pe... paccupaṭṭhitaṃ ti = 「『歩いているときも...（中略）...現前している』とは」
* idaṃ pana = 「しかしこの部分は」
* na anujānitabbaṃ = 「認容されるべきではない」

→ 「『歩いているときも...（中略）...現前している』という、この部分は認容されるべきではない。」

10. Sabbaññutaññāṇena hi āvajjitvā pajānāti.

* sabbaññuta-ñāṇena = 「一切智の智によって」
* hi = 「なぜなら」
* āvajjitvā = 「転向して（心を向けて）」
* pajānāti = 「知る（からである）」

→ 「なぜなら、（世尊は）一切智の智によって、心を向けた（対象について）知るからである。」

11. Tasmā ananuññāya ṭhatvā anuññampi paṭikkhipanto evamāha.

* tasmā = 「それゆえに」
* ananuññāya ṭhatvā = 「認容されない部分に立脚して」
* anuññaṃ pi paṭikkhipanto = 「認容される部分をも拒否しながら」
* evam āha = 「このように言った（のである）」

→ 「それゆえに、認容されない部分（常に智見が現前しているという主張）に立脚して、認容される部分（一切智者であるという事実）をも（一括して）拒絶しながら、このように言われたのである。」

## 186 (1)

186.Āsavānaṃ khayāti ettha sakiṃ khīṇānaṃ āsavānaṃ puna khepetabbābhāvā yāvadevāti na vuttaṃ. Pubbenivāsañāṇena cettha bhagavā atītajānanaguṇaṃ dasseti, dibbacakkhuñāṇena paccuppannajānanaguṇaṃ, āsavakkhayañāṇena lokuttaraguṇanti. Iti imāhi tīhi vijjāhi sakalabuddhaguṇe saṃkhipitvā kathesi.

Gihisaṃyojananti gihibandhanaṃ gihiparikkhāresu nikantiṃ. Natthi kho vacchāti gihisaṃyojanaṃ appahāya dukkhassantakaro nāma natthi. Yepi hi santatimahāmatto uggaseno seṭṭhiputto vītasokadārakoti gihiliṅge ṭhitāva arahattaṃ pattā, tepi maggena sabbasaṅkhāresu nikantiṃ sukkhāpetvā pattā. Taṃ patvā pana na tena liṅgena aṭṭhaṃsu, gihiliṅgaṃ nāmetaṃ hīnaṃ, uttamaguṇaṃ dhāretuṃ na sakkoti. Tasmā tattha ṭhito arahattaṃ patvā taṃdivasameva pabbajati vā parinibbāti vā. Bhūmadevatā pana tiṭṭhanti. Kasmā? Nilīyanokāsassa atthitāya. Sesakāmabhave manussesu sotāpannādayo tayo tiṭṭhanti, kāmāvacaradevesu sotāpannā sakadāgāmino ca, anāgāmikhīṇāsavā panettha na tiṭṭhanti. Kasmā? Tañhi ṭhānaṃ laḷitajanassa āvāso, natthi tattha tesaṃ pavivekārahaṃ paṭicchannaṭṭhānañca. Iti tattha khīṇāsavo parinibbāti, anāgāmī cavitvā suddhāvāse nibbattati. Kāmāvacaradevato upari pana cattāropi ariyā tiṭṭhanti.

1. 186. Āsavānaṃ khayāti ettha sakiṃ khīṇānaṃ āsavānaṃ puna khepetabbābhāvā yāvadevāti na vuttaṃ.

* 186 = （経典の段落番号に対応）
* āsavānaṃ khayāti = 「『煩悩が尽きたことによって』とは」
* ettha = 「ここで（この箇所において）」
* sakiṃ = 「一度」
* khīṇānaṃ = 「尽きた」
* āsavānaṃ = 「煩悩を」
* puna = 「再び」
* khepetabbābhāvā = （khepetabba + abhāvā）「滅尽させるべき状態がないため」
* yāvadevāti = 「『望むがままに』とは」
* na vuttaṃ = 「語られていない」

→ 「186. ここでの『煩悩が尽きたことによって』の箇所については、一度尽きた煩悩を再び滅ぼすべき状態が存在しないため、『望むがままに（yāvadeva）』とは語られていないのである。」

2. Pubbenivāsañāṇena cettha bhagavā atītajānanaguṇaṃ dasseti, dibbacakkhuñāṇena paccuppannajānanaguṇaṃ, āsavakkhayañāṇena lokuttaraguṇanti.

* pubbenivāsañāṇena = 「宿住智（前世の知識）によって」
* cettha = （ca + ettha）「そしてここで」
* bhagavā = 「世尊は」
* atītajānanaguṇaṃ = 「過去を知る徳を」
* dasseti = 「示す」
* dibbacakkhuñāṇena = 「天眼智によって」
* paccuppannajānanaguṇaṃ = 「現在を知る徳を」
* āsavakkhayañāṇena = 「漏尽智（煩悩を滅ぼす智慧）によって」
* lokuttaraguṇanti = 「出世間の徳を（示す）と」

→ 「そしてここで世尊は、宿住智によって過去を知る徳を示し、天眼智によって現在を知る徳を、漏尽智によって出世間の徳を示しているのである。」

4. Gihisaṃyojananti gihibandhanaṃ gihiparikkhāresu nikantiṃ.

* gihisaṃyojananti = 「『在家の結縛』とは」

5. Natthi kho vacchāti gihisaṃyojanaṃ appahāya dukkhassantakaro nāma natthi.

* natthi kho vacchāti = 「『ヴァッチャよ、いない』とは」

* appahāya = 「断ち切らずに」
* dukkhassantakaro nāma = 「苦の終滅をなす者というものは」

→ 「『ヴァッチャよ、いない』とは、在家の結縛を断ち切らずに苦の終滅をなす者などいない、という意味である。」

→ 「というのも、サンタティ大臣、ウッガセーナ長者の息子、ヴィータソーカ童子などのように、在家の姿のままとどまって阿羅漢果に達した者たちであっても、」

7. tepi maggena sabbasaṅkhāresu nikantiṃ sukkhāpetvā pattā.

8. Taṃ patvā pana na tena liṅgena aṭṭhaṃsu, gihiliṅgaṃ nāmetaṃ hīnaṃ, uttamaguṇaṃ dhāretuṃ na sakkoti.

* gihiliṅgaṃ = 「在家の姿」
* nāmetaṃ = （nāma + etaṃ）「という、それは」
* hīnaṃ = 「劣っている（ため）」
* uttamaguṇaṃ = 「最高の徳を」
* dhāretuṃ = 「保つ（保持する）ことは」
* na sakkoti = 「できない（からである）」

→ 「しかし、それに達してその姿（在家の姿）のままとどまることはなかった。在家の姿というものは劣っているため、最高の徳を保ち続けることができないからである。」

9. Tasmā tattha ṭhito arahattaṃ patvā taṃdivasameva pabbajati vā parinibbāti vā.

* tasmā = 「それゆえに」
* tattha = 「そこに（在家の状態に）」

10. Bhūmadevatā pana tiṭṭhanti. Kasmā? Nilīyanokāsassa atthitāya.

11. Sesakāmabhave manussesu sotāpannādayo tayo tiṭṭhanti, kāmāvacaradevesu sotāpannā sakadāgāmino ca, anāgāmikhīṇāsavā panettha na tiṭṭhanti.

## 186 (2)

Sopāsi kammavādīti sopi kammavādī ahosi, kiriyampi na paṭibāhittha. Tañhi ekanavutikappamatthake attānaṃyeva gahetvā katheti. Tadā kira mahāsatto pāsaṇḍapariggaṇhanatthaṃ pabbajito tassapi pāsaṇḍassa nipphalabhāvaṃ jānitvā vīriyaṃ na hāpesi, kiriyavādī hutvā sagge nibbattati. Tasmā evamāha. Sesaṃ sabbattha uttānamevāti.

* paṭibāhittha = 「否定しなかった（拒絶しなかった）」

→ 「『彼もまた業を説く者であった』とは、彼もまた業を説く者であり、行為（の果報）をも否定しなかったということである。」

2. Tañhi ekanavutikappamatthake attānaṃyeva gahetvā katheti.

* tañhi = （taṃ + hi）「というのも、その者を（そのことについて）」
* ekanavutikappamatthake = 「九十一劫の昔の」
* attānaṃyeva = （attānaṃ + yeva）「まさに自分自身を」
* gahetvā = 「取り上げて（指して）」
* katheti = 「語っているのである」

→ 「というのも、世尊は九十一劫の昔のその者について、まさに自分自身を指して語っているのである。」

3. Tadā kira mahāsatto pāsaṇḍapariggaṇhanatthaṃ pabbajito tassapi pāsaṇḍassa nipphalabhāvaṃ jānitvā vīriyaṃ na hāpesi,

* tadā = 「その時」
* kira = 「実に（伝え聞くところによれば）」
* mahāsatto = 「偉大なる衆生（菩薩）は」
* pāsaṇḍapariggaṇhanatthaṃ = 「外道の教えを調査（把握）するために」
* pabbajito = 「出家し」
* tassapi = （tassa + pi）「その」
* pāsaṇḍassa = 「外道の（教えの）」
* nipphalabhāvaṃ = 「無益な性質を（結果がないことを）」
* jānitvā = 「知って」
* vīriyaṃ = 「精進を」
* na hāpesi = 「怠らなかった（減らさなかった）」

→ 「実にその時、菩薩は外道の教えを調査するために（その教団に）出家し、その外道の教えが無益であることを知っても、精進を怠ることはなく、」

4. kiriyavādī hutvā sagge nibbattati.

* kiriyavādī = 「行為（の果報）を説く者」
* hutvā = 「となって」
* sagge = 「天に」
* nibbattati = 「生まれた（歴史的現在）」

→ 「行為（の果報）を説く者となって、天に生まれたのである。」

5. Tasmā evamāha.

* tasmā = 「それゆえに」
* evamāha = （evaṃ + āha）「このように語ったのである」

→ 「それゆえに、（世尊は）このように語ったのである。」

6. Sesaṃ sabbattha uttānamevāti.

* sesaṃ = 「残りは」
* sabbattha = 「すべての箇所において」
* uttānamevāti = （uttānaṃ + eva + ti）「まさに明らかである」

→ 「残りの部分は、すべて明らかである。」

7. Papañcasūdaniyā majjhimanikāyaṭṭhakathāya

* papañcasūdaniyā = 「『パパンチャスーダニー（戯論粉砕）』という」
* majjhimanikāyaṭṭhakathāya = 「中部註釈書における」

→ 「『パパンチャスーダニー』という中部の註釈書における、」

8. Tevijjavacchasuttavaṇṇanā niṭṭhitā.

* tevijjavacchasuttavaṇṇanā = 「三明ヴァッチャ経の解説が」
* niṭṭhitā = 「終了した」

→ 「三明ヴァッチャ経の解説が終了した。」

# 3. Mahāvacchasuttavaṇṇanā

## 193 (1)

193.Evaṃme sutanti mahāvacchasuttaṃ. Tattha sahakathīti saddhiṃvādo, bahuṃ mayā tumhehi saddhiṃ kathitapubbanti kathaṃ sāreti mettiṃ ghaṭeti. Purimāni hi dve suttāni etasseva kathitāni, saṃyuttake abyākatasaṃyuttaṃ (saṃ. ni. 4.416 ādayo) nāma etasseva kathitaṃ – ‘‘kiṃ nu kho, bho gotama, sassato loko idameva saccaṃ moghamaññanti abyākatameta’’nti evaṃ ekuttaranikāyepi iminā saddhiṃ kathitaṃ atthiyeva. Tasmā evamāha. Sammāsambuddhopi tassa āgatāgatassa saṅgahaṃ katvā okāsamakāsiyeva. Kasmā? Ayañhi sassatadiṭṭhiko, sassatadiṭṭhikā ca sīghaṃ laddhiṃ na vissajjenti, vasātelamakkhitapilotikā viya cirena sujjhanti. Passati ca bhagavā – ‘‘ayaṃ paribbājako kāle gacchante gacchante laddhiṃ vissajjetvā mama santike pabbajitvā cha abhiññāyo sacchikatvā abhiññātasāvako bhavissatī’’ti. Tasmā tassa āgatāgatassa saṅgahaṃ katvā okāsamakāsiyeva. Idaṃ panassa pacchimagamanaṃ.

4. Etamatthaṃ pucchissāmīti pucchati.

* etamatthaṃ = この事を（etaṃ atthaṃ）
* pucchissāmī ti = 「私は尋ねよう」と思って（pucchissāmi + iti）
* pucchati = （彼は）尋ねているのである。

→ 「この事を尋ねよう」と思って、彼は尋ねているのである。

5. Tattha tiṭṭhatūti bhavaṃ tāva gotamo tiṭṭhatu, bhavañhi loke pākaṭo arahāti attho.

* tattha = そこで（その文のなかで）
* tiṭṭhatū ti = 「tiṭṭhatu（さておき）」とは
* bhavaṃ tāva gotamo = 尊きゴータマはまず
* tiṭṭhatu = 置いておこう
* bhavañ hi = 実に尊者は（bhavaṃ + hi）
* loke = 世間において
* pākaṭo = 周知の、明らかな
* arahā ti = 阿羅漢であるから、と
* attho = という意味である。

→ そこで「tiṭṭhatu（さておき）」とは、「尊きゴータマのことはまず置いておこう。実に尊者は世間において周知の阿羅漢であるからだ」という意味である。

6. Tasmiṃ byākate uttari bhikkhunīādīnaṃ vasena pañhaṃ pucchi, bhagavāpissa byākāsi.

* tasmiṃ byākate = それ（最初の問い）が記別された（答えられた）とき
* uttari = さらに
* bhikkhunī-ādīnaṃ = 比丘尼などの
* vasena = 観点から、～に関して
* pañhaṃ = 質問を
* pucchi = 尋ねた
* bhagavā pi = 世尊もまた
* assa = 彼に対して
* byākāsi = 答えた（記別した）。

→ それが答えられたとき、さらに比丘尼などに関しても質問を尋ね、世尊もまた彼に答えたのである。

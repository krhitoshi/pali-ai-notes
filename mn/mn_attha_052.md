3. Tattha beluvagāmaketi vesāliyā dakkhiṇapasse avidūre beluvagāmako nāma atthi, taṃ gocaragāmaṃ katvāti attho.

* tattha = 「そこ（その経文の中）で」

* beluvagāmake ti = 「『ベールヴァ村において』とは」

* vesāliyā dakkhiṇapasse = 「ヴェーサーリーの南側に」

* avidūre = 「遠くないところに」

* beluvagāmako nāma atthi = 「ベールヴァ村という（村）がある」

* taṃ gocaragāmaṃ katvā = 「それを托鉢の村として」

  * gocara-gāma = 「托鉢して食を得る村（行乞の場とする村）」

* ti attho = 「という意味である」

→「そこで『ベールヴァ村において』とは、ヴェーサーリーの南側の遠くないところにベールヴァ村という村があり、それを托鉢の村として、という意味である。」

* jāti-gotta-vasena c'eva = 「生まれと氏姓の点でも」

* sārappatta-kula-gaṇanāya ca = 「富裕に達した家々の数え上げにおいても」

  * sārappatta = 「実質（財の中核・富裕）に達した」（sāra-ppatta）

  * kula-gaṇanā = 「家々の数え上げ、家格の勘定」

* dasame ṭhāne gaṇīyati = 「第十の位置に数えられる」

* tena assa = 「それゆえ彼に」

* dasamo tv eva nāmaṃ jātaṃ = 「まさに『ダサマ（第十）』という名が生まれた」

→「『ダサマ』とは、実に彼は生まれと氏姓の点でも、富裕に達した家々の数え上げにおいても第十の位置に数えられ、それゆえ彼にまさに『ダサマ（第十）』という名が生まれたのである。」

5. Aṭṭhakanāgaroti aṭṭhakanagaravāsī.

  * vāsī = 「住む者、住人」

→「『アッタカナーガラ』とは、アッタカ都市の住人である。」

6. Kukkuṭārāmoti kukkuṭaseṭṭhinā kārito ārāmo.

→「『クックタ園』とは、クックタ長者によって造られた園である。」

## 19

19. Abhisaṅkhatanti kataṃ uppāditaṃ. Abhisañcetayitanti cetayitaṃ pakappitaṃ. So tattha ṭhitoti so tasmiṃ samathavipassanādhamme ṭhito. Dhammarāgena dhammanandiyāti padadvayehi samathavipassanāsu chandarāgo vutto. Samathavipassanāsu hi sabbena sabbaṃ chandarāgaṃ pariyādiyituṃ sakkonto arahā hoti, asakkonto anāgāmī hoti. So samathavipassanāsu chandarāgassa appahīnattā catutthajjhānacetanāya suddhāvāse nibbattati, ayaṃ ācariyānaṃ samānakathā.

Vitaṇḍavādī panāha ‘‘teneva dhammarāgenāti vacanato akusalena suddhāvāse nibbattatī’’ti so ‘‘suttaṃ āharā’’ti vattabbo, addhā aññaṃ apassanto idameva āharissati, tato vattabbo ‘‘kiṃ panidaṃ suttaṃ neyyatthaṃ nītattha’’nti, addhā nītatthanti vakkhati. Tato vattabbo – evaṃ sante anāgāmiphalatthikena samathavipassanāsu chandarāgo kattabbo bhavissati, chandarāge uppādite anāgāmiphalaṃ paṭividdhaṃ bhavissati ‘‘mā suttaṃ me laddha’’nti yaṃ vā taṃ vā dīpehi. Pañhaṃ kathentena hi ācariyassa santike uggahetvā attharasaṃ paṭivijjhitvā kathetuṃ vaṭṭati, akusalena hi sagge, kusalena vā apāye paṭisandhi nāma natthi. Vuttañhetaṃ bhagavatā –

‘‘Na, bhikkhave, lobhajena kammena dosajena kammena mohajena kammena devā paññāyanti, manussā paññāyanti, yā vā panaññāpi kāci sugatiyo, atha kho, bhikkhave, lobhajena kammena dosajena kammena mohajena kammena nirayo paññāyati, tiracchānayoni paññāyati, pettivisayo paññāyati, yā vā panaññāpi kāci duggatiyo’’ti –

Evaṃ paññāpetabbo. Sace sañjānāti sañjānātu, no ce sañjānāti, ‘‘gaccha pātova vihāraṃ pavisitvā yāguṃ pivāhī’’ti uyyojetabbo.

Yathā ca pana imasmiṃ sutte, evaṃ mahāmālukyovādepi mahāsatipaṭṭhānepi kāyagatāsatisuttepi samathavipassanā kathitā. Tattha imasmiṃ sutte samathavasena gacchatopi vipassanāvasena gacchatopi samathadhurameva dhuraṃ, mahāmālukyovāde vipassanāva dhuraṃ, mahāsatipaṭṭhānaṃ pana vipassanuttaraṃ nāma kathitaṃ, kāyagatāsatisuttaṃ samathuttaranti.

Ayaṃ kho gahapati…pe… ekadhammo akkhātoti ekadhammaṃ pucchitena ayampi ekadhammoti evaṃ pucchāvasena kathitattā ekādasapi dhammā ekadhammo nāma jāto. Mahāsakuludāyisuttasmiñhi ekūnavīsati pabbāni paṭipadāvasena ekadhammo nāma jātāni, idha ekādasapucchāvasena ekadhammoti āgatāni. Amatuppattiyatthena vā sabbānipi ekadhammoti vattuṃ vaṭṭati.

### 対訳

1. 19. Abhisaṅkhatanti kataṃ uppāditaṃ.

* abhisaṅkhataṃ = 「形成されたもの」（abhisaṅkharoti の過去分詞）

* kataṃ = 「作られたもの」

* uppāditaṃ = 「生じさせられたもの」

→「『abhisaṅkhataṃ（形成された）』とは、作られ、生じさせられた（もの）である。」

2. Abhisañcetayitanti cetayitaṃ pakappitaṃ.

* abhisañcetayitaṃ = 「思念されたもの」（abhisañcetayati の過去分詞）

* cetayitaṃ = 「意図されたもの」

* pakappitaṃ = 「計画されたもの、企図されたもの」

→「『abhisañcetayitaṃ（思念された）』とは、意図され、計画された（もの）である。」

3. So tattha ṭhitoti so tasmiṃ samathavipassanādhamme ṭhito.

* so = 「彼は」

* tattha ṭhito = 「そこに立った者」

* tasmiṃ samatha-vipassanā-dhamme = 「その止と観の法において」

* ṭhito = 「立った、とどまった者」

→「『彼はそこに立って』とは、彼はその止と観の法にとどまって、という意味である。」

4. Dhammarāgena dhammanandiyāti padadvayehi samathavipassanāsu chandarāgo vutto.

* dhamma-rāgena = 「法への貪によって」

* dhamma-nandiyā = 「法への喜びによって」

* pada-dvayehi = 「（この）二つの句によって」

* samatha-vipassanāsu = 「止と観に対する」

* chanda-rāgo = 「欲貪」

* vutto = 「説かれた」

→「『法への貪によって、法への喜びによって』という二つの句によって、止と観に対する欲貪が説かれている。」

5. Samathavipassanāsu hi sabbena sabbaṃ chandarāgaṃ pariyādiyituṃ sakkonto arahā hoti, asakkonto anāgāmī hoti.

* samatha-vipassanāsu = 「止と観に対する」

* hi = 「実に」

* sabbena sabbaṃ = 「完全に、すっかり」

* chandarāgaṃ pariyādiyituṃ = 「欲貪を滅ぼし尽くすことを」

* sakkonto = 「できる者は」

* arahā hoti = 「阿羅漢となる」

* asakkonto = 「できない者は」

* anāgāmī hoti = 「不還者となる」

→「実に、止と観に対する欲貪を完全に滅ぼし尽くすことができる者は阿羅漢となり、できない者は不還者となる。」

6. So samathavipassanāsu chandarāgassa appahīnattā catutthajjhānacetanāya suddhāvāse nibbattati, ayaṃ ācariyānaṃ samānakathā.

* so = 「彼は」

* chandarāgassa appahīnattā = 「欲貪が捨てられていないので」

* catutthajjhāna-cetanāya = 「第四禅の思によって」

* suddhāvāse nibbattati = 「浄居天に生まれる」

* ahaṃ pana = 「しかし私は」

* evaṃvidhe niyyānikasāsane = 「このような出離に導く教えにおいて」

* ekādasavidhaṃ amatuppattipaṭipadaṃ = 「十一種の、不死に到達する行道を」

  * amata-uppatti-paṭipadā = 「不死（涅槃）への到達の道」

* desentassa ācariyassa = 「説いてくださる師のために」

* pūjaṃ kiṃ na karissāmi = 「どうして供養をしないだろうか」

* karissāmi yevā ti vadati = 「『必ずしよう』と言うのである」

→「『まして私は』とは──外道の者たちでさえ、出離に導かない教えにおいてすら、単なる技芸を与えるだけの者のために財を探し求める。まして私は、このような出離に導く教えにおいて、十一種の不死に到達する行道を説いてくださる師のために、どうして供養をしないだろうか、必ずしよう、と言うのである。」

12. Paccekadussayugena acchādesīti ekamekassa bhikkhuno ekekaṃ dussayugamadāsīti attho.

* paccekadussayugena acchādesī ti = 「『それぞれ一組の布で覆った』とは」

  * pacceka-dussa-yuga = 「各自への一組の布」

  * acchādesi = 「覆った、着せた」（acchādeti のアオリスト）

* ekamekassa bhikkhuno = 「一人一人の比丘に」

* ekekaṃ dussayugam adāsi = 「一組ずつの布を与えた」

* ti attho = 「という意味である」

→「『それぞれ一組の布で覆った』とは、一人一人の比丘に一組ずつの布を与えた、という意味である。」

13. Samudācāravacanaṃ panettha evarūpaṃ hoti, tasmā acchādesīti vuttaṃ.

* samudācāravacanaṃ = 「慣用の言い方」（samudācāra-vacana）

* pana ettha = 「しかしここでは」

* evarūpaṃ hoti = 「このようなものである」

* tasmā = 「それゆえ」

* acchādesī ti vuttaṃ = 「『覆った』と言われたのである」

→「しかし（布の布施については）このような慣用の言い方があるので、それゆえ『覆った』と言われたのである。」

14. Pañcasatavihāranti pañcasatagghanikaṃ paṇṇasālaṃ kāresīti attho.

* pañcasatavihāran ti = 「『五百の精舎を』とは」（pañca-sata-vihāra）

* pañcasatagghanikaṃ = 「五百（金）の価値のある」

  * agghanika = 「～の価値がある」

* paṇṇasālaṃ = 「葉葺きの小屋を」（paṇṇa-sālā）

* kāresī ti attho = 「作らせた、という意味である」（kāreti のアオリスト・使役）

→「『五百の精舎を』とは、五百の価値のある葉葺きの小屋を作らせた、という意味である。」

15. Sesaṃ sabbattha uttānamevāti.

* sesaṃ = 「残り（の語句）は」

* sabbattha = 「すべての箇所で」

* uttānam evā ti = 「明白である、と」

→「残りの語句は、すべての箇所で明白である。」

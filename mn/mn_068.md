* pe = 「省略（peyyāla）」
* evaṃ-pañño = 「このような慧（を持つ者）」
* evaṃ-vihārī = 「このような住（暮らし方をする者）」
* evaṃ-vimutto = 「このような解脱（を得た者）」
→ 「『その具寿はこのように戒を持つ者であった』とも、『その具寿はこのように法を持つ者であった』とも、…（中略）…『慧』…『住』…『その具寿はこのように解脱した者であった』とも。」

6. So tassa saddhañca sīlañca sutañca cāgañca paññañca anussaranto tadatthāya cittaṃ upasaṃharati.
* so = 「彼（聞いた比丘）は」
* tassa = 「その（亡くなった比丘）の」
* saddhañca = saddhaṃ（信）+ ca（と）
* sīlañca = sīlaṃ（戒）+ ca
* sutañca = sutaṃ（多聞）+ ca
* cāgañca = cāgaṃ（施捨）+ ca
* paññañca = paññaṃ（慧）+ ca
* anussaranto = 「随念しながら、思い起こしながら」
* tadatthāya = 「その（同じ）状態・目的のために」
* cittaṃ upasaṃharati = 「心を向ける」
→ 「彼は、その（亡くなった比丘の）信・戒・多聞・施捨・慧を思い起こしながら、自らも同じ境地に至るべく心を向ける。」

7. Evampi kho, anuruddhā, bhikkhuno phāsuvihāro hoti.
* evaṃ pi = 「このようにして（も）」
* kho = 「実に」
* anuruddhā = 「アヌルッダたちよ」
* bhikkhuno = 「比丘にとっての」
* phāsu-vihāro = 「安楽な住（安らかな暮らし）」
* hoti = 「となる、である」
→ 「アヌルッダたちよ、このようにしても実に、比丘にとって安楽な住となるのである。」

8. ‘‘Idhānuruddhā, bhikkhu suṇāti –
* idha + anuruddhā = 「アヌルッダたちよ、ここに」
* bhikkhu suṇāti = 「比丘が（次のように）聞く（とする）」
→ 「アヌルッダたちよ、ここに比丘が（次のように）聞くとしよう。」

9. ‘itthannāmo bhikkhu kālaṅkato;
* itthan-nāmo bhikkhu = 「これこれの名前の比丘が」
* kālaṅkato = 「死んだ（命終した）」
→ 「『これこれの名前の比丘が命終した。』」

10. so bhagavatā byākato – tiṇṇaṃ saṃyojanānaṃ parikkhayā sotāpanno avinipātadhammo niyato sambodhiparāyaṇo’ti.
* so bhagavatā byākato = 「彼は世尊によって（次のように）記別された」
* tiṇṇaṃ saṃyojanānaṃ = 「三つの結（煩悩）の」
* parikkhayā = 「完全なる滅尽により」
* sotāpanno = 「預流果（須陀洹）の者（となり）」
* a-vinipāta-dhammo = 「悪趣に堕ちない性質の者（となり）」
* niyato = 「（正しさに）定まった者（となり）」
* sambodhi-parāyaṇo = 「正覚を最終の目的（帰趣）とする者である」
* ti = 「と」
→ 「『彼は世尊によって、「三結を完全に滅尽したことにより預流（須陀洹）となり、悪趣に堕ちない性質の者で、正しさに定まり、正覚を最終の目的とする者である」と記別された』と。」

11. So kho panassa āyasmā sāmaṃ diṭṭho vā hoti anussavassuto vā –
* so kho pana assa āyasmā = 「さらに、その具寿は彼にとって」
* sāmaṃ diṭṭho vā hoti = 「自ら直接見知られていたか」
* anussava-ssuto vā = 「あるいは伝聞によって聞いて知られていたとする」
→ 「さらに、その具寿（亡くなった比丘）は、彼自身によって直接見知られていたか、あるいは伝聞によって聞いて知られていたとする。」

12. ‘evaṃsīlo so āyasmā ahosi itipi, evaṃdhammo…pe… evaṃpañño… evaṃvihārī… evaṃvimutto so āyasmā ahosi itipī’ti.
* pe = 「省略（peyyāla）」
→ 「『その具寿はこのように戒を持つ者であった』とも、『その具寿はこのように法を持つ者であった』とも、…（中略）…『慧』…『住』…『その具寿はこのように解脱した者であった』とも。」

13. So tassa saddhañca sīlañca sutañca cāgañca paññañca anussaranto tadatthāya cittaṃ upasaṃharati.
* anussaranto = 「信・戒・多聞・施捨・慧を思い起こしながら」
* tadatthāya = 「同じ境地に至るべく」
* cittaṃ upasaṃharati = 「心を向ける」
→ 「彼は、その（亡くなった比丘の）信・戒・多聞・施捨・慧を思い起こしながら、自らも同じ境地に至るべく心を向ける。」

14. Evampi kho, anuruddhā, bhikkhuno phāsuvihāro hoti.
* evaṃ pi kho = 「このようにして実に」
* anuruddhā = 「アヌルッダたちよ」
* bhikkhuno phāsuvihāro hoti = 「比丘にとって安楽な住となる」
→ 「アヌルッダたちよ、このようにしても実に、比丘にとって安楽な住となるのである。」

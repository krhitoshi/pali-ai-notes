* avinipāta-dhammā = 「悪趣に堕ちない性質のものであり」
* niyatā = 「決定しており（正しさに定まっており）」
* sambodhi-parāyaṇā ti = 「正覚を究極の目的とすると。」

→ 「アヌルッダたちよ、ここに、ある女性信徒が『かくかくしかじかの名前の女性信徒が亡くなった。彼女は世尊によって、三つの結の完全な滅尽により、預流果となり、悪趣に堕ちない性質のものであり、決定しており、正覚を究極の目的とするものであると記別された』と聞きます。」

9. Sā kho panassā bhaginī sāmaṃ diṭṭhā vā hoti anussavassutā vā – ‘evaṃsīlā sā bhaginī ahosi itipi, evaṃdhammā sā bhaginī ahosi itipi, evaṃpaññā sā bhaginī ahosi itipi, evaṃvihārinī sā bhaginī ahosi itipi, evaṃvimuttā sā bhaginī ahosi itipī’ti.

* (単語の構成は2と同じ、省略されずに記述されている)

→ 「さらに、彼女にとってその姉妹は、自ら直接見知った者であったり、あるいは『その姉妹はこのような戒を持つ者であった、その姉妹はこのような法を持つ者であった、その姉妹はこのような智慧を持つ者であった、その姉妹はこのように生きる者であった、その姉妹はこのように解脱した者であった』と伝聞で聞いた者であったりします。」

10. Sā tassā saddhañca sīlañca sutañca cāgañca paññañca anussarantī tadatthāya cittaṃ upasaṃharati. Evampi kho, anuruddhā, upāsikāya phāsuvihāro hoti.

* (単語の構成は3, 4と同じ)

→ 「彼女は、亡くなった姉妹の信・戒・聞・施・慧を随念し、その同じ状態を目指して心を向けます。アヌルッダたちよ、実にこのようにして、女性信徒に安楽な住が生じるのです。」

11. ‘‘Iti kho, anuruddhā, tathāgato na janakuhanatthaṃ na janalapanatthaṃ na lābhasakkārasilokānisaṃsatthaṃ na ‘iti maṃ jano jānātū’ti sāvake abbhatīte kālaṅkate upapattīsu byākaroti – ‘asu amutra upapanno, asu amutra upapanno’ti.

* iti kho = 「実にこのように」
* anuruddhā = 「アヌルッダたちよ」
* tathāgato = 「如来は」
* na jana-kuhana-atthaṃ = 「人々を欺く（誑惑する）ためでなく」
* na jana-lapana-atthaṃ = 「人々に媚びへつらうためでなく」
* na lābha-sakkāra-siloka-ānisaṃsa-atthaṃ = 「利得、尊敬、名声の利益のためでなく」
* na 'iti maṃ jano jānātū' ti = 「『人々が私のことをこのように知るように』というためでもなく」
* sāvake = 「弟子たちが」
* abbhatīte = 「過ぎ去り（亡くなり）」
* kālaṅkate = 「命終した」
* upapattīsu = 「生まれ変わり（再生）について」
* byākaroti = 「記別する（明言する）」
* asu = 「あの人は」
* amutra = 「あそこに」
* upapanno = 「生まれ変わった」
* asu amutra upapanno ti = 「あの人はあそこに生まれ変わった、と。」

→ 「アヌルッダたちよ、実にこのように、如来は人々を欺くためではなく、人々に媚びへつらうためではなく、利得や尊敬や名声の利益のためではなく、『人々が私のことをこのように知るように』というためでもなく、亡くなり命終した弟子たちの生まれ変わりについて『あの人はあそこに生まれ変わった、あの人はあそこに生まれ変わった』と記別するのです。」

12. Santi ca kho, anuruddhā, kulaputtā saddhā uḷāravedā uḷārapāmojjā.

* santi = 「いる」
* ca kho = 「しかし」
* anuruddhā = 「アヌルッダたちよ」
* kulaputtā = 「良家の子らが」
* saddhā = 「信を持った」
* uḷāra-vedā = 「広大な（優れた）歓喜を持った」
* uḷāra-pāmojjā = 「広大な（優れた）喜びを持った」

→ 「しかし、アヌルッダたちよ、信を持ち、広大な歓喜と広大な喜びを持った良家の子らがいます。」

13. Te taṃ sutvā tadatthāya cittaṃ upasaṃharanti. Tesaṃ taṃ, anuruddhā, hoti dīgharattaṃ hitāya sukhāyā’’ti.

* te = 「彼らは」
* taṃ = 「それを」
* sutvā = 「聞いて」
* tadatthāya = 「その（同じ）状態を目指して」
* cittaṃ = 「心を」
* upasaṃharanti = 「向ける。」
* tesaṃ = 「彼らにとって」
* taṃ = 「それは」
* anuruddhā = 「アヌルッダたちよ」
* hoti = 「なる」
* dīgha-rattaṃ = 「長い間（長夜にわたり）」
* hitāya = 「利益と」
* sukhāyā ti = 「安楽のために、と。」

→ 「彼らはそれを聞いて、その状態を目指して心を向けます。アヌルッダたちよ、それは彼らにとって、長い間、利益と安楽のためになるのです。」

14. Idamavoca bhagavā. Attamano āyasmā anuruddho bhagavato bhāsitaṃ abhinandīti.

* idam = 「これを」
* avoca = 「語った」
* bhagavā = 「世尊は」
* attamano = 「心喜んだ」
* āyasmā = 「尊者」
* anuruddho = 「アヌルッダは」
* bhagavato = 「世尊の」
* bhāsitaṃ = 「語った言葉を」
* abhinandī ti = 「歓喜した。」

→ 「世尊はこれを語られた。心喜んだ尊者アヌルッダは、世尊の語った言葉に歓喜した。」

15. Naḷakapānasuttaṃ niṭṭhitaṃ aṭṭhamaṃ.

* Naḷakapāna-suttaṃ = 「ナラカパーナ経（葦飲経）が」
* niṭṭhitaṃ = 「終わった」
* aṭṭhamaṃ = 「8番目の」

→ 「第8のナラカパーナ経が終わった。」

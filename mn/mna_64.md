
# 4. Mahāmālukyasuttavaṇṇanā

## 129

129.Evaṃme sutanti mahāmālukyasuttaṃ. Tattha orambhāgiyānīti heṭṭhā koṭṭhāsikāni kāmabhave nibbattisaṃvattanikāni. Saṃyojanānīti bandhanāni. Kassakho nāmāti kassa devassa vā manussassa vā desitāni dhāresi, kiṃ tvameveko assosi, na añño kocīti? Anusetīti appahīnatāya anuseti. Anusayamāno saṃyojanaṃ nāma hoti.

Ettha ca bhagavatā saṃyojanaṃ pucchitaṃ, therenapi saṃyojanameva byākataṃ. Evaṃ santepi tassa vāde bhagavatā doso āropito. So kasmāti ce? Therassa tathāladdhikattā. Ayañhi tassa laddhi ‘‘samudācārakkhaṇeyeva kilesehi saṃyutto nāma hoti, itarasmiṃ khaṇe asaṃyutto’’ti. Tenassa bhagavatā doso āropito. Athāyasmā ānando cintesi – ‘‘bhagavatā bhikkhusaṅghassa dhammaṃ desessāmīti attano dhammatāyeva ayaṃ dhammadesanā āraddhā, sā iminā apaṇḍitena bhikkhunā visaṃvāditā. Handāhaṃ bhagavantaṃ yācitvā bhikkhūnaṃ dhammaṃ desessāmī’’ti. So evamakāsi. Taṃ dassetuṃ ‘‘evaṃ vutte āyasmā ānando’’tiādi vuttaṃ.

Tattha sakkāyadiṭṭhipariyuṭṭhitenāti sakkāyadiṭṭhiyā gahitena abhibhūtena. Sakkāyadiṭṭhiparetenāti sakkāyadiṭṭhiyā anugatena. Nissaraṇanti diṭṭhinissaraṇaṃ nāma nibbānaṃ, taṃ yathābhūtaṃ nappajānāti. Appaṭivinītāti avinoditā anīhaṭā. Orambhāgiyaṃ saṃyojananti heṭṭhābhāgiyasaṃyojanaṃ nāma hoti. Sesapadesupi eseva nayo. Sukkapakkho uttānatthoyeva. ‘‘Sānusayā pahīyatī’’ti vacanato panettha ekacce ‘‘aññaṃ saṃyojanaṃ añño anusayo’’ti vadanti. ‘‘Yathā hi sabyañjanaṃ bhatta’’nti vutte bhattato aññaṃ byañjanaṃ hoti, evaṃ ‘‘sānusayā’’ti vacanato pariyuṭṭhānasakkāyadiṭṭhito aññena anusayena bhavitabbanti tesaṃ laddhi. Te ‘‘sasīsaṃ pārupitvā’’tiādīhi paṭikkhipitabbā. Na hi sīsato añño puriso atthi. Athāpi siyā – ‘‘yadi tadeva saṃyojanaṃ so anusayo, evaṃ sante bhagavatā therassa taruṇūpamo upārambho duāropito hotī’’ti. Na duāropito, kasmā? Evaṃladdhikattāti vitthāritametaṃ. Tasmā soyeva kileso bandhanaṭṭhena saṃyojanaṃ, appahīnaṭṭhena anusayoti imamatthaṃ sandhāya bhagavatā ‘‘sānusayā pahīyatī’’ti evaṃ vuttanti veditabbaṃ.

* rūpāvacarajjhānaṃ = 「色界の禅定に」
* samāpajjitvā = 「入って（到達して）」
* rūpaṃ = 「色を」
* atikkamitvā = 「超越して」
* arūpāvacarasamāpattiṃ = 「無色界の定に」
* samāpanno ti = 「達した、ということによって」
* samathavasena pi = 「止（サマタ）の観点からも」
* nena (anena) = 「彼によって」
* rūpaṃ = 「色は」
* atikkantaṃ = 「超越されている」

→ 「実にこの者は、下位である色界の禅定に入り、色を超越して無色界の定に到達したのであるから、止（サマタ）の観点からも彼によって色は超越されている。」

4. heṭṭhā rūpaṃ sammadeva sammasitvā taṃ atikkamma idāni arūpaṃ sammasatīti vipassanāvasenapinena rūpaṃ atikkantaṃ.

* heṭṭhā = 「下位で」
* rūpaṃ = 「色を」
* sammadeva = 「正しく」
* sammasitvā = 「随観（観察）して」
* taṃ = 「それを」
* atikkamma = 「超越して」
* idāni = 「今」
* arūpaṃ = 「無色を」
* sammasatī ti = 「随観している、ということによって」
* vipassanāvasena pi = 「観（ヴィパッサナー）の観点からも」
* nena = 「彼によって」
* rūpaṃ = 「色は」
* atikkantaṃ = 「超越されている」

→ 「下位で色を正しく観察し、それを超越して今は無色を観察しているのであるから、観（ヴィパッサナー）の観点からも彼によって色は超越されている。」

5. Arūpe pana sabbasopi rūpaṃ natthīti taṃ sandhāyapi idha rūpaṃ na gahitaṃ.

* arūpe pana = 「さらに、無色界においては」
* sabbasopi (sabbaso pi) = 「あらゆる点において」
* rūpaṃ natthi ti = 「色は存在しない、ということから」
* taṃ sandhāya pi = 「それを意図して（根拠として）も」
* idha = 「ここに」
* rūpaṃ = 「色は」
* na gahitaṃ = 「含まれていない」

→ 「さらに、無色界においては色は完全に存在しないことから、それを意図しても、ここには色が含まれていないのである。」

6. Atha kiñcarahīti kiṃ pucchāmīti pucchati?

* atha kiñcarahī ti = 「『それでは、いったい何か（atha kiñcarahi）』とは」
* kiṃ pucchāmī ti = 「『何を問おう』として」
* pucchati = 「（長老は）尋ねているのか？」

→ 「『それでは、いったい何か』とは、何を問おうとして尋ねているのか？」

7. Samathavasena gacchato cittekaggatā dhuraṃ hoti, so cetovimutto nāma.

* samathavasena = 「止の道筋で」
* gacchato = 「進む者には」
* cittekaggatā = 「心の一境性が」
* dhuraṃ hoti = 「中心（首座）となる」
* so = 「彼は」
* cetovimutto nāma = 「心解脱と呼ばれる」

→ 「止（サマタ）の道筋で進む者には、心の一境性が中心となり、彼は心解脱と呼ばれる。」

8. Vipassanāvasena gacchato paññā dhuraṃ hoti, so paññāvimutto nāmāti ettha therassa kaṅkhā natthi.

* vipassanāvasena = 「観の道筋で」
* gacchato = 「進む者には」
* paññā = 「智慧が」
* dhuraṃ hoti = 「中心となる」
* so = 「彼は」
* paññāvimutto nāmā ti = 「慧解脱と呼ばれる、ということについて」
* ettha = 「ここには」
* therassa = 「長老（アーナンダ）に」
* kaṅkhā natthi = 「疑いはない」

→ 「観（ヴィパッサナー）の道筋で進む者には智慧が中心となり、彼は慧解脱と呼ばれる、ということについて、ここでは長老（アーナンダ）に疑いはない。」

9. Ayaṃ sabhāvadhammoyeva, samathavaseneva pana gacchantesu eko cetovimutto nāma hoti, eko paññāvimutto.

* ayaṃ = 「これは」
* sabhāvadhammoyeva = 「まさに自然の理法（事の道理）である」
* samathavaseneva pana = 「しかしながら、まさに止の道筋で」
* gacchantesu = 「進む者たちの中で」
* eko = 「ある者は」
* cetovimutto nāma hoti = 「心解脱となる」
* eko = 「ある者は」
* paññāvimutto = 「慧解脱（となる）」

→ 「これはまさに事の道理である。しかしながら、まさに止（サマタ）の道筋で進む者たちの中でも、ある者は心解脱となり、ある者は慧解脱となる。」

10. Vipassanāvasena gacchantesupi eko paññāvimutto nāma hoti, eko cetovimuttoti ettha kiṃ kāraṇanti pucchati.

* vipassanāvasena = 「観の道筋で」
* gacchantesupi = 「進む者たちの中でも」
* eko paññāvimutto nāma hoti = 「ある者は慧解脱となる」
* eko cetovimutto ti = 「ある者は心解脱である、ということ」
* ettha = 「ここには」
* kiṃ kāraṇan ti = 「何が理由か、と」
* pucchati = 「尋ねているのである」

→ 「観（ヴィパッサナー）の道筋で進む者たちの中でも、ある者は慧解脱となり、ある者は心解脱となる。ここには何が理由なのか、と尋ねているのである。」

11. Indriyavemattataṃ vadāmīti indriyanānattataṃ vadāmi.

* indriyavemattataṃ vadāmī ti = 「『根の差異を私は説く』とは」
* indriyanānattataṃ = 「根（機能）の多様性を」
* vadāmi = 「私は説く（という意味である）」

→ 「『根の差異を私は説く』とは、『根の多様性を私は説く』という意味である。」

12. Idaṃ vuttaṃ hoti, na tvaṃ, ānanda, dasa pāramiyo pūretvā sabbaññutaṃ paṭivijjhi, tena te etaṃ apākaṭaṃ.

* idaṃ vuttaṃ hoti = 「これは次のような意味である」
* na tvaṃ = 「あなたは〜ない」
* ānanda = 「アーナンダよ」
* dasa pāramiyo = 「十波羅蜜を」
* pūretvā = 「満たして」
* sabbaññutaṃ = 「一切智を」
* paṭivijjhi = 「悟った（わけでは）」
* tena = 「それゆえに」
* te = 「あなたには」
* etaṃ = 「このことが」
* apākaṭaṃ = 「明らかでない」

13. Ahaṃ pana paṭivijjhiṃ, tena me etaṃ pākaṭaṃ.

* ahaṃ pana = 「しかし私は」

14. Ettha hi indriyanānattatā kāraṇaṃ.

* ettha hi = 「実にここには」
* indriyanānattatā = 「根の多様性が」
* kāraṇaṃ = 「理由（原因）である」

15. Samathavaseneva hi gacchantesu ekassa bhikkhuno cittekaggatā dhuraṃ hoti, so cetovimutto nāma hoti.

* samathavaseneva hi = 「実に、まさに止の道筋で」

16. Ekassa paññā dhuraṃ hoti, so paññāvimutto nāma hoti.

* paññā dhuraṃ hoti = 「智慧が中心となる」

17. Vipassanāvaseneva ca gacchantesu ekassa paññā dhuraṃ hoti, so paññāvimutto nāma hoti.

18. Ekassa cittekaggatā dhuraṃ hoti, so cetovimutto nāma hoti.

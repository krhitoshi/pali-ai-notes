# pali-ai-notes 運用ルール

## プロジェクトの目的

パーリ仏典・パーリ文をユーザ本人が読んで理解できるようになること, その習得度を
向上させるための分析やツールを提供すること.

- 訳語が平易かどうかはユーザの習得度による主観的な基準であり, 誰にでも平易で
  ある必要はない. 一般読者向けの平易化を目指すプロジェクトではない
- 訳語選定や平易化の作業では, 一般的な読みやすさではなくユーザの理解に資するか
  を基準に判断する. 迷った場合はユーザに確認する

## 公開リポジトリ

このリポジトリは公開リポジトリ. ファイルを作成・編集する際はプライベート情報を
含めない.

- 絶対パス (`/Users/<user>/...` のようにユーザー名が出るもの) を書かない.
  リポジトリ相対パスを使う
- 個人設定 (グローバルの `~/.claude/CLAUDE.md` など) の中身に言及しない
- メールアドレスなどの個人情報を書かない
- README.md の `note.com/krhitoshi` はユーザーが公開しているブログ URL のため
  問題ない

## 翻訳の方針

- assāsa / assasati は「出息 (外へ出る息)」, passāsa / passasati は「入息
  (内へ入る息)」と訳す (律註の定義を採用). 経註や現代の慣用ではこの逆に
  なることがあるが, 本プロジェクトでは出入の方向をこの向きに統一する

## 漢語・古語の平易化

- 文語漢語・古語表現を平易語に置換する改訂を行う際は `docs/plain_terms.md`
  (平易化リスト) を参照する. 置換語・保持する技術用語・機械置換の落とし穴が
  まとまっている


# 10. Āsavakkhayasuttaṃ

## 996

996. Āsavānaṃ khayāya saṃvattati. Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato saṃyojanappahānāya saṃvattati… anusayasamugghātāya saṃvattati… addhānapariññāya saṃvattati… āsavānaṃ khayāya saṃvattati? Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā…pe. … paṭinissaggānupassī assasissāmīti sikkhati, paṭinissaggānupassī passasissāmīti sikkhati. Evaṃ bhāvito kho, bhikkhave, ānāpānassatisamādhi evaṃ bahulīkato saṃyojanappahānāya saṃvattati…pe… anusayasamugghātāya saṃvattati…pe… addhānapariññāya saṃvattati…pe… āsavānaṃ khayāya saṃvattatīti. Dasamaṃ.

### 対訳

10. Āsavakkhayasuttaṃ

* āsava = 「漏（煩悩）」

* khaya = 「滅尽」

* suttaṃ = 「経」

→ 「漏尽経」

996-1. Āsavānaṃ khayāya saṃvattati.

* āsavānaṃ = 「諸々の漏の（属格複数）」

* khayāya = 「滅尽のために（与格）」

* saṃvattati = 「導く、資する」

→ 「（それは）諸々の漏の滅尽に導く。」

996-2. Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato saṃyojanappahānāya saṃvattati…

* kathaṃ = 「どのように」

* bhāvito = 「修習された（bhāveti の過去分詞）」

* ca = 「そして」

* bhikkhave = 「比丘たちよ（呼格）」

* ānāpānassatisamādhi = 「出入息念の定」

  * ānāpāna = 「出入息」

  * sati = 「念」

  * samādhi = 「定」

* bahulīkato = 「多く実践された」

* saṃyojana-ppahānāya = 「結の捨断のために」

  * saṃyojana = 「結（束縛する煩悩）」

  * pahāna = 「捨断」

* saṃvattati = 「導く」

→ 「比丘たちよ、出入息念の定はどのように修習され、どのように多く実践されると、結の捨断に導き…」

996-3. anusayasamugghātāya saṃvattati…

* anusaya = 「随眠（潜在煩悩）」

* samugghāta = 「根絶」

* saṃvattati = 「導く」

→ 「随眠の根絶に導き…」

996-4. addhānapariññāya saṃvattati…

* addhāna = 「行程、（輪廻の）長い道のり」

* pariññā = 「遍知」

→ 「行程の遍知に導き…」

996-5. āsavānaṃ khayāya saṃvattati?

* āsavānaṃ khayāya = 「諸々の漏の滅尽のために」

→ 「諸々の漏の滅尽に導くのか？」

996-6. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā…pe.

* idha = 「ここに（この教えにおいて）」

* bhikkhu = 「比丘は」

* arañña-gato = 「森に行った者」

* rukkha-mūla-gato = 「樹の根元に行った者」

* vā ... vā = 「あるいは～あるいは～」

* …pe. = 「（中略）」

→ 「比丘たちよ、ここに比丘は森に行き、あるいは樹の根元に行き…（中略）」

996-7. paṭinissaggānupassī assasissāmīti sikkhati, paṭinissaggānupassī passasissāmīti sikkhati.

* paṭinissagga = 「捨遣（手放すこと）」

* anupassī = 「随観する者」

* assasissāmi = 「私は出息しよう（assasati の未来形一人称単数）」

* passasissāmi = 「私は入息しよう（passasati の未来形一人称単数）」

* iti = 「～と」

* sikkhati = 「学ぶ、訓練する」

→ 「『捨遣を随観しながら出息しよう』と学び、『捨遣を随観しながら入息しよう』と学ぶ。」

996-8. Evaṃ bhāvito kho, bhikkhave, ānāpānassatisamādhi evaṃ bahulīkato saṃyojanappahānāya saṃvattati…pe…

* evaṃ = 「このように」

* bhāvito = 「修習された」

* kho = 「実に」

* bahulīkato = 「多く実践された」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念の定は、結の捨断に導き…（中略）」

996-9. anusayasamugghātāya saṃvattati…pe… addhānapariññāya saṃvattati…pe… āsavānaṃ khayāya saṃvattatīti.

* anusaya-samugghātāya = 「随眠の根絶のために」

* addhāna-pariññāya = 「行程の遍知のために」

* āsavānaṃ khayāya = 「諸々の漏の滅尽のために」

* saṃvattati = 「導く」

* iti = 「～と（経の結び）」

→ 「随眠の根絶に導き…（中略）行程の遍知に導き…（中略）諸々の漏の滅尽に導く、と。」

996-10. Dasamaṃ.

* dasamaṃ = 「第十（の経）」

→ 「第十（経）である。」

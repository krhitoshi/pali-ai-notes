
# 1. Ekadhammasuttaṃ

## 977

977. Sāvatthinidānaṃ . Tatra kho…pe… etadavoca – ‘‘ekadhammo, bhikkhave, bhāvito bahulīkato mahapphalo hoti mahānisaṃso. Katamo ekadhammo? Ānāpānassati [ānāpānasati (sī. pī.)]. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā? Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova [sato (bahūsu) tatiyapārājikepi] passasati. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti, dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti; rassaṃ vā assasanto ‘rassaṃ assasāmī’ti pajānāti, rassaṃ vā passasanto ‘rassaṃ passasāmī’ti pajānāti; ‘sabbakāyappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sabbakāyappaṭisaṃvedī passasissāmī’ti sikkhati; ‘passambhayaṃ kāyasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ kāyasaṅkhāraṃ passasissāmī’ti sikkhati; ‘pītippaṭisaṃvedī assasissāmī’ti sikkhati, ‘pītippaṭisaṃvedī passasissāmī’ti sikkhati; ‘sukhappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sukhappaṭisaṃvedī passasissāmī’ti sikkhati; ‘cittasaṅkhārappaṭisaṃvedī assasissāmī’ti sikkhati, ‘cittasaṅkhārappaṭisaṃvedī passasissāmī’ti sikkhati; ‘passambhayaṃ cittasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ cittasaṅkhāraṃ passasissāmī’ti sikkhati; ‘cittappaṭisaṃvedī assasissāmī’ti sikkhati, ‘cittappaṭisaṃvedī passasissāmī’ti sikkhati; ‘abhippamodayaṃ cittaṃ assasissāmī’ti sikkhati, ‘abhippamodayaṃ cittaṃ passasissāmī’ti sikkhati; ‘samādahaṃ cittaṃ assasissāmī’ti sikkhati , ‘samādahaṃ cittaṃ passasissāmī’ti sikkhati; ‘vimocayaṃ cittaṃ assasissāmī’ti sikkhati, ‘vimocayaṃ cittaṃ passasissāmī’ti sikkhati; ‘aniccānupassī assasissāmī’ti sikkhati, ‘aniccānupassī passasissāmī’ti sikkhati; ‘virāgānupassī assasissāmī’ti sikkhati, ‘virāgānupassī passasissāmī’ti sikkhati; ‘nirodhānupassī assasissāmī’ti sikkhati, ‘nirodhānupassī passasissāmī’ti sikkhati; ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā’’ti. Paṭhamaṃ.

### 対訳

1. Ekadhammasuttaṃ

* eka = 「一つの」
* dhamma = 「法」
* suttaṃ = 「経」

→ 「一法経」

2. Sāvatthinidānaṃ.

* Sāvatthi = 「サーヴァッティー(舎衛城)」
* nidānaṃ = 「因縁、由来(が舎衛城であること)」

→ 「サーヴァッティーでの因縁(この経は舎衛城で説かれた)。」

3. Tatra kho …pe… etadavoca –

* tatra = 「そこで」
* kho = 「実に(強調の不変化詞)」
* …pe… = 「(中略)」
* etad avoca = 「このように言った」
  * etad = 「これを」
  * avoca = 「言った(vacati の アオリスト)」

→ 「そこで(世尊は)…(中略)…このように言った。」

4. ‘‘ekadhammo, bhikkhave, bhāvito bahulīkato mahapphalo hoti mahānisaṃso.

* ekadhammo = 「一つの法は」
* bhikkhave = 「比丘たちよ(呼格)」
* bhāvito = 「修習された(bhāveti の過去分詞)」
* bahulīkato = 「多く実践された」
* mahapphalo = 「大きな果報のある」
* hoti = 「～となる」
* mahānisaṃso = 「大きな利益のある」

→ 「比丘たちよ、一つの法は、修習され、多く実践されたとき、大きな果報があり、大きな利益がある。」

5. Katamo ekadhammo? Ānāpānassati.

* katamo = 「どの、いかなる」
* ekadhammo = 「一つの法(とは)」
* ānāpānassati = 「出入息念」
  * āna = 「出息」/ apāna = 「入息」/ sati = 「念」(要確認: āna/apāna の出入対応は註釈により異なる)

→ 「どの一つの法か。出入息念である。」

6. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā?

* kathaṃ = 「どのように」
* bhāvitā = 「修習された(女性形)」
* ca = 「そして」
* kathaṃ bahulīkatā = 「どのように多く実践された(とき)」
* mahapphalā hoti mahānisaṃsā = 「大きな果報があり、大きな利益がある(のか)」

→ 「比丘たちよ、出入息念はどのように修習され、どのように多く実践されたとき、大きな果報があり、大きな利益があるのか。」

7. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに(この教えにおいて)」
* bhikkhu = 「比丘が」
* araññagato vā = 「森に行った者、あるいは」
* rukkhamūlagato vā = 「木の根元に行った者、あるいは」
* suññāgāragato vā = 「空き家に行った者、あるいは」
* nisīdati = 「坐る」
* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」
* ujuṃ kāyaṃ paṇidhāya = 「体をまっすぐに保って」
* parimukhaṃ satiṃ upaṭṭhapetvā = 「面前に念を確立して」

→ 「比丘たちよ、ここに比丘が、森に行き、あるいは木の根元に行き、あるいは空き家に行って、結跏趺坐を組み、体をまっすぐに保ち、面前に念を確立して坐る。」

8. So satova assasati, satova passasati.

* so = 「彼は」
* sato va = 「念をもってこそ」
* assasati = 「出息する(息を吐く)」
* passasati = 「入息する(息を吸う)」

→ 「彼は念をもって出息し、念をもって入息する。」

9. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti, dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti;

* dīghaṃ = 「長く」
* assasanto = 「出息しながら(現在分詞)」
* ‘dīghaṃ assasāmī’ti = 「『私は長く出息している』と」
* pajānāti = 「知る、了知する」
* passasanto = 「入息しながら」

→ 「長く出息しながら『私は長く出息している』と知り、長く入息しながら『私は長く入息している』と知る。」

10. rassaṃ vā assasanto ‘rassaṃ assasāmī’ti pajānāti, rassaṃ vā passasanto ‘rassaṃ passasāmī’ti pajānāti;

* rassaṃ = 「短く」

→ 「短く出息しながら『私は短く出息している』と知り、短く入息しながら『私は短く入息している』と知る。」

11. ‘sabbakāyappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sabbakāyappaṭisaṃvedī passasissāmī’ti sikkhati;

* sabba-kāya-paṭisaṃvedī = 「全身を感受する者(となって)」
  * sabba = 「すべての」/ kāya = 「身」/ paṭisaṃvedin = 「感受する者」
* assasissāmi = 「私は出息しよう(未来形)」
* sikkhati = 「学ぶ、訓練する」

→ 「『全身を感受しつつ私は出息しよう』と学び、『全身を感受しつつ私は入息しよう』と学ぶ。」

12. ‘passambhayaṃ kāyasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ kāyasaṅkhāraṃ passasissāmī’ti sikkhati;

* passambhayaṃ = 「静めながら(passambheti の現在分詞)」
* kāyasaṅkhāraṃ = 「身行を(呼吸のはたらき)」

→ 「『身行を静めながら私は出息しよう』と学び、『身行を静めながら私は入息しよう』と学ぶ。」

13. ‘pītippaṭisaṃvedī assasissāmī’ti sikkhati, ‘pītippaṭisaṃvedī passasissāmī’ti sikkhati;

* citta-saṅkhāra-paṭisaṃvedī = 「心行を感受する者(となって)」
  * cittasaṅkhāra = 「心行(想と受)」

16. ‘passambhayaṃ cittasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ cittasaṅkhāraṃ passasissāmī’ti sikkhati;

17. ‘cittappaṭisaṃvedī assasissāmī’ti sikkhati, ‘cittappaṭisaṃvedī passasissāmī’ti sikkhati;

25. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā’’ti. Paṭhamaṃ.

* evaṃ = 「このように」
* bhāvitā = 「修習された」
* bahulīkatā = 「多く実践された」
* mahapphalā hoti mahānisaṃsā = 「大きな果報があり、大きな利益がある」
* ti = 「～と(引用終わり)」
* paṭhamaṃ = 「第一(経)」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念は、大きな果報があり、大きな利益がある。」第一(経)。


# 9. Vesālīsuttaṃ

## 985 (1)

985. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā vesāliyaṃ viharati mahāvane kūṭāgārasālāyaṃ. Tena kho pana samayena bhagavā bhikkhūnaṃ anekapariyāyena asubhakathaṃ katheti, asubhāya vaṇṇaṃ bhāsati, asubhabhāvanāya vaṇṇaṃ bhāsati.

Atha kho bhagavā bhikkhū āmantesi – ‘‘icchāmahaṃ, bhikkhave, aḍḍhamāsaṃ paṭisallīyituṃ. Nāmhi kenaci upasaṅkamitabbo, aññatra ekena piṇḍapātanīhārakenā’’ti. ‘‘Evaṃ, bhante’’ti kho te bhikkhū bhagavato paṭissutvā nāssudha koci bhagavantaṃ upasaṅkamati, aññatra ekena piṇḍapātanīhārakena.

Atha kho te bhikkhū – ‘‘bhagavā anekapariyāyena asubhakathaṃ katheti, asubhāya vaṇṇaṃ bhāsati , asubhabhāvanāya vaṇṇaṃ bhāsatī’’ti anekākāravokāraṃ asubhabhāvanānuyogamanuyuttā viharanti. Te iminā kāyena aṭṭīyamānā [aṭṭiyamānā (sī. syā. kaṃ. pī. ka.)] harāyamānā jigucchamānā satthahārakaṃ pariyesanti. Dasapi bhikkhū ekāhena satthaṃ āharanti, vīsampi…pe… tiṃsampi bhikkhū ekāhena satthaṃ āharanti.

### 対訳

9. Vesālīsuttaṃ

* vesālī-sutta = 「ヴェーサーリー経」
  * 依主釈（ヴェーサーリーにおける経）
→「第九 ヴェーサーリー経」

985.

1. Evaṃ me sutaṃ –

* evaṃ = 「このように」
* me = 「私によって」（具格の意味で用いられた属格）
* sutaṃ = 「聞かれた」（√su の過去分詞・中性単数）
→「このように私は聞いた。」

2. ekaṃ samayaṃ bhagavā vesāliyaṃ viharati mahāvane kūṭāgārasālāyaṃ.

* ekaṃ samayaṃ = 「ある時」（時を表す対格）
* bhagavā = 「世尊は」（主格単数）
* vesāliyaṃ = 「ヴェーサーリーに」（処格）
* viharati = 「住している」（vi + √har、歴史的現在）
* mahāvane = 「大林に」（処格）
* kūṭāgārasālāyaṃ = 「重閣講堂に」（処格）
  * kūṭāgāra（尖った屋根の建物）+ sālā（堂）＝依主釈
→「ある時、世尊はヴェーサーリーの大林にある重閣講堂に住しておられた。」

3. Tena kho pana samayena bhagavā bhikkhūnaṃ anekapariyāyena asubhakathaṃ katheti, asubhāya vaṇṇaṃ bhāsati, asubhabhāvanāya vaṇṇaṃ bhāsati.

* tena kho pana samayena = 「さて、その時」（具格）
* bhikkhūnaṃ = 「比丘たちに」（与格複数）
* anekapariyāyena = 「多くの方法で」（具格）
  * aneka（多くの）+ pariyāya（方法）＝持業釈
* asubhakathaṃ = 「不浄についての話を」（対格）
  * asubha（不浄）+ kathā（話）＝依主釈
* katheti = 「語る」（√kath の使役形）
* asubhāya vaṇṇaṃ bhāsati = 「不浄（の観察）の称賛を説く」
  * vaṇṇa = 「称賛、功徳」
  * bhāsati = 「説く、語る」（√bhās）
* asubhabhāvanāya = 「不浄の修習の」（属格）
  * asubha + bhāvanā（修習）＝依主釈
→「さてその時、世尊は比丘たちに多くの方法で不浄についての話を語り、不浄（の観察）の称賛を説き、不浄の修習の称賛を説いておられた。」

4. Atha kho bhagavā bhikkhū āmantesi – ‘‘icchāmahaṃ, bhikkhave, aḍḍhamāsaṃ paṭisallīyituṃ.

* atha kho = 「そこで」
* bhikkhū = 「比丘たちに」（対格複数）
* āmantesi = 「呼びかけた」（ā + √mant、アオリスト）
* icchāmi ahaṃ = 「私は望む」（icchāmahaṃ は連声）
* bhikkhave = 「比丘たちよ」（呼格）
* aḍḍhamāsaṃ = 「半月の間」（期間を表す対格）
  * aḍḍha（半分）+ māsa（月）
* paṭisallīyituṃ = 「独坐すること、独り静かに籠もること」（paṭi + saṃ + √lī の不定詞）
→「そこで世尊は比丘たちに呼びかけられた。『比丘たちよ、私は半月の間、独坐したいと思う。」

5. Nāmhi kenaci upasaṅkamitabbo, aññatra ekena piṇḍapātanīhārakenā’’ti.

* na amhi = 「私は～ない」（nāmhi は連声）
* kenaci = 「誰によっても」（具格）
* upasaṅkamitabbo = 「近づかれるべき」（upa + saṃ + √kam の義務分詞）
* aññatra = 「～を除いて」（具格を取る不変化詞）
* ekena piṇḍapātanīhārakena = 「一人の托鉢食を運ぶ者を除いて」
  * piṇḍapāta（托鉢食）+ nīhāraka（運ぶ者、nī + √har）＝依主釈
→「托鉢食を運ぶ一人を除いて、誰も私のところへ来てはならない』と。」

6. ‘‘Evaṃ, bhante’’ti kho te bhikkhū bhagavato paṭissutvā nāssudha koci bhagavantaṃ upasaṅkamati, aññatra ekena piṇḍapātanīhārakena.

* evaṃ bhante = 「かしこまりました、尊師よ」
* te bhikkhū = 「それらの比丘たちは」
* bhagavato = 「世尊に」（与格）
* paṭissutvā = 「応答して」（paṭi + √su の連続体）
* na assudha = 「実に～ない」（assudha は強意の不変化詞）
* koci = 「誰も」
* bhagavantaṃ upasaṅkamati = 「世尊に近づく」（歴史的現在）
→「『かしこまりました、尊師よ』と、それらの比丘たちは世尊に応答し、托鉢食を運ぶ一人を除いて、実に誰も世尊のもとへ近づかなかった。」

7. Atha kho te bhikkhū – ‘‘bhagavā anekapariyāyena asubhakathaṃ katheti, asubhāya vaṇṇaṃ bhāsati, asubhabhāvanāya vaṇṇaṃ bhāsatī’’ti anekākāravokāraṃ asubhabhāvanānuyogamanuyuttā viharanti.

* atha kho te bhikkhū = 「そこで、それらの比丘たちは」
* ‘‘bhagavā … bhāsatī’’ti = 「『世尊は…説かれた』と（考えて）」（ti は引用・理由）
* anekākāravokāraṃ = 「多種多様な仕方で」（副詞的対格）
  * aneka（多くの）+ ākāra（行相、仕方）+ vokāra（種類、区別）
* asubhabhāvanānuyogam anuyuttā = 「不浄の修習の実践に専心して」
  * asubha + bhāvanā + anuyoga（実践、専修）＝依主釈
  * anuyuttā = 「従事した、専心した」（anu + √yuj の過去分詞・複数）
* viharanti = 「住する」
→「そこで、それらの比丘たちは『世尊は多くの方法で不浄についての話を語り、不浄の称賛を説き、不浄の修習の称賛を説かれた』と（考えて）、多種多様な仕方で不浄の修習の実践に専心して住した。」

8. Te iminā kāyena aṭṭīyamānā harāyamānā jigucchamānā satthahārakaṃ pariyesanti.

* te = 「彼らは」
* iminā kāyena = 「この身体によって、この身体を」（具格）
* aṭṭīyamānā = 「悩み苦しみ」（aṭṭīyati の現在分詞・複数）〔異読〕aṭṭiyamānā（sī. syā. kaṃ. pī. ka.）
* harāyamānā = 「恥じ」（harāyati の現在分詞）
* jigucchamānā = 「嫌悪して」（√gup の強意動詞 jigucchati の現在分詞）
* satthahārakaṃ = 「刀を持って来る者（＝自分の命を断ってくれる者）を」（対格）
  * sattha（刀）+ hāraka（持って来る者、√har）＝依主釈
* pariyesanti = 「探し求める」（pari + √es）
→「彼らはこの身体に悩み、恥じ、嫌悪して、刀を持って来る者（命を断ってくれる者）を探し求めた。」

9. Dasapi bhikkhū ekāhena satthaṃ āharanti, vīsampi…pe… tiṃsampi bhikkhū ekāhena satthaṃ āharanti.

* dasa pi bhikkhū = 「十人もの比丘たちが」
* ekāhena = 「一日で」（eka + aha〔日〕の具格）
* satthaṃ āharanti = 「刀を取る（＝自ら命を断つ）」
  * sattha = 「刀」
  * āharanti = 「持って来る、（自らに）用いる」（ā + √har）
* vīsam pi = 「二十人も」（中略）
* tiṃsam pi = 「三十人も」
→「一日に十人もの比丘たちが刀を取り（自ら命を断ち）、二十人も（中略）三十人もの比丘たちが一日に刀を取った。」

### 対訳

1. Atha kho bhagavā tassa aḍḍhamāsassa accayena paṭisallānā vuṭṭhito āyasmantaṃ ānandaṃ āmantesi –

* atha kho = 「そこで」
* bhagavā = 「世尊は」（主格単数）
* tassa aḍḍhamāsassa accayena = 「その半月が過ぎたとき」
  * accayena = 「経過によって」（accaya〔ati + √i、経過〕の具格）
* paṭisallānā = 「独坐から」（奪格）
* vuṭṭhito = 「出て」（vi + uṭ + √ṭhā の過去分詞・主格単数）
* āyasmantaṃ ānandaṃ = 「尊者アーナンダに」（対格）
* āmantesi = 「呼びかけた」（ā + √mant、アオリスト）
→「そこで世尊は、その半月が過ぎたとき独坐から出て、尊者アーナンダに呼びかけられた。」

2. ‘‘kiṃ nu kho, ānanda, tanubhūto viya bhikkhusaṅgho’’ti?

* kiṃ nu kho = 「いったいなぜ～か」（疑問の不変化詞）
* ānanda = 「アーナンダよ」（呼格）
* tanubhūto = 「少なくなった」
  * tanu（少ない、細い）+ bhūta（√bhū の過去分詞、～となった）
* viya = 「～のように、～のようである」
* bhikkhusaṅgho = 「比丘僧団は」（主格単数）
  * bhikkhu + saṅgha ＝ 依主釈
→「『アーナンダよ、比丘僧団はどうして少なくなったように（見える）のか』と。」

3. ‘‘Tathā hi pana, bhante, ‘bhagavā bhikkhūnaṃ anekapariyāyena asubhakathaṃ katheti, asubhāya vaṇṇaṃ bhāsati, asubhabhāvanāya vaṇṇaṃ bhāsatī’ti anekākāravokāraṃ asubhabhāvanānuyogamanuyuttā viharanti.

* tathā hi pana = 「実はこういうわけです、尊師よ」（理由を導く定型句）
* bhante = 「尊師よ」（呼格）
* bhagavā bhikkhūnaṃ anekapariyāyena asubhakathaṃ katheti = 「世尊は比丘たちに多くの方法で不浄についての話を語られる」
* asubhāya vaṇṇaṃ bhāsati = 「不浄（の観察）の称賛を説かれる」
* asubhabhāvanāya vaṇṇaṃ bhāsatī ti = 「不浄の修習の称賛を説かれる、と（考えて）」
* anekākāravokāraṃ = 「多種多様な仕方で」（副詞的対格）
  * aneka + ākāra（行相、仕方）+ vokāra（種類、区別）
* asubhabhāvanānuyogam anuyuttā = 「不浄の修習の実践に専心して」
  * asubha + bhāvanā + anuyoga（実践、専修）＝依主釈
  * anuyuttā = 「専心した」（anu + √yuj の過去分詞・複数）
* viharanti = 「住しています」（歴史的現在）
→「『実はこういうわけです、尊師よ。比丘たちは〈世尊は多くの方法で不浄についての話を語り、不浄の称賛を説き、不浄の修習の称賛を説かれた〉と（考えて）、多種多様な仕方で不浄の修習の実践に専心して住しています。」

4. Te iminā kāyena aṭṭīyamānā harāyamānā jigucchamānā satthahārakaṃ pariyesanti.

* te = 「彼らは」
* iminā kāyena = 「この身体に」（具格）
* aṭṭīyamānā = 「悩み苦しみ」（aṭṭīyati の現在分詞・複数）
* harāyamānā = 「恥じ」（harāyati の現在分詞）
* jigucchamānā = 「嫌悪して」（jigucchati〔√gup の強意動詞〕の現在分詞）
* satthahārakaṃ = 「刀を持って来る者（命を断ってくれる者）を」（対格）
  * sattha（刀）+ hāraka（持って来る者、√har）＝依主釈
* pariyesanti = 「探し求めています」（pari + √es）
→「彼らはこの身体に悩み、恥じ、嫌悪して、刀を持って来る者（命を断ってくれる者）を探し求めています。」

5. Dasapi bhikkhū ekāhena satthaṃ āharanti, vīsampi bhikkhū… tiṃsampi bhikkhū ekāhena satthaṃ āharanti.

* dasa pi bhikkhū = 「十人もの比丘たちが」
* ekāhena = 「一日で」（eka + aha〔日〕の具格）
* satthaṃ āharanti = 「刀を取ります（自ら命を断ちます）」
* vīsam pi bhikkhū =「二十人もの比丘たちが」（中略）
* tiṃsam pi bhikkhū = 「三十人もの比丘たちが」
→「一日に十人もの比丘たちが刀を取り、二十人もの比丘たちが（中略）三十人もの比丘たちが一日に刀を取っています。」

6. Sādhu, bhante, bhagavā aññaṃ pariyāyaṃ ācikkhatu yathāyaṃ bhikkhusaṅgho aññāya saṇṭhaheyyā’’ti.

* sādhu = 「どうか、なにとぞ」（願望を導く不変化詞）
* bhagavā … ācikkhatu = 「世尊は～をお説きください」（ā + √cikkh〔√khyā の強意形〕、命令法三人称単数）
* aññaṃ pariyāyaṃ = 「別の方法を」（対格）
* yathā ayaṃ bhikkhusaṅgho = 「この比丘僧団が～するように」（yathāyaṃ は連声）
* aññāya = 「完全智（阿羅漢果の智）において」（aññā の処格）
* saṇṭhaheyyā ti = 「確立するように、と」（saṃ + √ṭhā の願望法三人称単数）
→「尊師よ、どうか世尊は、この比丘僧団が完全智において確立するような別の方法をお説きください』と。」

7. ‘‘Tenahānanda, yāvatikā bhikkhū vesāliṃ upanissāya viharanti te sabbe upaṭṭhānasālāyaṃ sannipātehī’’ti.

* tena hi ānanda = 「それではアーナンダよ」（tenahānanda は連声）
* yāvatikā bhikkhū = 「～する限りの比丘たちを」（関係詞的形容詞・主格複数）
* vesāliṃ upanissāya = 「ヴェーサーリーに依って、～の近くで」
  * upanissāya = 「依って」（upa + ni + √si の連続体、対格を取る）
* viharanti = 「住している」
* te sabbe = 「それらすべてを」（対格）
* upaṭṭhānasālāyaṃ = 「集会堂に」（処格）
  * upaṭṭhāna（奉仕、伺候）+ sālā（堂）＝依主釈
* sannipātehī ti = 「集めなさい、と」（saṃ + ni + √pat の使役・命令法二人称単数）
→「『それではアーナンダよ、ヴェーサーリーの近くに住している限りの比丘たちを、すべて集会堂に集めなさい』と。」

8. ‘‘Evaṃ, bhante’’ti kho āyasmā ānando bhagavato paṭissutvā yāvatikā bhikkhū vesāliṃ upanissāya viharanti te sabbe upaṭṭhānasālāyaṃ sannipātetvā yena bhagavā tenupasaṅkami;

* evaṃ bhante ti = 「『かしこまりました、尊師よ』と」
* āyasmā ānando = 「尊者アーナンダは」（主格）
* bhagavato paṭissutvā = 「世尊に応答して」（paṭi + √su の連続体）
* sannipātetvā = 「集めて」（sannipāteti の連続体）
* yena bhagavā tena upasaṅkami = 「世尊のおられる所へ近づいた」
  * yena … tena … = 「～の所へ」（方向を表す定型句）
  * upasaṅkami = 「近づいた」（upa + saṃ + √kam、アオリスト）
→「『かしこまりました、尊師よ』と、尊者アーナンダは世尊に応答し、ヴェーサーリーの近くに住している限りの比丘たちをすべて集会堂に集めてから、世尊のおられる所へ近づいた。」

9. upasaṅkamitvā bhagavantaṃ etadavoca – ‘‘sannipatito, bhante, bhikkhusaṅgho. Yassa dāni, bhante, bhagavā kālaṃ maññatī’’ti.

* upasaṅkamitvā = 「近づいてから」（連続体）
* bhagavantaṃ etad avoca = 「世尊にこう言った」
  * etad = 「これを」（対格）
  * avoca = 「言った」（√vac、アオリスト）
* sannipatito = 「集まりました」（saṃ + ni + √pat の過去分詞・主格単数）〔異読〕sannipātito（sī.）＝「集められました」（使役の過去分詞）
* bhikkhusaṅgho = 「比丘僧団は」
* yassa dāni bhagavā kālaṃ maññatī ti = 「今や世尊がその時であるとお考えになることのために（＝今がその時とお考えでしたら、どうぞ）」
  * yassa = 「そのことのために」（属格）
  * dāni = 「今や」
  * kālaṃ maññati = 「時であると考える」（√man）
→「近づいてから、世尊にこう言った。『尊師よ、比丘僧団が集まりました。尊師よ、今がその時であると世尊がお考えでしたら、どうぞ』と。」

## 985 (3)

Atha kho bhagavā yena upaṭṭhānasālā tenupasaṅkami; upasaṅkamitvā paññatte āsane nisīdi. Nisajja kho bhagavā bhikkhū āmantesi – ‘‘ayampi kho, bhikkhave, ānāpānassatisamādhi bhāvito bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasameti’’.

‘‘Seyyathāpi , bhikkhave, gimhānaṃ pacchime māse ūhataṃ rajojallaṃ, tamenaṃ mahāakālamegho ṭhānaso antaradhāpeti vūpasameti; evameva kho, bhikkhave, ānāpānassatisamādhi bhāvito bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasameti. Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasameti?

‘‘Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvito kho, bhikkhave, ānāpānassatisamādhi evaṃ bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasametī’’ti. Navamaṃ.

### 対訳

1. Atha kho bhagavā yena upaṭṭhānasālā tenupasaṅkami; upasaṅkamitvā paññatte āsane nisīdi.

* atha kho = 「そこで」
* bhagavā = 「世尊は」（主格単数）
* yena upaṭṭhānasālā tena upasaṅkami = 「集会堂のある所へ近づいた」
  * yena … tena … = 「～の所へ」（方向を表す定型句）
  * upaṭṭhānasālā = 「集会堂」（upaṭṭhāna〔奉仕、伺候〕+ sālā〔堂〕＝依主釈）
  * upasaṅkami = 「近づいた」（upa + saṃ + √kam、アオリスト）
* upasaṅkamitvā = 「近づいてから」（連続体）
* paññatte āsane = 「設けられた座に」（処格）
  * paññatta = 「設けられた」（pa + √ñap の過去分詞）
* nisīdi = 「坐った」（ni + √sad、アオリスト）
→「そこで世尊は集会堂のある所へ近づき、近づいてから設けられた座に坐られた。」

2. Nisajja kho bhagavā bhikkhū āmantesi –

* nisajja = 「坐って」（ni + √sad の連続体）
* bhikkhū = 「比丘たちに」（対格複数）
* āmantesi = 「呼びかけた」（ā + √mant、アオリスト）
→「坐ってから、世尊は比丘たちに呼びかけられた。」

3. ‘‘ayampi kho, bhikkhave, ānāpānassatisamādhi bhāvito bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasameti’’.

* ayam pi kho = 「この～もまた」
* bhikkhave = 「比丘たちよ」（呼格）
* ānāpānassatisamādhi = 「出入息念の定は」（主格単数）
  * ānāpāna（出入息）+ sati（念）+ samādhi（定）＝依主釈
* bhāvito = 「修習され」（√bhū の使役の過去分詞）
* bahulīkato = 「多く実践されるならば」（bahulī + kata、「多く為された」）
* santo ca eva = 「寂静であり」
* paṇīto ca = 「勝妙であり→すぐれたものであり」（pa + √nī の過去分詞）
* asecanako ca = 「混ぜ物を要しないものであり」
  * a + secanaka（√sic〔注ぐ〕から）＝「（何かを）加え注ぐ必要のない、それ自体で完結して美味なもの」
* sukho ca vihāro = 「楽の住である」
* uppannuppanne ca pāpake akusale dhamme = 「生じ生じた悪しき不善の諸法を」（対格複数）
  * uppannuppanna = 「生じるたびごとの」（uppanna の畳語）
* ṭhānaso = 「即座に」（ṭhāna + so〔奪格的接尾辞〕）
* antaradhāpeti = 「消失させ」（antara + √dhā の使役）
* vūpasameti = 「静める」（vi + upa + √sam の使役）
→「『比丘たちよ、この出入息念の定もまた、修習され多く実践されるならば、寂静であり、すぐれたものであり、混ぜ物を要しないものであり、楽の住であって、生じ生じた悪しき不善の諸法を即座に消失させ、静めるのである。」

4. ‘‘Seyyathāpi, bhikkhave, gimhānaṃ pacchime māse ūhataṃ rajojallaṃ, tamenaṃ mahāakālamegho ṭhānaso antaradhāpeti vūpasameti;

* seyyathāpi = 「たとえば～のように」
* gimhānaṃ pacchime māse = 「夏の最後の月に」
  * gimhānaṃ = 「夏の」（属格複数）
  * pacchime māse = 「最後の月に」（処格）
* ūhataṃ rajojallaṃ = 「舞い上がった塵埃は」
  * ūhata = 「舞い上がった、かき立てられた」（ū〔ud〕+ √han の過去分詞）
  * rajojalla = 「塵と埃」（rajo〔塵〕+ jalla〔垢、埃〕＝相違釈）
* tam enaṃ = 「それを」（対格・強調の重複）
* mahāakālamegho = 「大きな時ならぬ雨雲が」（主格）
  * mahā（大）+ akāla（時ならぬ、季節はずれの）+ megha（雲、雨雲）＝持業釈
* ṭhānaso antaradhāpeti vūpasameti = 「即座に消失させ、静める」
→「たとえば比丘たちよ、夏の最後の月に舞い上がった塵埃を、大きな時ならぬ雨雲が即座に消失させ、静めるように、」

5. evameva kho, bhikkhave, ānāpānassatisamādhi bhāvito bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasameti.

* evam eva kho = 「まさにそのように」
* （以下、チャンク3と同文）
→「まさにそのように比丘たちよ、出入息念の定は、修習され多く実践されるならば、寂静であり、すぐれたものであり、混ぜ物を要しないものであり、楽の住であって、生じ生じた悪しき不善の諸法を即座に消失させ、静めるのである。」

6. Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasameti?

* kathaṃ bhāvito = 「どのように修習され」
* kathaṃ bahulīkato = 「どのように多く実践されるならば」
* （以下、チャンク3と同文）
→「では比丘たちよ、出入息念の定はどのように修習され、どのように多く実践されるならば、寂静であり、すぐれたものであり、混ぜ物を要しないものであり、楽の住であって、生じ生じた悪しき不善の諸法を即座に消失させ、静めるのであるか。」

7. ‘‘Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘が」（主格単数）
* araññagato vā = 「林に行き」
  * arañña（林）+ gata（行った）＝依主釈
* rukkhamūlagato vā = 「樹の根元に行き」
  * rukkha（樹）+ mūla（根元）+ gata ＝依主釈
* suññāgāragato vā = 「空き家に行き」
  * suñña（空の）+ agāra（家）+ gata ＝依主釈
* nisīdati = 「坐る」（ni + √sad）
* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」
  * ābhujitvā = 「組んで」（ā + √bhuj の連続体）
* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに定め置いて」
  * uju = 「まっすぐな」
  * paṇidhāya = 「定め置いて」（pa + ni + √dhā の連続体）
* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に現前させて」
  * parimukhaṃ = 「面前に」（pari + mukha の副詞的対格）
  * upaṭṭhapetvā = 「現前させて」（upa + √ṭhā の使役の連続体）
→「ここに比丘たちよ、比丘は林に行き、あるいは樹の根元に行き、あるいは空き家に行って、結跏趺坐を組み、身体をまっすぐに定め置き、念を面前に現前させて坐る。」

8. So satova assasati, satova passasati…pe…

* so = 「彼は」
* sato va = 「念をそなえたままに」
  * sata = 「念のある」（√sar〔憶念する〕の過去分詞）
  * va（eva）= 「まさに～のままに」
* assasati = 「出息する」（ā + √sas）
* passasati = 「入息する」（pa + √sas）
* …pe… = （中略）
→「彼は念をそなえたままに出息し、念をそなえたままに入息する。（中略）」

9. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissaggānupassī = 「捨て去ることを随観する者として」
  * paṭinissagga（捨て去ること、捨遣。paṭi + nis + √saj から）+ anupassin（随観する者）＝依主釈
* assasissāmī ti = 「出息しよう、と」（未来形一人称単数）
* sikkhati = 「学ぶ、修学する」（√sikkh）
* passasissāmī ti = 「入息しよう、と」
→「『捨て去ることを随観しつつ出息しよう』と学び、『捨て去ることを随観しつつ入息しよう』と学ぶ。」

10. Evaṃ bhāvito kho, bhikkhave, ānāpānassatisamādhi evaṃ bahulīkato santo ceva paṇīto ca asecanako ca sukho ca vihāro uppannuppanne ca pāpake akusale dhamme ṭhānaso antaradhāpeti vūpasametī’’ti. Navamaṃ.

* evaṃ bhāvito = 「このように修習され」
* evaṃ bahulīkato = 「このように多く実践されるならば」
* （以下、チャンク3と同文）
* navamaṃ = 「第九（経）」
→「比丘たちよ、出入息念の定はこのように修習され、このように多く実践されるならば、寂静であり、すぐれたものであり、混ぜ物を要しないものであり、楽の住であって、生じ生じた悪しき不善の諸法を即座に消失させ、静めるのである』と。第九（経）である。」

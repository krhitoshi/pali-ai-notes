
# 1. Icchānaṅgalasuttaṃ

## 987 (1)

987. Ekaṃ samayaṃ bhagavā icchānaṅgale viharati icchānaṅgalavanasaṇḍe. Tatra kho bhagavā bhikkhū āmantesi – ‘‘icchāmahaṃ, bhikkhave, temāsaṃ paṭisallīyituṃ. Nāmhi kenaci upasaṅkamitabbo, aññatra ekena piṇḍapātanīhārakenā’’ti. ‘‘Evaṃ, bhante’’ti kho te bhikkhū bhagavato paṭissutvā nāssudha koci bhagavantaṃ upasaṅkamati, aññatra ekena piṇḍapātanīhārakena.

Atha kho bhagavā tassa temāsassa accayena paṭisallānā vuṭṭhito bhikkhū āmantesi – ‘‘sace kho, bhikkhave, aññatitthiyā paribbājakā evaṃ puccheyyuṃ – ‘katamenāvuso, vihārena samaṇo gotamo vassāvāsaṃ bahulaṃ vihāsī’ti, evaṃ puṭṭhā tumhe, bhikkhave, tesaṃ aññatitthiyānaṃ paribbājakānaṃ evaṃ byākareyyātha – ‘ānāpānassatisamādhinā kho, āvuso, bhagavā vassāvāsaṃ bahulaṃ vihāsī’ti. Idhāhaṃ, bhikkhave, sato assasāmi, sato passasāmi. Dīghaṃ assasanto ‘dīghaṃ assasāmī’ti pajānāmi, dīghaṃ passasanto ‘dīghaṃ passasāmī’ti pajānāmi; rassaṃ assasanto ‘rassaṃ assasāmī’ti pajānāmi, rassaṃ passasanto ‘rassaṃ passasāmī’ti pajānāmi; ‘sabbakāyappaṭisaṃvedī assasissāmī’ti pajānāmi…pe… ‘paṭinissaggānupassī assasissāmī’ti pajānāmi, ‘paṭinissaggānupassī passasissāmī’ti pajānāmi’’.

### 対訳

1. Icchānaṅgalasuttaṃ

→ 「イッチャーナンガラ経」

1. Ekaṃ samayaṃ bhagavā icchānaṅgale viharati icchānaṅgalavanasaṇḍe.

* ekaṃ samayaṃ = 「ある時」
* bhagavā = 「世尊は」
* icchānaṅgale = 「イッチャーナンガラ（村）に」
* viharati = 「住んでいる」
* icchānaṅgala-vana-saṇḍe = 「イッチャーナンガラの密林に」（vana = 林、saṇḍa = 茂み・群れ）

→ 「ある時、世尊はイッチャーナンガラの、イッチャーナンガラの密林に住んでいた。」

2. Tatra kho bhagavā bhikkhū āmantesi –

* tatra kho = 「そこで」
* bhagavā = 「世尊は」
* bhikkhū = 「比丘たちに」
* āmantesi = 「呼びかけた（āmanteti の過去形）」

→ 「そこで世尊は比丘たちに呼びかけた。」

3. ‘‘icchāmahaṃ, bhikkhave, temāsaṃ paṭisallīyituṃ.

* icchāmi ahaṃ = 「私は望む」
* bhikkhave = 「比丘たちよ」
* temāsaṃ = 「三か月間」（ti + māsa）
* paṭisallīyituṃ = 「独りで坐ること、独坐すること（paṭisallīyati の不定詞）」

→ 「比丘たちよ、私は三か月間、独坐したいと思う。」

4. Nāmhi kenaci upasaṅkamitabbo, aññatra ekena piṇḍapātanīhārakenā’’ti.

* na amhi = 「私は～されるべきでない」
* kenaci = 「誰によっても」
* upasaṅkamitabbo = 「近づかれるべき（upasaṅkamati の未来受動分詞）」
* aññatra = 「～を除いて」
* ekena piṇḍapāta-nīhārakena = 「一人の、托鉢食を運ぶ者を（除いて）」（nīhāraka = 運び出す者）

→ 「托鉢食を運ぶ一人の者を除いて、誰も私のところに来てはならない。」

5. ‘‘Evaṃ, bhante’’ti kho te bhikkhū bhagavato paṭissutvā nāssudha koci bhagavantaṃ upasaṅkamati, aññatra ekena piṇḍapātanīhārakena.

* evaṃ bhante ti = 「『はい、尊師よ』と」
* te bhikkhū = 「かの比丘たちは」
* bhagavato paṭissutvā = 「世尊に応答して（paṭissuṇāti の連続体）」
* na assudha koci = 「実に誰も～ない」（assudha = 強調の不変化辞）
* bhagavantaṃ upasaṅkamati = 「世尊に近づく」
* aññatra ekena piṇḍapāta-nīhārakena = 「托鉢食を運ぶ一人の者を除いて」

→ 「『はい、尊師よ』とかの比丘たちは世尊に応答し、托鉢食を運ぶ一人の者を除いて、実に誰も世尊のところに近づかなかった。」

6. Atha kho bhagavā tassa temāsassa accayena paṭisallānā vuṭṭhito bhikkhū āmantesi –

* atha kho = 「さて」
* tassa temāsassa accayena = 「その三か月が過ぎた後に」（accaya = 経過）
* paṭisallānā vuṭṭhito = 「独坐から出て（vuṭṭhita = 立ち上がった、出定した）」
* bhikkhū āmantesi = 「比丘たちに呼びかけた」

→ 「さて世尊は、その三か月が過ぎた後、独坐から出て、比丘たちに呼びかけた。」

7. ‘‘sace kho, bhikkhave, aññatitthiyā paribbājakā evaṃ puccheyyuṃ –

* sace = 「もし」
* añña-titthiyā paribbājakā = 「他の宗派の遍歴行者たちが」（titthiya = 外道、異教徒）
* evaṃ puccheyyuṃ = 「このように問うならば（pucchati の願望法・複数）」

→ 「比丘たちよ、もし他の宗派の遍歴行者たちがこのように問うならば――」

8. ‘katamenāvuso, vihārena samaṇo gotamo vassāvāsaṃ bahulaṃ vihāsī’ti,

* katamena āvuso vihārena = 「友よ、どのような住によって」
* samaṇo gotamo = 「沙門ゴータマは」
* vassa-āvāsaṃ = 「雨安居を」（vassa = 雨期、āvāsa = 住まい）
* bahulaṃ = 「多く、主に」
* vihāsi = 「住んだ（viharati の過去形）」

→ 「『友よ、沙門ゴータマはどのような住によって雨安居を主に過ごしたのか』と、」

9. evaṃ puṭṭhā tumhe, bhikkhave, tesaṃ aññatitthiyānaṃ paribbājakānaṃ evaṃ byākareyyātha –

* evaṃ puṭṭhā = 「このように問われたならば（puṭṭha = pucchati の過去分詞）」
* tumhe = 「あなたたちは」
* tesaṃ aññatitthiyānaṃ paribbājakānaṃ = 「それら他宗派の遍歴行者たちに」
* evaṃ byākareyyātha = 「このように答えるべきである（byākaroti の願望法・二人称複数）」

→ 「比丘たちよ、このように問われたならば、あなたたちはそれら他宗派の遍歴行者たちにこのように答えるべきである。」

10. ‘ānāpānassatisamādhinā kho, āvuso, bhagavā vassāvāsaṃ bahulaṃ vihāsī’ti.

* ānāpāna-sati-samādhinā = 「出入息念の定によって」（āna = 出息、apāna = 入息、sati = 念、samādhi = 定）
* kho āvuso = 「実に、友よ」
* bhagavā = 「世尊は」
* vassāvāsaṃ bahulaṃ vihāsi = 「雨安居を主に過ごした」

→ 「『友よ、世尊は出入息念の定によって雨安居を主に過ごした』と。」

11. Idhāhaṃ, bhikkhave, sato assasāmi, sato passasāmi.

* idha ahaṃ = 「ここで私は」
* bhikkhave = 「比丘たちよ」
* sato = 「念をそなえて（sata = sarati の過去分詞）」
* assasāmi = 「出息する（息を外へ出す）」
* passasāmi = 「入息する（息を内へ入れる）」

→ 「比丘たちよ、ここで私は、念をそなえて出息し、念をそなえて入息する。」

12. Dīghaṃ assasanto ‘dīghaṃ assasāmī’ti pajānāmi, dīghaṃ passasanto ‘dīghaṃ passasāmī’ti pajānāmi;

* dīghaṃ = 「長く」
* assasanto = 「出息しながら（現在分詞）」
* dīghaṃ assasāmī ti pajānāmi = 「『私は長く出息する』と了知する」（pajānāti = はっきり知る）
* passasanto = 「入息しながら」
* dīghaṃ passasāmī ti pajānāmi = 「『私は長く入息する』と了知する」

→ 「長く出息しながら『私は長く出息する』と了知し、長く入息しながら『私は長く入息する』と了知する。」

13. rassaṃ assasanto ‘rassaṃ assasāmī’ti pajānāmi, rassaṃ passasanto ‘rassaṃ passasāmī’ti pajānāmi;

* rassaṃ = 「短く」
* assasanto ... pajānāmi = 「出息しながら～と了知する」
* passasanto ... pajānāmi = 「入息しながら～と了知する」

→ 「短く出息しながら『私は短く出息する』と了知し、短く入息しながら『私は短く入息する』と了知する。」

14. ‘sabbakāyappaṭisaṃvedī assasissāmī’ti pajānāmi…pe…

* sabba-kāya-paṭisaṃvedī = 「全身を感受する者として」（sabba = すべて、kāya = 身、paṭisaṃvedin = 感受する者）
* assasissāmi = 「私は出息しよう（未来形）」
* ti pajānāmi = 「と了知する」
* …pe… = 「（中略）」

→ 「『全身を感受しながら私は出息しよう』と了知する。（中略）」

15. ‘paṭinissaggānupassī assasissāmī’ti pajānāmi, ‘paṭinissaggānupassī passasissāmī’ti pajānāmi’’.

* paṭinissagga-anupassī = 「捨て去ることを随観する者として」（paṭinissagga = 捨遣、手放すこと、anupassin = 随観する者）
* assasissāmī ti pajānāmi = 「『私は出息しよう』と了知する」
* passasissāmī ti pajānāmi = 「『私は入息しよう』と了知する」

→ 「『捨て去ることを随観しながら私は出息しよう』と了知し、『捨て去ることを随観しながら私は入息しよう』と了知する。」

## 987 (2)

‘‘Yañhi taṃ, bhikkhave, sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipi. Ānāpānassatisamādhiṃ sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipi. Ye te, bhikkhave, bhikkhū sekhā appattamānasā anuttaraṃ yogakkhemaṃ patthayamānā viharanti tesaṃ ānāpānassatisamādhi bhāvito bahulīkato āsavānaṃ khayāya saṃvattati. Ye ca kho te, bhikkhave, bhikkhū arahanto khīṇāsavā vusitavanto katakaraṇīyā ohitabhārā anuppattasadatthā parikkhīṇabhavasaṃyojanā sammadaññā vimuttā tesaṃ ānāpānassatisamādhi bhāvito bahulīkato diṭṭhadhammasukhavihārāya ceva saṃvattati satisampajaññāya ca.

‘‘Yañhi taṃ, bhikkhave, sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipi. Ānāpānassatisamādhiṃ sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipī’’ti. Paṭhamaṃ.

### 対訳

1. Yañhi taṃ, bhikkhave, sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipi.

* yaṃ hi taṃ = 「実に、およそそれについて」

* bhikkhave = 「比丘たちよ」

* sammā vadamāno vadeyya = 「正しく語る者が語るであろう」

  * sammā = 「正しく」

  * vadamāno = 「語りつつある者（vadati の現在分詞）」

  * vadeyya = 「語るであろう（願望法）」

* ariyavihāro itipi = 「『聖なる住（ありよう）』とも」

* brahmavihāro itipi = 「『梵住』とも」

* tathāgatavihāro itipi = 「『如来の住』とも」

→ 「比丘たちよ、実に、正しく語る者が『聖なる住』とも『梵住』とも『如来の住』とも語るであろうもの――」

2. Ānāpānassatisamādhiṃ sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipi.

* ānāpānassatisamādhiṃ = 「出入息念の定（三昧）を」

  * āna = 「出息（外へ出る息）」／apāna = 「入息（内へ入る息）」（律註の定義による向き）

  * sati = 「念」

  * samādhi = 「定、等持」

→ 「正しく語る者は、出入息念の定を『聖なる住』とも『梵住』とも『如来の住』とも語るであろう。」

3. Ye te, bhikkhave, bhikkhū sekhā appattamānasā anuttaraṃ yogakkhemaṃ patthayamānā viharanti

* ye te bhikkhū = 「およそ～である比丘たちは」

* sekhā = 「有学（まだ学ぶべきことの残る者）」

* appattamānasā = 「まだ目的（阿羅漢果）に達していない者」

  * a + patta（到達した）+ mānasa（意、目的）

* anuttaraṃ yogakkhemaṃ = 「無上の軛安穏（束縛からの安穏、涅槃）を」

* patthayamānā viharanti = 「願いながら住している」

→ 「比丘たちよ、有学であり、まだ目的に達しておらず、無上の軛安穏を願いながら住している比丘たち――」

4. tesaṃ ānāpānassatisamādhi bhāvito bahulīkato āsavānaṃ khayāya saṃvattati.

* tesaṃ = 「彼らにとって」

* bhāvito = 「修習された」

* bahulīkato = 「多く実践された」

* āsavānaṃ khayāya = 「諸々の漏の滅尽のために」

* saṃvattati = 「導く、資する」

→ 「彼らにとって、出入息念の定が修習され、多く実践されると、諸々の漏の滅尽に導く。」

5. Ye ca kho te, bhikkhave, bhikkhū arahanto khīṇāsavā vusitavanto katakaraṇīyā ohitabhārā anuppattasadatthā parikkhīṇabhavasaṃyojanā sammadaññā vimuttā

* ye ca kho te bhikkhū = 「また、およそ～である比丘たちは」

* arahanto = 「阿羅漢であり」

* khīṇāsavā = 「漏が尽きた者であり」

* vusitavanto = 「（梵行を）住み終えた者であり」

* katakaraṇīyā = 「なすべきことをなし終えた者であり」

* ohitabhārā = 「重荷を下ろした者であり」

* anuppattasadatthā = 「自らの目的に到達した者であり」

  * anuppatta = 「到達した」＋ sadattha = 「自らの目的（saka + attha）」

* parikkhīṇabhavasaṃyojanā = 「有（存在）への結を滅ぼし尽くした者であり」

* sammadaññā vimuttā = 「正しく了知して解脱した者である」

→ 「比丘たちよ、また、阿羅漢であり、漏が尽き、（梵行を）住み終え、なすべきことをなし終え、重荷を下ろし、自らの目的に到達し、有への結を滅ぼし尽くし、正しく了知して解脱した比丘たち――」

6. tesaṃ ānāpānassatisamādhi bhāvito bahulīkato diṭṭhadhammasukhavihārāya ceva saṃvattati satisampajaññāya ca.

* tesaṃ = 「彼らにとって」

* diṭṭhadhammasukhavihārāya = 「現法楽住（今この世での安楽な生活）のために」

  * diṭṭhadhamma = 「現法、今この世」＋ sukha-vihāra = 「安楽な住」

* ceva ... ca = 「～と～の両方に」

* satisampajaññāya = 「念と正知のために」

→ 「彼らにとって、出入息念の定が修習され、多く実践されると、現法楽住と、念と正知とに導く。」

7. Yañhi taṃ, bhikkhave, sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipi. Ānāpānassatisamādhiṃ sammā vadamāno vadeyya – ‘ariyavihāro’ itipi, ‘brahmavihāro’ itipi, ‘tathāgatavihāro’ itipī ti.

* （第1・第2チャンクの繰り返し。結びの ti が付く）

→ 「比丘たちよ、実に、正しく語る者が『聖なる住』とも『梵住』とも『如来の住』とも語るであろうもの――正しく語る者は、出入息念の定を『聖なる住』とも『梵住』とも『如来の住』とも語るであろう」と。

8. Paṭhamaṃ.

* paṭhamaṃ = 「第一（の経、終わる）」

→ 「第一（経）。」

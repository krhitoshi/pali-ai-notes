
# 3. Suddhikasuttaṃ

## 979

979. ‘‘Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā? Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā’’ti. Tatiyaṃ.

### 対訳

3. Suddhikasuttaṃ

→ 「清浄経」(第三経)

1. Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā.

* ānāpānassati = 「出入息念(呼吸への念)」
  * āna = 「出息」(方針: assāsa=出息に対応)
  * apāna = 「入息」
  * sati = 「念、気づき」
* bhikkhave = 「比丘たちよ」(呼格・複数)
* bhāvitā = 「修習された」(bhāveti の過去分詞・女性)
* bahulīkatā = 「多く実践された、繰り返し行われた」
* mahapphalā = 「大きな果報がある」(mahā + phala)
* hoti = 「~である」
* mahānisaṃsā = 「大きな利益がある」(mahā + ānisaṃsa)

→ 「比丘たちよ、出入息念は、修習され、多く実践されると、大きな果報があり、大きな利益がある。」

2. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā?

* kathaṃ = 「どのように」
* bhāvitā = 「修習された」
* ca = 「そして」
* kathaṃ bahulīkatā = 「どのように多く実践された(とき)」

→ 「では比丘たちよ、出入息念はどのように修習され、どのように多く実践されると、大きな果報があり、大きな利益があるのか。」

3. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに(この教えにおいて)」
* bhikkhu = 「比丘は」
* araññagato = 「森に行った(者)」(arañña + gata)
* vā ... vā ... vā = 「あるいは~あるいは~」
* rukkhamūlagato = 「木の根元に行った(者)」(rukkha「木」+ mūla「根元」+ gata)
* suññāgāragato = 「空き家に行った(者)」(suñña「空の」+ agāra「家」+ gata)
* nisīdati = 「坐る」
* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」
  * pallaṅka = 「結跏趺坐」
  * ābhujitvā = 「組んで」(ābhujati の連続体)
* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに定めて」
  * uju = 「まっすぐな」
  * kāya = 「身体」
  * paṇidhāya = 「定めて、置いて」(paṇidahati の連続体)
* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に確立して」
  * parimukhaṃ = 「面前に、鼻先のあたりに」
  * satiṃ = 「念を」
  * upaṭṭhapetvā = 「確立して、現前させて」(upaṭṭhāpeti の連続体)

→ 「比丘たちよ、ここに比丘は、森に行き、あるいは木の根元に行き、あるいは空き家に行って、結跏趺坐を組み、身体をまっすぐに定め、念を面前に確立して坐る。」

4. So satova assasati, satova passasati …pe…

* so = 「彼は」
* sato va = 「まさに念をもって」
  * sato = 「念のある(者)」
  * va (eva) = 「まさに、こそ」
* assasati = 「出息する(息を外に出す)」
* passasati = 「入息する(息を内に入れる)」
* …pe… = 「(中略)」(十六事の定型句が省略されている)

→ 「彼はまさに念をもって出息し、まさに念をもって入息する。(中略)」

5. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissaggānupassī = 「捨遣を随観する者(として)」
  * paṭinissagga = 「捨遣、手放すこと」
  * anupassī = 「随観する者」
* assasissāmi = 「私は出息しよう」(assasati の未来・一人称単数)
* passasissāmi = 「私は入息しよう」(passasati の未来・一人称単数)
* ti = 「~と」(引用)
* sikkhati = 「学ぶ、訓練する」

→ 「『捨遣を随観しながら出息しよう』と学び、『捨遣を随観しながら入息しよう』と学ぶ。」

6. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā’’ti. Tatiyaṃ.

* evaṃ = 「このように」
* bhāvitā = 「修習された」
* kho = 「実に」(強調の小辞)
* evaṃ bahulīkatā = 「このように多く実践された(とき)」
* mahapphalā hoti mahānisaṃsā = 「大きな果報があり、大きな利益がある」
* ti = 「~と」(経の結び)
* tatiyaṃ = 「第三(経)」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念は、大きな果報があり、大きな利益がある。」第三(経)。

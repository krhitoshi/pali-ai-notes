
# 6. Ariṭṭhasuttaṃ

## 982

982. Sāvatthinidānaṃ. Tatra kho bhagavā…pe… etadavoca – ‘‘bhāvetha no tumhe bhikkhave, ānāpānassati’’nti? Evaṃ vutte āyasmā ariṭṭho bhagavantaṃ etadavoca – ‘‘ahaṃ kho, bhante, bhāvemi ānāpānassati’’nti. ‘‘Yathā kathaṃ pana tvaṃ, ariṭṭha, bhāvesi ānāpānassati’’nti? ‘‘Atītesu me, bhante, kāmesu kāmacchando pahīno, anāgatesu me kāmesu kāmacchando vigato, ajjhattabahiddhā [ajjhattaṃ bahiddhā (syā. kaṃ. pī. ka.)] ca me dhammesu paṭighasaññā suppaṭivinītā. So [sohaṃ (?)] satova assasissāmi, satova passasissāmi. Evaṃ khvāhaṃ, bhante, bhāvemi ānāpānassati’’nti.

‘‘‘Atthesā, ariṭṭha, ānāpānassati, nesā natthī’ti vadāmi. Api ca, ariṭṭha, yathā ānāpānassati vitthārena paripuṇṇā hoti taṃ suṇāhi, sādhukaṃ manasi karohi; bhāsissāmī’’ti. ‘‘Evaṃ, bhante’’ti kho āyasmā ariṭṭho bhagavato paccassosi. Bhagavā etadavoca –

‘‘Kathañca, ariṭṭha, ānāpānassati vitthārena paripuṇṇā hoti? Idha, ariṭṭha, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ kho, ariṭṭha, ānāpānassati vitthārena paripuṇṇā hotī’’ti. Chaṭṭhaṃ.

### 対訳

6. Ariṭṭhasuttaṃ

→ 「アリッタ経」

1. Sāvatthinidānaṃ.

* Sāvatthi = 「サーヴァッティー（舎衛城）」

* nidānaṃ = 「因縁、由来」

→ 「サーヴァッティーでの因縁（この経はサーヴァッティーで説かれた）。」

2. Tatra kho bhagavā…pe… etadavoca –

* tatra = 「そこで」

* kho = 「実に（強調の不変化詞）」

* bhagavā = 「世尊は」

* …pe… = 「（中略）」

* etad avoca = 「このように言った」

→ 「そこで世尊は……（中略）……このように言った。」

3. ‘‘bhāvetha no tumhe bhikkhave, ānāpānassati’’nti?

* bhāvetha = 「修習するか（bhāveti の現在二人称複数）」

* no = 「（疑問を示す不変化詞）～かどうか」

* tumhe = 「あなたたちは」

* bhikkhave = 「比丘たちよ（呼格）」

* ānāpānassatiṃ = 「出入息念を」

  * āna = 「出息」、apāna = 「入息」、sati = 「念」

* ti = 「～と（引用）」

→ 「『比丘たちよ、あなたたちは出入息念を修習しているか』と。」

4. Evaṃ vutte āyasmā ariṭṭho bhagavantaṃ etadavoca –

* evaṃ vutte = 「このように言われたとき（絶対処格）」

* āyasmā = 「尊者」

* ariṭṭho = 「アリッタは」

* bhagavantaṃ = 「世尊に」

* etad avoca = 「このように言った」

→ 「このように言われたとき、尊者アリッタは世尊にこのように言った。」

5. ‘‘ahaṃ kho, bhante, bhāvemi ānāpānassati’’nti.

* ahaṃ = 「私は」

* kho = 「実に」

* bhante = 「尊師よ（呼格）」

* bhāvemi = 「修習しています（現在一人称単数）」

* ānāpānassatiṃ = 「出入息念を」

→ 「『尊師よ、私は出入息念を修習しています』と。」

6. ‘‘Yathā kathaṃ pana tvaṃ, ariṭṭha, bhāvesi ānāpānassati’’nti?

* yathā kathaṃ = 「どのように」

* pana = 「では、しかし」

* tvaṃ = 「あなたは」

* ariṭṭha = 「アリッタよ（呼格）」

* bhāvesi = 「修習しているのか（現在二人称単数）」

* ānāpānassatiṃ = 「出入息念を」

→ 「『では、アリッタよ、あなたはどのように出入息念を修習しているのか』と。」

7. ‘‘Atītesu me, bhante, kāmesu kāmacchando pahīno,

* atītesu = 「過去の（処格複数）」

* me = 「私の」

* bhante = 「尊師よ」

* kāmesu = 「諸々の欲の対象に対して」

* kāmacchando = 「欲への意欲（欲貪）は」

* pahīno = 「捨てられた（pajahati の過去分詞）」

→ 「『尊師よ、私には過去の諸々の欲の対象に対する欲貪は捨てられ、」

8. anāgatesu me kāmesu kāmacchando vigato,

* anāgatesu = 「未来の（処格複数）」

* kāmesu = 「諸々の欲の対象に対して」

* kāmacchando = 「欲貪は」

* vigato = 「離れ去った（vigacchati の過去分詞）」

→ 「未来の諸々の欲の対象に対する欲貪は離れ去り、」

9. ajjhattabahiddhā ca me dhammesu paṭighasaññā suppaṭivinītā.

* ajjhatta-bahiddhā = 「内と外の」

* ca = 「そして」

* me = 「私の」

* dhammesu = 「諸法に対して」

* paṭigha-saññā = 「瞋恚の想は」

  * paṭigha = 「瞋恚、反発」、saññā = 「想」

* suppaṭivinītā = 「よく除き去られた（su + paṭivineti の過去分詞）」

→ 「そして内と外の諸法に対する瞋恚の想はよく除き去られています。」

10. So satova assasissāmi, satova passasissāmi.

* so = 「その私は」

* sato va = 「念をそなえたままで（sata = 念ある者、eva = まさに）」

* assasissāmi = 「出息するでしょう（未来一人称単数）」

* passasissāmi = 「入息するでしょう（未来一人称単数）」

→ 「その私は、念をそなえたままで出息し、念をそなえたままで入息します。」

11. Evaṃ khvāhaṃ, bhante, bhāvemi ānāpānassati’’nti.

* evaṃ = 「このように」

* kho ahaṃ = 「実に私は（khvāhaṃ は連声）」

* bhante = 「尊師よ」

* bhāvemi = 「修習しています」

* ānāpānassatiṃ = 「出入息念を」

→ 「尊師よ、このように私は出入息念を修習しています』と。」

12. ‘‘‘Atthesā, ariṭṭha, ānāpānassati, nesā natthī’ti vadāmi.

* atthi esā = 「これは存在する（atthesā は連声）」

* ariṭṭha = 「アリッタよ」

* ānāpānassati = 「出入息念は」

* na esā natthi = 「これは存在しないのではない」

* ti vadāmi = 「～と私は言う」

→ 「『アリッタよ、それも出入息念であり、それが出入息念でないとは私は言わない。」

13. Api ca, ariṭṭha, yathā ānāpānassati vitthārena paripuṇṇā hoti taṃ suṇāhi, sādhukaṃ manasi karohi; bhāsissāmī’’ti.

* api ca = 「しかしながら」

* yathā = 「どのように～か」

* ānāpānassati = 「出入息念が」

* vitthārena = 「詳細に」

* paripuṇṇā hoti = 「完成される（paripuṇṇa = 円満な、完全な）」

* taṃ = 「それを」

* suṇāhi = 「聞きなさい（命令法二人称単数）」

* sādhukaṃ = 「よく」

* manasi karohi = 「心に留めなさい（作意しなさい）」

* bhāsissāmi = 「私は説くでしょう」

* ti = 「～と」

→ 「しかしながら、アリッタよ、どのように出入息念が詳細に完成されるか、それを聞きなさい。よく心に留めなさい。私は説こう』と。」

14. ‘‘Evaṃ, bhante’’ti kho āyasmā ariṭṭho bhagavato paccassosi.

* evaṃ bhante ti = 「『はい、尊師よ』と」

* kho = 「実に」

* āyasmā ariṭṭho = 「尊者アリッタは」

* bhagavato = 「世尊に」

* paccassosi = 「応答した（paṭissuṇāti のアオリスト）」

→ 「『はい、尊師よ』と尊者アリッタは世尊に応答した。」

15. Bhagavā etadavoca –

* bhagavā = 「世尊は」

* etad avoca = 「このように言った」

→ 「世尊はこのように言った。」

16. ‘‘Kathañca, ariṭṭha, ānāpānassati vitthārena paripuṇṇā hoti?

* kathaṃ ca = 「では、どのように」

* ānāpānassati = 「出入息念が」

* vitthārena = 「詳細に」

* paripuṇṇā hoti = 「完成されるのか」

→ 「『では、アリッタよ、どのように出入息念が詳細に完成されるのか。」

17. Idha, ariṭṭha, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati

* idha = 「ここに（この教えにおいて）」

* bhikkhu = 「比丘が」

* arañña-gato vā = 「森に行って、あるいは」

* rukkhamūla-gato vā = 「樹の根もとに行って、あるいは」

* suññāgāra-gato vā = 「空き家に行って、あるいは」

  * suñña = 「空の」、agāra = 「家」

* nisīdati = 「坐る」

→ 「アリッタよ、ここに比丘が森に行き、あるいは樹の根もとに行き、あるいは空き家に行って坐り、」

18. pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」

* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに定めて」

  * uju = 「まっすぐな」、paṇidhāya = 「定めて（paṇidahati の連続体）」

* parimukhaṃ = 「面前に」

* satiṃ upaṭṭhapetvā = 「念を確立して（upaṭṭhapeti の連続体）」

→ 「結跏趺坐を組み、身体をまっすぐに定め、面前に念を確立する。」

19. So satova assasati, satova passasati.

* so = 「彼は」

* sato va = 「念をそなえたままで」

* assasati = 「出息する」

* passasati = 「入息する」

→ 「彼は、念をそなえたままで出息し、念をそなえたままで入息する。」

20. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti…pe…

* dīghaṃ = 「長く」

* vā = 「あるいは」

* assasanto = 「出息しながら（現在分詞）」

* ‘dīghaṃ assasāmi’ ti = 「『私は長く出息している』と」

* pajānāti = 「はっきりと知る」

* …pe… = 「（中略）」

→ 「あるいは長く出息しながら『私は長く出息している』とはっきりと知る……（中略）……」

21. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissagga-anupassī = 「捨遣を随観する者として」

  * paṭinissagga = 「捨遣、放棄」、anupassin = 「随観する者」

* assasissāmi = 「出息しよう」

* passasissāmi = 「入息しよう」

* ti sikkhati = 「～と学ぶ」

→ 「『捨遣を随観しながら出息しよう』と学び、『捨遣を随観しながら入息しよう』と学ぶ。」

22. Evaṃ kho, ariṭṭha, ānāpānassati vitthārena paripuṇṇā hotī’’ti. Chaṭṭhaṃ.

* evaṃ kho = 「このようにして」

* ānāpānassati = 「出入息念が」

* vitthārena = 「詳細に」

* paripuṇṇā hoti = 「完成される」

* ti = 「～と（経の結び）」

* chaṭṭhaṃ = 「第六（経）」

→ 「アリッタよ、このようにして出入息念が詳細に完成されるのである』と。第六経。」

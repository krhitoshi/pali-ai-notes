
# 8. Padīpopamasuttaṃ

## 984 (1)

984. ‘‘Ānāpānassatisamādhi , bhikkhave, bhāvito bahulīkato mahapphalo hoti mahānisaṃso. Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso?

‘‘Idha , bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvito kho, bhikkhave, ānāpānassatisamādhi evaṃ bahulīkato mahapphalo hoti mahānisaṃso.

‘‘Ahampi sudaṃ, bhikkhave, pubbeva sambodhā anabhisambuddho bodhisattova samāno iminā vihārena bahulaṃ viharāmi. Tassa mayhaṃ, bhikkhave, iminā vihārena bahulaṃ viharato neva kāyo kilamati na cakkhūni; anupādāya ca me āsavehi cittaṃ vimucci.

8. Padīpopamasuttaṃ

→ 「灯明の譬えの経」

984.

1. Ānāpānassatisamādhi, bhikkhave, bhāvito bahulīkato mahapphalo hoti mahānisaṃso.

* ānāpānassati-samādhi = 「出入息念による定」
  * ānāpāna = 「出入息（āna = 出息、apāna = 入息）」
  * sati = 「念」
  * samādhi = 「定、等持」
* bhikkhave = 「比丘たちよ（呼格・複数）」
* bhāvito = 「修習された（bhāveti の過去分詞）」
* bahulīkato = 「多く実践された、繰り返し行われた」
* mahapphalo = 「大きな果報のある（mahā + phala）」
* hoti = 「～である」
* mahānisaṃso = 「大きな利益のある（mahā + ānisaṃsa）」

→ 「比丘たちよ、出入息念の定は、修習され、多く実践されると、大きな果報があり、大きな利益がある。」

2. Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso?

* kathaṃ = 「どのように」
* bhāvito = 「修習され」
* bahulīkato = 「多く実践され」

→ 「比丘たちよ、では出入息念の定は、どのように修習され、どのように多く実践されると、大きな果報があり、大きな利益があるのか。」

3. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘は」
* araññagato vā = 「林に行き、あるいは」
* rukkhamūlagato vā = 「木の根元に行き、あるいは」
* suññāgāragato vā = 「空き家に行き、あるいは」
* nisīdati = 「坐る」
* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」
* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに保って」
  * ujuṃ = 「まっすぐに」
  * kāyaṃ = 「身体を」
  * paṇidhāya = 「定め置いて（paṇidahati の連続体）」
* parimukhaṃ satiṃ upaṭṭhapetvā = 「面前に念を確立して」
  * parimukhaṃ = 「面前に、（業処の）まわりに」
  * satiṃ = 「念を」
  * upaṭṭhapetvā = 「確立して、現前させて（upaṭṭhāpeti の連続体）」

→ 「比丘たちよ、ここに比丘は、林に行き、あるいは木の根元に行き、あるいは空き家に行き、結跏趺坐を組み、身体をまっすぐに保ち、面前に念を確立して坐る。」

4. So satova assasati, satova passasati.

* so = 「彼は」
* sato va = 「念をそなえた者としてまさに（sata = 念のある者、eva の連声形 va）」
* assasati = 「出息する（息を外へ出す）」
* passasati = 「入息する（息を内へ入れる）」

→ 「彼は念をそなえてまさに出息し、念をそなえてまさに入息する。」

5. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti…pe…

* dīghaṃ vā assasanto = 「あるいは長く出息しながら」
  * dīghaṃ = 「長く」
  * assasanto = 「出息している（現在分詞）」
* ‘dīghaṃ assasāmī’ti = 「『私は長く出息している』と」
* pajānāti = 「知る、了知する」
* …pe… = 「（中略）」

→ 「あるいは長く出息しながら『私は長く出息している』と知る……（中略）……」

6. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissagga-anupassī = 「捨遣を随観する者として」
  * paṭinissagga = 「捨遣、放棄」
  * anupassī = 「随観する者」
* assasissāmī’ti = 「『私は出息しよう』と（未来形）」
* passasissāmī’ti = 「『私は入息しよう』と（未来形）」
* sikkhati = 「学ぶ、修学する」

→ 「『捨遣を随観しながら私は出息しよう』と学び、『捨遣を随観しながら私は入息しよう』と学ぶ。」

7. Evaṃ bhāvito kho, bhikkhave, ānāpānassatisamādhi evaṃ bahulīkato mahapphalo hoti mahānisaṃso.

* evaṃ = 「このように」
* kho = 「実に（強意の小辞）」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念の定は、大きな果報があり、大きな利益がある。」

8. Ahampi sudaṃ, bhikkhave, pubbeva sambodhā anabhisambuddho bodhisattova samāno iminā vihārena bahulaṃ viharāmi.

* ahampi = 「私もまた（ahaṃ + pi）」
* sudaṃ = 「実に（強意の小辞）」
* pubbeva sambodhā = 「正覚の以前に（pubbe + eva + sambodhā 従格）」
* anabhisambuddho = 「まだ完全に覚っていない者として」
* bodhisatto va samāno = 「まさに菩薩であったとき」
  * bodhisatto = 「菩薩」
  * samāno = 「～である者として（atthi の現在分詞）」
* iminā vihārena = 「この住（この出入息念の住まい方）によって」
* bahulaṃ = 「多く、しばしば」
* viharāmi = 「住していた（歴史的現在）」

→ 「比丘たちよ、私もまた正覚の以前、まだ完全に覚っておらず、まさに菩薩であったとき、この住によって多く住していた。」

9. Tassa mayhaṃ, bhikkhave, iminā vihārena bahulaṃ viharato neva kāyo kilamati na cakkhūni;

* tassa mayhaṃ = 「その私にとって（属格）」
* iminā vihārena bahulaṃ viharato = 「この住によって多く住している（者にとって）」
  * viharato = 「住している（現在分詞・属格）」
* neva = 「決して～ない（na + eva）」
* kāyo = 「身体は」
* kilamati = 「疲れる」
* na cakkhūni = 「眼も（疲れ）ない」
  * cakkhūni = 「両眼（複数）」

→ 「比丘たちよ、この住によって多く住していた私には、身体も疲れず、眼も疲れなかった。」

10. anupādāya ca me āsavehi cittaṃ vimucci.

* anupādāya = 「取著することなく（an + upādāya）」
* ca = 「そして」
* me = 「私の」
* āsavehi = 「諸々の漏から（従格・複数）」
* cittaṃ = 「心は」
* vimucci = 「解脱した（vimuccati のアオリスト）」

→ 「そして私の心は、取著することなく諸々の漏から解脱した。」

### 対訳

1. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya –

* tasmātiha = 「それゆえ、ここに」（tasmā = それゆえ、iha = ここに）

* bhikkhave = 「比丘たちよ」（呼格・複数）

* bhikkhu = 「比丘が」

* ce pi = 「もし～ならば」

* ākaṅkheyya = 「願うならば」（ākaṅkhati の願望法・三人称単数）

→ 「それゆえここに、比丘たちよ、もし比丘が～と願うならば」

2. ‘neva me kāyo kilameyya na cakkhūni, anupādāya ca me āsavehi cittaṃ vimucceyyā’ti,

* na eva ... na = 「～も～もないように」

* me = 「私の」

* kāyo = 「身体が」

* kilameyya = 「疲れるように」（kilamati の願望法）

* cakkhūni = 「両眼が」（cakkhu の複数）

* anupādāya = 「取著せずに」（an + upādāya、upādiyati の連続体）

* ca = 「そして」

* āsavehi = 「諸々の漏から」（āsava の奪格・複数）

* cittaṃ = 「心が」

* vimucceyya = 「解脱するように」（vimuccati の願望法）

* ti = 「～と」（引用の終わり）

→ 「『私の身体も両眼も疲れることなく、そして取著することなく、私の心が諸々の漏から解脱するように』と」

3. ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* ayam eva = 「まさにこの」

* ānāpānassati-samādhi = 「出入息念の定が」

  * āna = 「出息」、apāna = 「入息」、sati = 「念」、samādhi = 「定」

* sādhukaṃ = 「よく、十分に」

* manasi kātabbo = 「注意されるべきである」（manasi karoti「作意する」の義務分詞）

→ 「まさにこの出入息念の定が、よく作意されるべきである。」

4. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya – ‘ye me gehasitā sarasaṅkappā te pahīyeyyu’nti, ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* ye ... te = 「～であるものは、それらが」（関係代名詞の構文）

* me = 「私の」

* gehasitā = 「在家（世俗）に依存した」（geha = 家、sita = 依った）

* sara-saṅkappā = 「記憶と思考が」

  * sara = 「記憶、想起」

  * saṅkappa = 「思考、思惟」

* pahīyeyyuṃ = 「捨てられるように」（pahīyati の願望法・三人称複数）

→ 「それゆえここに、比丘たちよ、もし比丘が『私の在家に依存した記憶と思考が捨てられるように』と願うならば、まさにこの出入息念の定が、よく作意されるべきである。」

5. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya – ‘appaṭikūle paṭikūlasaññī vihareyya’nti, ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* appaṭikūle = 「厭わしくないものにおいて」（a + paṭikūla の処格）

* paṭikūla-saññī = 「厭わしいという想を持つ者として」

  * paṭikūla = 「厭わしいもの」

  * saññī = 「想を持つ者」（saññā から）

* vihareyyaṃ = 「私は住むように」（viharati の願望法・一人称単数。vihareyyan ti = vihareyyaṃ + ti）

→ 「それゆえここに、比丘たちよ、もし比丘が『厭わしくないものにおいて、厭わしいという想を持って住むように』と願うならば、まさにこの出入息念の定が、よく作意されるべきである。」

6. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya – ‘paṭikūle appaṭikūlasaññī vihareyya’nti, ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* paṭikūle = 「厭わしいものにおいて」（処格）

* appaṭikūla-saññī = 「厭わしくないという想を持つ者として」

→ 「それゆえここに、比丘たちよ、もし比丘が『厭わしいものにおいて、厭わしくないという想を持って住むように』と願うならば、まさにこの出入息念の定が、よく作意されるべきである。」

7. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya – ‘paṭikūle ca appaṭikūle ca paṭikūlasaññī vihareyya’nti, ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* paṭikūle ca appaṭikūle ca = 「厭わしいものにおいても、厭わしくないものにおいても」

* paṭikūla-saññī = 「厭わしいという想を持つ者として」

→ 「それゆえここに、比丘たちよ、もし比丘が『厭わしいものにおいても厭わしくないものにおいても、厭わしいという想を持って住むように』と願うならば、まさにこの出入息念の定が、よく作意されるべきである。」

8. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya – ‘paṭikūle ca appaṭikūle ca appaṭikūlasaññī vihareyya’nti, ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* paṭikūle ca appaṭikūle ca = 「厭わしいものにおいても、厭わしくないものにおいても」

* appaṭikūla-saññī = 「厭わしくないという想を持つ者として」

→ 「それゆえここに、比丘たちよ、もし比丘が『厭わしいものにおいても厭わしくないものにおいても、厭わしくないという想を持って住むように』と願うならば、まさにこの出入息念の定が、よく作意されるべきである。」

9. Tasmātiha, bhikkhave, bhikkhu cepi ākaṅkheyya – ‘appaṭikūlañca paṭikūlañca tadubhayaṃ abhinivajjetvā upekkhako vihareyyaṃ sato sampajāno’ti, ayameva ānāpānassatisamādhi sādhukaṃ manasi kātabbo.

* appaṭikūlañ ca paṭikūlañ ca = 「厭わしくないものも、厭わしいものも」

* tad-ubhayaṃ = 「その両者を」（tad = それ、ubhaya = 両方）

* abhinivajjetvā = 「避けて、退けて」（abhinivajjeti の連続体）

* upekkhako = 「捨（平静）を保つ者として」（upekkhā から）

* vihareyyaṃ = 「私は住むように」（願望法・一人称単数）

* sato = 「念を持ち」（sata、sarati の過去分詞）

* sampajāno = 「正知を持って」

→ 「それゆえここに、比丘たちよ、もし比丘が『厭わしくないものも厭わしいものも、その両者を避けて、念と正知を持ち、捨（平静）を保って住むように』と願うならば、まさにこの出入息念の定が、よく作意されるべきである。」

* vaṭṭiñca paṭicca = 「灯芯に縁って（vaṭṭi = 灯芯）」

* tela-ppadīpo = 「油灯が（padīpa = 灯火）」

* jhāyeyya = 「燃えるだろう（jhāyati の願望法。ここでは「燃える」の意）」

→ 「比丘たちよ、たとえば油に縁り、灯芯に縁って油灯が燃えるとして、」

11. tasseva telassa ca vaṭṭiyā ca pariyādānā anāhāro nibbāyeyya;

* tasseva = 「まさにその（tassa + eva）」

* telassa ca vaṭṭiyā ca pariyādānā = 「油と灯芯の消尽によって（奪格）」

* anāhāro = 「食（燃料）がなくなって（an + āhāra = 食、供給）」

* nibbāyeyya = 「消えるだろう（nibbāyati の願望法）」

→ 「まさにその油と灯芯が尽きることによって、燃料がなくなり消えるだろう。」

12. evameva kho, bhikkhave, bhikkhu kāyapariyantikaṃ vedanaṃ vedayamāno ‘kāyapariyantikaṃ vedanaṃ vedayāmī’ti pajānāti,

* evameva kho = 「まさにそのように」

* bhikkhu = 「比丘は」

→ 「比丘たちよ、まさにそのように、比丘は身体を限りとする受を感受しつつ、『私は身体を限りとする受を感受する』と了知し、」

13. jīvitapariyantikaṃ vedanaṃ vedayamāno ‘jīvitapariyantikaṃ vedanaṃ vedayāmī’ti pajānāti,

→ 「命を限りとする受を感受しつつ、『私は命を限りとする受を感受する』と了知し、」

14. ‘kāyassa bhedā uddhaṃ jīvitapariyādānā idheva sabbavedayitāni anabhinanditāni sītībhavissantī’ti pajānātī ti.

* ti = 「～と（経文の引用終わりを示す）」

→ 「『身体の破壊の後、命が尽きたとき、まさにここですべての感受されたものは歓喜されることなく冷たくなるだろう』と了知する、と。」

15. Aṭṭhamaṃ.

* aṭṭhamaṃ = 「第八（経）」

→ 「第八（経、終わり）。」

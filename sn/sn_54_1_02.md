
# 2. Bojjhaṅgasuttaṃ

## 978

978. ‘‘Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā? Idha, bhikkhave, bhikkhu ānāpānassatisahagataṃ satisambojjhaṅgaṃ bhāveti vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ, ānāpānassatisahagataṃ dhammavicayasambojjhaṅgaṃ bhāveti…pe… ānāpānassatisahagataṃ upekkhāsambojjhaṅgaṃ bhāveti vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā’’ti. Dutiyaṃ.

### 対訳

2. Bojjhaṅgasuttaṃ

→ 「覚支経」

1. Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā.

* ānāpānassati = 「出入息念（呼吸への念）」
  * āna = 「出息」/ apāna = 「入息」/ sati = 「念」
* bhikkhave = 「比丘たちよ（呼格・複数）」
* bhāvitā = 「修習された（bhāveti の過去分詞・女性）」
* bahulīkatā = 「多く実践された、繰り返し行われた」
* mahapphalā = 「大きな果報がある」
* hoti = 「～となる」
* mahānisaṃsā = 「大きな利益がある」

→ 「比丘たちよ、出入息念は、修習され、多く実践されると、大きな果報があり、大きな利益がある。」

2. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā?

* kathaṃ = 「どのように」
* bhāvitā ca = 「修習され」
* kathaṃ bahulīkatā = 「どのように多く実践されると」
* mahapphalā hoti mahānisaṃsā = 「大きな果報があり、大きな利益があるのか」

→ 「では比丘たちよ、出入息念はどのように修習され、どのように多く実践されると、大きな果報があり、大きな利益があるのか。」

3. Idha, bhikkhave, bhikkhu ānāpānassatisahagataṃ satisambojjhaṅgaṃ bhāveti vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ,

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘は」
* ānāpānassati-sahagataṃ = 「出入息念を伴った」
  * sahagata = 「共に行く、伴った（saha + gata）」
* sati-sambojjhaṅgaṃ = 「念覚支を」
* bhāveti = 「修習する」
* viveka-nissitaṃ = 「遠離に依った」
  * viveka = 「遠離」/ nissita = 「依った、依止した」
* virāga-nissitaṃ = 「離貪に依った」
* nirodha-nissitaṃ = 「滅に依った」
* vossagga-pariṇāmiṃ = 「捨遣へと向かう」
  * vossagga = 「捨遣、手放すこと」/ pariṇāmin = 「向かう、変化していく」

→ 「比丘たちよ、ここに比丘は、遠離に依り、離貪に依り、滅に依り、捨遣へと向かう、出入息念を伴った念覚支を修習する。」

4. ānāpānassatisahagataṃ dhammavicayasambojjhaṅgaṃ bhāveti…pe…

* dhammavicaya-sambojjhaṅgaṃ = 「択法覚支を」
  * dhamma = 「法」/ vicaya = 「調べること、簡択」
* …pe… = 「（中略）」— 精進・喜・軽安・定の各覚支が省略されている

→ 「出入息念を伴った択法覚支を修習する……（中略：精進覚支・喜覚支・軽安覚支・定覚支）……」

5. ānāpānassatisahagataṃ upekkhāsambojjhaṅgaṃ bhāveti vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ.

* upekkhā-sambojjhaṅgaṃ = 「捨覚支を」

→ 「遠離に依り、離貪に依り、滅に依り、捨遣へと向かう、出入息念を伴った捨覚支を修習する。」

6. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā’’ti. Dutiyaṃ.

* evaṃ = 「このように」
* bhāvitā kho = 「修習され」
* evaṃ bahulīkatā = 「このように多く実践されると」
* mahapphalā hoti mahānisaṃsā = 「大きな果報があり、大きな利益がある」
* ti = 「～と（引用の終わり）」
* dutiyaṃ = 「第二（経）」

→ 「比丘たちよ、このように修習され、このように多く実践されると、出入息念は大きな果報があり、大きな利益がある。」第二（経）。

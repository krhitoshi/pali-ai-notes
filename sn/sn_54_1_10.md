
# 10. Kimilasuttaṃ

## 986 (1)

986. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā kimilāyaṃ [kimbilāyaṃ (sī. pī.)] viharati veḷuvane. Tatra kho bhagavā āyasmantaṃ kimilaṃ āmantesi – ‘‘kathaṃ bhāvito nu kho, kimila, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso’’ti?

Evaṃ vutte āyasmā kimilo tuṇhī ahosi. Dutiyampi kho bhagavā…pe… tatiyampi kho bhagavā āyasmantaṃ kimilaṃ āmantesi – ‘‘kathaṃ bhāvito nu kho, kimila, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso’’ti? Tatiyampi kho āyasmā kimilo tuṇhī ahosi.

Evaṃ vutte āyasmā ānando bhagavantaṃ etadavoca – ‘‘etassa, bhagavā, kālo; etassa, sugata, kālo! Yaṃ bhagavā ānāpānassatisamādhiṃ bhāseyya . Bhagavato sutvā bhikkhū dhāressantī’’ti.

### 対訳

10. Kimilasuttaṃ

* kimila = 「キミラ(人名・地名)」

* suttaṃ = 「経」

→ 「キミラ経」

1. Evaṃ me sutaṃ – ekaṃ samayaṃ bhagavā kimilāyaṃ [kimbilāyaṃ (sī. pī.)] viharati veḷuvane.

* evaṃ = 「このように」

* me = 「私によって」

* sutaṃ = 「聞かれた(su の過去分詞)」

* ekaṃ samayaṃ = 「あるとき」

* bhagavā = 「世尊は」

* kimilāyaṃ = 「キミラー(という町)において」(異読: kimbilāyaṃ「キンビラー」)

* viharati = 「住んでいた(現在形だが歴史的現在)」

* veḷuvane = 「竹林において」(veḷu = 竹, vana = 林)

→ 「このように私は聞いた――あるとき世尊はキミラーの竹林に住んでいた。」

2. Tatra kho bhagavā āyasmantaṃ kimilaṃ āmantesi –

* tatra = 「そこで」

* kho = 「実に(強調の不変化詞)」

* bhagavā = 「世尊は」

* āyasmantaṃ kimilaṃ = 「尊者キミラに」

* āmantesi = 「呼びかけた(āmanteti のアオリスト)」

→ 「そこで世尊は尊者キミラに呼びかけた――」

3. ‘‘kathaṃ bhāvito nu kho, kimila, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso’’ti?

* kathaṃ = 「どのように」

* bhāvito = 「修習された(bhāveti の過去分詞)」

* nu kho = 「いったい(疑問の不変化詞)」

* kimila = 「キミラよ(呼格)」

* ānāpānassatisamādhi = 「出入息念による定」

  * ānāpāna = 「出入息」

  * sati = 「念」

  * samādhi = 「定、等持」

* kathaṃ bahulīkato = 「どのように多く実践された」(bahulīkata = 多修された)

* mahapphalo = 「大きな果報のあるもの」(mahā + phala)

* hoti = 「〜となる」

* mahānisaṃso = 「大きな利益のあるもの」(mahā + ānisaṃsa)

* ti = 「〜と(引用の終わり)」

→ 「『キミラよ、出入息念による定は、どのように修習され、どのように多く実践されると、大きな果報、大きな利益のあるものとなるのか』と。」

4. Evaṃ vutte āyasmā kimilo tuṇhī ahosi.

* evaṃ vutte = 「このように言われたとき(絶対処格)」

* āyasmā kimilo = 「尊者キミラは」

* tuṇhī = 「沈黙して」

* ahosi = 「あった(hoti のアオリスト)」

→ 「このように言われたとき、尊者キミラは沈黙していた。」

5. Dutiyampi kho bhagavā…pe… tatiyampi kho bhagavā āyasmantaṃ kimilaṃ āmantesi –

* dutiyampi = 「二度目にも」(dutiyaṃ + pi)

* …pe… = 「(中略)」

* tatiyampi = 「三度目にも」(tatiyaṃ + pi)

* kho = 「実に」

* bhagavā = 「世尊は」

* āyasmantaṃ kimilaṃ āmantesi = 「尊者キミラに呼びかけた」

→ 「二度目にも世尊は……(中略)……三度目にも世尊は尊者キミラに呼びかけた――」

6. ‘‘kathaṃ bhāvito nu kho, kimila, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso’’ti?

* (第3チャンクと同文)

→ 「『キミラよ、出入息念による定は、どのように修習され、どのように多く実践されると、大きな果報、大きな利益のあるものとなるのか』と。」

7. Tatiyampi kho āyasmā kimilo tuṇhī ahosi.

* tatiyampi = 「三度目にも」

* āyasmā kimilo = 「尊者キミラは」

* tuṇhī ahosi = 「沈黙していた」

→ 「三度目にも尊者キミラは沈黙していた。」

8. Evaṃ vutte āyasmā ānando bhagavantaṃ etadavoca –

* evaṃ vutte = 「このように言われたとき」

* āyasmā ānando = 「尊者アーナンダは」

* bhagavantaṃ = 「世尊に(対格)」

* etadavoca = 「このことを言った」(etad + avoca, vacati のアオリスト)

→ 「このように言われたとき、尊者アーナンダは世尊にこう言った――」

9. ‘‘etassa, bhagavā, kālo; etassa, sugata, kālo!

* etassa = 「それの(属格)」

* bhagavā = 「世尊よ(呼格)」

* kālo = 「時、時機」

* sugata = 「善逝よ(呼格)」

→ 「『世尊よ、それには今が時です。善逝よ、それには今が時です。」

10. Yaṃ bhagavā ānāpānassatisamādhiṃ bhāseyya.

* yaṃ = 「〜ということ(関係代名詞、前文の etassa を受ける)」

* bhagavā = 「世尊が」

* ānāpānassatisamādhiṃ = 「出入息念による定を(対格)」

* bhāseyya = 「説かれますように(bhāsati の願望法)」

→ 「世尊が出入息念による定をお説きになる、その時です。」

11. Bhagavato sutvā bhikkhū dhāressantī’’ti.

* bhagavato = 「世尊から(属格・奪格的用法)」

* sutvā = 「聞いて(suṇāti の連続体)」

* bhikkhū = 「比丘たちは」

* dhāressanti = 「保持するでしょう(dhāreti の未来形)」

* ti = 「〜と(引用の終わり)」

→ 「比丘たちは世尊から聞いて保持するでしょう』と。」

## 986 (2)

‘‘Tenahānanda, suṇāhi, sādhukaṃ manasi karohi; bhāsissāmī’’ti. ‘‘Evaṃ, bhante’’ti kho āyasmā ānando bhagavato paccassosi. Bhagavā etadavoca – ‘‘kathaṃ bhāvito ca, ānanda, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso? So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvito kho, ānanda, ānāpānassatisamādhi evaṃ bahulīkato mahapphalo hoti mahānisaṃso’’.

### 対訳

1. ‘‘Tenahānanda, suṇāhi, sādhukaṃ manasi karohi; bhāsissāmī’’ti.

* tena hi = 「それでは」
* Ānanda = 「アーナンダよ」
* suṇāhi = 「聞きなさい（suṇāti の命令法二人称単数）」
* sādhukaṃ = 「よく、しっかりと」
* manasi karohi = 「作意しなさい、心に留めなさい」
* bhāsissāmi = 「私は説くだろう（bhāsati の未来形一人称単数）」
* ti = 「～と（引用の終わり）」

→ 「『それでは、アーナンダよ、聞きなさい。よく作意しなさい。私は説きましょう』と。」

2. ‘‘Evaṃ, bhante’’ti kho āyasmā ānando bhagavato paccassosi.

* evaṃ = 「そのように、はい」
* bhante = 「尊師よ」
* ti kho = 「～と」
* āyasmā Ānando = 「尊者アーナンダは」
* bhagavato = 「世尊に（与格）」
* paccassosi = 「応えた（paṭissuṇāti のアオリスト）」

→ 「『はい、尊師よ』と、尊者アーナンダは世尊に応えた。」

3. Bhagavā etadavoca –

* bhagavā = 「世尊は」
* etad avoca = 「このことを言った（etad = これを、avoca = vacati のアオリスト）」

→ 「世尊はこう言った。」

4. ‘‘kathaṃ bhāvito ca, ānanda, ānāpānassatisamādhi kathaṃ bahulīkato mahapphalo hoti mahānisaṃso?

* kathaṃ = 「どのように」
* bhāvito = 「修習された（bhāveti の過去分詞）」
* ca = 「そして」
* Ānanda = 「アーナンダよ」
* ānāpānassati-samādhi = 「出入息念の定」
  * āna = 「出息」、apāna = 「入息」、sati = 「念」、samādhi = 「定」
* kathaṃ bahulīkato = 「どのように多く実践された（bahulīkaroti の過去分詞）」
* mahapphalo hoti = 「大きな果報がある（mahā + phala）」
* mahānisaṃso = 「大きな利益がある（mahā + ānisaṃsa）」

→ 「アーナンダよ、出入息念の定はどのように修習され、どのように多く実践されるならば、大きな果報があり、大きな利益があるのか。」

5. Idhānanda, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati

* idha = 「ここに（この教えにおいて）」
* Ānanda = 「アーナンダよ」
* bhikkhu = 「比丘が」
* arañña-gato vā = 「林に行った者、あるいは」
* rukkha-mūla-gato vā = 「樹の根元に行った者、あるいは」
* suññāgāra-gato vā = 「空き家に行った者、あるいは」
  * suñña = 「空の」、agāra = 「家」
* nisīdati = 「坐る」

→ 「ここにアーナンダよ、比丘は林に行き、あるいは樹の根元に行き、あるいは空き家に行って坐る。」

6. pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで（ābhujati の連続体）」
* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに定め置いて」
  * uju = 「まっすぐな」、kāya = 「身体」、paṇidhāya = 「定め置いて（paṇidahati の連続体）」
* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に現前させて」
  * parimukhaṃ = 「面前に、口の周りに」
  * satiṃ = 「念を」
  * upaṭṭhapetvā = 「現前させて、確立して（upaṭṭhāpeti の連続体）」

→ 「結跏趺坐を組み、身体をまっすぐに定め置き、念を面前に現前させて。」

7. So satova assasati, satova passasati…pe…

* so = 「彼は」
* sato va = 「念をそなえた者としてこそ（sata = 念ある、va = eva 強調）」
* assasati = 「出息する（息を出す）」
* passasati = 「入息する（息を入れる）」
* …pe… = 「（中略、peyyāla）」

→ 「彼は念をそなえて出息し、念をそなえて入息する。……（中略）……」

8. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissagga-anupassī = 「捨遣を随観する者として」
  * paṭinissagga = 「捨遣、捨て放つこと」
  * anupassī = 「随観する者」
* assasissāmi = 「私は出息するだろう（assasati の未来形一人称単数）」
* passasissāmi = 「私は入息するだろう」
* ti sikkhati = 「～と（心がけて）学ぶ、修学する」

→ 「『捨遣を随観しながら出息しよう』と学び、『捨遣を随観しながら入息しよう』と学ぶ。」

9. Evaṃ bhāvito kho, ānanda, ānāpānassatisamādhi evaṃ bahulīkato mahapphalo hoti mahānisaṃso’’.

* evaṃ bhāvito kho = 「実に、このように修習され」
* Ānanda = 「アーナンダよ」
* ānāpānassati-samādhi = 「出入息念の定は」
* evaṃ bahulīkato = 「このように多く実践されるならば」
* mahapphalo hoti = 「大きな果報がある」
* mahānisaṃso = 「大きな利益がある」

→ 「アーナンダよ、このように修習され、このように多く実践されるならば、出入息念の定は大きな果報があり、大きな利益があるのである。」

## 986 (3)

‘‘Yasmiṃ samaye, ānanda, bhikkhu dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti, dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti; rassaṃ vā assasanto ‘rassaṃ assasāmī’ti pajānāti, rassaṃ vā passasanto ‘rassaṃ passasāmī’ti pajānāti; ‘sabbakāyappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sabbakāyappaṭisaṃvedī passasissāmī’ti sikkhati; ‘passambhayaṃ kāyasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ kāyasaṅkhāraṃ passasissāmī’ti sikkhati – kāye kāyānupassī, ānanda, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ. Taṃ kissa hetu? Kāyaññatarāhaṃ, ānanda, etaṃ vadāmi yadidaṃ – assāsapassāsaṃ. Tasmātihānanda, kāye kāyānupassī bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

‘‘Yasmiṃ samaye, ānanda, bhikkhu ‘pītippaṭisaṃvedī assasissāmī’ti sikkhati, ‘pītippaṭisaṃvedī passasissāmī’ti sikkhati; ‘sukhappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sukhappaṭisaṃvedī passasissāmī’ti sikkhati; ‘cittasaṅkhārappaṭisaṃvedī assasissāmī’ti sikkhati, ‘cittasaṅkhārappaṭisaṃvedī passasissāmī’ti sikkhati; ‘passambhayaṃ cittasaṅkhāraṃ passasissāmī’ti sikkhati – vedanāsu vedanānupassī, ānanda, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ. Taṃ kissa hetu? Vedanāññatarāhaṃ, ānanda, etaṃ vadāmi, yadidaṃ – assāsapassāsānaṃ [assāsapassāsaṃ (pī. ka.) ma. ni. 3.145] sādhukaṃ manasikāraṃ. Tasmātihānanda, vedanāsu vedanānupassī bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

### 対訳

1. Yasmiṃ samaye, ānanda, bhikkhu dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti, dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti;

* yasmiṃ samaye = 「そのときに（yasmiṃ = 関係代名詞・処格, samaye = 時に）」
* ānanda = 「アーナンダよ（呼格）」
* bhikkhu = 「比丘は」
* dīghaṃ = 「長く」
* vā = 「あるいは」
* assasanto = 「出息しながら（assasati の現在分詞）」
* ‘dīghaṃ assasāmī’ti = 「『私は長く出息する』と」
* pajānāti = 「了知する（pa + jānāti）」
* passasanto = 「入息しながら（passasati の現在分詞）」
* ‘dīghaṃ passasāmī’ti = 「『私は長く入息する』と」

→ 「アーナンダよ、比丘が長く出息しながら『私は長く出息する』と了知し、長く入息しながら『私は長く入息する』と了知するとき、」

2. rassaṃ vā assasanto ‘rassaṃ assasāmī’ti pajānāti, rassaṃ vā passasanto ‘rassaṃ passasāmī’ti pajānāti;

* rassaṃ = 「短く」

→ 「あるいは短く出息しながら『私は短く出息する』と了知し、短く入息しながら『私は短く入息する』と了知するとき、」

3. ‘sabbakāyappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sabbakāyappaṭisaṃvedī passasissāmī’ti sikkhati;

* sabba-kāya-paṭisaṃvedī = 「全身を感受する者（sabba = すべての, kāya = 身, paṭisaṃvedī = 感受する者）」
* assasissāmī’ti = 「『私は出息しよう』と（未来形）」
* sikkhati = 「学ぶ、修習する」
* passasissāmī’ti = 「『私は入息しよう』と（未来形）」

→ 「『全身を感受しつつ出息しよう』と学び、『全身を感受しつつ入息しよう』と学ぶとき、」

4. ‘passambhayaṃ kāyasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ kāyasaṅkhāraṃ passasissāmī’ti sikkhati –

* passambhayaṃ = 「静めながら（passambheti の現在分詞）」
* kāya-saṅkhāraṃ = 「身行を（身体の働き、ここでは出入息のこと）」

→ 「『身行を静めつつ出息しよう』と学び、『身行を静めつつ入息しよう』と学ぶとき、」

5. kāye kāyānupassī, ānanda, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* kāye = 「身において（処格）」
* kāyānupassī = 「身を随観する者（kāya + anupassī）」
* tasmiṃ samaye = 「そのときに」
* viharati = 「住する」
* ātāpī = 「熱心な者」
* sampajāno = 「正知ある者」
* satimā = 「念ある者」
* vineyya = 「調伏して（vineti の連続体）」
* loke = 「世間における」
* abhijjhā-domanassaṃ = 「貪欲と憂いを（abhijjhā = 貪欲, domanassa = 憂い）」

→ 「アーナンダよ、そのとき比丘は、熱心に、正知をそなえ、念をそなえて、世間における貪欲と憂いを調伏し、身において身を随観して住する。」

6. Taṃ kissa hetu?

* taṃ = 「それは」
* kissa hetu = 「何の理由で（kissa = 何の・属格, hetu = 理由）」

→ 「それはなぜか。」

7. Kāyaññatarāhaṃ, ānanda, etaṃ vadāmi yadidaṃ – assāsapassāsaṃ.

* kāya-aññataraṃ = 「身の一種（aññatara = ある一つの）」
* ahaṃ = 「私は」
* etaṃ = 「これを」
* vadāmi = 「言う」
* yadidaṃ = 「すなわち」
* assāsa-passāsaṃ = 「出息と入息を」

→ 「アーナンダよ、私はこれ、すなわち出息と入息を、身の一種であると言うからである。」

8. Tasmātihānanda, kāye kāyānupassī bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* tasmātiha = 「それゆえここに（tasmā + iha）」

→ 「それゆえアーナンダよ、そのとき比丘は、熱心に、正知をそなえ、念をそなえて、世間における貪欲と憂いを調伏し、身において身を随観して住するのである。」

9. Yasmiṃ samaye, ānanda, bhikkhu ‘pītippaṭisaṃvedī assasissāmī’ti sikkhati, ‘pītippaṭisaṃvedī passasissāmī’ti sikkhati;

* pīti-paṭisaṃvedī = 「喜を感受する者（pīti = 喜）」

→ 「アーナンダよ、比丘が『喜を感受しつつ出息しよう』と学び、『喜を感受しつつ入息しよう』と学ぶとき、」

10. ‘sukhappaṭisaṃvedī assasissāmī’ti sikkhati, ‘sukhappaṭisaṃvedī passasissāmī’ti sikkhati;

* sukha-paṭisaṃvedī = 「楽を感受する者（sukha = 楽）」

14. Taṃ kissa hetu?

→ 「それはなぜか。」

* ahaṃ = 「私は」
* etaṃ vadāmi = 「これを言う」
* yadidaṃ = 「すなわち」
* assāsa-passāsānaṃ = 「出息と入息への（属格・複数）」
* sādhukaṃ = 「十分な、よき」
* manasikāraṃ = 「作意を」
* ※ 異読: assāsapassāsaṃ（ある版および中部 3.145）

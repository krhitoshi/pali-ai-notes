
# 3. Paṭhamaānandasuttaṃ

## 989 (1)

989. Sāvatthinidānaṃ. Atha kho āyasmā ānando yena bhagavā tenupasaṅkami ; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdi. Ekamantaṃ nisinno kho āyasmā ānando bhagavantaṃ etadavoca – ‘‘atthi nu kho, bhante , ekadhammo [eko dhammo (sī.)] bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti?

‘‘Atthi kho, ānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti.

‘‘Katamo pana, bhante, ekadhammo [eko dhammo (sī.)] bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti? ‘‘Ānāpānassatisamādhi kho, ānanda, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti, cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti, satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrenti’’.

3. Paṭhamaānandasuttaṃ

→ 「第一アーナンダ経」

989.

1. Sāvatthinidānaṃ.

* Sāvatthi = 「サーヴァッティー（舎衛城）」

* nidānaṃ = 「因縁、由来（この経の説かれた場所の定型句）」

→ 「サーヴァッティーでのことである。」

2. Atha kho āyasmā ānando yena bhagavā tenupasaṅkami;

* atha kho = 「さて」

* āyasmā = 「尊者」

* ānando = 「アーナンダは」

* yena bhagavā tena = 「世尊のいるところへ」

* upasaṅkami = 「近づいた（upa + saṃ + √kam のアオリスト）」

→ 「さて、尊者アーナンダは世尊のいるところへ近づいた。」

3. upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdi.

* upasaṅkamitvā = 「近づいて（連続体）」

* bhagavantaṃ = 「世尊に（対格）」

* abhivādetvā = 「礼拝して（連続体）」

* ekamantaṃ = 「一方に、傍らに」

* nisīdi = 「坐った（アオリスト）」

→ 「近づいて世尊に礼拝し、一方に坐った。」

4. Ekamantaṃ nisinno kho āyasmā ānando bhagavantaṃ etadavoca –

* ekamantaṃ nisinno = 「一方に坐った（者として）」

* kho = 「実に（強調の小辞）」

* bhagavantaṃ etad avoca = 「世尊にこう言った」

  * etad = 「これを」

  * avoca = 「言った（√vac のアオリスト）」

→ 「一方に坐った尊者アーナンダは世尊にこう言った。」

5. ‘‘atthi nu kho, bhante, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti,

* atthi nu kho = 「～はあるのでしょうか（nu は疑問の小辞）」

* bhante = 「尊師よ」

* ekadhammo = 「一つの法が」

* bhāvito = 「修習された（bhāveti の過去分詞）」

* bahulīkato = 「多く行われた（bahulī-karoti の過去分詞）」

* cattāro dhamme = 「四つの法を（対格）」

* paripūreti = 「完成させる、満たす（pari + √pūr の使役）」

→ 「『尊師よ、修習され、多く行われて、四つの法を完成させる一つの法はあるのでしょうか。」

6. cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti,

* cattāro dhammā = 「四つの法が（主格）」

* bhāvitā bahulīkatā = 「修習され、多く行われて」

* satta dhamme = 「七つの法を」

* paripūrenti = 「完成させる（複数）」

→ 「四つの法が修習され、多く行われて、七つの法を完成させ、」

7. satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti?

* satta dhammā = 「七つの法が」

* dve dhamme = 「二つの法を」

* paripūrenti = 「完成させる」

* ti = 「～と（引用の終わり、疑問文）」

→ 「七つの法が修習され、多く行われて、二つの法を完成させる（そのような一つの法はあるのでしょうか）』と。」

8. ‘‘Atthi kho, ānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti,

* atthi kho = 「実にある」

* ānanda = 「アーナンダよ」

→ 「『アーナンダよ、修習され、多く行われて、四つの法を完成させる一つの法は実にある。」

9. cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti.

→ 「四つの法は修習され、多く行われて、七つの法を完成させ、七つの法は修習され、多く行われて、二つの法を完成させる』と。」

10. ‘‘Katamo pana, bhante, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti,

* katamo = 「どれが、いかなるものが（疑問代名詞）」

* pana = 「では、しかし」

→ 「『では尊師よ、修習され、多く行われて、四つの法を完成させる一つの法とは、どのようなものでしょうか。」

11. cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti?

→ 「四つの法が修習され、多く行われて、七つの法を完成させ、七つの法が修習され、多く行われて、二つの法を完成させる（その一つの法とはどのようなものでしょうか）』と。」

12. ‘‘Ānāpānassatisamādhi kho, ānanda, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti,

* ānāpānassati-samādhi = 「出入息念の定（ānāpāna = 出息と入息、sati = 念、samādhi = 定）」

* ekadhammo = 「一つの法として」

* cattāro satipaṭṭhāne = 「四念処を（対格）」

* paripūreti = 「完成させる」

→ 「『アーナンダよ、出入息念の定こそ、修習され、多く行われて、四念処を完成させる一つの法である。」

13. cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti,

* cattāro satipaṭṭhānā = 「四念処は（主格）」

* satta bojjhaṅge = 「七覚支を（対格）」

* paripūrenti = 「完成させる」

→ 「四念処は修習され、多く行われて、七覚支を完成させ、」

14. satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrenti’’.

* satta bojjhaṅgā = 「七覚支は（主格）」

* vijjā-vimuttiṃ = 「明と解脱を（vijjā = 明・智、vimutti = 解脱）」

* paripūrenti = 「完成させる」

→ 「七覚支は修習され、多く行われて、明と解脱を完成させる。』」

### 対訳

1. ‘‘Kathaṃ bhāvito, ānanda, ānāpānassatisamādhi kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti?

* kathaṃ = 「どのように」

* bhāvito = 「修習された（bhāveti の過去分詞）」

* ānanda = 「アーナンダよ（呼格）」

* ānāpānassati-samādhi = 「出入息念の定」

  * āna = 「出息」 apāna = 「入息」 sati = 「念」 samādhi = 「定、精神統一」

* bahulīkato = 「多修された、繰り返し行われた」

* cattāro satipaṭṭhāne = 「四つの念処を（対格）」

* paripūreti = 「完成させる、円満にする」

→ 「アーナンダよ、出入息念の定はどのように修習され、どのように多修されると、四念処を完成させるのか？」

2. Idhānanda, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに、この（教え）において」

* bhikkhu = 「比丘は」

* arañña-gato vā = 「林に行き、あるいは」

* rukkhamūla-gato vā = 「木の根元に行き、あるいは」

* suññāgāra-gato vā = 「空き家に行き、あるいは」

* nisīdati = 「坐る」

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」

* ujuṃ kāyaṃ paṇidhāya = 「体をまっすぐに定めて」

* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に確立して」

→ 「アーナンダよ、ここに比丘は、林に行き、あるいは木の根元に行き、あるいは空き家に行き、結跏趺坐を組み、体をまっすぐに定め、念を面前に確立して坐る。」

3. So satova assasati, satova passasati.

* so = 「彼は」

* sato va = 「まさに念をもって（sato = 念ある者として、va = eva の短縮「まさに」）」

* assasati = 「出息する（息を外へ出す）」

* passasati = 「入息する（息を内へ入れる）」

→ 「彼はまさに念をもって出息し、まさに念をもって入息する。」

4. Dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti, dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti…pe…

* dīghaṃ vā assasanto = 「あるいは長く出息しながら」

* ‘dīghaṃ assasāmī’ti = 「『私は長く出息している』と」

* pajānāti = 「知る、はっきりと知る」

* dīghaṃ vā passasanto = 「あるいは長く入息しながら」

* ‘dīghaṃ passasāmī’ti pajānāti = 「『私は長く入息している』と知る」

* …pe… = 「（中略）」

→ 「あるいは長く出息しながら『私は長く出息している』と知り、あるいは長く入息しながら『私は長く入息している』と知る……（中略）……」

5. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati’’.

* paṭinissagga-anupassī = 「捨遣を随観する者として」

  * paṭinissagga = 「捨遣、手放すこと」 anupassī = 「随観する者」

* assasissāmī ti = 「『私は出息しよう』と（未来形）」

* sikkhati = 「学ぶ、訓練する」

* passasissāmī ti sikkhati = 「『私は入息しよう』と学ぶ」

→ 「『捨遣を随観しながら私は出息しよう』と学び、『捨遣を随観しながら私は入息しよう』と学ぶ。」

6. ‘‘Yasmiṃ samaye, ānanda, bhikkhu dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti, dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti;

* yasmiṃ samaye = 「その時に（関係代名詞句：～する時には）」

* bhikkhu = 「比丘が」

* dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti = 「あるいは長く出息しながら『私は長く出息している』と知り」

* dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti = 「あるいは長く入息しながら『私は長く入息している』と知る」

→ 「アーナンダよ、比丘が、あるいは長く出息しながら『私は長く出息している』と知り、あるいは長く入息しながら『私は長く入息している』と知る時、」

7. rassaṃ vā…pe… ‘passambhayaṃ kāyasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ kāyasaṅkhāraṃ passasissāmī’ti sikkhati –

* rassaṃ vā = 「あるいは短く」

* …pe… = 「（中略）」

* passambhayaṃ = 「静めながら（passambheti の現在分詞）」

* kāya-saṅkhāraṃ = 「身行を（体の働き、ここでは呼吸のこと）」

* ‘…assasissāmī’ti sikkhati = 「『……私は出息しよう』と学び」

* ‘…passasissāmī’ti sikkhati = 「『……私は入息しよう』と学ぶ」

→ 「あるいは短く……（中略）……『身行を静めながら私は出息しよう』と学び、『身行を静めながら私は入息しよう』と学ぶ時、」

8. kāye kāyānupassī, ānanda, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* kāye = 「身において（処格）」

* kāya-anupassī = 「身を随観する者として」

* tasmiṃ samaye = 「その時に」

* viharati = 「住する、過ごす」

* ātāpī = 「熱心に努める者として」

* sampajāno = 「正知ある者として」

* satimā = 「念ある者として」

* vineyya = 「取り除いて（vineti の連続体）」

* loke = 「世間における」

* abhijjhā-domanassaṃ = 「貪欲と憂いを」

→ 「アーナンダよ、その時、比丘は、世間における貪欲と憂いを取り除いて、熱心に、正知をもち、念をもって、身において身を随観して住する。」

9. Taṃ kissa hetu?

* taṃ = 「それは」

* kissa hetu = 「何の理由によるのか（kissa = kiṃ の属格）」

→ 「それはなぜか？」

10. Kāyaññatarāhaṃ, ānanda, etaṃ vadāmi, yadidaṃ – assāsapassāsaṃ.

* kāya-aññataraṃ = 「身の一種、身のうちのあるもの」

  * aññatara = 「（～のうちの）一つ、ある一種」

* ahaṃ = 「私は」

* etaṃ vadāmi = 「これを～と言う」

* yadidaṃ = 「すなわち」

* assāsa-passāsaṃ = 「出息と入息を」

→ 「アーナンダよ、私はこれ、すなわち出息と入息を、身の一種であると言うからである。」

11. Tasmātihānanda, kāye kāyānupassī bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ’’.

* tasmā-t-iha = 「それゆえにここで（tasmā = それゆえ、iha = ここで）」

* kāye kāyānupassī = 「身において身を随観する者として」

* bhikkhu tasmiṃ samaye viharati = 「比丘はその時に住する」

* ātāpī sampajāno satimā = 「熱心に、正知をもち、念をもって」

* vineyya loke abhijjhādomanassaṃ = 「世間における貪欲と憂いを取り除いて」

→ 「それゆえに、アーナンダよ、その時、比丘は、世間における貪欲と憂いを取り除いて、熱心に、正知をもち、念をもって、身において身を随観して住するのである。」

## 989 (3)

### 対訳

1. Yasmiṃ samaye, ānanda, bhikkhu ‘pītippaṭisaṃvedī assasissāmī’ti sikkhati

* yasmiṃ samaye = 「その時に（yasmiṃ = 関係代名詞・処格, samaye = 時に）」
* ānanda = 「アーナンダよ（呼格）」
* bhikkhu = 「比丘が」
* pīti-ppaṭisaṃvedī = 「喜を感受する者（pīti = 喜, paṭisaṃvedī = 感受する者）」
* assasissāmi = 「私は出息しよう（assasati の未来形一人称単数）」
* ti = 「～と（引用）」
* sikkhati = 「学ぶ、修学する」

→ 「アーナンダよ、比丘が『喜を感受しながら出息しよう』と学ぶ時、」

2. …pe… sukhappaṭisaṃvedī …pe… cittasaṅkhārappaṭisaṃvedī …pe…

* pe = 「（中略）」
* sukha-ppaṭisaṃvedī = 「楽を感受する者（sukha = 楽）」
* cittasaṅkhāra-ppaṭisaṃvedī = 「心行を感受する者（citta = 心, saṅkhāra = 行）」

→ 「（中略）『楽を感受しながら…』（中略）『心行を感受しながら…』（中略）」

3. ‘passambhayaṃ cittasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ cittasaṅkhāraṃ passasissāmī’ti sikkhati –

* passambhayaṃ = 「静めながら（passambheti の現在分詞・主格単数）」
* cittasaṅkhāraṃ = 「心行を（対格）」
* assasissāmi = 「私は出息しよう」
* passasissāmi = 「私は入息しよう（passasati の未来形一人称単数）」
* ti sikkhati = 「～と学ぶ」

→ 「『心行を静めながら出息しよう』と学び、『心行を静めながら入息しよう』と学ぶ時、」

4. vedanāsu vedanānupassī, ānanda, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* bhāveti = 「修習する」

* …pe… = 「（中略。精進覚支・喜覚支・軽安覚支・定覚支が略されている）」

* upekkhāsambojjhaṅgaṃ = 「捨覚支を」

  * upekkhā = 「捨」

* vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ = 「遠離に依り、離貪に依り、滅に依り、放棄へと向かう（前出と同じ）」

→ 「遠離に依り、離貪に依り、滅に依り、放棄へと向かう択法覚支を修習する。（中略）遠離に依り、離貪に依り、滅に依り、放棄へと向かう捨覚支を修習する。」

4. Evaṃ bhāvitā kho, ānanda, satta bojjhaṅgā evaṃ bahulīkatā vijjāvimuttiṃ paripūrentī’’ti.

* evaṃ = 「このように」

* bhāvitā = 「修習された」

* kho = 「実に（強調の不変化詞）」

* ānanda = 「アーナンダよ」

* satta bojjhaṅgā = 「七覚支は」

* evaṃ bahulīkatā = 「このように多く実践されると」

* vijjāvimuttiṃ paripūrenti = 「明知と解脱を完成させる」

* ti = 「～と（引用の終わり）」

→ 「アーナンダよ、このように修習され、このように多く実践されると、七覚支は明知と解脱を完成させる。」

5. Tatiyaṃ.

* tatiyaṃ = 「第三（経）」

→ 「（この品の）第三経（が終わった）。」

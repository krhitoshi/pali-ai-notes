
# 4. Paṭhamaphalasuttaṃ

## 980

980. ‘‘Ānāpānassati , bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā? Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā. Evaṃ bhāvitāya kho, bhikkhave, ānāpānassatiyā evaṃ bahulīkatāya dvinnaṃ phalānaṃ aññataraṃ phalaṃ pāṭikaṅkhaṃ – diṭṭheva dhamme aññā, sati vā upādisese anāgāmitā’’ti. Catutthaṃ.

### 対訳

4. Paṭhamaphalasuttaṃ

→ 「第一果経」（第4経）

1. Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā.

* ānāpānassati = 「出入息念（ānāpāna = 出息と入息 + sati = 念）」

* bhikkhave = 「比丘たちよ（呼格・複数）」

* bhāvitā = 「修習された（bhāveti の過去分詞・女性）」

* bahulīkatā = 「多く実践された（bahulīkaroti の過去分詞・女性）」

* mahapphalā = 「大きな果報がある（mahā + phala）」

* hoti = 「～である（bhavati）」

* mahānisaṃsā = 「大きな利益がある（mahā + ānisaṃsa）」

→ 「比丘たちよ、出入息念は、修習され、多く実践されると、大きな果報があり、大きな利益がある。」

2. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā?

* kathaṃ = 「どのように」

* bhāvitā ca = 「修習され」

* kathaṃ bahulīkatā = 「どのように多く実践されると」

→ 「では比丘たちよ、出入息念はどのように修習され、どのように多く実践されると、大きな果報があり、大きな利益があるのか。」

3. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに（この教えにおいて）」

* bhikkhu = 「比丘は」

* araññagato vā = 「林に行き、あるいは（arañña + gata）」

* rukkhamūlagato vā = 「樹の根もとに行き、あるいは（rukkha + mūla + gata）」

* suññāgāragato vā = 「空き家に行き、あるいは（suñña + agāra + gata）」

* nisīdati = 「坐る」

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで（ābhujati の連続体）」

* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに定めて（uju = まっすぐ、paṇidahati の連続体）」

* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に確立して（upaṭṭhapeti の連続体）」

→ 「比丘たちよ、ここに比丘は、林に行き、あるいは樹の根もとに行き、あるいは空き家に行って、結跏趺坐を組み、身体をまっすぐに定め、念を面前に確立して坐る。」

4. So satova assasati, satova passasati…pe…

* so = 「彼は」

* sato va = 「まさに念をもって（sata = 念ある者 + eva）」

* assasati = 「出息する（外へ息を出す）」

* passasati = 「入息する（内へ息を入れる）」

* …pe… = 「（中略）」

→ 「彼はまさに念をもって出息し、まさに念をもって入息する。……（中略）……」

5. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissagga-anupassī = 「捨て去ることを随観する者（paṭinissagga = 捨遣 + anupassin = 随観者）」

* assasissāmi = 「出息しよう（assasati の未来形・一人称単数）」

* passasissāmi = 「入息しよう（passasati の未来形・一人称単数）」

* ti = 「～と（引用）」

* sikkhati = 「学ぶ、訓練する」

→ 「『捨て去ることを随観しながら出息しよう』と学び、『捨て去ることを随観しながら入息しよう』と学ぶ。」

6. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā.

* evaṃ = 「このように」

* kho = 「実に（強調の小辞）」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念は、大きな果報があり、大きな利益がある。」

7. Evaṃ bhāvitāya kho, bhikkhave, ānāpānassatiyā evaṃ bahulīkatāya dvinnaṃ phalānaṃ aññataraṃ phalaṃ pāṭikaṅkhaṃ –

* evaṃ bhāvitāya … ānāpānassatiyā … bahulīkatāya = 「このように修習され、このように多く実践された出入息念において（処格または属格的用法・女性単数）」

* dvinnaṃ phalānaṃ = 「二つの果報のうち（属格・複数）」

* aññataraṃ phalaṃ = 「いずれか一つの果報が」

* pāṭikaṅkhaṃ = 「期待されるべきである（paṭikaṅkhati からの未来受動分詞）」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念においては、二つの果報のうちいずれか一つの果報が期待される。すなわち――」

8. diṭṭheva dhamme aññā, sati vā upādisese anāgāmitā’’ti. Catutthaṃ.

* diṭṭhe eva dhamme = 「まさに現世において（diṭṭha-dhamma = 見られた法、現世）」

* aññā = 「完全智（阿羅漢果の智）」

* sati vā upādisese = 「あるいは執着の残余があるならば（sati = atthi の現在分詞・処格、upādisesa = 取り残り、処格絶対節）」

* anāgāmitā = 「不還であること（anāgāmin + tā）」

* ti = 「～と（経の結び）」

* catutthaṃ = 「第四（経）」

→ 「まさに現世における完全智か、あるいは執着の残余があるならば不還の位である、と。第四（経）。」

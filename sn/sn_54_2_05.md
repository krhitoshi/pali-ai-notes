
# 5. Paṭhamabhikkhusuttaṃ

## 991

991. Atha kho sambahulā bhikkhū yena bhagavā tenupasaṅkamiṃsu; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdiṃsu. Ekamantaṃ nisinnā kho te bhikkhū bhagavantaṃ etadavocuṃ – ‘‘atthi nu kho, bhante, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti? ‘‘Atthi kho, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti.

‘‘Katamo pana, bhante, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti? ‘‘Ānāpānassatisamādhi kho, bhikkhave, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti, cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti, satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrentī’’ti.

‘‘Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti? Idha, bhikkhave, bhikkhu araññagato vā…pe… evaṃ bhāvitā kho, bhikkhave, satta bojjhaṅgā evaṃ bahulīkatā vijjāvimuttiṃ paripūrentī’’ti. Pañcamaṃ.

### 対訳

**5. Paṭhamabhikkhusuttaṃ（第一比丘経）**

1. Atha kho sambahulā bhikkhū yena bhagavā tenupasaṅkamiṃsu;

* atha kho = 「さて」
* sambahulā bhikkhū = 「多くの比丘たちが」
* yena bhagavā tena upasaṅkamiṃsu = 「世尊のいる所へ近づいた」
  * upasaṅkamiṃsu = 「近づいた（upasaṅkamati のアオリスト・複数）」

→ 「さて、多くの比丘たちが世尊のもとへ近づいた。」

2. upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdiṃsu.

* upasaṅkamitvā = 「近づいて（連続体）」
* bhagavantaṃ abhivādetvā = 「世尊に敬礼して」
* ekamantaṃ nisīdiṃsu = 「一方に坐った」

→ 「近づいて世尊に敬礼し、一方に坐った。」

3. Ekamantaṃ nisinnā kho te bhikkhū bhagavantaṃ etadavocuṃ –

* ekamantaṃ nisinnā = 「一方に坐った（者たちは）」
* te bhikkhū = 「それらの比丘たちは」
* bhagavantaṃ etad avocuṃ = 「世尊にこう言った」

→ 「一方に坐ったそれらの比丘たちは、世尊にこう言った。」

4. ‘‘atthi nu kho, bhante, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti,

* atthi nu kho = 「～はあるのでしょうか」
* bhante = 「尊師よ」
* ekadhammo = 「一つの法が」
* bhāvito bahulīkato = 「修習され、多く実践されて」
  * bhāvita = 「修習された（bhāveti の過去分詞）」
  * bahulīkata = 「多く実践された、繰り返し行われた」
* cattāro dhamme paripūreti = 「四つの法を完成させる」
  * paripūreti = 「満たす、完成させる」

→ 「『尊師よ、修習され多く実践されて四つの法を完成させる一つの法はあるのでしょうか。』」

5. cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti,

* cattāro dhammā = 「四つの法が」
* bhāvitā bahulīkatā = 「修習され、多く実践されて」
* satta dhamme paripūrenti = 「七つの法を完成させる」

→ 「『（また）修習され多く実践されて七つの法を完成させる四つの法は、』」

6. satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti?

* satta dhammā = 「七つの法が」
* dve dhamme paripūrenti = 「二つの法を完成させる」
* ti = 「～と（引用の終わり）」

→ 「『（また）修習され多く実践されて二つの法を完成させる七つの法は、あるのでしょうか。』と。」

7. ‘‘Atthi kho, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti.

* atthi kho, bhikkhave = 「比丘たちよ、～はある」
* （以下は前の問いと同じ構文）

→ 「『比丘たちよ、修習され多く実践されて四つの法を完成させる一つの法はある。四つの法は修習され多く実践されて七つの法を完成させる。七つの法は修習され多く実践されて二つの法を完成させる。』と。」

8. ‘‘Katamo pana, bhante, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti?

* katamo pana = 「では、どれが～か」
* bhante = 「尊師よ」

→ 「『では尊師よ、どの一つの法が、修習され多く実践されて四つの法を完成させ、四つの法が修習され多く実践されて七つの法を完成させ、七つの法が修習され多く実践されて二つの法を完成させるのでしょうか。』と。」

9. ‘‘Ānāpānassatisamādhi kho, bhikkhave, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti,

* ānāpānassati-samādhi = 「出入息念の等持（定）」
  * āna = 「出息」・apāna = 「入息」（本方針：assāsa=出息 / passāsa=入息 に対応）
  * sati = 「念」
  * samādhi = 「等持、定」
* ekadhammo = 「一つの法（として）」
* cattāro satipaṭṭhāne paripūreti = 「四念処を完成させる」
  * satipaṭṭhāna = 「念処」

→ 「『比丘たちよ、出入息念の等持という一つの法が、修習され多く実践されて四念処を完成させる。』」

10. cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti,

* cattāro satipaṭṭhānā = 「四念処が」
* satta bojjhaṅge paripūrenti = 「七覚支を完成させる」
  * bojjhaṅga = 「覚支（bodhi + aṅga）」

→ 「『四念処は修習され多く実践されて七覚支を完成させる。』」

11. satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrentī’’ti.

* satta bojjhaṅgā = 「七覚支が」
* vijjā-vimuttiṃ paripūrenti = 「明と解脱を完成させる」
  * vijjā = 「明、明知」
  * vimutti = 「解脱」

→ 「『七覚支は修習され多く実践されて明と解脱を完成させる。』と。」

12. ‘‘Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti?

* kathaṃ bhāvito = 「どのように修習された（とき）」
* kathaṃ bahulīkato = 「どのように多く実践された（とき）」
* cattāro satipaṭṭhāne paripūreti = 「四念処を完成させる（のか）」

→ 「『比丘たちよ、出入息念の等持は、どのように修習され、どのように多く実践されて、四念処を完成させるのか。』」

13. Idha, bhikkhave, bhikkhu araññagato vā…pe…

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘が」
* arañña-gato vā = 「林に行った者、あるいは」
* …pe… = 「（中略。以下、定型の出入息念の修習と四念処・七覚支の展開が省略されている）」

→ 「『比丘たちよ、ここに比丘は林に行き、あるいは……（中略）……』」

14. evaṃ bhāvitā kho, bhikkhave, satta bojjhaṅgā evaṃ bahulīkatā vijjāvimuttiṃ paripūrentī’’ti.

* evaṃ bhāvitā = 「このように修習され」
* evaṃ bahulīkatā = 「このように多く実践されて」
* vijjāvimuttiṃ paripūrenti = 「明と解脱を完成させる」

→ 「『比丘たちよ、このように修習され、このように多く実践されて、七覚支は明と解脱を完成させる。』と。」

15. Pañcamaṃ.

* pañcamaṃ = 「第五（の経、了）」

→ 「第五（経、終わり）。」


# 5. Dutiyaphalasuttaṃ

## 981

981. ‘‘Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā? Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā.

‘‘Evaṃ bhāvitāya kho, bhikkhave, ānāpānassatiyā evaṃ bahulīkatāya satta phalā sattānisaṃsā pāṭikaṅkhā. Katame satta phalā sattānisaṃsā? Diṭṭheva dhamme paṭikacca aññaṃ ārādheti; no ce diṭṭheva dhamme paṭikacca aññaṃ ārādheti. Atha maraṇakāle aññaṃ ārādheti; no ce diṭṭheva dhamme paṭikacca aññaṃ ārādheti , no ce maraṇakāle aññaṃ ārādheti. Atha pañcannaṃ orambhāgiyānaṃ saṃyojanānaṃ parikkhayā antarāparinibbāyī hoti… upahaccaparinibbāyī hoti… asaṅkhāraparinibbāyī hoti… sasaṅkhāraparinibbāyī hoti… uddhaṃsoto hoti akaniṭṭhagāmī – evaṃ bhāvitāya kho, bhikkhave, ānāpānassatiyā evaṃ bahulīkatāya ime satta phalā sattānisaṃsā pāṭikaṅkhā’’ti. Pañcamaṃ.

### 対訳

5. Dutiyaphalasuttaṃ

→ 「第二の果経」

1. ‘‘Ānāpānassati, bhikkhave, bhāvitā bahulīkatā mahapphalā hoti mahānisaṃsā.

* ānāpānassati = 「出入息念（呼吸への念）」
  * āna = 「出息」 / apāna = 「入息」 / sati = 「念」
* bhikkhave = 「比丘たちよ（呼格複数）」
* bhāvitā = 「修習された（bhāveti の過去分詞・女性）」
* bahulīkatā = 「多く実践された、繰り返し行われた」
* mahapphalā = 「大きな果報がある」
* hoti = 「～である」
* mahānisaṃsā = 「大きな利益がある」

→ 「比丘たちよ、出入息念は、修習され、多く実践されると、大きな果報があり、大きな利益がある。」

2. Kathaṃ bhāvitā ca, bhikkhave, ānāpānassati kathaṃ bahulīkatā mahapphalā hoti mahānisaṃsā?

* kathaṃ = 「どのように」
* bhāvitā ca = 「修習され」
* kathaṃ bahulīkatā = 「どのように多く実践されると」

→ 「では比丘たちよ、出入息念はどのように修習され、どのように多く実践されると、大きな果報があり、大きな利益があるのか。」

3. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘は」
* araññagato vā = 「林に行き、あるいは」
* rukkhamūlagato vā = 「木の根元に行き、あるいは」
* suññāgāragato vā = 「空き家に行き、あるいは」
* nisīdati = 「坐る」
* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」
* ujuṃ kāyaṃ paṇidhāya = 「体をまっすぐに保って」
* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に確立して」

→ 「比丘たちよ、ここに比丘は、林に行き、あるいは木の根元に行き、あるいは空き家に行って、結跏趺坐を組み、体をまっすぐに保ち、念を面前に確立して坐る。」

4. So satova assasati, satova passasati…pe…

* so = 「彼は」
* sato va = 「念をもってこそ（sata = 念のある者）」
* assasati = 「出息する（息を外に出す）」
* passasati = 「入息する（息を内に入れる）」
* …pe… = 「（中略）」

→ 「彼は念をもって出息し、念をもって入息する。（中略）」

5. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissagga-anupassī = 「捨て去ることを随観する者」
  * paṭinissagga = 「捨て去ること、放棄」
  * anupassī = 「随観する者」
* assasissāmi = 「私は出息しよう（未来形一人称単数）」
* passasissāmi = 「私は入息しよう」
* ti = 「～と」
* sikkhati = 「学ぶ、訓練する」

→ 「『捨て去ることを随観しながら出息しよう』と学び、『捨て去ることを随観しながら入息しよう』と学ぶ。」

6. Evaṃ bhāvitā kho, bhikkhave, ānāpānassati evaṃ bahulīkatā mahapphalā hoti mahānisaṃsā.

* evaṃ = 「このように」
* kho = 「実に（強調の不変化詞）」

→ 「比丘たちよ、出入息念はこのように修習され、このように多く実践されると、大きな果報があり、大きな利益がある。」

7. ‘‘Evaṃ bhāvitāya kho, bhikkhave, ānāpānassatiyā evaṃ bahulīkatāya satta phalā sattānisaṃsā pāṭikaṅkhā.

* evaṃ bhāvitāya ... ānāpānassatiyā ... bahulīkatāya = 「このように修習され、このように多く実践された出入息念には（処格または属格的な絶対構文的用法）」
* satta phalā = 「七つの果報」
* satta ānisaṃsā = 「七つの利益」
* pāṭikaṅkhā = 「期待されるべき、当然見込まれる」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念には、七つの果報、七つの利益が期待される。」

8. Katame satta phalā sattānisaṃsā?

* katame = 「どのような（疑問代名詞複数）」

→ 「どのような七つの果報、七つの利益か。」

9. Diṭṭheva dhamme paṭikacca aññaṃ ārādheti;

* diṭṭhe eva dhamme = 「まさに現世において（現法において）」
* paṭikacca = 「早くに、あらかじめ」
* aññaṃ = 「完全智（阿羅漢果の智）」
* ārādheti = 「達成する、成就する」

→ 「まさに現世において早くに完全智を達成する。」

10. no ce diṭṭheva dhamme paṭikacca aññaṃ ārādheti. Atha maraṇakāle aññaṃ ārādheti;

* no ce = 「もし～でなければ」
* atha = 「そのときは、それならば」
* maraṇakāle = 「死のときに（処格）」

→ 「もし現世において早くに完全智を達成しなければ、そのときは死のときに完全智を達成する。」

11. no ce diṭṭheva dhamme paṭikacca aññaṃ ārādheti, no ce maraṇakāle aññaṃ ārādheti.

→ 「もし現世において早くに完全智を達成せず、死のときにも完全智を達成しなければ、」

12. Atha pañcannaṃ orambhāgiyānaṃ saṃyojanānaṃ parikkhayā antarāparinibbāyī hoti…

* pañcannaṃ orambhāgiyānaṃ saṃyojanānaṃ = 「五つの下分結の（属格複数）」
  * orambhāgiya = 「下分の（欲界に結びつける）」
  * saṃyojana = 「結（束縛）」
* parikkhayā = 「完全に滅尽することによって（奪格）」
* antarāparinibbāyī = 「中間般涅槃者（次の生の中途で般涅槃する者）」
* hoti = 「～となる」

→ 「そのときは、五つの下分結が完全に滅尽することによって中間般涅槃者となる…」

13. upahaccaparinibbāyī hoti… asaṅkhāraparinibbāyī hoti… sasaṅkhāraparinibbāyī hoti… uddhaṃsoto hoti akaniṭṭhagāmī –

* upahaccaparinibbāyī = 「損害般涅槃者（寿命の半ばを過ぎてから般涅槃する者）」
* asaṅkhāraparinibbāyī = 「無行般涅槃者（努力なしに般涅槃する者）」
* sasaṅkhāraparinibbāyī = 「有行般涅槃者（努力をともなって般涅槃する者）」
* uddhaṃsoto = 「上流者（上方の世界へ流れのぼる者）」
* akaniṭṭhagāmī = 「色究竟天（アカニッタ天）へ行く者」

→ 「損害般涅槃者となる… 無行般涅槃者となる… 有行般涅槃者となる… 上流者として色究竟天へ行く者となる――」

14. evaṃ bhāvitāya kho, bhikkhave, ānāpānassatiyā evaṃ bahulīkatāya ime satta phalā sattānisaṃsā pāṭikaṅkhā’’ti. Pañcamaṃ.

* ime = 「これらの」
* ti = 「～と（経の結び）」
* pañcamaṃ = 「第五（経の番号）」

→ 「比丘たちよ、このように修習され、このように多く実践された出入息念には、これらの七つの果報、七つの利益が期待される。」第五経。

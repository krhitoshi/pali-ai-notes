
# 7. Saṃyojanappahānasuttaṃ

## 993

993. Ānāpānassatisamādhi, bhikkhave, bhāvito bahulīkato saṃyojanappahānāya saṃvattati…pe…. Sattamaṃ.

### 対訳

1. Saṃyojanappahānasuttaṃ

  * saṃyojana = 「結（衆生を輪廻に縛りつける束縛）」

  * pahāna = 「捨断、捨てること」

  * suttaṃ = 「経」

→ 「結の捨断経」

2. Ānāpānassatisamādhi, bhikkhave, bhāvito bahulīkato saṃyojanappahānāya saṃvattati

  * ānāpānassati-samādhi = 「出入息念の定」

    * āna = 「出息（外へ出る息）」

    * apāna = 「入息（内へ入る息）」

    * sati = 「念」

    * samādhi = 「定、等持」

  * bhikkhave = 「比丘たちよ（呼格・複数）」

  * bhāvito = 「修習された（bhāveti の過去分詞）」

  * bahulīkato = 「多く実践された、繰り返し行われた」

  * saṃyojanappahānāya = 「結の捨断のために（与格）」

  * saṃvattati = 「導く、資する」

→ 「比丘たちよ、出入息念の定が修習され、多く実践されると、結の捨断に導く。」

3. …pe…

  * pe = 「（中略）」（peyyāla の略。前経と同様の定型句が省略されている）

→ 「（中略）」

4. Sattamaṃ.

  * sattamaṃ = 「第七（の経）」

→ 「第七経（が終わった）。」

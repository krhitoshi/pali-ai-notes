
# 2. Kaṅkheyyasuttaṃ

## 988 (1)

988. Ekaṃ samayaṃ āyasmā lomasakaṃbhiyo [lomasavaṅgiso (sī. pī.)] sakkesu viharati kapilavatthusmiṃ nigrodhārāme. Atha kho mahānāmo sakko yenāyasmā lomasakaṃbhiyo tenupasaṅkami; upasaṅkamitvā āyasmantaṃ lomasakaṃbhiyaṃ abhivādetvā ekamantaṃ nisīdi. Ekamantaṃ nisinno kho mahānāmo sakko āyasmantaṃ lomasakaṃbhiyaṃ etadavoca – ‘‘so eva nu kho, bhante, sekho vihāro so tathāgatavihāro, udāhu aññova [añño (syā. kaṃ. pī. ka.)] sekho vihāro añño tathāgatavihāro’’ti?

‘‘Na kho, āvuso mahānāma, sveva sekho vihāro, so tathāgatavihāro. Añño kho, āvuso mahānāma, sekho vihāro, añño tathāgatavihāro. Ye te, āvuso mahānāma, bhikkhū sekhā appattamānasā anuttaraṃ yogakkhemaṃ patthayamānā viharanti, te pañca nīvaraṇe pahāya viharanti. Katame pañca? Kāmacchandanīvaraṇaṃ pahāya viharanti, byāpādanīvaraṇaṃ…pe… thinamiddhanīvaraṇaṃ…pe… uddhaccakukkuccanīvaraṇaṃ…pe… vicikicchānīvaraṇaṃ pahāya viharanti.

### 対訳

2. Kaṅkheyyasuttaṃ

→ 「疑いの経」

1. Ekaṃ samayaṃ āyasmā lomasakaṃbhiyo sakkesu viharati kapilavatthusmiṃ nigrodhārāme.

* ekaṃ samayaṃ = 「あるとき」

* āyasmā = 「尊者」

* lomasakaṃbhiyo = 「ローマサカンビヤ（比丘の名。異読: lomasavaṅgiso ローマサヴァンギサ）」

* sakkesu = 「釈迦族の（国）において」

* viharati = 「住んでいる（vi + √har）」

* kapilavatthusmiṃ = 「カピラヴァットゥにおいて」

* nigrodhārāme = 「ニグローダ園において（nigrodha「バニヤン樹」+ ārāma「園」）」

→ 「あるとき、尊者ローマサカンビヤは釈迦族の国のカピラヴァットゥ、ニグローダ園に住んでいた。」

2. Atha kho mahānāmo sakko yenāyasmā lomasakaṃbhiyo tenupasaṅkami;

* atha kho = 「そのとき」

* mahānāmo sakko = 「釈迦族のマハーナーマ」

* yena ... tena = 「～のところへ（方向を示す関係構文）」

* upasaṅkami = 「近づいた（upasaṅkamati のアオリスト）」

→ 「そのとき、釈迦族のマハーナーマは尊者ローマサカンビヤのところへ近づいた。」

3. upasaṅkamitvā āyasmantaṃ lomasakaṃbhiyaṃ abhivādetvā ekamantaṃ nisīdi.

* upasaṅkamitvā = 「近づいて（連続体）」

* āyasmantaṃ lomasakaṃbhiyaṃ = 「尊者ローマサカンビヤに（対格）」

* abhivādetvā = 「敬礼して（連続体）」

* ekamantaṃ = 「一方に、傍らに」

* nisīdi = 「坐った（nisīdati のアオリスト）」

→ 「近づいて、尊者ローマサカンビヤに敬礼して、一方に坐った。」

4. Ekamantaṃ nisinno kho mahānāmo sakko āyasmantaṃ lomasakaṃbhiyaṃ etadavoca –

* ekamantaṃ nisinno = 「一方に坐った（nisinna = nisīdati の過去分詞）」

* etad avoca = 「このことを言った（avoca = vacati のアオリスト）」

→ 「一方に坐った釈迦族のマハーナーマは、尊者ローマサカンビヤにこう言った。」

5. ‘‘so eva nu kho, bhante, sekho vihāro so tathāgatavihāro, udāhu aññova sekho vihāro añño tathāgatavihāro’’ti?

* so eva nu kho = 「まさにそれが～なのでしょうか（nu = 疑問の小詞）」

* bhante = 「尊者よ」

* sekho vihāro = 「有学の住（sekha = 学びの途上にある者、有学）」

* so tathāgatavihāro = 「それが如来の住（なのでしょうか）」

* udāhu = 「それとも」

* añño va ... añño = 「～は別であり、～も別（である）」

* ti = 「～と（引用の小詞）」

→ 「『尊者よ、有学の住がそのまま如来の住なのでしょうか。それとも、有学の住と如来の住とは別のものなのでしょうか』と。」

6. ‘‘Na kho, āvuso mahānāma, sveva sekho vihāro, so tathāgatavihāro.

* na kho = 「決して～ではない」

* āvuso mahānāma = 「友、マハーナーマよ」

* sveva = so eva = 「まさにそれが」

→ 「『友、マハーナーマよ、有学の住がそのまま如来の住なのではありません。』」

7. Añño kho, āvuso mahānāma, sekho vihāro, añño tathāgatavihāro.

* añño ... añño = 「一方は別であり、他方も別である」

→ 「友、マハーナーマよ、有学の住は別のものであり、如来の住は別のものです。」

8. Ye te, āvuso mahānāma, bhikkhū sekhā appattamānasā anuttaraṃ yogakkhemaṃ patthayamānā viharanti, te pañca nīvaraṇe pahāya viharanti.

* ye te ... bhikkhū = 「およそ～である比丘たちは（関係代名詞構文）」

* sekhā = 「有学の（者たち）」

* appattamānasā = 「まだ目的（阿羅漢果）に達していない（a + patta「達した」+ mānasa「意・目的」）」

* anuttaraṃ = 「無上の」

* yogakkhemaṃ = 「軛からの安穏を（yoga「軛・束縛」+ khema「安穏」）」

* patthayamānā = 「願いながら（pattheti の現在分詞）」

* viharanti = 「住んでいる」

* te = 「彼らは」

* pañca nīvaraṇe = 「五蓋を（nīvaraṇa = 蓋、心を覆う障害）」

* pahāya = 「捨てて（pajahati の連続体）」

→ 「友、マハーナーマよ、およそ有学であり、まだ目的に達しておらず、無上の軛からの安穏を願いながら住んでいる比丘たちは、五蓋を捨てて住んでいます。」

9. Katame pañca?

* katame = 「どのような（疑問代名詞・複数）」

* pañca = 「五つ」

→ 「どのような五つでしょうか。」

10. Kāmacchandanīvaraṇaṃ pahāya viharanti, byāpādanīvaraṇaṃ…pe… thinamiddhanīvaraṇaṃ…pe… uddhaccakukkuccanīvaraṇaṃ…pe… vicikicchānīvaraṇaṃ pahāya viharanti.

* kāmacchandanīvaraṇaṃ = 「欲貪の蓋を（kāma「欲」+ chanda「意欲」）」

* byāpādanīvaraṇaṃ = 「瞋恚の蓋を（byāpāda = 悪意、怒り）」

* thinamiddhanīvaraṇaṃ = 「惛沈睡眠の蓋を（thina「心の沈み」+ middha「眠気」）」

* uddhaccakukkuccanīvaraṇaṃ = 「掉挙後悔の蓋を（uddhacca「浮つき」+ kukkucca「後悔」）」

* vicikicchānīvaraṇaṃ = 「疑の蓋を（vicikicchā = 疑い）」

* pahāya viharanti = 「捨てて住んでいる」

* …pe… = 「（中略の符号）」

→ 「欲貪の蓋を捨てて住み、瞋恚の蓋を…（中略）…惛沈睡眠の蓋を…（中略）…掉挙後悔の蓋を…（中略）…疑の蓋を捨てて住んでいます。」

## 988 (2)

‘‘Yepi te, āvuso mahānāma, bhikkhū sekhā appattamānasā anuttaraṃ yogakkhemaṃ patthayamānā viharanti, te ime pañca nīvaraṇe pahāya viharanti.

‘‘Ye ca kho te, āvuso mahānāma, bhikkhū arahanto khīṇāsavā vusitavanto katakaraṇīyā ohitabhārā anuppattasadatthā parikkhīṇabhavasaṃyojanā sammadaññā vimuttā, tesaṃ pañca nīvaraṇā pahīnā ucchinnamūlā tālāvatthukatā anabhāvaṃkatā [anabhāvakatā (sī. pī.)] āyatiṃ anuppādadhammā. Katame pañca? Kāmacchandanīvaraṇaṃ pahīnaṃ ucchinnamūlaṃ tālāvatthukataṃ anabhāvaṃkataṃ āyatiṃ anuppādadhammaṃ; byāpādanīvaraṇaṃ pahīnaṃ…pe… thinamiddhanīvaraṇaṃ…pe… uddhaccakukkuccanīvaraṇaṃ…pe… vicikicchānīvaraṇaṃ pahīnaṃ ucchinnamūlaṃ tālāvatthukataṃ anabhāvaṃkataṃ āyatiṃ anuppādadhammaṃ.

‘‘Ye te, āvuso mahānāma, bhikkhū arahanto khīṇāsavā vusitavanto katakaraṇīyā ohitabhārā anuppattasadatthā parikkhīṇabhavasaṃyojanā sammadaññā vimuttā, tesaṃ ime pañca nīvaraṇā pahīnā ucchinnamūlā tālāvatthukatā anabhāvaṃkatā āyatiṃ anuppādadhammā. Tadamināpetaṃ, āvuso mahānāma, pariyāyena veditabbaṃ yathā – aññova sekho vihāro, añño tathāgatavihāro.

### 対訳

1. Yepi te, āvuso mahānāma, bhikkhū sekhā appattamānasā anuttaraṃ yogakkhemaṃ patthayamānā viharanti,

* ye pi te = 「およそ～であるところの彼ら（比丘たち）もまた」

* āvuso mahānāma = 「友、マハーナーマよ」

* bhikkhū sekhā = 「有学の比丘たち」

  * sekha = 「有学（まだ学ぶべきことが残っている者、阿羅漢果に達していない聖者）」

* appattamānasā = 「（阿羅漢果という）心の目的にまだ達していない者たち」

  * appatta = 「達していない（a + patta, pāpuṇāti の過去分詞）」

  * mānasa = 「意、心の目的（ここでは阿羅漢果を指す）」

* anuttaraṃ yogakkhemaṃ = 「無上の軛安穏（束縛からの安穏、すなわち阿羅漢果）を」

  * anuttara = 「無上の」

  * yogakkhema = 「軛からの安穏（yoga = 軛・束縛、khema = 安穏）」

* patthayamānā = 「願いながら（pattheti の現在分詞）」

* viharanti = 「住している、暮らしている」

→ 「友、マハーナーマよ、有学であり、まだ心の目的に達しておらず、無上の軛安穏を願って住している比丘たちもまた、」

2. te ime pañca nīvaraṇe pahāya viharanti.

* te = 「彼らは」

* ime pañca nīvaraṇe = 「これら五つの蓋（心を覆う障害）を」

* pahāya = 「捨てて（pajahati の連続体）」

* viharanti = 「住している」

→ 「彼らはこれら五つの蓋を捨てて住している。」

3. Ye ca kho te, āvuso mahānāma, bhikkhū arahanto khīṇāsavā vusitavanto katakaraṇīyā ohitabhārā anuppattasadatthā parikkhīṇabhavasaṃyojanā sammadaññā vimuttā,

* ye ca kho te = 「しかしまた、およそ～であるところの彼ら（比丘たち）は」

* āvuso mahānāma = 「友、マハーナーマよ」

* bhikkhū arahanto = 「阿羅漢である比丘たち」

* khīṇāsavā = 「諸々の漏（煩悩）が尽きた者たち」

  * khīṇa = 「尽きた（khīyati の過去分詞）」

  * āsava = 「漏（心から流れ出る煩悩）」

* vusitavanto = 「（梵行を）住み終えた者たち、修行を完成した者たち」

* katakaraṇīyā = 「なすべきことをなし終えた者たち」

  * kata = 「なされた」 + karaṇīya = 「なすべきこと」

* ohitabhārā = 「重荷を下ろした者たち」

  * ohita = 「下ろした」 + bhāra = 「重荷（五蘊・煩悩・行作の重荷）」

* anuppattasadatthā = 「自らの目的（阿羅漢果）に到達した者たち」

  * anuppatta = 「到達した」 + sadattha = 「自らの目的（saka + attha）」

* parikkhīṇabhavasaṃyojanā = 「有（迷いの生存）への結縛が完全に尽きた者たち」

  * parikkhīṇa = 「完全に尽きた」 + bhava = 「有、生存」 + saṃyojana = 「結、結縛」

* sammadaññā vimuttā = 「正しく了知して解脱した者たち」

  * sammā = 「正しく」 + aññā = 「了知（完全智）」

  * vimutta = 「解脱した（vimuccati の過去分詞）」

→ 「しかしまた、友、マハーナーマよ、阿羅漢であり、諸々の漏が尽き、修行を住み終え、なすべきことをなし終え、重荷を下ろし、自らの目的に到達し、有への結縛が完全に尽き、正しく了知して解脱した比丘たち、」

4. tesaṃ pañca nīvaraṇā pahīnā ucchinnamūlā tālāvatthukatā anabhāvaṃkatā āyatiṃ anuppādadhammā.

* tesaṃ = 「彼らにとって」

* pañca nīvaraṇā = 「五つの蓋は」

* pahīnā = 「捨てられた（pajahati の過去分詞）」

* ucchinnamūlā = 「根が断ち切られた」

  * ucchinna = 「断ち切られた」 + mūla = 「根」

* tālāvatthukatā = 「（根を切られた）ターラ樹の切り株のようにされた」

  * tāla = 「ターラ樹（パルミラ椰子）」 + avatthu = 「土台のない状態」 + kata = 「された」（再生しないことの譬え）

* anabhāvaṃkatā = 「存在しないものとされた、消滅させられた」

* āyatiṃ anuppādadhammā = 「未来に生じない性質のもの（となった）」

  * āyatiṃ = 「未来に」

  * anuppāda = 「不生起」 + dhamma = 「～する性質のもの」

→ 「彼らにとって五つの蓋は捨てられ、根が断ち切られ、ターラ樹の切り株のようにされ、消滅させられ、未来に生じない性質のものとなっている。」

5. Katame pañca?

* katame = 「どの、いかなる」

* pañca = 「五つ（とは）」

→ 「五つとは何か。」

6. Kāmacchandanīvaraṇaṃ pahīnaṃ ucchinnamūlaṃ tālāvatthukataṃ anabhāvaṃkataṃ āyatiṃ anuppādadhammaṃ;

* kāmacchanda-nīvaraṇaṃ = 「欲貪の蓋は」

  * kāma = 「欲」 + chanda = 「欲求」

* pahīnaṃ = 「捨てられ」

* ucchinnamūlaṃ = 「根が断ち切られ」

* tālāvatthukataṃ = 「ターラ樹の切り株のようにされ」

* anabhāvaṃkataṃ = 「消滅させられ」

* āyatiṃ anuppādadhammaṃ = 「未来に生じない性質のもの（となっている）」

→ 「欲貪の蓋は捨てられ、根が断ち切られ、ターラ樹の切り株のようにされ、消滅させられ、未来に生じない性質のものとなっている。」

7. byāpādanīvaraṇaṃ pahīnaṃ …pe… thinamiddhanīvaraṇaṃ …pe… uddhaccakukkuccanīvaraṇaṃ …pe… vicikicchānīvaraṇaṃ pahīnaṃ ucchinnamūlaṃ tālāvatthukataṃ anabhāvaṃkataṃ āyatiṃ anuppādadhammaṃ.

* byāpāda-nīvaraṇaṃ = 「瞋恚の蓋は」

* pahīnaṃ …pe… = 「捨てられ……（中略）」

* thinamiddha-nīvaraṇaṃ = 「惛沈・睡眠の蓋は」

  * thina = 「惛沈（心の沈鬱）」 + middha = 「睡眠（心所の鈍重さ）」

* uddhaccakukkucca-nīvaraṇaṃ = 「掉挙・後悔の蓋は」

  * uddhacca = 「掉挙（心の浮動）」 + kukkucca = 「後悔、悪作」

* vicikicchā-nīvaraṇaṃ = 「疑の蓋は」

* pahīnaṃ ucchinnamūlaṃ tālāvatthukataṃ anabhāvaṃkataṃ āyatiṃ anuppādadhammaṃ = 「捨てられ、根が断ち切られ、ターラ樹の切り株のようにされ、消滅させられ、未来に生じない性質のもの（となっている）」

→ 「瞋恚の蓋は捨てられ……（中略）……惛沈・睡眠の蓋は……（中略）……掉挙・後悔の蓋は……（中略）……疑の蓋は捨てられ、根が断ち切られ、ターラ樹の切り株のようにされ、消滅させられ、未来に生じない性質のものとなっている。」

8. Ye te, āvuso mahānāma, bhikkhū arahanto khīṇāsavā vusitavanto katakaraṇīyā ohitabhārā anuppattasadatthā parikkhīṇabhavasaṃyojanā sammadaññā vimuttā, tesaṃ ime pañca nīvaraṇā pahīnā ucchinnamūlā tālāvatthukatā anabhāvaṃkatā āyatiṃ anuppādadhammā.

* tad aminā pi etaṃ = 「それゆえ、この（次に述べる）～によってもこのことが」

  * aminā = 「これによって（ayaṃ の具格）」

## 988 (3)

‘‘Ekamidaṃ, āvuso mahānāma, samayaṃ bhagavā icchānaṅgale viharati icchānaṅgalavanasaṇḍe. Tatra kho, āvuso mahānāma, bhagavā bhikkhū āmantesi – ‘icchāmahaṃ, bhikkhave, temāsaṃ paṭisallīyituṃ. Nāmhi kenaci upasaṅkamitabbo, aññatra ekena piṇḍapātanīhārakenā’ti. ‘Evaṃ, bhante’ti kho, āvuso mahānāma, te bhikkhū bhagavato paṭissutvā nāssudha koci bhagavantaṃ upasaṅkamati, aññatra ekena piṇḍapātanīhārakena.

‘‘Atha kho, āvuso, bhagavā tassa temāsassa accayena paṭisallānā vuṭṭhito bhikkhū āmantesi – ‘sace kho, bhikkhave, aññatitthiyā paribbājakā evaṃ puccheyyuṃ – katamenāvuso, vihārena samaṇo gotamo vassāvāsaṃ bahulaṃ vihāsīti, evaṃ puṭṭhā tumhe, bhikkhave, tesaṃ aññatitthiyānaṃ paribbājakānaṃ evaṃ byākareyyātha – ānāpānassatisamādhinā kho, āvuso, bhagavā vassāvāsaṃ bahulaṃ vihāsīti. Idhāhaṃ, bhikkhave , sato assasāmi, sato passasāmi. Dīghaṃ assasanto dīghaṃ assasāmīti pajānāmi, dīghaṃ passasanto dīghaṃ passasāmīti pajānāmi…pe… paṭinissaggānupassī assasissāmīti pajānāmi, paṭinissaggānupassī passasissāmī’ti pajānāmi’’.

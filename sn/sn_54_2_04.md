
# 4. Dutiyaānandasuttaṃ

## 990

990. Atha kho āyasmā ānando yena bhagavā tenupasaṅkami; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdi. Ekamantaṃ nisinnaṃ kho āyasmantaṃ ānandaṃ bhagavā etadavoca – ‘‘atthi nu kho, ānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā, satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti. Bhagavaṃmūlakā no, bhante, dhammā…pe… ‘‘atthānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti.

‘‘Katamo cānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti? Ānāpānassatisamādhi, ānanda, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti, cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti, satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrentī’’ti. ‘‘Kathaṃ bhāvito cānanda, ānāpānassatisamādhi, kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti? Idhānanda, bhikkhu araññagato vā…pe… evaṃ bhāvitā kho, ānanda, satta bojjhaṅgā evaṃ bahulīkatā vijjāvimuttiṃ paripūrentī’’ti. Catutthaṃ.

### 対訳

1. Dutiyaānandasuttaṃ

* dutiya = 「第二の」
* ānanda = 「アーナンダ（人名）」
* suttaṃ = 「経」

→ 「第二アーナンダ経」

2. 990. Atha kho āyasmā ānando yena bhagavā tenupasaṅkami; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdi.

* atha kho = 「そこで」
* āyasmā ānando = 「尊者アーナンダは」
* yena bhagavā tena upasaṅkami = 「世尊のいるところへ近づいた」
  * yena ... tena = 「～のところへ」（慣用表現）
  * upasaṅkami = 「近づいた（upasaṅkamati のアオリスト）」
* upasaṅkamitvā = 「近づいてから（連続体）」
* bhagavantaṃ abhivādetvā = 「世尊に礼拝して」
* ekamantaṃ nisīdi = 「一方に坐った」

→ 「そこで尊者アーナンダは世尊のもとへ近づいた。近づいて世尊に礼拝し、一方に坐った。」

3. Ekamantaṃ nisinnaṃ kho āyasmantaṃ ānandaṃ bhagavā etadavoca –

* ekamantaṃ nisinnaṃ = 「一方に坐った（対格、アーナンダを修飾）」
* āyasmantaṃ ānandaṃ = 「尊者アーナンダに（対格）」
* bhagavā = 「世尊は」
* etad avoca = 「このことを言った」

→ 「一方に坐った尊者アーナンダに、世尊はこう言った。」

4. ‘‘atthi nu kho, ānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti,

* atthi nu kho = 「はたして～があるだろうか」
  * nu kho = 疑問の小辞
* ānanda = 「アーナンダよ（呼格）」
* ekadhammo = 「一つの法が」
* bhāvito = 「修習され（bhāveti の過去分詞）」
* bahulīkato = 「多く実践され（bahulīkaroti の過去分詞）」
* cattāro dhamme = 「四つの法を」
* paripūreti = 「完成させる、満たす」

→ 「アーナンダよ、修習され、多く実践されると四つの法を完成させる一つの法があるだろうか。」

5. cattāro dhammā bhāvitā bahulīkatā, satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti.

* cattāro dhammā = 「四つの法が（主格）」
* bhāvitā bahulīkatā = 「修習され、多く実践されると」
* satta dhamme = 「七つの法を」
* paripūrenti = 「完成させる（複数）」
* satta dhammā = 「七つの法が」
* dve dhamme = 「二つの法を」
* ti = 引用の終わりを示す小辞

→ 「四つの法は修習され、多く実践されると七つの法を完成させ、七つの法は修習され、多く実践されると二つの法を完成させる（そのような法があるだろうか）」と。

6. Bhagavaṃmūlakā no, bhante, dhammā…pe…

* bhagavaṃmūlakā = 「世尊を根本とする」
  * bhagavaṃ = 「世尊」 + mūlaka = 「根本とする」
* no = 「私たちの（属格複数）」
* bhante = 「尊師よ」
* dhammā = 「諸法は」
* …pe… = 中略（「世尊を導き手とし、世尊を依りどころとします。尊師よ、どうかこの言葉の意味を世尊が明らかにしてくださいますように…」という定型句の省略）

→ 「尊師よ、私たちの諸法は世尊を根本とし……（中略）」

7. ‘‘atthānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti.

* atthi + ānanda → atthānanda = 「アーナンダよ、ある」

→ 「アーナンダよ、修習され、多く実践されると四つの法を完成させる一つの法がある。四つの法は修習され、多く実践されると七つの法を完成させ、七つの法は修習され、多く実践されると二つの法を完成させる。」

8. ‘‘Katamo cānanda, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti?

* katamo = 「どれか、いかなるものか（疑問代名詞）」
* ca + ānanda → cānanda = 「そして、アーナンダよ」

→ 「では、アーナンダよ、修習され、多く実践されると四つの法を完成させ、（その）四つの法が修習され、多く実践されると七つの法を完成させ、（その）七つの法が修習され、多く実践されると二つの法を完成させる一つの法とは、どれか。」

9. Ānāpānassatisamādhi, ānanda, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti,

* ānāpānassatisamādhi = 「出入息念の定（安般念定）」
  * āna = 「出息」
  * apāna = 「入息」
  * sati = 「念」
  * samādhi = 「定、等持」
* cattāro satipaṭṭhāne = 「四念処を」

→ 「アーナンダよ、出入息念の定という一つの法が、修習され、多く実践されると四念処を完成させる。」

10. cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti, satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrentī’’ti.

* cattāro satipaṭṭhānā = 「四念処が（主格）」
* satta bojjhaṅge = 「七覚支を」
  * bojjhaṅga = bodhi（覚り）+ aṅga（支分）
* satta bojjhaṅgā = 「七覚支が（主格）」
* vijjāvimuttiṃ = 「明と解脱を」
  * vijjā = 「明、明智」
  * vimutti = 「解脱」

→ 「四念処は修習され、多く実践されると七覚支を完成させ、七覚支は修習され、多く実践されると明と解脱を完成させる」と。

11. ‘‘Kathaṃ bhāvito cānanda, ānāpānassatisamādhi, kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti?

* kathaṃ = 「どのように」
* bhāvito = 「修習されると」
* bahulīkato = 「多く実践されると」

→ 「では、アーナンダよ、出入息念の定はどのように修習され、どのように多く実践されると四念処を完成させるのか。」

12. Idhānanda, bhikkhu araññagato vā…pe…

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘が」
* araññagato vā = 「林に行き、あるいは」
  * arañña = 「林、森」 + gata = 「行った」
* …pe… = 中略（「樹の下に行き、あるいは空き家に行き、結跏趺坐を組み、身体をまっすぐに保ち、念を面前に現前させて坐る。彼はまさに念じつつ出息し、念じつつ入息する……」以下、出入息念の十六事と四念処・七覚支の修習の定型句の省略）

→ 「アーナンダよ、ここに比丘が林に行き、あるいは……（中略）」

13. evaṃ bhāvitā kho, ānanda, satta bojjhaṅgā evaṃ bahulīkatā vijjāvimuttiṃ paripūrentī’’ti.

* evaṃ = 「このように」
* bhāvitā = 「修習された」
* satta bojjhaṅgā = 「七覚支は」
* vijjāvimuttiṃ paripūrenti = 「明と解脱を完成させる」

→ 「アーナンダよ、このように修習され、このように多く実践されると、七覚支は明と解脱を完成させる」と。

14. Catutthaṃ.

* catutthaṃ = 「第四（の経）」

→ 「以上、第四経。」

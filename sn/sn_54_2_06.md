
# 6. Dutiyabhikkhusuttaṃ

## 992 (1)

992. Atha kho sambahulā bhikkhū yena bhagavā tenupasaṅkamiṃsu; upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdiṃsu. Ekamantaṃ nisinne kho te bhikkhū bhagavā etadavoca – ‘‘atthi nu kho, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti? ‘‘Bhagavaṃmūlakā no, bhante, dhammā…pe… bhagavato sutvā bhikkhū dhāressantī’’ti. ‘‘Atthi, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti’’.

‘‘Katamo ca, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti? Ānāpānassatisamādhi, bhikkhave, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti, cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti, satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrentīti.

‘‘Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti? Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati’’.

992.

1. Atha kho sambahulā bhikkhū yena bhagavā tenupasaṅkamiṃsu;

* atha kho = 「さて」
* sambahulā bhikkhū = 「多くの比丘たちが」
* yena bhagavā tena = 「世尊のいるところへ」
* upasaṅkamiṃsu = 「近づいた（upasaṅkamati の過去・複数）」

→ 「さて、多くの比丘たちが世尊のもとへ近づいた。」

2. upasaṅkamitvā bhagavantaṃ abhivādetvā ekamantaṃ nisīdiṃsu.

* upasaṅkamitvā = 「近づいて（連続体）」
* bhagavantaṃ abhivādetvā = 「世尊に敬礼して」
* ekamantaṃ = 「一方に、傍らに」
* nisīdiṃsu = 「坐った（過去・複数）」

→ 「近づいて世尊に敬礼し、一方に坐った。」

3. Ekamantaṃ nisinne kho te bhikkhū bhagavā etadavoca –

* ekamantaṃ nisinne = 「一方に坐った（者たちに）」
* te bhikkhū = 「それらの比丘たちに」
* bhagavā = 「世尊は」
* etad avoca = 「このように言った」

→ 「一方に坐ったそれらの比丘たちに、世尊はこのように言った。」

4. ‘‘atthi nu kho, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti,

* atthi nu kho = 「〜はあるだろうか」
* bhikkhave = 「比丘たちよ」
* ekadhammo = 「一つの法が」
* bhāvito = 「修習され（bhāveti の過去分詞）」
* bahulīkato = 「多く実践され」
* cattāro dhamme = 「四つの法を」
* paripūreti = 「完成させる」

→ 「比丘たちよ、修習され、多く実践されて、四つの法を完成させる一つの法はあるだろうか。」

5. cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti,

* cattāro dhammā = 「四つの法が」
* bhāvitā bahulīkatā = 「修習され、多く実践されて」
* satta dhamme = 「七つの法を」
* paripūrenti = 「完成させる（複数）」

→ 「四つの法が修習され、多く実践されて、七つの法を完成させ、」

6. satta dhammā bhāvitā bahulīkatā dve dhamme paripūrentī’’ti?

* satta dhammā = 「七つの法が」
* dve dhamme = 「二つの法を」
* paripūrenti = 「完成させる」
* ti = 引用の終わり（疑問文）

→ 「七つの法が修習され、多く実践されて、二つの法を完成させる（そのような一法はあるだろうか）」と。

7. ‘‘Bhagavaṃmūlakā no, bhante, dhammā…pe… bhagavato sutvā bhikkhū dhāressantī’’ti.

* bhagavaṃ-mūlakā = 「世尊を根本とする」
* no = 「私たちの」
* bhante = 「尊師よ」
* dhammā = 「諸法は」
* …pe… = 「（中略）」
* bhagavato sutvā = 「世尊から聞いて」
* bhikkhū dhāressanti = 「比丘たちは記憶するでしょう（dhāreti の未来・複数）」

→ 「尊師よ、私たちの諸法は世尊を根本とするものです。（中略）世尊から聞いて、比丘たちは記憶するでしょう」と。

8. ‘‘Atthi, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti’’.

* atthi = 「ある」（以下は前出と同じ）

→ 「比丘たちよ、修習され、多く実践されて、四つの法を完成させる一つの法はある。四つの法が修習され、多く実践されて、七つの法を完成させ、七つの法が修習され、多く実践されて、二つの法を完成させる。」

9. ‘‘Katamo ca, bhikkhave, ekadhammo bhāvito bahulīkato cattāro dhamme paripūreti, cattāro dhammā bhāvitā bahulīkatā satta dhamme paripūrenti, satta dhammā bhāvitā bahulīkatā dve dhamme paripūrenti?

* katamo ca = 「では、どのような〜か」

→ 「比丘たちよ、では、修習され、多く実践されて、四つの法を完成させ、（その）四つの法が修習され、多く実践されて、七つの法を完成させ、（その）七つの法が修習され、多く実践されて、二つの法を完成させる一つの法とは、どのようなものか。」

10. Ānāpānassatisamādhi, bhikkhave, ekadhammo bhāvito bahulīkato cattāro satipaṭṭhāne paripūreti,

* ānāpānassati-samādhi = 「出入息念の定（三昧）」
  * āna = 「出息」/ apāna = 「入息」（本プロジェクトの方針による向き）
  * sati = 「念」/ samādhi = 「定、三昧」
* cattāro satipaṭṭhāne = 「四念処を」
* paripūreti = 「完成させる」

→ 「比丘たちよ、出入息念の定という一つの法が修習され、多く実践されて、四念処を完成させ、」

11. cattāro satipaṭṭhānā bhāvitā bahulīkatā satta bojjhaṅge paripūrenti,

* cattāro satipaṭṭhānā = 「四念処が」
* satta bojjhaṅge = 「七覚支を」
* paripūrenti = 「完成させる」

→ 「四念処が修習され、多く実践されて、七覚支を完成させ、」

12. satta bojjhaṅgā bhāvitā bahulīkatā vijjāvimuttiṃ paripūrentīti.

* satta bojjhaṅgā = 「七覚支が」
* vijjā-vimuttiṃ = 「明と解脱を」
  * vijjā = 「明、明知」/ vimutti = 「解脱」
* paripūrenti = 「完成させる」

→ 「七覚支が修習され、多く実践されて、明と解脱を完成させる。」

13. ‘‘Kathaṃ bhāvito ca, bhikkhave, ānāpānassatisamādhi kathaṃ bahulīkato cattāro satipaṭṭhāne paripūreti?

* kathaṃ bhāvito = 「どのように修習され」
* kathaṃ bahulīkato = 「どのように多く実践されて」

→ 「比丘たちよ、出入息念の定はどのように修習され、どのように多く実践されて、四念処を完成させるのか。」

14. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati

* idha = 「ここに（この教えにおいて）」
* bhikkhu = 「比丘が」
* arañña-gato vā = 「林に行き、あるいは」
* rukkhamūla-gato vā = 「樹の根もとに行き、あるいは」
* suññāgāra-gato vā = 「空き家に行き」
* nisīdati = 「坐る」

→ 「比丘たちよ、ここに比丘が、林に行き、あるいは樹の根もとに行き、あるいは空き家に行って坐る。」

15. pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」
* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに定めて」
* parimukhaṃ = 「面前に」
* satiṃ upaṭṭhapetvā = 「念を現前させて（upaṭṭhapeti の連続体）」

→ 「結跏趺坐を組み、身体をまっすぐに定め、面前に念を現前させて（坐る）。」

16. So satova assasati, satova passasati…pe…

* so = 「彼は」
* sato va = 「念をそなえたままで」
* assasati = 「出息する（息を外へ出す）」
* passasati = 「入息する（息を内へ入れる）」
* …pe… = 「（中略）」

→ 「彼は念をそなえたままで出息し、念をそなえたままで入息する。（中略）」

17. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati’’.

* paṭinissagga-anupassī = 「捨て去ることを随観する者として」
  * paṭinissagga = 「捨て去ること、放棄」
  * anupassī = 「随観する者」
* assasissāmi = 「私は出息しよう（未来・一人称単数）」
* passasissāmi = 「私は入息しよう」
* ti sikkhati = 「〜と学ぶ、訓練する」

→ 「『捨て去ることを随観する者として出息しよう』と学び、『捨て去ることを随観する者として入息しよう』と学ぶ。」

### 対訳

【第1段落：身随観(第一の四法)】

1. Yasmiṃ samaye, bhikkhave, bhikkhu dīghaṃ vā assasanto ‘dīghaṃ assasāmī’ti pajānāti,

* yasmiṃ samaye = 「～する時に」(関係代名詞 + 処格)

* bhikkhave = 「比丘たちよ」(呼格)

* bhikkhu = 「比丘が」

* dīghaṃ vā assasanto = 「長く出息しながら」

  * dīghaṃ = 「長く」

  * assasanto = 「出息している」(assasati の現在分詞)

* ‘dīghaṃ assasāmī’ti pajānāti = 「『私は長く出息している』と了知する」

  * pajānāti = 「了知する、はっきりと知る」

→ 「比丘たちよ、比丘が長く出息しながら『私は長く出息している』と了知し、」

2. dīghaṃ vā passasanto ‘dīghaṃ passasāmī’ti pajānāti,

* dīghaṃ vā passasanto = 「あるいは長く入息しながら」

  * passasanto = 「入息している」(passasati の現在分詞)

→ 「あるいは長く入息しながら『私は長く入息している』と了知し、」

3. rassaṃ vā assasanto ‘rassaṃ assasāmī’ti pajānāti…pe…

* rassaṃ = 「短く」

* …pe… = 「(中略)」(peyyāla、繰り返し部分の省略)

→ 「あるいは短く出息しながら『私は短く出息している』と了知し……(中略)……」

4. sabbakāyappaṭisaṃvedī…pe…

* sabba-kāya-paṭisaṃvedī = 「全身を感受する者(となって)」

  * sabba = 「すべての」

  * kāya = 「身体」(ここでは息の全体、すなわち息の初め・中間・終わりを指すと註釈される)

  * paṭisaṃvedī = 「感受する者、体験して知る者」

→ 「『全身を感受して……』(中略)……」

5. ‘passambhayaṃ kāyasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ kāyasaṅkhāraṃ passasissāmī’ti sikkhati –

* passambhayaṃ = 「静めながら」(passambheti の現在分詞)

* kāya-saṅkhāraṃ = 「身行を」(身体の行、すなわち出入息そのもの)

* assasissāmī’ti = 「『私は出息しよう』と」(未来形)

* passasissāmī’ti = 「『私は入息しよう』と」(未来形)

* sikkhati = 「学ぶ、修学する」

→ 「『身行を静めながら私は出息しよう』と学び、『身行を静めながら私は入息しよう』と学ぶ――」

6. kāye kāyānupassī, bhikkhave, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* kāye kāyānupassī = 「身において身を随観する者として」

  * kāye = 「身において」(処格)

  * kāya-anupassī = 「身を随観する者」

* tasmiṃ samaye = 「その時に」

* viharati = 「住する」

* ātāpī = 「熱心に(精進を具えた者として)」

* sampajāno = 「正知ある者として」

* satimā = 「念ある者として」

* vineyya = 「調伏して、除いて」(vineti の連続体)

* loke = 「世間における」

* abhijjhā-domanassaṃ = 「貪欲と憂いを」

→ 「比丘たちよ、その時、比丘は身において身を随観する者として住する。熱心に、正知をもち、念をもち、世間における貪欲と憂いを除いて。」

7. Taṃ kissa hetu?

* taṃ = 「それは」

* kissa hetu = 「何の因によってか」→「なぜか」

→ 「それはなぜか。」

8. Kāyaññatarāhaṃ, bhikkhave, etaṃ vadāmi, yadidaṃ – assāsapassāsaṃ.

* kāya-aññataraṃ = 「身のうちの一つ(である)」

  * aññatara = 「あるもの、一種のもの」

* ahaṃ = 「私は」

* etaṃ vadāmi = 「これを～と説く」

* yadidaṃ = 「すなわち」

* assāsa-passāsaṃ = 「出息と入息を」

→ 「比丘たちよ、私はこれを、すなわち出息と入息を、身のうちの一つ(ある種の身)であると説く。」

9. Tasmātiha, bhikkhave, kāye kāyānupassī bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* tasmātiha = 「それゆえここに」(tasmā + iha)

→ 「それゆえここに、比丘たちよ、その時、比丘は身において身を随観する者として住する。熱心に、正知をもち、念をもち、世間における貪欲と憂いを除いて。」

10. Yasmiṃ samaye, bhikkhave, bhikkhu pītippaṭisaṃvedī…pe…

  * pīti = 「喜」

→ 「比丘たちよ、比丘が『喜を感受して……』(中略)……」

11. sukhappaṭisaṃvedī…pe…

12. cittasaṅkhārappaṭisaṃvedī…pe…

  * citta-saṅkhāra = 「心行」(受と想、心にともなう行)

13. ‘passambhayaṃ cittasaṅkhāraṃ assasissāmī’ti sikkhati, ‘passambhayaṃ cittasaṅkhāraṃ passasissāmī’ti sikkhati –

14. vedanāsu vedanānupassī, bhikkhave, bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

* vedanāsu vedanānupassī = 「受において受を随観する者として」

  * vedanāsu = 「諸々の受において」(処格・複数)

  * vedanā-anupassī = 「受を随観する者」

15. Taṃ kissa hetu?

16. Vedanāññatarāhaṃ, bhikkhave, etaṃ vadāmi, yadidaṃ – assāsapassāsānaṃ sādhukaṃ manasikāraṃ.

* vedanā-aññataraṃ = 「受のうちの一つ(である)」

* assāsa-passāsānaṃ = 「出息と入息に対する」(属格・複数)

* sādhukaṃ = 「よく、十分に」

* manasikāraṃ = 「作意を、注意を向けることを」

→ 「比丘たちよ、私はこれを、すなわち出息と入息に対するよき作意を、受のうちの一つ(ある種の受)であると説く。」

17. Tasmātiha, bhikkhave, vedanāsu vedanānupassī bhikkhu tasmiṃ samaye viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ.

→ 「それゆえここに、比丘たちよ、その時、比丘は受において受を随観する者として住する。熱心に、正知をもち、念をもち、世間における貪欲と憂いを除いて。」

## 992 (3)

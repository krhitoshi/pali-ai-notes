
# 7. Mahākappinasuttaṃ

## 983 (1)

983. Sāvatthinidānaṃ. Tena kho pana samayena āyasmā mahākappino bhagavato avidūre nisinno hoti pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. Addasā kho bhagavā āyasmantaṃ mahākappinaṃ avidūre nisinnaṃ pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. Disvāna bhikkhū āmantesi –

‘‘Passatha no tumhe, bhikkhave, etassa bhikkhuno kāyassa iñjitattaṃ vā phanditattaṃ vā’’ti? ‘‘Yadāpi mayaṃ, bhante, taṃ āyasmantaṃ passāma saṅghamajjhe vā nisinnaṃ ekaṃ vā raho nisinnaṃ, tadāpi mayaṃ tassa āyasmato na passāma kāyassa iñjitattaṃ vā phanditattaṃ vā’’ti.

### 対訳

1. Mahākappinasuttaṃ

* mahākappina = 「マハーカッピナ（長老の名）」

* suttaṃ = 「経」

→ 「マハーカッピナ経」

2. Sāvatthinidānaṃ.

* sāvatthi = 「サーヴァッティー（舎衛城）」

* nidānaṃ = 「因縁、由来（この経の説かれた場所を示す定型句）」

→ 「サーヴァッティーでのことである。」

3. Tena kho pana samayena āyasmā mahākappino bhagavato avidūre nisinno hoti pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* tena kho pana samayena = 「さてその時」

* āyasmā mahākappino = 「尊者マハーカッピナは」

* bhagavato avidūre = 「世尊から遠くない所に」

* nisinno hoti = 「坐っていた」

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで」

* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに保って」

  * ujuṃ = 「まっすぐに」

  * kāyaṃ = 「身体を」

  * paṇidhāya = 「置いて、保って（paṇidahati の連続体）」

* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を面前に確立して」

  * parimukhaṃ = 「面前に」

  * satiṃ = 「念を」

  * upaṭṭhapetvā = 「確立して、現前させて（upaṭṭhāpeti の連続体）」

→ 「さてその時、尊者マハーカッピナは世尊から遠くない所で、結跏趺坐を組み、身体をまっすぐに保ち、念を面前に確立して坐っていた。」

4. Addasā kho bhagavā āyasmantaṃ mahākappinaṃ avidūre nisinnaṃ pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* addasā kho = 「見た（dassati/passati のアオリスト）」

* bhagavā = 「世尊は」

* āyasmantaṃ mahākappinaṃ = 「尊者マハーカッピナを」

* avidūre nisinnaṃ = 「遠くない所に坐っているのを」

* pallaṅkaṃ ābhujitvā ... upaṭṭhapetvā = 「結跏趺坐を組み～念を面前に確立して（前チャンクと同じ）」

→ 「世尊は、尊者マハーカッピナが遠くない所で、結跏趺坐を組み、身体をまっすぐに保ち、念を面前に確立して坐っているのを見た。」

5. Disvāna bhikkhū āmantesi –

* disvāna = 「見て（passati の連続体）」

* bhikkhū = 「比丘たちに」

* āmantesi = 「呼びかけた、告げた（āmanteti のアオリスト）」

→ 「見て、比丘たちに告げた。」

6. ‘‘Passatha no tumhe, bhikkhave, etassa bhikkhuno kāyassa iñjitattaṃ vā phanditattaṃ vā’’ti?

* passatha = 「あなたたちは見るか」

* no = 「（疑問の強調の小辞）はたして」

* tumhe = 「あなたたちは」

* bhikkhave = 「比丘たちよ」

* etassa bhikkhuno = 「この比丘の」

* kāyassa = 「身体の」

* iñjitattaṃ vā = 「動揺すること、あるいは」

  * iñjita = 「動いた（iñjati の過去分詞）」+ -tta（抽象名詞化）

* phanditattaṃ vā = 「震えること、あるいは」

  * phandita = 「震えた（phandati の過去分詞）」+ -tta（抽象名詞化）

* ti = 「～と（引用の終わり）」

→ 「『比丘たちよ、あなたたちは、この比丘の身体が動揺したり震えたりするのを見るか』と。」

7. ‘‘Yadāpi mayaṃ, bhante, taṃ āyasmantaṃ passāma saṅghamajjhe vā nisinnaṃ ekaṃ vā raho nisinnaṃ, tadāpi mayaṃ tassa āyasmato na passāma kāyassa iñjitattaṃ vā phanditattaṃ vā’’ti.

* yadāpi = 「～する時でも」

* mayaṃ = 「私たちが」

* bhante = 「尊師よ」

* taṃ āyasmantaṃ = 「その尊者を」

* passāma = 「見る」

* saṅghamajjhe vā nisinnaṃ = 「僧団の中に坐っているのを、あるいは」

  * saṅgha-majjhe = 「僧団の中に」

* ekaṃ vā raho nisinnaṃ = 「一人で人のいない所に坐っているのを、あるいは」

  * ekaṃ = 「一人で」

  * raho = 「人のいない所で、ひそかに」

* tadāpi = 「その時でも」

* mayaṃ = 「私たちは」

* tassa āyasmato = 「その尊者の」

* na passāma = 「見ない」

* kāyassa iñjitattaṃ vā phanditattaṃ vā = 「身体が動揺したり震えたりするのを」

* ti = 「～と（引用の終わり）」

→ 「『尊師よ、私たちがその尊者を、僧団の中に坐っている時でも、一人で人のいない所に坐っている時でも見る時、その時でも私たちは、その尊者の身体が動揺したり震えたりするのを見ません』と。」

## 983 (2)

‘‘Yassa, bhikkhave, samādhissa bhāvitattā bahulīkatattā neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā, tassa so, bhikkhave, bhikkhu samādhissa nikāmalābhī akicchalābhī akasiralābhī. Katamassa ca, bhikkhave, samādhissa bhāvitattā bahulīkatattā neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā?

‘‘Ānāpānassatisamādhissa, bhikkhave, bhāvitattā bahulīkatattā neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā. Kathaṃ bhāvite ca, bhikkhave, ānāpānassatisamādhimhi kathaṃ bahulīkate neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā?

‘‘Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā. So satova assasati, satova passasati…pe… ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati. Evaṃ bhāvite ca kho, bhikkhave, ānāpānassatisamādhimhi evaṃ bahulīkate neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā’’ti. Sattamaṃ.

### 対訳

1. Yassa, bhikkhave, samādhissa bhāvitattā bahulīkatattā neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā,

* yassa samādhissa = 「その三昧の（関係代名詞 ya の属格）」

* bhikkhave = 「比丘たちよ（呼格）」

* bhāvitattā = 「修習されたことによって（bhāvita + -tta の奪格）」

* bahulīkatattā = 「多く行われたことによって（bahulīkata + -tta の奪格）」

* neva = na + eva = 「決して～ない」

* kāyassa = 「身体の」

* iñjitattaṃ = 「動揺すること（iñjita + -tta）」

* vā ... vā = 「～や～」

* hoti = 「ある、生じる」

* phanditattaṃ = 「震動すること（phandita + -tta）」

* na cittassa = 「心の～もない」

→ 「比丘たちよ、その三昧が修習され、多く行われたことによって、身体の動揺も震動もなく、心の動揺も震動もないならば、」

2. tassa so, bhikkhave, bhikkhu samādhissa nikāmalābhī akicchalābhī akasiralābhī.

* tassa samādhissa = 「その三昧を（lābhī が属格を取る）」

* so bhikkhu = 「その比丘は」

* nikāmalābhī = 「望みのままに得る者（nikāma = 望み + lābhī = 得る者）」

* akicchalābhī = 「困難なく得る者（a + kiccha = 困難）」

* akasiralābhī = 「苦労なく得る者（a + kasira = 苦労）」

→ 「比丘たちよ、その比丘は、その三昧を望みのままに得る者、困難なく得る者、苦労なく得る者である。」

3. Katamassa ca, bhikkhave, samādhissa bhāvitattā bahulīkatattā neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā?

* katamassa samādhissa = 「どの三昧の（katama = どの、疑問代名詞）」

* ca = 「では、そして」

→ 「では比丘たちよ、どの三昧が修習され、多く行われたことによって、身体の動揺も震動もなく、心の動揺も震動もないのか？」

4. Ānāpānassatisamādhissa, bhikkhave, bhāvitattā bahulīkatattā neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā.

* ānāpānassatisamādhissa = 「出入息念三昧の（āna + apāna + sati + samādhi の属格）」

→ 「比丘たちよ、出入息念三昧が修習され、多く行われたことによって、身体の動揺も震動もなく、心の動揺も震動もない。」

5. Kathaṃ bhāvite ca, bhikkhave, ānāpānassatisamādhimhi kathaṃ bahulīkate neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā?

* kathaṃ = 「どのように」

* bhāvite = 「修習されたときに（処格、絶対処格構文）」

* ānāpānassatisamādhimhi = 「出入息念三昧が（処格）」

* bahulīkate = 「多く行われたときに（処格）」

→ 「では比丘たちよ、出入息念三昧がどのように修習され、どのように多く行われたときに、身体の動揺も震動もなく、心の動揺も震動もないのか？」

6. Idha, bhikkhave, bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに（この教えにおいて）」

* araññagato = 「森に行った者（arañña = 森 + gata）」

* rukkhamūlagato = 「樹の根元に行った者（rukkha = 樹 + mūla = 根元）」

* suññāgāragato = 「空き家に行った者（suñña = 空の + agāra = 家）」

* nisīdati = 「坐る」

* pallaṅkaṃ ābhujitvā = 「結跏趺坐を組んで（ābhujitvā = 曲げて、組んで）」

* ujuṃ kāyaṃ paṇidhāya = 「身体をまっすぐに保って（uju = まっすぐ、paṇidhāya = 定め置いて）」

* parimukhaṃ satiṃ upaṭṭhapetvā = 「面前に念を現前させて（parimukhaṃ = 面前に、upaṭṭhapetvā = 現前させて）」

→ 「比丘たちよ、ここに比丘は、森に行き、あるいは樹の根元に行き、あるいは空き家に行って、結跏趺坐を組み、身体をまっすぐに保ち、面前に念を現前させて坐る。」

7. So satova assasati, satova passasati…pe…

* so = 「彼は」

* sato va = 「念をそなえた者としてまさに（sata = 念ある + eva）」

* assasati = 「出息する（外へ息を出す）」

* passasati = 「入息する（内へ息を入れる）」

* …pe… = 「（中略。十六事の定型句が続く）」

→ 「彼は念をそなえて出息し、念をそなえて入息する。……（中略）……」

8. ‘paṭinissaggānupassī assasissāmī’ti sikkhati, ‘paṭinissaggānupassī passasissāmī’ti sikkhati.

* paṭinissaggānupassī = 「捨遣を随観する者（paṭinissagga = 捨て去ること + anupassī = 随観する者）」

* assasissāmi = 「私は出息しよう（未来形）」

* passasissāmi = 「私は入息しよう（未来形）」

* ti = 「～と（引用）」

* sikkhati = 「学ぶ、修学する」

→ 「『捨遣を随観しながら出息しよう』と学び、『捨遣を随観しながら入息しよう』と学ぶ。」

9. Evaṃ bhāvite ca kho, bhikkhave, ānāpānassatisamādhimhi evaṃ bahulīkate neva kāyassa iñjitattaṃ vā hoti phanditattaṃ vā, na cittassa iñjitattaṃ vā hoti phanditattaṃ vā’’ti.

* evaṃ = 「このように」

* bhāvite ... bahulīkate = 「修習され～多く行われたときに（絶対処格）」

* ca kho = 「実に」

* ti = 「～と（経の結び）」

→ 「比丘たちよ、実にこのように出入息念三昧が修習され、このように多く行われたときに、身体の動揺も震動もなく、心の動揺も震動もない。」

10. Sattamaṃ.

* sattamaṃ = 「第七（経）」

→ 「第七（経、終わり）。」

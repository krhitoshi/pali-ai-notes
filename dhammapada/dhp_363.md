
363.

‘‘Yo mukhasaṃyato bhikkhu, mantabhāṇī anuddhato;

Atthaṃ dhammañca dīpeti, madhuraṃ tassa bhāsita’’nti.

Tattha mukhasaṃyatoti dāsacaṇḍālādayopi ‘‘tvaṃ dujjāto, tvaṃ dussīlo’’tiādīnaṃ avacanatāya mukhena saṃyato. Mantabhāṇīti mantā vuccati paññā, tāya bhaṇanasīlo. Anuddhatoti nibbutacitto. Atthaṃ dhammañca dīpetīti bhāsitatthañceva desanādhammañca katheti. Madhuranti evarūpassa bhikkhuno bhāsitaṃ madhuraṃ nāma. Yo pana atthameva sampādeti, na pāḷiṃ, pāḷiṃyeva sampādeti, na atthaṃ, ubhayaṃ vā pana na sampādeti, tassa bhāsitaṃ madhuraṃ nāma na hotīti.

## 対訳

1. Yo mukhasaṃyato bhikkhu,

* yo = 「～である者は」
* mukhasaṃyato = mukha + saṃyata
  * mukha = 「口」→ ここでは「言葉・発言」
  * saṃyata = 「制御された・抑制された」
  → 「言葉において抑制された」
* bhikkhu = 「比丘、出家修行者」

→「口（言葉）をよく抑制している比丘である者は」

---

2. mantabhāṇī anuddhato;

* mantabhāṇī = manta + bhāṇin
  * manta = 「思慮・智慧・よく考えた判断（= paññā）」
  * bhāṇin / bhāṇī = 「語る者」
  → 「よく考え（智慧をもって）語る者」
* anuddhato
  * a-（否定）+ uddhata = 「興奮した、はねあがった、浮ついた」
  → 「はねあがっていない、興奮していない、静まった」

→「智慧をもって語り、心が浮ついていない者であり」

---

3. Atthaṃ dhammañca dīpeti,

* atthaṃ
  * attha = 「意味・趣旨・利益・目的」
* dhammañ ca = dhammaṃ + ca
  * dhammaṃ = 「法、教え（ブッダの教え）」
  * ca = 「そして」
  → 「意味と法（教え）とを」
* dīpeti = 「明らかにする・説き示す・解き明かす」

→「その（語の）意味と法（教え）とを解き明かし」

---

4. madhuraṃ tassa bhāsita’’nti.

* madhuraṃ = 「甘い、ここでは『心地よい・耳に快い』」
* tassa = 「その人の、その比丘の」
* bhāsitaṃ = 「話した言葉、発言」
* ’nti = iti = 「～である、と」

→「その人の語ることは『甘美である（心地よい）』と言われる。」

---

5. Tattha mukhasaṃyatoti dāsacaṇḍālādayopi ‘‘tvaṃ dujjāto, tvaṃ dussīlo’’tiādīnaṃ avacanatāya mukhena saṃyato.

* tattha = 「ここで、この文句においては」
* mukhasaṃyato ti = 「mukhasaṃyata とは…という意味である」
* dāsacaṇḍālādayo pi
  * dāsa = 「奴隷」
  * caṇḍāla = 「賤民、被差別階層」
  * -ādayo = 「～など」
  * pi = 「～でさえも」
  → 「奴隷や賤民などに対してさえも」
* “tvaṃ dujjāto, tvaṃ dussīlo” ti ādināṃ
  * tvaṃ = 「お前は」
  * dujjāto = 「悪い家柄の者、卑しい生まれの者」
  * dussīlo = 「戒を守らない者、品行の悪い者」
  * ti ādināṃ = 「『…』などという（種々の言葉）の」
* avacanatāya
  * a-vacana-tā = 「言わないこと、そのように言わない状態」
* mukhena saṃyato = 「口によって抑制されている、口で自制している」

→「ここで『口を抑制した者』とは、奴隷や賤民などに対してさえも『お前は卑しい生まれだ』『お前は悪行の人だ』などといった悪口を決して口にしないことによって、言葉の上で自らを抑制している者、という意味である。」

---

6. Mantabhāṇīti mantā vuccati paññā, tāya bhaṇanasīlo.

* mantabhāṇī ti = 「mantabhāṇī とは…という意味である」
* mantā = 「manta」→「思慮・熟慮・智慧」
* vuccati = 「～と呼ばれる」
* paññā = 「智慧」
  → 「‘manta’ とは智慧と呼ばれる」
* tāya = 「その（智慧）によって」
* bhaṇana-sīlo
  * bhaṇana = 「語ること」
  * sīla = 「性質、習性」
  → 「語ることを習いとしている者」

→「『mantabhāṇī』とは、‘manta’ すなわち智慧と呼ばれるものによって語ることを習いとしている者、という意味である。」

---

7. Anuddhatoti nibbutacitto.

* anuddhato ti = 「anuddhata とは…という意味である」
* nibbutacitto
  * nibbuta = 「冷めた、鎮まった、涼やかな」
  * citto = 「心」
  → 「心が鎮まっている、静まった心の者」

→「『anuddhato』とは、心が鎮まって静まり返っている者、という意味である。」

---

8. Atthaṃ dhammañca dīpetīti bhāsitatthañceva desanādhammañca katheti.

* atthaṃ dhammañ ca dīpetī ti = 「『意味と法とを明らかにする』とは…という意味である」
* bhāsita-atthaṃ ca eva
  * bhāsita = 「語られたもの、言葉」
  * atthaṃ = 「意味」
  * ca eva = 「そしてちょうどその」
  → 「語られた（言葉）の意味そのものと」
* desanā-dhammaṃ ca
  * desanā = 「説法、教説」
  * dhammaṃ = 「法、教え」
  → 「説法としての教えとを」
* katheti = 「語る、説く」

→「『意味と法とを明らかにする』とは、語られた文句の意味そのものと、説法としての教えの内容とを、きちんと語り明かすことである。」

---

9. Madhuranti evarūpassa bhikkhuno bhāsitaṃ madhuraṃ nāma.

* madhuraṃ ti = 「madhuraṃ とは…という意味である」
* evarūpassa bhikkhuno = 「このような性質を具えた比丘の」
* bhāsitaṃ = 「語るところ、発言」
* madhuraṃ nāma = 「『甘美である』と呼ばれる」

→「『madhuraṃ（甘美である）』とは、このような比丘の語ることを『甘美である』と呼ぶ、ということである。」

---

10. Yo pana atthameva sampādeti, na pāḷiṃ, pāḷiṃyeva sampādeti, na atthaṃ, ubhayaṃ vā pana na sampādeti, tassa bhāsitaṃ madhuraṃ nāma na hotīti.

* yo pana = 「しかし、～である者がいる」
* atthaṃ eva sampādeti, na pāḷiṃ
  * atthaṃ eva = 「もっぱら意味だけを」
  * sampādeti = 「整える、成就させる、よく説明する」
  * na pāḷiṃ = 「しかしパーリ（原文）は整えない」
  → 「意味だけはきちんと説明し、パーリの文句は整えない者」
* pāḷiṃ yeva sampādeti, na atthaṃ
  * pāḷiṃ yeva = 「パーリの文句だけを」
  * na atthaṃ = 「しかし意味は整えない」
  → 「パーリの文句だけを立派に唱え、意味を整えない者」
* ubhayaṃ vā pana na sampādeti
  * ubhayaṃ = 「両方」
  * na sampādeti = 「どちらも整えない」
  → 「その両方を整えない者」
* tassa bhāsitaṃ = 「その者の語るところは」
* madhuraṃ nāma na hoti = 「『甘美である』とは言われない」

→「しかし、意味だけをよく説明してパーリ（原文）の扱いを整えない者、あるいはパーリの文句だけをうまく整えて唱えるが意味をきちんと示さない者、さらにはその両方とも十分に整えない者――そのような者の語るところは、『甘美である』とは決して呼ばれないのである。」

---

## 意訳

「口をよく抑制している比丘、つまり奴隷や賤民に対してでさえ『お前は卑しい生まれだ』『お前は品行が悪い』などの言葉を決して口にしない者であって、  
智慧によってよく考えながら語り、心が興奮せず静まりきっている者がいる。

その比丘は、単に文句を唱えるだけではなく、その語の意味と説法としての教えの内容を、両方ともはっきりと説き明かす。  
このような比丘の語る言葉こそ、『甘美な言葉』と呼ばれる。

これに対して、意味だけを説明してパーリの原文をおろそかにする者、  
あるいはパーリの文句だけをうまく整えて唱えながら、その意味を明らかにしない者、  
さらには意味もパーリもどちらもきちんと整えない者――  
そのような者の語る言葉は、『甘美な言葉』とは言えないのである。」

## 対訳

1. Yo mukhasaṃyato bhikkhu, mantabhāṇī anuddhato;

* yo = 「～であるところの（関係代名詞）」
* mukhasaṃyato = 「口を慎んでいる、言葉を制している（mukha + saṃyata）」
* bhikkhu = 「比丘、修行僧」
* mantabhāṇī = 「知恵をもって語る者（mantā + bhāṇin）」
* anuddhato = 「浮ついていない、落ち着いた（a + uddhata）」

→「口を慎み、知恵をもって語り、浮ついたところのない比丘は、」

2. Atthaṃ dhammañca dīpeti, madhuraṃ tassa bhāsitanti.

* atthaṃ = 「義、意味、目的」
* dhammañca = 「法（真理）を、教えを（dhammaṃ + ca）」
* dīpeti = 「明らかにする、説き示す」
* madhuraṃ = 「甘い、心地よい、麗しい」
* tassa = 「彼の」
* bhāsitaṃ = 「語られたこと、言葉」

→「義（意味）と法（教え）を説き示す。彼の言葉は麗しい。」

3. Tattha mukhasaṃyatoti dāsacaṇḍālādayopi ‘‘tvaṃ dujjāto, tvaṃ dussīlo’’tiādīnaṃ avacanatāya mukhena saṃyato.

* tattha = 「そこ（その句）において」
* mukhasaṃyato ti = 「『口を慎んでいる』とは」
* dāsa-caṇḍāla-ādayo pi = 「奴隷やチャンダーラ（不可触民）らに対しても」
* tvaṃ dujjāto = 「お前は卑しい生まれだ」
* tvaṃ dussīlo = 「お前は戒行が悪い」
* ti-ādīnaṃ = 「～など（の言葉）」
* avacanatāya = 「言わないことによって」
* mukhena saṃyato = 「口によって制していること」

→「そのうち『口を慎んでいる』とは、奴隷やチャンダーラらに対しても『お前は卑しい生まれだ』『お前は不道徳だ』などと言わないことで、口を制している者のことである。」

4. Mantabhāṇīti mantā vuccati paññā, tāya bhaṇanasīlo.

* mantabhāṇī ti = 「『知恵をもって語る者』とは」
* mantā = 「マンター（計策・知恵）」
* vuccati = 「～と言われる」
* paññā = 「知恵、般若」
* tāya = 「それ（知恵）によって」
* bhaṇanasīlo = 「語る性質、語る習慣があること」

→「『知恵をもって語る者』とは、マンター（知恵）とは知恵のことであり、それ（知恵）に基づいて語る習慣がある者のことである。」

5. Anuddhatoti nibbutacitto.

* anuddhato ti = 「『浮ついていない』とは」
* nibbutacitto = 「静まりかえった心をもつ者、寂静なる心の者」

→「『浮ついていない』とは、心が静まりかえっている者のことである。」

6. Atthaṃ dhammañca dīpetīti bhāsitatthañceva desanādhammañca katheti.

* atthaṃ dhammañca dīpeti ti = 「『義と法を説き示す』とは」
* bhāsitatthañ ca eva = 「語られたことの意味（義）と」
* desanādhammañ ca = 「説示された教え（法）を」
* katheti = 「語る、述べる」

→「『義と法を説き示す』とは、語られる内容の意味と、説示される教えの両方を語るということである。」

7. Madhuranti evarūpassa bhikkhuno bhāsitaṃ madhuraṃ nāma.

* madhuraṃ ti = 「『麗しい（甘い）』とは」
* evarūpassa bhikkhuno = 「このような（徳を備えた）比丘の」
* bhāsitaṃ = 「言葉」
* madhuraṃ nāma = 「麗しい（甘い）と呼ばれる」

→「『麗しい』とは、このような比丘の言葉こそが麗しいと言われるのである。」

8. Yo pana atthameva sampādeti, na pāḷiṃ, pāḷiṃyeva sampādeti, na atthaṃ, ubhayaṃ vā pana na sampādeti, tassa bhāsitaṃ madhuraṃ nāma na hotīti.

* yo pana = 「しかしながら、～である者は」
* atthaṃ eva sampādeti = 「意味（義）だけを達成し（説き）」
* na pāḷiṃ = 「聖典（原文）を説かない」
* pāḷim eva sampādeti = 「聖典（原文）だけを達成し（説き）」
* na atthaṃ = 「意味（義）を説かない」
* ubhayaṃ vā pana na sampādeti = 「あるいは両方とも達成しない（説かない）」
* tassa bhāsitaṃ = 「彼の言葉は」
* madhuraṃ nāma na hoti = 「麗しいとは呼ばれない」

→「しかし、意味だけを説いて聖典を説かない者、聖典だけを説いて意味を説かない者、あるいはその両方とも説かない者の言葉は、麗しいとは言われないのである。」

## 意訳

口を慎み、知恵に基づいて語り、心が浮つくことなく落ち着いている比丘は、教えの正しい意味とその真理の両方を説き示します。そのような比丘の言葉こそが、麗しく響くのです。

ここで「口を慎む」とは、たとえ身分の低い者に対しても「卑しい生まれだ」「不道徳だ」といった罵倒を口にしないことを指します。「知恵をもって語る」とは、常に智慧に基づいて語ることです。「浮つかない」とは、心が静まりかえっている状態をいいます。
また、「義と法を説き示す」とは、教えの言葉（聖典）とその深い意味の両方を正しく伝えることを意味します。もし、意味だけを説いて原文を疎かにしたり、原文だけを唱えて意味を解かなかったり、あるいはその両方を欠いたりするならば、その言葉が麗しいと言われることはありません。聖典の言葉と意味の両方を備えてこそ、比丘の言葉は真に麗しいものとなるのです。

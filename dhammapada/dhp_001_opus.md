1.

‘‘Manopubbaṅgamā dhammā, manoseṭṭhā manomayā;

Manasā ce paduṭṭhena, bhāsati vā karoti vā;

Tato naṃ dukkhamanveti, cakkaṃva vahato pada’’nti.

Tattha manoti kāmāvacarakusalādibhedaṃ sabbampi catubhūmikacittaṃ. Imasmiṃ pana pade tadā tassa vejjassa uppannacittavasena niyamiyamānaṃ vavatthāpiyamānaṃ paricchijjiyamānaṃ domanassasahagataṃ paṭighasampayuttacittameva labbhati. Pubbaṅgamāti tena paṭhamagāminā hutvā samannāgatā. Dhammāti guṇadesanāpariyattinissattanijjīvavasena cattāro dhammā nāma. Tesu –

‘‘Na hi dhammo adhammo ca, ubho samavipākino;

Adhammo nirayaṃ neti, dhammo pāpeti suggati’’nti. (theragā. 304; jā. 1.15.386) –

Ayaṃ guṇadhammo nāma. ‘‘Dhammaṃ vo, bhikkhave, desessāmi ādikalyāṇa’’nti (ma. ni. 3.420) ayaṃ desanādhammo nāma. ‘‘Idha pana, bhikkhave, ekacce kulaputtā dhammaṃ pariyāpuṇanti suttaṃ geyya’’nti (ma. ni. 1.239) ayaṃ pariyattidhammo nāma. ‘‘Tasmiṃ kho pana samaye dhammā honti, khandhā hontī’’ti (dha. sa. 121) ayaṃ nissattadhammo nāma, nijjīvadhammotipi eso eva. Tesu imasmiṃ ṭhāne nissattanijjīvadhammo adhippeto. So atthato tayo arūpino khandhā vedanākkhandho saññākkhandho saṅkhārakkhandhoti. Ete hi mano pubbaṅgamo etesanti manopubbaṅgamā nāma.

Kathaṃ panetehi saddhiṃ ekavatthuko ekārammaṇo apubbaṃ acarimaṃ ekakkhaṇe uppajjamāno mano pubbaṅgamo nāma hotīti? Uppādapaccayaṭṭhena. Yathā hi bahūsu ekato gāmaghātādīni kammāni karontesu ‘‘ko etesaṃ pubbaṅgamo’’ti vutte yo nesaṃ paccayo hoti, yaṃ nissāya te taṃ kammaṃ karonti, so datto vā mitto vā tesaṃ pubbaṅgamoti vuccati, evaṃsampadamidaṃ veditabbaṃ. Iti uppādapaccayaṭṭhena mano pubbaṅgamo etesanti manopubbaṅgamā. Na hi te mane anuppajjante uppajjituṃ sakkonti, mano pana ekaccesu cetasikesu anupajjantesupi uppajjatiyeva. Adhipativasena pana mano seṭṭho etesanti manoseṭṭho. Yathā hi corādīnaṃ corajeṭṭhakādayo adhipatino seṭṭhā. Tathā tesampi mano adhipati manova seṭṭhā. Yathā pana dāruādīhi nipphannāni tāni tāni bhaṇḍāni dārumayādīni nāma honti, tathā tepi manato nipphannattā manomayā nāma.

Paduṭṭhenāti āgantukehi abhijjhādīhi dosehi paduṭṭhena. Pakatimano hi bhavaṅgacittaṃ, taṃ apaduṭṭhaṃ. Yathā hi pasannaṃ udakaṃ āgantukehi nīlādīhi upakkiliṭṭhaṃ nīlodakādibhedaṃ hoti, na ca navaṃ udakaṃ, nāpi purimaṃ pasannaudakameva, tathā tampi āgantukehi abhijjhādīhi dosehi paduṭṭhaṃ hoti, na ca navaṃ cittaṃ, nāpi purimaṃ bhavaṅgacittameva, tenāha bhagavā – ‘‘pabhassaramidaṃ, bhikkhave, cittaṃ, tañca kho āgantukehi upakkilesehi upakkiliṭṭha’’nti (a. ni. 1.49). Evaṃ manasā ce paduṭṭhena, bhāsati vā karoti vā so bhāsamāno catubbidhaṃ vacīduccaritameva bhāsati, karonto tividhaṃ kāyaduccaritameva karoti, abhāsanto akaronto tāya abhijjhādīhi paduṭṭhamānasatāya tividhaṃ manoduccaritaṃ pūreti. Evamassa dasa akusalakammapathā pāripūriṃ gacchanti.

Tato naṃ dukkhamanvetīti tato tividhaduccaritato taṃ puggalaṃ dukkhaṃ anveti, duccaritānubhāvena catūsu apāyesu, manussesu vā tamattabhāvaṃ gacchantaṃ kāyavatthukampi itarampīti iminā pariyāyena kāyikacetasikaṃ vipākadukkhaṃ anugacchati. Yathā kiṃ? Cakkaṃva vahato padanti dhure yuttassa dhuraṃ vahato balibaddassa padaṃ cakkaṃ viya. Yathā hi so ekampi divasaṃ dvepi pañcapi dasapi aḍḍhamāsampi māsampi vahanto cakkaṃ nivattetuṃ jahituṃ na sakkoti , atha khvassa purato abhikkamantassa yugaṃ gīvaṃ bādhati, pacchato paṭikkamantassa cakkaṃ ūrumaṃsaṃ paṭihanati. Imehi dvīhi ākārehi bādhantaṃ cakkaṃ tassa padānupadikaṃ hoti; tatheva manasā paduṭṭhena tīṇi duccaritāni pūretvā ṭhitaṃ puggalaṃ nirayādīsu tattha tattha gatagataṭṭhāne duccaritamūlakaṃ kāyikampi cetasikampi dukkhamanubandhatīti.

### 偈文（ダンマパダ第1偈）

1. Manopubbaṅgamā dhammā, manoseṭṭhā manomayā;
   Manasā ce paduṭṭhena, bhāsati vā karoti vā;
   Tato naṃ dukkhamanveti, cakkaṃva vahato padanti.

* mano-pubbaṅgamā = 「意（こころ）を先導者とする」
  * mano = 「意、心」
  * pubbaṅgama = 「先行する者、先導者」
* dhammā = 「諸法（もろもろの法）」
* mano-seṭṭhā = 「意を最上とする」
* mano-mayā = 「意より成る、意から生じた」
* manasā ce paduṭṭhena = 「もし汚れた意によって」
  * paduṭṭha = 「汚された、穢れた（padussati の過去分詞）」
* bhāsati vā = 「語るならば」
* karoti vā = 「行うならば」
* tato = 「それゆえに、そこから」
* naṃ = 「その者を」
* dukkham anveti = 「苦が追随する」
* cakkaṃ va vahato padaṃ = 「荷を引く（牛の）足跡に車輪が（つき従う）ように」
  * cakka = 「車輪」
  * vahato = 「（荷を）運ぶ者の（vahati の現在分詞・属格）」
  * pada = 「足、足跡」

→ 「諸法は意を先導者とし、意を最上とし、意より成る。
　もし汚れた意をもって語り、あるいは行うならば、
　そこから苦がその者に従う——荷を引く牛の足に車輪が従うように。」

### 註釈

2. Tattha mano ti kāmāvacarakusalādibhedaṃ sabbam pi catubhūmikacittaṃ.

* tattha = 「そこで（この偈において）」
* mano ti = 「『意』とは」
* kāmāvacara-kusala-ādi-bhedaṃ = 「欲界善〔心〕などの区別をもつ」
  * kāmāvacara = 「欲界の」
  * kusala = 「善」
  * ādi = 「など」
  * bheda = 「区別、種類」
* sabbam pi = 「すべての」
* catubhūmika-cittaṃ = 「四地の心（欲界・色界・無色界・出世間）」

→ 「ここで『意（mano）』とは、欲界善心などの区別をもつ、すべての四地の心のことである。」

3. Imasmiṃ pana pade tadā tassa vejjassa uppannacittavasena niyamiyamānaṃ vavatthāpiyamānaṃ paricchijjiyamānaṃ domanassasahagataṃ paṭighasampayuttacittam eva labbhati.

* imasmiṃ pana pade = 「しかし、この（偈の）句においては」
* tadā = 「その時」
* tassa vejjassa = 「かの医者の」
* uppanna-citta-vasena = 「生じた心に基づいて」
* niyamiyamānaṃ = 「限定されるところの」
* vavatthāpiyamānaṃ = 「確定されるところの」
* paricchijjiyamānaṃ = 「区分されるところの」
* domanassa-sahagataṃ = 「憂（ドーマナッサ）を伴う」
* paṭigha-sampayutta-cittam eva = 「瞋恚（パティガ）と相応した心にほかならない」
* labbhati = 「得られる（＝そのように理解される）」

→ 「しかしこの句においては、その時かの医者に生じた心に基づいて
　限定・確定・区分されるところの、
　憂を伴い瞋恚と相応した心のみが〔ここでの『意』として〕理解される。」

4. Pubbaṅgamā ti tena paṭhamagāminā hutvā samannāgatā.

* pubbaṅgamā ti = 「『先導者とする（pubbaṅgamā）』とは」
* tena = 「それ（心）を」
* paṭhama-gāminā = 「最初に行く者として」
* hutvā = 「あって」
* samannāgatā = 「具えた」

→ 「『先導者とする』とは、それ（心）を最初に行く者として具えている、ということである。」

5. Dhammā ti guṇa-desanā-pariyatti-nissatta-nijjīva-vasena cattāro dhammā nāma.

* dhammā ti = 「『諸法（dhammā）』とは」
* guṇa = 「徳、功徳」
* desanā = 「教説」
* pariyatti = 「学文、聖典」
* nissatta = 「無有情（有情なきもの）」
* nijjīva = 「無命（生命なきもの）」
* vasena = 「～として、～の意味で」
* cattāro dhammā nāma = 「四種の法と呼ばれる」

→ 「『諸法』とは、功徳・教説・学文・無有情無命という意味で、四種の法がある。」

6. Tesu –
   "Na hi dhammo adhammo ca, ubho samavipākino;
   Adhammo nirayaṃ neti, dhammo pāpeti suggatī" ti. –
   Ayaṃ guṇadhammo nāma.

* tesu = 「それらのうち」
* na hi = 「実に～ない」
* dhammo adhammo ca = 「法と非法とは」
* ubho = 「両者は」
* sama-vipākino = 「等しい果報をもつものではない」
* adhammo = 「非法（悪）は」
* nirayaṃ neti = 「地獄に導く」
* dhammo = 「法（善）は」
* pāpeti suggatī ti = 「善趣に至らしめる」
* ayaṃ guṇadhammo nāma = 「これが功徳としての法（guṇa-dhamma）と呼ばれる」

→ 「それらのうち——
　『法と非法とは、両者ともに等しい果報あるにあらず。
　非法は地獄に導き、法は善趣に至らしめる。』
　これが功徳としての法（guṇa-dhamma）と呼ばれるものである。」

7. "Dhammaṃ vo, bhikkhave, desessāmi ādikalyāṇan" ti ayaṃ desanādhammo nāma.

* dhammaṃ vo = 「汝らに法を」
* bhikkhave = 「比丘たちよ」
* desessāmi = 「わたしは説こう」
* ādi-kalyāṇaṃ = 「初め善き〔法〕を」
* ayaṃ desanā-dhammo nāma = 「これが教説としての法（desanā-dhamma）と呼ばれる」

→ 「『比丘たちよ、わたしは汝らに初め善き法を説こう』——
　これが教説としての法と呼ばれるものである。」

8. "Idha pana, bhikkhave, ekacce kulaputtā dhammaṃ pariyāpuṇanti suttaṃ geyyan" ti ayaṃ pariyattidhammo nāma.

* idha pana = 「さてここに」
* bhikkhave = 「比丘たちよ」
* ekacce kulaputtā = 「ある良家の子らは」
* dhammaṃ pariyāpuṇanti = 「法を学ぶ」
* suttaṃ geyyaṃ = 「経・応頌（を）」
* ayaṃ pariyatti-dhammo nāma = 「これが学文としての法（pariyatti-dhamma）と呼ばれる」

→ 「『さてここに、比丘たちよ、ある良家の子らは法を学ぶ——経・応頌〔など〕を。』
　これが学文としての法と呼ばれるものである。」

9. "Tasmiṃ kho pana samaye dhammā honti, khandhā hontī" ti ayaṃ nissattadhammo nāma, nijjīvadhammoti pi eso eva.

* tasmiṃ kho pana samaye = 「さてその時には」
* dhammā honti = 「諸法がある」
* khandhā honti = 「諸蘊がある」
* ayaṃ nissatta-dhammo nāma = 「これが無有情としての法（nissatta-dhamma）と呼ばれる」
* nijjīva-dhammo ti pi eso eva = 「無命としての法（nijjīva-dhamma）とも、それはまさにこの同じものである」

→ 「『さてその時、諸法があり、諸蘊がある。』
　これが無有情としての法と呼ばれる。無命としての法も、まさにこれと同じである。」

10. Tesu imasmiṃ ṭhāne nissattanijjīvadhammo adhippeto.

* tesu = 「それら（四種の法）のうち」
* imasmiṃ ṭhāne = 「この箇所においては」
* nissatta-nijjīva-dhammo = 「無有情・無命の法が」
* adhippeto = 「意図されている」

→ 「それらのうち、この箇所においては無有情・無命の法が意図されている。」

11. So atthato tayo arūpino khandhā vedanākkhandho saññākkhandho saṅkhārakkhandho ti.

* so = 「それは」
* atthato = 「意味としては」
* tayo arūpino khandhā = 「三つの非色蘊である」
* vedanā-kkhandho = 「受蘊」
* saññā-kkhandho = 「想蘊」
* saṅkhāra-kkhandho = 「行蘊」

→ 「それは意味としては三つの非色蘊、すなわち受蘊・想蘊・行蘊のことである。」

12. Ete hi mano pubbaṅgamo etesaṃ ti manopubbaṅgamā nāma.

* ete hi = 「これらは実に」
* mano pubbaṅgamo etesaṃ = 「意がこれらの先導者である」
* ti = 「ということで」
* mano-pubbaṅgamā nāma = 「『意を先導者とする（manopubbaṅgamā）』と呼ばれる」

→ 「これら〔三蘊〕は、意がそれらの先導者であるから、
　『意を先導者とする（manopubbaṅgamā）』と呼ばれるのである。」

13. Kathaṃ pan' etehi saddhiṃ eka-vatthuko ek'ārammaṇo apubbaṃ acarimaṃ ekakkhaṇe uppajjamāno mano pubbaṅgamo nāma hotī ti?

* kathaṃ pana = 「しかしどのようにして」
* etehi saddhiṃ = 「これら〔の心所〕とともに」
* eka-vatthuko = 「同一の所依をもち」
* ek'ārammaṇo = 「同一の所縁をもち」
* apubbaṃ acarimaṃ = 「前後なく」
* eka-kkhaṇe uppajjamāno = 「一刹那に生じる」
* mano = 「意（心）が」
* pubbaṅgamo nāma hoti = 「先導者となるのか」

→ 「しかし、これら〔心所〕とともに同一の所依・同一の所縁をもち、
　前後なく同一刹那に生じる意が、どのようにして先導者となるのか？」

14. Uppādapaccayaṭṭhena.

* uppāda-paccaya-aṭṭhena = 「生起の縁という意味において」

→ 「〔答える——〕生起の縁という意味においてである。」

15. Yathā hi bahūsu ekato gāmaghātādīni kammāni karontesu "ko etesaṃ pubbaṅgamo" ti vutte yo nesaṃ paccayo hoti, yaṃ nissāya te taṃ kammaṃ karonti, so datto vā mitto vā tesaṃ pubbaṅgamo ti vuccati, evaṃsampadam idaṃ veditabbaṃ.

* yathā hi = 「たとえば」
* bahūsu ekato = 「多くの者が一緒になって」
* gāma-ghāta-ādīni kammāni karontesu = 「村の略奪などの行為をする時に」
* ko etesaṃ pubbaṅgamo ti vutte = 「『彼らの首謀者は誰か』と問われた時」
* yo nesaṃ paccayo hoti = 「彼らの縁（原因）となる者」
* yaṃ nissāya = 「誰に依って」
* te taṃ kammaṃ karonti = 「彼らがその行為をなすか」
* so datto vā mitto vā = 「そのダッタとかミッタとかいう者が」
* tesaṃ pubbaṅgamo ti vuccati = 「彼らの首謀者（先導者）と呼ばれる」
* evaṃ-sampadam idaṃ veditabbaṃ = 「このように、これも同様に理解すべきである」

→ 「たとえば、多くの者が共に村の略奪などの行為をなす時、
　『彼らの首謀者は誰か』と問えば、
　彼らの縁となり、彼に依って彼らがその行為をなすところの、
　ダッタなりミッタなりが、彼らの首謀者と呼ばれる。
　このように、ここでも同様に理解すべきである。」

16. Iti uppādapaccayaṭṭhena mano pubbaṅgamo etesaṃ ti manopubbaṅgamā.

* iti = 「このように」
* uppāda-paccaya-aṭṭhena = 「生起の縁という意味で」
* mano pubbaṅgamo etesaṃ = 「意がこれらの先導者である」
* ti = 「ということで」
* manopubbaṅgamā = 「『意を先導者とする』〔と言われる〕」

→ 「このように、生起の縁という意味で、意がこれらの先導者であるから、
　『意を先導者とする』と言われるのである。」

17. Na hi te mane anuppajjante uppajjituṃ sakkonti, mano pana ekaccesu cetasikesu anuppajjantesu pi uppajjati yeva.

* na hi = 「なぜなら」
* te = 「それら〔心所〕は」
* mane anuppajjante = 「意が生じないのに」
* uppajjituṃ sakkonti = 「生じることはできない」
* mano pana = 「しかし意は」
* ekaccesu cetasikesu anuppajjantesu pi = 「ある心所が生じなくても」
* uppajjati yeva = 「まさに生じる」

→ 「なぜなら、それら〔心所〕は意が生じなければ生じることができない。
　しかし意は、ある心所が生じなくても、まさに生じるからである。」

18. Adhipativasena pana mano seṭṭho etesaṃ ti manoseṭṭhā.

* adhipati-vasena = 「主宰（支配者）という意味で」
* pana = 「さらに」
* mano seṭṭho etesaṃ = 「意がこれらの最上である」
* ti = 「ということで」
* manoseṭṭhā = 「『意を最上とする』〔と言われる〕」

→ 「さらに、主宰という意味で意がこれらの最上であるから、
　『意を最上とする（manoseṭṭhā）』と言われる。」

19. Yathā hi corādīnaṃ corajeṭṭhakādayo adhipatino seṭṭhā. Tathā tesam pi mano adhipati mano va seṭṭhā.

* yathā hi = 「たとえば」
* corādīnaṃ = 「盗賊たちなどにとって」
* cora-jeṭṭhaka-ādayo = 「盗賊の首領などが」
* adhipatino = 「主宰者であり」
* seṭṭhā = 「最上者である」
* tathā = 「そのように」
* tesam pi = 「それら〔心所〕にとっても」
* mano adhipati = 「意が主宰者であり」
* mano va seṭṭhā = 「意こそが最上者である」

→ 「たとえば盗賊たちにとって盗賊の首領が主宰者であり最上者であるように、
　それら〔心所〕にとっても、意が主宰者であり、意こそが最上者である。」

20. Yathā pana dārv'ādīhi nipphannāni tāni tāni bhaṇḍāni dārumayādīni nāma honti, tathā te pi manato nipphannattā manomayā nāma.

* yathā pana = 「また、たとえば」
* dāru-ādīhi nipphannāni = 「木などから作られた」
* tāni tāni bhaṇḍāni = 「さまざまの品物が」
* dāru-maya-ādīni nāma honti = 「『木製の』などと呼ばれる」
* tathā = 「そのように」
* te pi = 「それら〔心所〕も」
* manato nipphannattā = 「意から生じたものであるから」
* manomayā nāma = 「『意より成る（manomayā）』と呼ばれる」

→ 「また、たとえば木などから作られたさまざまの品物が『木製の』などと呼ばれるように、
　それら〔心所〕も、意から生じたものであるから、
　『意より成る（manomayā）』と呼ばれるのである。」

21. Paduṭṭhenā ti āgantukehi abhijjhādīhi dosehi paduṭṭhena.

* paduṭṭhena = 「『汚れた（paduṭṭhena）』とは」
* āgantukehi = 「外来の」
* abhijjhā-ādīhi = 「貪欲（abhijjhā）などの」
* dosehi = 「垢（汚れ）によって」
* paduṭṭhena = 「汚されたもの〔という意味〕である」

→ 「『汚れた（paduṭṭhena）』とは、外来の貪欲などの垢によって汚されたもの、という意味である。」

22. Pakati-mano hi bhavaṅga-cittaṃ, taṃ apaduṭṭhaṃ.

* pakati-mano = 「自然のままの心（本来の心）とは」
* hi = 「すなわち」
* bhavaṅga-cittaṃ = 「有分心（bhavaṅga-citta）のことである」
* taṃ apaduṭṭhaṃ = 「それは汚れていない」

→ 「本来の心とは有分心のことであり、それは汚れていない。」

23. Yathā hi pasannaṃ udakaṃ āgantukehi nīlādīhi upakkiliṭṭhaṃ nīlodakādibhedaṃ hoti, na ca navaṃ udakaṃ, nāpi purimaṃ pasannaudakam eva,

* yathā hi = 「たとえば」
* pasannaṃ udakaṃ = 「清らかな水が」
* āgantukehi nīla-ādīhi = 「外来の青色などによって」
* upakkiliṭṭhaṃ = 「汚染されると」
* nīla-udaka-ādi-bhedaṃ hoti = 「青い水などの区別となる」
* na ca navaṃ udakaṃ = 「しかし新しい水ではなく」
* nāpi purimaṃ pasanna-udakam eva = 「かつての清らかな水そのものでもない」

→ 「たとえば清らかな水が外来の青色などによって汚染され、青い水などとなるが、
　それは新しい水ではなく、またかつての清浄な水そのものでもない。」

24. tathā tam pi āgantukehi abhijjhādīhi dosehi paduṭṭhaṃ hoti, na ca navaṃ cittaṃ, nāpi purimaṃ bhavaṅgacittam eva,

* tathā = 「そのように」
* tam pi = 「その〔心〕も」
* āgantukehi abhijjhā-ādīhi dosehi = 「外来の貪欲などの垢によって」
* paduṭṭhaṃ hoti = 「汚されるのであるが」
* na ca navaṃ cittaṃ = 「しかし新しい心ではなく」
* nāpi purimaṃ bhavaṅga-cittam eva = 「またかつての有分心そのものでもない」

→ 「そのようにその心もまた、外来の貪欲などの垢によって汚されるが、
　それは新しい心ではなく、またかつての有分心そのものでもない。」

25. ten'āha bhagavā – "pabhassaram idaṃ, bhikkhave, cittaṃ, tañ ca kho āgantukehi upakkilesehi upakkiliṭṭhan" ti.

* ten'āha bhagavā = 「それゆえ世尊はこう仰せられた——」
* pabhassaram idaṃ cittaṃ = 「この心は光り輝くものである」
  * pabhassara = 「光明ある、輝く」
* bhikkhave = 「比丘たちよ」
* tañ ca kho = 「しかしそれは」
* āgantukehi upakkilesehi = 「外来の随煩悩（汚れ）によって」
* upakkiliṭṭhaṃ = 「汚染されている」

→ 「それゆえ世尊はこう仰せられた——
　『比丘たちよ、この心は光り輝くものである。しかしそれは外来の随煩悩によって汚染されている』と。」

26. Evaṃ manasā ce paduṭṭhena, bhāsati vā karoti vā so bhāsamāno catubbidhaṃ vacīduccaritam eva bhāsati, karonto tividhaṃ kāyaduccaritam eva karoti,

* evaṃ = 「このように」
* manasā ce paduṭṭhena = 「もし汚れた意をもって」
* bhāsati vā karoti vā = 「語り、あるいは行うならば」
* so bhāsamāno = 「その者は語る時に」
* catubbidhaṃ vacī-duccaritam eva bhāsati = 「四種の口悪行のみを語る」
  * 四種の口悪行：妄語・離間語・粗悪語・綺語
* karonto = 「行う時には」
* tividhaṃ kāya-duccaritam eva karoti = 「三種の身悪行のみを行う」
  * 三種の身悪行：殺生・偸盗・邪淫

→ 「このように汚れた意をもって語り、あるいは行うならば、
　その者は語る時には四種の口悪行のみを語り、
　行う時には三種の身悪行のみを行うのである。」

27. abhāsanto akaronto tāya abhijjhādīhi paduṭṭhamānasatāya tividhaṃ manoduccaritaṃ pūreti.

* abhāsanto = 「語らず」
* akaronto = 「行わずとも」
* tāya abhijjhā-ādīhi = 「かの貪欲などによる」
* paduṭṭha-mānasatāya = 「汚れた意の状態によって」
* tividhaṃ mano-duccaritaṃ pūreti = 「三種の意悪行を満たす」
  * 三種の意悪行：貪欲・瞋恚・邪見

→ 「語らず行わずとも、かの貪欲などによる汚れた意の状態によって、
　三種の意悪行を満たすのである。」

28. Evam assa dasa akusalakammapathā pāripūriṃ gacchanti.

* evam = 「このようにして」
* assa = 「その者の」
* dasa akusala-kamma-pathā = 「十不善業道が」
* pāripūriṃ gacchanti = 「円満に成就する」

→ 「このようにして、その者の十不善業道が円満に成就するのである。」

29. Tato naṃ dukkham anvetī ti tato tividhaduccaritato taṃ puggalaṃ dukkhaṃ anveti,

* tato naṃ dukkham anveti = 「『それゆえ苦がその者に従う』とは」
* tato = 「それ（三種の悪行）から」
* tividha-duccaritato = 「三種の悪行から」
* taṃ puggalaṃ = 「その人を」
* dukkhaṃ anveti = 「苦が追随する」

→ 「『それゆえ苦がその者に従う』とは、三種の悪行から、その人に苦が追随する、ということである。」

30. duccaritānubhāvena catūsu apāyesu, manussesu vā tam attabhāvaṃ gacchantaṃ kāyavatthukam pi itaram pī ti iminā pariyāyena kāyikacetasikaṃ vipākadukkhaṃ anugacchati.

* duccarita-ānubhāvena = 「悪行の威力によって」
* catūsu apāyesu = 「四悪趣（地獄・餓鬼・畜生・阿修羅）において」
* manussesu vā = 「あるいは人間界において」
* taṃ attabhāvaṃ gacchantaṃ = 「その自体（個体的存在）を得る者に」
* kāya-vatthukam pi = 「身体に基づくものも」
* itaram pi = 「その他のものも」
* iminā pariyāyena = 「この理由によって」
* kāyika-cetasikaṃ = 「身体的および精神的な」

→ 「悪行の威力によって、四悪趣において、あるいは人間界においてその自体を得る者に、
　身体に基づく苦もその他の苦も——
　この理由によって身体的および精神的な異熟の苦が追随するのである。」

31. Yathā kiṃ? Cakkaṃ va vahato padan ti dhure yuttassa dhuraṃ vahato balibaddassa padaṃ cakkaṃ viya.

* yathā kiṃ = 「それはどのようにか？」
* cakkaṃ va vahato padaṃ = 「『荷を引く者の足に車輪が〔従う〕ように』とは」
* dhure yuttassa = 「轅（ながえ）に繋がれた」
* dhuraṃ vahato = 「荷を引く」
* balibaddassa = 「牡牛の」
* padaṃ cakkaṃ viya = 「足跡に車輪が〔従う〕ように」

→ 「それはどのようにか？ 『荷を引く者の足に車輪が従うように』とは、
　轅に繋がれて荷を引く牡牛の足跡に車輪が従うように、ということである。」

32. Yathā hi so ekam pi divasaṃ dve pi pañca pi dasa pi aḍḍhamāsam pi māsam pi vahanto cakkaṃ nivattetuṃ jahituṃ na sakkoti,

* yathā hi = 「たとえば」
* so = 「その〔牡牛〕は」

* aḍḍhamāsam pi = 「半月でも」

* vahanto = 「引き続けて」

→ 「たとえば、その〔牡牛〕は一日でも二日でも五日でも十日でも半月でも一月でも
　引き続けて、車輪を退けることも捨て去ることもできない。」

33. atha khv'assa purato abhikkamantassa yugaṃ gīvaṃ bādhati, pacchato paṭikkamantassa cakkaṃ ūrumaṃsaṃ paṭihanati.

* atha kho = 「そして」
* assa = 「その〔牡牛〕が」
* purato abhikkamantassa = 「前方に進む時には」

* pacchato paṭikkamantassa = 「後方に退く時には」
* cakkaṃ ūru-maṃsaṃ paṭihanati = 「車輪が腿の肉を打つ」

→ 「そして、前方に進む時には軛が首を圧迫し、
　後方に退く時には車輪が腿の肉を打つ。」

34. Imehi dvīhi ākārehi bādhantaṃ cakkaṃ tassa padānupadikaṃ hoti;

* imehi dvīhi ākārehi = 「これら二つの仕方で」
* bādhantaṃ = 「苦しめる」
* cakkaṃ = 「車輪は」
* tassa = 「その〔牡牛〕の」
* padānupadikaṃ hoti = 「足跡に従うものとなる」
  * pada + anupadika = 「足跡に随って行くもの」

→ 「これら二つの仕方で苦しめる車輪は、その〔牡牛の〕足跡に従い続けるのである。」

* tath'eva = 「まさにそのように」
* manasā paduṭṭhena = 「汚れた意によって」
* tīṇi duccaritāni pūretvā = 「三種の悪行を満たして」
* ṭhitaṃ puggalaṃ = 「存する人を」
* nirayādīsu = 「地獄などにおいて」
* tattha tattha gatagata-ṭṭhāne = 「あちこちの赴いた赴いた場所において」
* duccarita-mūlakaṃ = 「悪行を根本とする」
* kāyikam pi cetasikam pi = 「身体的にも精神的にも」
* dukkhaṃ anubandhatī ti = 「苦が追随し続ける、ということである」

→ 「まさにそのように、汚れた意によって三種の悪行を満たした人を、
　地獄などのあちこちの赴いた先において、
　悪行を根本とする身体的・精神的な苦が追随し続けるのである。」

### 意訳

**【偈】**
諸法は意（こころ）を先導者とし、意を最上とし、意より成る。
もし汚れた意をもって語り、あるいは行うならば、
苦はその者に従う——荷を引く牛の足跡に車輪が従うように。

**【註釈】**
ここで「意」とは、欲界善心をはじめとするすべての四地の心のことである。
ただしこの偈においては、かの医者にその時に生じた、
憂を伴い瞋恚と相応した心のみが該当する。

「先導者とする」とは、心を最初に行くものとして具えている、ということである。

「諸法」には功徳・教説・学文・無有情無命の四種の意味があるが、
ここでは無有情・無命の法、すなわち受蘊・想蘊・行蘊の三つの非色蘊が意図されている。
意がそれらの先導者であるから「意を先導者とする」と呼ばれる。

ところで、これら心所とともに同一の所依・同一の所縁をもち、
前後なく同一刹那に生じる意が、どうして先導者となるのか。
それは生起の縁という意味においてである。
たとえば多くの者が共に村の略奪を行う時、
彼らを駆り立てた者が首謀者と呼ばれるように、
心所は意なしには生じえないが、意はある心所がなくとも生じうる。
このゆえに意が先導者なのである。

盗賊の首領が仲間の主宰者であり最上者であるように、
意も心所にとっての主宰者であり最上者であるから「意を最上とする」と言われる。
木から作られた品物が「木製の」と呼ばれるように、
心所は意から生じたものであるから「意より成る」と言われる。

「汚れた」とは、外来の貪欲などの垢によって汚されたもののことである。
本来の心とは有分心のことであり、それは汚れていない。
清らかな水が外から混入した染料によって色づくように、
心もまた外来の煩悩によって汚されるのであるが、
それは新しい心でもなく、もとの有分心そのものでもない。
それゆえ世尊は「この心は光り輝くものである。
しかしそれは外来の随煩悩によって汚染されている」と仰せられたのである。

このように汚れた意をもって語れば四種の口悪行を語り、
行えば三種の身悪行を行い、
語らず行わずとも汚れた意の状態によって三種の意悪行を満たす。
こうして十不善業道が円満に成就するのである。

三種の悪行から苦がその人に従い、
悪行の威力によって四悪趣において、あるいは人間界において、
身体的・精神的な異熟の苦が追随する。
それはあたかも轅に繋がれた牡牛の足跡に車輪が従うようなものである。
その牡牛は一日でも半月でも一月でも引き続けて車輪を逃れることができず、
前に進めば軛が首を圧迫し、後に退けば車輪が腿を打つ。
まさにそのように、汚れた意によって三種の悪行を満たした者には、
地獄などいかなる赴いた先においても、
悪行を根本とする身体的・精神的な苦が追随し続けるのである。

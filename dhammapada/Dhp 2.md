→「『柔らかくなれ』『荒くなれ』と言ったり、殴ったりして影だけを引き返らせることはできない。」

## 意訳

「心が先に立ち、あらゆる現象は心に従って起こる。心が最上であり、心によって形づくられている。  
もし清浄に澄みきった心をもって語り、また行為をなすならば、その人には、幸福がそのあとから、影のように離れることなくついて来る。

ここで『心』と言うとき、本来は欲界・色界・無色界・出世間界に属するすべての心を指すが、この句では意味を限定して、八種の欲界の善心、さらに言えば喜を伴い智慧と相応した心を指す。『法』とは受・想・行の三蘊であり、それらにとって、心は生起の条件として必ず先に生じるから、『心を先頭とする』『心が最上である』『心によって作られた』と言う。

人々が大勢で布施や供養などの功徳をしているとき、そのきっかけをつくった人が『先導者』と呼ばれるように、三蘊にとっての先導者・支配者が心である。金から作られた器が『金製』と呼ばれるように、心から生じたこれらも『心によって成ったもの』と呼ばれる。

『清浄な心』とは、無貪などの徳によって澄みきった心をいう。そのような心で語るときは、四種の善い言葉だけを語り、行為するときは三種の善い身の行為だけを行う。たとえ語らず行わなくても、その澄んだ心であることによって三種の善い心の行為を満たしている。このようにして、その人には十種の善業道が完全に具わる。

その善行為のゆえに、その人には楽（幸福）が伴う。ここでは三界にわたるあらゆる善が意図されており、その善行為の力によって善趣に生まれた者にはもちろん、たとえ悪趣にあっても、楽を受けるべき状態にあるときには、身体をもつ者にも、他の基盤をもつ者にも、あるいは無色界のように基盤なき者にも、身と心の果報としての楽が付き従い、離れることがない。

ちょうど影というものが身体に結びつき、身体が歩けば影も歩き、立てば立ち、座れば座り、言葉で命じたり打ったりして影だけを引き返らせることなどできないのと同じである。身体に結びついているからである。同様に、十種の善業道に根ざし、繰り返し培われた善から生じる、欲界などさまざまな世界における身と心の楽は、どこに生まれようとも、離れることのない影のように付き従って、その有情を決して離れないのである。」

### 註釈（Commentary）

4. Tattha kiñcāpi manoti avisesena sabbampi catubhūmikacittaṃ vuccati,

*   tattha = 「そこにおいて（その偈文において）」

*   avisesena = 「無差別に、区別なく」

*   vuccati = 「言われる、呼ばれる」

→ 「そこで、いかに『意』といっても、（一般的には）区別なく四つの地界（欲界・色界・無色界・出世間）に属するすべての心が言及されるのではあるが、」

5. imasmiṃ pana pade niyamiyamānaṃ vavatthāpiyamānaṃ paricchijjiyamānaṃ aṭṭhavidhaṃ kāmāvacarakusalacittaṃ labbhati.

*   imasmiṃ pana pade = 「しかしながら、この句（詩節）においては」
*   niyamiyamānaṃ = 「限定されつつ」
*   vavatthāpiyamānaṃ = 「定義されつつ（特定され）」
*   paricchijjiyamānaṃ = 「区分（判断）されつつ」
*   aṭṭhavidhaṃ = 「八種の」
*   kāmāvacara-kusala-cittaṃ = 「欲界の善心」が
*   labbhati = 「得られる（適用される）」

→ 「しかしこの詩句においては、限定され、特定され、区分されて、『八種の欲界善心』が得られる（を意味する）。」

6. Vatthuvasena panāhariyamānaṃ tatopi somanassasahagataṃ ñāṇasampayuttacittameva labbhati.

*   āhariyamānaṃ = 「取り上げられるならば」

*   labbhati = 「得られる」

→ 「さらに（マッタカクンダリーの）因縁物語に基づいて取り上げるならば、それらの中でも『喜びを伴う、智恵と結びついた心（喜倶行智相応心）』のみが得られる。」

→ 「あるいは悪趣にあっても楽を享受できる境遇にいる者に対して、身体的な楽も、精神的な楽も、あるいは（物質的身体を）依り所としない楽も、すなわち身体的・精神的な果報としての幸福が付き従い、決して離れないという意味であると理解すべきである。」

*   yathā kiṃ = 「何のごとくか？」

→ 「何のごとくか。『離れない影のごとく』とは、たとえば影というものは身体に結びついており、」

→ 「『戻れ』と優しく言ったり厳しく言ったりして、あるいは叩いたりして、影を引き返させることはできない。なぜか。身体に結びついているからである。」

*   kāmāvacara-ādi-bhedaṃ = 「欲界などの区分による」

*   gata-gata-ṭṭhāne = 「至るところで」
*   anapāyinī chāyā viya hutvā = 「離れない影のようになって」
*   na vijahatī ti = 「（その人を）捨て去らないのである」

→ 「まさにそのように、これら十善業道を習慣的に実践した善を根源とし、欲界などに分類される身体的および精神的な幸福は、どこへ行こうとも、離れない影のようになって、決してその人を捨て去らないのである。」

## 意訳

「私たちのあらゆる精神的な現象（心所）は、心が先導し、心が主であり、心によって造り出されます。もしも、清らかな心で言葉を語り、あるいは行いを行うならば、それによって幸福（楽）はその人に付き従います。あたかも、決して身体から離れることのない影のように。

ここで『心（意）』というのは、本来はあらゆる領域の心を指しますが、この詩句においては特に『欲界の善き心』、因縁物語に即して言えば『喜びを伴い、智慧と結びついた心』を意味しています。
心が『先導する』というのは、心が物事の生起の原因であり、常に先頭に立つからです。集団で寄付や供養を行う際に主導者がいるように、心の働きにおいては心がリーダーなのです。また、黄金製品が黄金でできているように、精神的な現象は心から生じたものなので『心によって造られたもの』と言われます。

『清らかな心』とは、貪りなどの汚れがない心のことです。このような心で語れば、嘘や悪口のない正しい言葉となり、行えば殺生や盗みのない正しい行いとなります。また、何もしていない時でも心の中で正しい思いを持ち続けます。こうして十種の善き行い（十善業道）が完成されます。

『幸福が付き従う』とは、影が身体の動きに合わせて移動し、誰もそれを追い払うことができないのと同じです。善い行いによって生み出された身体的・精神的な幸福は、その人がどこへ行こうとも、決してその人を見捨てることなく、影のように寄り添い続けるのです。」

    * na vijahati = 「離れない」
    * ti attho veditabbo = 「という意味であると理解すべきである」

    → 「身を所依とするもの・他の所依をもつもの・所依なきもの、すなわち身体的・精神的な異熟の楽が追随して離れない、という意味であると理解すべきである。」

26. Yathā kiṃ? Chāyāva anapāyinīti

    * yathā kiṃ = 「どのようにか？」
    * chāyā va anapāyinī ti = 「『離れることのない影のように』とは」

    → 「どのようにか？ 『離れることのない影のように』とは――」

27. yathā hi chāyā nāma sarīrappaṭibaddhā sarīre gacchante gacchati, tiṭṭhante tiṭṭhati, nisīdante nisīdati,

    * yathā hi = 「たとえば」
    * chāyā nāma = 「影というものは」
    * sarīra-ppaṭibaddhā = 「身体に結びついたものであって」
    * sarīre gacchante gacchati = 「身体が行けば行き」
    * tiṭṭhante tiṭṭhati = 「立てば立ち」
    * nisīdante nisīdati = 「座れば座る」

    → 「たとえば、影というものは身体に結びついたものであり、身体が行けば行き、立てば立ち、座れば座る。」

28. na sakkoti, ''saṇhena vā pharusena vā nivattāhī''ti vatvā vā pothetvā vā nivattāpetuṃ.

    * na sakkoti = 「～することはできない」
    * saṇhena vā = 「柔和な（言葉）によっても」
    * pharusena vā = 「粗暴な（言葉）によっても」
    * nivattāhī ti vatvā vā = 「『戻れ』と言っても」
    * pothetvā vā = 「叩いても」
    * nivattāpetuṃ = 「戻らせること（はできない）」

    → 「柔和な言葉で『戻れ』と言っても、叩いても、引き返させることはできない。」

29. Kasmā? Sarīrappaṭibaddhattā.

    * kasmā = 「なぜか？」
    * sarīra-ppaṭibaddhattā = 「身体に結びついているがゆえに」

    → 「なぜか？ 身体に結びついているがゆえに。」

30. Evameva imesaṃ dasannaṃ kusalakammapathānaṃ āciṇṇasamāciṇṇakusalamūlikaṃ kāmāvacarādibhedaṃ kāyikacetasikasukhaṃ gatagataṭṭhāne anapāyinī chāyā viya hutvā na vijahatīti.

    * evam eva = 「まさにそのように」
    * imesaṃ dasannaṃ kusala-kamma-pathānaṃ = 「これら十善業道の」
    * āciṇṇa-samāciṇṇa-kusala-mūlikaṃ = 「繰り返し積み重ねられた善根に基づく」
    * kāmāvacara-ādi-bhedaṃ = 「欲界などの種類に分かれた」
    * kāyika-cetasika-sukhaṃ = 「身体的・精神的な楽が」
    * gata-gata-ṭṭhāne = 「行く先々の場所において」
    * anapāyinī chāyā viya hutvā = 「離れることのない影のようになって」
    * na vijahati = 「離れない」
    * ti = 「ということである」

    → 「まさにそのように、これら十善業道の、繰り返し積み重ねられた善根に基づく、欲界などの種類に分かれた身体的・精神的な楽が、行く先々において離れることのない影のようになって離れないのである。」

## 意訳

【偈文】
「諸々の法（心所法）は意が先導し、意が最も勝れ、意から生じる。
もし澄んだ心をもって語り、あるいは行うならば、
その人に楽が従い行く――離れることのない影のように。」

【註釈】
ここで「意（mano）」とは、広い意味では四地すべての心を指すが、この偈の文脈では八種の欲界善心に限定され、さらに文脈上の根拠から言えば喜倶・智相応の心のみが得られる。

「先導する」とは、生起の縁という意味で、喜相応の意がこれら〔受・想・行の三蘊〕の先導者であることを言う。「諸法」とは受などの三蘊である。ちょうど多くの人が集まって大比丘僧伽への布施や供養などの福業を行うとき、「誰が先導者か」と問われれば、彼らの縁となりそれに依って彼らがその福業を行うティッサやプッサこそが先導者と呼ばれるように、意もまた生起の縁として諸法の先導者なのである。心所法は意が生じなければ生じることができないが、意はある種の心所が生じなくとも生じうるからである。

「意が最勝」とは、主宰者という意味である。集団の長や組合の長と呼ばれるように、諸法にとって意こそが最勝者である。「意より成る」とは、金で造った品物が「金製の～」と呼ばれるように、諸法も意から生じたものであるゆえにそう呼ばれる。

「澄んだ（pasannena）」とは、無貪欲などの徳によって澄浄となった心による、ということである。このような心をもって語る者は四種の語善行のみを語り、行う者は三種の身善行のみを行い、語らず行わずとも無貪欲などによる澄浄なる心によって三種の意善行を成就する。こうして十善業道が円満に至る。

「それからその人に楽が従う」とは、その三種の善行から、その人に楽が従うということである。ここでは三地（欲界・色界・無色界）の善が意図されており、その善行の威力によって善趣に生まれた人、あるいは悪趣にあっても楽を受けうる境遇にある人に、身体的・精神的な異熟の楽が追随して離れない、という意味と理解すべきである。

どのようにか？――影というものは身体に結びつき、身体が行けば行き、立てば立ち、座れば座る。柔和に「戻れ」と言っても、叩いても、戻らせることはできない。なぜなら身体に結びついているからである。まさにそのように、十善業道の繰り返し積み重ねられた善根に基づく、欲界その他の種類に分かれた身体的・精神的な楽は、行く先々において離れることのない影のようになって、その者を離れないのである。

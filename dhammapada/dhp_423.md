423.

‘‘Pubbenivāsaṃ yo vedi, saggāpāyañca passati;

Atho jātikkhayaṃ patto, abhiññāvosito muni; (Saṃ. ni. 1.199);

Sabbavositavosānaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tassattho – yo pubbenivāsaṃ pākaṭaṃ katvā jānāti, chabbīsatidevalokabhedaṃ saggañca catubbidhaṃ apāyañca dibbacakkhunā passati, atho jātikkhayasaṅkhātaṃ arahattaṃ patto, abhiññeyyaṃ dhammaṃ abhijānitvā pariññeyyaṃ parijānitvā pahātabbaṃ pahāya sacchikātabbaṃ sacchikatvā vosiko niṭṭhānaṃ patto, vusitavosānaṃ vā patto, āsavakkhayapaññāya monabhāvaṃ pattattā muni, tamahaṃ sabbesaṃ kilesānaṃ vosānaṃ arahattamaggañāṇaṃ brahmacariyavāsaṃ vutthabhāvena sabbavositavosānaṃ brāhmaṇaṃ vadāmīti.

## 対訳

1. ‘‘Pubbenivāsaṃ yo vedi,

* pubbenivāsaṃ = 「過去の住処・前住」「前世の生存（＝過去世）」を
* yo = 「・・・である者は」
* vedi = 「知る」（*vedeti* の三人称・完了／現在形）

→ 「前世の生存を知る者は」

---

2. saggāpāyañca passati;

* sagga-apāyañca = 「天界と、そして悪趣とを」
  * sagga = 「天界」
  * apāya = 「悪趣、苦しみの境域（地獄・餓鬼・畜生・阿修羅など）」
  * -ñca = 「・・・と、そして・・・を」
* passati = 「見る」

→ 「天界と悪趣とを（も）見る者であり」

---

3. Atho jātikkhayaṃ patto,

* atho = 「さらに、そしてまた」
* jātikkhayaṃ = 「生の滅尽（生まれの終滅）」を
  * jāti = 「生まれ、生」
  * khaya = 「滅、尽きること」
* patto = 「到達した、得た」

→ 「さらに、生まれの滅尽に到達し」

---

4. abhiññāvosito muni; (Saṃ. ni. 1.199);

* abhiññā-vosito = 「（諸）神通・勝れた直接知によって完成した／成就しおわった」
  * abhiññā = 「勝れた直接知、神通智」
  * vosita (pp. of *vas* ‘住む・修める’ / *vusita*) = 「修行し終えた・成就した」
* muni = 「牟尼、沈黙の聖者・賢者」

→ 「勝れた直接知を成就し終えた聖者である者」

---

5. Sabbavositavosānaṃ,

* sabba-vosita-vosānaṃ = 「すべてを修し終え、すべてを終結させた者を」
  * sabba = 「すべて」
  * vusita/vosita = 「修行し終えた・過ごし終えた」
  * vosāna = 「終結・やむこと・滅尽」

→ 「一切を修し終え、なすべきことをすべて終えた者を」

---

6. tamahaṃ brūmi brāhmaṇa’’nti.

* taṃ = 「そのような人を」
* ahaṃ = 「私は」
* brūmi = 「・・・と呼ぶ／言う」
* brāhmaṇaṃ = 「バラモン（ここでは『清らかな聖者』の意）」
* nti = 「・・・と（世尊は）言った、という」

→ 「その人を、私は『バラモン（真の聖者）』と呼ぶのである、と。」

---

7. Tassattho –

* tassa = 「その（偈の）」
* attho = 「意味は」
* – = 「・・・である。」

→ 「その意味は、次のようである。」

---

8. yo pubbenivāsaṃ pākaṭaṃ katvā jānāti,

* yo = 「・・・である者は」
* pubbenivāsaṃ = 「前住、前世の生存（過去世）」を
* pākaṭaṃ katvā = 「明らかにして、顕わにして」
  * pākaṭa = 「明らかな、はっきりした」
  * katvā = 「・・・して（*karoti* の絶対分詞）」
* jānāti = 「知る」

→ 「前世の生存を、明らかにして知る者であり」

---

9. chabbīsatidevalokabhedaṃ saggañca catubbidhaṃ apāyañca dibbacakkhunā passati,

* chabbīsati-devaloka-bhedaṃ = 「二十六種の天界の区別を」
  * chabbīsati = 「二十六」
  * devaloka = 「天界」
  * bheda = 「区分、種類」
* saggañca = 「そして天界を（＝それらの天界全体を）」
* catubbidhaṃ apāyañca = 「四種の悪趣をも」
  * catubbidhaṃ = 「四種の」
  * apāyaṃ = 「悪趣」
* dibba-cakkhunā = 「天眼によって」
  * dibba = 「天の、神の」
  * cakkhu = 「眼」
* passati = 「見る」

→ 「二十六種の天界の区別と、天界全体、さらに四種の悪趣とを、天眼によって見る者であり」

---

10. atho jātikkhayasaṅkhātaṃ arahattaṃ patto,

* atho = 「そしてまた」
* jātikkhaya-saṅkhātaṃ = 「『生の滅尽』と呼ばれる」
  * jāti = 「生まれ、生」
  * khaya = 「滅、尽きること」
  * saṅkhāta = 「・・・と名づけられる、呼ばれる」
* arahattaṃ = 「阿羅漢果、阿羅漢の境地」を
* patto = 「得た、到達した」

→ 「さらに、『生の滅尽』と名づけられる阿羅漢の境地に到達し」

---

11. abhiññeyyaṃ dhammaṃ abhijānitvā pariññeyyaṃ parijānitvā pahātabbaṃ pahāya sacchikātabbaṃ sacchikatvā vosiko niṭṭhānaṃ patto, vusitavosānaṃ vā patto,

* abhiññeyyaṃ dhammaṃ = 「直接に知るべき法」を
  * abhiññeyya = 「直接に知るべき」
  * dhamma = 「法、対象」
* abhijānitvā = 「直接に知り了って」
* pariññeyyaṃ (dhammaṃ) = 「徹底的に知り尽くすべき（法）を」
  * pariññeyya = 「完全認知すべき」
* parijānitvā = 「知り尽くし了って」
* pahātabbaṃ (dhammaṃ) = 「捨て去るべき（法・煩悩など）を」
  * pahātabba = 「捨てるべき、放棄すべき」
* pahāya = 「捨て去って」
* sacchikātabbaṃ (dhammaṃ) = 「証得すべき（法）を」
  * sacchikātabba = 「自ら体証すべき」
* sacchikatvā = 「証得し終えて」
* vosiko = 「なすべきことをやり終えた者、もはやするべきことがない者」
* niṭṭhānaṃ patto = 「究竟の終局・完成に到達した」
  * niṭṭhāna = 「終局、完成、目的地」
  * patto = 「到達した」
* vusita-vosānaṃ vā patto = 「あるいは、（聖なる）生活を修し終え、終結に達した者となった」
  * vusita = 「修し終えた、住し終えた」
  * vosāna = 「終わり、やみ、完了」
  * vā = 「あるいは」
  * patto = 「到達した」

→ 「直接に知るべき法を直接に知り、知り尽くすべき法を知り尽くし、捨て去るべきものを捨て去り、証得すべきものを証得して、なすべきことをすべてやり終えた者、究竟の完成に達した者、あるいは聖なる生活を修し終えて終結に達した者であり」

---

12. āsavakkhayapaññāya monabhāvaṃ pattattā muni,

* āsavakkhaya-paññāya = 「漏尽の智慧によって」
  * āsava = 「漏（煩悩の根本的流出）」
  * khaya = 「滅、尽きること」
  * paññā = 「智慧」
  * -ya (inst.) = 「・・・によって」
* mona-bhāvaṃ = 「聖なる沈黙／牟尼の境地」を
  * mona = 「牟尼であること、沈黙の聖者であるありさま」
  * bhāva = 「状態、ありさま」
* patta-ttā = 「到達したがゆえに」
  * patta = 「到達した」
  * -ttā = 「・・・であるがゆえに」
* muni = 「牟尼、聖者である」

→ 「漏尽の智慧によって聖なる沈黙の境地に到達しているがゆえに、牟尼（聖者）であり」

---

13. tamahaṃ sabbesaṃ kilesānaṃ vosānaṃ arahattamaggañāṇaṃ brahmacariyavāsaṃ vutthabhāvena sabbavositavosānaṃ brāhmaṇaṃ vadāmīti.

* taṃ = 「そのような人を」
* ahaṃ = 「私は」
* sabbesaṃ kilesānaṃ vosānaṃ = 「一切の煩悩の滅尽（という点）において」
  * sabbesaṃ = 「一切の」
  * kilesānaṃ = 「煩悩の」
  * vosānaṃ = 「滅尽、終息」
* arahattamagga-ñāṇaṃ = 「阿羅漢道の智（を得ているという点）において」
  * arahattamagga = 「阿羅漢の道」
  * ñāṇa = 「智」
* brahmacariya-vāsaṃ vuttha-bhāvena = 「梵行の生活を修し終えたというありさまによって」
  * brahmacariya = 「梵行、清浄行、聖なる生活」
  * vāsa = 「住すること、生活」
  * vuttha (pp. of *viharati*) = 「住し終えた、過ごし終えた」
  * -bhāva = 「状態、性質」
* sabba-vosita-vosānaṃ = 「一切を修し終え、終結し終えた者として」
  * sabba = 「一切」
  * vosita/vusita = 「修し終えた・行じ終えた」
  * vosānaṃ = 「終結、やみ、完了」
* brāhmaṇaṃ = 「（真の）バラモンを」
* vadāmi = 「私は・・・と呼ぶ、言う」
* iti = 「・・・と（いうのである）」

→ 「その人を、私は、一切の煩悩の滅尽、阿羅漢道の智、梵行の生活を修し終えたというありさまによって、一切を修し終え、なすべきことをすべて終えたバラモンであると言うのである。」

---

## 意訳

前世の生存を見通して知り、  
二十六の天界と四つの悪趣を、天眼によってありのままに見ることができ、  
さらに「生まれの滅尽」と呼ばれる阿羅漢の境地に到達し、  

本来、直接に知られるべきものを知り尽くし、  
明らかにし尽くすべきものを明らかにし、  
捨て去るべきものをすべて捨て去り、  
証得すべき真理を自ら証し終えて、  
もはやなすべきことの残っていない完成の境地に達し、  
梵行の生活を修しきって終わりに至った者──  

漏（煩悩）がすべて尽きる智慧によって、  
沈黙の聖者の境地に到りきっているがゆえに聖者と呼ばれるその人を、  
私は、一切の煩悩を滅し、阿羅漢道の智を具え、  
聖なる梵行の生を修め終え、  
なすべきことをすべて成就し終えた者として、  
「真のバラモン」と呼ぶのである。

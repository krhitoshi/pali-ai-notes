399.

‘‘Akkosaṃ vadhabandhañca, aduṭṭho yo titikkhati;

Khantībalaṃ balānīkaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha aduṭṭhoti etaṃ dasahi akkosavatthūhi akkosañca pāṇiādīhi pothanañca andubandhanādīhi bandhanañca yo akuddhamānaso hutvā adhivāseti , khantibalena samannāgatattā khantibalaṃ, punappunaṃ uppattiyā anīkabhūtena teneva khantibalena samannāgatattā balānīkaṃ taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Akkosaṃ vadhabandhañca, aduṭṭho yo titikkhati;

* akkosaṃ = 「罵り、ののしり、暴言を」
* vadha-bandhaṃ ca = 「打擲（vadha：打つ・傷つけること／殺害）と、拘束・縛り付けること（bandhana）とを、また」
  * vadha = 「打つこと、傷つけること、殺すこと」
  * bandhana = 「縛ること、つなぐこと、拘禁・監禁」
  * -ñca (＝ca) = 「そして」
* aduṭṭho = 「怒っていない者、悪意を抱かない者（a- + duṭṭha「汚れた・怒った」の否定）」
* yo = 「…である者は」
* titikkhati = 「耐え忍ぶ、堪え忍んで受け止める」

→「罵りや、打ち叩き・縛り付けることをも、怒ることなく耐え忍ぶ者は」

---

2. Khantībalaṃ balānīkaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* khantī-balaṃ = 「忍耐という力を具えた者」
  * khantī = 「忍耐、堪忍、堪え忍ぶこと」
  * balaṃ = 「力」
* balānīkaṃ = 「（その）力の軍勢（のごとき者）、力の軍陣を備えた者」
  * bala = 「力」
  * anīka = 「軍勢、戦列、軍陣」
  * balānīkaṃ = 「力そのものが軍勢のようである（ほど強い）者」
* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ、言う」
* brāhmaṇaṃ = 「バラモンと」
* ’nti (ti) = 「～と（ブッダの説示の言い終わりを示す引用句）」

→「忍耐の力を備え、力の軍勢のような（強大な）者――その人を、私は『バラモン』と呼ぶのである。」

---

3. Tattha aduṭṭhoti

* tattha = 「ここで、この偈においては」
* aduṭṭho ti = 「『aduṭṭha（怒らぬ者）』とは」

→「ここで、『aduṭṭho（怒らぬ者）』とは、」

---

4. etaṃ dasahi akkosavatthūhi akkosañca

* etaṃ = 「この（akkosaṃ vadhabandhañca という一連の害悪を）」
* dasahi akkosavatthūhi = 「十種の罵詈の対象によって（十種類の罵倒の仕方によって）」
  * dasahi = 「十の」
  * akkosavatthūhi = 「罵りの事柄（テーマ）によって」
* akkosañca = 「罵倒をも」

→「この十種の罵詈の対象による罵りをも」

---

5. pāṇiādīhi pothanañca

* pāṇi-ādīhi = 「手などによる」
  * pāṇi = 「手、こぶし」
  * -ādīhi = 「などによって」
* pothanañca = 「打ち叩くことをも」

→「手などで打ち叩くことをも」

---

6. andubandhanādīhi bandhanañca

* andu-bandhana-ādīhi = 「縄や拘束具などで縛り付けることなどによる」
  * andu-bandhana = 「縄などによる縛り付け・拘束」
  * -ādīhi = 「などによって」
* bandhanañca = 「拘束・縛ることをも」

→「縄などで縛り付けるような拘束をも」

---

7. yo akuddhamānaso hutvā adhivāseti,

* yo = 「…である者は」
* akuddha-mānaso = 「怒りを起こさない心の者」
  * a-kuddha = 「怒っていない」
  * mānasa = 「心」
* hutvā = 「…となって、…の状態で」
* adhivāseti = 「（それらを）堪え忍ぶ、受け容れる、黙って耐える」

→「怒りの心を起こすことなく、そのすべてを堪え忍ぶ者であり」

---

8. khantibalena samannāgatattā khantibalaṃ,

* khantibalena = 「忍耐の力によって」
  * khantī-bala = 「忍耐の力」
* samannāgatattā = 「備わっているがゆえに、具えているがゆえに」
* khantibalaṃ = 「『忍耐の力を具えた者』と（いうのであり）」

→「忍耐の力を具えているがゆえに、『忍耐の力を有する者（khantībala）』と呼ばれ」

---

9. punappunaṃ uppattiyā anīkabhūtena teneva khantibalena samannāgatattā balānīkaṃ

* punappunaṃ = 「何度も何度も、繰り返し」
* uppattiyā = 「生起することにより、起こってくることによって」
* anīka-bhūtena = 「軍勢のようなものになった」
  * anīka = 「軍勢・戦列」
  * bhūta = 「…となった」
* teneva khantibalena = 「まさにその同じ忍耐の力によって」
* samannāgatattā = 「具わっているがゆえに」
* balānīkaṃ = 「『力の軍勢（のごとき者）』と（言うのであり）」

→「その忍耐の力が、何度も何度も起こることによって、まさに軍勢のようなものとなった、その同じ忍耐の力を備えているがゆえに、『力の軍勢（balānīka）』と呼ばれ」

---

10. taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「そのような者を」
* evarūpaṃ = 「このような種類の」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモンと」
* vadāmi ti attho = 「呼ぶのである、という意味である」

→「このような者をこそ、私は『バラモン』と呼ぶのである、という意味である。」

---

## 意訳

罵られ、手などで打たれ、縄などで縛られても、心に怒りを起こすことなく、静かにそれらを堪え忍ぶ人がいる。  
この人は、忍耐という力を身にそなえ、その忍耐が何度も何度も起こって、あたかも強大な軍勢のような力となっている。  
そのように、忍耐の力の軍勢を備えた人を、私は「真のバラモン」と呼ぶ。

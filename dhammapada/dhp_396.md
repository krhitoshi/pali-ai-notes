396.

‘‘Na cāhaṃ brāhmaṇaṃ brūmi, yonijaṃ mattisambhavaṃ;

Bhovādi nāma so hoti, sace hoti sakiñcano;

Akiñcanaṃ anādānaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha yonijanti yoniyaṃ jātaṃ. Mattisambhavanti brāhmaṇiyā mātu santake udarasmiṃ sambhūtaṃ. Bhovādīti so pana āmantanādīsu ‘‘bho, bho’’ti vatvā vicaranto bhovādi nāma hoti, sace rāgādīhi kiñcanehi sakiñcano. Ahaṃ pana rāgādīhi akiñcanaṃ catūhi upādānehi anādānaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Na cāhaṃ brāhmaṇaṃ brūmi, yonijaṃ mattisambhavaṃ;

* na ca ahaṃ = 「そして（かつ）私は～しない」→「私は…とは呼ばない」
* brāhmaṇaṃ brūmi = 「バラモンと呼ぶ」  
  * brāhmaṇaṃ = 「バラモンを」
  * brūmi = 「私は言う・呼ぶ」
* yonijaṃ = 「胎（yoni）から生じた者、胎に生まれた者」
* mattisambhavaṃ = 「母（matī / mātā）から生じた者、母胎に由来する者」

→「私は、胎から生まれ、母胎に由来する者を、バラモンとは呼ばない。」

2. Bhovādi nāma so hoti, sace hoti sakiñcano;

* bhovādi = 「『bho, bho』と言う者、『ボー』と呼びかける者」
* nāma = 「名の、名づけて」
* so hoti = 「その者は～である」
* sace = 「もし…であるならば」
* hoti = 「～である」
* sakiñcano = 「何ものかを所有している者、煩悩という〈何か〉をもっている者」
  * sa- = 「共に・有する」
  * kiñcana = 「何ものか、所有物・汚れ・しがらみ」

→「もしその者が、何ものか（煩悩）を有しているならば、その者は名づけて『bhovādi（ボーなどと言う者）』である。」

3. Akiñcanaṃ anādānaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* akiñcanaṃ = 「何ものも有さない者、〈何か〉を全く持たない者」
  * a- = 「ない」
  * kiñcana = 「何ものか、執着の対象」
* anādānaṃ = 「取らない者、執着しない者」
  * an-ādāna = 「（upādānaを）取ることがない」
* tam = 「その者を」
* ahaṃ brūmi = 「私は～と呼ぶ／言う」
* brāhmaṇaṃ = 「バラモンと」

→「何ものも有さず、執着しないその者を、私はバラモンと呼ぶのである。」

4. Tattha yonijanti yoniyaṃ jātaṃ.

* tattha = 「そこ（この偈の中）において」
* yonijan ti = 「『yonijaṃ』とは」
* yoniyaṃ = 「胎（腹・母胎）の中で」
* jātaṃ = 「生じた者、生まれた者」

→「ここで『yonijaṃ』とは、胎の中に生まれた者（胎内で生じた者）のことである。」

5. Mattisambhavanti brāhmaṇiyā mātu santake udarasmiṃ sambhūtaṃ.

* mattisambhavan ti = 「『mattisambhavaṃ』とは」
* brāhmaṇiyā = 「バラモン女（の）」
* mātu santake = 「母に属する（母のもとにある）」
  * mātu = 「母（mātā）の」
  * santake = 「属している・所有のもとにある」
* udarasmiṃ = 「腹の中で」
* sambhūtaṃ = 「生じた者、生まれ出た者」

→「『mattisambhavaṃ』とは、バラモン女という母に属する腹の中で生じた者（母胎で生まれた者）のことである。」

6. Bhovādīti so pana āmantanādīsu ‘‘bho, bho’’ti vatvā vicaranto bhovādi nāma hoti, sace rāgādīhi kiñcanehi sakiñcano.

* bhovādī ti = 「『bhovādi』とは」
* so pana = 「さて、その者は」
* āmantanādīsu = 「呼びかけなどをするときに」
  * āmantana = 「呼びかけ」
  * -ādīsu = 「などにおいて」
* ‘‘bho, bho’’ti = 「『bho, bho（おい、おい）』と」
* vatvā = 「言って」
* vicaranto = 「歩き回っている・行動している（者であり）」
* bhovādi nāma hoti = 「『bhovādi』という名であるようになる」
* sace = 「もし」
* rāgādīhi = 「貪欲などによって」
  * rāga = 「愛欲・貪欲」
  * -ādī = 「など」
* kiñcanehi = 「汚れ・しがらみとなる〈何ものか〉によって」
* sakiñcano = 「何ものかを有している（者）」

→「『bhovādi』とは、呼びかけなどのときに『bho, bho』と言いながら歩き回るので『bhovādi』という名になる者であって、もし貪欲などの〈何かの汚れ〉を有している者のことである。」

7. Ahaṃ pana rāgādīhi akiñcanaṃ catūhi upādānehi anādānaṃ brāhmaṇaṃ vadāmīti attho.

* ahaṃ pana = 「しかし、私は」
* rāgādīhi = 「貪欲などによって」
* akiñcanaṃ = 「何ものも有さない者（とされるような者）」
* catūhi upādānehi = 「四つの取（upādāna）に関して」
  * catūhi = 「四つの（具格）」  
  * upādāna = 「執着・取」
* anādānaṃ = 「取らない者、執着しない者」
* brāhmaṇaṃ vadāmi = 「バラモンであると言う／呼ぶ」
* ti attho = 「という意味である」

→「しかし私は、貪欲などによっては何ものも有さず、四つの取に関しては取らない者を、バラモンであると言うのである、という意味である。」

---

## 意訳

「私は、胎から生まれ、母の腹から生じた者を、バラモンとは呼ばない。  
そのような者は、もしなお何らかの煩悩を有しているならば、人に『ボー、ボー』と呼びかけて歩き回るだけの『bhovādi（ボーなどと言う人）』にすぎない。

一切の〈何か〉を持たず、四つの執着対象をいっさい取らない者――その人こそを、私はバラモンと呼ぶのである。」

400.

‘‘Akkodhanaṃ vatavantaṃ, sīlavantaṃ anussadaṃ;

Dantaṃ antimasārīraṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha vatavantanti dhutavatena, samannāgataṃ catupārisuddhisīlena sīlavantaṃ, taṇhāussadābhāvena anussadaṃ, chaḷindriyadamanena dantaṃ, koṭiyaṃ ṭhitena attabhāvena antimasarīraṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Akkodhanaṃ vatavantaṃ, sīlavantaṃ anussadaṃ;

* akkodhanaṃ  
  * a- = 「～でない」  
  * kodhana = 「怒りっぽい、怒る人」  
  → 「怒りのない者、怒らない者」

* vatavantaṃ  
  * vata = 「誓戒・苦行・修道作法（特に頭陀行など）」  
  * -vanta = 「それを備えた者」  
  → 「誓戒（頭陀行など）を具えた者」

* sīlavantaṃ  
  * sīla = 「戒、身口の行いの律し方」  
  * -vanta = 「それを具えた者」  
  → 「戒を具えた者、戒行ある者」

* anussadaṃ  
  * an- = 「無い」  
  * ussada = 「腫れ・膿・はれ物、汚れの塊（比喩的に煩悩の膿）」  
  → 「膿（のような汚れ）のない者、汚れのない者」

→ 「怒りのない者で、誓戒を具え、戒行ある者で、膿（のような汚れ）のない者であり」

---

2. Dantaṃ antimasārīraṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* dantaṃ  
  * danta = 「調御された、よく制御された」  
  → 「よく制御された者（調御された者）」

* antimasārīraṃ  
  * antima = 「最後の」  
  * sarīra = 「身、身体」  
  → 「最後の身体を持つ者（＝この生を最後とする者）」

* tam ahaṃ brūmi brāhmaṇaṃ  
  * tam = 「そのような者を」  
  * ahaṃ = 「私は」  
  * brūmi = 「言う、呼ぶ」  
  * brāhmaṇaṃ = 「バラモン（真のバラモン・聖者）」  
  → 「その者を、私は『バラモン』と呼ぶ」

→ 「よく制御され、この世の最後の身体をもつ者、その者を私は『バラモン』と呼ぶのである。」

---

3. Tattha vatavantanti dhutavatena,

* tattha = 「ここで、この句について」  
* vatavantaṃ ti = 「『vatavantaṃ』とは」  
* dhutavatena = 「頭陀行の誓戒を（も）つことによって」  

→ 「ここで『誓戒を具えた者（vatavantaṃ）』とは、頭陀行の誓戒を具えた者ということである。」

---

4. samannāgataṃ catupārisuddhisīlena sīlavantaṃ,

* samannāgataṃ = 「そなわった、具えた」  
* catupārisuddhisīlena = 「四種において清浄な戒とともに（＝四波羅提木叉清浄を具えた）」  
  * catu = 「四つの」  
  * pārisuddhi-sīla = 「四種の完全に清浄な戒」  
* sīlavantaṃ = 「戒を具えた者」

→ 「四種において清浄な戒を具えた、すなわち戒行ある者であり」

---

5. taṇhāussadābhāvena anussadaṃ,

* taṇhā-ussadā-bhāvena  
  * taṇhā = 「渇愛」  
  * ussada = 「腫れ・はれ物・膿の塊」  
  * abhāva = 「無いこと、不在」  
  → 「渇愛という腫れ物（膿の塊）が無いことによって」  

* anussadaṃ  
  * an-ussada = 「腫れ（膿）がない者、膿のない者」

→ 「渇愛という腫れ物がないがゆえに『膿のない者（anussadaṃ）』であり」

---

6. chaḷindriyadamanena dantaṃ,

* chaḷ-indriya-damanena  
  * chaḷ = 「六つの」  
  * indriya = 「諸根、感覚器官（眼・耳・鼻・舌・身・意）」  
  * damana = 「調伏、制御」  
  → 「六つの諸根を調伏することによって」

* dantaṃ = 「調御された者、よく制御された者」

→ 「六根を調伏することによって『調御された者（dantaṃ）』であり」

---

7. koṭiyaṃ ṭhitena attabhāvena antimasarīraṃ

* koṭiyaṃ ṭhitena  
  * koṭi = 「限界、終点、おわり」  
  * ṭhita = 「立っている、存している」  
  → 「終極（解脱）に至っているがゆえに」

* attabhāvena  
  * attabhāva = 「この有身、この個体としての存在」  
  → 「（今ある）有身として」

* antima-sarīraṃ  
  * antima = 「最後の」  
  * sarīra = 「身体」  
  → 「最後の身体（を持つ者）」

→ 「終極に至った有身として、これは『最後の身体（antimasarīraṃ）』であり」

---

8. tamahaṃ brāhmaṇaṃ vadāmīti attho.

* tam ahaṃ brāhmaṇaṃ vadāmi ti attho  
  * tam = 「そのような者を」  
  * ahaṃ = 「私は」  
  * brāhmaṇaṃ = 「バラモン（聖者）」  
  * vadāmi = 「と言う、呼ぶ」  
  * ti attho = 「という意味である」

→ 「そのような者を、私は『バラモン』と呼ぶのである、という意味である。」

---

## 意訳

「怒りがなく、頭陀行などの誓戒を具え、  
四種において清浄な戒をそなえ、  
渇愛という膿のような汚れがなく、  
六つの感覚をよく調御しており、  
この身をもって最後の生を終えようとしている者──  
そのような者を、私はまことの『バラモン』と呼ぶ。」

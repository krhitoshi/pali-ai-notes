414.

‘‘Yomaṃ palipathaṃ duggaṃ, saṃsāraṃ mohamaccagā;

Tiṇṇo pāraṅgato jhāyī, anejo akathaṃkathī;

Anupādāya nibbuto, tamahaṃ brūmi brāhmaṇa’’nti.

Tassattho – yo bhikkhu imaṃ rāgapalipathañceva kilesaduggañca saṃsāravaṭṭañca catunnaṃ ariyasaccānaṃ appaṭivijjhanakamohañca atīto, cattāro oghe tiṇṇo hutvā pāraṃ anuppatto, duvidhena jhānena jhāyī, taṇhāya abhāvena anejo, kathaṃkathāya abhāvena akathaṃkathī, upādānānaṃ abhāvena anupādiyitvā kilesanibbānena nibbuto, tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Yomaṃ palipathaṃ duggaṃ, saṃsāraṃ moham accagā;

* yo = 「～である者は」
* maṃ = 「この（imaṃ の語中音変化）」
* imaṃ = 「この」
* palipathaṃ = 「泥道、ぬかるんだ道」
* duggaṃ = 「難所である、通りにくい、危険な」
  * rāgapalipatha（註）= 「泥道のような貪欲、貪欲というぬかるみ」
  * kilesadugga（註）= 「煩悩という難所・危険な道」
* saṃsāraṃ = 「輪廻（生死流転）を」
* mohaṃ = 「迷い、無明を」
* accagā = 「越えた、乗り越えた（atikkamati のアオリスト）」

→「このぬかるんだ難所の道、輪廻と迷いとを越えた者は」

---

2. Tiṇṇo pāraṅgato jhāyī, anejo akathaṃkathī;

* tiṇṇo = 「渡り終えた者、渡りきった者（tarati の過去分詞）」
* pāraṅgato = 「彼岸に至った者、向こう岸に到達した者」
  * pāraṃ = 「彼岸、向こう側」
  * gato = 「行った、至った」
* jhāyī = 「禅定にふける者、瞑想に専心する者」
* anejo = 「揺らぎなき者、動揺なき者、欲動のない者」
  * eja = 「震え・動き・欲動」
  * an‑eja = 「動きのない、動揺しない」
* akathaṃkathī = 「疑いなき者、二心を持たぬ者」
  * kathaṃkathā = 「あれこれ思い悩むこと、疑惑」
  * a‑kathaṃkathī = 「疑いを持たない者」

→「洪水を渡り終えて彼岸に至り、禅定にふけり、動揺なく、疑いなき者は」

---

3. Anupādāya nibbuto, tam ahaṃ brūmi brāhmaṇa’’nti.

* anupādāya = 「執着することなく、取着することなく」
  * upādāna = 「執着、取着」
  * an‑upādāya = 「執着せずに、執らずに（絶対分詞）」
* nibbuto = 「涅槃に達した者、冷却した者、鎮まった者」
  * nibbuta = 「消えた、鎮まった、涅槃に入った」
* taṃ = 「その人を、その者を」
* ahaṃ = 「私は」
* brūmi = 「～と呼ぶ、言う」
* brāhmaṇaṃ = 「バラモン（真の聖者）」
* ’nti = 「～と（言った）」を示す引用の終わり

→「執着することなく涅槃に達した、その者を、私は『バラモン』と呼ぶのである。」

---

4. Tassattho –

* tassa = 「その（偈の）」
* attho = 「意味は」

→「その意味は、次のようである。」

---

5. yo bhikkhu imaṃ rāgapalipathañceva kilesaduggañca saṃsāravaṭṭañca catunnaṃ ariyasaccānaṃ appaṭivijjhanakamohañca atīto,

* yo = 「いかなる～であれ、その者は」
* bhikkhu = 「比丘は、出家修行者は」
* imaṃ = 「この」
* rāga‑palipathaṃ‑ca‑eva = 「貪欲という泥道そのものを」
  * rāga = 「貪欲」
  * palipatha = 「泥道、ぬかるみの道」
  * ca eva = 「そしてまさに」
* kilesa‑duggaṃ‑ca = 「煩悩という難所を」
  * kilesa = 「煩悩」
  * dugga = 「難所、危険な地、通りにくい場所」
* saṃsāra‑vaṭṭaṃ‑ca = 「輪廻の輪、輪廻の回転を」
  * saṃsāra = 「輪廻、生死流転」
  * vaṭṭa = 「回転、輪、流転」
* catunnaṃ ariyasaccānaṃ appaṭivijjhana‑kamohaṃ‑ca = 「四つの聖なる真理を悟らないという、その悟れなさに根ざす無明を」
  * catunnaṃ = 「四つの（属格複数）」
  * ariya‑saccānaṃ = 「聖なる真理（四聖諦）の」
  * appaṭivijjhana = 「洞察されないこと、悟られないこと」
  * moha = 「迷い、無明」
  * appaṭivijjhana‑kamoha = 「（四聖諦が）悟られないことというかたちをとる無明」
* atīto = 「超えた者であり、乗り越えた者であり」

→「すなわち、どのような比丘であれ、この貪欲という泥道と、煩悩という難所と、輪廻の回転と、四つの聖なる真理を悟らないというその無明とを超え去った者であり」

---

6. cattāro oghe tiṇṇo hutvā pāraṃ anuppatto,

* cattāro = 「四つの」
* oghā = 「氾濫、洪水」
  * ogha = 「氾濫、押し流すもの（欲・有・見・無明の四氾濫）」
* tiṇṇo hutvā = 「渡り終えて、越え終えて」
  * tiṇṇa = 「渡りきった」
  * hutvā = 「～してから、～となって（絶対分詞）」
* pāraṃ = 「彼岸へ、向こう岸へ」
* anuppatto = 「到達した者、至った者（anupapajjati の過去分詞）」

→「四つの氾濫を渡り終えて彼岸に到達した者であり」

---

7. duvidhena jhānena jhāyī,

* duvidhena = 「二種のかたちによる、二通りの」
  * du‑ = 「二」
  * vidha = 「種類、かたち」
* jhānena = 「禅定によって」
* jhāyī = 「禅定にふける者、瞑想に打ち込む者」

→「二種の禅定によって禅定にふける者であり」

---

8. taṇhāya abhāvena anejo,

* taṇhāya = 「渇愛の」
* abhāvena = 「無さによって、存在しないがゆえに」
  * abhāva = 「不在、無いこと」
* anejo = 「動揺なき者、揺らぎなき者、欲動のない者」

→「渇愛が存在しないがゆえに動揺なき者であり」

---

9. kathaṃkathāya abhāvena akathaṃkathī,

* kathaṃkathāya = 「疑い・あれこれ思い悩む心の」
* abhāvena = 「無さによって、存在しないがゆえに」
* akathaṃkathī = 「疑いなき者、迷いを持たない者」

→「疑いが存在しないがゆえに疑いなき者であり」

---

10. upādānānaṃ abhāvena anupādiyitvā kilesanibbānena nibbuto,

* upādānānaṃ = 「執着・取着の（複数属格）」
* abhāvena = 「無さによって、存在しないがゆえに」
* anupādiyitvā = 「執着することなくして、取着することなくして」
  * an‑upādiyitvā = 「掴みとることなく、取りつくことなく（絶対分詞）」
* kilesa‑nibbānena = 「煩悩が涅槃したことによって、煩悩の消滅という涅槃によって」
  * kilesa = 「煩悩」
  * nibbāna = 「涅槃、吹き消し・消滅」
* nibbuto = 「涅槃に達した者、冷却しきった者、鎮まりきった者」

→「執着が存在しないがゆえに何ものにも執着することなく、煩悩の消滅という涅槃によって完全に涅槃に達した者であり」

---

11. tam ahaṃ brāhmaṇaṃ vadāmi ti attho.

* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（真の聖者）と」
* vadāmi = 「呼ぶ、言う」
* ti attho = 「という意味である」

→「そのような者を、私は『バラモン』と呼ぶのである、という意味である。」

---

## 意訳

「このぬかるんだ危険な道、輪廻という迷いの流れを越え去った者、
四つの氾濫を渡りきって彼岸に至り、二種の禅定にふけり、
渇愛がないがゆえに揺らぐことなく、疑いがないがゆえに迷うこともなく、
いかなるものにも執着することなく、煩悩の滅尽という涅槃によって
すっかり鎮まりきっている者──
その人を、わたしは真のバラモンと呼ぶ。」

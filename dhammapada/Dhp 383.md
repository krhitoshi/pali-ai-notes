383.

‘‘Chinda sotaṃ parakkamma, kāme panuda brāhmaṇa;

Saṅkhārānaṃ khayaṃ ñatvā, akataññūsi brāhmaṇā’’ti.

Tattha parakkammāti taṇhāsotaṃ nāma na appamattakena vāyāmena chindituṃ sakkā, tasmā ñāṇasampayuttena mahantena parakkamena parakkamitvā taṃ sotaṃ chinda. Ubhopi kāme panuda nīhara. Brāhmaṇāti khīṇāsavānaṃ ālapanametaṃ. Saṅkhārānanti pañcannaṃ khandhānaṃ khayaṃ jānitvā. Akataññūti evaṃ sante tvaṃ suvaṇṇādīsu kenaci akatassa nibbānassa jānanato akataññū nāma hosīti.

## 対訳

### 1. Chinda sotaṃ parakkamma, kāme panuda brāhmaṇa;

* chinda = 「断て」  
  * chindati の命令形・2人称単数。「切れ、断ち切れ」

* sotaṃ = 「流れを」  
  * sota（流れ）の単数対格。ここでは「渇愛の流れ」

* parakkamma = 「奮励して・力を尽くして」  
  * parakkamati（奮励する）の絶対分詞。「努力してから」

→「奮励して、（渇愛という）流れを断て。」

* kāme = 「欲（か）を」  
  * kāma の複数対格。感官的欲楽・欲対象

* panuda = 「払いのけよ・追い払え」  
  * panudati の命令形・2人称単数

* brāhmaṇa = 「バラモンよ、聖者よ」  
  * 呼格。ここでは阿羅漢への呼びかけの語

→「バラモンよ、欲を追い払いなさい。」

――全体として  
→「奮励して渇愛の流れを断て。バラモンよ、欲を払いのけよ。」


### 2. Saṅkhārānaṃ khayaṃ ñatvā, akataññūsi brāhmaṇā’’ti.

* saṅkhārānaṃ = 「諸行の」  
  * saṅkhāra の複数属格。「形成された諸法（行）」  
  * 註：ここでは五取蘊としての諸行を指す

* khayaṃ = 「滅尽を・消滅を」  
  * khaya の対格。「尽きること」

* ñatvā = 「知って・悟って」  
  * jānāti の絶対分詞。「知ったのち」

→「諸行の滅尽を悟って」

* akataññūsi = akataññū + asi  
  * akata = 「作られていないもの・不作」  
  * ññū（ñātā） = 「知る者」  
  * asi = 「～である（あなたは）」  
→「あなたは『不作（不生）のものを知る者』である」

* brāhmaṇā = 「バラモンよ、聖者よ」  
  * 呼格。語尾が -ā になった形

→「バラモンよ、あなたは不作にして作られざるものを知る者である。」


---

### 3. Tattha parakkammāti taṇhāsotaṃ nāma na appamattakena vāyāmena chindituṃ sakkā, tasmā ñāṇasampayuttena mahantena parakkamena parakkamitvā taṃ sotaṃ chinda.

* tattha = 「ここで」  
  * 当該偈の語句の説明を導く語

* parakkammā ti = 「『parakkamma（奮励して）』とは」  

* taṇhāsotaṃ nāma = 「いわゆる『渇愛の流れ』を」  
  * taṇhā = 「渇愛」  
  * sotaṃ = 「流れ」  
  * nāma = 「～と名づけられる、いわゆる」

* na appamattakena vāyāmena = 「わずかな努力によっては……ない」  
  * na = 「～ではない」  
  * appamattakena = 「わずかな、少しばかりの」  
  * vāyāmena = 「精進・努力によって」

* chindituṃ sakkā = 「断ち切ることができない」  
  * chindituṃ = 「断つこと（不定詞）」  
  * sakkā = 「可能である」→na と呼応して「不可能である」

→「いわゆる渇愛という流れは、少しばかりの努力では断ち切ることができない。」

* tasmā = 「それゆえ」  

* ñāṇasampayuttena = 「智慧と結びついた」  
  * ñāṇa = 「智慧・知」  
  * sampayutta = 「伴った、随伴した」

* mahantena parakkamena = 「大いなる奮励によって」  
  * mahanta = 「大きい、偉大な」  
  * parakkama = 「奮励、強い努力」

* parakkamitvā = 「奮励して・力を尽くして」  
  * parakkamati の絶対分詞。「奮励したのち」

* taṃ sotaṃ chinda = 「その流れを断て」  
  * taṃ = 「その」  
  * sotaṃ = 「流れを」  
  * chinda = 「断ち切れ」

→「それゆえ、智慧を伴った大いなる奮励をもって努めて、その流れを断て、ということである。」


### 4. Ubhopi kāme panuda nīhara.

* ubho pi = 「両方の……もまた」  
  * ubho = 「両方」  
  * pi = 「～もまた」

* kāme = 「欲を」  
  * ここでは二種の欲、すなわち  
    * kilesakāma（煩悩としての欲）  
    * vatthukāma（対象としての欲）  
    を指すと解される

* panuda = 「払いのけよ」  
* nīhara = 「取り除け、追い出せ」  
  * nīharati の命令形。「外へ出せ → 取り去れ」

→「二種の欲の両方を払いのけ、取り除け。」


### 5. Brāhmaṇāti khīṇāsavānaṃ ālapanametaṃ.

* brāhmaṇā ti = 「『brāhmaṇa（バラモンよ）』とは」  

* khīṇāsavānaṃ = 「煩悩の漏尽した者たちへの」  
  * khīṇa = 「尽きた」  
  * āsava = 「漏（煩悩）」  
→「煩悩の漏尽した聖者たちの」

* ālapanam etaṃ = 「呼びかけの語である」  
  * ālapana = 「呼びかけ、呼格」  
  * etaṃ = 「これは」

→「『バラモンよ』というのは、漏尽の聖者たちへの呼びかけの語である。」


### 6. Saṅkhārānanti pañcannaṃ khandhānaṃ khayaṃ jānitvā.

* saṅkhārānaṃ ti = 「『saṅkhārānaṃ（諸行の）』とは」  

* pañcannaṃ khandhānaṃ = 「五つの蘊の」  
  * pañca = 「五」  
  * khandha = 「蘊、集まり（色・受・想・行・識）」

* khayaṃ jānitvā = 「その滅尽を知って」  
  * khayaṃ = 「滅、消滅」  
  * jānitvā = 「知って、悟って」

→「五蘊という諸行の滅尽を悟って、という意味である。」


### 7. Akataññūti evaṃ sante tvaṃ suvaṇṇādīsu kenaci akatassa nibbānassa jānanato akataññū nāma hosīti.

* akataññū ti = 「『akataññū』とは」  

* evaṃ sante = 「このようであるときには」  
  * evaṃ = 「このように」  
  * sante（sati） = 「～であるならば、～であるとき」

* tvaṃ = 「あなたは」  

* suvaṇṇādīsu kenaci = 「金などいかなるものによっても」  
  * suvaṇṇa = 「金」  
  * ādi = 「～など」  
  * kenaci = 「いかなるものによっても」

* akatassa nibbānassa = 「（そのようなものによって）作られたのではない涅槃の」  
  * akata = 「作られていない、不作の」  
  * nibbāna = 「涅槃」

* jānanato = 「それを知るがゆえに」  
  * jānanato（jānana + tto） = 「知るということによって」

* akataññū nāma hosī ti = 「『不作のものを知る者』と名づけられるであろう」  
  * akataññū = 「不作のもの（＝無為・涅槃）を知る者」  
  * nāma = 「～と呼ばれる」  
  * hosi = 「～であろう、～となるであろう」

→「このようであるとき、あなたは、金などいかなるものによっても作られたのではない涅槃を知るがゆえに、『不作のものを知る者』と名づけられるであろう、という意味である。」


## 意訳

奮励して渇愛という流れを断ち切れ。  
真のバラモンよ、二種の欲をことごとく払いのけ、取り除きなさい。

五つの蘊としての諸行の滅び尽くすことを悟ったなら、  
金などいかなるものによっても作られたのではない無為の涅槃を知る  
「不作のものを知る者」と呼ぶべき聖なるバラモンとなるのである。

379.

‘‘Attanā codayattānaṃ, paṭimaṃsetha attanā;

So attagutto satimā, sukhaṃ bhikkhu vihāhisi.

380.

‘‘Attā hi attano nātho, ko hi nātho paro siyā;

Attā hi attano gati;

Tasmā saṃyamamattānaṃ, assaṃ bhadraṃva vāṇijo’’ti.

Tattha codayattānanti attanāva attānaṃ codaya sāraya. Paṭimaṃsethāti attanāva attānaṃ parivīmaṃsatha. Soti so tvaṃ, bhikkhu, evaṃ sante attanāva guttatāya attagutto, upaṭṭhitasatitāya satimā hutvā sabbiriyāpathesu sukhaṃ viharissasīti attho.

Nāthoti avassayo patiṭṭhā. Ko hi nātho paroti yasmā parassa attabhāve patiṭṭhāya kusalaṃ vā katvā saggaparāyaṇena maggaṃ vā bhāvetvā sacchikataphalena bhavituṃ na sakkā, tasmā ko hi nāma paro nātho bhaveyyāti attho. Tasmāti yasmā attāva attano gati patiṭṭhā saraṇaṃ, tasmā yathā bhadraṃ assājānīyaṃ nissāya lābhaṃ patthayanto vāṇijo tassa visamaṭṭhānacāraṃ pacchinditvā divasassa tikkhattuṃ nahāpento bhojento saṃyameti paṭijaggati, evaṃ tvampi anuppannassa akusalassa uppādaṃ nivārento satisammosena uppannaṃ akusalaṃ pajahanto attānaṃ saṃyama gopaya, evaṃ sante paṭhamajjhānaṃ ādiṃ katvā lokiyalokuttaravisesaṃ adhigamissasīti attho.

## 対訳

### 379偈

1. Attanā codayattānaṃ, paṭimaṃsetha attanā;

* attanā = 「自分によって、自らによって」（具格）
* codaya = 「奮い立たせよ、叱咤せよ」（動詞 codeti の命令形・二人称複数「奮起させよ」）
* attānaṃ = 「自分自身を」
  * codayattānaṃ = codayatha + attānaṃ の連声（「自分自身を奮い立たせよ」）
* paṭimaṃsetha = 「よく吟味せよ・検討せよ」（動詞 paṭimaṃsati の命令・二人称複数）
* attanā = 「自分自身によって」

→「自ら自分自身を奮い立たせよ、自ら自分自身をよく吟味せよ。」

2. So attagutto satimā, sukhaṃ bhikkhu vihāhisi.

* so = 「その者は、そのようにしたなら、その人は」
* attagutto = atta + gutto
  * atta = 「自己」
  * gutto = 「護られた、守られた」（gopeti/guṇāti の過去分詞）
  → 「自己によって護られている者、自らを守っている者」
* satimā = 「念のある者、気づきに満ちた者」（sati + mant）
* sukhaṃ = 「安楽に、幸福に」
* bhikkhu = 「比丘よ」
* vihāhisi = 「住むであろう・過ごすであろう」（vihārissasi の縮約・未来二人称単数）

→「そのように自己によって自らを守り、念ある者となるならば、比丘よ、あなたは安楽に住するであろう。」

### 380偈

3. Attā hi attano nātho, ko hi nātho paro siyā;

* attā = 「自己」
* hi = 「まさに～である、というのも」（強意・説明）
* attano = 「自分自身の」（属格）
* nātho = 「依り所、守り手、庇護者」
→ attā hi attano nātho = 「自己こそまさに自分自身の依り所である。」

* ko = 「誰が」
* hi = 「いったい、まさに」（反問・強意）
* nātho = 「依り所、守り手」
* paro = 「他者、他人」
* siyā = 「ありうるだろうか」（動詞 atthi の仮定法・三人称単数）
→ ko hi nātho paro siyā = 「他の者が依り所でありうるだろうか。」

→「自己こそ自分自身の依り所である。他の誰が依り所たりうるだろうか。」

4. Attā hi attano gati;

* attā = 「自己」
* hi = 「まさに～である」
* attano = 「自分自身の」
* gati = 「行き先、ゆくところ、帰趨・行路・運命、到達先」
→「自己こそまさに自分自身の行き先（帰趨・ゆく末）である。」

5. Tasmā saṃyamamattānaṃ, assaṃ bhadraṃva vāṇijo’’ti.

* tasmā = 「それゆえに、そのために」
* saṃyamaṃ = 「抑制、制御、節制」
* attānaṃ = 「自己を」
  * saṃyamam attānaṃ = 「自己を抑制すること、自己を制御すること」
* assaṃ = 「馬」
* bhadraṃ = 「良馬、立派な、善き」
* va = 「…のように」
* vāṇijo = 「商人」
→「それゆえ、自分自身を、良い馬を（扱う）商人が（そうする）ように、よく制御せよ。」

### 註釈部分

6. Tattha codayattānanti attanāva attānaṃ codaya sāraya.

* tattha = 「そこにおいて、この句において」
* codayattānan-ti = 「codayattānaṃ という語は…という意味である」
* attanā eva (attanāva) = 「まさに自分自身によって」
* attānaṃ = 「自分自身を」
* codaya = 「奮い立たせよ、叱咤せよ」
* sāraya = 「促せ、励ませ、進ませよ」（動詞 sārati の命令形）

→「ここで『codayattānaṃ』とは、『まさに自分自身によって自分自身を奮い立たせ、励ませ』という意味である。」

7. Paṭimaṃsethāti attanāva attānaṃ parivīmaṃsatha.

* paṭimaṃsethā-ti = 「paṭimaṃsetha というのは」
* attanā eva (attanāva) = 「まさに自分自身によって」
* attānaṃ = 「自分自身を」
* parivīmaṃsatha = 「よく調べよ・吟味せよ・省みよ」（parivīmaṃsati の命令・二人称複数）

→「『paṭimaṃsetha』とは、『まさに自分自身によって自分自身をよく省みよ・吟味せよ』という意味である。」

8. Soti so tvaṃ, bhikkhu, evaṃ sante attanāva guttatāya attagutto, upaṭṭhitasatitāya satimā hutvā sabbiriyāpathesu sukhaṃ viharissasīti attho.

* so-ti = 「so とは」
* so tvaṃ = 「その者とは、つまり汝である、比丘よ」
* bhikkhu = 「比丘よ」
* evaṃ sante = 「このようであるならば、かくある時」
* attanā eva (attanāva) = 「自己によって」
* guttatāya = 「守られているということによって」（gutta + tā → 「保護されている性質」）
* attagutto = 「自己によって守られた者、自らを守っている者」
* upaṭṭhita-satitāya = 「よく確立された念を持つことによって」
* satimā hutvā = 「念ある者となって」
* sabb-iriyāpathesu = 「すべての威儀・すべての姿勢・行動（四威儀）において」
* sukhaṃ = 「安楽に」
* viharissasi = 「住するであろう・過ごすであろう」（未来二人称単数）
* ti attho = 「という意味である」

→「『so』とは、すなわち汝である、比丘よ、という意味である。  
このようであるならば、すなわち、自己によって守られているゆえに自己によって護られた者となり、よく確立された念を有して念ある者となって、すべての威儀において安楽に住するであろう、という意味である。」

9. Nāthoti avassayo patiṭṭhā.

* nātho-ti = 「nātho とは」
* avassayo = 「依り所、頼みとするもの、庇護」
* patiṭṭhā = 「基盤、拠り所、支え」

→「『nātho』とは、依り所・拠り所のことである。」

10. Ko hi nātho paroti yasmā parassa attabhāve patiṭṭhāya kusalaṃ vā katvā saggaparāyaṇena maggaṃ vā bhāvetvā sacchikataphalena bhavituṃ na sakkā,

* ko hi nātho paro-ti = 「『他者が依り所でありうるだろうか』というのは」
* yasmā = 「なぜならば」
* parassa attabhāve = 「他人の身（他者の身体・存在）に依拠して」
* patiṭṭhāya = 「依りどころとして、基盤として」
* kusalaṃ vā katvā = 「善を作しても、善行をしても」
* saggaparāyaṇena = 「天界を行き先とするものとして（天界往生を目的とする行路として）」
* maggaṃ vā bhāvetvā = 「あるいは道（聖道）を修習して」
* sacchikataphalena = 「証悟された果（悟りの果）に伴って、証得された果によって」
* bhavituṃ na sakkā = 「存在することはできない、そうなることはできない」

→「『他者が依り所でありうるだろうか』とは、なぜならば、他人の身を基盤として善を為したり、天界を行き先とする行路としての道を修習したりして、その証得された果によって（自ら）そうなって在ることは〔決して〕できないからである、という意味である。」

11. Tasmā ko hi nāma paro nātho bhaveyyāti attho.

* tasmā = 「それゆえ」
* ko hi nāma = 「いったい誰が、いかなる人が」
* paro = 「他者が」
* nātho bhaveyya = 「依り所たりえようか」
* ti attho = 「という意味である」

→「それゆえ、『いったいどのような他者が依り所でありえようか』という意味である。」

12. Tasmāti yasmā attāva attano gati patiṭṭhā saraṇaṃ,

* tasmā-ti = 「『tasmā（それゆえ）』とは」
* yasmā = 「なぜならば」
* attā eva (attāva) = 「まさに自己こそ」
* attano = 「自分自身の」
* gati = 「行き先・帰趨・ゆく末」
* patiṭṭhā = 「拠り所・基盤」
* saraṇaṃ = 「帰依、依り所、保護」

→「『それゆえ』とは、『なぜならば、自己こそまさに自分自身の行き先であり、拠り所であり、帰依すべき依り所だからである』という意味である。」

13. Tasmā yathā bhadraṃ assājānīyaṃ nissāya lābhaṃ patthayanto vāṇijo tassa visamaṭṭhānacāraṃ pacchinditvā divasassa tikkhattuṃ nahāpento bhojento saṃyameti paṭijaggati,

* tasmā = 「それゆえ」
* yathā = 「ちょうど～するように」
* bhadraṃ = 「良い、立派な」
* assa-ājānīyaṃ = 「良種の馬・名馬」
* nissāya = 「それに依り頼んで」
* lābhaṃ = 「利得、利益」
* patthayanto = 「望み求めている（現在分詞）」
* vāṇijo = 「商人は」
* tassa = 「その（馬）の」
* visama-ṭṭhāna-cāraṃ = 「悪路への出歩き、険しい場所への行き来」
  * visama = 「でこぼこな、危険な、険しい」
  * ṭṭhāna = 「場所」
  * cāra = 「歩行、動き、行き来」
* pacchinditvā = 「断ち切って、やめさせて」
* divasassa tikkhattuṃ = 「一日に三度」
* nahāpento = 「（その馬を）洗わせつつ、沐浴させつつ」
* bhojento = 「食べさせつつ、養いつつ」
* saṃyameti = 「抑制する、節度をもって扱う、制御する」
* paṭijaggati = 「よく世話をする、養護する」

→「それゆえ、ちょうど良い名馬に依り頼んで利益を望む商人が、その馬に険しい場所を歩かせることをやめさせ、一日に三度沐浴させ、食べさせて、抑制しつつよく世話をするように、」

14. evaṃ tvampi anuppannassa akusalassa uppādaṃ nivārento satisammosena uppannaṃ akusalaṃ pajahanto attānaṃ saṃyama gopaya,

* evaṃ = 「このように、同じように」
* tvaṃ pi (tvampi) = 「あなたもまた」
* anuppannassa = 「まだ起こっていない」
* akusalassa = 「不善（の心・状態）の」
* uppādaṃ = 「生起、起こること」
* nivārento = 「防いでいる、阻止している（現在分詞）」
* sati-sammosena = 「念の欠如・念の失念によって」
  * sati = 「念」
  * sammoṣa = 「失念、失われること」
* uppannaṃ = 「すでに起こった」
* akusalaṃ = 「不善（の心・状態）を」
* pajahanto = 「捨てている、断っている（現在分詞）」
* attānaṃ = 「自己を」
* saṃyama = 「抑制、制御、節制」
* gopaya = 「守れ、護れ」（動詞 gopeti の命令形）

→「同じように、あなたもまた、まだ起こっていない不善の生起を防ぎ、念の失念によってすでに起こってしまった不善を捨て去りながら、自己を抑制によって守れ。」

15. evaṃ sante paṭhamajjhānaṃ ādiṃ katvā lokiyalokuttaravisesaṃ adhigamissasīti attho.

* evaṃ sante = 「このようであるならば、かくあれば」

* ādiṃ katvā = 「それを初めとして、それを始まりとして」

  * lokiya = 「世間的、世俗的」
  * lokuttara = 「出世間的、聖なる」
  * visesa = 「殊勝、格別の境地」
* adhigamissasi = 「獲得するであろう、到達するであろう」（未来二人称単数）
* ti attho = 「という意味である」

→「このようであるならば、初禅をはじめとして、世間的および出世間的な殊勝な境地を獲得するであろう、という意味である。」

## 意訳

379

自ら、自分自身を奮い立たせよ。  
自ら、自分自身をよく省みよ。  

そうすれば、自己の力によって自らを守り、  
念をしっかりと保つ者となって、  
比丘よ、あなたはあらゆる行いにおいて安らかに暮らすことができるだろう。

380

自己こそ、自分自身の拠り所である。  
ほかの誰が拠り所たりうるだろうか。  

自己こそ、自分自身の行き先・行く末を決めるものである。  
だからこそ、自分自身を、  
立派な名馬を扱う商人がそれを大切に制御して世話するように、  
よく抑え、よく守らなければならない。

というのも、他人の身体を拠り所として善を行ったり、  
天界行きを目指して道を修めたりしても、  
その果を自分が得ることはできないからである。  

このように、まだ起こっていない不善が起こるのを防ぎ、  
念を失ったことによってすでに起こってしまった不善を捨て去り、  
自己を抑制によって守るならば、  

あなたは初禅をはじめとして、  
世間の上でも、また出世間の悟りにおいても、  
すぐれた境地に到達することができるであろう。

389.

‘‘Na brāhmaṇassa pahareyya, nāssa muñcetha brāhmaṇo;

Dhī brāhmaṇassa hantāraṃ, tato dhī yassa muñcati.

390.

‘‘Na brāhmaṇassetadakiñci seyyo, yadā nisedho manaso piyehi;

Yato yato hiṃsamano nivattati, tato tato sammatimeva dukkha’’nti.

Tattha pahareyyāti ‘‘khīṇāsavabrāhmaṇohamasmī’’ti jānanto khīṇāsavassa vā aññatarassa vā jātibrāhmaṇassa na pahareyya. Nāssa muñcethāti sopi pahaṭo khīṇāsavabrāhmaṇo assa paharitvā ṭhitassa veraṃ na muñcetha, tasmiṃ kopaṃ na kareyyāti attho. Dhī brāhmaṇassāti khīṇāsavabrāhmaṇassa hantāraṃ garahāmi. Tato dhīti yo pana taṃ paharantaṃ paṭipaharanto tassa upari veraṃ muñcati, taṃ tatopi garahāmiyeva.

Etadakiñci seyyoti yaṃ khīṇāsavassa akkosantaṃ vā apaccakkosanaṃ, paharantaṃ vā appaṭipaharaṇaṃ, etaṃ tassa khīṇāsavabrāhmaṇassa na kiñci seyyo, appamattakaṃ seyyo na hoti, adhimattameva seyyoti attho. Yadā nisedho manaso piyehīti kodhanassa hi kodhuppādova manaso piyo nāma. Kodho hi panesa mātāpitūsupi buddhādīsupi aparajjhati. Tasmā yo assa tehi manaso nisedho kodhavasena uppajjamānassa cittassa niggaho, etaṃ na kiñci seyyoti attho. Hiṃsamanoti kodhamano. So tassa yato yato vatthuto anāgāmimaggena samugghātaṃ gacchanto nivattati . Tato tatoti tato tato vatthuto sakalampi vaṭṭadukkhaṃ nivattatiyevāti attho.

## 対訳

### 偈頌 389

1. Na brāhmaṇassa pahareyya,

* na = 「～してはならない」
* brāhmaṇassa = 「バラモンに（対して）」※ここでは「煩悩を滅した聖者＝真のバラモン」
* pahareyya = 「打つべきだ、打ってよい」という意味の仮定法 → 「打ってはならない」

→「バラモンには打ちかかってはならない。」

2. nāssa muñcetha brāhmaṇo;

* na = 「～してはならない」
* assa = 「その者に対して／その者に向けて（paharitvā ṭhitassa：打って立っている者に）」
* muñcetha = 「放つべきだ、解き放つべきだ」という意味の仮定法 → 「（瞋恚・怨みなどを）放ってはならない」
* brāhmaṇo = 「バラモンは（が）」

→「また、バラモンもその者に対して（怨みを）放ってはならない。」

3. Dhī brāhmaṇassa hantāraṃ,

* dhī = 「呪うべきだ、恥ずべきだ、軽蔑すべきだ」
* brāhmaṇassa = 「バラモンを」
* hantāraṃ = 「殺す者を、殺害者を」

→「バラモンを殺す者は恥ずべきである。」

4. tato dhī yassa muñcati.

* tato = 「それよりもなお、さらにそれ以上に」
* dhī = 「呪うべきだ、恥ずべきだ」
* yassa = 「（その）者が」
* muñcati = 「放つ、解き放つ（ここでは『怨み・瞋恚を放つ』）」

→「しかし、その上、（打たれて）怨みを放つ者は、なおさら恥ずべきである。」

### 偈頌 390

5. Na brāhmaṇassetadakiñci seyyo,

* na = 「～ではない」
* brāhmaṇassa = 「バラモンにとって」
* etad = 「これ（＝先に説かれた『打たれても打ち返さないこと』）」
* akiñci = 「少しも、いささかも」
* seyyo = 「よりよい、すぐれている」

→「バラモンにとって、このことにまさるものは、いささかも存在しない。」

6. yadā nisedho manaso piyehi;

* yadā = 「…するとき」
* nisedho = 「抑止、制止、抑えつけること」
* manaso = 「心が（から）、心に関して」
* piyehi = 「好むもの・愛するものから、心地よいと感じるものから」

→「すなわち、心が愛着しているものどもから抑え止められるときである。」

7. Yato yato hiṃsamano nivattati,

* yato yato = 「どこからどこから、どの対象からどの対象から」
* hiṃsa-mano = 「害意ある心、他を害そうとする心 → 瞋恚の心」
* nivattati = 「退く、離れる、背を向ける、やめる」

→「害意ある心が、どの対象から離れ去るところであれ、」

8. tato tato sammatimeva dukkha’’nti.

* tato tato = 「ちょうどその対象その対象から」
* sammati = 「止む、鎮まる、終息する」
* eva = 「まさに、ちょうどそのように」
* dukkhaṃ = 「苦しみ」

→「まさにその対象その対象から、苦しみは止んでいくのである。」

### 註釈部分

9. Tattha pahareyyāti ‘‘khīṇāsavabrāhmaṇohamasmī’’ti jānanto khīṇāsavassa vā aññatarassa vā jātibrāhmaṇassa na pahareyya.

* tattha = 「そこにおいて、この箇所において」
* pahareyyā ti = 「『pahareyya（打ってはならない）』というのは…という意味である」
* ‘‘khīṇāsavabrāhmaṇohamasmī’’ti = 「『私は煩悩を滅したバラモンである』と」
* jānanto = 「知っている者は、自覚している者は」
* khīṇāsavassa vā = 「（相手が）煩悩を滅した者であろうと」
* aññatarassa vā = 「あるいは、その他の（誰か）であろうと」
* jātibrāhmaṇassa = 「生まれによるバラモンである者に対してであろうと」
* na pahareyya = 「打ってはならない」

→「ここで『打ってはならない』とは、『私は煩悩を滅したバラモンである』と知っている者は、煩悩を滅した者であろうと、あるいは生まれによるバラモンである者であろうと、誰に対しても打ちかかってはならない、という意味である。」

10. Nāssa muñcethāti sopi pahaṭo khīṇāsavabrāhmaṇo assa paharitvā ṭhitassa veraṃ na muñcetha, tasmiṃ kopaṃ na kareyyāti attho.

* na assa muñcethā ti = 「『nāssa muñcetha』とは…という意味である」
* so pi = 「その者もまた」
* pahaṭo = 「打たれたとき、傷つけられたとき」
* khīṇāsava-brāhmaṇo = 「煩悩を滅したバラモンは」
* assa = 「その者（自分を打った者）の」
* paharitvā ṭhitassa = 「打って立っている者の」
* veraṃ = 「怨み、敵意」
* na muñcetha = 「解き放ってはならない、向けてはならない」
* tasmiṃ = 「その者に対して」
* kopaṃ = 「怒り」
* na kareyya = 「起こしてはならない」

→「『また、その者に対して（怨みを）放ってはならない』とは、煩悩を滅したバラモンもまた、打たれたときに、自分を打って立っている者に怨みを向けてはならず、その者に対して怒りを起こしてはならない、という意味である。」

11. Dhī brāhmaṇassāti khīṇāsavabrāhmaṇassa hantāraṃ garahāmi.

* dhī brāhmaṇassā ti = 「『Dhī brāhmaṇassa』とは」
* khīṇāsava-brāhmaṇassa = 「煩悩を滅したバラモンを」
* hantāraṃ = 「殺す者を、殺害者を」
* garahāmi = 「私は非難する、咎める」

→「『バラモンに対して dhī である』とは、『煩悩を滅したバラモンを殺す者を私は非難する』という意味である。」

12. Tato dhīti yo pana taṃ paharantaṃ paṭipaharanto tassa upari veraṃ muñcati, taṃ tatopi garahāmiyeva.

* tato dhī ti = 「『tato dhī』とは」
* yo pana = 「しかし、…なる者は」
* taṃ = 「その（自分を）打つ者を」
* paharantaṃ = 「打っている者を」
* paṭi-paharanto = 「打ち返しつつ、報復しつつ」
* tassa upari = 「その者に対して」
* veraṃ = 「怨み、敵意」
* muñcati = 「放つ、向ける」
* taṃ = 「その者を」
* tato pi = 「それ（単に殺す者）よりもさらに」
* garahāmi yeva = 「私はまさに非難する」

→「『それよりもなお dhī である』とは、しかし自分を打つ者に打ち返しつつ、その者に対して怨みを向ける者を、私はその（単に殺す者）よりも、さらにいっそう非難する、という意味である。」

13. Etadakiñci seyyoti yaṃ khīṇāsavassa akkosantaṃ vā apaccakkosanaṃ, paharantaṃ vā appaṭipaharaṇaṃ, etaṃ tassa khīṇāsavabrāhmaṇassa na kiñci seyyo, appamattakaṃ seyyo na hoti, adhimattameva seyyoti attho.

* etad akiñci seyyo ti = 「『etadakiñci seyyo』とは」
* yaṃ = 「…ということを」
* khīṇāsavassa = 「煩悩を滅した者が」
* akkosantaṃ vā = 「罵ってくる者に対しても」
* apaccakkosanaṃ = 「罵り返さないこと」
* paharantaṃ vā = 「打ってくる者に対しても」
* appaṭipaharaṇaṃ = 「打ち返さないこと」
* etaṃ = 「このようなことは」
* tassa khīṇāsava-brāhmaṇassa = 「その煩悩を滅したバラモンにとって」
* na kiñci seyyo = 「『ほんの少しよい』という程度のものではない」
* appamattakaṃ seyyo na hoti = 「わずかに勝れているというのではない」
* adhimattaṃ eva seyyo = 「きわめて、最高度に勝れているのである」

→「『これが、いささかもよりよいものではない』とは、煩悩を滅した者が、罵ってくる者に対しても罵り返さないこと、打ってくる者に対しても打ち返さないこと、これらはその煩悩を滅したバラモンにとって、単に『少しばかりよい』ものではなく、わずかに勝れているにとどまるのではなく、きわめて高度に勝れている、という意味である。」

14. Yadā nisedho manaso piyehīti kodhanassa hi kodhuppādova manaso piyo nāma.

* yadā nisedho manaso piyehī ti = 「『心の好むものからの抑止があるときに』とは」
* kodhanassa hi = 「怒りっぽい人にとっては、まさしく」
* kodhu-uppādo eva = 「怒りの生起そのものが」
* manaso piyo nāma = 「『心にとって好ましいもの』と呼ばれるのである」

→「『心の好むものから抑え止められるとき』とは、怒りっぽい人にとっては、まさに怒りの起こること自体が『心に好ましいもの』と見なされている、ということである。」

15. Kodho hi panesa mātāpitūsupi buddhādīsupi aparajjhati.

* kodho hi pana esā = 「ところが、この怒りというものは」
* mātā-pitūsu pi = 「父母に対してさえ」
* buddha-ādīsu pi = 「仏陀などに対してさえ」
* aparajjhati = 「罪を犯させる、過ちを犯させる」

→「というのも、この怒りというものは、父母に対してでさえ、また仏陀などに対してでさえ、過ちを犯させるからである。」

16. Tasmā yo assa tehi manaso nisedho kodhavasena uppajjamānassa cittassa niggaho, etaṃ na kiñci seyyoti attho.

* tasmā = 「それゆえ」
* yo assa tehi = 「それ（怒りなど、心の好むものら）からの」
* manaso nisedho = 「心の抑止」
* kodha-vasena uppajjamānassa cittassa = 「怒りの支配のもとに起こりつつある心の」
* niggaho = 「制御、抑え込み」
* etaṃ = 「このことが」
* na kiñci seyyo ti attho = 「『いささかも（それにまさるものが）ないほどによい』という意味である」

→「それゆえ、そのような怒りなどから心を抑え止めること、すなわち怒りのもとに起こりつつある心を制御することこそが、『これにまさるものは、いささかも存在しないほどによい』という意味なのである。」

17. Hiṃsamanoti kodhamano.

* hiṃsamanoti = 「『hiṃsamano（害意ある心）』とは」
* kodha-mano = 「怒りの心、瞋恚の心のことである」

→「『害意ある心』とは、怒りの心のことである。」

18. So tassa yato yato vatthuto anāgāmimaggena samugghātaṃ gacchanto nivattati .

* so = 「その（怒りの）心は」
* tassa = 「それの」
* yato yato vatthuto = 「どの対象どの対象から」
* anāgāmi-maggena = 「不還果に至る道によって」
* samugghātaṃ gacchanto = 「根絶へと向かっていくときに」
* nivattati = 「（そこから）退く、離れ去る」

→「その怒りの心が、不還道によって根絶へと向かい、ある対象ある対象から退いていくところでは、」

19. Tato tatoti tato tato vatthuto sakalampi vaṭṭadukkhaṃ nivattatiyevāti attho.

* tato tato ti = 「『tato tato』とは」
* tato tato vatthuto = 「まさにその対象その対象から」
* sakalam pi vaṭṭa-dukkhaṃ = 「輪廻全体の苦しみがことごとく」
* nivattati eva = 「退いていく、止んでいくのである」
* ti attho = 「という意味である」

→「『そのところそのところから』とは、まさにその対象その対象から、輪廻全体の苦しみがことごとく退いていくのである、という意味である。」

## 意訳

バラモン（ここでは煩悩をすでに滅ぼした聖者）に対しては、誰であれ暴力をふるってはならない。また、そのバラモン自身も、たとえ人から打たれても、その相手に対して怨みや怒りを向けてはならない。  
煩悩を滅したバラモンを殺す者は、確かに非難されるべきである。しかし、それ以上に非難されるべきなのは、打たれたときに打ち返し、その相手に対して怨みを解き放つバラモンである。

煩悩を滅したバラモンにとって、罵られても罵り返さず、打たれても打ち返さないこと──これにまさるものは、少しも存在しない。それは、わずかに勝れているどころではなく、きわめて最高に勝れたあり方なのである。  
怒りっぽい人にとっては、怒りの起こること自体が、実は「心にとって好ましいもの」と感じられている。しかし、この怒りというものは、父母に対してでさえ、さらには仏陀に対してでさえ、過ちを犯させてしまう。だからこそ、そうした「心の好むもの」から心を引き戻し、怒りによって起こりつつある心を抑え制御すること──これに勝る善は他にないのである。

害意に満ちた怒りの心が、不還道によって根こそぎ滅ぼされていく過程で、ある対象から、またある対象からと、順次そこから退いていくとき、その対象その対象に結びついた輪廻の苦しみは、まさにそこから鎮まり、止んでいくのである。

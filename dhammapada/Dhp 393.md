393.

‘‘Na jaṭāhi na gottena, na jaccā hoti brāhmaṇo;

Yamhi saccañca dhammo ca, so sucī so ca brāhmaṇo’’ti.

Tattha saccanti yasmiṃ puggale cattāri saccāni soḷasahākārehi paṭivijjhitvā ṭhitaṃ saccañāṇañceva navavidho ca lokuttaradhammo atthi, so suci, so brāhmaṇo cāti attho.

## 対訳

1. ‘‘Na jaṭāhi na gottena, na jaccā hoti brāhmaṇo;

* na = 「～ではない」
* jaṭāhi = 「髪を束ねた髪（ジャター）によって」〔jaṭā の具格複数〕
* na = 「～によってではなく」
* gottena = 「家系・氏族（ゴットラ）によって」〔gotta の具格単数〕
* na = 「～によってではない」
* jaccā = 「生まれによって、生まれによる身分によって」〔jāti の具格・副詞的用法〕
* hoti = 「なる、…となる」
* brāhmaṇo = 「バラモン、ブラーフマナ（聖者／バラモン種姓の人）」

→「髪を束ねた髪によってでもなく、家系によってでもなく、生まれによってでも、バラモン（聖者）となるのではない。」

2. Yamhi saccañca dhammo ca, so sucī so ca brāhmaṇo’’ti.

* yamhi = 「その人のうちに（…が）あるところの者においては」〔関係代名詞 yo の処格〕
* saccaṃ = 「真理、真実」
* ca = 「そして」
* dhammo = 「法（ダンマ）、正法、義にかなった行い・状態」
* ca = 「また」
* so = 「その者は」
* sucī = 「清浄である、清らかな者である」
* so = 「その者は」
* ca = 「そして」
* brāhmaṇo = 「バラモン（聖者）である」
* ’’ti = 「～と（説かれている）」を示す引用終止

→「そのうちに真理と法とが具わっている者こそ、その者は清浄であり、その者こそバラモン（聖者）である、と。」

3. Tattha saccanti yasmiṃ puggale cattāri saccāni soḷasahākārehi paṭivijjhitvā ṭhitaṃ saccañāṇañceva navavidho ca lokuttaradhammo atthi, so suci, so brāhmaṇo cāti attho.

* tattha = 「そこにおいては、この偈においては」
* saccaṃ ti = 「『saccaṃ（真理）』という語は」
* yasmiṃ puggale = 「ある人物の中において、その人において」〔puggala = 「人、人格」〕
* cattāri saccāni = 「四つの諦、四聖諦」
* soḷasahākārehi = 「十六のあり方・側面によって」  
  * soḷasa = 「16」  
  * ākāra = 「あり方、様態」 → 「四諦を十六の様態において」
* paṭivijjhitvā = 「（それらを）洞察し悟り取って、徹底的に理解して」
* ṭhitaṃ = 「確立している、確定している、（動揺せず）安立している」
* saccañāṇaṃ = 「諦についての智、四諦智」
* ca eva (ceva) = 「そしてまさに、そのうえ」
* navavidho = 「九種の」
* ca = 「そして」
* lokuttaradhammo = 「出世間の法（超世間の法）」  
  〔四道・四果と涅槃から成る九種の出世間法〕
* atthi = 「がある、備わっている」
* so suci = 「その者は清浄である」
* so brāhmaṇo ca = 「その者こそバラモン（聖者）である」
* ti attho = 「という意味である」

→「ここで『真理（saccaṃ）』とは、ある人のうちに四つの聖なる真理が十六の側面をもって洞察され、確立している四諦智、そして九種の出世間法が存することを意味する。そのような者は清浄であり、その者こそバラモン（聖者）である、という意味である。」

---

## 意訳

「髪を束ねているからといっても、名門の家系だからといっても、生まれがバラモンだからといっても、人はバラモン（真の聖者）にはならない。

その人の内に、真理と正しい法が確立している者こそ、清らかな人であり、その人こそまことのバラモンである。

ここでいう『真理』とは、その人が四つの聖なる真理を十六の側面にわたって悟り取り、それに基づく四諦の智と、四道・四果および涅槃から成る九種の出世間の法がその人に備わっていることを指す。そのような者が清浄であり、その人こそ真のバラモンである、ということである。」


369.

‘‘Siñca bhikkhu imaṃ nāvaṃ, sittā te lahumessati;

Chetvā rāgañca dosañca, tato nibbānamehisi.

Siñca bhikkhu imaṃ nāvanti bhikkhu imaṃ attabhāvasaṅkhātaṃ nāvaṃ micchāvitakkodakaṃ chaḍḍento siñca. Sittā te lahumessatīti yathā hi mahāsamudde udakasseva bharitā nāvā chiddāni pidahitvā udakassa sittatāya sittā sallahukā hutvā mahāsamudde anosīditvā sīghaṃ supaṭṭanaṃ gacchati, evaṃ tavāpi ayaṃ micchāvitakkodakabharitā attabhāvanāvā cakkhudvārādichiddāni saṃvarena pidahitvā uppannassa micchāvitakkodakassa sittatāya sittā sallahukā saṃsāravaṭṭe anosīditvā sīghaṃ nibbānaṃ gamissati. Chetvāti rāgadosabandhanāni chinda. Etāni hi chinditvā arahattappatto tato aparabhāge anupādisesanibbānameva ehisi, gamissasīti attho.

## 対訳

1. ‘‘Siñca bhikkhu imaṃ nāvaṃ, sittā te lahumessati;

* siñca = 「汲み出せ・排水せよ・空にせよ」（siñcati の命令形）
* bhikkhu = 「比丘よ」
* imaṃ = 「この」
* nāvaṃ = 「舟を・船を」
* sittā = 「汲み出されて・水を捨てられて（空にされて）」
* te = 「あなたにとって・あなたのために」
* lahum essati = 「軽くなるであろう」（lahu「軽い」＋将来形 essati）

→ 「比丘よ、この舟の水を汲み出せ。この舟が水を捨てられて空になれば、あなたにとって軽くなるであろう。」

2. Chetvā rāgañca dosañca, tato nibbānamehisi.

* chetvā = 「断ち切って・切り捨てて」（chindati の絶対分詞）
* rāgaṃ ca = 「愛欲を、また」
* dosaṃ ca = 「瞋恚をも」
* tato = 「それから・そののちに」
* nibbānaṃ = 「涅槃に」
* ehisi = 「到るであろう・赴くであろう」（ehati の将来形・2人称）

→ 「愛欲と瞋恚を断ち切ったなら、そののちに、あなたは涅槃へと到るであろう。」

---

3. Siñca bhikkhu imaṃ nāvanti bhikkhu imaṃ attabhāvasaṅkhātaṃ nāvaṃ micchāvitakkodakaṃ chaḍḍento siñca.

* Siñca bhikkhu imaṃ nāvanti = 「『Siñca bhikkhu imaṃ nāvaṃ（比丘よ、この舟の水を汲み出せ）』というのは」
* bhikkhu = 「比丘よ」
* imaṃ = 「この」
* attabhāva‑saṅkhātaṃ = 「attabhāva（この身・個体的存在）と呼ばれる」
  * attabhāva = 「この身・自らの存在・五取蘊としての個体」
  * saṅkhāta = 「…と名づけられた・…と呼ばれる」
* nāvaṃ = 「舟を」
* micchā‑vitakka‑udakaṃ = 「邪思惟という水を」
  * micchā‑vitakka = 「邪な思惟・誤った思考」
  * udaka = 「水」
* chaḍḍento = 「捨て去りつつ・投げ捨てつつ」
* siñca = 「汲み出せ・排水せよ」

→ 「『比丘よ、この舟の水を汲み出せ』とは、『比丘よ、この身という attabhāva と呼ばれる舟から、邪思惟という水を捨て去りつつ汲み出せ（空にせよ）』という意味である。」

---

4. Sittā te lahumessatīti yathā hi mahāsamudde udakasseva bharitā nāvā chiddāni pidahitvā udakassa sittatāya sittā sallahukā hutvā mahāsamudde anosīditvā sīghaṃ supaṭṭanaṃ gacchati,

* Sittā te lahum essatīti = 「『水を捨てられてあなたにとって軽くなるであろう』とは」
* yathā hi = 「ちょうど…のように」
* mahā‑samudde = 「大海において」
* udakassa‑eva bharitā nāvā = 「水だけで満たされた舟は」
  * udakassa eva = 「まさに水で」
  * bharitā = 「満たされた」
  * nāvā = 「舟が」
* chiddāni = 「穴を」
* pidahitvā = 「ふさいでから・塞いで」
* udakassa sittatāya = 「水が汲み出されたことによって」
  * sittatā = 「汲み出されているという状態」
* sittā = 「（舟が）水を汲み出されて空にされ」
* sallahukā hutvā = 「きわめて軽くなって」
* mahā‑samudde anosīditvā = 「大海の中で沈むことなく」
  * anosīditvā = 「沈まずに」
* sīghaṃ = 「速やかに」
* su‑paṭṭanaṃ = 「良い港へ・安穏な港へ」
* gacchati = 「到る・進む」

→ 「『水を捨てられて軽くなる』とは、たとえば大海において、水でいっぱいになった舟が、穴をふさぎ、水を汲み出されたことによって空にされて非常に軽くなり、大海に沈むことなく、速やかに良き港へと進むようなものである。」

---

5. evaṃ tavāpi ayaṃ micchāvitakkodakabharitā attabhāvanāvā cakkhudvārādichiddāni saṃvarena pidahitvā uppannassa micchāvitakkodakassa sittatāya sittā sallahukā saṃsāravaṭṭe anosīditvā sīghaṃ nibbānaṃ gamissati.

* evaṃ = 「このように・同様に」
* tava‑api = 「あなたにとってもまた」
* ayaṃ attabhāva‑nāvā = 「この attabhāva という舟は」
  * attabhāva‑nāvā = 「この身という名の舟」
* micchā‑vitakka‑udaka‑bharitā = 「邪思惟という水で満たされている」
  * bharitā = 「満たされた」
* cakkhu‑dvāra‑ādi‑chiddāni = 「眼門などの諸々の穴を」
  * cakkhu‑dvāra = 「眼の門（眼根の門）」
  * ādi = 「など」
  * chiddāni = 「穴」
* saṃvarena = 「抑制によって・防護によって」
* pidahitvā = 「ふさいでから・閉ざして」
* uppannassa micchā‑vitakka‑udakassa = 「すでに生じた邪思惟という水を」
* sittatāya = 「汲み出したことによって」
* sittā = 「（その舟が）水を汲み出されて空にされ」
* sallahukā = 「非常に軽くなって」
* saṃsāra‑vaṭṭe = 「輪廻の流転の中で・生死流転において」
* anosīditvā = 「沈み込むことなく・沈没することなく」
* sīghaṃ = 「速やかに」
* nibbānaṃ = 「涅槃へ」
* gamissati = 「到るであろう・赴くであろう」

→ 「同様に、あなたにとっても、この身という舟は、邪思惟という水で満たされているが、眼門などの穴を防護によってふさぎ、生じた邪思惟という水を汲み出すことによって空にされ、非常に軽くなって、生死流転の大海の中に沈み込むことなく、速やかに涅槃へと到るであろう。」

---

6. Chetvāti rāgadosabandhanāni chinda.

* Chetvāti = 「『chetvā（断ち切って）』とは」
* rāga‑dosa‑bandhanāni = 「愛欲と瞋恚という繋縛を」
  * rāga = 「愛欲・貪愛」
  * dosa = 「瞋恚・怒り」
  * bandhana = 「繋縛・束縛」
* chinda = 「断て・切り離せ」

→ 「『断ち切って』とは、愛欲と瞋恚という繋縛を断て、という意味である。」

---

7. Etāni hi chinditvā arahattappatto tato aparabhāge anupādisesanibbānameva ehisi, gamissasīti attho.

* etāni hi = 「まさにこれら（の繋縛）を」
* chinditvā = 「断ち切って」
* arahattappatto = 「阿羅漢果に到達した者は」
  * arahattā‑p‑patta = 「阿羅漢という境地に達した」
* tato = 「それから・そののち」
* aparabhāge = 「さらに先において・後の段階で」
* anupādisesa‑nibbānaṃ‑eva = 「無余依涅槃そのものへ」
  * anupādisesa = 「余りの取蘊（執着の依り所）をもたない」
  * nibbānaṃ = 「涅槃」
  * eva = 「こそ・まさに」
* ehisi = 「到るであろう・赴くであろう」
* gamissasīti attho = 「すなわち『行くであろう』という意味である」

→ 「というのも、まさにこれら（愛欲と瞋恚の繋縛）を断ち切って阿羅漢果に到達した者は、そののちには、無余依涅槃そのものへと赴くであろう、すなわち『行くであろう』という意味なのである。」

---

## 意訳

「比丘よ、この舟の水を汲み出しなさい。この舟が水を捨てて空になれば、あなたにとって軽くなるだろう。  
愛欲と瞋恚を断ち切るなら、そののち、あなたは涅槃へと到ることになる。

ここで『この舟』とは、この身、この個体的存在を舟にたとえたものである。これを満たしている邪な思考という水を捨て去りつつ、汲み出しなさい。

ちょうど大海の中で、水でいっぱいになった舟が、穴という穴をふさぎ、水を汲み出して空にされ、とても軽くなって大海に沈むことなく、速やかに良い港へと進んで行くように──

それと同じく、あなたのこの身という舟も、邪な思惟という水で満ちているが、眼の門などの感覚の穴を防護によってふさぎ、生じた邪思惟という水を汲み出して空にすれば、きわめて軽くなって、生死流転の大海に沈み込むことなく、速やかに涅槃へと到るであろう。

ここで『断ち切って』とは、愛欲と瞋恚という束縛を断て、ということである。  
これらの束縛を断ち切って阿羅漢の境地に達したなら、そののちには、無余依涅槃そのものへと赴くことになるのである。」

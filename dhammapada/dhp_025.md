25.

‘‘Uṭṭhānenappamādena , saṃyamena damena ca;

Dīpaṃ kayirātha medhāvī, yaṃ ogho nābhikīratī’’ti.

Tattha dīpaṃ kayirāthāti vīriyasaṅkhātena uṭṭhānena, satiyā avippavāsākārasaṅkhātena appamādena, catupārisuddhisīlasaṅkhātena saṃyamena, indriyadamena cāti imehi kāraṇabhūtehi catūhi dhammehi dhammojapaññāya samannāgato medhāvī imasmiṃ ativiya dullabhapatiṭṭhatāya atigambhīre saṃsārasāgare attano patiṭṭhānabhūtaṃ arahattaphalaṃ dīpaṃ kayirātha kareyya, kātuṃ sakkuṇeyyāti attho. Kīdisaṃ? Yaṃ ogho nābhikīratīti yaṃ catubbidhopi kilesogho abhikirituṃ viddhaṃsetuṃ na sakkoti. Na hi sakkā arahattaṃ oghena abhikiritunti.

## 対訳

### 偈文 (Gāthā)

1. Uṭṭhānenappamādena, saṃyamena damena ca;

  * uṭṭhānena = 「奮起・努力によって」
  * appamādena = 「不放逸（気づきを怠らないこと）によって」
  * saṃyamena = 「自制によって」
  * damena = 「調御によって」
  * ca = 「そして」

→「奮起と不放逸により、また自制と調御によって」

2. Dīpaṃ kayirātha medhāvī, yaṃ ogho nābhikīratī ti.

  * dīpaṃ = 「島（避難所）を」
    * 文脈上の ogha（洪水）に対応するため「灯明」ではなく「島（中洲）」と解する。
  * kayirātha = 「作るべきである（optative）」
  * medhāvī = 「賢者は」
  * yaṃ = 「いかなるものを（関係代名詞）」
  * ogho = 「洪水が」
  * na abhikīrati = 「圧倒しない、押し流さない」
  * ti = （引用の終わり）

→「賢者は、洪水が押し流すことのできない島を作るべきである。」

---

### 註釈 (Aṭṭhakathā)

3. Tattha dīpaṃ kayirāthāti

  * tattha = 「そこで（その偈において）」
  * dīpaṃ kayirātha iti = 「『島を作るべし』とは」

→「そこでの『島を作るべし』という言葉の解説は以下の通りである。」

4. vīriyasaṅkhātena uṭṭhānena,

  * vīriya-saṅkhātena = 「精進と呼ばれる」
  * uṭṭhānena = 「奮起によって」

→「精進と呼ばれる『奮起』によって、」

5. satiyā avippavāsākārasaṅkhātena appamādena,

  * satiyā = 「念（サティ）から」
  * avippavāsa-ākāra = 「離れない状態」
  * saṅkhātena = 「～と呼ばれる」
  * appamādena = 「不放逸によって」

→「念（気づき）から離れない状態である『不放逸』によって、」

6. catupārisuddhisīlasaṅkhātena saṃyamena,

  * catu-pārisuddhi-sīla = 「四種の清浄なる戒（別解脱律儀、根律儀、活命清浄、資具依止）」
  * saṅkhātena = 「～と呼ばれる」
  * saṃyamena = 「自制によって」

→「四種清浄戒と呼ばれる『自制』によって、」

7. indriyadamena cāti

  * indriya-damena = 「感官（根）の調御によって」
  * ca iti = 「～と（いうことである）」

→「そして感官を制御する『調御』によって、」

8. imehi kāraṇabhūtehi catūhi dhammehi

  * imehi = 「これらの」
  * kāraṇabhūtehi = 「原因（手段）となる」
  * catūhi dhammehi = 「四つの法によって」

→「これらの原因となる四つの法を用いて、」

9. dhammojapaññāya samannāgato medhāvī

  * dhamma-oja-paññāya = 「法の滋養（あるいは本質）を知る智慧を」
  * samannāgato = 「備えた」
  * medhāvī = 「賢者は」

→「法の滋養を知る智慧を備えた賢者は、」

10. imasmiṃ ativiya dullabhapatiṭṭhatāya atigambhīre saṃsārasāgare

  * imasmiṃ = 「この」
  * ativiya = 「極めて」
  * dullabha-patiṭṭhatāya = 「足場を得ることが困難であるため」
  * atigambhīre = 「非常に深い」
  * saṃsāra-sāgare = 「輪廻の大海において」

→「足場を得ることが極めて難しく、非常に深いこの輪廻の大海において、」

11. attano patiṭṭhānabhūtaṃ arahattaphalaṃ dīpaṃ kayirātha kareyya, kātuṃ sakkuṇeyyāti attho.

  * attano = 「自己の」
  * patiṭṭhāna-bhūtaṃ = 「よりどころ（足場）となる」
  * arahatta-phalaṃ = 「阿羅漢果という」
  * dīpaṃ = 「島を」
  * kayirātha = 「作るべきである（偈の語）」
  * kareyya = 「作るべきである（解説）」
  * kātuṃ sakkuṇeyya = 「作ることができるであろう」
  * iti attho = 「という意味である」

→「自分自身のよりどころとなる阿羅漢果という島を作るべきである、作ることができる、という意味である。」

12. Kīdisaṃ? Yaṃ ogho nābhikīratīti

  * kīdisaṃ = 「どのような（島か）？」
  * yaṃ ogho na abhikīrati iti = 「『それを洪水が押し流さない』とは」

→「どのような島か？ 『それを洪水が押し流さない』とは、」

13. yaṃ catubbidhopi kilesogho abhikirituṃ viddhaṃsetuṃ na sakkoti.

  * yaṃ = 「その（島）を」
  * catubbidho pi = 「四種（欲・有・見・無明）の～であっても」
  * kilesa-ogho = 「煩悩の洪水が」
  * abhikirituṃ = 「圧倒すること」
  * viddhaṃsetuṃ = 「破壊すること」
  * na sakkoti = 「できない」

→「四種の煩悩の洪水であっても、それを圧倒したり破壊したりすることができない（ということである）。」

14. Na hi sakkā arahattaṃ oghena abhikiritunti.

  * na hi sakkā = 「～は不可能であるからだ」
  * arahattaṃ = 「阿羅漢果を」
  * oghena = 「（煩悩の）洪水によって」
  * abhikirituṃ iti = 「圧倒することは」

→「なぜなら、阿羅漢果を洪水によって圧倒することは不可能だからである。」

## 意訳

奮起（努力）と不放逸（気づき）により、また自制と調御によって、
賢者は、洪水によって押し流されることのない島（避難所）を作るべきである。

**【註釈による解説】**
ここで「島を作るべし」とは、次のような意味である。
精進という名の「奮起」、常に気づきを保つことである「不放逸」、四種の清浄な戒めである「自制」、そして感覚器官を制御する「調御」。
これら四つの法を手段として、法の真髄を知る智慧を備えた賢者は、足場を見つけることが極めて難しく深遠なこの輪廻の大海原において、自分自身の確固たるよりどころとなる「阿羅漢果（悟りの境地）」という島を築くべきであり、また築くことができるのである。

それはどのような島か。「洪水が押し流さない」と言われる通り、四つの煩悩の洪水（欲・有・見・無明の激流）であっても、決して圧倒したり破壊したりすることのできない島である。なぜなら、阿羅漢果の境地を煩悩の洪水によって押し流すことは不可能だからである。

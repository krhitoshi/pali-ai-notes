# Gemini

## 対訳

### 偈文（Gāthā）

1. Akkocchi maṃ avadhi maṃ, ajini maṃ ahāsi me;

* akkocchi = 「（彼は私を）罵った」（akkosati のアオリスト）
* maṃ = 「私を」
* avadhi = 「（彼は私を）打った、傷つけた」（vadhati のアオリスト）
* ajini = 「（彼は私に）勝った、負かした」（jināti のアオリスト）
* ahāsi = 「（彼は私から）奪った」（harati のアオリスト）
* me = 「私から、私の（ものを）」

→ 「『（彼が）私を罵った、私を打った、私を負かした、私から奪った』」

2. Ye ca taṃ nupanayhanti, veraṃ tesūpasammatī’’ti.

* ye ca = 「また、～である人々は」
* taṃ = 「それ（その思い、その恨み）を」
* na upanayhanti = 「結びつけない、抱き続けない」
    * upa + nah (結ぶ) → 「恨みを結んで心に留め置くこと」を否定
* veraṃ = 「怨み、敵意」
* tesu = 「彼らにおいて、彼らの中では」
* upasammati = 「静まる、止滅する」

→ 「また、その（思い）を抱き続けない人々においては、怨みは静まる。」

### 註釈（Aṭṭhakathā）

3. Ye ca taṃ nupanayhantīti

→ 「『その（思い）を抱き続けない人々』というのは、以下の通りである。」

4. asatiyā amanasikāravasena vā kammapaccavekkhaṇādivasena vā

* asatiyā = 「念（sati）がないことによって（＝忘れることによって）」
* amanasikāra-vasena = 「作意（manasikāra）をしない力によって（＝気に留めないことによって）」
* vā ... vā = 「あるいは～、あるいは～」
* kamma-paccavekkhaṇa-ādi-vasena = 「業（kamma）を省察すること等の力によって」

→ 「（過ぎ去ったことへの）念がないためか、あるいは気に留めないことによってか、あるいは『業の省察』などを行うことによって、」

5. ye taṃ akkosādivatthukaṃ kodhaṃ

* ye = 「（～である）人々は」
* taṃ = 「その」
* akkosa-ādi-vatthukaṃ = 「罵りなどを事由（対象）とする」
* kodhaṃ = 「怒りを」

→ 「罵りなどを原因とするその怒りを（抱き続けない）。」
（※以下は「業の省察」の具体的な内容と思考過程）

6. tayāpi koci niddoso purimabhave akkuṭṭho bhavissati, pahaṭo bhavissati,

* tayā pi = 「あなたによってもまた」
* koci niddoso = 「ある罪なき（無実の）者が」
* purima-bhave = 「前世において」
* akkuṭṭho bhavissati = 「罵られたであろう」（未来形による推測）
* pahaṭo bhavissati = 「打たれたであろう」

→ 「『あなたによってもまた、前世において、ある罪なき者が罵られたり、打たれたりしたことであろう。』」

7. kūṭasakkhiṃ otāretvā jito bhavissati, kassaci te pasayha kiñci acchinnaṃ bhavissati,

* kūṭasakkhiṃ = 「偽証を」
* otāretvā = 「もたらして、立てて」
* jito bhavissati = 「（不当に）負かされたことであろう」
* kassaci = 「誰かの（ものを）」
* te = 「あなたによって」
* pasayha = 「力ずくで、無理やりに」
* kiñci acchinnaṃ bhavissati = 「何かが奪い取られたことであろう」

→ 「『（前世であなたが）偽証を立てて（誰かを）打ち負かしたことであろう。あなたによって誰かの何かが無理やりに奪われたことであろう。』」

8. tasmā niddoso hutvāpi akkosādīni pāpuṇāsīti evaṃ na upanayhanti.

* tasmā = 「それゆえに」
* niddoso hutvā pi = 「（今生では）罪なき者でありながらも」
* akkosādīni = 「罵りなどを」
* pāpuṇāsi = 「（あなたは）受けるのだ、達するのだ」
* iti evaṃ = 「～というように」
* na upanayhanti = 「（怒りを）結びつけない、恨みに思わない」

→ 「『それゆえに、（今生では）罪がないとしても、（過去の業の結果として）罵りなどを受けるのだ』と考えて、このように（怒りを）抱き続けないのである。」

9. Tesu pamādena uppannampi veraṃ

* tesu = 「彼ら（そのような省察をする人々）において」
* pamādena = 「怠慢（放逸）によって」
* uppannaṃ pi = 「生じたとしても」
* veraṃ = 「怨みは」

→ 「彼らにおいて、ふとした油断から怨みが生じたとしても、」

10. iminā anupanayhanena nirindhano viya jātavedo vūpasammatīti.

* iminā anupanayhanena = 「この『結びつけないこと（執着しないこと）』によって」
* nirindhano = 「燃料のない、薪のない」
* viya = 「～のように」
* jātavedo = 「火（jātaveda）」
* vūpasammati = 「静まる、消える」
* ti = 「（という意味である）」

→ 「この『（怒りを）結びつけないこと』によって、薪のない火のように、（怨みは）鎮火するのである。」

## 意訳

「『彼が私を罵った、私を打った、私を打ち負かした、私のものを奪った』。
人々が、そのような被害の思いを心に結びつけて抱き続けないならば、
その人々において、怨みは静まり消えていく。

（註釈）
『その思いを抱き続けない人々』とは、以下のいずれかの方法による人々のことである。
一つには、過ぎ去ったことを忘れることによって、あるいは意図的に気に留めないようにすることによって。
あるいはまた、『業（カルマ）の省察』を行うことによってである。

業の省察とは次のように考えることである。
『今生では自分に非がなくても、前世において、私は罪なき誰かを罵ったり、打ったりしたことがあるに違いない。偽証によって誰かを陥れたり、力ずくで誰かの物を奪ったりしたことがあるに違いない。それゆえに、過去の報いとして、今このような罵りなどを受けているのだ』と。

このように考えて、罵りなどをきっかけとした怒りを心に結びつけず、恨みに思わないのである。
そのような人々においては、たとえふとした油断から怨みが生じたとしても、この『恨みを結びつけないこと』によって、燃料の尽きた火のように、その怨みは速やかに鎮火するのである。」

## 意訳

「『あいつは私を罵った、私を打った、私に勝った、私の物を奪った』――
このように（過去の仕打ちを）思い続ける者もいる。
しかし、そのことを心の中で握りしめず、恨みとして抱きつづけない者たちには、
怨みは静まっていく。

ここで『恨みを抱き続けない者』とは、
うっかり忘れてしまうという意味だけでなく、
あえて心に取り上げないようにしたり、
「これは自分自身の過去の業の結果なのだ」と
自分の行為を省みて考える者たちのことである。

彼らは、たとえば次のように考える。
――自分も前世には、罪のない誰かを罵ったことがあり、
傷つけたこともあり、
偽証人を仕立て上げて裁きに勝ったこともあり、
誰かの物を力ずくで奪ったこともあったに違いない。
その結果として、今世の自分は、罪のない身でありながら、
罵倒などの仕打ちを受けているのだ――と。

このように考える者は、その出来事を恨みとして抱き続けない。
そのような人たちにおいては、
たとえ不注意から一時的に怨みが起こったとしても、
恨みを抱え込まずにいることによって、
燃料のない火が自然に消え去るように、
その怨みも静まり、消え去ってしまうのである。

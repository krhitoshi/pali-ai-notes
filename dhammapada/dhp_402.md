402.

‘‘Yo dukkhassa pajānāti, idheva khayamattano;

Pannabhāraṃ visaṃyuttaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha dukkhassāti khandhadukkhassa. Pannabhāranti ohitakhandhabhāraṃ catūhi yogehi sabbakilesehi vā visaṃyuttaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Yo dukkhassa pajānāti,

* yo = 「～である者は」「…なる人は」
* dukkhassa = 「苦の」「苦しみの」（属格）
* pajānāti = 「よく理解する」「正しく知る」

→「苦を、〔その本質を〕よく理解する者は」

2. idheva khayamattano;

* idha = 「ここで」「この世で」「現世で」
* eva = 「まさに」「こそ」
* khayaṃ = 「滅尽」「終滅」
* attano = 「自らの」「自分自身の」（属格）

→「この現世においてこそ、自らの〔苦の〕滅尽を〔理解する〕」

3. Pannabhāraṃ visaṃyuttaṃ,

* pannabhāraṃ = 「負担を下ろした者」
  * panna = 「置いた、下ろした」（paneti「置く・下ろす」の過去分詞）
  * bhāra = 「重荷、負担」
  →「（五蘊という）重荷を下ろした者」
* visaṃyuttaṃ = 「離れた」「解き放たれた」「解脱した」
  * vi- + saṃyutta = 「結びつきから離れた」「束縛から解き放たれた」

→「重荷を下ろし、束縛から解き放たれた者を」

4. tamahaṃ brūmi brāhmaṇa’’nti.

* taṃ = 「その者を」
* ahaṃ = 「私は」
* brūmi = 「言う」「呼ぶ」
* brāhmaṇaṃ = 「バラモンと」「聖なる人と」

→「その者を、私は『バラモン（真の聖者）』と呼ぶのである。」

---

5. Tattha dukkhassāti khandhadukkhassa.

* tattha = 「ここで」「この句においては」
* dukkhassāti = 「『dukkhassa』とは、の意で」
  * 「dukkhassa という語は」
* khandhadukkhassa = 「諸蘊の苦の」「五蘊としての苦の」
  * khandha = 「（色受想行識の）蘊、集まり」
  * dukkha = 「苦」
  →「五蘊としての苦、五蘊の性質としての苦」

→「ここで『dukkhassa』とは、五蘊としての苦を意味する。」

6. Pannabhāranti ohitakhandhabhāraṃ catūhi yogehi sabbakilesehi vā visaṃyuttaṃ

* pannabhāran-ti = 「『pannabhāraṃ』とは、の意で」
* ohita-khandha-bhāraṃ = 「下ろされた〔＝捨て去られた〕五蘊の重荷」
  * ohita = 「置かれた、下ろされた」（ava- + hita）
  * khandha = 「蘊」
  * bhāra = 「重荷」
→「五蘊という重荷を下ろした者であり」
* catūhi yogehi = 「四つの瑜伽（束縛）から」
  * catu = 「四つの」
  * yoga = 「結びつき、束縛（欲・有・見・無明の四有結）」
  * -hi = 具格「～によって、～から」
→「四つの束縛から」
* sabba-kilesehi vā = 「あるいは一切の煩悩から」
  * sabba = 「すべての」
  * kilesa = 「煩悩」
  * -hi = 具格「～から」
  * vā = 「または」
→「あるいは一切の煩悩から」
* visaṃyuttaṃ = 「離れた者」「解き放たれた者」

→「『pannabhāraṃ』とは、五蘊という重荷を下ろし、四つの束縛から、あるいは一切の煩悩から離れた者のことであり」

7. tamahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「その者を」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（聖者）と」
* vadāmi = 「言う」「呼ぶ」
* ti attho = 「という意味である」

→「その者を、私は『バラモン（真の聖者）』と呼ぶ、という意味である。」

---

## 意訳

「この身の五蘊としての『苦』をよく悟り、
この現世においてこそ、自らの苦の滅尽を明らかに知り、
五蘊という重荷を下ろし、
四つの有の束縛、あるいは一切の煩悩から解き放たれている人、
その人こそを、私は真の『バラモン（聖者）』と呼ぶのである。」

368.

‘‘Mettāvihārī yo bhikkhu, pasanno buddhasāsane;

Adhigacche padaṃ santaṃ, saṅkhārūpasamaṃ sukhaṃ.

Tattha mettāvihārīti mettākammaṭṭhāne kammaṃ karontopi mettāvasena tikacatukkajjhāne nibbattetvā ṭhitopi mettāvihārīyeva nāma. Pasannoti yo pana buddhasāsane pasanno hoti, pasādaṃ rocetiyevāti attho. Padaṃ santanti nibbānassetaṃ nāmaṃ. Evarūpo hi bhikkhu santaṃ koṭṭhāsaṃ sabbasaṅkhārānaṃ upasantatāya saṅkhārūpasamaṃ, paramasukhatāya sukhanti laddhanāmaṃ nibbānaṃ adhigacchati, vindatiyevāti attho.

## 対訳

1. Mettāvihārī yo bhikkhu, pasanno buddhasāsane;

* mettā‑vihārī = 「慈（mettā）を住処とする者」→「慈に住する者」「慈の心で住する者」
* yo ... bhikkhu = 「…であるところの比丘は」
  * yo = 「…である者」
  * bhikkhu = 「比丘」
* pasanno = 「信を起こした、歓喜・信頼に満ちた」
* buddha‑sāsane = 「ブッダの教え（教法・教えの制度）において」
  * buddha = 「ブッダの」
  * sāsana = 「教え、教法、教えの体系」

→「慈を住処とする比丘で、ブッダの教えに信を起こしている者は」

---

2. Adhigacche padaṃ santaṃ, saṅkhārūpasamaṃ sukhaṃ.

* adhigacche = 「到達するであろう、悟るであろう」（adhigacchati の願望法）
* padaṃ = 「境地・状態・究極のところ」
* santaṃ = 「静寂なる、寂静の」
* saṅkhāra‑upasamaṃ = 「諸行（saṅkhāra）の静まるところ／諸行の鎮静」
  * saṅkhāra = 「諸行、形成された一切のもの」
  * upasama = 「鎮静、しずまり」
* sukhaṃ = 「楽、安楽、最高の幸福」

→「諸行の静まる、静寂なる楽（境地）に到達するであろう。」

---

3. Tattha mettāvihārīti mettākammaṭṭhāne kammaṃ karontopi mettāvasena tikacatukkajjhāne nibbattetvā ṭhitopi mettāvihārīyeva nāma.

* tattha = 「そこで、この偈において」
* mettā‑vihārī‑ti = 「『mettāvihārī（慈を住処とする者）』とは」
* mettā‑kammaṭṭhāne = 「慈の業処において、慈の瞑想対象において」
  * mettā = 「慈（メッター）」
  * kammaṭṭhāna = 「業処、修行の対象・瞑想のテーマ」
* kammaṃ karontopi = 「（その）業をなしている場合でも」→「その瞑想を実践していても」
  * kammaṃ karonta = 「業（実践・作業）を行っている」
  * pi = 「…であっても」
* mettā‑vasena = 「慈によって、慈という仕方で」
* tika‑catukka‑jjhāne = 「三種および四種の禅定（に分別される禅定）に」
  * tika = 「三種の（tika＝三分類）」
  * catukka = 「四種の（四分類）」
  * jhāna = 「禅定」
* nibbattetvā = 「（それらの禅定を）生起させて、成就させて」
* ṭhito‑pi = 「その境地に立っていても、そこにとどまっていても」
  * ṭhita = 「安住している、立っている」
  * pi = 「…であっても」
* mettā‑vihārī‑eva nāma = 「まさしく『慈に住する者』と呼ばれるのである」
  * eva = 「まさしく」
  * nāma = 「…と名づけられる」

→「ここで『慈に住する者』とは、慈の業処（慈の瞑想対象）において修行している場合でも、慈によって三種・四種の禅定を生起させてその境地に安住している場合でも、まさしく『慈に住する者』と名づけられる、という意味である。」

---

4. Pasannoti yo pana buddhasāsane pasanno hoti, pasādaṃ rocetiyevāti attho.

* pasanno‑ti = 「『pasanno（信を起こした者）』とは」
* yo pana ... pasanno hoti = 「さて、…である者が信を起こしている、ということである」
  * yo = 「…である者」
  * pana = 「さて、ところで」
  * pasanno hoti = 「信を起こしている、信じている・歓喜している」
* buddha‑sāsane = 「ブッダの教えにおいて」
* pasādaṃ roceti‑eva‑ti attho = 「まさしく信心を喜ぶ（＝信心を生起させる）という意味である」
  * pasāda = 「信心、清浄な信」
  * roceti = 「喜ぶ・好む・生起させる」
  * eva = 「まさしく」
  * ti attho = 「という意味である」

→「『信を起こした者』とは、ブッダの教えに対して信を起こし、まさしく信心を喜び起こす者を言う、という意味である。」

---

5. Padaṃ santanti nibbānassetaṃ nāmaṃ.

* padaṃ santaṃ‑ti = 「『静寂なる境地』とは」
* nibbānassa‑etaṃ nāmaṃ = 「これは涅槃の名称である」
  * nibbāna = 「涅槃」
  * etaṃ nāmaṃ = 「この名である、この呼び名である」

→「『静寂なる境地』とは、涅槃の名称である。」

---

6. Evarūpo hi bhikkhu santaṃ koṭṭhāsaṃ sabbasaṅkhārānaṃ upasantatāya saṅkhārūpasamaṃ, paramasukhatāya sukhanti laddhanāmaṃ nibbānaṃ adhigacchati, vindatiyevāti attho.

* evarūpo hi bhikkhu = 「このような（前述のような）比丘は」
  * evarūpa = 「このような」
  * hi = 「まことに、というのも」
  * bhikkhu = 「比丘」
* santaṃ koṭṭhāsaṃ sabba‑saṅkhārānaṃ = 「一切の諸行の静まった側面（部分）を」
  * santaṃ = 「静まった、寂静である」
  * koṭṭhāsa = 「部分・側面」
  * sabba‑saṅkhāra = 「一切の諸行」
* upasantatāya = 「（それら諸行が）静まっていることによって」
  * upasantatā = 「静まっている状態、鎮静」
* saṅkhāra‑upasamaṃ = 「諸行の鎮静たるもの（＝諸行の静まりであるところの涅槃）」
* paramasukhatāya = 「最高の楽であることによって、最上の幸福であるという性質ゆえに」
  * parama‑sukhatā = 「最高楽であること」
* sukha‑nti laddha‑nāmaṃ = 「『楽』という名を得た」
  * sukha = 「楽・安楽・幸福」
  * laddha‑nāma = 「…という名を得た、そう呼ばれる」
* nibbānaṃ = 「涅槃を」
* adhigacchati = 「到達する、悟る」
* vindati‑eva‑ti attho = 「まさしく獲得するのである、という意味である」
  * vindati = 「獲得する、得る」
  * eva = 「まさしく」
  * ti attho = 「という意味である」

→「というのも、このような比丘は、一切の諸行が静まっているゆえに『諸行の鎮静』といわれ、また最高の楽であるがゆえに『楽』という名を得た涅槃という、諸行の静まった側面を、到達し、まさしく獲得するのである、という意味である。」

---

## 意訳

慈の瞑想に住し、仏陀の教えを深く信じる比丘は、  
諸行のいっさいが静まりきった、静寂の安楽――涅槃――という境地に到達する。

ここで「慈に住する者」とは、慈の瞑想対象を修しつつある者であっても、  
慈の修習によって三種・四種に分類される諸禅定を成就し、それに安住している者であっても、  
いずれも「慈に住する者」と呼ばれることである。

また「信を起こした者」とは、仏陀の教えに対して信心をおこし、  
その信をよろこび育てている者をいう。

「静寂なる境地」とは、涅槃の名であり、  
このような比丘は、一切の諸行が静まりきっているという意味で「諸行の鎮静」と呼ばれ、  
また最高無上の楽であるという意味で「楽」と名づけられる  
その涅槃という境地を、実際に到達し、獲得するのである。

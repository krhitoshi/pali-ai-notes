386.

‘‘Jhāyiṃ virajamāsīnaṃ, katakiccamanāsavaṃ;

Uttamatthamanuppattaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha jhāyinti duvidhena jhānena jhāyantaṃ kāmarajena virajaṃ vane ekakamāsīnaṃ catūhi maggehi soḷasannaṃ kiccānaṃ katattā katakiccaṃ āsavānaṃ abhāvena anāsavaṃ uttamatthaṃ arahattaṃ anuppattaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Jhāyiṃ virajamāsīnaṃ, katakiccamanāsavaṃ;

* jhāyiṃ = 「禅定にふけっている者を」
  * jhāyin（禅に入る者）の対格形
* virajaṃ = 「汚れ（煩悩の塵）のない」
  * vi + rajas（塵・汚れ）→「無垢の、離欲の」
* āsīnaṃ = 「坐っている者を」
  * āsīna = 「坐している、着座している」
* katakiccaṃ = 「なすべきことをすでに成し終えた者を」
  * kata = 「なされた」
  * kicca = 「なすべきこと、務め」
* anāsavaṃ = 「漏（āsava）のない者を」
  * an-āsava = 「煩悩漏のない、煩悩を断じ尽くした」

→「禅定にふけり、垢なきまま坐し、なすべきことを成し終え、煩悩漏をもたぬ者を」

2. Uttamatthamanuppattaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* uttamatthaṃ = 「最上の目的を」
  * uttama = 「最上の、最高の」
  * attha = 「目的・利益・意味」ここでは「究竟の目的」
* anuppattaṃ = 「到達した者を」
  * anupapta/anuppatta（anu + √pat）= 「到達した、得た」
* taṃ = 「その者を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ、言う」
* brāhmaṇaṃ = 「バラモンであると言う」
  * ここでは「真のバラモン・聖者」

→「最上の目的に到達したその者を、私は『バラモン』と呼ぶのである。」

---

3. Tattha jhāyinti duvidhena jhānena jhāyantaṃ

* tattha = 「ここで、この偈（の語句）の中で」
* jhāyin ti = 「『jhāyin（禅定にふける者）』とは」
* duvidhena = 「二種のしかたの（双の）」
  * du-vidha = 「二種の」
* jhānena = 「禅定によって」
  * ここは具格で「～という二種の禅定をもって」
  * 通例：世俗的禅（samatha-jhāna）と観禅（vipassanā-jhāna）などを指す
* jhāyantaṃ = 「禅定にふけっている者」
  * jhāyati（禅定に入る）の現在分詞

→「ここで『禅定にふける者（jhāyin）』とは、二種の禅定によって禅にふける者のことである。」

4. kāmarajena virajaṃ vane ekakamāsīnaṃ

* kāma-rajena = 「欲の塵によって（＝貪欲という）垢から」
  * kāma = 「感官的欲、欲楽」
  * raja = 「塵、汚れ」→「欲という汚れ」
  * -ena = 「～によって、～に関して」→「欲の汚れから」
* virajaṃ = 「離れた者、汚れのない者」
* vane = 「林の中で、森の中で」
* ekakaṃ = 「独りで、一人きりで」
* āsīnaṃ = 「坐している者を」

→「欲の塵から離れ、林の中で独り坐している者のことであり」

5. catūhi maggehi soḷasannaṃ kiccānaṃ katattā katakiccaṃ

* catūhi maggehi = 「四つの道によって」
  * cattāro maggā = 四聖道（預流・一来・不還・阿羅漢道）
  * 具格複数「四つの聖道をもって、四道によることによって」
* soḷasannaṃ kiccānaṃ = 「十六の任務（なすべき働き）の」
  * soḷasa = 「十六」
  * kicca = 「なすべきこと、任務、役割」
  * 十六の任務＝四聖諦それぞれに四つの「なすべき作用」（知る・捨てる・証する・修習する）を配した十六行相
* katattā = 「なしたがゆえに、成就したがゆえに」
* katakiccaṃ = 「なすべきことを終えた者であり」
  * kata + kicca = 「すべきことがやり遂げられた」

→「四つの聖道によって十六種の任務をやり遂げたがゆえに、『なすべきことを終えた者』であり」

6. āsavānaṃ abhāvena anāsavaṃ

* āsavānaṃ = 「漏（āsava）たちの」
  * āsava = 「漏、煩悩漏：欲漏、有漏、見漏、無明漏など」
* abhāvena = 「存在しないことによって、無いがゆえに」
* an-āsavaṃ = 「漏のない者」
  * 「一切の漏を滅した者、煩悩漏のない者」

→「諸々の漏が存在しないがゆえに、『漏のない者』であり」

7. uttamatthaṃ arahattaṃ anuppattaṃ

* uttama-atthaṃ = 「最上の目的を」
  * ここでは涅槃そのものとしての「阿羅漢果」を指す
* arahattaṃ = 「阿羅漢の位、阿羅漢果」
* anuppattaṃ = 「到達した者」

→「最上の目的である阿羅漢果に到達した者であり」

8. ahaṃ brāhmaṇaṃ vadāmīti attho.

* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（と呼ばれる者）を」
  * ここでは血統上のバラモンではなく、「煩悩を焼き尽くした真のバラモン＝聖者」
* vadāmi = 「と言う、呼ぶ」
* ti attho = 「という意味である」

→「そのような者をこそ、私は『バラモン』と呼ぶのである、という意味である。」

---

## 意訳

「禅定にふけり、欲の汚れから離れ、林の中でひとり静かに坐し、  
四つの聖なる道によって四聖諦に関わる十六のなすべきことをすべて成し遂げ、  
なすべきことをやり終え、あらゆる煩悩漏を断ち、  
最高の目的たる阿羅漢の境地に到達した者――  
その者をこそ、私は真の『バラモン』と呼ぶのである。」

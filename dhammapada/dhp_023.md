## 対訳

### 偈文（Gāthā）

1. ‘‘Te jhāyino sātatikā, niccaṃ daḷhaparakkamā;

* Te = 「彼ら（前偈で説かれた不放逸な人々）は」
* jhāyino = 「禅定を修する者であり」
* sātatikā = 「根気強く（絶えず励み）」
* niccaṃ = 「常に」
* daḷha-parakkamā = 「堅固な勇猛（努力）をなす者である」

→「彼らは禅定を修し、根気強く、常に堅固な努力をなす者である。」

2. Phusanti dhīrā nibbānaṃ, yogakkhemaṃ anuttara’’nti.

* phusanti = 「（彼らは）触れる、達する」
* dhīrā = 「賢者たちは」
* nibbānaṃ = 「涅槃（ニッバーナ）に」
* yoga-kkhemaṃ = 「軛（くびき・束縛）からの安穏に」
* anuttaraṃ = 「この上なき（無上の）」
* ti = （引用の終わり）

→「賢者たちは、軛からの安穏である無上のニッバーナに達する（触れる）。」

### 註釈（Aṭṭhakathā）

3. Tejhāyinoti te appamattā paṇḍitā

* Te jhāyino ti = 「『Te jhāyino（彼らは禅定者）』とは」
* te appamattā paṇḍitā = 「彼ら、不放逸なる賢者たちは」

→「『Te jhāyino』とは、彼ら不放逸なる賢者たちは、という意味である。」

4. aṭṭhasamāpattisaṅkhātena ārammaṇūpanijjhānena

* aṭṭha-samāpatti-saṅkhātena = 「八等至（八種の定）と呼ばれる」
* ārammaṇa-upanijjhānena = 「所縁（対象）を凝視する禅定（所縁禅）と」

→「八等至と呼ばれる所縁を凝視する禅定と、」

5. vipassanāmaggaphalasaṅkhātena lakkhaṇūpanijjhānena cāti duvidhenapi jhānena jhāyino.

* vipassanā-magga-phala-saṅkhātena = 「ヴィパッサナー（観）・道・果と呼ばれる」
* lakkhaṇa-upanijjhānena ca = 「（無常などの）相を凝視する禅定（相禅）と」
* iti duvidhena pi jhānena = 「という二種の禅定によって」
* jhāyino = 「禅定者（禅定を修する者）なのである」

→「ヴィパッサナー・道・果と呼ばれる相を凝視する禅定という、これら二種の禅定によって『禅定者』なのである。」

6. Sātatikāti abhinikkhamanakālato paṭṭhāya yāva arahattamaggā satataṃ pavattakāyikacetasikavīriyā.

* Sātatikā ti = 「『Sātatikā（根気強い者）』とは」
* abhinikkhamana-kālato paṭṭhāya = 「（世俗を離れて）出家した時から始まって」
* yāva arahatta-maggā = 「阿羅漢道に至るまで」
* satataṃ pavatta- = 「絶えず（隙間なく）生起させている」
* kāyika-cetasika-vīriyā = 「身体的および精神的な精進を持つ者である」

→「『Sātatikā』とは、出家の時から阿羅漢道に至るまで、絶えず心身の精進を働かせている者、という意味である。」

7. Niccaṃ daḷhaparakkamāti yaṃ taṃ purisathāmena purisavīriyena purisaparakkamena pattabbaṃ,

* Niccaṃ daḷha-parakkamā ti = 「『Niccaṃ daḷhaparakkamā（常に堅固な勇猛ある者）』とは」
* yaṃ taṃ = 「いかなるものであれ」
* purisa-thāmena = 「人の力によって」
* purisa-vīriyena = 「人の精進によって」
* purisa-parakkamena = 「人の勇猛によって」
* pattabbaṃ = 「到達されるべきもの（があるならば）」

→「『Niccaṃ daḷhaparakkamā』とは、人の力、人の精進、人の勇猛によって到達すべきものがあるならば、」

8. na taṃ apāpuṇitvā vīriyassa saṇṭhānaṃ bhavissatīti

* na taṃ apāpuṇitvā = 「それに到達せずして～することはない」
* vīriyassa saṇṭhānaṃ = 「精進の停止（休息）が」
* bhavissatī ti = 「あるだろう（＝停止することはない）」

→「それに到達せずして精進が止まることはないであろう、と（決意し）」

9. evarūpena vīriyena antarā anosakkitvā niccappavattena daḷhaparakkamena samannāgatā.

* evarūpena vīriyena = 「このような（質の）精進によって」
* antarā anosakkitvā = 「途中で退くことなく」
* nicca-ppavattena = 「常に行われる（継続した）」
* daḷha-parakkamena = 「堅固な勇猛を」
* samannāgatā = 「具えている（者たちである）」

→「このような精進をもって、途中で退くことなく、常に行われる堅固な勇猛さを具えている、という意味である。」

10. Phusantīti ettha dve phusanā ñāṇaphusanā ca, vipākaphusanā ca.

* Phusanti ti = 「『Phusanti（触れる）』とは」
* ettha dve phusanā = 「ここには二種の『触（到達）』がある」
* ñāṇa-phusanā ca = 「智による触と」
* vipāka-phusanā ca = 「果報（異熟）による触である」

→「『Phusanti』について。ここには『智による触』と『果報による触』の二種の触がある。」

11. Tattha cattāro maggā ñāṇaphusanā nāma, cattāri phalāni vipākaphusanā nāma.

* Tattha = 「その中で」
* cattāro maggā = 「四つの道（聖道）は」
* ñāṇa-phusanā nāma = 「『智による触』と呼ばれる」
* cattāri phalāni = 「四つの果（聖果）は」
* vipāka-phusanā nāma = 「『果報による触』と呼ばれる」

→「その中で、四つの道は『智による触』、四つの果は『果報による触』と呼ばれる。」

12. Tesu idha vipākaphusanā adhippetā.

* Tesu = 「それら（二つ）のうち」
* idha = 「ここでは」
* vipāka-phusanā adhippetā = 「『果報による触』が意図されている」

→「ここでは『果報による触』が意図されている。」

13. Ariyaphalena nibbānaṃ sacchikaronto dhīrā paṇḍitā

* ariya-phalena = 「聖なる果によって」
* nibbānaṃ sacchikaronto = 「ニッバーナを現証（直感）しつつ」
* dhīrā paṇḍitā = 「賢者たちは」

→「聖なる果によってニッバーナを現証しつつ、賢者たちは、」

14. tāya vipākaphusanāya phusanti, nibbānaṃ sacchikaronti.

* tāya vipāka-phusanāya = 「その『果報による触』によって」
* phusanti = 「（ニッバーナに）触れる」
* nibbānaṃ sacchikaronti = 「すなわちニッバーナを現証するのである」

→「その『果報による触』によって触れる、つまりニッバーナを現証するのである。」

15. Yogakkhemaṃ anuttaranti ye cattāro yogā mahājanaṃ vaṭṭe osīdāpenti,

* Yogakkhemaṃ anuttaraṃ ti = 「『Yogakkhemaṃ anuttaraṃ（無上の安穏）』とは」
* ye cattāro yogā = 「四つの軛（ヨーガ）があり」
* mahājanaṃ = 「多くの人々を」
* vaṭṭe = 「輪廻の中に」
* osīdāpenti = 「沈没させる（沈める）」

→「『Yogakkhemaṃ anuttaraṃ』について。多くの人々を輪廻の中に沈める四つの軛（ヨーガ）があるが、」

16. tehi khemaṃ nibbhayaṃ sabbehi lokiyalokuttaradhammehi seṭṭhattā anuttaranti.

* tehi = 「それら（四つの軛）から」
* khemaṃ = 「安穏であり」
* nibbhayaṃ = 「恐怖がない（のがニッバーナである）」
* sabbehi lokiya-lokuttara-dhammehi = 「全ての世間・出世間の法の中で」
* seṭṭhattā = 「最も優れているため」
* anuttaraṃ ti = 「『anuttara（無上の）』と言うのである」

→「それらから安穏であり、恐怖がないこと、また、全ての世間的・出世間的な法の中で最も優れているために『無上』と言うのである。」

## 意訳

不放逸な賢者たちは、対象に集中する禅定（八等至）と、真理の相を観察する禅定（ヴィパッサナー・道・果）の両方を修める瞑想者である。
彼らは出家してから阿羅漢の境地に達するまで、心身の精進を絶やすことなく根気強く励む。
「人の力で到達できる場所ならば、たどり着くまでは決して諦めない」という、退くことのない堅固な努力を常に続けている。

彼ら賢者は、このようにして「無上の安穏」であるニッバーナに到達する。
それは、人々を輪廻に沈める四つの束縛（軛）から離れた安らぎの境地であり、あらゆる世間・出世間の法の中で最も優れたものである。
彼らは聖なる果を得ることによって、そのニッバーナを直接体験（現証）するのである。

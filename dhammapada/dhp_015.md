
15.

‘‘Idha socati pecca socati,

Pāpakārī ubhayattha socati;

So socati so vihaññati;

Disvā kammakiliṭṭhamattano’’ti.

Tattha pāpakārīti nānappakārassa pāpakammassa kārako puggalo ‘‘akataṃ vata me kalyāṇaṃ, kataṃ pāpa’’nti ekaṃseneva maraṇasamaye idha socati, idamassa kammasocanaṃ. Vipākaṃ anubhavanto pana pecca socati. Idamassa paraloke vipākasocanaṃ. Evaṃ so ubhayattha socatiyeva. Teneva kāraṇena jīvamānoyeva so cundasūkarikopi socati. Disvā kammakiliṭṭhanti attano kiliṭṭhakammaṃ passitvā nānappakārakaṃ vilapanto vihaññatīti.

## 対訳

1. Idha socati pecca socati,

* idha = 「ここで、現世で」
* socati = 「憂える、悲しむ」
* pecca = 「死んでから、来世で」
* socati = 「憂える、悲しむ」
→ 「現世で憂え、来世で憂える。」

2. Pāpakārī ubhayattha socati;

* pāpa-kārī = 「悪を行う者は」
* ubhayattha = 「両方で（現世と来世で）」
* socati = 「憂える」
→ 「悪を行う者は、両方で憂える。」

3. So socati so vihaññati;

* so = 「彼は」
* socati = 「憂える」
* so = 「彼は」
* vihaññati = 「苦しむ、悩む」
→ 「彼は憂え、彼は苦しむ。」

4. Disvā kammakiliṭṭhamattano''ti.

* disvā = 「見て」
* kamma-kiliṭṭhaṃ = 「汚れた行為（悪業）を」
* attano = 「自分の」
* ti = 引用詞
→ 「自分の汚れた行いを見て（彼は憂え、苦しむのである）。」

5. Tattha pāpakārīti nānappakārassa pāpakammassa kārako puggalo

* tattha = 「そこに（その偈において）」
* pāpakārī ti = 「『悪を行う者』とは」
* nānappakārassa = 「様々な種類の」
* pāpakammassa = 「悪業の」
* kārako = 「実行者である」
* puggalo = 「人が」
→ 「その偈において、『悪を行う者』とは、様々な種類の悪業を行う人が」

6. ‘‘akataṃ vata me kalyāṇaṃ, kataṃ pāpa’’nti

* akataṃ = 「なされなかった」
* vata = 「実に」
* me = 「私によって」
* kalyāṇaṃ = 「善が」
* kataṃ = 「なされた」
* pāpaṃ = 「悪が」
* ti = 引用詞
→ 「『実に私は善を行わず、悪を行った』と（思って）」

7. ekaṃseneva maraṇasamaye idha socati, idamassa kammasocanaṃ.

* ekaṃsena eva = 「確かに、まさしく」
* maraṇa-samaye = 「死の時に」
* idha = 「ここで、現世で」
* socati = 「憂える」
* idaṃ = 「これが」
* assa = 「彼の」
* kamma-socanaṃ = 「業による憂いである」
→ 「死の時に、現世においてまさしく憂える。これが彼の業による憂いである。」

8. Vipākaṃ anubhavanto pana pecca socati. Idamassa paraloke vipākasocanaṃ.

* vipākaṃ = 「果報を」
* anubhavanto = 「経験しながら、受けながら」
* pana = 「そして、また」
* pecca = 「死んでから、来世で」
* socati = 「憂える」
* idaṃ = 「これが」
* assa = 「彼の」
* paraloke = 「来世における」
* vipāka-socanaṃ = 「果報による憂いである」
→ 「また、果報を受けながら、来世で憂える。これが彼の来世における果報による憂いである。」

9. Evaṃ so ubhayattha socatiyeva. Teneva kāraṇena jīvamānoyeva so cundasūkarikopi socati.

* evaṃ = 「このようにして」
* so = 「彼は」
* ubhayattha = 「両方で」
* socati yeva = 「憂えるのである」
* tena eva = 「まさしくその」
* kāraṇena = 「理由で」
* jīvamāno eva = 「生きながらにして」
* so = 「その」
* cunda-sūkariko pi = 「豚殺しのチュンダも」
* socati = 「憂える（憂えた）」
→ 「このようにして、彼は両方で憂えるのである。まさしくその理由で、豚殺しのチュンダも生きながらにして憂えたのである。」

10. Disvā kammakiliṭṭhanti attano kiliṭṭhakammaṃ passitvā nānappakārakaṃ vilapanto vihaññatīti.

* disvā kammakiliṭṭhan ti = 「『汚れた行いを見て』とは」
* attano = 「自分の」
* kiliṭṭha-kammaṃ = 「汚れた行い（悪業）を」
* passitvā = 「見て」
* nānappakārakaṃ = 「様々に」
* vilapanto = 「嘆き悲しみながら」
* vihaññati = 「苦しむ（悩む）」
* ti = 引用詞
→ 「『汚れた行いを見て』とは、自分の汚れた行いを見て、様々に嘆き悲しみながら苦しむ、という意味である。」

## 意訳

現世で憂え、来世でも憂える。
悪を行う者は、その両方で憂える。
自分の汚れた行いを見て、
彼は憂え、そして苦しむのである。

註釈：
その偈において「悪を行う者」とは、様々な種類の悪業を行った人が「実に私は善きことを行わず、悪きことを行った」と思い、死の時に現世においてまさしく憂えることである。これが彼の業（行い自体）による憂いである。また、果報を受けながら来世でも憂える。これが彼の来世における果報による憂いである。
このようにして、彼は両方で憂えるのである。まさしくその理由から、豚殺しのチュンダも生きながらにして憂え苦しんだのである。
「汚れた行いを見て」とは、自分の汚れた行い（悪業）を見て、様々に嘆き悲しみながら苦しむ、という意味である。

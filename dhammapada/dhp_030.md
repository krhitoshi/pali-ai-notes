‘‘Appamādena maghavā, devānaṃ seṭṭhataṃ gato;

## 対訳
1. Appamādena maghavā, devānaṃ seṭṭhataṃ gato;

* appamādena = 「不放逸によって（appamāda の具格）」
* maghavā = 「マガヴァー（帝釈天）は」
* devānaṃ = 「神々の（属格）」
* seṭṭhataṃ = 「最上であること、最上の地位に（seṭṭhatā の対格）」
* gato = 「至った（gam の過去分詞）」

→ 「不放逸によって、マガヴァーは神々の最上の地位に至った。」

2. Appamādaṃ pasaṃsanti, pamādo garahito sadā''ti.

* appamādaṃ = 「不放逸を（対格）」
* pasaṃsanti = 「称賛する（pasaṃsati の三人称複数）」

* garahito = 「非難される（garahati の過去分詞）」
* sadā = 「常に」
* ti = 「～と（引用を示す不変化詞）」

→ 「（賢者たちは）不放逸を称賛する。放逸は常に非難される。」

3. Tattha appamādenāti macalagāme bhūmippadesasodhanaṃ ādiṃ katvā katena appamādena.

* tattha = 「そこにおいて（この偈において）」
* appamādenā ti = 「『appamādena（不放逸によって）』とは」
* macalagāme = 「マチャラ村において（処格）」
* bhūmi-ppadesa-sodhanaṃ = 「土地の清掃を」
* ādiṃ katvā = 「～を初めとして」
* katena = 「なされた（kata の具格）」
* appamādena = 「不放逸によって」

→ 「ここでの『不放逸によって』とは、マチャラ村において土地の清掃などを初めとしてなされた不放逸によって（という意味である）。」

4. Maghavāti idāni ‘‘maghavā’’tipaññāto magho māṇavo dvinnaṃ devalokānaṃ rājabhāvena devānaṃ seṭṭhataṃ gato.

* "maghavā" ti paññāto = 「『マガヴァー』として知られている」
* magho māṇavo = 「マガという若者が」
* dvinnaṃ = 「二つの」
* deva-lokānaṃ = 「天界の（属格）」
* rāja-bhāvena = 「王であることによって、王の地位によって（具格）」
* devānaṃ = 「神々の」

* gato = 「至った」

→ 「『マガヴァーは』とは、現在『マガヴァー』として知られているマガという若者が、二つの天界の王となることで、神々の最上の地位に至った（ということである）。」

5. Pasaṃsantīti buddhādayo paṇḍitā appamādameva thomenti vaṇṇayanti.

* Pasaṃsantī ti = 「『pasaṃsanti（称賛する）』とは」
* buddha-ādayo = 「仏陀をはじめとする」
* paṇḍitā = 「賢者たちが」
* appamādaṃ eva = 「不放逸こそを」

→ 「『称賛する』とは、仏陀をはじめとする賢者たちが、不放逸こそを称え、賞賛する（という意味である）。」

6. Kiṃ kāraṇā? Sabbesaṃ lokiyalokuttarānaṃ visesānaṃ paṭilābhakāraṇattā.

* Kiṃ kāraṇā? = 「何が理由か？（なぜか？）」
* sabbesaṃ = 「すべての」

→ 「何が理由か？ すべての世間的および出世間的な優れた境地を獲得する原因であるからである。」

7. Pamādo garahito sadāti pamādo pana tehi ariyehi niccaṃ garahito nindito.

→ 「何が理由か？ すべての破滅の根本であるからである。」

9. Manussadobhaggaṃ vā hi apāyuppatti vā sabbā pamādamūlikāyevāti.

* Manussa-dobhaggaṃ = 「人間としての不幸（不運）」

* hi = 「実に」

→ 「実に、人間としての不幸も、悪趣に生まれることも、すべては放逸を根本としているからである。」

## 意訳

【偈文】
不放逸によって、マガヴァー（帝釈天）は神々の最上の地位に至った。
（賢者たちは）不放逸を称賛する。放逸は常に非難される。

【註釈】
ここで「不放逸によって」とは、マチャラ村において土地の清掃などを初めとしてなされた不放逸によって、という意味である。
「マガヴァーは」とは、現在「マガヴァー」として知られているマガという若者が、二つの天界の王の地位に就くことで、神々の最上の地位に至ったということである。
「称賛する」とは、仏陀をはじめとする賢者たちが、不放逸こそを称え、賞賛するということである。
なぜか？ それは、不放逸がすべての世間的・出世間的な優れた境地を獲得する原因であるからである。
「放逸は常に非難される」とは、一方、放逸はそれらの聖者たちによって常に非難され、軽蔑されるという意味である。
なぜか？ それは、放逸がすべての破滅の根本であるからである。実に、人間としての不幸も、悪趣（苦しみの世界）に生まれることも、すべては放逸を根本としているからである。

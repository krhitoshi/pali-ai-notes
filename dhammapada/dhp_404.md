404.

‘‘Asaṃsaṭṭhaṃ gahaṭṭhehi, anāgārehi cūbhayaṃ;

Anokasārimappicchaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha asaṃsaṭṭhanti dassanasavanasamullapanaparibhogakāyasaṃsaggānaṃ abhāvena asaṃsaṭṭhaṃ. Ubhayanti gihīhi ca anāgārehi cāti ubhayehipi asaṃsaṭṭhaṃ . Anokasārinti anālayacāriṃ taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Asaṃsaṭṭhaṃ gahaṭṭhehi, anāgārehi cūbhayaṃ;

* asaṃsaṭṭhaṃ = 「交わっていない・混じり合っていない者を」  
  * saṃsaṭṭha（saṃ + sajjati の過去分詞）=「親しく交わる・混じる」→ a- + saṃsaṭṭha = 「交わらない」
* gahaṭṭhehi = 「在家の人々と」  
  * gahaṭṭha = 「家を持つ者＝在家者・俗人」
  * -ehi（具格複数）=「～と共に、～と」
* anāgārehi = 「出家者たちと」  
  * anāgāra = 「家なき者＝出家者」
  * -ehi（具格複数）
* ca = 「そして」
* ubhayaṃ = 「その両方と（に対しても）」  
  * ubhaya = 「両者・両方」

→「在家の人々とも、出家者たちとも、その両方と交わらない者を」

---

2. Anokasārimappicchaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* anokasāriṃ = 「住処を本拠とせずに歩む者を」  
  * an-oka-sārin  
    * oka = 「家・住処・依り所」
    * anoka = 「家（住処）なき、住処を自分の本拠としない」
    * sārin（＝cārin）=「行く者・歩む者・暮らす者」
  →「住処を我がものとして頼みとせずに行動する者」
* appicchaṃ = 「欲望の少ない者を」  
  * appa = 「少ない」
  * icchā = 「欲望、望み」
  * appiccha = 「欲が少ない、少欲の」
* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ・言う」
* brāhmaṇaṃ = 「バラモン（真の聖者）と」

→「住処を本拠とせずに行動し、欲の少ないそのような者を、私は『バラモン（聖者）』と呼ぶのである。」

---

3. Tattha asaṃsaṭṭhanti dassanasavanasamullapanaparibhogakāyasaṃsaggānaṃ abhāvena asaṃsaṭṭhaṃ.

* tattha = 「そこで、その句においては」
* asaṃsaṭṭhan ti = 「『asaṃsaṭṭhaṃ（交わらない）』とは、…という意味である」
* dassana-sa­vana-samullāpana-paribhoga-kāya-saṃsaggānaṃ =  
  * dassana-saṃsagga = 「見ることによる交わり」
  * savana-saṃsagga = 「聞くことによる交わり」
  * samullāpana-saṃsagga = 「会話・語り合いによる交わり」
  * paribhoga-saṃsagga = 「共に用いること（享受）による交わり」
  * kāya-saṃsagga = 「身体的接触による交わり」
  * … の saṃsagga たち（複数属格）=「それらの交わり」
* abhāvena = 「（それら）が無いことによって・無いがゆえに」
* asaṃsaṭṭhaṃ = 「交わらない（者である）」

→「ここで『交わらない』とは、見る・聞く・語り合う・共に用いる・身体的接触を通じての種々の交わりが無いがゆえに、（他者と）交わらない者である、という意味である。」

---

4. Ubhayanti gihīhi ca anāgārehi cāti ubhayehipi asaṃsaṭṭhaṃ.

* ubhayan ti = 「『ubhayaṃ（両者）』とは」
* gihīhi = 「在家者たちと」  
  * gihi（= gahaṭṭha）=「在家の人」
* ca = 「および」
* anāgārehi = 「出家者たちと」
* cā ti = 「～ということで」
* ubhayehi pi = 「その両方とさえ（いずれとも）」  
  * -hi（具格複数）
  * pi = 「さえも」
* asaṃsaṭṭhaṃ = 「交わらない（者である）」

→「『両者』とは在家者と出家者のことであり、そのいずれとも交わらない者である、という意味である。」

---

5. Anokasārinti anālayacāriṃ taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* anokasārin ti = 「『anokasāriṃ』とは」
* anālaya-cāriṃ = 「執着なき所を行ずる者、住処に執着せずに行動する者を」  
  * ālaya = 「寄りつく所・住処・依りつき・執着」
  * anālaya = 「執着なき、寄りつかない」
  * cārin = 「行動する者・暮らす者」
→「住処・依り所に執着せずに行動する者」
* taṃ = 「そのような者を」
* evarūpaṃ = 「かくのごとき者を」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（聖者）と」
* vadāmi ti = 「呼ぶのである、と」
* attho = 「いう意味である」

→「『anokasāriṃ』とは、住処・依り所に執着せずに行動する者をいう。私はそのような者をバラモン（聖者）と呼ぶ、という意味である。」

---

## 意訳

在家の人々とも、出家者たちとも、どちらとも親しく交わることなく、  
住処を自分の拠り所とせずにさまよう者で、  
欲の少ない人――そのような人を、私は真のバラモン（聖者）と呼ぶ。

ここで「交わらない」とは、見る・聞く・語り合う・物を共に用いる・身体的に接触するといった形での親しい交際がないことをいう。  
また「両者」とは在家者と出家者のことであり、そのどちらとも親しく交わらないことを示す。  
「住処を本拠としない」とは、住む場所や依り所に執着することなく行動する者のことであり、そうした人をこそバラモンと呼ぶのである。

394.

‘‘Kiṃ te jaṭāhi dummedha, kiṃ te ajinasāṭiyā;

Abbhantaraṃ te gahanaṃ, bāhiraṃ parimajjasī’’ti. (jā. 1.4.98);

Tattha kiṃ te jaṭāhīti ambho duppañña tava baddhāhipi imāhi jaṭāhi sakhurāya nivatthāyapi imāya ajinacammasāṭikāya ca kimatthoti. Abbhantaranti abbhantarañhi te rāgādikilesagahanaṃ, kevalaṃ hatthilaṇḍaṃ assalaṇḍaṃ viya maṭṭhaṃ bāhiraṃ parimajjasīti attho.

## 対訳

1. ‘‘Kiṃ te jaṭāhi dummedha,

- kiṃ = 「何の役に立つのか、どんな意味があるのか」
- te = 「お前には／お前にとって」
- jaṭāhi = 「ジャータ（束ねたもじゃもじゃ頭の髪）で」（具格複数）
- dummedha = 「愚かな者よ、知恵のない者よ」（呼格）

→ 「愚か者よ、お前にとって、そのジャータの髪は何の役に立つのか。」

2. kiṃ te ajinasāṭiyā;

- kiṃ = 「何の役に立つのか」
- te = 「お前には／お前にとって」
- ajina-sāṭiyā = 「鹿皮の衣で、鹿皮の衣など着ていても」（ajina「鹿皮」+ sāṭi「衣」の具格単数）

→ 「お前にとって、その鹿皮の衣にどんな意味があるのか。」

3. Abbhantaraṃ te gahanaṃ,

- abbhantaraṃ = 「内側は、内面には」
- te = 「お前の」
- gahanaṃ = 「藪、密林、こんがらがった茂み」

→ 「お前の内面は、（煩悩という）密林であるのに」

4. bāhiraṃ parimajjasī’’ti.

- bāhiraṃ = 「外側だけを、外面を」
- parimajjasī = 「（お前は）こすり清めている、さすっている」（parimajjati の二人称単数・現在）
- ti = 「～と（言った）」の引用標識

→ 「お前は外側だけをこすり清めているのだ。」


5. Tattha kiṃ te jaṭāhīti

- tattha = 「ここで、この偈において」
- kiṃ te jaṭāhīti = 「『kiṃ te jaṭāhi』という（語句）は」

→ 「ここで『kiṃ te jaṭāhi』とは、」

6. ambho duppañña tava baddhāhipi imāhi jaṭāhi

- ambho = 「おい、…よ」（呼びかけの間投詞）
- duppañña = 「愚かな者、知恵の乏しい者」
- tava = 「お前には／お前の」
- baddhāhi = 「束ねられた、結わえられた」（baddha の具格複数）
- pi = 「～であっても、～すらも」
- imāhi = 「これらの」（具格複数）
- jaṭāhi = 「ジャータの髪、もじゃもじゃ頭の髪」（具格複数）

→ 「『おい愚か者よ、お前には、こうした束ねられたジャータの髪があったところで』」

7. sakhurāya nivatthāyapi imāya ajinacammasāṭikāya ca kimatthoti.

- sakhurāya = 「ごわごわした、粗い（質の）」※鹿皮衣の性質を形容
- nivatthāya pi = 「身にまとっていても、着ていても」（nivattha「身につけた」の具格）
- imāya = 「この」（具格単数）
- ajina-camma-sāṭikāya = 「鹿皮の衣である上衣」（ajina「鹿」+ camma「皮」+ sāṭikā「衣」の具格単数）
- ca = 「そして、～もまた」
- kim-attho = 「どんな役に立つのか、何の意味があるのか」
- ti = 「～という意味である、と」

→ 「『この粗い鹿皮の衣を身にまとっていたところで、お前にはそれがいったい何の役に立つのか』という意味である。」

8. Abbhantaranti abbhantarañhi te rāgādikilesagahanaṃ,

- abbhantaraṃ ti = 「『abbhantaraṃ』とは」
- abbhantaraṃ hi = 「実に内側には」
- te = 「お前の／お前には」
- rāga-ādi-kilesa-gahanaṃ = 「欲などの諸煩悩という藪・密林」
  - rāga = 「欲、愛着」
  - ādi = 「など、…をはじめとする」
  - kilesa = 「煩悩」
  - gahanaṃ = 「藪、密林」

→ 「『abbhantaraṃ』とは、『実にお前の内側には、欲などの煩悩という密林があり』ということであり、」

9. kevalaṃ hatthilaṇḍaṃ assalaṇḍaṃ viya maṭṭhaṃ bāhiraṃ parimajjasīti attho.

- kevalaṃ = 「ただ、単に」
- hatthi-laṇḍaṃ = 「象の胴体、象の体」
- assa-laṇḍaṃ = 「馬の胴体、馬の体」
- viya = 「～のように」
- maṭṭhaṃ = 「表面だけ、うわべだけ」
- bāhiraṃ = 「外側を」
- parimajjasī ti = 「（お前は）こすり清めている、なで回している」
- attho = 「という意味である」

→ 「ただ象や馬の体の表面ばかりをなで回すように、お前は外側のうわべだけをこすり清めているのだ、という意味である。」


## 意訳

「愚か者よ、お前の束ねたジャータの髪など、いったい何の役に立つというのか。
お前のまとっている鹿皮の粗い衣など、何の意味があろうか。

お前の内側には、欲をはじめとする煩悩の密林がそのまま茂っているのに、
お前がしているのは、象や馬の体の表面をなでて光らせるように、
うわべの外側だけをこすり清めていることにすぎないのだ。」

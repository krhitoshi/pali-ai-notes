## 対訳

### 偈文（Gāthā）

1. Evaṃ visesato ñatvā, appamādamhi paṇḍitā;

* evaṃ = 「このように」
* visesato = 「殊勝として、区別として（visesa + abl. sfx -to）」
  → 「明確な差異（区別）があることとして」
* ñatvā = 「知って（jānāti の絶対分詞）」
* appamādamhi = 「不放逸（気づきを保つこと）において」
* paṇḍitā = 「賢者たちは」

→ 「賢者たちは、不放逸について、このように（放逸との）違いを明確に知って、」

2. Appamāde pamodanti, ariyānaṃ gocare ratā.

* appamāde = 「不放逸を（不放逸において）」
* pamodanti = 「大いに喜ぶ」
* ariyānaṃ = 「聖者たちの」
* gocare = 「行境（ぎょうきょう：活動領域、牧場）」において
* ratā = 「楽しむ、喜びとする（ramati の過去分詞）」

→ 「不放逸を喜び、聖者たちの行境（活動領域）を楽しむ。」

### 註釈（Aṭṭhakathā）

3. Evaṃ visesato ñatvāti pamattassa vaṭṭato nissaraṇaṃ natthi, appamattassa atthīti evaṃ visesatova jānitvā.

* evaṃ visesato ñatvāti = 「『このように違いを知って』とは」
* pamattassa = 「放逸な者には」
* vaṭṭato = 「輪廻から」
* nissaraṇaṃ = 「出離（脱出）は」
* natthi = 「ない」
* appamattassa = 「不放逸な者には」
* atthīti = 「（出離が）ある、ということ」
* evaṃ = 「このように」
* visesato eva = 「まさに（両者の）違いとして」
* jānitvā = 「知って」

→ 「『このように違いを知って』とは、放逸な者には輪廻からの脱出はなく、不放逸な者には（脱出が）ある、というように、まさに明確な違いとして理解して、という意味である。」

4. Ke panetaṃ visesaṃ jānantīti? Appamādamhi paṇḍitāti

* ke pana etaṃ = 「では、誰がこの」
* visesaṃ = 「違いを」
* jānanti iti? = 「知るのか？」
* appamādamhi paṇḍitā ti = 「『不放逸における賢者たち』とは」

→ 「では、誰がこの違いを知るのか？ 『不放逸における賢者たち』とは、」

5. ye paṇḍitā medhāvino sappaññā attano appamāde ṭhatvā appamādaṃ vaḍḍhenti, te evaṃ visesakāraṇaṃ jānanti.

* ye paṇḍitā = 「賢明であり」
* medhāvino = 「知恵があり」
* sappaññā = 「（真の）慧を持つ人々が」
* attano = 「自らの」
* appamāde = 「不放逸（の状態）に」
* ṭhatvā = 「立って、留まって」
* appamādaṃ = 「不放逸を」
* vaḍḍhenti = 「増大させる」
* te = 「彼らが」
* evaṃ = 「このような」
* visesakāraṇaṃ = 「違いの理由を」
* jānanti = 「知っている」

→ 「賢明で、知恵があり、慧を持つ人々が、自ら不放逸に留まり、不放逸を増大させていく。彼らが、このように（放逸と不放逸の）違いの理由を知っているのである。」

6. Appamāde pamodantīti te evaṃ ñatvā tasmiṃ attano appamāde pamodanti, pahaṃsitamukhā tuṭṭhapahaṭṭhā honti.

* appamāde pamodanti ti = 「『不放逸を喜ぶ』とは」
* te evaṃ ñatvā = 「彼らはこのように知って」
* tasmiṃ attano appamāde = 「その自らの不放逸（の実践）において」
* pamodanti = 「大いに喜ぶ」
* pahaṃsita-mukhā = 「晴れやかな顔をして（笑みを浮かべ）」
* tuṭṭha-pahaṭṭhā = 「満足し、心躍らせて」
* honti = 「いる」

→ 「『不放逸を喜ぶ』とは、彼らはこのように（不放逸の価値を）知って、その自らの不放逸の実践を喜び、晴れやかな顔つきで、満足し心躍らせている、ということである。」

7. Ariyānaṃ gocare ratāti te evaṃ appamāde pamodantā taṃ appamādaṃ vaḍḍhetvā

* ariyānaṃ gocare ratā ti = 「『聖者たちの行境を楽しむ』とは」
* te evaṃ = 「彼らはこのように」
* appamāde pamodantā = 「不放逸を喜びながら」
* taṃ appamādaṃ = 「その不放逸を」
* vaḍḍhetvā = 「増大させて（成長させて）」

→ 「『聖者たちの行境を楽しむ』とは、彼らはこのように不放逸を喜びつつ、その不放逸をさらに成長させて、」

8. ariyānaṃ buddhapaccekabuddhabuddhasāvakānaṃ gocarasaṅkhāte catusatipaṭṭhānādibhede sattatiṃsa bodhipakkhiyadhamme

* ariyānaṃ = 「聖者たちの、すなわち」
* buddha-paccekabuddha-buddhasāvakānaṃ = 「仏陀、独覚（縁覚）、仏弟子たちの」
* gocara-saṅkhāte = 「行境（活動領域）と呼ばれる」
* catusatipaṭṭhāna-ādi-bhede = 「四念処などをはじめとする区分をもつ」
* sattatiṃsa = 「三十七の」
* bodhipakkhiyadhamme = 「菩提分法（悟りを助ける法）において」

→ 「聖者たち、すなわち仏陀・独覚・仏弟子たちの『行境』と呼ばれる、四念処をはじめとする三十七の菩提分法（悟りの構成要素）、」

9. navavidhalokuttaradhamme ca ratā niratā, abhiratā hontīti attho.

* navavidha-lokuttaradhamme ca = 「および、九種の出世間の法（四向・四果・涅槃）において」
* ratā = 「楽しみ」
* niratā = 「深く楽しみ」
* abhiratā = 「喜びとする」
* hontīti attho = 「ということである」

→ 「また九種の出世間法において、楽しみ、深く没頭し、喜びとする、という意味である。」

## 意訳

賢者たちは、放逸と不放逸の違いをこのように明確に知っています。
すなわち、放逸な者には輪廻からの脱出はなく、不放逸な者にはそれがあるという事実を理解しているのです。
そのため賢者たちは、不放逸（気づきを保つこと）の実践を大いに喜び、
ブッダや阿羅漢といった聖者たちの活動領域である「三十七の悟りへ導く法（三十七菩提分法）」や「九種の出世間の法（涅槃に至る階梯と涅槃そのもの）」を楽しみとしているのです。

384.

‘‘Yadā dvayesu dhammesu, pāragū hoti brāhmaṇo;

Athassa sabbe saṃyogā, atthaṃ gacchanti jānato’’ti.

Tattha yadāti yasmiṃ kāle dvidhā ṭhitesu samathavipassanādhammesu abhiññāpāragādivasena ayaṃ khīṇāsavo pāragū hoti, athassa vaṭṭasmiṃ saṃyojanasamatthā sabbe kāmayogādayo saṃyogā evaṃ jānantassa atthaṃ parikkhayaṃ gacchantīti attho.

## 対訳

1. Yadā dvayesu dhammesu,

- yadā = 「…するとき」「…であるとき」
- dvayesu = 「二つのものにおいて」「二種の」
- dhammesu = 「法において」「状態・要素において」

→ 「二種の法において〔ある状態が〕成り立つときに」

2. pāragū hoti brāhmaṇo;

- pāragū = 「彼岸に到達した者」「向こう岸に達した者」
  - pāra = 「彼岸・向こう岸」
  - gū（= gata）= 「到達した」
- hoti = 「…である／…となる」
- brāhmaṇo = 「バラモン。ここでは煩悩の尽きた聖者（阿羅漢）」

→ 「バラモンたる聖者は、彼岸に到達した者となる。」

3. Athassa sabbe saṃyogā,

- atha = 「そのとき」「それから」
- assa = 「彼において／彼には」（代名詞の与格）
- sabbe = 「すべての」
- saṃyogā = 「結びつき、結縛、束縛（サンヨーガ）」

→ 「すると、その人にとって、すべての結縛は」

4. atthaṃ gacchanti jānato’’ti.

- atthaṃ = 「消滅に」「滅尽へ」「存在しない状態へ」
- gacchanti = 「赴く」「至る」「行き着く」
- jānato = 「知っている者（のためには／にとって）」「かく知る者には」
- ti = （引用の終わりを示す）

→ 「かく知る者にとっては、滅尽へと行き着くのである、ということである。」


5. Tattha yadāti

- tattha = 「そこにおいて」「この偈においては」
- yadā ti = 「『yadā（…するとき）』という語は」

→ 「ここで『yadā』とは、」

6. yasmiṃ kāle dvidhā ṭhitesu samathavipassanādhammesu

- yasmiṃ kāle = 「いかなる時に」「その時に」
- dvidhā = 「二分して」「二つに分けられた」
- ṭhitesu = 「立っている、存している」「位置づけられている」
- samatha-vipassanā-dhammesu = 「止と観という法において」
  - samatha = 「止・奢摩他・静慮」
  - vipassanā = 「観・ヴィパッサナー・洞察」
  - dhammesu = 「法・状態・修行要素」

→ 「止と観という二種に分かれて存している法において、その時に」

7. abhiññāpāragādivasena

- abhiññā-pāragū-ādi-vasena = 「『abhiññā（阿毘知）』『pāragū（彼岸に至った者）』等という観点によって／という区別の仕方によって」
  - abhiññā = 「阿毘知・超知・最高の知」
  - pāragū = 「彼岸に至った者」
  - ādi = 「等」
  - vasena = 「…によって」「…という観点から」

→ 「『阿毘知』『彼岸に至った者』等という観点によって」

8. ayaṃ khīṇāsavo pāragū hoti,

- ayaṃ = 「この者は」
- khīṇāsavo = 「漏の尽きた者」「煩悩が尽きた者（阿羅漢）」
  - khīṇa = 「尽きた、滅した」
  - āsava = 「漏・煩悩」
- pāragū hoti = 「彼岸に到達した者となる」

→ 「この漏尽の人は、彼岸に到達した者となる。」

9. athassa vaṭṭasmiṃ saṃyojanasamatthā

- atha = 「そのとき」「すると」
- assa = 「彼において／彼には」
- vaṭṭasmiṃ = 「輪廻の流転において」「生死流転の過程において」
  - vaṭṭa = 「輪、輪廻（行相の循環）」
- saṃyojana-samatthā = 「結縛となる能力をもった」「結びつける働きをなすことのできる」
  - saṃyojana = 「結びつけるもの、結縛、（十）結」
  - samattha = 「能力をもつ、有能な」

→ 「そして、輪廻の流転において結縛となる能力をもつ」

10. sabbe kāmayogādayo saṃyogā

- sabbe = 「すべての」
- kāma-yoga-ādayo = 「欲のヨーガなど（以下同類のものすべて）」
  - kāma-yoga = 「欲（愛欲）というくびき・束縛」
  - ādi = 「等」
- saṃyogā = 「結縛・絆・つながり」

→ 「欲のヨーガなど、すべてのこれらの結びつき（結縛）は」

11. evaṃ jānantassa atthaṃ parikkhayaṃ gacchantīti attho.

- evaṃ = 「このように」
- jānantassa = 「知る者にとって」「かく知る人において」
- atthaṃ = 「滅に、消滅に」
- parikkhayaṃ = 「全き滅尽へ」「完全な消耗・尽き果てることへ」
- gacchanti = 「至る」「行き着く」
- ti attho = 「という意味である」

→ 「このように（法を）知る者にとっては、それらは滅尽へと行き着く、という意味である。」


## 意訳

「二つの法、すなわち止と観という修行の法において完成しているとき、そのバラモンたる聖者――煩悩の尽きた阿羅漢は、阿毘知を得て彼岸に到達した者となる。

そのとき、このように止観の法を理解しているその人にとっては、輪廻の流転において結縛となるはずの一切の束縛──欲のくびきなど、あらゆる結びつきは、ことごとく滅び尽くしてしまうのである。」

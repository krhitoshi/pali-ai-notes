406.

‘‘Aviruddhaṃ viruddhesu, attadaṇḍesu nibbutaṃ;

Sādānesu anādānaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha aviruddhanti āghātavasena viruddhesupi lokiyamahājanesu āghātābhāvena aviruddhaṃ. Hatthagate daṇḍe vā satthe vā avijjamānepi paresaṃ pahāradānato aviratattā attadaṇḍesu janesu nibbutaṃ nikkhittadaṇḍaṃ, pañcannaṃ khandhānaṃ ahaṃ mamanti gahitattā sādānesu tassa gahaṇassa abhāvena anādānaṃ taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Aviruddhaṃ viruddhesu,

* Aviruddhaṃ = 「争わない者、反抗しない者」
* viruddhesu = 「敵対する者たちの中において、敵意ある人々の中にあって」

→ 「敵対する者たちの中にあっても争わない者を」

2. attadaṇḍesu nibbutaṃ;

* attadaṇḍesu = 「自らの（暴力の）杖・武器を持っている者たちの中において」
  * atta = 「自分自身の」
  * daṇḍa = 「杖、棒、刑杖、武器」
  * -esu = 「～の中において」
* nibbutaṃ = 「鎮まりきった者、静まった者」

→ 「自ら武器を持つ人々の中にあっても、（暴力を）鎮めきった者を」

3. Sādānesu anādānaṃ,

* Sādānesu = 「取ること、執着という行為・対象のあるところにおいて」
* anādānaṃ = 「取らない者、掴み取らないこと、非取得」

→ 「（ものを）取り、執着することのあるところにあっても、取らない者を」

4. tamahaṃ brūmi brāhmaṇa’’nti.

* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「言う、呼ぶ」
* brāhmaṇaṃ = 「バラモンであると、真のバラモンと」

→ 「私はそのような者を、バラモンであると言うのである。」

---

5. Tattha aviruddhanti āghātavasena viruddhesupi lokiyamahājanesu āghātābhāvena aviruddhaṃ.

* Tattha = 「ここで、この偈において」
* aviruddhanti = 「『aviruddhaṃ（争わない者）』とは」
* āghāta-vasena = 「怒り・怨みという観点から」
* viruddhesupi = 「敵対している者たちの中にあってさえも」
  * viruddhesu = 「敵対する者たちの中に」
  * -pi = 「～でさえも」
* lokiya-mahā-janesu = 「世間の多くの人々の中において」
* āghāta-abhāvena = 「怒りがないことによって」
* aviruddhaṃ = 「争わない者である」

→ 「ここで『争わない者』とは、怒りという点から言えば、敵対している世間一般の人々の中にあってさえも、怒りがないことによって争わない者のことである。」

6. Hatthagate daṇḍe vā satthe vā avijjamānepi paresaṃ pahāradānato aviratattā attadaṇḍesu janesu nibbutaṃ nikkhittadaṇḍaṃ,

* Hatthagate = 「手にある、手に握られている」
* daṇḍe vā = 「棒、杖、殴打の道具、また」
* satthe vā = 「あるいは刃物、武器」
* avijjamānepi = 「（それらが）存在していなくても、たとえ持っていなくても」
* paresaṃ = 「他人に対して」
* pahāra-dānato = 「打撃を与えることから、殴ることをすることから」
* aviratattā = 「やめていないことのゆえに、離れていないために」
* attadaṇḍesu = 「自分自身の（暴力の）杖を持つ者たちの中において」
* janesu = 「人々の中に」
* nibbutaṃ = 「『nibbutaṃ（鎮まりきった者）』とは」
* nikkhitta-daṇḍaṃ = 「杖を捨てた者、暴力の道具を置き捨てた者である」

→ 「『自ら杖を持つ者たちの中にあって鎮まりきった者』とは、たとえ手に棒や刃物がなくても、他人を打つことをやめていないために（なお）自分の暴力の杖を持っている人々の中にあって、その中で杖を捨てた者、暴力の道具を置き捨てた者である。」

7. pañcannaṃ khandhānaṃ ahaṃ mamanti gahitattā sādānesu tassa gahaṇassa abhāvena anādānaṃ

* pañcannaṃ khandhānaṃ = 「五つの蘊（五蘊）の」
* ahaṃ mamanti = 「『これは私だ・これは私のものだ』と言って」
* gahitattā = 「取られているので、執着されているために」
* sādānesu = 「取られるものどもにおいて、取得の対象となるものの中において」
* tassa = 「その（＝「私・私のもの」としての）」
* gahaṇassa = 「取ること、執着することの」
* abhāvena = 「ないことによって、不在であることにより」
* anādānaṃ = 「『取らないこと、非取得であること』である」

→ 「『取るべきものどもの中で取らない者』とは、『これは私だ・これは私のものだ』として取られている五蘊という取得の対象となるものどもの中にあって、そのように取ることがないことによって非取得である、という意味である。」

8. taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「そのような者を」
* eva-rūpaṃ = 「まさにこのような性質の」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモンであると」
* vadāmi = 「言う、呼ぶ」
* ti attho = 「という意味である」

→ 「このような性質をそなえた者を、私はバラモンであると言うのである、という意味である。」

---

## 意訳

敵対し怒りを向けてくる人々の中にあっても、心に怒りを起こさず争わない者。
手に武器を携え、他人を打ち傷つけている人々の中にあっても、その暴力の杖を投げ捨てて、すっかり鎮まりきっている者。
五つの蘊を「これは私だ、私のものだ」とつかみ取って執着する人々の中にあっても、それらを取って自分のものとしない者――
私はその人を、真のバラモンと呼ぶのである。

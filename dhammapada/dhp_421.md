421.

‘‘Yassa pure ca pacchā ca, majjhe ca natthi kiñcanaṃ;

Akiñcanaṃ anādānaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha pureti atītesu khandhesu. Pacchāti anāgatesu khandhesu. Majjheti paccuppannesu khandhesu. Natthi kiñcananti yassetesu ṭhānesu taṇhāgāhasaṅkhātaṃ kiñcanaṃ natthi, tamahaṃ rāgakiñcanādīhi akiñcanaṃ kassaci gahaṇassa abhāvena anādānaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Yassa pure ca pacchā ca, majjhe ca natthi kiñcanaṃ;

* yassa = 「その人には／誰においては」
* pure ca = 「前に（pure）も」→「過去においても」
* pacchā ca = 「後に（pacchā）も」→「未来においても」
* majjhe ca = 「中間においても」→「現在においても」
* natthi = 「ない」
* kiñcanaṃ = 「何ものか・しがらみ・汚れ・執着物」

→「その人には、過去においても、未来においても、現在においても、何もののしがらみもない。」

2. Akiñcanaṃ anādānaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* akiñcanaṃ = 「何もののしがらみも持たない者」
  * a-（否定）+ kiñcana = 「しがらみ・執着物・汚れ」が「ない」
* anādānaṃ = 「取らない者・受け取らない者」
  * an-（否定）+ ādāna（取ること・執着してつかみ取ること）
* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「説く、呼ぶ、言う」
* brāhmaṇaṃ = 「バラモン（真実の意味での清らかな聖者）」

→「そのように一切のしがらみを持たず、取ることのない者を、私は『バラモン』と呼ぶのである。」

3. Tattha pureti atītesu khandhesu.

* tattha = 「ここで（この偈において）」
* pure ti = 「‘pure’という語は」
* atītesu khandhesu = 「過ぎ去った（過去の）五蘊に関して」

→「ここで ‘pure’ とは、過去に属する諸々の五蘊を意味する。」

4. Pacchāti anāgatesu khandhesu.

* pacchā ti = 「‘pacchā’ とは」
* anāgatesu khandhesu = 「まだ来ていない（未来の）五蘊に関して」

→「‘pacchā’ とは、未来に属する諸々の五蘊を意味する。」

5. Majjheti paccuppannesu khandhesu.

* majjhe ti = 「‘majjhe’ とは」
* paccuppannesu khandhesu = 「現に起こっている（現在の）五蘊に関して」

→「‘majjhe’ とは、現在に属する諸々の五蘊を意味する。」

6. Natthi kiñcananti yassetesu ṭhānesu taṇhāgāhasaṅkhātaṃ kiñcanaṃ natthi,

* natthi kiñcanaṃ ti = 「『kiñcanaṃ は存在しない』とは」
* yassa = 「誰において」
* etesu ṭhānesu = 「これらの立場（過去・未来・現在の五蘊）において」
* taṇhā-gāha-saṅkhātaṃ = 「渇愛による把持と呼ばれる」
  * taṇhā = 「渇愛」
  * gāha = 「つかむこと、把持」
  * saṅkhātaṃ = 「…と呼ばれる、…と名づけられる」
* kiñcanaṃ = 「しがらみ・執着物」
* natthi = 「存在しない」

→「『しがらみがない』とは、過去・未来・現在というこれらの場面において、渇愛の把持と呼ばれるしがらみが全く存在しない者である、ということである。」

7. tamahaṃ rāgakiñcanādīhi akiñcanaṃ kassaci gahaṇassa abhāvena anādānaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ ahaṃ = 「その者を、私は」
* rāga-kiñcana-ādīhi = 「貪欲というしがらみなどによって」
  * rāga = 「愛着・貪欲」
  * kiñcana = 「しがらみ」
  * -ādīhi = 「…など（その他の煩悩のしがらみ）によって」
* akiñcanaṃ = 「何もののしがらみも持たない者と」
* kassaci gahaṇassa = 「何ものをもつかみ取ることの」
* abhāvena = 「不在・無いがゆえに」
* anādānaṃ = 「取らない者・受け取らない者と」
* brāhmaṇaṃ = 「真のバラモン（聖者）と」
* vadāmi = 「私は説く・呼ぶ」
* ti attho = 「という意味である」

→「その者を、私は、貪欲などのしがらみによっても汚されていない『しがらみなき者』であり、何ものをもつかみ取ることがないがゆえに『取らない者』であり、そのような真のバラモンである、と説くのである、という意味である。」

---

## 意訳

「過去においても、未来においても、そして現在においても、  
いかなるものにも渇愛のしがらみを持たない者、  
一切のしがらみを離れて、何ものをもつかみ取らない者、  
その人こそ、私は真のバラモンと呼ぶ。

ここで『前』とは過去の五蘊、『後』とは未来の五蘊、  
『中間』とは現在の五蘊を指す。  
それら過去・未来・現在のいずれにおいても、  
渇愛による把持というしがらみがまったく存在しない者を、  
貪欲などのあらゆるしがらみを離れ、何ものも取らない者として、  
真のバラモンと説くのである。」

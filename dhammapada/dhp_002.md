### 註釈（Commentary）

4. Tattha kiñcāpi manoti avisesena sabbampi catubhūmikacittaṃ vuccati,

*   tattha = 「そこにおいて（その偈文において）」
*   kiñcāpi = 「～ではあるが、いかに～といえども」
*   mano ti = 「『意（mano）』とは」
*   avisesena = 「無差別に、区別なく」
*   sabbam pi = 「すべて」
*   catubhūmika-cittaṃ = 「四界の心（欲界・色界・無色界・出世間界の全意識）」
*   vuccati = 「言われる、呼ばれる」

→ 「そこで、いかに『意』といっても、（一般的には）区別なく四つの地界（欲界・色界・無色界・出世間）に属するすべての心が言及されるのではあるが、」

5. imasmiṃ pana pade niyamiyamānaṃ vavatthāpiyamānaṃ paricchijjiyamānaṃ aṭṭhavidhaṃ kāmāvacarakusalacittaṃ labbhati.

*   imasmiṃ pana pade = 「しかしながら、この句（詩節）においては」
*   niyamiyamānaṃ = 「限定されつつ」
*   vavatthāpiyamānaṃ = 「定義されつつ（特定され）」
*   paricchijjiyamānaṃ = 「区分（判断）されつつ」
*   aṭṭhavidhaṃ = 「八種の」
*   kāmāvacara-kusala-cittaṃ = 「欲界の善心」が
*   labbhati = 「得られる（適用される）」

→ 「しかしこの詩句においては、限定され、特定され、区分されて、『八種の欲界善心』が得られる（を意味する）。」

6. Vatthuvasena panāhariyamānaṃ tatopi somanassasahagataṃ ñāṇasampayuttacittameva labbhati.

*   vatthu-vasena = 「（この偈が説かれた）事由（物語）に基づいて」
*   pana = 「さらに」
*   āhariyamānaṃ = 「取り上げられるならば」
*   tato pi = 「それ（八種）の中でも」
*   somanassa-sahagataṃ = 「喜悦（喜び）倶行の」
*   ñāṇa-sampayutta-cittam eva = 「智相応（智慧を伴う）心のみが」
*   labbhati = 「得られる」

→ 「さらに（マッタカクンダリーの）因縁物語に基づいて取り上げるならば、それらの中でも『喜びを伴う、智恵と結びついた心（喜倶行智相応心）』のみが得られる。」

7. Pubbaṅgamāti tena paṭhamagāminā hutvā samannāgatā.

*   pubbaṅgamā ti = 「『前導とする』とは」
*   tena = 「それ（心）が」
*   paṭhamagāminā hutvā = 「先に行くものとなって」
*   samannāgatā = 「（諸法が）伴われている、具わっている」

→ 「『前導とする』とは、それ（意）が先に行くものとなり、（諸法がそれに）伴われているということである。」

8. Dhammāti vedanādayo tayo khandhā.

*   dhammā ti = 「『法（諸事象）』とは」
*   vedanā-ādayo = 「受（感受作用）などをはじめとする」
*   tayo khandhā = 「三つの蘊（受蘊・想蘊・行蘊）」

→ 「『法』とは、受などをはじめとする三つの蘊（心の構成要素）のことである。」

9. Ete hi uppādapaccayaṭṭhena somanassasampayuttamano pubbaṅgamo etesanti manopubbaṅgamā nāma.

*   ete hi = 「実にこれら（三蘊）は」
*   uppāda-paccaya-aṭṭhena = 「生起の原因という意味によって」
*   somanassa-sampayutta-mano = 「喜悦と相応した意が」
*   pubbaṅgamo etesan ti = 「これらの先導者である、という理由で」
*   manopubbaṅgamā nāma = 「『意を前導とするもの』と呼ばれる」

→ 「実にこれらは、生起の原因であるという意味において、喜悦と相応した意がそれらの先導者であるため、『意を前導とするもの』と呼ばれる。」

10. Yathā hi bahūsu ekato hutvā mahābhikkhusaṅghassa cīvaradānādīni vā uḷārapūjādhammassavanādīni vā mālāgandhasakkārakaraṇādīni vā puññāni karontesu

*   yathā hi = 「たとえば実に」
*   bahūsu ekato hutvā = 「多くの人々が一つになって（団結して）」
*   mahā-bhikkhu-saṅghassa = 「大比丘サンガに対して」
*   cīvara-dāna-ādīni vā = 「衣の布施などを、あるいは」
*   uḷāra-pūjā-dhammassavana-ādīni vā = 「盛大な供養や聴法などを、あるいは」
*   mālā-gandha-sakkāra-karaṇa-ādīni vā = 「花や香による尊敬行為などを、あるいは」
*   puññāni karontesu = 「（それらの）功徳を行っている時に」

→ 「たとえば、多くの人々が団結して、大比丘サンガに対して衣の布施や、盛大な供養や聴法、あるいは花や香による尊敬行為などの功徳を行っている時に、」

11. ‘‘ko etesaṃ pubbaṅgamo’’ti vutte yo tesaṃ paccayo hoti, yaṃ nissāya te tāni puññāni karonti, so tisso vā phusso vā tesaṃ pubbaṅgamoti vuccati, evaṃsampadamidaṃ veditabbaṃ.

*   ko etesaṃ pubbaṅgamo ti vutte = 「『誰が彼らの主導者か』と問われた時」
*   yo tesaṃ paccayo hoti = 「彼らの（活動の）縁（きっかけ）となる者が」
*   yaṃ nissāya = 「誰に依拠して」
*   te tāni puññāni karonti = 「彼らがそれらの功徳を行う（その人が）」
*   so tisso vā phusso vā = 「そのティッサなり、フッサなりが（※具体的な人名）」
*   tesaṃ pubbaṅgamo ti vuccati = 「彼らの主導者であると言われる」
*   evaṃ sampadam idaṃ veditabbaṃ = 「このように、このこと（意が前導であること）も理解されるべきである」

→ 「『誰が彼らの主導者か』と問われた際、彼らの活動のきっかけとなり、その人に依って彼らがそれらの功徳を行うところの人物、そのティッサなりフッサなりが『彼らの主導者である』と言われる。これと同様に、このこと（意が法を導くこと）も理解されるべきである。」

12. Iti uppādapaccayaṭṭhena mano pubbaṅgamo etesanti manopubbaṅgamā.

*   iti = 「このように」
*   uppāda-paccaya-aṭṭhena = 「生起の原因という意味によって」
*   mano pubbaṅgamo etesan ti = 「意がそれらの先導者であるゆえに」
*   manopubbaṅgamā = 「『意を前導とする』のである」

→ 「このように、生起の原因であるという意味で、意がそれらの先導者であるから『意を前導とする（マノプッバンガマー）』のである。」

13. Na hi te mane anuppajjante uppajjituṃ sakkonti, mano pana ekaccesu cetasikesu anuppajjantesupi uppajjatiyeva.

*   na hi = 「～ではないからである」
*   te = 「それら（心所法）は」
*   mane anuppajjante = 「意が生じない時には」
*   uppajjituṃ sakkonti = 「生じることができない」
*   mano pana = 「しかし、意は」
*   ekaccesu cetasikesu anuppajjantesu pi = 「一部の心所が生じない場合でも」
*   uppajjati yeva = 「まさに生じる」

→ 「なぜなら、それら（心所）は意が生じなければ生じることができないが、意の方は一部の心所が生じない場合であっても生じるからである。」

14. Evaṃ adhipativasena pana mano seṭṭho etesanti manoseṭṭhā.

*   evaṃ = 「同様に」
*   adhipati-vasena pana = 「さらに増上（支配的地位）という意味によって」
*   mano seṭṭho etesan ti = 「意がそれらにとって最勝（長）であるから」
*   manoseṭṭhā = 「『意を最勝とする』のである」

→ 「同様に、支配的な地位にあるという意味で、意がそれらにとって最勝であるから『意を最勝とする（マノセッダー）』のである。」

15. Yathā hi gaṇādīnaṃ adhipati puriso gaṇaseṭṭho seṇiseṭṭhoti vuccati, tathā tesampi manova seṭṭho.

*   yathā hi = 「たとえば」
*   gaṇa-ādīnaṃ = 「群衆などの」
*   adhipati puriso = 「支配者である人が」
*   gaṇa-seṭṭho seṇi-seṭṭho ti = 「群衆の長、ギルド（組合）の長であると」
*   vuccati = 「呼ばれるように」
*   tathā = 「そのように」
*   tesam pi = 「それら（心所法）にとっても」
*   mano va seṭṭho = 「意こそが最勝なのである」

→ 「たとえば、群衆などの支配者である人が『群衆の長』や『ギルドの長』と呼ばれるように、それら（心所）にとっても意こそが最勝なのである。」

16. Yathā pana suvaṇṇādīhi nipphāditāni bhaṇḍāni suvaṇṇamayādīni nāma honti, tathā etepi manato nipphannattā manomayā nāma.

*   yathā pana = 「また、たとえば」
*   suvaṇṇa-ādīhi = 「黄金などによって」
*   nipphāditāni bhaṇḍāni = 「作られた物品が」
*   suvaṇṇamayā-ādīni nāma honti = 「『黄金製』などという名になる（ように）」
*   tathā = 「そのように」
*   ete pi = 「これら（心所法）も」
*   manato nipphannattā = 「意から生じたものであるがゆえに」
*   manomayā nāma = 「『意より成る（マノマヤー）』と呼ばれる」

→ 「また、たとえば黄金などで作られた物品が『黄金製』などと呼ばれるように、これらも意から生じたものであるがゆえに『意より成る』と呼ばれる。」

17. Pasannenāti anabhijjhādīhi guṇehi pasannena.

*   pasannenā ti = 「『清らかな（心）で』とは」
*   anabhijjhā-ādīhi guṇehi = 「無貪などの徳によって」
*   pasannena = 「清らかになった（心で）」

*   bhāsati vā karoti vā ti = 「『語り、あるいは行う』とは」

→ 「『語り、あるいは行う』とは、このような心で語る者は、四種の口の善行のみを語り、」

*   karonto = 「行う者は」
*   tividhaṃ kāya-sucaritam eva = 「三種の身の善行（不殺生・不偸盗・不邪淫）のみを」
*   karoti = 「行う」

→ 「行う者は、三種の身の善行のみを行う。」

*   abhāsanto akaronto = 「（言葉を）発さず、（行為を）行わない時も」
*   tāya = 「その」
*   anabhijjhā-ādīhi pasanna-mānasatāya = 「無貪などによって清らかな心の状態によって」

21. Evamassa dasa kusalakammapathā pāripūriṃ gacchanti.

*   evam assa = 「このように彼に」
*   dasa kusala-kamma-pathā = 「十善業道が」
*   pāripūriṃ gacchanti = 「完成に至る（成就する）」

→ 「このようにして、彼において十善業道が成就する。」

22. Tato naṃ sukhamanvetīti tato tividhasucaritato naṃ puggalaṃ sukha manveti.

*   tato naṃ sukham anvetī ti = 「『それによって彼に楽が従う』とは」
*   tato = 「その」
*   tividha-sucaritato = 「三種の（身口意の）善行によって」
*   naṃ puggalaṃ = 「その人物に」
*   sukhaṃ anveti = 「楽が従う」

→ 「『それによって彼に楽が従う』とは、その三種の善行によって、その人物に楽が付き従うという意味である。」

23. Idha tebhūmikampi kusalaṃ adhippetaṃ,

*   idha = 「ここでは」
*   tebhūmikam pi kusalaṃ = 「三界（欲・色・無色）の善であっても」
*   adhippetaṃ = 「意図されている（含まれる）」

→ 「ここでは（本来の意味としては）、三界に属する善行も意図されている。」

*   tasmā = 「それゆえ」
*   tebhūmika-sucarita-anubhāvena = 「三界の善行の威力によって」
*   sugati-bhave = 「善趣（天界や人間界）の生存に」
*   nibbattaṃ puggalaṃ = 「生まれた人物に」

→ 「それゆえ、三界の善行の威力によって善き生存（善趣）に生まれた人物に対して、」

25. duggatiyaṃ vā sukhānubhavanaṭṭhāne ṭhitaṃ kāyavatthukampi itaravatthukampi avatthukampīti kāyikacetasikaṃ vipākasukhaṃ anugacchati, na vijahatīti attho veditabbo.

→ 「何のごとくか。『離れない影のごとく』とは、たとえば影というものは身体に結びついており、」

27. sarīre gacchante gacchati, tiṭṭhante tiṭṭhati , nisīdante nisīdati,

*   sarīre gacchante = 「身体が行く時には」
*   gacchati = 「（影も）行き」
*   tiṭṭhante tiṭṭhati = 「（身体が）立つ時には立ち」
*   nisīdante nisīdati = 「（身体が）座る時には座る」

→ 「身体が行く時には行き、立つ時には立ち、座る時には座り、」

28. na sakkoti, ‘‘saṇhena vā pharusena vā nivattāhī’’ti vatvā vā pothetvā vā nivattāpetuṃ. Kasmā? Sarīrappaṭibaddhattā.

*   na sakkoti = 「（誰も）～できない」
*   saṇhena vā pharusena vā = 「穏やかな言葉や、粗暴な言葉で」
*   nivattāhī ti vatvā vā = 「『戻れ』と言ったり」
*   pothetvā vā = 「あるいは（影を）叩いたりして」
*   nivattāpetuṃ = 「（影を）引き返させることは（できない）」
*   kasmā = 「なぜか」
*   sarīra-paṭibaddhattā = 「身体に結びついているからである」

→ 「『戻れ』と優しく言ったり厳しく言ったりして、あるいは叩いたりして、影を引き返させることはできない。なぜか。身体に結びついているからである。」

29. Evameva imesaṃ dasannaṃ kusalakammapathānaṃ āciṇṇasamāciṇṇakusalamūlikaṃ kāmāvacarādibhedaṃ kāyikacetasikasukhaṃ gatagataṭṭhāne anapāyinī chāyā viya hutvā na vijahatīti.

*   evam eva = 「まさにそのように」
*   imesaṃ dasannaṃ kusala-kamma-pathānaṃ = 「これら十善業道の」
*   āciṇṇa-samāciṇṇa-kusala-mūlikaṃ = 「習慣的に行われた善を根源とする」
*   kāmāvacara-ādi-bhedaṃ = 「欲界などの区分による」
*   kāyika-cetasika-sukhaṃ = 「身体的・精神的な楽（幸福）は」
*   gata-gata-ṭṭhāne = 「至るところで」
*   anapāyinī chāyā viya hutvā = 「離れない影のようになって」
*   na vijahatī ti = 「（その人を）捨て去らないのである」

→ 「まさにそのように、これら十善業道を習慣的に実践した善を根源とし、欲界などに分類される身体的および精神的な幸福は、どこへ行こうとも、離れない影のようになって、決してその人を捨て去らないのである。」

---

### 意訳

「私たちのあらゆる精神的な現象（心所）は、心が先導し、心が主であり、心によって造り出されます。もしも、清らかな心で言葉を語り、あるいは行いを行うならば、それによって幸福（楽）はその人に付き従います。あたかも、決して身体から離れることのない影のように。

ここで『心（意）』というのは、本来はあらゆる領域の心を指しますが、この詩句においては特に『欲界の善き心』、因縁物語に即して言えば『喜びを伴い、智慧と結びついた心』を意味しています。
心が『先導する』というのは、心が物事の生起の原因であり、常に先頭に立つからです。集団で寄付や供養を行う際に主導者がいるように、心の働きにおいては心がリーダーなのです。また、黄金製品が黄金でできているように、精神的な現象は心から生じたものなので『心によって造られたもの』と言われます。

『清らかな心』とは、貪りなどの汚れがない心のことです。このような心で語れば、嘘や悪口のない正しい言葉となり、行えば殺生や盗みのない正しい行いとなります。また、何もしていない時でも心の中で正しい思いを持ち続けます。こうして十種の善き行い（十善業道）が完成されます。

『幸福が付き従う』とは、影が身体の動きに合わせて移動し、誰もそれを追い払うことができないのと同じです。善い行いによって生み出された身体的・精神的な幸福は、その人がどこへ行こうとも、決してその人を見捨てることなく、影のように寄り添い続けるのです。」

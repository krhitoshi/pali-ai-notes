## 対訳

### 偈文 26

1. Pamādamanuyuñjanti, bālā dummedhino janā;

* pamādam-anuyuñjanti = pamādaṃ（放逸、怠惰）+ anuyuñjanti（専心する、耽る）
  → 「放逸に耽っている」
* bālā = 「愚かな者たち」
* dummedhino = du（悪しき）+ medhā（智慧）+ in（持つ）
  → 「智慧なき者たち、愚鈍な者たち」
* janā = 「人々は」

→ 「智慧なき愚かな人々は、放逸に耽っている。」

2. Appamādañca medhāvī, dhanaṃ seṭṭhaṃva rakkhati.

* appamādañ-ca = appamādaṃ（不放逸、勤勉）+ ca（一方、しかし）
* medhāvī = 「賢者は（智慧ある者は）」
* dhanaṃ = 「財産を」
* seṭṭhaṃ-va = seṭṭhaṃ（最上の）+ iva（～のごとく）
* rakkhati = 「守る」

→ 「一方、賢者は最上の財産（を守る）のごとく、不放逸を守る。」

### 偈文 27

3. ‘‘Mā pamādamanuyuñjetha, mā kāmaratisanthavaṃ;

* mā = 「～するなかれ（禁止）」
* pamādam-anuyuñjetha = 「放逸に専心してはならない」
* kāma-rati-santhavaṃ = kāma（諸欲）+ rati（喜び）+ santhava（親密、耽溺）
  → 「諸欲の喜びに耽溺すること（をっしてはならない）」

→ 「放逸に専念してはならない。諸欲の喜びに耽溺してはならない。」

4. Appamatto hi jhāyanto, pappoti vipulaṃ sukha’’nti.

* appamatto = 「不放逸なる者は」
* hi = 「実に」
* jhāyanto = 「禅定（瞑想）を修する者は」
* pappoti = pāpuṇāti（達する、獲得する）
* vipulaṃ = 「広大な、大きな」
* sukhaṃ-ti = 「安楽に」+ ti（引用終わり）

→ 「実に、不放逸にして禅定を修する者は、広大な安楽に達するからである。」

### 註釈（Aṭṭhakathā）

5. Tattha bālāti bālyena samannāgatā idhalokaparalokatthaṃ ajānantā.

* tattha = 「そこにおいて（その偈の中で）」
* bālā ti = 「『bālā（愚者）』とは」
* bālyena samannāgatā = 「愚かさを具備しており」
* idhaloka-paraloka-atthaṃ = 「現世と来世の義（利益・幸福）を」
* ajānantā = 「知らない者たちのことである」

→ 「そこで『愚者』とは、愚かさを具えており、現世と来世の利益を知らない者たちのことである。」

6. Dummedhinoti nippaññā.

* dummedhino ti = 「『dummedhino（悪しき智慧ある者）』とは」
* nippaññā = 「智慧がない者（無知な者）のことである」

→ 「『悪慧なる者』とは、無知な者のことである。」

7. Te pamāde ādīnavaṃ apassantā pamādaṃ anuyuñjanti pavattenti, pamādena kālaṃ vītināmenti.

* te = 「彼らは」
* pamāde = 「放逸の中に」
* ādīnavaṃ apassantā = 「過患（災い、欠点）を見ずに」
* pamādaṃ anuyuñjanti pavattenti = 「放逸を専修し、行い」
* pamādena kālaṃ vītināmenti = 「放逸によって時を過ごす」

→ 「彼らは放逸における災いを見ず、放逸に耽り、それを行い、放逸によって時を過ごすのである。」

8. Medhāvīti dhammojapaññāya samannāgato pana paṇḍito

* medhāvī ti = 「『賢者』とは」
* dhammoja-paññāya = 「法の滋養ある（優れた）智慧を」
* samannāgato pana = 「具えているところの」
* paṇḍito = 「賢人である」

→ 「『賢者』とは、法の滋養ある智慧を具えた賢人のことである。」

9. kulavaṃsāgataṃ seṭṭhaṃ uttamaṃ sattaratanadhanaṃ viya appamādaṃ rakkhati.

* kulavaṃsa-āgataṃ = 「家系に伝わる」
* seṭṭhaṃ uttamaṃ = 「最上の、至高の」
* sattaratana-dhanaṃ viya = 「七宝という財産のように」
* appamādaṃ rakkhati = 「不放逸を守る（大切にする）」

→ 「（賢者は）家系に伝わる最上の至高なる七宝の財産のように、不放逸を大切に守る。」

10. Yathā hi uttamaṃ dhanaṃ nissāya

* yathā hi = 「たとえば実に～のように」
* uttamaṃ dhanaṃ nissāya = 「最上の財産に依拠して」

→ 「たとえば、最上の財産を元手にして、」

11. ‘‘kāmaguṇasampattiṃ pāpuṇissāma, puttadāraṃ posessāma, paralokagamanamaggaṃ sodhessāmā’’ti dhane ānisaṃsaṃ passantā taṃ rakkhanti,

* kāmaguṇa-sampattiṃ = 「五欲の功徳（妙なる対象）の成就を」
* pāpuṇissāma = 「我らは得るであろう」
* puttadāraṃ posessāma = 「妻子を養うであろう」
* paraloka-gamana-maggaṃ = 「来世へ行く道を」
* sodhessāmā ti = 「浄めるであろう、といって」
* dhane ānisaṃsaṃ passantā = 「財産における功徳（利益）を見ながら」

→ 「『（この財によって）五欲の楽しみを得よう、妻子を養おう、来世への道を浄めよう』と、財産がもたらす利益を見据えてそれを守るように、」

12. evaṃ paṇḍitopi appamatto

13. ‘‘paṭhamajjhānādīni paṭilabhissāmi, maggaphalādīni pāpuṇissāmi, tisso vijjā, cha abhiññā sampādessāmī’’ti

* paṭhama-jhāna-ādīni = 「初禅をはじめとする（禅定を）」
* paṭilabhissāmi = 「私は獲得するであろう」
* magga-phala-ādīni = 「道と果（悟りの階梯）などを」
* pāpuṇissāmi = 「達するであろう」
* tisso vijjā = 「三明を」
* cha abhiññā = 「六神通を」

14. appamāde ānisaṃsaṃ passanto dhanaṃ seṭṭhaṃva appamādaṃ rakkhatīti attho.

* appamāde = 「不放逸の中に」

* mā pamādam-anuyuñjetha = 「放逸に専心してはならない」
* mā pamādena kālaṃ vītināmayittha = 「放逸によって時を過ごしてはならない（という勧めである）」

→ 「『放逸になるな』とは、それゆえ『あなた方は放逸に耽ってはならない、放逸に時を費やしてはならない』ということである。」

16. Mā kāmaratisanthavanti vatthukāmakilesakāmesu ratisaṅkhātaṃ taṇhāsanthavampi mā anuyuñjetha mā cintayittha mā paṭilabhittha.

* mā kāmaratisanthavan ti = 「『諸欲の喜びに親しむな』とは」
* vatthukāma-kilesakāmesu = 「事物の欲（対象）と煩悩の欲において」
* rati-saṅkhātaṃ = 「喜びと呼ばれる」
* taṇhā-santhavam-pi = 「渇愛による親密さ（執着）をも」
* mā anuyuñjetha = 「専修してはならない」
* mā cintayittha = 「（それを）考えてはならない」
* mā paṭilabhittha = 「（それを）得てはならない」

→ 「『諸欲の喜びに親しむな』とは、欲の対象や煩悩の欲における喜びとしての『渇愛による執着』にも専念してはならない、心を向けてはならない、それを得てはならないということである。」

17. Appamatto hīti upaṭṭhitassatitāya hi appamatto jhāyanto puggalo vipulaṃ uḷāraṃ nibbānasukhaṃ pāpuṇātīti.

* appamatto hī ti = 「『実に不放逸なる者は』とは」
* upaṭṭhita-satitāya hi = 「気づき（念）が確立していることによって、実に」
* appamatto = 「不放逸であり」
* jhāyanto puggalo = 「禅定（瞑想）している人は」
* vipulaṃ uḷāraṃ = 「広大で崇高な」
* nibbāna-sukhaṃ pāpuṇātī ti = 「涅槃の安楽に達する、という意味である」

→ 「『実に不放逸なる者は』とは、念（サティ）が確立しているがゆえに不放逸であり、瞑想に励むその人は、広大かつ崇高な涅槃の安楽に到達する、という意味である。」

## 意訳

愚かで智慧のない人々は、放逸（怠け、不注意）に耽っている。
しかし賢者は、最上の財産を大切に守るように、不放逸（勤勉、気づき）を守る。

放逸に耽ってはならない。諸欲の喜びに溺れてはならない。
不放逸にして瞑想に励む者こそが、広大な安楽を得るのだから。

**【註釈】**
ここで「愚者」とは、愚かさを備え、現世や来世における真の幸福を知らない人々のことである。「智慧なき人々」とは無知な者のことである。彼らは放逸の危険性を見抜けず、放逸に耽って時を無駄に過ごす。

一方「賢者」とは、法の滋養となる智慧を備えた賢人のことである。彼は、代々伝わる最上の七宝の財産を大切にするように、不放逸を大切に守る。
たとえば、人々が最上の財産を元手にして「これで五欲の楽しみを得よう、妻子を養おう、来世への道を整えよう」とその利益を見込んで財産を守るように、賢者もまた「不放逸によって初禅などを得よう、聖道と聖果に達しよう、三明・六神通を完成させよう」と、不放逸がもたらす功徳を見据え、最上の財産のごとく不放逸を守るのである。

「放逸に耽るな」とは、それゆえにあなた方は放逸に専念してはならず、放逸に時を費やしてはならないという教えである。「諸欲の喜びに溺れるな」とは、欲望の対象や煩悩における楽しみ、すなわち渇愛による執着に専心したり、思いを巡らせたり、それを求めてはならないということである。

「不放逸な者」とは、念（サティ）を確立して不放逸であり、禅定を実践する人のことである。そのような人は、広大かつ崇高な涅槃の安楽に到達する、という意味である。

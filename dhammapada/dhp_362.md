原文（偈文）

362.

‘‘Hatthasaṃyato pādasaṃyato,
Vācāsaṃyato saṃyatuttamo;
Ajjhattarato samāhito,
Eko santusito tamāhu bhikkhu’’nti.

原文（註釈）

Tattha hatthasaṃyatoti hatthakīḷāpanādīnaṃ vā hatthena paresaṃ paharaṇādīnaṃ vā abhāvena hatthasaṃyato. Dutiyapadepi eseva nayo. Vācāya pana musāvādādīnaṃ akaraṇato vācāya saṃyato. Saṃyatuttamoti saṃyatattabhāvo, kāyacalanasīsukkhipanabhamukavikārādīnaṃ akārakoti attho. Ajjhattaratoti gocarajjhattasaṅkhātāya kammaṭṭhānabhāvanāya rato. Samāhitoti suṭṭhu ṭhapito. Eko santusitoti ekavihārī hutvā suṭṭhu tusito vipassanācārato paṭṭhāya attano adhigamena tuṭṭhamānaso. Puthujjanakalyāṇakañhi ādiṃ katvā sabbepi sekhā attano adhigamena santussantīti santusitā, arahā pana ekantasantusitova. Taṃ sandhāyetaṃ vuttaṃ.

---

語句分解と文法（偈文）

① Hatthasaṃyato pādasaṃyato,

* hattha = 「手」
* saṃyata = 「制御された・抑制された」
* hatthasaṃyato = 「手を制御している者」

* pāda = 「足」
* pādasaṃyato = 「足を制御している者」

→ 「手をよく制御し、足をよく制御している者は」

---

② Vācāsaṃyato saṃyatuttamo;

* vācā = 「言葉、言語、発語」
* vācāsaṃyato = 「言葉を制御している者」

* saṃyata = 「抑制された、制御された」
* uttama = 「最上の、最も優れた」
* saṃyatuttamo = saṃyata + uttama
  * 直訳：「抑制された者たちの中で最上の者」
  * 文脈上：「最上の自制心をそなえた者」

→ 「言葉を制御し、自制の面で最も優れた者であり」

---

③ Ajjhattarato samāhito,

* ajjhattā = 「内なる、内側の、自分の内面」
* rato（rata）= 「喜ぶ、楽しむ、愛好する」
* ajjhattarato = 「内面を喜びとする者、内面的な対象を楽しむ者」
  * ここでは「内観・内面的な修行を楽しむ者」のニュアンス

* samāhita（samāhita）= 「（心が）よく定まった、安定した、統一された」
  * 「心が定まっている」「よく集中している」

→ 「内面を喜びとし、心がよく定まっている者であり」

---

④ Eko santusito tamāhu bhikkhu’’nti.

* eko = 「一人で、独りで」
  * 修行生活を一人で営む、独居瞑想を示唆

* santusita（santusito）= 「満足している、足るを知る」
  * santusita < *santusita = 「満ち足りている、満足している」

* taṃ = 「その者を」
* āhu = 「（賢者たちは）…と言う」
* bhikkhuṃ = 「比丘を」
* tam āhu bhikkhuṃ → sandhi で tamāhu bhikkhu

→ 「一人でいて満ち足りている、そのような者を、賢者たちは『比丘』と呼ぶのである。」

---

語句分解と文法（註釈）

① Tattha hatthasaṃyatoti hatthakīḷāpanādīnaṃ vā hatthena paresaṃ paharaṇādīnaṃ vā abhāvena hatthasaṃyato.

* tattha = 「ここで、この句においては」
* hatthasaṃyato ti = 「『hatthasaṃyata』とは～という意味である」

* hatthakīḷāpana = 「手で遊ぶこと、手遊び、無駄な手の動き」
  * -ādīnaṃ = 「～などの」

* vā … vā = 「あるいは～あるいは～」→「…や…などが」

* hatthena = 「手で」
* paresaṃ = 「他人を」
* paharaṇa = 「打つこと、傷つけること、暴力を加えること」
  * -ādīnaṃ = 「～などの」

* abhāvena = 「…が無いことによって、…をしないことによって」

→
「ここで『手を制御した者』とは、手遊びなどを行わないこと、また手で他人を打つなどの行為を行わないことによって、手を制御した者である、という意味である。」

---

② Dutiyapadepi eseva nayo.

* dutiyapade pi = 「第二の句（pāda）についてもまた」
* eseva nayo = 「同じ道理である、同様に理解すべきである」

→ 「第二句（pādasaṃyato）についても、同じ理解の仕方である。」

---

③ Vācāya pana musāvādādīnaṃ akaraṇato vācāya saṃyato.

* vācāya pana = 「さて、言葉については」
* musāvāda = 「虚偽語、嘘」
* -ādīnaṃ = 「～などの」
* akaraṇato = 「行わないことによって」

* vācāya saṃyato = 「言葉を制御した者である」

→ 「言葉については、嘘をつくなどの（悪しき）言葉を用いないことによって、言葉を制御した者である。」

---

④ Saṃyatuttamoti saṃyatattabhāvo, kāyacalanasīsukkhipanabhamukavikārādīnaṃ akārakoti attho.

* saṃyatuttamo ti = 「『saṃyatuttama』とは～という意味である」

* saṃyatta-ttabhāvo（= saṃyatattabhāvo）
  * saṃyata = 「制御された」
  * -ttabhāva = 「その状態・ありさま」
  * → 「よく制御された状態、自制されたあり方」

* kāya-calana = 「身体を揺り動かすこと、無駄に動かすこと」
* sīsa-ukkhipana = 「頭を持ち上げること、頭を振り回すこと」
* bhā-mukavikāra
  * bhā（bhamma 等）= 「眉・額」
  * mukavikāra = 「口のゆがめ方、表情の変形」
  * → 「眉や口をゆがめるような表情の変化」

* -ādīnaṃ = 「～などの」
* akārako = 「行わない者」
* ti attho = 「という意味である」

→
「『最上に自制された者』とは、自制された状態を備え、身体をむやみに動かすことや、頭を振り上げること、眉や口をゆがめるような表情の変化などを行わない者、という意味である。」

---

⑤ Ajjhattaratoti gocarajjhattasaṅkhātāya kammaṭṭhānabhāvanāya rato.

* ajjhattarato ti = 「『ajjhattarata』とは～という意味である」

* gocara = 「行動範囲、牧草地、心の対象領域」
* ajjhatta-saṅkhātāya = 「『内面』と呼ばれる」
* kammaṭṭhāna-bhāvanā = 「禅定の対象とする修行、瞑想の実習」

* rato = 「喜ぶ者、愛好する者」

→
「『内面を喜びとする者』とは、内面を対象領域とする禅定の修習（内観の瞑想）を喜びとしている者のことである。」

---

⑥ Samāhitoti suṭṭhu ṭhapito.

* samāhito ti = 「『samāhita』とは～という意味である」
* suṭṭhu = 「よく、十分に」
* ṭhapito = 「置かれている、確立している」

→ 「『よく定まった者』とは、心がよく確立している者である。」

---

⑦ Eko santusitoti ekavihārī hutvā suṭṭhu tusito vipassanācārato paṭṭhāya attano adhigamena tuṭṭhamānaso.

* eko santusito ti = 「『一人で満足している者』とは～という意味である」

* ekavihārī hutvā = 「独りで住む者となって」
* suṭṭhu tusito = 「よく満ち足りている、よく満足している」

* vipassanā-cāra-to paṭṭhāya
  * vipassanā = 「観（内観の智慧）」
  * cāra = 「行い、実践」
  * -to paṭṭhāya = 「～の実践から始めて」
  * → 「vipassanā の実践から始めて」

* attano adhigamena = 「自分自身の悟り・到達によって」
* tuṭṭha-mānaso = 「心が満足している者」

→
「『一人で満足している者』とは、独りで住む者となり、観（vipassanā）の実践から始めて、自らの悟り（到達）によって心がよく満ち足りている者のことである。」

---

⑧ Puthujjanakalyāṇakañhi ādiṃ katvā sabbepi sekhā attano adhigamena santussantīti santusitā, arahā pana ekantasantusitova.

* puthujjana-kalyāṇakaṃ hi = 「凡夫であっても善性を持つ者（善良な凡夫）は」
* ādiṃ katvā = 「初めとして、はじめに挙げて」
* sabbe pi sekhā = 「一切の『学びの身（sekhā）』たちもまた」
  * sekha = 「修行中の聖者（預流・一来・不還・修行中の阿羅漢候補など）」

* attano adhigamena = 「それぞれ自らの到達によって」
* santussanti ti = 「満足している」
* santusitā = 「満足している者たちである」

* arahā pana = 「しかし、阿羅漢は」
* ekanta-santusito va = 「全く・偏りなく・徹底的に満足しているだけである」
  * ekanta = 「完全に、絶対的に」
  * → 「決定的な満足・完全な満足」

→
「実際、善良な凡夫を初めとして、一切の『学びの身』である聖者たちも、それぞれ自らの到達によって満足しているので『満足した者たち』と言える。しかし阿羅漢は、他と比べものにならない、完全な満足そのものである。」

---

⑨ Taṃ sandhāyetaṃ vuttaṃ.

* taṃ sandhāya = 「そのことを指して、そのことを念頭に置いて」
* etaṃ vuttaṃ = 「このように説かれたのである」

→ 「このことをふまえて、この偈が説かれたのである。」

---

逐語訳（偈文）

「手を制御し、足を制御し、
言葉を制御し、自制において最も優れた者で、
内面を喜びとし、よく定まった者であり、
一人でいて満足している、そのような者を、
賢者たちは『比丘』と言う。」

---

逐語訳（註釈の要旨）

「ここで『手を制御した者』とは、手で遊ぶことや、手を使って他人を打つような行為などを行わないことによって、手をよく制御した者を意味する。第二句の『足を制御した者』も同様に理解すべきである。

『言葉を制御した者』とは、嘘をつくなどの悪しき言葉を用いないことによる。

『最上に自制された者』とは、自制された状態をそなえ、身体を無駄に動かしたり、頭を振り上げたり、眉や口をゆがめるような表情の変化などを行わない者のことである。

『内面を喜びとする者』とは、内面を対象領域とする禅定の修習、すなわち内観の瞑想を喜びとしている者である。『よく定まった者』とは、心がよく確立している者である。

『一人で満足している者』とは、独りで住む者となり、観（vipassanā）の実践から始めて、自らの悟り・到達によって心がよく満ち足りている者のことである。

善良な凡夫をはじめとして、一切の学びの身（聖者たち）は、それぞれ自らの到達によって満足しているので『満足した者たち』と言える。しかし阿羅漢は、他と比較にならない、完全な満足そのものである。そのことをふまえて、この偈が説かれたのである。」

自然な日本語訳（偈文）

「手を慎み、足を慎み、
言葉を慎み、自制において最もすぐれ、
内なるものを喜びとし、心の定まった者で、
ひとり静かに暮らし、満ち足りている人――
賢者たちは、その人こそを『比丘』と呼ぶ。」

---

自然な日本語訳（註釈）

「ここで『手を慎む者』とは、手遊びをしたり、手で人を打つような行為などをしないことによって、手をよく制御している人のことをいう。第二句の『足を慎む者』も同じように理解すべきである。

『言葉を慎む者』とは、嘘をつくなどの悪い言葉を用いない人のことである。

『自制において最もすぐれた者』とは、自制されたあり方を備え、身体をむやみに揺り動かしたり、頭を振り回したり、眉や口をゆがめるような身ぶりや表情をとらない人を意味する。

『内なるものを喜びとする者』とは、外界ではなく内面を対象とする瞑想の修行に喜びを見いだしている人であり、『心の定まった者』とは、心がよく落ち着き、確立している人である。

『ひとり満ち足りている者』とは、独りで暮らし、観（ヴィパッサナー）の実践に励むことからはじまり、自ら体験した悟り・到達によって、心から満ち足りている人のことである。

実際、善良な凡夫をはじめ、一切の修行途上の聖者たちも、それぞれ自分の到達に応じて満足しているので、『満足している者たち』と言うことができる。しかし阿羅漢は、その中でも別格であり、完全な満足そのものである。このような意味をふまえて、この偈が説かれているのである。」

407.

‘‘Yassa rāgo ca doso ca, māno makkho ca pātito;

Sāsaporiva āraggā, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha āraggāti yassete rāgādayo kilesā, ayañca paraguṇamakkhanalakkhaṇo makkho āraggā sāsapo viya pātito, yathā sāsapo āragge na santiṭṭhati, evaṃ citte na santiṭṭhati, tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Yassa rāgo ca doso ca, māno makkho ca pātito;

* yassa = 「その人には／その人において」
* rāgo = 「愛欲・貪欲（ラーカ）」  
* ca = 「そして」
* doso = 「瞋恚・怒り（ドーサ）」
* ca = 「そして」
* māno = 「慢・我慢（マーナ）」
* makkho = 「嫉み・恩知らず・他人を軽んじる心（マッカ）」
* ca = 「そして」
* pātito = 「落とされた・捨て去られた・消え去った」

→「その人には、愛欲と怒りと、慢と嫉みとが、すでに捨て去られており、」

2. Sāsaporiva āraggā, tamahaṃ brūmi brāhmaṇanti.

* sāsapo = 「芥子の実」
* iva = 「…のように」
* āraggā = 「針先から・針から（ āragga＝針・その先端 ）」
* tam = 「その人を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ・言う」
* brāhmaṇaṃ = 「バラモン（真のバラモン・聖者）」
* nti = （引用の終わりを示す）

→「芥子の実が針先から（はじき落とされる）ように（落ちている）。その人を、私は『バラモン』と呼ぶのである。」

---

3. Tattha āraggāti

* tattha = 「そこで／その偈において」
* āraggā ti = 「『āraggā（針先から）』というのは」

→「そこで『āraggā（針先から）』とは、」

4. yassete rāgādayo kilesā,

* yassa = 「その人には／その人において」
* ete = 「これらの」
* rāga-ādayo = 「愛欲など（愛欲をはじめとする）」
* kilesā = 「煩悩たち」

→「その人には、これら愛欲などの煩悩と、」

5. ayañca paraguṇamakkhanalakkhaṇo makkho āraggā sāsapo viya pātito,

* ayaṃ ca = 「そしてこの」
* para-guṇa-makkhana-lakkhaṇo = 「他人の功徳を塗りつぶす（＝けなす）という性質をもった」
  * para = 「他人の」
  * guṇa = 「長所・功徳」
  * makkhanā = 「塗りつぶすこと・消し去ること」
  * lakkhaṇa = 「特徴・性質」
* makkho = 「マッカ（嫉み・恩知らず）」  
* āraggā = 「針先から」
* sāsapo viya = 「芥子の実のように」
* pātito = 「落ちている・振り落とされている」

→「また、他人の長所をけなす性質をもったマッカ（嫉み）という煩悩も、針先から振り落とされた芥子の実のように落ちていて、」

6. yathā sāsapo āragge na santiṭṭhati,

* yathā = 「ちょうど…であるように」
* sāsapo = 「芥子の実が」
* āragge = 「針先に」
* na santiṭṭhati = 「とどまらない・留まっていない」

→「ちょうど芥子の実が針先にとどまっていないように、」

7. evaṃ citte na santiṭṭhati,

* evaṃ = 「そのように・同様に」
* citte = 「（その人の）心に」
* na santiṭṭhati = 「とどまらない・留まっていない」

→「同様に、それらの煩悩はその人の心にとどまっていないのであり、」

8. tamahaṃ brāhmaṇaṃ vadāmīti attho.

* tam = 「そのような人を」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（真実のバラモン）」
* vadāmi = 「と呼ぶ・と言う」
* ti attho = 「という意味である」

→「そのような人を私はバラモンと呼ぶのである、という意味である。」

---

## 意訳

「その人には、愛欲も怒りも、慢心も、  
他人の善いところをけなすような嫉みも、  
すでにすっかり捨て去られている。  

それらの煩悩は、芥子の実が針先にとどまらず、  
すぐに落ちてしまうように、  
その人の心にはまったくとどまることがない。  

そのような人こそ、私は真のバラモンと呼ぶ。」

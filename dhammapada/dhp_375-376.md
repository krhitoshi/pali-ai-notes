375.

‘‘Tatrāyamādi bhavati, idha paññassa bhikkhuno;

Indriyagutti santuṭṭhi, pātimokkhe ca saṃvaro.

376.

‘‘Mitte bhajassu kalyāṇe, suddhājīve atandite;

Paṭisanthāravutyassa, ācārakusalo siyā;

Tato pāmojjabahulo, dukkhassantaṃ karissatī’’ti.

Tatrāyamādi bhavatīti tatra ayaṃ ādi, idaṃ pubbaṭṭhānaṃ hoti. Idha paññassāti imasmiṃ sāsane paṇḍitabhikkhuno. Idāni ‘‘taṃ ādī’’ti vuttaṃ pubbaṭṭhānaṃ dassento indriyaguttītiādimāha. Catupārisuddhisīlañhi pubbaṭṭhānaṃ nāma. Tattha indriyaguttīti indriyasaṃvaro. Santuṭṭhīti catupaccayasantoso. Tena ājīvapārisuddhi ceva paccayasannissitañca sīlaṃ kathitaṃ. Pātimokkheti pātimokkhasaṅkhāte jeṭṭhakasīle paripūrakāritā kathitā.

Mitte bhajassu kalyāṇeti vissaṭṭhakammante apatirūpasahāye vajjetvā sādhujīvitāya suddhājīve jaṅghabalaṃ nissāya jīvikakappanāya akusīte atandite kalyāṇamitte bhajassu, sevassūti attho. Paṭisanthāravutyassāti āmisapaṭisanthārena ca dhammapaṭisanthārena ca sampannavuttitāya paṭisanthāravutti assa, paṭisanthārassa kārakā bhaveyyāti attho. Ācārakusaloti sīlampi ācāro, vattapaṭivattampi ācāro. Tattha kusalo siyā, cheko bhaveyyāti attho. Tato pāmojjabahuloti tato paṭisanthāravuttito ca ācārakosallato ca uppannena dhammapāmojjena pāmojjabahulo hutvā taṃ sakalassāpi vaṭṭadukkhassa antaṃ karissatīti attho.

## 対訳

### 375偈

1. Tatrāyamādi bhavati, idha paññassa bhikkhuno;

- tatra = 「そこで／その場合に」
- ayaṃ = 「この～は」
- ādi = 「初め、始まり、端緒」
- bhavati = 「なる、ある」
  → tatrāyamādi bhavati = 「そこで、このような初め（端緒）がある」

- idha = 「ここにおいて、この教えにおいて」
- paññassa = 「知恵ある者の、賢い者の」（*pañña*「智慧」の属格）
- bhikkhuno = 「比丘の」

→「そこで、このような始め（端緒）がある。ここにおいて、賢い比丘には――」

2. Indriyagutti santuṭṭhi, pātimokkhe ca saṃvaro.

- indriya-gutti = 「諸根の守護、諸感官の防護」
  - indriya = 「（眼・耳・鼻・舌・身・意の）諸根」
  - gutti = 「守護、ガードすること」
- santuṭṭhi = 「満足、知足」
- pātimokkhe = 「波羅提木叉（律儀）において」
- ca = 「そして」
- saṃvaro = 「抑制、律儀、禁戒」

→「（まず）諸根の守護と知足、そして波羅提木叉における（戒の）抑制である。」

### 376偈

3. Mitte bhajassu kalyāṇe, suddhājīve atandite;

- mitte = 「友人たちを」（*mitta* の対格複数）
- bhajassu = 「交わりなさい、親しみなさい」（命令）
- kalyāṇe = 「善い、善良な、よき（人々である）」
- suddha-jīve = 「清浄な生活をする（者たち）、清らかな生活を送る友」
  - suddha = 「清浄な」
  - jīva = 「生活（する者）」
- atandite = 「怠らない、たるまない、精勤な」

→「（汚れた）仕事を離れ、清浄な生活を送り、怠りない善き友人たちと交わりなさい。」

4. Paṭisanthāravutyassa, ācārakusalo siyā;

- paṭisanthāra-vutti assa = 「もてなし（歓待）を行う性行があるように」
  - paṭisanthāra = 「接待、もてなし、歓待」
  - vutti = 「行い、あり方、性行」
  - assa = 「…であるように／…を備えているように」
- ācāra-kusalo = 「行儀に巧みな、礼儀作法に通じた」
  - ācāra = 「行儀、ふるまい、慣行」
  - kusala = 「巧みな、習熟した」
- siyā = 「…であるように／…であれ」（願望・命令）

→「物資のもてなしと法のもてなしにかなった行いを備え、行儀作法に通じた者であれ。」

5. Tato pāmojjabahulo, dukkhassantaṃ karissatī’’ti.

- tato = 「それによって、そののち、それらから」
  - ここでは「その接待の行いと行儀の巧みさによって」
- pāmojja-bahulo = 「喜びに満ちた、多くの歓喜をもつ」
  - pāmojja = 「歓喜、喜び」
  - bahulo = 「多い、満ちた」
- dukkhassa = 「苦しみの、苦の」
- antaṃ = 「終わり、尽きること」
- karissati = 「つくるであろう、成就するであろう」

→「そのことによって歓喜に満たされ、苦しみの終わりを実現するであろう。」

### 註釈部分

6. Tatrāyamādi bhavatīti tatra ayaṃ ādi, idaṃ pubbaṭṭhānaṃ hoti.

- tatrāyamādi bhavatīti = 「『tatrāyaṃ ādi bhavati』というのは」
- tatra = 「そこで、その場合に」
- ayaṃ ādi = 「この初め（この始まり）」
- idaṃ = 「この（ことは）」
- pubbaṭṭhānaṃ = 「前提、前置き、前提条件・先行条件」
- hoti = 「である」

→「『そこでこのような初めがある』とは、そこでこの初め、すなわちこれが前提条件（前置き）である、という意味である。」

7. Idha paññassāti imasmiṃ sāsane paṇḍitabhikkhuno.

- idha paññassāti = 「『idha paññassa』とは」
- imasmiṃ sāsane = 「この教え（仏法）の中で」
- paṇḍita-bhikkhuno = 「賢い比丘の」

→「『ここにおいて賢い者の』とは、この教えにおける賢い比丘の（ことである）。」

8. Idāni ‘‘taṃ ādī’’ti vuttaṃ pubbaṭṭhānaṃ dassento indriyaguttītiādimāha.

- idāni = 「さて今」
- ‘‘taṃ ādī’’ti vuttaṃ pubbaṭṭhānaṃ = 「『それが初めである』と言われた前提条件」
- dassento = 「示しつつ、明らかにしつつ」
- indriyagutti-ādiṃ āha = 「『indriyagutti』等（の偈）を説いた」
  - indriyagutti-ādi = 「indriyagutti から始まる（諸語）」

→「さて今、『それが初めである』と述べられた前提条件を示すために、『indriyagutti』等を説いたのである。」

9. Catupārisuddhisīlañhi pubbaṭṭhānaṃ nāma.

- catu-pārisuddhi-sīlaṃ hi = 「四種の波羅夷清浄（四波羅提清浄）という戒こそは」
  - catu = 「四つの」
  - pārisuddhi-sīla = 「完全に清浄な戒、四波羅提清浄戒
    （＝波羅提木叉・戒学における四種の清浄：pātimokkha-saṃvara, indriya-saṃvara,
      ājīva-pārisuddhi, paccaya-sannissita）」
- pubbaṭṭhānaṃ = 「前提、基盤」
- nāma = 「と呼ばれる」

→「四つの波羅夷清浄（四種清浄戒）こそが前提条件と呼ばれるのである。」

10. Tattha indriyaguttīti indriyasaṃvaro.

- tattha = 「そこで、その中で」
- indriyagutti = 「諸根の守護」
- indriya-saṃvaro = 「諸根の抑制、諸根に対する抑制律儀」

→「その中で、『indriyagutti』とは諸根の抑制のことである。」

11. Santuṭṭhīti catupaccayasantoso.

- santuṭṭhi = 「知足」
- catu-paccaya-santoso = 「四資具（衣・食・住・薬）についての満足・知足」

→「『santuṭṭhi』とは、四つの生活必需品に対する知足である。」

12. Tena ājīvapārisuddhi ceva paccayasannissitañca sīlaṃ kathitaṃ.

- tena = 「そのことによって、それをもって」
  - ここでは「santuṭṭhi（知足）の語によって」
- ājīva-pārisuddhi ca eva = 「生計の清浄という戒と」
  - ājīva = 「生活・生計・生業」
  - pārisuddhi = 「全き清浄」
- paccaya-sannissitañca sīlaṃ = 「諸資具に対する依止（の正しさ）に関する戒」
  - paccaya = 「生活資具（衣・食・住・薬）」
  - sannissita = 「依存のあり方」
- kathitaṃ = 「説き表された」

→「この語（知足）によって、生活手段の清浄と、諸資具への依り方に関する戒とが説き表されている。」

13. Pātimokkheti pātimokkhasaṅkhāte jeṭṭhakasīle paripūrakāritā kathitā.

- pātimokkheti = 「『pātimokkhe』とは」
- pātimokkha-saṅkhāte = 「波羅提木叉と呼ばれる」
- jeṭṭhaka-sīle = 「最上の戒・大戒（根本律儀）において」
- paripūra-kāritā = 「完全に守り行うこと、充実して実践すること」
- kathitā = 「説かれた」

→「『pātimokkhe』とは、波羅提木叉と名づけられる最上の戒を完全に守り行うことが説かれているのである。」

14. Mitte bhajassu kalyāṇeti vissaṭṭhakammante apatirūpasahāye vajjetvā

- mitte bhajassu kalyāṇe ti = 「『mitte bhajassu kalyāṇe』とは」
- vissaṭṭha-kammante = 「放逸な仕事、卑しい仕事・不適切な業に没頭する（人々）」

15. sādhujīvitāya suddhājīve jaṅghabalaṃ nissāya jīvikakappanāya akusīte atandite kalyāṇamitte bhajassu, sevassūti attho.

- sādhujīvitāya = 「善き生活のために、正しい生き方のために」
- suddha-jīve = 「清浄に生きる者たちを」
- jaṅgha-balaṃ nissāya = 「足の力に依って、徒歩托鉢（巡回）に依存して」
  - jaṅgha-bala = 「脚力」→比丘が托鉢などを歩行によって行うこと
- jīvika-kappanāya = 「生計を成り立たせるために」
- akusīte = 「怠惰でない」
- atandite = 「怠らない、精勤な」
- kalyāṇa-mitte = 「善き友人たちを」

- sevassu ti attho = 「『親しみなさい』という意味である」

→「善き生活を送るために、清浄な生活をいとなみ、足の力に依って（托鉢して）生計を立て、怠慢でなく精勤である善き友人たちと交わり、親しみなさい、という意味である。」

16. Paṭisanthāravutyassāti āmisapaṭisanthārena ca dhammapaṭisanthārena ca sampannavuttitāya paṭisanthāravutti assa, paṭisanthārassa kārakā bhaveyyāti attho.

- paṭisanthāra-vutti assāti = 「『paṭisanthāravutti assa』とは」
- āmi sa-paṭisanthārena ca = 「物質（衣・食など）のもてなしによっても」
- dhamma-paṭisanthārena ca = 「法（教え）のもてなしによっても」
- sampanna-vuttitāya = 「それらをそなえた行い（性行）によって」
- paṭisanthāra-vutti assa = 「もてなしの性行がその人にある」
- paṭisanthārassa kārakā bhaveyyā ti attho = 「もてなしを行う者であるべきである、という意味である」

→「『もてなしの行いを備えよ』とは、物資によるもてなしと、法によるもてなしの双方にすぐれた行いをもって、もてなしを行う性行を備え、もてなしを実行する者であるべきだ、という意味である。」

17. Ācārakusaloti sīlampi ācāro, vattapaṭivattampi ācāro.

- ācāra-kusalo ti = 「『ācārakusalo』とは」
- sīlaṃ pi ācāro = 「戒もまた『ācāra（行儀・行い）』であり」
- vatta-paṭivattaṃ pi ācāro = 「守るべき作法と、その返礼の作法もまた『ācāra（行儀）』である」
  - vatta = 「なすべき行い・作法」
  - paṭivatta = 「それに対する応答・返礼の作法」

→「『行儀に巧みな者』とは、戒も行儀であり、作法とその返礼もまた行儀である。」

18. Tattha kusalo siyā, cheko bhaveyyāti attho.

- tattha = 「その点において、それらのことにおいて」
- kusalo siyā = 「巧みであるべきである」
- cheko bhaveyyā ti attho = 「熟練しているべきだ、という意味である」

→「そのような行儀一切に通じ、熟練した者であるべきだ、という意味である。」

19. Tato pāmojjabahuloti tato paṭisanthāravuttito ca ācārakosallato ca uppannena dhammapāmojjena pāmojjabahulo hutvā

- tato = 「それによって、それから」
  - ここでは「もてなしの行いと、行儀の巧みさによって」
- pāmojja-bahulo ti = 「『pāmojjabahulo』とは」
- tato paṭisanthāra-vuttito ca = 「そのもてなしの行いによって、また」
- ācāra-kosallato ca = 「行儀に巧みであることによっても」
- uppannena dhamma-pāmojjena = 「そこから生じた法（正しい行い）による歓喜をもって」
- pāmojja-bahulo hutvā = 「歓喜に満ちた者となって」

→「『歓喜に満ちた者となり』とは、そのもてなしの行いと、行儀の巧みさから生じた法による歓喜をもって、歓喜に満ちた者となって、という意味である。」

20. taṃ sakalassāpi vaṭṭadukkhassa antaṃ karissatīti attho.

- taṃ = 「その者は」
- sakalassa pi vaṭṭa-dukkhassa = 「輪廻全体の一切の苦しみの」
  - vaṭṭa = 「輪廻の流転（煩悩・業・果の循環）」
  - dukkha = 「苦しみ」
- antaṃ = 「終わりを」
- karissati ti attho = 「終焉をもたらすであろう、という意味である」

→「その者は、輪廻全体の一切の苦しみの終わりを成就するであろう、という意味である。」

## 意訳

そこで、賢い比丘にとっての出発点が説かれる。

それは、まず諸根をよく守り、四つの生活必需品に対して知足であること、
そして波羅提木叉という最上の戒を、完全に守り行うことである。
この四種の清浄戒こそが、道の前提となる基礎なのである。

次にこう教える。

放逸な仕事にふける者や、ふさわしくない仲間を避けて、
清らかな生き方をし、歩いて托鉢して生計を立て、
怠りなく精進する善き友人たちと交わり、親しみなさい。

物質によるもてなしと、法によるもてなしの双方をよく実践して、
人を迎えもてなす行いを身につけ、
また、戒そのものも、種々の作法とその応対の作法も、
いずれにも通じて巧みであるようにしなさい。

そのような、もてなしの行いと行儀の巧みさから、
正しい法にもとづく歓喜が生じ、
その歓喜に満たされた者は、
ついには輪廻の一切の苦しみの終わりを実現するであろう。

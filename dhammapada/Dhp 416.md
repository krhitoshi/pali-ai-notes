416.

‘‘Yodha taṇhaṃ pahantvāna, anāgāro paribbaje;

Taṇhābhavaparikkhīṇaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tassattho – yo idha loke chadvārikaṃ taṇhaṃ vā mānaṃ vā jahitvā gharāvāsena anatthiko anāgāro hutvā paribbajati, taṇhāya ceva bhavassa ca parikkhīṇattā taṇhābhavaparikkhīṇaṃ tamahaṃ brāhmaṇaṃ vadāmīti.

## 対訳

1. ‘‘Yodha taṇhaṃ pahantvāna, anāgāro paribbaje;

* yo-dha = yo + dha  
  * yo = 「…である者」  
  * dha（強意の助辞）= 「まさに、その」  
  → 「よくぞそのような者は」→「…である者は」

* taṇhaṃ = 「渇愛を」  
  * taṇhā = 「渇愛、欲望、執着」

* pahantvāna = 「捨て去ってしまって、断ってしまって」  
  * pahantvā = 「捨て去って、離れて」（pajahati の絶対分詞）  
  * -na（強意）= 「まったく、十分に」

→ 「渇愛を捨て去ってしまい」

* anāgāro = 「家なき人、出家者」  
  * an-āgāra = 「住家（āgāra）のない」→「在家生活を捨てた者」

* paribbaje = 「遊行すべきである、遍歴すべきである」  
  * paribbajati（出家して托鉢遊行する）の願望・仮定法

→ 「家なき人として遊行すべきであり（＝出家遊行し）」

→ 全体：「渇愛を捨て去り、家なき人となって遊行する者は」

---

2. Taṇhābhavaparikkhīṇaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* taṇhā-bhava-parikkhīṇaṃ = 「渇愛と有（生存）が尽きはてた者を」  
  * taṇhā = 「渇愛」  
  * bhava = 「有（生存、存在）・三界のいずれかの生存状態」  
  * parikkhīṇa = 「すっかり尽きた、完全に断ち切られた」  
→ 「渇愛と有とが完全に尽きはてている者を」

* taṃ = 「その人を」  

* ahaṃ = 「私は」  

* brūmi = 「（～と）呼ぶ、言う」  

* brāhmaṇaṃ = 「バラモンである、と（呼ぶ）」  
  * ここではダンマ的な意味での「真の聖者」

→ 「その者を、私は『バラモン（真の聖者）』と呼ぶのである」

---

3. Tassattho –

* tassa attho = 「その（偈の）意味は」  

→ 「その意味は次のとおりである。」

---

4. yo idha loke chadvārikaṃ taṇhaṃ vā mānaṃ vā jahitvā

* yo = 「…である者は」  

* idha = 「ここ、この（現）世において」  

* loke = 「世において、この世の中で」

→ 「この世において」

* chadvārikaṃ = 「六つの門に属する、六門による」  
  * cha-dvāra = 「六つの門」＝「六処」（眼・耳・鼻・舌・身・意）  
  * chadvārikaṃ taṇhaṃ = 「六門を通じた渇愛」＝「六根門を通じて起こる渇愛」

* taṇhaṃ vā = 「渇愛であろうと」  

* mānaṃ vā = 「または慢（うぬぼれ・我慢）であろうと」  
  * māna = 「慢、我慢、優越感・劣等感・同等感の三種を含む『私は…である』という思い上がり」

* jahitvā = 「捨て去って、放棄して」  

→ 「この世において、六つの門から起こる渇愛であれ、または慢であれ、（それらを）捨て去って」

---

5. gharāvāsena anatthiko anāgāro hutvā paribbajati,

* gharāvāsena = 「家住みの生活によって、在家生活による」  
  * ghara = 「家」  
  * vāsena（vasa）= 「住むこと、生活」

* anatthiko = 「求めない者、欲しない者」  
  * an-atthika = 「利益・用事・欲求（attha）を求めない」

→ 「在家の家庭生活を欲しがらず」

* anāgāro hutvā = 「家なき人となって」  
  * hutvā = 「…となって、…になって」

* paribbajati = 「遊行する、出家遊行生活を送る」

→ 「在家生活を欲しがらず、家なき人となって遊行し」

---

6. taṇhāya ceva bhavassa ca parikkhīṇattā taṇhābhavaparikkhīṇaṃ

* taṇhāya ceva = 「渇愛がまさしく」  

* bhavassa ca = 「そして有（生存）がまた」  

* parikkhīṇattā = 「完全に尽きているがゆえに」  
  * parikkhīṇa-ttā（parikkhīṇatā の具格）= 「尽きてしまっているという状態によって」

→ 「渇愛と有とが完全に尽きているがゆえに」

* taṇhā-bhava-parikkhīṇaṃ = 「渇愛と有が尽きた者（である）」  

→ 「『渇愛と有とが尽きた者』であり」

---

7. tamahaṃ brāhmaṇaṃ vadāmīti.

* taṃ = 「その者を」  

* ahaṃ = 「私は」  

* brāhmaṇaṃ = 「バラモン（真の聖者）であると」  

* vadāmi = 「言う、呼ぶ」  

* ti = 引用標識  
* iti（省略）= 「…と（いう意味である）」

→ 「その者を私は『バラモン（真の聖者）』と呼ぶのである、という意味である。」

---

## 意訳

「この世において、六つの感官の門から起こる渇愛や慢心を捨て去り、  
在家の家庭生活を求めることなく、家なき出家者として遊行し、  
渇愛と生存への執着とをすっかり断ち尽くしている者、  
そのような『渇愛と有が完全に尽きた人』こそ、  
私は真のバラモン（聖者）と呼ぶのである。」

360.

‘‘Cakkhunā saṃvaro sādhu, sādhu sotena saṃvaro;
Ghānena saṃvaro sādhu, sādhu jivhāya saṃvaro.

Tattha cakkhunāti yadā hi bhikkhuno cakkhudvāre rūpārammaṇaṃ āpāthamāgacchati, tadā iṭṭhārammaṇe arajjantassa aniṭṭhārammaṇe adussantassa asamapekkhanena mohaṃ anuppādentassa tasmiṃ dvāre saṃvaro thakanaṃ pidahanaṃ gutti katā nāma hoti. Tassa so evarūpo cakkhunā saṃvaro sādhu. Esa nayo sotadvārādīsupi. Cakkhudvārādīsuyeva pana saṃvaro vā asaṃvaro vā nuppajjati, parato pana javanavīthiyaṃ esa labbhati. Tadā hi asaṃvaro uppajjanto assaddhā akkhanti kosajjaṃ muṭṭhasaccaṃ aññāṇanti akusalavīthiyaṃ ayaṃ pañcavidho labbhati. Saṃvaro uppajjanto saddhā khanti vīriyaṃ sati ñāṇanti kusalavīthiyaṃ ayaṃ pañcavidho labbhati.’’


偈文（360）の語句分解と文法

1) Cakkhunā saṃvaro sādhu
- cakkhunā = cakkhu（眼）/ 具格単数「眼によって・眼において（眼の門を通して）」
- saṃvaro = 「制御・防護・抑制（根の防護）」主語
- sādhu = 「善い・よい」述語（評価）
→「眼の（根の）防護は善い」

2) sādhu sotena saṃvaro
- sādhu = 「善い」
- sotena = sota（耳）/ 具格単数「耳によって」
- saṃvaro = 「防護」
→「耳の防護も善い」

3) Ghānena saṃvaro sādhu
- ghānena = ghāna（鼻）/ 具格単数
- saṃvaro = 「防護」
- sādhu = 「善い」
→「鼻の防護は善い」

4) sādhu jivhāya saṃvaro
- sādhu = 「善い」
- jivhāya = jivhā（舌）/ 具格単数
- saṃvaro = 「防護」
→「舌の防護も善い」


偈文（360）の逐語訳

「眼の防護は善い。耳の防護も善い。
鼻の防護は善い。舌の防護も善い。」


偈文（360）の自然な日本語訳

「眼についての自制はよい。耳についての自制もよい。
鼻についての自制はよい。舌についての自制もよい。」


註釈（Tattha〜）の語句分解と文法・内容説明（文ごと）

A) Tattha cakkhunāti ...
- Tattha = 「ここで（この偈において）」註釈の導入
- cakkhunā ti = 「『cakkhunā（眼によって）』とは〜」と語釈を立てる

B) yadā hi bhikkhuno cakkhudvāre rūpārammaṇaṃ āpāthamāgacchati,
- yadā ... tadā ... = 「〜するとき、〜そのとき」相関構文
- hi = 「実に・というのも」説明の強調
- bhikkhuno = bhikkhu（比丘）/ 属格単数「比丘の」
- cakkhudvāre = cakkhu-dvāra（眼の門）/ 処格単数「眼門において」
- rūpārammaṇaṃ = rūpa-ārammaṇa（色・形の対象）/ 主格（または対格的に扱われることもあるが、ここでは主語）「色の対象が」
- āpāthaṃ āgacchati = 「感官の射程（āpātha）に来る／現前する」
→「比丘の眼門に、色の対象が現前するとき」

C) tadā iṭṭhārammaṇe arajjantassa aniṭṭhārammaṇe adussantassa ...
- tadā = 「そのとき」
- iṭṭhārammaṇe = iṭṭha-ārammaṇa（好ましい対象）/ 処格単数「好ましい対象に対して」
- arajjantassa = a-rajjanta（貪らない）/ 現在分詞・属格単数「貪らない者の（＝その比丘が貪らず）」
- aniṭṭhārammaṇe = aniṭṭha-ārammaṇa（好ましくない対象）/ 処格単数「好ましくない対象に対して」
- adussantassa = a-dussanta（瞋らない）/ 現在分詞・属格単数「憎まない者の（＝その比丘が怒らず）」
→「好ましい対象に対して貪らず、好ましくない対象に対して瞋らない（その比丘が）」

D) asamapekkhanena mohaṃ anuppādentassa ...
- asamapekkhanena = asama-apekkhanā（偏った見方・不平等な観方）/ 具格単数「偏った見方によって」
  ※偏った見方は痴（moha）を生じさせやすいが、ここでは「それによって痴を起こさない」ことを言う
- mohaṃ = 「痴・迷妄」対格単数
- anuppādentassa = an-uppādenta（生じさせない）/ 現在分詞・属格単数
→「偏った見方によって痴を生じさせない（その比丘が）」

E) tasmiṃ dvāre saṃvaro thakanaṃ pidahanaṃ gutti katā nāma hoti.
- tasmiṃ dvāre = 「その門において」処格単数（＝眼の門）
- saṃvaro = 「防護・制御」
- thakanaṃ = 「塞ぐこと・遮ること」
- pidahanaṃ = 「閉じること・蓋をすること」
- gutti = 「護り・守護」
- katā nāma hoti = 「〜と名づけられる／〜と呼ばれる状態になる」
→「その門（眼門）における“防護”とは、塞ぎ・閉鎖・守護としてなされることだ、と言われる」

F) Tassa so evarūpo cakkhunā saṃvaro sādhu.
- tassa = 「その人にとって／その比丘の」属格
- so evarūpo = 「そのような性質の」
- cakkhunā saṃvaro = 「眼による防護（眼門の防護）」
- sādhu = 「善い」
→「このような眼の防護は、その比丘にとって善い」

G) Esa nayo sotadvārādīsupi.
- esa nayo = 「この方法（説明の仕方）は同様である」
- sota-dvāra-ādīsu pi = 「耳の門 आदि（鼻・舌など）の諸門においても」
→「同様の理屈が耳門などにも当てはまる」

H) Cakkhudvārādīsuyeva pana saṃvaro vā asaṃvaro vā nuppajjati, parato pana javanavīthiyaṃ esa labbhati.
- cakkhudvārādīsu yeva = 「眼門 आदिにおいてこそ」強調（yeva）
- pana = 「しかし」
- saṃvaro vā asaṃvaro vā = 「防護であれ不防護であれ」
- (n)uppajjati = 「（そこで）起こる／生起する」
- parato pana = 「その後に（さらに後段では）」
- javanavīthiyaṃ = javana-vīthi（速行の心路）/ 処格「速行心路において」
- esa labbhati = 「これは（よりはっきり）得られる／把握される」
→「防護・不防護は眼門などを契機として起こるが、その内実は後段の速行心路において把握される」

I) Tadā hi asaṃvaro uppajjanto assaddhā akkhanti kosajjaṃ muṭṭhasaccaṃ aññāṇanti akusalavīthiyaṃ ayaṃ pañcavidho labbhati.
- tadā hi = 「そのとき実に」
- asaṃvaro uppajjanto = 「不防護が生起するとき」
- assaddhā = 「不信（信の欠如）」
- akkhanti = 「不忍・不堪（忍耐の欠如）」
- kosajjaṃ = 「怠惰」
- muṭṭhasaccaṃ = 「念失（正念の喪失）」
- aññāṇaṃ = 「無智（智慧の欠如）」
- ti = 「〜という」
- akusalavīthiyaṃ = 「不善の心路において」処格
- ayaṃ pañcavidho labbhati = 「これは五種として見いだされる」
→「不防護が起こると、不善の心路においてそれは五つ――不信・不忍・怠惰・念失・無智――として現れる」

J) Saṃvaro uppajjanto saddhā khanti vīriyaṃ sati ñāṇanti kusalavīthiyaṃ ayaṃ pañcavidho labbhati.
- saṃvaro uppajjanto = 「防護が生起するとき」
- saddhā = 「信」
- khanti = 「忍（忍耐）」
- vīriyaṃ = 「精進」
- sati = 「念（正念）」
- ñāṇaṃ = 「智（智慧）」
- ti = 「〜という」
- kusalavīthiyaṃ = 「善の心路において」処格
- ayaṃ pañcavidho labbhati = 「これは五種として見いだされる」
→「防護が起こると、善の心路においてそれは五つ――信・忍・精進・念・智――として現れる」


註釈（Tattha〜）の逐語訳

「ここで『眼によって』とは――
実に、比丘の眼の門に色の対象が現前するとき、そのとき、
好ましい対象に貪らず、好ましくない対象に瞋らず、
偏った見方によって痴を生じさせない者にとって、
その門における防護は、塞ぎ・閉じ・守護としてなされたものだと呼ばれる。
そのような眼の防護は善い。耳の門などについても同様である。
しかし、防護または不防護は眼の門などを契機として起こるが、
（その内実は）後の速行心路において把握される。
そのとき不防護が生起すると、不善の心路において、
不信・不忍・怠惰・念失・無智という、この五種として見いだされる。
防護が生起すると、善の心路において、
信・忍・精進・念・智という、この五種として見いだされる。」


註釈（Tattha〜）の自然な日本語訳

「ここで言う『眼による（防護）』とはこういうことだ。
比丘の眼の門に色（目に映る対象）が入ってきたとき、
好ましいものに貪らず、好ましくないものに腹を立てず、
偏った見方から迷いを起こさないなら、
その眼の門は“塞ぎ、閉じ、守る”という意味で防護されていると言う。
このような眼の自制はよい。耳などの感官についても同様である。

ただし、防護・不防護は眼門などをきっかけに起こるが、
心の働きとしては後段の速行（javana）の流れの中で具体的に現れる。
不防護が起これば、不善の流れでは
信の欠如・忍耐の欠如・怠惰・念の喪失・無智という五つの形で現れ、
防護が起これば、善の流れでは
信・忍耐・精進・正念・智慧という五つの形で現れる。」

361.

‘‘Kāyena saṃvaro sādhu, sādhu vācāya saṃvaro;

Manasā saṃvaro sādhu, sādhu sabbattha saṃvaro;

Sabbattha saṃvuto bhikkhu, sabbadukkhā pamuccatī’’ti.

Kāyena saṃvaroti ettha pana pasādakāyopi copanakāyopi labbhati. Ubhayampi panetaṃ kāyadvārameva. Tattha pasādadvāre saṃvarāsaṃvaro kathitova. Copanadvārepi taṃvatthukā pāṇātipātaadinnādānakāmesumicchācārā. Tehi pana saddhiṃ akusalavīthiyaṃ uppajjantehi taṃ dvāraṃ asaṃvutaṃ hoti, kusalavīthiyaṃ uppajjantehi pāṇātipātāveramaṇiādīhi saṃvutaṃ. Sādhu vācāyāti etthāpi copanavācāpi vācā. Tāya saddhiṃ uppajjantehi musāvādādīhi taṃ dvāraṃ asaṃvutaṃ hoti, musāvādāveramaṇiādīhi saṃvutaṃ. Manasā saṃvaroti etthāpi javanamanato aññena manena saddhiṃ abhijjhādayo natthi. Manodvāre pana javanakkhaṇe uppajjamānehi abhijjhādīhi taṃ dvāraṃ asaṃvutaṃ hoti, anabhijjhādīhi saṃvutaṃ hoti. Sādhu sabbatthāti tesu cakkhudvārādīsu sabbesupi saṃvaro sādhu. Ettāvatā hi aṭṭha saṃvaradvārāni aṭṭha ca asaṃvaradvārāni kathitāni. Tesu aṭṭhasu asaṃvaradvāresu ṭhito bhikkhu sakalavaṭṭamūlakadukkhato na muccati, saṃvaradvāresu pana ṭhito sabbasmāpi vaṭṭamūlakadukkhā muccati. Tena vuttaṃ – ‘‘sabbattha saṃvuto bhikkhu, sabbadukkhā pamuccatī’’ti.


偈文（361）の語句分解と文法

1) Kāyena saṃvaro sādhu, sādhu vācāya saṃvaro;
- kāyena = kāya（身体）/ 具格単数「身によって・身において」
- saṃvaro = 「制御・防護」
- sādhu = 「善い」
- vācāya = vācā（言葉・言語）/ 具格単数「言葉によって」
→「身の防護は善い、言葉の防護も善い」

2) Manasā saṃvaro sādhu, sādhu sabbattha saṃvaro;
- manasā = mana（意・心）/ 具格単数
- sabbattha = 「あらゆる所で・すべての（門）において」
→「意の防護は善い。あらゆるところでの防護も善い」

3) Sabbattha saṃvuto bhikkhu, sabbadukkhā pamuccatī’’ti.
- sabbattha = 「あらゆる所で」
- saṃvuto = saṃvuta（防護された、制御された）/ 過去分詞 主格単数
- bhikkhu = 「比丘」主語
- sabba-dukkhā = 「あらゆる苦から」離格（複合語としての離格的用法）
- pamuccati = 「解放される・離れる」現在
- ti = 引用終止
→「あらゆる所で制御された比丘は、あらゆる苦から解放される」


偈文（361）の逐語訳

「身の防護は善い。言葉の防護も善い。
意の防護は善い。すべての所での防護も善い。
すべての所で防護された比丘は、あらゆる苦から解放される。」


偈文（361）の自然な日本語訳

「身体をよく制御するのはよい。言葉をよく制御するのもよい。
心をよく制御するのはよい。あらゆる面で制御が整っているのはさらによい。
あらゆる面で自制を保つ比丘は、あらゆる苦から解き放たれる。」


註釈（Kāyena saṃvaro ti〜）の語句分解と文法・内容説明（文ごと）

A) Kāyena saṃvaroti ettha pana pasādakāyopi copanakāyopi labbhati.
- kāyena saṃvaro ti = 「『身による防護』とは」
- ettha pana = 「ここではしかし」
- pasādakāyo pi = pasādakāya（浄・感官の身体＝五根側）もまた
- copanakāyo pi = copanakāya（動作する身体＝身業側）もまた
- labbhati = 「得られる／含意される」
→「ここで『身』とは、感官としての身体（パサーダ・カーヤ）も、動作としての身体（コーパナ・カーヤ）も含む」

B) Ubhayampi panetaṃ kāyadvārameva.
- ubhayaṃ pi = 「両方とも」
- pana = 「しかし」
- etaṃ = 「これは」
- kāya-dvāraṃ eva = 「身の門にほかならない」
→「しかし、そのどちらも結局は身門（kāyadvāra）である」

C) Tattha pasādadvāre saṃvarāsaṃvaro kathitova.
- tattha = 「そのうち」
- pasāda-dvāre = 「感官（パサーダ）の門において」
- saṃvara-asaṃvaro = 「防護と不防護」
- kathito eva = 「すでに語られた（既出）」
→「感官の門における防護・不防護は、すでに（前段で）説明済みである」

D) Copanadvārepi taṃvatthukā pāṇātipātaadinnādānakāmesumicchācārā.
- copana-dvāre pi = 「動作の門においても」
- taṃ-vatthukā = 「それ（＝身業）を対象・根拠とする」
- pāṇātipāta-adinnādāna-kāmesu micchācārā = 「殺生・不与取・欲における邪行（邪淫）」
→「動作の身門にも、それを基盤とする（身の）悪行として、殺生・盗み・邪淫がある」

E) Tehi pana saddhiṃ akusalavīthiyaṃ uppajjantehi taṃ dvāraṃ asaṃvutaṃ hoti,
- tehi saddhiṃ = 「それらとともに」
- akusalavīthiyaṃ = 「不善の心路において」
- uppajjantehi = 「生起するものによって（生起しつつある場合）」具格複数（文意上は随伴）
- taṃ dvāraṃ = 「その門（身門）は」
- asaṃvutaṃ hoti = 「不防護となる」
→「それら（殺生等）とともに不善の心路が起これば、その門は不防護である」

F) kusalavīthiyaṃ uppajjantehi pāṇātipātāveramaṇiādīhi saṃvutaṃ.
- kusalavīthiyaṃ = 「善の心路において」
- uppajjantehi = 「生起するものによって」
- pāṇātipātā-veramaṇi-ādīhi = 「殺生を離れること等（不殺生・不盗・不邪淫の戒）」具格複数
- saṃvutaṃ (hoti) = 「防護される」
→「善の心路が起こり、不殺生など（離れる戒）が伴えば、その門は防護されている」

G) Sādhu vācāyāti etthāpi copanavācāpi vācā.
- sādhu vācāyā ti = 「『言葉によって（の防護）は善い』とは」
- ettha api = 「ここでもまた」
- copanavācā pi = 「動作としての言葉（発語の働き）もまた」
- vācā (labbhati) = 「vācā（語）として含まれる」
→「『言葉』も、実際に発語として働く言葉（コーパナ・ヴァーチャー）を指す」

H) Tāya saddhiṃ uppajjantehi musāvādādīhi taṃ dvāraṃ asaṃvutaṃ hoti,
- tāya saddhiṃ = 「それ（言葉）とともに」
- uppajjantehi = 「生起するものによって」
- musāvāda-ādīhi = 「妄語など（四種の悪口：妄語・離間語・粗悪語・雑穢語などに代表される）」具格複数
- taṃ dvāraṃ = 「その門（語門）は」
- asaṃvutaṃ hoti = 「不防護である」
→「妄語などがともに生起すれば、語の門は不防護となる」

I) musāvādāveramaṇiādīhi saṃvutaṃ.
- musāvāda-veramaṇi-ādīhi = 「妄語を離れること等（正語の側）」具格複数
- saṃvutaṃ (hoti) = 「防護される」
→「妄語を離れること等が伴えば、語の門は防護されている」

J) Manasā saṃvaroti etthāpi javanamanato aññena manena saddhiṃ abhijjhādayo natthi.
- manasā saṃvaro ti = 「『意による防護』とは」
- ettha api = 「ここでもまた」
- javanamanato aññena manena = 「速行の意（javana-mana）以外の意とともには」
- saddhiṃ = 「ともに」
- abhijjhā-ādayo = 「貪欲（abhijjhā）など」
- natthi = 「ない」
→「ここで問題になる（不善の）貪欲などは、速行（javana）の心以外の“意”には伴わない」

K) Manodvāre pana javanakkhaṇe uppajjamānehi abhijjhādīhi taṃ dvāraṃ asaṃvutaṃ hoti,
- manodvāre pana = 「しかし意の門においては」
- javanakkhaṇe = 「速行の刹那において」
- uppajjamānehi = 「生起しているものによって」
- abhijjhā-ādīhi = 「貪欲などによって」
- taṃ dvāraṃ = 「その門（意門）は」
- asaṃvutaṃ hoti = 「不防護となる」
→「意門では、速行の瞬間に貪欲などが起これば、その門は不防護となる」

L) anabhijjhādīhi saṃvutaṃ hoti.
- an-abhijjhā-ādīhi = 「無貪欲など（貪欲がない等の善側）」具格複数
- saṃvutaṃ hoti = 「防護される」
→「無貪欲などが起これば、その門は防護されている」

M) Sādhu sabbatthāti tesu cakkhudvārādīsu sabbesupi saṃvaro sādhu.
- sādhu sabbatthā ti = 「『すべての所で（の防護は）善い』とは」
- tesu ... sabbesu pi = 「それら（眼門など）のすべてにおいても」
- saṃvaro sādhu = 「防護は善い」
→「眼門など、諸門すべてにおいて防護が善い、という意味」

N) Ettāvatā hi aṭṭha saṃvaradvārāni aṭṭha ca asaṃvaradvārāni kathitāni.
- ettāvatā = 「これだけで／この範囲で」
- hi = 「というのも」
- aṭṭha = 「八つ」
- saṃvara-dvārāni = 「防護の門」
- aṭṭha ca asaṃvara-dvārāni = 「不防護の門も八つ」
- kathitāni = 「説かれた」
→「ここまでで、防護の門八つと、不防護の門八つが説かれた」

O) Tesu aṭṭhasu asaṃvaradvāresu ṭhito bhikkhu sakalavaṭṭamūlakadukkhato na muccati,
- tesu aṭṭhasu = 「それら八つにおいて」
- asaṃvara-dvāresu = 「不防護の門に」
- ṭhito = 「立っている・留まっている（＝そこにいる、そこに依る）」過去分詞
- bhikkhu = 「比丘」
- sakala-vaṭṭa-mūlaka-dukkhato = 「輪廻を根とする全体の苦から」離格
- na muccati = 「解放されない」
→「不防護の八つの門に留まる比丘は、輪廻を根とする苦の全体から解放されない」

P) saṃvaradvāresu pana ṭhito sabbasmāpi vaṭṭamūlakadukkhā muccati.
- saṃvara-dvāresu pana = 「しかし防護の門においては」
- ṭhito = 「留まっている」
- sabbasmā pi = 「あらゆるものからも」
- vaṭṭa-mūlaka-dukkhā = 「輪廻を根とする苦から」
- muccati = 「解放される」
→「防護の門に留まる者は、輪廻を根とするあらゆる苦から解放される」

Q) Tena vuttaṃ – ‘‘sabbattha saṃvuto bhikkhu, sabbadukkhā pamuccatī’’ti.
- tena = 「それゆえに」
- vuttaṃ = 「言われた」
→「それゆえ、『あらゆる所で制御された比丘は、あらゆる苦から解放される』と言われたのである」


註釈（Kāyena saṃvaro〜）の逐語訳

「『身による防護』とは、ここでは感官としての身体（パサーダ・カーヤ）も、
動作する身体（コーパナ・カーヤ）も含まれる。しかし両者とも身門にほかならない。
そのうち感官の門における防護・不防護はすでに説かれた。
動作の門については、それを基盤とする殺生・不与取・欲における邪行がある。
それらとともに不善の心路が起こればその門は不防護となり、
善の心路が起こり不殺生など（離れる戒）が伴えば防護される。
『言葉によって（の防護は善い）』についても、動作する言葉が言葉である。
それとともに妄語などが起こればその門は不防護となり、
妄語を離れることなどが伴えば防護される。
『意による防護』についても、速行の意以外の意には貪欲などはない。
しかし意門では、速行の刹那に貪欲などが起これば不防護となり、
無貪欲などによって防護される。
『すべての所で（の防護は善い）』とは、眼門などすべての門で防護が善いということ。
ここまでで、防護の門八つと不防護の門八つが説かれた。
不防護の八つの門に留まる比丘は輪廻を根とする苦から解放されず、
防護の門に留まる者は輪廻を根とするあらゆる苦から解放される。
それゆえに『あらゆる所で制御された比丘は、あらゆる苦から解放される』と言われた。」


註釈（Kāyena saṃvaro〜）の自然な日本語訳

「ここでいう『身の制御』の“身”には二つが含まれる。
感官としての身体（五根の側）と、実際に行為を起こす身体である。
どちらも結局は身の門に属する。
感官の門における防護・不防護はすでに前で説明したので、ここでは行為の身門を述べる。
身の行為を土台とする悪は、殺生・盗み・邪淫である。
これらが不善の心の流れとともに起これば身門は“防護されていない”。
反対に、善の心の流れが起こり、不殺生などの戒（離れること）が伴えば“防護されている”。

『言葉の制御』も同様で、実際に発語する言葉が問題となる。
妄語などが起これば語門は不防護となり、妄語を離れるなどがあれば防護となる。

『意の制御』については、貪欲などの不善は速行（javana）の心で問題になる。
意門で速行の瞬間に貪欲などが起これば不防護であり、
無貪欲などが起これば防護である。

そして『あらゆるところでの制御がよい』とは、
眼門などすべての門で制御が整っているのがよい、という意味である。
ここまでで“防護の門”が八つ、“不防護の門”も八つ述べられた。
不防護の諸門にとどまる比丘は輪廻に根ざす苦から解放されないが、
防護の諸門にとどまる者は輪廻に根ざすあらゆる苦から解放される。
それゆえ偈に『あらゆる面で制御された比丘は、あらゆる苦から解き放たれる』と説かれたのである。」

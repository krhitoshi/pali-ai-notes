5.

‘‘Na hi verena verāni, sammantīdha kudācanaṃ;

Averena ca sammanti, esa dhammo sanantano’’ti.

Tattha na hi verenāti yathā hi kheḷasiṅghāṇikādīhi asucīhi makkhitaṃ ṭhānaṃ teheva asucīhi dhovantā suddhaṃ niggandhaṃ kātuṃ na sakkonti, atha kho taṃ ṭhānaṃ bhiyyosomattāya asuddhatarañceva duggandhatarañca hoti; evameva akkosantaṃ paccakkosanto paharantaṃ paṭipaharanto verena veraṃ vūpasametuṃ na sakkoti, atha kho bhiyyo bhiyyo verameva karoti. Iti verāni nāma verena kismiñci kāle na sammanti, atha kho vaḍḍhantiyeva. Averena ca sammantīti yathā pana tāni kheḷādīni asucīni vippasannena udakena dhoviyamānāni nassanti, taṃ ṭhānaṃ suddhaṃ hoti sugandhaṃ; evameva averena khantimettodakena yoniso manasikārena paccavekkhaṇena verāni vūpasammanti paṭippassambhanti abhāvaṃ gacchanti. Esa dhammo sanantanoti esa averena verūpasamanasaṅkhāto porāṇako dhammo; sabbesaṃ buddhapaccekabuddhakhīṇāsavānaṃ gatamaggoti.

# Gemini

## 対訳

1. ‘‘Na hi verena verāni, sammantīdha kudācanaṃ;

* na hi = 「決して～ない」（否定の強調）
* verena = 「怨みによって、憎しみによって」
* verāni = 「（複数の）怨みは、敵意は」
* sammantīdha = sammanti + idha
    * sammanti = 「鎮まる、止む」
    * idha = 「ここで、この世で」
* kudācanaṃ = 「いついかなる時も」

→ 「実にこの世において、怨みは怨みによって鎮まることは、いかなる時もない。」

2. Averena ca sammanti, esa dhammo sanantano’’ti.

* averena = 「怨みなきことによって、不怨によって」
* ca = 「一方、しかし」
* sammanti = 「鎮まる」
* esa = 「これは」
* dhammo = 「法（真理、法則）」
* sanantano = 「永遠の、古来の」
    * ti = （引用の終わり）

→ 「しかし、怨まないことによって（怨みは）鎮まる。これは永遠の（古来の）真理である。」

3. Tattha na hi verenāti

* tattha = 「その（詩句の）中で」
* na hi verena ti = 「『na hi verena』というのは（次のような意味である）」

→ 「そこで『na hi verena（決して怨みによっては～ない）』というのは、次のような意味である。」

4. yathā hi kheḷasiṅghāṇikādīhi asucīhi makkhitaṃ ṭhānaṃ teheva asucīhi dhovantā suddhaṃ niggandhaṃ kātuṃ na sakkonti,

* yathā hi = 「たとえば～のように」
* kheḷa-siṅghāṇika-ādīhi = 「唾や鼻水などの」
* asucīhi = 「不浄物によって」
* makkhitaṃ ṭhānaṃ = 「汚れた場所を」
* teheva (tehi eva) = 「まさにそれら（同じ不浄物）によって」
* asucīhi = 「不浄物によって」
* dhovantā = 「洗おうとする人々は」
* suddhaṃ = 「清浄に」
* niggandhaṃ = 「無臭に」
* kātuṃ na sakkonti = 「することはできない」

→ 「たとえば、唾や鼻水などの不浄物によって汚れた場所を、まさにそれらと同じ不浄物を使って洗おうとしても、清浄にすることも臭いをなくすこともできない。」

5. atha kho taṃ ṭhānaṃ bhiyyosomattāya asuddhatarañceva duggandhatarañca hoti;

* atha kho = 「それどころか、むしろ」
* taṃ ṭhānaṃ = 「その場所は」
* bhiyyosomattāya = 「いっそう甚だしく」
* asuddhatarañceva = asuddhataraṃ + ca + eva = 「さらに汚くなり、かつまた」
* duggandhatarañca = duggandhataraṃ + ca = 「さらに悪臭を放つようになる」
* hoti = 「～となる」

→ 「それどころか、その場所は、いっそう汚くなり、かつ悪臭がひどくなるだけである。」

6. evameva akkosantaṃ paccakkosanto paharantaṃ paṭipaharanto verena veraṃ vūpasametuṃ na sakkoti,

* evameva = 「まさにそのように」
* akkosantaṃ = 「（自分を）罵る者に対して」
* paccakkosanto = 「罵り返す者は」
* paharantaṃ = 「（自分を）打つ者に対して」
* paṭipaharanto = 「打ち返す者は」
* verena = 「怨み（報復）によって」
* veraṃ = 「怨みを」
* vūpasametuṃ na sakkoti = 「鎮めることはできない」

→ 「まさにそのように、罵る者を罵り返し、打つ者を打ち返す者は、怨みによって怨みを鎮めることはできない。」

7. atha kho bhiyyo bhiyyo verameva karoti. Iti verāni nāma verena kismiñci kāle na sammanti, atha kho vaḍḍhantiyeva.

* atha kho = 「それどころか」
* bhiyyo bhiyyo = 「ますます、いっそう」
* verameva karoti = 「ただ怨みを（新たに）作るだけである」
* iti = 「このように」
* verāni nāma = 「怨みというものは」
* verena = 「怨みによって」
* kismiñci kāle = 「いかなる時においても」
* na sammanti = 「鎮まらない」
* atha kho = 「かえって」
* vaḍḍhantiyeva = vaḍḍhanti + eva = 「ただ増大するだけである」

→ 「それどころか、ますます怨みを増やすだけである。このように、怨みというものは、怨み（報復）によってはいつまでも鎮まることなく、かえって増大するばかりである。」

8. Averena ca sammantīti yathā pana tāni kheḷādīni asucīni vippasannena udakena dhoviyamānāni nassanti, taṃ ṭhānaṃ suddhaṃ hoti sugandhaṃ;

* averena ca sammanti ti = 「『averena ca sammanti（不怨によって鎮まる）』というのは」
* yathā pana = 「一方、たとえば～のように」
* tāni kheḷādīni asucīni = 「それらの唾などの不浄は」
* vippasannena udakena = 「澄んだ水によって」
* dhoviyamānāni = 「洗い流されると」
* nassanti = 「消滅する」
* taṃ ṭhānaṃ = 「その場所は」
* suddhaṃ hoti = 「清浄になり」
* sugandhaṃ = 「良い香り（清潔）となる」

→ 「『不怨によって鎮まる』というのは、たとえばそれらの唾などの汚れも、清らかな水によって洗い流されれば消滅し、その場所は清浄となり、清潔になるように（という意味である）。」

9. evameva averena khantimettodakena yoniso manasikārena paccavekkhaṇena verāni vūpasammanti paṭippassambhanti abhāvaṃ gacchanti.

* evameva = 「まさにそのように」
* averena = 「不怨によって（すなわち）」
* khanti-metta-udakena = 「忍辱（耐え忍び）と慈しみという水によって」
* yoniso manasikārena = 「理に適った作意（正しい注意・洞察）によって」
* paccavekkhaṇena = 「（自身の行為や心の）観察・反省によって」
* verāni = 「怨みは」
* vūpasammanti = 「鎮まり」
* paṭippassambhanti = 「静まり返り」
* abhāvaṃ gacchanti = 「無くなる（消滅に至る）」

→ 「まさにそのように、『不怨』すなわち忍耐と慈愛という水、そして理に適った正しい思惟と観察によって、怨みは鎮まり、静止し、消滅に至るのである。」

10. Esa dhammo sanantanoti esa averena verūpasamanasaṅkhāto porāṇako dhammo;

* esa dhammo sanantano ti = 「『これは古来の法である』というのは」
* esa = 「この」
* averena vera-upasamana-saṅkhāto = 「不怨によって怨みを鎮めることと呼ばれる」
* porāṇako dhammo = 「古来の（昔からの）法、伝統」

→ 「『これは古来の法である』というのは、この『怨みを持たないことによって怨みを鎮めること』こそが、古来からの法である（という意味である）。」

11. sabbesaṃ buddhapaccekabuddhakhīṇāsavānaṃ gatamaggoti.

* sabbesaṃ = 「すべての」
* buddha-paccekabuddha-khīṇāsavānaṃ = 「仏（正等覚者）、独覚（縁覚）、漏尽者（阿羅漢）たちの」
* gata-maggo ti = 「歩まれた道である、ということである」

→ 「それは、すべての仏たち、独覚たち、そして煩悩を滅尽した阿羅漢たちが歩まれた道である、という意味である。」

## 意訳

「実にこの世において、怨みが怨み（報復）によって鎮まることは絶対にない。
しかし、怨みを抱かないことによって、怨みは鎮まる。
これは永遠の真理（古来からの法）である。」

註釈によると、その意味は以下の通りである。

「決して怨みによっては～ない」というのは、たとえば唾や鼻水などの汚物で汚れた場所を、同じ汚物を使って洗おうとしても、きれいにすることも悪臭を消すこともできないのと同様である。むしろその場所は、さらに汚れ、悪臭がひどくなるだけである。
それと同じように、罵られたからといって罵り返し、打たれたからといって打ち返すならば、怨みをもって怨みを鎮めることはできない。それどころか、ますます怨みを作り出すだけである。このように、怨みというものは、怨み（報復）によってはいつまでも鎮まることなく、かえって増大するばかりである。

一方、「不怨によって鎮まる」というのは、汚れた場所も清らかな水で洗えば汚れが落ち、その場所が清潔になるようなものである。
まさにそのように、「不怨」すなわち忍耐と慈しみという水、そして理に適った正しい思惟と観察によってこそ、怨みは鎮まり、消え去り、無くなるのである。

「これは古来の法である」というのは、この「怨まないことによって怨みを鎮める」という在り方こそが、古来からの正しい法（伝統）であり、すべての仏陀、独覚、阿羅漢たちが歩まれた道である、という意味である。

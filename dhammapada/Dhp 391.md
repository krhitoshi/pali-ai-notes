391.

‘‘Yassa kāyena vācāya, manasā natthi dukkaṭaṃ;

Saṃvutaṃ tīhi ṭhānehi, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha dukkaṭanti sāvajjaṃ dukkhudrayaṃ apāyasaṃvattanikaṃ kammaṃ. Tīhi ṭhānehīti etehi kāyādīhi tīhi kāraṇehi kāyaduccaritādipavesanivāraṇatthāya dvāraṃ pihitaṃ, taṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Yassa kāyena vācāya, manasā natthi dukkaṭaṃ;

* yassa = 「その人には、その者には」（関係代名詞「ya」の属格・単数）  
* kāyena = 「身によって、身体で」（kāya「身」の具格・単数）  
* vācāya = 「言葉によって、言語で」（vācā「言葉」の具格・単数）  
* manasā = 「心によって」（manas「心」の具格・単数）  
* natthi = 「無い」  
* dukkaṭaṃ = 「悪い行為、罪となる行為」

→「その者には、身によっても、言葉によっても、心によっても、悪い行為が無い者」

---

2. Saṃvutaṃ tīhi ṭhānehi, tamahaṃ brūmi brāhmaṇa’’nti.

* saṃvutaṃ = 「抑制された、制御された、よく守られた（saṃvara の過去分詞）」  
* tīhi = 「三つのもので、三つの点において」（数詞「ti」の具格・複数）  
* ṭhānehi = 「基盤・方面・点・状態において」（ṭhāna「場所・状態」の具格・複数）  
　→ tīhi ṭhānehi = 「三つの方面・三つの基盤（＝身・口・意）において」  
* taṃ = 「その者を」  
* ahaṃ = 「私は」  
* brūmi = 「（…と）言う、呼ぶ」  
* brāhmaṇaṃ = 「バラモンであると、聖なるバラモンと」（brāhmaṇa「バラモン」の対格・単数）  

→「三つの方面において抑制されているその者を、私は『バラモン』と呼ぶのである。」

---

3. Tattha dukkaṭanti sāvajjaṃ dukkhudrayaṃ apāyasaṃvattanikaṃ kammaṃ.

* tattha = 「ここで、この句において」  
* dukkaṭaṃ ti = 「『dukkaṭaṃ』という（語は）」  
* sāvajjaṃ = 「非難されるべき、咎ある」  
* dukkhudrayaṃ = 「苦のもととなるもの、苦を生み出すもの」  
　* dukkha + udraya（udaya）「苦の生起・苦の源」  
* apāyasaṃvattanikaṃ = 「悪趣（四悪趣）にいたらしめる、悪趣に趣く結果をもたらす」  
　* apāya「悪趣」＋ saṃvattanika「もたらす・赴かせる」  
* kammaṃ = 「行為、業」

→「ここで『dukkaṭa（悪行）』とは、非難されるべきもので、苦の原因となり、悪趣に赴かせるような行為をいう。」

---

4. Tīhi ṭhānehīti etehi kāyādīhi tīhi kāraṇehi kāyaduccaritādipavesanivāraṇatthāya dvāraṃ pihitaṃ, taṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* tīhi ṭhānehi ti = 「『三つの方面において』というのは」  
* etehi = 「これらの」  
* kāyādīhi = 「身などによって（＝身・口・意という）」  
　* kāya「身」+ ādi「など」＋複数具格  
* tīhi kāraṇehi = 「三つの原因・三つの要因によって」  
　→ すなわち「身・口・意という三つの原因によって」  
* kāyaduccaritādi = 「身の悪行など（身・口・意の不善行一切）」  
　* kāya-duccarita「身の悪行」＋ ādi「等」  
* pavesa-nivāraṇa-atthāya = 「入り込むことを防ぐために」  
　* pavesa「侵入・入り込み」  
　* nivāraṇa「防止・遮断」  
　* atthāya（atthaṃ + aya）「…のために」  
* dvāraṃ = 「門、門戸、出入り口」（ここでは身・口・意の「門」）  
* pihitaṃ = 「閉ざされている、塞がれている」  
* taṃ = 「そのような者を」  
* ahaṃ = 「私は」  
* brāhmaṇaṃ = 「バラモンであると」  
* vadāmi ti attho = 「と言うのである、という意味である」

→「『三つの方面において』とは、ここでは、この身など三つの原因（＝身・口・意）によって、身の悪行などが入り込むことを防ぐために、その門が閉ざされている（＝三つの門がよく守られている）者を指すのである。そのような者を私は『バラモン』と言う、という意味である。」

---

## 意訳

「その人には、身によっても、言葉によっても、心によっても、いかなる悪行もない。  
身・口・意という三つの面でよく制御され、門を固く閉ざして悪い行いが入り込む余地のない者、  
その人こそを、私はまことの『バラモン』と呼ぶのである。」

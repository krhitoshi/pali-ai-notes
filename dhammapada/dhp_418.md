418.

‘‘Hitvā ratiñca aratiñca, sītibhūtaṃ nirūpadhiṃ;

Sabbalokābhibhuṃ vīraṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha ratinti pañcakāmaguṇaratiṃ. Aratinti araññavāse ukkaṇṭhitattaṃ. Sītibhūtanti nibbutaṃ. Nirūpadhinti nirupakkilesaṃ. Vīranti taṃ evarūpaṃ sabbaṃ khandhalokaṃ abhibhavitvā ṭhitaṃ vīriyavantaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Hitvā ratiñca aratiñca,

* hitvā = 「捨てて、離れて（捨て去って）」〔動詞 jahati「捨てる」の絶対分詞〕  
* ratiṃ ca = 「ラティ（快楽・愛着）をも」  
* aratiṃ ca = 「アラティ（不快・嫌悪・倦怠）をも」  

→「快楽への愛着も、嫌悪・倦怠も、ともに捨て去って」

---

2. sītibhūtaṃ nirūpadhiṃ;

* sītibhūtaṃ = 「冷却した、冷めきった、涼しくなった」  
  * sīti = 「冷たさ、涼しさ」＋ bhūta（bhavati の過去分詞）  
  → 「冷めきった、熱が完全に去った」＝「煩悩の火が消えた」  
* nirūpadhiṃ = 「ウパディ（所依・基礎）がない」  
  * niru-/nir- = 「～のない、離れた」  
  * upadhi = 「（再生・苦の）基礎、所依」「所持・所有」「煩悩の貯蔵」  
  → 「一切のウパディ（煩悩・有の基・所有）をもたない者」

→「すっかり冷めきり、いかなるウパディ（煩悩・有の基）ももたない者を」

---

3. Sabbalokābhibhuṃ vīraṃ,

* sabba-loka-abhibhuṃ = 「すべてのローカ（世界）を、凌駕した者・超克した者」  
  * sabba = 「一切の、すべての」  
  * loka = 「世界、世間」  
  * abhibhu(ṃ) = 「打ち勝った者、征服した者」  
→ 「一切の世間を超克した者」  
* vīraṃ = 「勇者を、力強き者を」  
  * vīra = 「勇ましい者、英雄、精進ある者」

→「一切の世間を超え征服した勇者を」

---

4. tamahaṃ brūmi brāhmaṇanti.

* taṃ = 「そのような者を」  
* ahaṃ = 「私は」  
* brūmi = 「言う、呼ぶ」  
* brāhmaṇaṃ = 「バラモンと、真のバラモンと」  
* ti = 引用マーカー  
* nti = 終止の語尾（「～と〔私は〕言う」との意）

→「その人こそを、私は『バラモン』と呼ぶのである。」

---

### 註釈文

5. Tattha ratinti pañcakāmaguṇaratiṃ.

* tattha = 「ここで、この句において」  
* ratiṃ ti = 「rati という語は」  
* pañca-kāma-guṇa-ratiṃ = 「五つの欲楽の対象に対する楽しみ」  
  * pañca = 「五つの」  
  * kāma-guṇa = 「欲の要素・欲の対象（色・声・香・味・触の五欲）」
  * rati = 「楽しみ、歓楽、愛着」

→「ここで『rati（快楽への愛着）』とは、五欲の対象への楽しみ（五欲楽の喜び）をいう。」

---

6. Aratinti araññavāse ukkaṇṭhitattaṃ.

* aratiṃ ti = 「arati という語は」  
* arañña-vāse = 「林・森での住まいにおいて、林住において」  
  * arañña = 「林、森、荒野」  
  * vāsa = 「住むこと、居住」  
* ukkaṇṭhitattaṃ = 「うんざりすること、倦怠・いや気がさすこと」  

→「『arati（不快・倦怠）』とは、林住生活に対していや気がさしている状態をいう。」

---

7. Sītibhūtanti nibbutaṃ.

* sītibhūtaṃ ti = 「sītibhūtaṃ 〔冷却した〕というのは」  
* nibbutaṃ = 「ニッブータ、すなわち涅槃に達した、煩悩が消滅した状態」  

→「『sītibhūtaṃ（冷めきった）』とは、すなわち『nibbutaṃ（寂静＝涅槃に達した）』ということである。」

---

8. Nirūpadhinti nirupakkilesaṃ.

* nirūpadhiṃ ti = 「nirūpadhiṃ というのは」  
* nir-upakkilesaṃ = 「汚れ（煩悩）がまったくない」  
  * upakkilesa = 「汚れ、混濁、煩悩」  
  * nir- = 「～のない、離れた」

→「『nirūpadhiṃ（ウパディなき者）』とは、すなわち『nirupakkilesaṃ（いかなる煩悩の汚れもない）』という意味である。」

---

9. Vīranti taṃ evarūpaṃ sabbaṃ khandhalokaṃ abhibhavitvā ṭhitaṃ vīriyavantaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* vīraṃ ti = 「vīra というのは」  
* taṃ = 「その者を、かの人を」  
* eva-rūpaṃ = 「このようなありさまの」  
  →「すなわち、上に説かれたような性質をそなえた」  
* sabbaṃ khandha-lokaṃ = 「一切の蘊（かたまり）の世界を」  
  * khandha = 「（五）蘊：色・受・想・行・識」  
  * loka = 「世間・世界」  
* abhibhavitvā = 「打ち克ち、超え、征服して」〔abhibhavati の絶対分詞〕  
* ṭhitaṃ = 「とどまっている、確立している」  
* vīriya-vantaṃ = 「精進ある者、エネルギーに満ちた者」  
* ahaṃ = 「私は」  
* brāhmaṇaṃ = 「（その者を）バラモンと」  
* vadāmi ti attho = 「呼ぶのである、という意味である」

→「『vīra（勇者）』とは、かくのごとき者――このようにして一切の五蘊の世間を征服し、そこを超えて確立している、精進力に富む人――この人をこそ、私は『バラモン』と呼ぶのである、という意味である。」

---

## 意訳

五欲の楽しみにふけることも、林の中の修行生活にうんざりすることも、ともに捨て去り、  
煩悩の火がすっかり鎮まりきって、いかなる「所有」や「再生の基」となるものも持たず、  
この五蘊の世間すべてを超え征服して、そこに動じることなく確立している精進の勇者──  
その人こそを、私は「真のバラモン」と呼ぶ。

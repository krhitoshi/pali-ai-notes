385.

‘‘Yassa pāraṃ apāraṃ vā, pārāpāraṃ na vijjati;

Vītaddaraṃ visaṃyuttaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha pāranti ajjhattikāni cha āyatanāni. Apāranti bāhirāni cha āyatanāni. Pārāpāranti tadubhayaṃ. Na vijjatīti yassa sabbampetaṃ ‘‘aha’’nti vā ‘‘mama’’nti vā gahaṇābhāvena natthi, taṃ kilesadarathānaṃ vigamena vītaddaraṃ sabbakilesehi visaṃyuttaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Yassa pāraṃ apāraṃ vā, pārāpāraṃ na vijjati;

* yassa = 「その人には／～する者には」
* pāraṃ = 「かなた（pāra）＝内なる岸、内側」※註で「内の六処」と解釈
* apāraṃ = 「こなた（a‑pāra）＝外なる岸、外側」※註で「外の六処」と解釈
* vā = 「あるいは」
* pārāpāraṃ = 「かなたとこなたの両方（pāra + apāra）」
* na vijjati = 「存在しない・見出されない」

→「その人には、内なる岸も外なる岸も、あるいはその両方も、存在しない。」

2. Vītaddaraṃ visaṃyuttaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* vītaddaraṃ = 「苦労（daratha）、わずらいを離れた（vīta + daratha）」＝「悩みのない」
* visaṃyuttaṃ = 「（…から）完全に離れた・切り離された」
* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「言う・呼ぶ」
* brāhmaṇaṃ = 「バラモン（真の聖者）」
* ti = （引用終わりを示す）
* nti = 「と」言った、の意の延長

→「煩いなく、完全に離れきったそのような者を、私は『バラモン（真の聖者）』と呼ぶのである。」

---

3. Tattha pāranti ajjhattikāni cha āyatanāni.

* tattha = 「ここで、この句において」
* pāraṃ ti = 「『pāraṃ』とは」
* ajjhattikāni = 「内なる・内側の」
* cha = 「六つの」
* āyatanāni = 「処（āyatana）、基盤、領域（六内処＝眼・耳・鼻・舌・身・意）」

→「ここで『pāraṃ（かなた）』とは、内なる六つの処（六内処）をいう。」

4. Apāranti bāhirāni cha āyatanāni.

* apāraṃ ti = 「『apāraṃ』とは」
* bāhirāni = 「外なる・外側の」
* cha = 「六つの」
* āyatanāni = 「処（六外処＝色・声・香・味・触・法）」

→「『apāraṃ（こなた）』とは、外なる六つの処（六外処）をいう。」

5. Pārāpāranti tadubhayaṃ.

* pārāpāraṃ ti = 「『pārāpāraṃ』とは」
* tad‑ubhayaṃ = 「その両方、すなわちそれら二つとも」

→「『pārāpāraṃ』とは、その両方（内外二種の六処）をいう。」

6. Na vijjatīti yassa sabbampetaṃ ‘‘aha’’nti vā ‘‘mama’’nti vā gahaṇābhāvena natthi,

* na vijjati ti = 「『na vijjati』とは」
* yassa = 「その人には／～する者には」
* sabbaṃ etaṃ = 「これら一切（前述の内外両六処）」
* “aha”nti vā = 「『私はこれだ（ahaṃ）』と、であるとか」
* “mama”nti vā = 「『これは私のものだ（mama）』と、であるとか」
* gahaṇa‑abhāvena = 「つかみ取ること（gahaṇa）が無いことによって、取著の欠如により」
* natthi = 「存在しない」

→「『存在しない』とは、これら一切を『これは私だ』とか『これは私のものだ』とかと執って取ることがないがゆえに、その人にはまったく存在しない、ということである。」

7. taṃ kilesadarathānaṃ vigamena vītaddaraṃ sabbakilesehi visaṃyuttaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「そのような人を」
* kilesa‑darathānaṃ = 「煩悩（kilesa）による疲労・わずらい（daratha）」
* vigamena = 「去ること・消滅によって」
* vītaddaraṃ = 「わずらいから離れた者、苦労なき者」
* sabba‑kilesehi = 「あらゆる煩悩から」
* visaṃyuttaṃ = 「完全に離れた、切り離された」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（真の聖者）」
* vadāmi ti = 「と説く／と呼ぶ」
* attho = 「という意味である」

→「そのように、煩悩によるあらゆる疲労・わずらいが滅したゆえに悩みなく、すべての煩悩から完全に離れきった人を、私は『バラモン（真の聖者）』と呼ぶのである、という意味である。」

---

## 意訳

「内なる六つの感官の領域にも、外なる六つの対象の領域にも、
またその両方にも、いささかの『私はこれだ』『これは私のものだ』という
執らわれが、まったく見いだされない人がいる。

その人は、煩悩によるあらゆる疲れとわずらいが消え去って、
一切の煩悩から完全に解き放たれている。

私は、そのような人こそ真の『バラモン（清らかな聖者）』であると説く。」

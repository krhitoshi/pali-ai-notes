364.

‘‘Dhammārāmo dhammarato, dhammaṃ anuvicintayaṃ;

Dhammaṃ anussaraṃ bhikkhu, saddhammā na parihāyatī’’ti.

Tattha nivāsanaṭṭhena samathavipassanādhammo ārāmo assāti dhammārāmo. Tasmiṃyeva dhamme ratoti dhammarato. Tasseva dhammassa punappunaṃ vicintanatāya dhammaṃ anuvicintayaṃ, taṃ dhammaṃ āvajjento manasikarontoti attho. Anussaranti tameva dhammaṃ anussaranto. Saddhammāti evarūpo bhikkhu sattatiṃsabhedā bodhipakkhiyadhammā navavidhalokuttaradhammā ca na parihāyatīti attho.

## 対訳

1. ‘‘Dhammārāmo dhammarato, dhammaṃ anuvicintayaṃ;

* dhammārāmo = 「dhamma（法）＋ ārāma（園・憩いの場・住処）」  
  → 「法を安住の場・住処とする者」

* dhammarato = 「dhamma（法）＋ rato（喜ぶ・楽しむ）」  
  → 「法を喜ぶ者・法を楽しむ者」

* dhammaṃ = 「法を」

* anuvicintayaṃ = 「anu（繰り返し・後から）＋ vicintayaṃ（考察している・思惟している）」  
  → 「法を繰り返し思惟している（者）」

→「法を安住の場とし、法を楽しみ、法を繰り返し思惟している者は……」

2. Dhammaṃ anussaraṃ bhikkhu, saddhammā na parihāyatī’’ti.

* dhammaṃ = 「法を」

* anussaraṃ = 「anu（たびたび・後から）＋ saraṃ（憶念している）」
  → 「たえず憶念している・思い起こしている（者）」

* bhikkhu = 「比丘は」

* saddhammā = 「真実なる法・正しい法は」  
  ※註：ここでは後に説明される「三十七菩提分法」と「九種の出世間法」を指す。

* na = 「…しない」

* parihāyatī = 「衰える・失われる・退転する」

→「法を憶念している比丘において、真実の法は退転することがないのである。」

---

3. Tattha nivāsanaṭṭhena samathavipassanādhammo ārāmo assāti dhammārāmo.

* tattha = 「ここで、この偈の中で」

* nivāsana-ṭṭhena = 「nivāsana（住すること・住処）＋ attha（意味）」  
  → 「住処とするという意味において」

* samatha-vipassanā-dhammo = 「止（samatha）と観（vipassanā）の法」  
  → 「止観の法」

* ārāmo assa = 「ārāmo（園・憩いの場・住処）が assa（彼のものである・彼にとってのものである）」  
  → 「彼にとっての住処・憩いの場である」

* ti = 「…と」

* dhammārāmo = 「“dhammārāmo”という（語）」  

→「ここで、『dhammārāmo（法を安住の場とする者）』とは、  
　住処とするという意味において、止観の法を憩いの場・住処とする者、ということである。」

4. Tasmiṃyeva dhamme ratoti dhammarato.

* tasmiṃ eva = 「まさにその（= 先の）」

* dhamme = 「法において」

* rato ti = 「rato（喜ぶ・楽しむ者）＋ ti（…という）」  
  → 「喜ぶ者であるということで」

* dhammarato = 「“dhammarato”という（語）」  

→「その同じ法を楽しむ者であるから、“dhammarato（法を楽しむ者）”というのである。」

5. Tasseva dhammassa punappunaṃ vicintanatāya dhammaṃ anuvicintayaṃ, taṃ dhammaṃ āvajjento manasikarontoti attho.

* tassa eva = 「まさにその（同じ）」

* dhammassa = 「法を」

* punappunaṃ = 「何度も何度も、繰り返し」

* vicintanatāya = 「vicintana（思惟・考察）＋ tā（…という状態・性質）」  
  → 「繰り返し思惟するというあり方によって」

* dhammaṃ anuvicintayaṃ = 「法を繰り返し考察している者（= anuvicintayaṃ）」  

* taṃ dhammaṃ = 「その法を」

* āvajjento = 「心を向けている・念頭に置いている」

* manasikaronto = 「注意を向けている・意にとめている」

* ti attho = 「という意味である」

→「『dhammaṃ anuvicintayaṃ（法を繰り返し思惟する者）』とは、  
　その同じ法を何度も何度も思惟するというあり方によって、  
　その法に心を向け、意にとめている者である、という意味である。」

6. Anussaranti tameva dhammaṃ anussaranto.

* anussaraṃ ti = 「anussaranto（憶念している者）というのは」

* taṃ eva dhammaṃ = 「まさにその同じ法を」

* anussaranto = 「たえず憶念している者」

→「『anussaraṃ（憶念している者）』とは、  
　その同じ法をたえず憶念している者、という意味である。」

7. Saddhammāti evarūpo bhikkhu sattatiṃsabhedā bodhipakkhiyadhammā navavidhalokuttaradhammā ca na parihāyatīti attho.

* saddhammā ti = 「“saddhammā（真実の法）”とは」

* evarūpo bhikkhu = 「このような（= 上述のような）比丘において」

* sattatiṃsa-bhedā = 「sattatiṃsa（37）＋ bhedā（種類に分けられた）」  
  → 「三十七に分類される」

* bodhipakkhiya-dhammā = 「菩提に属する法・菩提分法」  
  → 「三十七菩提分法（覚りに資する三十七の法）」

* nava-vidha-lokuttara-dhammā =  
  * nava-vidha = 「九種の」  
  * lokuttara-dhammā = 「出世間の法」  
    → 「九種の出世間法（四道・四果・涅槃）」

* ca = 「および」

* na parihāyati = 「衰えない・減少しない・退転しない」

* ti attho = 「という意味である」

→「『saddhammā（真実の法）』とは、  
　このような比丘においては、三十七種類に分かれる菩提分法と、  
　九種の出世間法とが、衰えることがない、という意味である。」

---

## 意訳

「法を安住の場とし、法を楽しみ、法を繰り返し思惟し、  
その法をたえず憶念している比丘においては、  
真実の法が退転することはない。」

すなわち、この偈において「法を安住の場とする者（dhammārāmo）」とは、  
止と観という法を、自らの住処・安住の境地としている者をいう。  
「法を楽しむ者（dhammarato）」とは、その同じ法を喜び楽しむ者である。  
「法を繰り返し思惟する者」とは、その法を何度も何度も考察し、  
そこに心を向け、注意深く意にとめている者をいい、  
「憶念する者」とは、その同じ法を絶えず思い起こしている者である。

そのような比丘においては、三十七菩提分法と、  
四道・四果および涅槃という九種の出世間法が、減少・退転することはない、  
という意味である。

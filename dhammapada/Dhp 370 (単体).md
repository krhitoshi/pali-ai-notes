370.

‘‘Pañca chinde pañca jahe, pañca cuttari bhāvaye;

Pañcasaṅgātigo bhikkhu, oghatiṇṇoti vuccati.

Pañcachindeti heṭṭhāapāyasampāpakāni pañcorambhāgiyasaṃyojanāni pāde baddharajjuṃ puriso satthena viya heṭṭhāmaggattayena chindeyya. Pañca jaheti uparidevalokasampāpakāni pañcuddhambhāgiyasaṃyojanāni puriso gīvāya baddharajjukaṃ viya arahattamaggena jaheyya pajaheyya, chindeyyāti attho. Pañca cuttari bhāvayeti uddhambhāgiyasaṃyojanānaṃ pahānatthāya saddhādīni pañcindriyāni uttari bhāveyya. Pañcasaṅgātigoti evaṃ sante pañcannaṃ rāgadosamohamānadiṭṭhisaṅgānaṃ atikkamanena pañcasaṅgātigo bhikkhu oghatiṇṇoti vuccati, cattāro oghe tiṇṇoyevāti vuccatīti attho.

## 対訳

### 偈頌

1. Pañca chinde pañca jahe,

* pañca = 「五つ（のもの）を」
* chinde = 「断つべきである・断ち切るべきである（chindati の願望法）」
* pañca = 「五つ（のもの）を」
* jahe = 「捨て去るべきである（jahati の願望法）」

→「五つを断ち、五つを捨て去り」

2. pañca cuttari bhāvaye;

* pañca = 「さらに他の五つを」
* cuttari (= uttari) = 「いっそう・さらに先に・さらに高く」
* bhāvaye = 「修習すべきである（bhāveti の願望法）」

→「さらに別の五つを、いっそう（高く）修習すべきである。」

3. Pañcasaṅgātigo bhikkhu,

* pañca-saṅga-atigo = 「五つの『結びつき（saṅga）』を・越えた（atigo = atikkanto）」
  * pañca = 「五つの」
  * saṅga = 「執着・束縛・結びつき」
  * atigo = 「越えた、超えた者」
* bhikkhu = 「比丘は」

→「五つの執着を超えた比丘は」

4. oghatiṇṇoti vuccati.

* ogha-tiṇṇo = 「（四つの）激流（ogha）を・渡り終えた者」
* ti = 「～と」
* vuccati = 「呼ばれる、言われる」

→「『煩悩の大洪水を渡り終えた者』と呼ばれるのである。」


### 註釈文

5. Pañcachindeti heṭṭhāapāyasampāpakāni pañcorambhāgiyasaṃyojanāni pāde baddharajjuṃ puriso satthena viya heṭṭhāmaggattayena chindeyya.

* Pañcachinde ti = 「『pañca chinde（五つを断て）』とは」
* heṭṭhā-apāya-sampāpakāni = 「下方の悪趣（apāya）に至らせる（sampāpakāni）」
  * heṭṭhā = 「下の方へ」
  * apāya = 「悪趣・地獄等」
  * sampāpakāni = 「到達させる・もたらすもの」
* pañca-orambhāgiya-saṃyojanāni = 「五つの下分結（下位に属する五つの結びつき）」
  * pañca = 「五つの」
  * orambhāgiya = 「下方に属する」
  * saṃyojana = 「結縛・束縛」
* pāde baddha-rajjuṃ = 「足に縛られた縄を」
  * pāda = 「足」
  * baddha = 「縛られた」
  * rajju = 「縄」
* puriso = 「人が」
* satthena viya = 「刀で切るように」
  * sattha = 「刀・刃物」
  * viya = 「～のように」
* heṭṭhā-magga-ttayena = 「下位の三つの道によって」
  * heṭṭhā = 「下位の」
  * magga = 「道（聖道）」
  * ttaya = 「三つ」
* chindeyya = 「断つべきである」

→「『五つを断て』とは、下方の悪趣に至らせる五つの下分結を、人が足に縛られた縄を刀で切るように、下位の三つの道によって断つべきである、ということである。」

6. Pañca jaheti uparidevalokasampāpakāni pañcuddhambhāgiyasaṃyojanāni puriso gīvāya baddharajjukaṃ viya arahattamaggena jaheyya pajaheyya, chindeyyāti attho.

* Pañca jahe ti = 「『pañca jahe（五つを捨てよ）』とは」
* upari-devaloka-sampāpakāni = 「上方の天界に至らせる（生まれさせる）」
  * upari = 「上方の」
  * deva-loka = 「天界」
  * sampāpakāni = 「到達させるもの」
* pañca-uddhambhāgiya-saṃyojanāni = 「五つの上分結（上位に属する五つの結びつき）」
  * uddhaṃbhāgiya = 「上方に属する」
* puriso = 「人が」
* gīvāya baddha-rajjukaṃ viya = 「首に縛られた縄を切るように」
  * gīvā = 「首」
  * baddha = 「縛られた」
  * rajjuka = 「縄」
  * viya = 「～のように」
* arahattamaggena = 「阿羅漢道によって」
* jaheyya = 「捨てるべきである」
* pajaheyya = 「完全に捨て去るべきである（pa + jahati）」
* chindeyyā ti attho = 「すなわち断ち切るべきであるという意味である」

→「『五つを捨てよ』とは、上方の天界に生まれさせる五つの上分結を、人が首に縛られた縄を切るように、阿羅漢道によって捨て、完全に捨て去り、断ち切るべきである、という意味である。」

7. Pañca cuttari bhāvayeti uddhambhāgiyasaṃyojanānaṃ pahānatthāya saddhādīni pañcindriyāni uttari bhāveyya.

* Pañca cuttari bhāvaye ti = 「『pañca cuttari bhāvaye（さらに五つを修習せよ）』とは」
* uddhambhāgiya-saṃyojanānaṃ = 「上分結の（＝上位に属する結びつき）」
* pahāna-atthāya = 「捨断のために」
  * pahāna = 「捨て去ること・断滅」
  * atthāya = 「～のために」
* saddhā-ādīni pañca-indriyāni = 「信などの五つの根（五根）を」
  * saddhā = 「信」
  * -ādīni = 「等（精進・念・定・慧を含む）」
  * pañca = 「五つの」
  * indriya = 「根・支配力（五根）」
* uttari bhāveyya = 「さらに高めて修習すべきである」

→「『さらに五つを修習せよ』とは、上分結を捨て去るために、信などの五根を、いっそう高く修習すべきである、ということである。」

8. Pañcasaṅgātigoti evaṃ sante pañcannaṃ rāgadosamohamānadiṭṭhisaṅgānaṃ atikkamanena pañcasaṅgātigo bhikkhu oghatiṇṇoti vuccati, cattāro oghe tiṇṇoyevāti vuccatīti attho.

* Pañcasaṅgātigo ti = 「『pañcasaṅgātigo（五つの執着を超えた者）』とは」
* evaṃ sante = 「このようである時（以上のようにしているならば）」
* pañcannaṃ = 「五つの」
* rāga-dosa-moha-māna-diṭṭhi-saṅgānaṃ = 「貪欲・瞋恚・愚癡・慢・邪見という執着（結びつき）」
  * rāga = 「貪欲」
  * dosa = 「瞋恚」
  * moha = 「愚癡」
  * māna = 「慢」
  * diṭṭhi = 「邪見」
  * saṅga = 「執着・結びつき」
* atikkamanena = 「それらを超越することによって」
* pañca-saṅga-atigo bhikkhu = 「五つの執着を超えた比丘は」
* ogha-tiṇṇo ti vuccati = 「『煩悩の洪水を渡り終えた者』と呼ばれる」
* cattāro oghe tiṇṇo yeva ti vuccati ti attho = 「（すなわち）四つの（煩悩の）大洪水をすでに渡り終えた者である、と言われるのである、という意味である」
  * cattāro oghā = 「四つの激流（欲・有・見・無明の四洪水）」
  * tiṇṇa = 「渡り終えた、越え終えた」

→「『五つの執着を超えた者』とは、このような状態で、貪・瞋・癡・慢・邪見という五つの執着を越えたことによって、五つの執着を超えた比丘は『煩悩の洪水を渡り終えた者』と呼ばれ、すなわち四つの大洪水をすでに渡り終えた者であると呼ばれる、という意味である。」


## 意訳

「五つのものを断ち、さらに五つを捨て去り、さらに別の五つを一層高く修めよ。こうして五つの執着を超えた比丘は、『煩悩の大洪水を渡り終えた者』と呼ばれる。

ここで『五つを断て』とは、下方の悪趣に堕とす五つの下分結を、足に縛りついた縄を刀で断つように、下位の三つの聖道によって断つことである。

『五つを捨てよ』とは、上方の天界に生まれさせる五つの上分結を、首に巻き付いた縄を断ち切るように、阿羅漢道によって捨て去り、完全に断ち切ることである。

『さらに五つを修習せよ』とは、上分結を捨て去るために、信・精進・念・定・慧という五根を、いっそう高く修習することである。

このようにして、貪・瞋・癡・慢・邪見という五つの執着を超えた比丘は、四つの煩悩の大洪水を渡り終えた者と呼ばれるのである。」

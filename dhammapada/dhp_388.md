388.

‘‘Bāhitapāpoti brāhmaṇo, samacariyā samaṇoti vuccati;

Pabbājayamattano malaṃ, tasmā pabbajitoti vuccatī’’ti.

Tattha samacariyāti sabbākusalāni sametvā caraṇena. Tasmāti yasmā bāhitapāpatāya brāhmaṇo, akusalāni sametvā caraṇena samaṇoti vuccati, tasmā yo attano rāgādimalaṃ pabbājayanto vinodento carati, sopi tena pabbājanena pabbajitoti vuccatīti attho.

## 対訳

1. ‘‘Bāhitapāpoti brāhmaṇo,

* bāhita-pāpo ti  
  * bāhita = 「取り除かれた、排除された」  
  * pāpa = 「悪、不善」  
  → 「悪を取り除いた者、悪を離れた者」である、という意味  
* brāhmaṇo = 「ブラフマン／バラモン、聖者」

→「『悪を取り除いた者』を、（ここでは）バラモン（真の聖者）と言う。」

---

2. samacariyā samaṇoti vuccati;

* samacariyā = 「平等な行ない、正しく整った行ないによって」  
  * sama = 「平らな、整った、平等な、正しい」  
  * caryā (cariyā) = 「行ない、歩み、生き方」  
→「正しく整った行ないによって」
* samaṇo ti = 「『沙門（出家修行者）』と言われる」  
  * samaṇa = 「沙門、修行者」
* vuccati = 「～と呼ばれる、言われる」

→「正しく整った行ないによって、『沙門』と言われる。」

---

3. Pabbājayamattano malaṃ,

* pabbājayaṃ = 「追い出して／追放して」  
  * pabbājeti = 「追い出す、追放する、除く」  
  → その絶対分詞「～を追い出して、除いて」
* attano = 「自らの、自分自身の」  
* malaṃ = 「垢、汚れ、煩悩」

→「自らの（心の）汚れを追い出して（除いて）」

---

4. tasmā pabbajitoti vuccatī’’ti.

* tasmā = 「それゆえに、そのために」  
* pabbajito ti = 「『pabbajita（出家者）』と言われる」  
  * pabbajita = 通常「出家した者（家を出た者）」だが、ここでは  
    * pabbā-（外へ）+ jita/jita（駆逐された）と語源的説明も含意  
    → 「（煩悩を）外へ追いやった者」とも掛けられている
* vuccati = 「～と呼ばれる、言われる」

→「そのために、『出家者（pabbajita）』と呼ばれるのである、という意味である。」

---

5. Tattha samacariyāti sabbākusalāni sametvā caraṇena.

* tattha = 「ここで、この文句において」  
* samacariyā ti = 「『samacariyā（正しい行ない）』とは」  
* sabbākusalāni = 「一切の不善法を」  
  * sabba + akusalāni = 「すべての不善なるもの（不善法）」  
* sametvā = 「まとめて、ことごとく押さえ込んで、制して」  
  * sameti = 「集める、整える、統合する、抑える」  
* caraṇena = 「そのように歩むこと、行ないによって」

→「ここで『正しい行ない（samacariyā）』とは、一切の不善法をことごとく制し抑えるような行ないによって、という意味である。」

---

6. Tasmāti yasmā bāhitapāpatāya brāhmaṇo,

* tasmā ti = 「それゆえに、というのは」  
* yasmā = 「なぜなら、…であるがゆえに」  
* bāhita-pāpatāya = 「悪を取り除いたという性格によって」  
  * bāhita = 「取り除かれた」  
  * pāpa = 「悪、不善」  
  * -tā = 「～という状態・性質」  
→「悪が取り除かれているという在り方によって」
* brāhmaṇo = 「（真の）バラモンであり」

→「それゆえに、なぜなら、悪が取り除かれているという在り方によって『バラモン』であり、」

---

7. akusalāni sametvā caraṇena samaṇoti vuccati,

* akusalāni = 「不善法（不善なる諸要素）」  
* sametvā = 「まとめて、ことごとく制して、抑えて」  
* caraṇena = 「そのような行ない・実践によって」  
* samaṇo ti = 「『沙門』であると言われ」  
* vuccati = 「～と呼ばれる」

→「不善法をことごとく制し抑えるような行ないによって『沙門』と言われ，」

---

8. tasmā yo attano rāgādimalaṃ pabbājayanto vinodento carati,

* tasmā = 「だから、したがって」  
* yo = 「（～である）者は」  
* attano = 「自らの、自身の」  
* rāga-ādi-malaṃ = 「貪欲などの（rāga-ādi）汚れ」  
  * rāga = 「貪り、愛着、欲望」  
  * ādi = 「など、…を始めとする」  
  * mala = 「垢、汚れ、煩悩」  
→「貪欲などの心の垢」
* pabbājayanto = 「追い出しつつ（絶えず追放しつつ）」  
  * pabbājeti の現在分詞
* vinodento = 「追い払いつつ、除き去りつつ」  
  * vinodeti = 「駆逐する、払いのける、除く」
* carati = 「歩む、修行する、生きる、行ずる」

→「だから、自らの貪欲などの汚れを追い出し、追い払いながら修行して歩む者は」

---

9. sopi tena pabbājanena pabbajitoti vuccatīti attho.

* so pi = 「その人もまた」  
* tena pabbājanena = 「そのような追放行為（汚れを追い出すこと）によって」  
  * tena = 「その」  
  * pabbājana = 「追放、追い出し（pabbājeti からの名詞）」  
* pabbajito ti = 「『pabbajita（出家者）』と呼ばれる」  
* vuccati = 「～と呼ばれる」  
* ti attho = 「という意味である」

→「そのような汚れの追放によって、その人もまた『出家者（pabbajita）』と呼ばれる、という意味である。」

---

## 意訳

「悪を取り除いた者を、（真の意味での）バラモンと言い、  
一切の不善法をことごとく制し抑える正しい行ないによって、沙門と言われる。  

自らの心の汚れを追い出しているがゆえに、そうした者は『出家者（pabbajita）』と呼ばれる。  

ここで『正しい行ない』とは、一切の不善法をことごとく制御する実践のことである。  
すなわち、悪を除き去っているのでバラモンであり、不善法を抑える行ないによって沙門と言われる。  

したがって、自らの貪欲などの煩悩という汚れを、追い出し追い払いながら生き、修行する者は、  
その煩悩の追放によって、真に『出家者（pabbajita）』と呼ばれるのである。」

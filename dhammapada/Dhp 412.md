412.

‘‘Yodha puññañca pāpañca, ubho saṅgamupaccagā;

Asokaṃ virajaṃ suddhaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha ubhoti dvepi puññāni ca pāpāni ca chaḍḍetvāti attho. Saṅganti rāgādibhedaṃ saṅgaṃ. Upaccagāti atikkanto. Vaṭṭamūlakasokābhāvena asokaṃ abbhantare rāgarajādīnaṃ abhāvena virajaṃ nirupakkilesatāya suddhaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Yodha puññañca pāpañca, ubho saṅgamupaccagā;

- yo dha = 「まさにそのような人は／かれは」
- puññaṃ ca = 「功徳（善業）をも」
- pāpaṃ ca = 「罪（悪業）をも」
- ubho = 「この両方を」
- saṅgaṃ upaccagā = 「執着を越え去った」
  - saṅgaṃ = 「執着・つながり（煩悩による結びつき）を」
  - upaccagā = 「乗り越えた・超越した（atikkanto と同義）」

→「まさに、功徳と罪とのこの両方について、その執着を越え去った者」

2. Asokaṃ virajaṃ suddhaṃ, tamahaṃ brūmi brāhmaṇanti.

- asokaṃ = 「憂いなき者」
  - a- (否定) + soka = 「悲しみ・憂い」→「悲しみのない」
- virajaṃ = 「塵（けがれ）のない者」
  - vi-raja = 「煩悩の塵の離れた、無垢の」
- suddhaṃ = 「清浄な者」
- taṃ ahaṃ = 「そのような者を、わたしは」
- brūmi = 「呼ぶ・言う」
- brāhmaṇaṃ = 「バラモン（真の聖者）と」

→「憂いなく、塵なき、清浄なその人を、わたしは『バラモン（真の聖者）』と呼ぶのである。」

3. Tattha ubhoti dvepi puññāni ca pāpāni ca chaḍḍetvāti attho.

- tattha = 「ここで、この句において」
- ubho ti = 「『ubho（両方）』とは」
- dve pi = 「二つとも」
- puññāni ca pāpāni ca = 「功徳（善）と罪（悪）とを」
- chaḍḍetvā ti attho = 「捨て去っている、という意味である」
  - chaḍḍetvā = 「投げ捨てて、放棄して」

→「ここで『両方』とは、善と悪という二つともを捨て去っている、という意味である。」

4. Saṅganti rāgādibhedaṃ saṅgaṃ.

- saṅgaṃ ti = 「『saṅga（執着）』とは」
- rāga-ādi-bhedaṃ = 「欲愛などの種類に分かれる」
  - rāga = 「愛欲・貪り」
  - ādi = 「など」
  - bhedaṃ = 「区分・種別」
- saṅgaṃ = 「執着（である）」

→「『執着』とは、愛欲などさまざまな種類に分かれる執着のことである。」

5. Upaccagāti atikkanto.

- upaccagā ti = 「『upaccagā（越えた）』とは」
- atikkanto = 「超え去った者・乗り越えた者である」

→「『越え去った』とは、（それを）超え去った者である、ということである。」

6. Vaṭṭamūlakasokābhāvena asokaṃ…

- vaṭṭa-mūlaka-soka-abhāvena = 「輪廻に根ざした悲しみがないことによって」
  - vaṭṭa = 「輪廻、有情の流転（生死の輪）」
  - mūlaka = 「根ざした、根本とする」
  - soka = 「悲しみ・憂い」
  - abhāva = 「不在・無いこと」
  - -ena（具格）= 「〜によって」
- asokaṃ = 「（その人は）憂いなき者である」

→「輪廻に根ざした悲しみが存在しないことによって、その人は『憂いなき者』である。」

7. …abbhantare rāgarajādīnaṃ abhāvena virajaṃ…

- abbhantare = 「内において、内面に」
- rāga-raja-ādīnaṃ = 「愛欲という塵などの」
  - rāga = 「愛欲」
  - raja = 「塵（＝煩悩の汚れ）」
  - ādi = 「など」
- abhāvena = 「ないために、不在であることによって」
- virajaṃ = 「塵なき者、無垢である」

→「内面に、愛欲という塵などが存在しないことによって、『塵なき者（無垢な者）』である。」

8. …nirupakkilesatāya suddhaṃ…

- nirupakkilesatāya = 「いかなる汚れもない状態であることによって」
  - nir- = 「無い」
  - upakkilesa = 「汚れ・垢・煩悩の汚染」
  - -tā = 「〜であるという性質・状態」
  - -ya（具格） = 「〜であることにより」
- suddhaṃ = 「清浄である」

→「いかなる汚れもない状態であることによって、『清浄である』。」

9. …tamahaṃ brāhmaṇaṃ vadāmīti attho.

- taṃ ahaṃ = 「そのような者を、わたしは」
- brāhmaṇaṃ vadāmi ti = 「『バラモン（真の聖者）』と説くのである」
- attho = 「という意味である」

→「そのような人を、わたしは『バラモン（真の聖者）』と説くのである、という意味である。」


## 意訳

「善と悪のどちらにも執着することなく、その双方を乗り越えた人──  
生死流転に根ざす悲しみから解放され、内には愛欲という塵すらなく、  
いかなる汚れもないがゆえに清浄であるその人を、  
わたしは『真のバラモン』と呼ぶのである。」

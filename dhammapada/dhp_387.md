387.

‘‘Divā tapati ādicco, rattimābhāti candimā;

Sannaddho khattiyo tapati, jhāyī tapati brāhmaṇo;

Atha sabbamahorattiṃ, buddho tapati tejasā’’ti.

Tattha divā tapatīti divā virocati, rattiṃ panassa gatamaggopi na paññāyati. Candimāti candopi abbhādīhi vimutto rattimeva virocati, no divā. Sannaddhoti suvaṇṇamaṇivicittehi sabbābharaṇehi paṭimaṇḍito caturaṅginiyā senāya parikkhittova rājā virocati, na aññātakavesena ṭhito. Jhāyīti khīṇāsavo pana gaṇaṃ vinodetvā jhāyantova virocati. Tejasāti sammāsambuddho pana sīlatejena dussīlyatejaṃ, guṇatejena nigguṇatejaṃ, paññātejena duppaññatejaṃ, puññatejena apuññatejaṃ, dhammatejena adhammatejaṃ pariyādiyitvā iminā pañcavidhena tejasā niccakālameva virocatīti attho.

## 対訳

1. ‘‘Divā tapati ādicco, rattimābhāti candimā;

* divā = 「昼に」
* tapati = 「輝く・照りつける」
* ādicco = 「太陽が」
→ 「昼には太陽が輝き」

* rattiṃ = 「夜に」
* ābhāti = 「光り映える・輝く」
* candimā = 「月が」
→ 「夜には月が光り映える。」

2. Sannaddho khattiyo tapati, jhāyī tapati brāhmaṇo;

* sannaddho = 「武装した・装備を整えた」
* khattiyo = 「クシャトリヤ（王族の戦士）、王」
* tapati = 「輝く」
→ 「武装した王族（戦士の王）は輝き」

* jhāyī = 「禅定に入った者・禅修行者」
* tapati = 「輝く」
* brāhmaṇo = 「バラモン（ここでは聖者）」
→ 「禅定に入ったバラモンは輝く。」

3. Atha sabbamahorattiṃ, buddho tapati tejasā’’ti.

* atha = 「しかし・ところが」
* sabba-maha-rattiṃ = 「一日一夜じゅう（昼夜を通して）」
* buddho = 「仏陀は」
* tapati = 「輝く」
* tejasā = 「威光によって・光輝によって」
→ 「しかし一日一夜を通しては、仏陀が威光によって輝くのである。」


4. Tattha divā tapatīti divā virocati, rattiṃ panassa gatamaggopi na paññāyati.

* tattha = 「ここで（この偈において）」
* divā tapatīti = 「『divā tapati（昼に輝く）』とは」
* divā = 「昼に」
* virocati = 「光り輝く、明らかに見える」
→ 「ここで『昼に輝く』とは、昼に光り輝くということであり」

* rattiṃ pana = 「しかし夜には」
* assa = 「その（太陽の）」
* gata-m-aggopi = 「通って行った道筋さえも」
  * gata = 「行った・通り過ぎた」
  * maggo = 「道・通路」
  * pi = 「～さえも」
* na paññāyati = 「知れない・見分けられない・見えない」
→ 「しかし夜になると、その通って行った道筋すら見えないのである。」


5. Candimāti candopi abbhādīhi vimutto rattimeva virocati, no divā.

* candimā ti = 「『candimā（月）』とは」
* cando pi = 「月もまた」
* abbhādīhi = 「雲など（雲等の覆い）から」
  * abbhā = 「雲」
  * ādīhi = 「～などによる（…などのものから）」
* vimutto = 「解放された・覆いを免れた時には」
* rattiṃ eva = 「夜にだけ」
* virocati = 「輝く」
* no divā = 「昼には（そう）輝かない」
→ 「『月』とは、月もまた雲などの覆いが取り除かれているときには夜にだけ輝き、昼には輝かないということである。」


6. Sannaddhoti suvaṇṇamaṇivicittehi sabbābharaṇehi paṭimaṇḍito caturaṅginiyā senāya parikkhittova rājā virocati, na aññātakavesena ṭhito.

* sannaddho ti = 「『sannaddho（武装した）』とは」
* suvaṇṇa-maṇi-vicittehi = 「金や宝石で飾り立てられた」
  * suvaṇṇa = 「金」
  * maṇi = 「宝石」
  * vicitta = 「美しく飾られた・種々に飾られた」
* sabba-ābharaṇehi = 「あらゆる装身具をもって」
* paṭimaṇḍito = 「身を装った・飾り立てられた」
* caturaṅginiyā senāya = 「四種の軍隊（象・馬・戦車・歩兵）から成る軍勢に」
* parikkhitto va = 「取り囲まれているときにこそ」
* rājā = 「王は」
* virocati = 「輝いて見える」
→ 「『武装した者』とは、金や宝石で飾られたあらゆる装身具を身につけ、四種の軍勢に取り囲まれているときにこそ王は輝いて見えるということであり」

* na aññātaka-vesena ṭhito = 「よそ者の姿・見慣れぬ服装で立っているときには（輝かない）」
  * aññātaka = 「他人の・よそ者の」
  * vesa = 「装い・服装」
  * ṭhito = 「立っている」
→ 「見知らぬ者のような装いで立っているときには輝かないのである。」


7. Jhāyīti khīṇāsavo pana gaṇaṃ vinodetvā jhāyantova virocati.

* jhāyī ti = 「『jhāyī（禅定に入った者）』とは」
* khīṇāsavo pana = 「しかるに、煩悩が尽きた聖者は」
  * khīṇa = 「滅した・尽きた」
  * āsava = 「煩悩の漏れ」
* gaṇaṃ vinodetvā = 「仲間の群れを退けて・群衆から離れて」
  * gaṇa = 「仲間・一団・集団」
  * vinodetvā = 「退けて・遠ざけて」
* jhāyanto va = 「ただ禅定しているときにこそ」
* virocati = 「輝く」
→ 「『禅定に入った者』とは、煩悩が尽きた聖者が、仲間の群れを離れ、ただ禅定にふけっているときにこそ輝くということである。」


8. Tejasāti sammāsambuddho pana sīlatejena dussīlyatejaṃ, guṇatejena nigguṇatejaṃ, paññātejena duppaññatejaṃ, puññatejena apuññatejaṃ, dhammatejena adhammatejaṃ pariyādiyitvā iminā pañcavidhena tejasā niccakālameva virocatīti attho.

* tejasā ti = 「『tejasā（威光によって）』とは」
* sammā-sambuddho pana = 「しかし正しくさとりを得た仏陀は」
* sīla-tejena = 「戒の威光によって」
* dussīlya-tejaṃ = 「悪しき戒行（不道徳）の威光を」
→ 「戒の威光によって不徳の威光を覆い尽くし」

* guṇa-tejena = 「徳の威光によって」
* nigguṇa-tejaṃ = 「無徳の威光を」
→ 「徳の威光によって無徳の威光を覆い尽くし」

* paññā-tejena = 「智慧の威光によって」
* duppañña-tejaṃ = 「無智の威光を」
→ 「智慧の威光によって愚かさの威光を覆い尽くし」

* puñña-tejena = 「功徳の威光によって」
* apuñña-tejaṃ = 「不善・罪業の威光を」
→ 「功徳の威光によって不善の威光を覆い尽くし」

* dhamma-tejena = 「正法の威光によって」
* adhamma-tejaṃ = 「不法の威光を」
→ 「法の威光によって不法の威光を覆い尽くして」

* pariyādiyitvā = 「完全に取り囲み・凌駕し・打ち消して」
* imina pañcavidhena tejasā = 「このような五種の威光によって」
* nicca-kālam eva = 「つねに・絶えず」
* virocati ti attho = 「輝いている、という意味である」
→ 「この五種の威光によって、常に絶えず輝いているのである、という意味である。」


## 意訳

「昼には太陽が輝き、夜には月が光り映える。
武装し飾り立てられた王は輝き、禅定に入った聖なるバラモンも輝く。
しかし、一日一夜を通してつねに輝いているのは、
その威光による仏陀である。

すなわち、太陽は昼だけ輝き、夜にはその通り過ぎた跡すら見えなくなる。
月もまた、雲などに覆われていないときには夜だけ輝き、昼には輝かない。
王は、金や宝石のあらゆる装身具をまとい、四種の軍勢に取り囲まれているときにこそ輝いて見え、
見知らぬ者のような姿で立っているときには輝かない。
煩悩の尽きた聖者も、群れから離れ、ひとり禅定にふけっているときにこそ輝く。

これに対して、正しく悟った仏陀は、
戒の威光によって不徳の威光を、
徳の威光によって無徳の威光を、
智慧の威光によって愚かさの威光を、
功徳の威光によって不善の威光を、
法の威光によって不法の威光を、
すべて覆い尽くし凌駕しながら、
この五種の威光によって、いついかなる時にも輝いているのである。」

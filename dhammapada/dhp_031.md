31.

‘‘Appamādarato bhikkhu, pamāde bhayadassi vā;

Saṃyojanaṃ aṇuṃ thūlaṃ, ḍahaṃ aggīva gacchatī’’ti.

Tattha appamādaratoti appamāde rato abhirato, appamādena vītināmentoti attho. Pamāde bhayadassi vāti nirayuppattiādikaṃ pamāde bhayaṃ bhayato passanto, tāsaṃ vā uppattīnaṃ mūlattā pamādaṃ bhayato passanto. Saṃyojananti vaṭṭadukkhena saddhiṃ yojanaṃ bandhanaṃ pajānaṃ vaṭṭe osīdāpanasamatthaṃ dasavidhaṃ saṃyojanaṃ. Aṇuṃ thūlanti mahantañca khuddakañca. Ḍahaṃ aggīva gacchatīti yathā ayaṃ aggī etaṃ mahantañca khuddakañca upādānaṃ ḍahantova gacchati. Evameso appamādarato bhikkhu appamādādhigatena ñāṇagginā etaṃ saṃyojanaṃ ḍahanto abhabbuppattikaṃ karonto gacchatīti attho.

## 対訳

1. ‘‘Appamādarato bhikkhu, pamāde bhayadassi vā;

* appamāda-rato = 「不放逸を楽しむ（不放逸を喜ぶ）」
* bhikkhu = 「比丘は」
* pamāde = 「放逸において」
* bhaya-dassi = 「恐れを見る者」
* vā = 「あるいは」

→ 「不放逸を楽しみ、あるいは放逸の中に恐れを見る比丘は、」

2. Saṃyojanaṃ aṇuṃ thūlaṃ, ḍahaṃ aggīva gacchatī’’ti.

* saṃyojanaṃ = 「結縛（煩悩の束縛）を」
* aṇuṃ = 「微細な」
* thūlaṃ = 「粗大な」
* ḍahaṃ = 「焼きつつ（ḍahatiの現在分詞）」
* aggīva (aggi + iva) = 「火のように」
* gacchati = 「進む、行く」
* ti = 「（引用符）」

→ 「粗大な結縛も微細な結縛も、火のように焼き尽くしながら進む。」

3. Tattha appamādaratoti appamāde rato abhirato, appamādena vītināmentoti attho.

* tattha = 「そこで、その中で」
* appamādarato ti = 「『appamādarato（不放逸を楽しむ）』とは」
* appamāde = 「不放逸において」
* rato = 「楽しむ者」
* abhirato = 「深く楽しむ者」
* appamādena = 「不放逸をもって」
* vītināmento = 「時を過ごす者」
* ti attho = 「という意味である」

→ 「そこで、『不放逸を楽しむ』とは、不放逸において楽しみ、深く楽しみ、不放逸をもって時を過ごす者である、という意味である。」

4. Pamāde bhayadassi vāti nirayuppattiādikaṃ pamāde bhayaṃ bhayato passanto, tāsaṃ vā uppattīnaṃ mūlattā pamādaṃ bhayato passanto.

* pamāde bhayadassi vā ti = 「『pamāde bhayadassi vā（あるいは放逸に恐れを見る）』とは」
* nirayuppatti-ādikaṃ = 「地獄への誕生などの」
* pamāde = 「放逸において」
* bhayaṃ = 「恐れを」
* bhayato = 「恐れとして」
* passanto = 「見る者」
* tāsaṃ vā uppattīnaṃ = 「あるいは、それらの（悪趣への）誕生の」
* mūlattā = 「根本であるために」
* pamādaṃ = 「放逸を」
* bhayato passanto = 「恐れとして見る者」

→ 「『あるいは放逸に恐れを見る』とは、放逸の中に地獄への誕生などの恐れを恐れとして見る者、あるいは、それらの誕生の根本であるがゆえに放逸そのものを恐れとして見る者のことである。」

5. Saṃyojananti vaṭṭadukkhena saddhiṃ yojanaṃ bandhanaṃ pajānaṃ vaṭṭe osīdāpanasamatthaṃ dasavidhaṃ saṃyojanaṃ.

* saṃyojanan ti = 「『saṃyojanaṃ（結縛）』とは」
* vaṭṭadukkhena saddhiṃ = 「輪廻の苦とともに」
* yojanaṃ = 「結びつけること」
* bandhanaṃ = 「縛ること」
* pajānaṃ = 「衆生（有情）の」
* vaṭṭe = 「輪廻に」
* osīdāpana-samatthaṃ = 「沈ませる力をもつ（osīdāpana:沈ませる + samattha:能力がある）」
* dasavidhaṃ saṃyojanaṃ = 「十種の結縛のことである」

→ 「『結縛』とは、衆生を輪廻の苦とともに結びつけ、縛りつけ、輪廻に沈ませる力をもつ十種の結縛のことである。」

6. Aṇuṃ thūlanti mahantañca khuddakañca.

* aṇuṃ thūlan ti = 「『aṇuṃ thūlaṃ（微細な、粗大な）』とは」
* mahantañca = 「大きなもの（粗大）も」
* khuddakañca = 「小さなもの（微細）も」

→ 「『微細な、粗大な』とは、大きな結縛も小さな結縛も（ということである）。」

7. Ḍahaṃ aggīva gacchatīti yathā ayaṃ aggī etaṃ mahantañca khuddakañca upādānaṃ ḍahantova gacchati.

* ḍahaṃ aggīva gacchati ti = 「『ḍahaṃ aggīva gacchati（火のように焼きながら進む）』とは」
* yathā = 「ちょうど～のように」
* ayaṃ aggī = 「この火が」
* etaṃ = 「その」
* mahantañca khuddakañca = 「大小の」
* upādānaṃ = 「燃料（薪・執着）を」
* ḍahanto eva = 「まさに焼き尽くしながら」
* gacchati = 「進む（ように）」

→ 「『火のように焼きながら進む』とは、ちょうどこの火が大小の燃料を焼き尽くしながら進むように、」

8. Evameso appamādarato bhikkhu appamādādhigatena ñāṇagginā etaṃ saṃyojanaṃ ḍahanto abhabbuppattikaṃ karonto gacchatīti attho.

* evaṃ eso = 「そのように、この」
* appamādarato bhikkhu = 「不放逸を楽しむ比丘は」
* appamādādhigatena = 「不放逸によって得られた」
* ñāṇagginā = 「智の火によって（ñāṇa + aggi + nā）」
* etaṃ saṃyojanaṃ = 「これらの結縛を」
* ḍahanto = 「焼きながら」
* abhabbuppattikaṃ = 「（二度と）生じ得ない状態に」
* karonto = 「しながら」
* gacchati ti attho = 「進む、という意味である」

→ 「そのように、この不放逸を楽しむ比丘が、不放逸によって得られた智の火によってこれらの結縛を焼き尽くし、二度と生じ得ない状態にしながら進む、という意味である。」

## 意訳

不放逸（怠りなく努めること）を楽しみ、あるいは放逸（怠惰）に恐れを見出している比丘は、大小あらゆる煩悩の束縛を、まるで火のように焼き尽くしながら進む。

註釈によれば、「不放逸を楽しむ」とは、不放逸を深く喜び、不放逸をもって時を過ごす者という意味である。「放逸に恐れを見る」とは、放逸が地獄などの悪趣へ生まれる原因であると知り、放逸の中に恐れを見出している者のことである。

「結縛」とは、衆生を輪廻の苦しみと結びつけて縛り、輪廻の中に沈ませる力を持つ十種の煩悩の束縛を指す。「微細な、粗大な」とは大小の結縛のことである。

「火のように焼きながら進む」とは、燃え広がる火が大小の燃料を焼き尽くしながら進むように、不放逸を楽しむ比丘が、不放逸の実践によって得られた「智慧の火」によってあらゆる結縛を焼き尽くし、それらが二度と生じない状態に滅しながら進む、という意味である。

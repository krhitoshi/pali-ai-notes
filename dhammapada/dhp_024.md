24.

‘‘Uṭṭhānavato satīmato,

Sucikammassa nisammakārino;

Saññatassa dhammajīvino,

Appamattassa yasobhivaḍḍhatī’’ti.

Tattha uṭṭhānavatoti uṭṭhānavīriyavantassa. Satimatoti satisampannassa. Sucikammassāti niddosehi niraparādhehi kāyakammādīhi samannāgatassa. Nisammakārinoti evañce bhavissati, evaṃ karissāmīti vā, imasmiṃ kamme evaṃ kate idaṃ nāma bhavissatīti vā evaṃ nidānaṃ sallakkhetvā rogatikicchanaṃ viya sabbakammāni nisāmetvā upadhāretvā karontassa. Saññatassāti kāyādīhi saññatassa nicchiddassa. Dhammajīvinoti agārikassa tulākūṭādīni vajjetvā kasigorakkhādīhi, anagārikassa vejjakammadūtakammādīni vajjetvā dhammena samena bhikkhācariyāya jīvikaṃ kappentassa. Appamattassāti avippavutthasatino. Yasobhivaḍḍhatīti issariyabhogasampannasaṅkhāto ceva kittivaṇṇabhaṇanasaṅkhāto ca yaso abhivaḍḍhatīti.

## 対訳

### 偈文（Gāthā）

1. Uṭṭhānavato satīmato,

* uṭṭhānavato = 「奮励努力する者の」（uṭṭhānavant の具格/属格・単数）
* satīmato = 「気づき（念）ある者の」（satimant の具格/属格・単数）
    * 通常は satimato と短音だが、韻律のために satīmato と長音化されている。

→ 「奮励努力し、気づきを保ち、」

2. Sucikammassa nisammakārino;

* suci-kammassa = 「清らかな行いのある者の」
    * suci = 清浄な
    * kamma = 行為
* nisamma-kārino = 「よく熟考してから行動する者の」
    * nisamma = 注意深く、熟考して（Gerund）
    * kārin = 行う人

→ 「行いが清らかで、よく熟考してから行動し、」

3. Saññatassa dhammajīvino,

* saññatassa = 「自制した者の、抑制された者の」（saññata の属格）
* dhamma-jīvino = 「法（ダンマ）に従って生活する者の」

→ 「自らを制し、法に従って生き、」

4. Appamattassa yasobhivaḍḍhatī’’ti.

* appamattassa = 「不放逸な者の、怠らざる者の」（appamatta の属格）
* yaso = 「名声が、誉れが」（yasa 中性・主格）
* abhivaḍḍhati = 「増大する」
* ti = 引用の終わり

→ 「怠りなき者の名声は、増大する。」

---

### 註釈（Atthakathā）

5. Tattha uṭṭhānavatoti uṭṭhānavīriyavantassa.

* tattha = 「その中で（その偈文において）」
* uṭṭhānavato ti = 「『uṭṭhānavato』とは」
* uṭṭhāna-vīriya-vantassa = 「奮起と精進（ヴィリヤ）を備えた者の（ことである）」

→ 「そこで『uṭṭhānavato』とは、奮起する精進力を持つ者のことである。」

6. Satimatoti satisampannassa.

* satimato ti = 「『satimato』とは」
* sati-sampannassa = 「念（サティ）を具足した者の（ことである）」

→ 「『satimato』とは、念（気づき）を完全に備えた者のことである。」

7. Sucikammassāti niddosehi niraparādhehi kāyakammādīhi samannāgatassa.

* sucikammassa ti = 「『sucikammassa』とは」
* niddosehi = 「過失のない」
* niraparādhehi = 「罪のない、無垢な」
* kāyakammādīhi = 「身業（身体的行為）などを」
* samannāgatassa = 「備えた者の（ことである）」

→ 「『sucikammassa』とは、過失がなく罪のない、身体的行為などを備えた者のことである。」

8. Nisammakārinoti evañce bhavissati, evaṃ karissāmīti vā, imasmiṃ kamme evaṃ kate idaṃ nāma bhavissatīti vā

* nisammakārino ti = 「『nisammakārino』とは」
* evañce bhavissati = 「もしこのようになるならば」
* evaṃ karissāmi ti vā = 「『私はこのようにしよう』とか」
* imasmiṃ kamme evaṃ kate = 「この行為がこのように為されたならば」
* idaṃ nāma bhavissati ti vā = 「『これこれの結果が生じるであろう』とか」

→ 「『nisammakārino』とは、『もしこの状況なら私はこうしよう』、あるいは『この行為をこのように行えば、これこれの結果になるだろう』と（考え）、」

9. evaṃ nidānaṃ sallakkhetvā rogatikicchanaṃ viya sabbakammāni nisāmetvā upadhāretvā karontassa.

* evaṃ = 「このように」
* nidānaṃ = 「原因・動機を」
* sallakkhetvā = 「よく観察して」
* roga-tikicchanaṃ viya = 「病気の治療（をする医者）のように」
* sabba-kammāni = 「すべての行為を」
* nisāmetvā = 「熟慮して」
* upadhāretvā = 「検討して、確かめて」
* karontassa = 「行う者の（ことである）」

→ 「このように原因をよく観察し、あたかも病気の治療（をする医者）のように、すべての行為を熟慮し、検討してから行う者のことである。」

10. Saññatassāti kāyādīhi saññatassa nicchiddassa.

* saññatassa ti = 「『saññatassa』とは」
* kāyādīhi saññatassa = 「身体（・言葉・心）等によって抑制された者の」
* nicchiddassa = 「（戒律や行いに）穴のない者、欠点のない者の（ことである）」

→ 「『saññatassa』とは、身・口・意において制御され、（行いに）隙のない者のことである。」

11. Dhammajīvinoti agārikassa tulākūṭādīni vajjetvā kasigorakkhādīhi,

* dhammajīvino ti = 「『dhammajīvino』とは」
* agārikassa = 「在家者にとっては」
* tulā-kūṭādīni = 「秤（はかり）のごまかし等を」
* vajjetvā = 「避けて」
* kasi-gorakkhādīhi = 「農業や牧畜（牛の保護）等によって（生計を立て）」

→ 「『dhammajīvino』とは、在家者であれば、秤のごまかし（不正）などを避け、農業や牧畜などによって（正当に暮らし）、」

12. anagārikassa vejjakammadūtakammādīni vajjetvā dhammena samena bhikkhācariyāya jīvikaṃ kappentassa.

* anagārikassa = 「出家者にとっては」
* vejjakamma-dūtakammādīni = 「医療行為や使い走り（使者）の行為等を」
    * 律において、比丘が生活の糧として医療や使者を務めることは邪命（邪な生活手段）とされる。
* vajjetvā = 「避けて」
* dhammena samena = 「法にのっとり、正しく」
* bhikkhācariyāya = 「托鉢（乞食行）によって」
* jīvikaṃ kappentassa = 「生計を維持する者の（ことである）」

→ 「出家者であれば、医療行為や使者の仕事などを避け、法にのっとり正しく托鉢によって生計を営む者のことである。」

13. Appamattassāti avippavutthasatino.

* appamattassa ti = 「『appamattassa』とは」
* avippavuttha-satino = 「念（サティ）が散乱していない者、失念していない者の（ことである）」

→ 「『appamattassa』とは、念（気づき）が散乱せず、常に目覚めている者のことである。」

14. Yasobhivaḍḍhatīti issariyabhogasampannasaṅkhāto ceva kittivaṇṇabhaṇanasaṅkhāto ca yaso abhivaḍḍhatīti.

* yaso abhivaḍḍhati ti = 「『yaso abhivaḍḍhati』とは」
* issariya-bhoga-sampanna-saṅkhāto ca eva = 「権勢や財産を具足するという意味での（名声）と」
* kitti-vaṇṇa-bhaṇana-saṅkhāto ca = 「賞賛や誉め言葉が語られるという意味での（名声）の」
* yaso = 「名声が」
* abhivaḍḍhatīti = 「増大する、ということである」

→ 「『yasobhivaḍḍhati』とは、権勢や財産に恵まれるという意味の名声と、人々から賞賛され誉め称えられるという意味の名声の、（その双方の）名声が増大する、という意味である。」

## 意訳

「奮起して努力し、常に気づき（念）を保ち、
身体・言葉・心の行いが清らかで、よく熟考してから行動し、
自らを制し、法に従って正しく生活し、
怠ることのない人、
そのような人の名声は、大いに高まる。」

**（註釈に基づく解説）**
ここで「奮起する」とは精進努力すること、「気づきある」とは念を完全に備えていることです。「行いが清らか」とは身・口・意に過ちがないことです。「熟考して行う」とは、医者が病状を見て治療法を決めるように、原因と結果をよく観察し、「こうすればこうなる」と慎重に判断して行動することです。「自らを制する」とは行いに隙がなく制御されていること。「法に従って生活する」とは、在家者であれば不正な商売を避けて正業に就き、出家者であれば医療や使者などの不適切な生計を避けて正しく托鉢で生きることです。「怠ることのない」とは念が散乱していないことです。
そのような人の「名声が増大する」とは、権力や財産といった実利的な面でも、人々からの賞賛や評判という面でも、その誉れが高まっていくという意味です。

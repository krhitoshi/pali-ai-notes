377.

‘‘Vassikā viya pupphāni, maddavāni pamuñcati;

Evaṃ rāgañca dosañca, vippamuñcetha bhikkhavo’’ti.

Tattha vassikāti sumanā. Maddavānīti milātāni. Idaṃ vuttaṃ hoti – yathā vassikā hiyyo pupphitapupphāni punadivase purāṇabhūtāni muñcati, vaṇṭato vissajjeti, evaṃ tumhepi rāgādayo dose vippamuñcethāti.

## 対訳

1. ‘‘Vassikā viya pupphāni, maddavāni pamuñcati;

* vassikā = 「ヴァッシカーの花（＝スマナ＝ジャスミン）」
* viya = 「～のように」
* pupphāni = 「花どもを」
* maddavāni = 「しおれたもの、萎れたものを」
* pamuñcati = 「放ち捨てる、落とす」

→「ヴァッシカーの花が、しおれた花を落とすように、」

2. Evaṃ rāgañca dosañca, vippamuñcetha bhikkhavo’’ti.

* evaṃ = 「このように」
* rāgañca = 「貪欲（愛欲）も」
  * rāga = 「貪欲、愛着」
  * -ñca（ca）= 「～も」
* dosañca = 「怒り（瞋恚）も」
  * dosa = 「怒り、悪意」
  * -ñca（ca）= 「～も」
* vippamuñcetha = 「完全に放ち捨てよ、離れよ」
  * vi- + pamuñcati の願望・命令形「徹底的に手放す、解き放つ」
* bhikkhavo = 「比丘たちよ」

→「同じように、貪欲も怒りも、比丘たちよ、すっかり手放しなさい。」

3. Tattha vassikāti sumanā.

* tattha = 「ここで、この句において」
* vassikā ti = 「『vassikā』とは」
* sumanā = 「スマナの花（ジャスミン）のことである」

→「ここで『vassikā』とは、スマナの花（ジャスミン）である。」

4. Maddavānīti milātāni.

* maddavāni ti = 「『maddavāni』とは」
* milātāni = 「しおれたもの、枯れかけたもののことである」

→「『maddavāni』とは、しおれた（花々）という意味である。」

5. Idaṃ vuttaṃ hoti – yathā vassikā hiyyo pupphitapupphāni punadivase purāṇabhūtāni muñcati, vaṇṭato vissajjeti, evaṃ tumhepi rāgādayo dose vippamuñcethāti.

* idaṃ vuttaṃ hoti = 「これは次のように言われたのである、という意味である」
* yathā = 「ちょうど～のように」
* vassikā = 「ヴァッシカー（スマナ）の花は」
* hiyyo = 「きのう」
* pupphita-pupphāni = 「咲いた花（咲いていた花々）を」
  * pupphita = 「咲いた」
  * pupphāni = 「花ども」
* puna-divase = 「翌日には」
* purāṇa-bhūtāni = 「古くなったものとなった（古びたものとなった）それらを」
* muñcati = 「落とす、手放す」
* vaṇṭato = 「花柄（茎）のところから」
* vissajjeti = 「離し去る、放ち捨てる」
* evaṃ = 「このように、それと同じく」
* tumhe pi = 「君たちもまた」
* rāgādayo = 「貪欲などの（～を始めとする）」
  * rāga = 「貪欲」
  * ādayo (ādi + 複数) = 「～など、～を始めとする」
* dose = 「煩悩・汚れ（悪徳）を」
  * dosa ではなく dosā（dosa の複数・中性形）＝「諸々の汚れ、煩悩」
* vippamuñcetha ti = 「完全に手放すべきである」
* attho（省略）= 「という意味である」

→「ここで言われたことの意味はこうである──ちょうどヴァッシカー（スマナの花）が、きのう咲いた花々で、翌日には古くなったものとなったそれらを、花柄から離して落とすように、同じく、あなたがたもまた、貪欲などの諸々の汚れを、すっかり手放さなければならない、という意味である。」


## 意訳

ヴァッシカー（スマナの花）が、しおれてしまった花を次々と落としていくように、
あなたがた比丘も、貪欲や怒りといった煩悩を、すっかり手放しなさい。

ここで「ヴァッシカー」とはスマナ（ジャスミン）の花のことであり、「しおれたもの」とは萎れた花を意味する。ちょうどその花が、昨日咲いた花で、翌日には古くなったそれらを茎から切り離して落とすように、同じように、あなたがたも貪欲などのあらゆる汚れを完全に捨て去らなければならない、という意味である。

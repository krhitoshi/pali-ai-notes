378.

‘‘Santakāyo santavāco, santavā susamāhito;

Vantalokāmiso bhikkhu, upasantoti vuccatī’’ti.

Tattha santakāyoti pāṇātipātādīnaṃ abhāvena santakāyo, musāvādādīnaṃ abhāvena santavāco, abhijjhādīnaṃ abhāvena santavā, kāyādīnaṃ tiṇṇampi suṭṭhu samāhitattā susamāhito, catūhi maggehi lokāmisassa vantatāya vantalokāmiso bhikkhu abbhantare rāgādīnaṃ upasantatāya upasantoti vuccatīti attho.

## 対訳

1. ‘‘Santakāyo santavāco, santavā susamāhito;

* santakāyo = santa（静まった・寂静な）+ kāya（身・身体）  
  → 「身が静まった者」

* santavāco = santa（静まった）+ vāco（言葉・語）  
  → 「語（ことば）が静まった者」

* santavā = santa（静まった）+ v ā（ここでは心・性向を表す）  
  → 「心静かな者・性向が静まった者」

* susamāhito = su-（よく）+ samāhita（定められた・三昧に入った）  
  → 「よく定められた者・よく三昧に入った者」

→「身が静まり、語が静まり、心静かな、よく定まった者は、」

2. Vantalokāmiso bhikkhu, upasantoti vuccatī’’ti.

* vanta-lokāmiso = vanta（吐き出された・吐き捨てられた）+ loka-āmisa（世間の甘餌・世俗の欲楽）  
  → 「世間の欲の餌を吐き捨てた者」

* bhikkhu = 比丘（出家修行者）

* upasanto = 「寂静した者・完全に静まった者」

* ti vuccati = 「～と呼ばれる」

→「世間の欲の餌を吐き捨てた比丘は、『寂静した者』と呼ばれるのである。」


3. Tattha santakāyoti pāṇātipātādīnaṃ abhāvena santakāyo,

* tattha = 「ここで」「この箇所においては」

* santakāyo ti = 「『santakāyo』とは」

* pāṇātipāta-ādīnaṃ = pāṇātipāta（殺生）+ ādī（など）+ の複数属格  
  → 「殺生などの」

* abhāvena = 「無いがゆえに・存在しないことによって」

* santakāyo = 「身が静まった者」

→「ここで『身が静まった者』とは、殺生などが存在しないがゆえに身が静まった者であり、」


4. musāvādādīnaṃ abhāvena santavāco,

* musāvāda-ādīnaṃ = musāvāda（妄語・嘘）+ ādī（など）+ 複数属格  
  → 「妄語などの」

* abhāvena = 「無いことによって」

* santavāco = 「語が静まった者」

→「妄語などが無いことによって『語が静まった者』であり、」


5. abhijjhādīnaṃ abhāvena santavā,

* abhijjhā-ādīnaṃ = abhijjhā（貪り・貪求）+ ādī（など）+ 複数属格  
  → 「貪りなどの」

* abhāvena = 「無いことによって」

* santavā = 「心静かな者・性向の静まった者」

→「貪りなどが無いことによって『心静かな者（性向の静まった者）』であり、」


6. kāyādīnaṃ tiṇṇampi suṭṭhu samāhitattā susamāhito,

* kāyādīnaṃ = kāya-ādi（身など＝身・語・意）+ 複数属格  
  → 「身などの」

* tiṇṇaṃ pi = 「この三つすべて（身・語・意の三つ）」

* suṭṭhu = 「よく・完全に」

* samāhitattā = samāhita（定められた）+ -tā（状態）  
  → 「よく定まっているという状態であるがゆえに」

* susamāhito = 「よく定められた者・よく三昧に入った者」

→「身など三つすべてがよく定められているがゆえに、『よく定まった者』であり、」


7. catūhi maggehi lokāmisassa vantatāya vantalokāmiso bhikkhu

* catūhi maggehi = catu（四）+ magga（道）+ 複数具格  
  → 「四つの道（四聖道）によって」

* lokāmisassa = loka-āmisa（世間の甘餌・世俗の欲楽）+ 単数属格  
  → 「世間の欲の餌の」

* vantatāya = vanta（吐き出された）+ -tā（状態）+ 具格  
  → 「吐き捨てられているということによって」

* vanta-lokāmiso = 「世間の欲の餌を吐き捨てた者」

* bhikkhu = 「比丘」

→「四つの聖なる道によって世間の欲の餌を吐き捨てているがゆえに、『世間の欲の餌を吐き捨てた者』である比丘は、」


8. abbhantare rāgādīnaṃ upasantatāya upasantoti vuccatīti attho.

* abbhantare = 「内において・内面において」

* rāga-ādīnaṃ = rāga（貪欲）+ ādī（など：瞋・痴など）+ 複数属格  
  → 「貪欲などの」

* upasantatāya = upasanta（静まった・寂静した）+ -tā（状態）+ 具格  
  → 「静まっているということによって」

* upasanto ti vuccati = 「『寂静した者』と呼ばれる」

* ti attho = 「という意味である」

→「内面において貪欲などが静まっているがゆえに、『寂静した者』と呼ばれるのである、という意味である。」


## 意訳

「身が静まり、言葉が静まり、心静かで、よく三昧に入っている者。
四つの聖なる道によって世間の欲の餌を吐き捨てた比丘こそ、
真に『寂静した者』と呼ばれる。

ここで『身が静まった者』とは、殺生などの行為がないために身が静まった者である。
『語が静まった者』とは、妄語などがないために語が静まった者であり、
『心静かな者』とは、貪りなどがないために心の傾向が静まった者である。
身・語・意の三つすべてがよく定められているので、『よく定まった者』と呼ばれる。

さらに、四つの聖道によって世間の欲の餌を吐き捨てた比丘は、
内面において貪欲などが静まりきっているがゆえに、
『寂静した者』と呼ばれるのである。」

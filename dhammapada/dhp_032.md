32.

‘‘Appamādarato bhikkhu, pamāde bhayadassi vā;

Abhabbo parihānāya, nibbānasseva santike’’ti.

Tattha abhabbo parihānāyāti so evarūpo bhikkhu samathavipassanādhammehi vā maggaphalehi vā parihānāya abhabbo, nāpi pattehi parihāyati, na appattāni na pāpuṇāti. Nibbānasseva santiketi kilesaparinibbānassapi anupādāparinibbānassāpi santikeyevāti.

## 対訳

1. ‘‘Appamādarato bhikkhu, pamāde bhayadassi vā;

* appamāda-rato = 「不放逸を楽しむ」
* bhikkhu = 「比丘は」
* pamāde = 「放逸の中に」
* bhaya-dassī = 「恐れを見る者」
* vā = 「あるいは（または）」

→ 「不放逸を楽しみ、あるいは放逸の中に恐れを見る比丘は、」

2. Abhabbo parihānāya, nibbānasseva santike’’ti.

* abhabbo = 「あり得ない、不能である」
* parihānāya = 「退転することは」
* nibbānassa eva = 「まさに涅槃の（nibbānassa + eva）」
* santike = 「近くに（いる）」
* ti = 「〜と」

→ 「退転することはあり得ず、まさに涅槃の近くにいるのである。」

3. Tattha abhabbo parihānāyāti

* tattha = 「そこにおいて、その中で」
* abhabbo parihānāya ti = 「『退転することはあり得ない』とは」

→ 「そこにおいて、『退転することはあり得ない』とは、」

4. so evarūpo bhikkhu samathavipassanādhammehi vā maggaphalehi vā parihānāya abhabbo,

* so evarūpo bhikkhu = 「そのような比丘は」
* samatha-vipassanā-dhammehi vā = 「止（サマタ）と観（ヴィパッサナー）の法から、あるいは」
* magga-phalehi vā = 「道と果から、あるいは」
* parihānāya = 「退転することは」
* abhabbo = 「あり得ない」

→ 「そのような比丘は、止と観の法から、あるいは道と果から退転することはあり得ない（という意味である）。」

5. nāpi pattehi parihāyati, na appattāni na pāpuṇāti.

* na api = 「〜もなく」
* pattehi = 「すでに得たものから」
* parihāyati = 「退転する（失う）」
* na = 「ない（否定）」
* appattāni = 「未だ得ていないものに」
* na pāpuṇāti = 「到達しないということは（二重否定により「必ず到達する」）」

→ 「すでに得たものから退転して失うこともなく、未だ得ていないものに到達しないということもない。」

6. Nibbānasseva santiketi

* nibbānassa eva santike ti = 「『まさに涅槃の近くに』とは」

→ 「『まさに涅槃の近くに』とは、」

7. kilesaparinibbānassapi anupādāparinibbānassāpi santikeyevāti.

* kilesa-parinibbānassa api = 「煩悩の完全なる消滅（有余涅槃）の」
* anupādā-parinibbānassa api = 「執着なき完全なる消滅（無余涅槃）の」
* santike yeva = 「まさに近くに」
* ti = 「という意味である」

→ 「煩悩の消滅の近くにも、執着なき消滅の近くにも、まさにいるのであるという意味である。」

## 意訳

「不放逸を楽しみ、あるいは放逸の中に恐れを見る比丘は、退転することはあり得ず、まさに涅槃の近くにいるのである。」

ここでの「退転することはあり得ない」とは、そのような比丘が止と観の法から、あるいは道と果から退転して離れてしまうことはあり得ないという意味である。すでに得たものから退転して失うこともなく、未だ得ていないものに到達しないということもない（必ず到達する）。

「まさに涅槃の近くに」とは、煩悩の完全なる消滅（有余涅槃）の近くにも、執着なき完全なる消滅（無余涅槃）の近くにも、まさにいるのであるという意味である。

395.

‘‘Paṃsukūladharaṃ jantuṃ, kisaṃ dhamanisanthataṃ;

Ekaṃ vanasmiṃ jhāyantaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha kisanti paṃsukūlikā hi attano anurūpaṃ paṭipadaṃ pūrentā appamaṃsalohitā ceva honti dhamanisanthatagattā ca, tasmā evamāha. Ekaṃ vanasminti vivittaṭṭhāne ekakaṃ vanasmiṃ jhāyantaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Paṃsukūladharaṃ jantuṃ, kisaṃ dhamanisanthataṃ;

- paṃsukūla-dharaṃ = 「糞掃衣（捨てられた布切れ）を身に着けている（者）」  
  - paṃsukūla = 「塵・ごみの中から拾った布、糞掃衣」  
  - dhara = 「身に帯びる、着る、そのようにしている者」
- jantuṃ = 「生きもの、人物、衆生」ここでは「人」を指す
- kisaṃ = 「やせ衰えた、痩せた」
- dhamani-santhataṃ = 「血管が張りめぐらされている（ように見える）」  
  - dhamanī = 「血管、筋」  
  - santhata = 「張り渡った、浮き出た」

→「糞掃衣をまとった、痩せて血管の浮き出たその人を」

2. Ekaṃ vanasmiṃ jhāyantaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

- ekaṃ = 「ひとりの、単独の」
- vanasmiṃ = 「林の中で、森の中で」
- jhāyantaṃ = 「禅定に入っている、瞑想している」  
  - jhāyati の現在分詞
- taṃ = 「その者を」
- ahaṃ = 「私は」
- brūmi = 「言う、呼ぶ」
- brāhmaṇaṃ = 「バラモン、真の聖者」

→「ひとりで林の中にあって、瞑想しているその人を、私は『バラモン』と呼ぶのである。」

---

3. Tattha kisanti

- tattha = 「ここで、その句において」
- kisaṃ ti = 「『kisaṃ（痩せた）』というのは」

→「ここで『kisaṃ（痩せた）』とは、というに」

4. paṃsukūlikā hi attano anurūpaṃ paṭipadaṃ pūrentā appamaṃsalohitā ceva honti dhamanisanthatagattā ca, tasmā evamāha.

- paṃsukūlikā hi = 「というのも、糞掃衣行者たちは」  
  - paṃsukūlikā = 「糞掃衣を常用する者たち」
- attano anurūpaṃ paṭipadaṃ pūrentā = 「自らにふさわしい行法を全うしているので」  
  - attano = 「自らの」  
  - anurūpaṃ = 「相応した、ふさわしい」  
  - paṭipadaṃ = 「修行のあり方、実践の道」  
  - pūrentā = 「満たしている、果たしている（現在分詞・主格複数）」
- appa-maṃsa-lohitā ceva honti = 「肉と血が少なくなり、痩せた身となり」  
  - appa = 「少ない」  
  - maṃsa = 「肉」  
  - lohita = 「血」  
  - appamaṃsalohitā = 「肉も血も少ない状態の者たち」
- dhamani-santhatagattā ca = 「また、血管の張りめぐらされた体つきになる」  
  - gatta = 「身体」  
  - dhamanisanthata-gattā = 「血管の浮き出た体をした者たち」
- tasmā evaṃ āha = 「だからこのように言ったのである」

→「糞掃衣行者たちは、自分にふさわしい修行の道を実践し続けるゆえに、肉や血が少なくなって痩せ、血管の浮き出た体つきになる。ゆえに、（釈尊は）このように『痩せて血管の張り出た』と言われたのである。」

5. Ekaṃ vanasminti vivittaṭṭhāne ekakaṃ vanasmiṃ jhāyantaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

- ekaṃ vanasmiṃ ti = 「『ekaṃ vanasmiṃ（林の中でひとり）』というのは」
- vivitta-ṭṭhāne = 「人里離れた、寂静な場所で」  
  - vivitta = 「隔絶した、静かな」  
  - ṭhāna = 「場所」
- ekakaṃ = 「ひとりきりの者を」
- vanasmiṃ jhāyantaṃ = 「（その）林の中で瞑想している」
- taṃ ahaṃ brāhmaṇaṃ vadāmi ti attho = 「そのような者を、私は『バラモンである』と言う、という意味である」  
  - taṃ = 「その者を」  
  - vadāmi = 「私は言う」  
  - ti attho = 「という意味である」

→「『林の中でひとり』とは、人里離れた静かな場所で、ひとりきりで林の中にあって瞑想している者を言うのであり、そのような者を私は『バラモンである』と言う、という意味である。」

---

## 意訳

「糞掃衣をまとい、痩せて血管が浮き出るほどの身となり、ただひとり森の奥で静かに禅定にふけっている人──その人こそ、私は真のバラモンと呼ぶ。

ここで『痩せた』と言うのは、糞掃衣行者が自分にふさわしい厳しい修行を実践し続ける結果、肉も血も少なくなり、血管が浮き出た体つきになることを指す。

また『ひとり森の中で』とは、人里離れた静かな場所で、ただひとり林の中にあって瞑想している者のことであり、そのような者をこそ、私はバラモンと呼ぶのである。」

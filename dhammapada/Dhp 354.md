原文（偈 354）

Sabbadānaṃ dhammadānaṃ jināti,
Sabbarasaṃ dhammaraso jināti;
Sabbaratiṃ dhammarati jināti,
Taṇhakkhayo sabbadukkhaṃ jinātī’’ti.

原文（註釈）

Tattha sabbadānaṃ dhammadānanti sacepi hi cakkavāḷagabbhe yāva brahmalokā nirantaraṃ katvā sannisinnānaṃ buddhapaccekabuddhakhīṇāsavānaṃ kadaligabbhasadisāni cīvarāni dadeyya, tasmiṃ samāgame catuppadikāya gāthāya katānumodanāva seṭṭhā. Tañhi dānaṃ tassā gāthāya soḷasiṃ kalaṃ nāgghati. Evaṃ dhammassa desanāpi vācanampi savanampi mahantaṃ. Yena ca puggalena bahūnaṃ taṃ dhammassavanaṃ kāritaṃ, tasseva ānisaṃso mahā. Tathārūpāya eva parisāya paṇītapiṇḍapātassa patte pūretvā dinnadānatopi sappitelādīnaṃ patte pūretvā dinnabhesajjadānatopi mahāvihārasadisānaṃ vihārānañca lohapāsādasadisānañca pāsādānaṃ anekāni satasahassāni kāretvā dinnasenāsanadānatopi anāthapiṇḍikādīhi vihāre ārabbha katapariccāgatopi antamaso catuppadikāya gāthāya anumodanāvasenāpi pavattitaṃ dhammadānameva varaṃ seṭṭhaṃ. Kiṃ kāraṇā? Evarūpāni hi puññāni karontā dhammaṃ sutvāva karonti, no asutvā. Sace hi ime sattā dhammaṃ na suṇeyyuṃ, uḷuṅkamattaṃ yāgumpi kaṭacchumattaṃ bhattampi na dadeyyuṃ. Iminā kāraṇena sabbadānehi dhammadānameva seṭṭhaṃ. Apica ṭhapetvā buddhe ca paccekabuddhe ca sakalakappaṃ deve vassante udakabindūni gaṇetuṃ samatthāya paññāya samannāgatā sāriputtādayopi attano dhammatāya sotāpattiphalādīni adhigantuṃ nāsakkhiṃsu, assajittherādīhi kathitadhammaṃ sutvā sotāpattiphalaṃ sacchikariṃsu, satthu dhammadesanāya sāvakapāramīñāṇaṃ sacchikariṃsu. Imināpi kāraṇena, mahārāja, dhammadānameva seṭṭhaṃ. Tena vuttaṃ – ‘‘sabbadānaṃ dhammadānaṃ jinātī’’ti.


────────────────────────
偈文：語句分解と文法

1) Sabbadānaṃ dhammadānaṃ jināti,
- sabba-dānaṃ = 「あらゆる施与（布施）」(sabba「すべての」+ dāna「布施」)
  - ここは比較の対象としての「一切の布施」。
- dhamma-dānaṃ = 「法の施与／法施（教えを与えること）」
- jināti = 「勝る、打ち勝つ」(√ji)
→ 「あらゆる布施に、法施は勝る。」

2) Sabbarasaṃ dhammaraso jināti;
- sabba-rasaṃ = 「あらゆる味（快楽の味・風味）」(rasa「味、趣味、快感」)
- dhamma-raso = 「法の味（教えを味わう喜び）」(主語扱い)
- jināti = 「勝る」
→ 「あらゆる味に、法の味は勝る。」

3) Sabbaratiṃ dhammarati jināti,
- sabba-ratiṃ = 「あらゆる楽しみ／歓喜」(rati「喜び、楽しみ」)
- dhamma-rati = 「法における歓喜／法を喜ぶこと」
- jināti = 「勝る」
→ 「あらゆる楽しみに、法の歓喜は勝る。」

4) Taṇhakkhayo sabbadukkhaṃ jinātī’’ti.
- taṇhā-kkhayo = 「渇愛の滅尽」(taṇhā「渇愛」+ khaya「滅尽」)
- sabba-dukkhaṃ = 「一切の苦」(dukkha「苦」)
- jināti = 「打ち勝つ」
→ 「渇愛の滅尽は、一切の苦に打ち勝つ。」

偈文：逐語訳
「いかなる布施にも、法の布施は勝る。
いかなる味にも、法の味は勝る。
いかなる楽しみにも、法の楽しみは勝る。
渇愛の滅尽は、一切の苦に打ち勝つ。」

偈文：自然な日本語訳
「どんな布施よりも、教えを分かち与えること（法施）が勝っている。
どんな快楽の味わいよりも、教えの味わいが勝っている。
どんな楽しみよりも、教えを喜ぶ楽しみが勝っている。
そして渇愛が尽きることは、あらゆる苦に勝つのである。」


────────────────────────
註釈：語句分解と文法・内容（文ごと）

A) Tattha sabbadānaṃ dhammadānanti ...
- tattha = 「ここで（この偈において）」(副詞)
- sabbadānaṃ dhammadānaṃ (iti) = 「『一切の布施に法施が勝る』とは（というのは）」
→ 以後、その理由説明。

B) sacepi hi ... cīvarāni dadeyya, ...
- sacepi hi = 「たとえ…であっても」(条件強調)
- cakkavāḷa-gabbhe yāva brahma-lokā = 「輪囲世界（世界体系）の内から梵天界に至るまで」
- nirantaraṃ katvā = 「隙間なく（びっしりと）して」
- sannisinnānaṃ = 「集まって座している者たちに」（sannisinna の複数与格・属格的用法の文脈）
- buddha-paccekabuddha-khīṇāsavānaṃ = 「仏・独覚・漏尽者（阿羅漢）たちの」
- kadali-gabbha-sadisāni cīvarāni = 「芭蕉の幹のように（重なり層をなすほどの）衣を」
- dadeyya = 「与えるとしよう／布施するとしよう」（仮定・願望の語調）
→ 「たとえ、世界体系から梵天界まで隙間なく集まって座る仏・独覚・漏尽者たちに、芭蕉の幹のように（大量に重なった）衣を与えるとしても、」

C) tasmiṃ samāgame ... katānumodanāva seṭṭhā.
- tasmiṃ samāgame = 「その集会において」
- catuppadikāya gāthāya = 「四句から成る偈によって」
- katā-anumodanā = 「なされた随喜（功徳の称賛・回向の言葉）」
- eva seṭṭhā = 「まさにそれこそ最上である」
→ 「そのような場では、四句偈によってなされる随喜の言葉こそが最上である。」

D) Tañhi dānaṃ ... soḷasiṃ kalaṃ nāgghati.
- taṃ hi dānaṃ = 「その布施は」
- tassā gāthāya = 「その偈（の功徳）に」
- soḷasiṃ kalaṃ = 「十六分の一（でさえ）」
- na agghati（= nāgghati）= 「値しない、及ばない」
→ 「その（莫大な）布施は、その偈（の功徳）の十六分の一にも及ばない。」

E) Evaṃ dhammassa desanāpi vācanampi savanampi mahantaṃ.
- evaṃ = 「このように」
- dhammassa desanā = 「法の説示（説法）」
- vācanā = 「読誦（唱えること）」
- savana = 「聴聞（聞くこと）」
- mahantaṃ = 「大いなるもの、偉大」
→ 「このように、法を説くことも、唱えることも、聞くことも大いなる功徳である。」

F) Yena ca puggalena ... tasseva ānisaṃso mahā.
- yena puggalena = 「それを（=法の聴聞を）…させたその人によって」
- bahūnaṃ ... kāritaṃ = 「多くの人々に…させられた」
- tassa eva = 「その人こそ」
- ānisaṃso = 「利益、功徳果」
- mahā = 「大きい」
→ 「多くの人に法の聴聞をさせた人、その人の利益（功徳果）は大きい。」

G) Tathārūpāya eva parisāya ... dhammadānameva varaṃ seṭṭhaṃ.
（長い列挙：どんな大布施よりも、法施が最上だ、という強調）
- paṇīta-piṇḍapātassa patte pūretvā dinnadānato pi = 「上等な食の托鉢物で鉢を満たして与える布施よりも」
- sappi-telādīnaṃ ... dinnabhesajja-dānato pi = 「酥油や油などの薬（医薬）を鉢いっぱいにして与える施薬よりも」
- mahā-vihāra-sadisānaṃ vihārānaṃ ... loha-pāsāda-sadisānaṃ pāsādānaṃ ... senāsana-dānato pi
  = 「大寺院のような僧院、ローハパーサーダのような宮殿楼閣を、何十万と建てて住処を布施する施住よりも」
- anāthapiṇḍikādīhi ... katapariccāgato pi = 「アナータピンディカ等が僧院を建立して行ったような大寄進よりも」
- antamaso ... anumodanāvasena pi pavattitaṃ dhammadānameva varaṃ seṭṭhaṃ
  = 「最小限に言っても、四句偈の随喜という形で行われる法施こそが、最上・最勝である」
→ 物質的な巨大布施をいくら並べても、教えを示し・聞かせ・随喜させる法施が上だ、とまとめる。

H) Kiṃ kāraṇā? ... no asutvā.
- kiṃ kāraṇā? = 「なぜか？」
- evarūpāni ... puññāni karontā = 「このような功徳行を行う人々は」
- dhammaṃ sutvā eva = 「法を聞いてこそ」
- karonti = 「行う」
- no asutvā = 「聞かずに（行うの）ではない」
→ 「なぜなら、人は法を聞いてこそ、そのような善行をするのであって、聞かずにはしないからである。」

I) Sace hi ime sattā ... na dadeyyuṃ.
- sace hi = 「もし本当に…なら」
- ime sattā = 「これらの衆生は」
- dhammaṃ na suṇeyyuṃ = 「法を聞かないなら」（仮定）
- uḷuṅka-mattaṃ yāguṃ pi = 「ひしゃく一杯ほどの粥でさえ」
- kaṭacchu-mattaṃ bhattaṃ pi = 「杓子一杯ほどの飯でさえ」
- na dadeyyuṃ = 「与えないだろう」
→ 「もし衆生が法を聞かなかったなら、ひしゃく一杯の粥すら、杓子一杯の飯すら施さないだろう。」

J) Iminā kāraṇena ... dhammadānameva seṭṭhaṃ.
- iminā kāraṇena = 「この理由によって」
- sabba-dānehi = 「あらゆる布施の中で」
- dhamma-dānaṃ eva = 「法施こそ」
- seṭṭhaṃ = 「最上」
→ 「ゆえに、一切の布施の中で法施こそ最上である。」

K) Apica ... (もう一つの理由：智慧第一の人々ですら“教えを聞いて”証果した)
- apica = 「さらにまた」
- ṭhapetvā buddhe ca paccekabuddhe ca = 「仏と独覚を除けば」
- sakalakappaṃ deve vassante ... udakabindūni gaṇetuṃ samatthāya paññāya = 「一劫の間、天が雨を降らせるならその水滴を数えられるほどの智慧によって」
- samannāgatā = 「具えた」
- sāriputtādayo pi = 「サーリプッタ等でさえ」
- attano dhammatāya ... adhigantuṃ nāsakkhiṃsu = 「自力だけでは、預流果などを証得できなかった」
- assaji-therādīhi kathita-dhammaṃ sutvā = 「アッサジ長老らの説いた法を聞いて」
- sotāpatti-phalaṃ sacchikariṃsu = 「預流果を現証した」
- satthu dhammadesanāya ... sāvakapāramīñāṇaṃ sacchikariṃsu
  = 「師（世尊）の説法によって、声聞波羅蜜の智を現証した」
→ 「仏・独覚を除けば、途方もない智慧を具えたサーリプッタ等ですら自力だけで証果できず、他者が説く法を“聞いて”預流果を現証し、師の説法によって声聞としての完成の智を現証した。」

L) Imināpi kāraṇena, mahārāja, dhammadānameva seṭṭhaṃ.
- iminā pi kāraṇena = 「この理由によってもまた」
- mahārāja = 「大王よ」（呼びかけ）
- dhammadānaṃ eva seṭṭhaṃ = 「法施こそ最上」
→ 「この理由によっても、大王よ、法施こそ最上である。」

M) Tena vuttaṃ – ‘‘sabbadānaṃ dhammadānaṃ jinātī’’ti.
- tena vuttaṃ = 「それゆえに言われた」
→ 「だから『一切の布施に法施は勝る』と言われたのである。」


註釈：逐語訳（要点を崩さずに）
「ここで『一切の布施に法施が勝る』とは、たとえ世界体系から梵天界に至るまで隙間なく集まって座る仏・独覚・漏尽者たちに、芭蕉の幹のように重なるほどの衣を布施するとしても、その集会では四句偈による随喜の言葉こそ最上である。その布施は、その偈の功徳の十六分の一にも及ばない。かくのごとく、法を説くこと・唱えること・聞くことは大いなる功徳であり、多くの人々に法の聴聞をさせた人の利益は大きい。上等な食の布施、医薬の布施、僧院や楼閣を何十万も建てて住処を施す布施、アナータピンディカ等の大寄進、そうしたものよりも、最小限に言っても四句偈の随喜という形で行われる法施こそ最上である。なぜなら、人は法を聞いてこそ布施等の功徳行を行い、聞かなければひしゃく一杯の粥すら与えないからである。さらにまた、仏と独覚を除けば、サーリプッタ等の大智慧者でさえ自力だけでは証果できず、他者の説く法を聞いて預流果を現証し、師の説法によって声聞の完成の智を現証した。この理由によっても、法施こそ最上である。ゆえに『一切の布施に法施は勝る』と言われたのである。」

註釈：自然な日本語訳
「この偈で言う『どんな布施よりも法施が勝る』とはこういう意味である。たとえ宇宙いっぱいに集まった仏や独覚や阿羅漢たちに、衣が幾重にも積み重なるほどの大布施をしたとしても、その場で四句の偈をもって功徳を称え、教えを示す随喜の言葉こそが最上なのだ。物の布施は、その偈の功徳の十六分の一にすら届かない。だから、法を説くこと、唱えること、聞くことの功徳は非常に大きい。さらに、多くの人に法を聞かせた人の功徳果は格別である。上等な食事を満たして施す布施や、医薬を満たして施す布施や、巨大な寺院や楼閣を何十万と建てて住処を施す布施、アナータピンディカたちのような大寄進──そうしたものよりも、たとえ四句偈の随喜という形であっても、教えを与える法施が最上である。なぜなら、人は教えを聞いてこそ善行を起こし、教えを聞かなければ、ひしゃく一杯の粥すら施さないからだ。しかも、仏と独覚を別にすれば、サーリプッタのような大智慧の弟子でさえ、ただ自分の力だけでは預流果などを得られず、他者が語る法を聞いて初めて預流果を証し、師の説法によって弟子としての完成の智慧を得た。この点からも、法施こそ最上である。だから『一切の布施に法施は勝る』と言われるのである。」

原文

Sabbe pana gandharasādayopi rasā ukkaṃsato devatānaṃ sudhābhojanarasopi saṃsāravaṭṭe pātetvā dukkhānubhavanasseva paccayo.
Yo panesa sattatiṃsabodhipakkhiyadhammasaṅkhāto ca navalokuttaradhammasaṅkhāto ca dhammaraso, ayameva sabbarasānaṃ seṭṭho.
Tena vuttaṃ – ‘‘sabbarasaṃ dhammaraso jinātī’’ti .

Yāpesā puttaratidhīturatidhanaratiitthiratinaccagītavāditādiratipabhedā ca anekappabhedā ratī, sāpi saṃsāravaṭṭe pātetvā dukkhānubhavanasseva paccayo.
Yā panesā dhammaṃ kathentassa vā suṇantassa vā vācentassa vā anto uppajjamānā pīti udaggabhāvaṃ janeti, assūni pavatteti, lomahaṃsaṃ janeti, sāyaṃ saṃsāravaṭṭassa antaṃ katvā arahattapariyosānā hoti.
Tasmā sabbaratīnaṃ evarūpā dhammaratiyeva seṭṭhā.
Tena vuttaṃ – ‘‘sabbaratiṃ dhammarati jinātī’’ti

taṇhakkhayo pana taṇhāya khayante uppannaṃ arahattaṃ sakalassapi vaṭṭadukkhassa abhibhavanato sabbaseṭṭhameva.
Tena vuttaṃ – ‘‘taṇhakkhayo sabbadukkhaṃ jinātī’’ti.


────────────────────────
語句分解と文法・内容（文ごと）

1) Sabbe pana gandharasādayopi rasā ... paccayo.
- sabbe ... rasā = 「あらゆる…味は」
- gandha-rasa-ādayo pi = 「香りの味などもまた」（gandha「香り」+ rasa「味」+ ādi「など」）
- ukkaṃsato = 「極致において、最高級のものとしてさえ」
- devatānaṃ sudhā-bhojana-rasa pi = 「天人たちの甘露（sudhā）の食の味でさえも」
- saṃsāra-vaṭṭe pātetvā = 「輪廻の渦（vaṭṭa）へ投げ落として」
- dukkha-anubhavanassa eva paccayo = 「苦を経験することの、まさに原因（条件）である」
→ 「香りの味など、あらゆる味は、極上のもの、たとえば天人の甘露の食の味でさえ、輪廻へと落とし、苦を味わわせる原因にすぎない。」

2) Yo panesa ... dhammaraso, ayameva sabbarasānaṃ seṭṭho.
- yo pana esa ... dhammaraso = 「しかしこの“法の味”は」
- sattatiṃsa-bodhipakkhiya-dhamma-saṅkhāto ca
  = 「三十七菩提分法（bodhipakkhiya-dhamma）と称されるものでもあり」
- nava-lokuttara-dhamma-saṅkhāto ca
  = 「九つの出世間法（lokuttara-dhamma）と称されるものでもある」
- ayaṃ eva = 「これこそまさに」
- sabba-rasānaṃ seṭṭho = 「一切の味の中で最上」
→ 「しかし、この三十七菩提分法であり、また九つの出世間法でもある“法の味”こそ、あらゆる味の中で最上である。」

※ 九つの出世間法（nava lokuttara dhammā）は、一般に
  四道（sotāpatti-, sakadāgāmi-, anāgāmi-, arahattamagga）
  ＋四果（…phala）
  ＋涅槃（nibbāna）
  の九を指す、という理解で読めます。

3) Tena vuttaṃ – ‘‘sabbarasaṃ dhammaraso jinātī’’ti.
- tena = 「それゆえに」
- vuttaṃ = 「言われた」
→ 「それゆえに『あらゆる味に、法の味は勝る』と言われたのである。」

4) Yāpesā ... ratī, sāpi ... paccayo.
- yāpi esā ... ratī = 「またこの（世間の）楽しみ（rati）も」
- putta-rati = 「息子を喜ぶ楽しみ」
- dhītu-rati = 「娘を喜ぶ楽しみ」
- dhana-rati = 「財を喜ぶ楽しみ」
- itthi-rati = 「異性（女性）に対する楽しみ」
- nacca-gīta-vāditā-ādi-rati-pabhedā = 「舞踊・歌・演奏などの楽しみの種別」
- ca anekappabhedā ratī = 「そして多種多様な楽しみ」
- sā pi ... paccayo = 「それもまた…原因」
→ 「息子・娘・財・異性、舞踊や歌や演奏など、さまざまな楽しみも、輪廻へ落とし、苦を経験させる原因にすぎない。」

5) Yā panesā ... arahattapariyosānā hoti.
- yā pana esā = 「しかしこちらの（楽しみ）は」
- dhammaṃ kathentassa vā suṇantassa vā vācentassa vā = 「法を説く者、聞く者、誦する者の」
- anto uppajjamānā = 「内に生じてくる」
- pīti = 「喜（よろこび、歓喜）」
- udagga-bhāvaṃ janeti = 「高揚を生じさせる」
- assūni pavatteti = 「涙を流させる」
- loma-haṃsaṃ janeti = 「鳥肌（毛が逆立つ感動）を起こさせる」
- sā ayaṃ ... hoti = 「その喜びは…である」
- saṃsāra-vaṭṭassa antaṃ katvā = 「輪廻の渦の終わりをなして」
- arahattapariyosānā = 「阿羅漢果を究竟（到達点）とする」
→ 「しかし、法を説く・聞く・誦するその人の内に起こる喜びは、高揚を生み、涙を流させ、身の毛のよだつ感動を起こさせる。その喜びは輪廻を終わらせ、阿羅漢果を到達点とするのである。」

6) Tasmā sabbaratīnaṃ evarūpā dhammaratiyeva seṭṭhā.
- tasmā = 「それゆえに」
- sabba-ratīnaṃ = 「あらゆる楽しみの中で」
- evarūpā dhamma-rati = 「このような法の歓喜（法を喜ぶ楽しみ）」
- eva seṭṭhā = 「こそ最上」
→ 「ゆえに、あらゆる楽しみの中で、このような法の歓喜こそ最上である。」

7) Tena vuttaṃ – ‘‘sabbaratiṃ dhammarati jinātī’’ti.
→ 「それゆえに『あらゆる楽しみに、法の歓喜は勝る』と言われたのである。」

8) taṇhakkhayo pana ... sabbaseṭṭhameva.
- taṇhā-kkhayo pana = 「さらに、渇愛の滅尽は」
- taṇhāya khayante = 「渇愛が滅しているときに」
- uppannaṃ arahattaṃ = 「生起した阿羅漢果（阿羅漢の境地）であり」
- sakalassa api vaṭṭa-dukkhassa = 「輪廻に属する苦のすべてに対しても」
- abhibhavanato = 「打ち勝つことによって」
- sabba-seṭṭhaṃ eva = 「まさに最上である」
→ 「渇愛の滅尽とは、渇愛が滅したときに現れる阿羅漢果であり、輪廻の苦の全体に打ち勝つがゆえに、何よりも最上である。」

9) Tena vuttaṃ – ‘‘taṇhakkhayo sabbadukkhaṃ jinātī’’ti.
→ 「それゆえに『渇愛の滅尽は、一切の苦に勝つ』と言われたのである。」

逐語訳（全体）

「香りの味など、どのような味であっても、極上のもの──天人の甘露の食の味でさえ──輪廻へと落とし、苦を経験させる原因にすぎない。
しかし、三十七菩提分法であり、九つの出世間法でもあるこの“法の味”こそ、あらゆる味の中で最上である。
ゆえに『あらゆる味に、法の味は勝る』と言われる。

また、息子や娘や財や異性、舞踊・歌・演奏など、無数の種類の楽しみも、輪廻へ落とし、苦を経験させる原因にすぎない。
しかし、法を説く者・聞く者・誦する者の内に起こる喜びは、高揚を生み、涙を流させ、身の毛のよだつ感動を起こさせる。そしてそれは輪廻を終わらせ、阿羅漢果を到達点とする。
だから、あらゆる楽しみの中で、このような法の歓喜こそ最上である。
ゆえに『あらゆる楽しみに、法の歓喜は勝る』と言われる。

さらに渇愛の滅尽とは、渇愛が滅したときに現れる阿羅漢果であり、輪廻の苦のすべてに打ち勝つがゆえに、まさに最上である。
ゆえに『渇愛の滅尽は、一切の苦に勝つ』と言われる。」

自然な日本語訳

「香りや味わいといった快楽は、どれほど上等であっても──天人が味わう甘露の食でさえ──結局は輪廻に沈ませ、苦を繰り返させる条件にしかならない。
それに対して、“法の味”とは、三十七の菩提分法であり、九つの出世間法（四道・四果・涅槃）として成り立つもので、これこそあらゆる味の中で最上である。だから『どんな味にも、法の味が勝る』と言われる。

同様に、子どもや財産や異性をめぐる喜び、舞踊や歌や演奏の楽しみなど、世間の楽しみは多種多様だが、それも輪廻に落として苦を招く原因にすぎない。
しかし、法を説いたり、聞いたり、誦したりする者の内に生じる歓喜は、人を高揚させ、ときに涙を流させ、強い感動を起こさせる。そしてその歓喜は輪廻を終わらせ、阿羅漢果へと至らせる。ゆえに、あらゆる楽しみの中で、この法の歓喜こそが最上である。だから『どんな楽しみにも、法の楽しみが勝る』と言われる。

さらに渇愛の滅尽とは、渇愛が尽きたところに現れる阿羅漢の境地であり、輪廻に属するあらゆる苦を制するがゆえに、最上である。だから『渇愛の滅尽は一切の苦に勝つ』と言われるのである。」

415.

‘‘Yodha kāme pahantvāna, anāgāro paribbaje;

Kāmabhavaparikkhīṇaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tassattho – yo puggalo idha loke ubhopi kāme hitvā anāgāro hutvā paribbajati, taṃ parikkhīṇakāmañceva parikkhīṇabhavañca ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Yodha kāme pahantvāna, anāgāro paribbaje;

* yo = 「…する者は」, 「～なる人は」
* dha = 強意・指示の語（「まさに、その」「さて」）
* kāme = 「欲楽を」（kāma の対格複数）  
  * ここでは五欲の対象・感官的欲望
* pahantvāna = 「捨て去って、断ち切って」（pajahati の絶対分詞）
* anāgāro = 「家なき者として、在家を離れた者として」, 「出家者として」
* paribbaje = 「遊行すべし、修行生活を送るであろう」（paribbajati の願望法・三人称単数）

→「まさに、欲楽を捨て去って、家なき者として遊行生活を送る人は」

2. Kāmabhavaparikkhīṇaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* kāma-bhava-parikkhīṇaṃ = 「欲と有（生存・輪廻としての存在）を完全に尽くした者を」
  * kāma = 「欲、感官的欲楽」
  * bhava = 「有、存在、輪廻としての生存」
  * parikkhīṇa = 「すっかり尽きた、完全に滅した」（parikkhayati の過去分詞）
  * 連結して「欲と有とが完全に滅した者」
* taṃ = 「その者を」
* ahaṃ = 「わたしは」
* brūmi = 「言う、呼ぶ」
* brāhmaṇaṃ = 「バラモンであると」, 「真のバラモンと」
* ti = 引用の終わり
* nti = 「と（言われている／と）」の意

→「欲と有とが完全に滅したその者を、わたしは『バラモン』と呼ぶのである」

3. Tassattho –

* tassa = 「それ（この偈）の」
* attho = 「意味は」
→「その意味は、次のようである。」

4. yo puggalo idha loke ubhopi kāme hitvā anāgāro hutvā paribbajati,

* yo puggalo = 「かの人、かの人物は」
  * yo = 関係代名詞「…する者は」
  * puggalo = 「人、人格」
* idha loke = 「この世において、この世界において」
  * idha = 「ここで、この世で」
  * loke = 「世の中で、世界において」
* ubho pi kāme = 「両方の欲を（ともに）」
  * ubho = 「両方、二種」
  * pi = 「～もまた、～さえ」
  * kāme = 「欲を、欲楽を」（対格複数）
    * 註釈的には、対象としての欲（vatthukāma）と、心の欲（kilesakāma）の二種を指す
* hitvā = 「捨てて、離れ去って」（jahati の絶対分詞）
* anāgāro hutvā = 「家なき者となって」, 「出家者となって」
  * anāgāro = 「家なき者、在家を離れた者」
  * hutvā = 「～となって」（hoti の絶対分詞）
* paribbajati = 「遊行生活を送る、出家修行を行う」

→「この世において、両種の欲をともに捨てて家なき者となり、出家の修行生活を送る人は」

5. taṃ parikkhīṇakāmañceva parikkhīṇabhavañca ahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「その人を」
* parikkhīṇa-kāmaṃ ceva = 「欲が完全に尽きた者であると、そしてまた」
  * parikkhīṇa = 「すっかり尽きた、完全に滅した」
  * kāmaṃ = 「欲、感官的欲望」
  * ca eva = 「そしてまさに、そしてまた」
* parikkhīṇa-bhavañca = 「有（生存）もまた完全に尽きた者であると」
  * bhavaṃ = 「有、存在、生存」
  * ca = 「そして」
* ahaṃ = 「わたしは」
* brāhmaṇaṃ vadāmi = 「バラモンであると言う」
  * vadāmi = 「言う、呼ぶ」
* ti attho = 「という意味である」

→「その人を、わたしは、欲が完全に尽き、さらに有（生存）も完全に尽きたバラモンであると言うのである、という意味である。」


## 意訳

「欲楽を断ち切り、在家を離れて家なき出家者として遊行生活を送る人――  
その者のうちに、欲もまた生存としての有もすっかり滅びているなら、  
その人こそ、わたしは真の『バラモン』と呼ぶのである。」

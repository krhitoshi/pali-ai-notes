382.

‘‘Yo have daharo bhikkhu, yuñjati buddhasāsane;

Somaṃ lokaṃ pabhāseti, abbhā muttova candimā’’ti.

Tattha yuñjatīti ghaṭati vāyamati. Pabhāsetīti so bhikkhu attano arahattamaggañāṇena abbhādīhi mutto candimā viya lokaṃ khandhādibhedaṃ lokaṃ obhāseti, ekālokaṃ karotīti attho.

## 対訳

1. ‘‘Yo have daharo bhikkhu, yuñjati buddhasāsane;

* yo = 「…である者は」
* have = 「まさに、実に」（強調の助詞）
* daharo = 「若い、年少の」
* bhikkhu = 「比丘、出家僧」
  → yo have daharo bhikkhu = 「まことに若き比丘であって、…する者は」
* yuñjati = 「身を入れて励む、専心する、精進する」
* buddha-sāsane = 「ブッダの教え（教えの体系・教法）の中で／において」

→「まことに、若い比丘であって、ブッダの教えに身を入れて励む者は」

2. Somaṃ lokaṃ pabhāseti, abbhā muttova candimā’’ti.

* so = 「そのようなその者は」
* imaṃ → maṃ（同化して）= 「この」
  → so imaṃ → somaṃ = 「その者はこの…を」
* lokaṃ = 「世間、この世、人々の世界」
* pabhāseti = 「照らし出す、光を与える」
* abbhā = 「雲（abbha の複数）」
* mutto = 「解き放たれた、離れた、晴れた」
* va = 「…のように」
* candimā = 「月」

→「その者は、雲から解き放たれた月のように、この世を照らし出すのである。」

3. Tattha yuñjatīti ghaṭati vāyamati.

* tattha = 「ここで、この句においては」
* yuñjati ti = 「『yuñjati』というのは」
* ghaṭati = 「懸命に取り組む」
* vāyamati = 「努力する、励む」

→「ここで『yuñjati』とは、『懸命に取り組み、努力する』という意味である。」

4. Pabhāsetīti so bhikkhu attano arahattamaggañāṇena abbhādīhi mutto candimā viya lokaṃ khandhādibhedaṃ lokaṃ obhāseti, ekālokaṃ karotīti attho.

* pabhāseti ti = 「『pabhāseti』とは」
* so bhikkhu = 「その比丘は」
* attano = 「自らの」
* arahatta-magga-ñāṇena = 「阿羅漢道の智慧（阿羅漢の道を悟る智慧）によって」
* abbhā-dīhi = 「雲などによって」
  * abbha = 「雲」
  * ādi = 「…など」
  → abbhādīhi mutto = 「雲などから解き放たれた」
* candimā viya = 「月のように」
* lokaṃ = 「世間を」
* khandha-ādi-bhedaṃ lokaṃ = 
  * khandha = 「蘊（五蘊）」
  * ādi = 「…など」
  * bhedaṃ = 「区別・分類された、分けられた」
  → 「五蘊などの区別によって構成されている世間を」
* obhāseti = 「照らし出す、明らかにする」
* ekālokaṃ = 「一つの光の場にする、全体を明るくする」
  * eka = 「一つの」
  * āloka = 「光、明るさ」
* karoti = 「…とする、…にする」
* ti attho = 「という意味である」

→「『pabhāseti』とは、この比丘が、自らの阿羅漢道の智慧によって、雲などから解き放たれた月のように、五蘊などの区別によって成り立つ世間を照らし出し、全体を一つの明るさにしてしまう、という意味である。」

---

## 意訳

「真に、若い比丘であっても、ブッダの教えに身を入れて励む者は、
雲から解き放たれた月のように、この世を照らし出す。

ここで『励む（yuñjati）』とは、懸命に取り組み努力することである。
『照らし出す（pabhāseti）』とは、その比丘が自らの阿羅漢道の智慧によって、
雲から解き放たれた月のように、五蘊などによって成り立つこの世間を
照らし明らかにし、全体を一面の明るさにしてしまうことである。」

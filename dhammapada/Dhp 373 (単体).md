373.

‘‘Suññāgāraṃ paviṭṭhassa, santacittassa bhikkhuno;

Amānusī ratī hoti, sammā dhammaṃ vipassato.

Suññāgāraṃ paviṭṭhassāti kismiñcideva vivittokāse kammaṭṭhānaṃ avijahitvā kammaṭṭhānamanasikārena nisinnassa. Santacittassāti nibbutacittassa. Sammāti hetunā kāraṇena dhammaṃ vipassantassa vipassanāsaṅkhātā amānusī rati aṭṭhasamāpattisaṅkhātā dibbāpi rati hoti uppajjatīti attho.

## 対訳

1. Suññāgāraṃ paviṭṭhassa, santacittassa bhikkhuno;

* suññāgāraṃ = 「空（くう／から）の住処・部屋・閑居」
  * suñña = 「空である、何もない、人がいない」
  * āgāra = 「家、住処」
* paviṭṭhassa = 「入った者の、入りこんだ者の」
  * √vis + pa + 入る → paviṭṭha（過去分詞）＋属格
* santa-cittassa = 「静まった心をもつ（＝寂静な心の）」
  * santa = 「静まった、寂静な、安らいだ」
  * citta = 「心」
* bhikkhuno = 「比丘の」

→「空寂な住処に入り、心静かな比丘においては」

---

2. Amānusī ratī hoti, sammā dhammaṃ vipassato.

* amānusī = 「人間ならざる、非人間的な、超人的な」
  * ここでは「通常の人間のレベルを超えた」という意味合い
* ratī = 「喜び、楽しみ、悦び」
* hoti = 「ある、生ずる」
* sammā = 「正しく、まさに、しかるべき因をもって」
* dhammaṃ = 「法を、諸法を、対象を」
* vipassato = 「洞察している者には、内観している者にとって」
  * vipassati（観察する、洞察する）の現在分詞属格

→「正しく法を洞察している者には、人間を超えた喜びが生じる。」

---

3. Suññāgāraṃ paviṭṭhassāti kismiñcideva vivittokāse kammaṭṭhānaṃ avijahitvā kammaṭṭhānamanasikārena nisinnassa.

* Suññāgāraṃ paviṭṭhassā ti = 「『suññāgāraṃ paviṭṭhassa』とは、…という意味である。」
* kismiñci-deva = 「いかなる～であれ、どれかの」
  * kismiñci = 「何かしらの、どれか一つの」
  * -eva（deva） = 「まさに、～だけ」
* vivitta-okāse = 「閑静な場所において、他から隔絶した所に」
  * vivitta = 「離れた、隔絶した、僻遠の、閑静な」
  * okāsa = 「場所」
* kammaṭṭhānaṃ = 「業処（ごうじょ）、瞑想の対象（瞑想テーマ）を」
* avijahitvā = 「捨てないで、放棄しないで、手放さないで」
  * vi + jahati（捨てる）＋不定詞的構文
* kammaṭṭhāna-manasikārena = 「業処を念じることによって、業処を心にかけることをもって」
  * manasikāra = 「注意・念じること、意識の向け方」
* nisinnassa = 「坐している者において、座っている者の」

→「『空寂な住処に入った』とは、どこかの閑静な場所において、業処（瞑想対象）を捨てることなく、その業処を念じつつ坐っている者のことである。」

---

4. Santacittassāti nibbutacittassa.

* santa-cittassā ti = 「『santacittassa』とは…という意味である。」
* nibbutacittassa = 「冷め静まった心をもつ者の」
  * nibbuta = 「消えた、鎮まった、涼んだ、涅槃した（火が消えたように冷えた）」  
  → ここでは「煩悩の炎が静まり、安らいだ心」

→「『心静かな（santacittassa）』とは、冷め静まった心をもつ者のことである。」

---

5. Sammāti hetunā kāraṇena dhammaṃ vipassantassa vipassanāsaṅkhātā amānusī rati aṭṭhasamāpattisaṅkhātā dibbāpi rati hoti uppajjatīti attho.

* Sammā ti = 「『sammā』とは…という意味である。」
* hetunā kāraṇena = 「因によって、理由によって、しかるべき根拠をもって」
  * hetu = 「因、原因、理由」
  * kāraṇa = 「理由、根拠、原因」
* dhammaṃ vipassantassa = 「法を洞察している者には、諸法を観察している者にとって」
  * vipassantassa = 「洞察しつつある（現在分詞属格）」
* vipassanā-saṅkhātā = 「『ヴィパッサナー』と名づけられる、ヴィパッサナーと呼ばれる」
  * saṅkhāta = 「…と名づけられた」
* amānusī rati = 「人間を超えた喜び」
* aṭṭha-samāpatti-saṅkhātā = 「八つの等至（八解脱・八定）と名づけられる」
  * aṭṭha = 「八」
  * samāpatti = 「等至、（禅）定の到達、八等至（四静慮＋四無色定）」
* dibbā pi rati = 「神々のような、天的な喜びさえも」
  * dibba = 「天の、神的な、清浄で微妙な」
* hoti uppajjati = 「ある、生じる、起こる」
* ti attho = 「という意味である。」

→「『正しく(sammā)法を洞察している者には人間を超えた喜びがある』とは、しかるべき因・理由によって法を洞察している者には、『ヴィパッサナー』と名づけられる人間を超えた喜び、さらに『八等至（八つの定）』と名づけられる天的な喜びさえも生じる、という意味である。」

---

## 意訳

「空寂な住処に入り、心静かな比丘には、  
正しく法を洞察しているかぎり、  
人の域を超えた深い喜びが湧き起こる。

ここで『空寂な住処に入る』とは、  
どこかの閑静な場所において、  
選び取った瞑想対象を手放さず、それに心を向けつつ静かに坐ることをいう。

『心静かな』とは、煩悩の炎が鎮まり、冷ややかに安らいだ心をもつことである。

このように、正当な因・道理にもとづいて法を洞察する者には、  
『ヴィパッサナー（内観）』と呼ばれる人間を超えた喜びが生じ、  
さらには、四禅と四無色定から成る『八等至』と名づけられる  
天的な喜びさえも生じるのである。」

372.

‘‘Natthi jhānaṃ apaññassa, paññā natthi ajhāyato;

Yamhi jhānañca paññā ca, sa ve nibbānasantike.

Natthi jhānanti jhānuppādikāya vāyāmapaññāya apaññassa jhānaṃ nāma natthi. Paññā natthīti ajhāyantassa ‘‘samāhito bhikkhu yathābhūtaṃ jānāti passatī’’ti vuttalakkhaṇā paññā natthi. Yamhi jhānañca paññā cāti yamhi puggale idaṃ ubhayampi atthi, so nibbānassa santike ṭhitoyevāti attho.

## 対訳

1. Natthi jhānaṃ apaññassa,

* natthi = 「ない、存在しない」
* jhānaṃ = 「禅定は」（jhāna = 「静慮・禅定」）
* apaññassa = 「智慧を欠いている者には、智慧のない者にとっては」
  * a- = 「無い、～のない」
  * paññā = 「智慧」
  * -ssa = 属格語尾「～の、～にとって」

→「智慧なき者には、禅定というものは存在しない。」

2. paññā natthi ajhāyato;

* paññā = 「智慧は」
* natthi = 「ない、存在しない」
* ajhāyato = 「禅定に入らない者には、修禅しない者には」
  * a- = 否定接頭辞「～しない」
  * jhāyati = 「禅定に入る、静慮する」
  * -to = 属格形「～の者に」

→「禅定を行なわない者には、智慧というものは存在しない。」

3. Yamhi jhānañca paññā ca,

* yamhi = 「いずれの（人）の中に、どの（人）において」
* jhānaṃ ca = 「禅定も」
* paññā ca = 「智慧も」
* ca ... ca = 「～も～も」

→「どの人において禅定もまた智慧も（ともに）あるならば」

4. sa ve nibbānasantike.

* sa = 「その人は、そのような者は」
* ve = 強意辞「まさに、確かに」
* nibbāna-santike = 「涅槃の近くに、涅槃のそばに」
  * nibbāna = 「涅槃」
  * santika = 「近く、そば」

→「その人こそ、まさしく涅槃の近くにいる。」

---

5. Natthi jhānanti jhānuppādikāya vāyāmapaññāya apaññassa jhānaṃ nāma natthi.

* Natthi jhānaṃ ti = 「『禅定はない』とは、というのは」
  * ti = 引用を示す語
* jhānuppādikāya = 「禅定を生起させる（ような）」
  * jhāna = 「禅定」
  * uppādika = 「生じさせる、起こさせる」
* vāyāma-paññāya = 「精進としての智慧において、努力という形の智慧が」
  * vāyāma = 「精進、努力」
  * paññā = 「智慧」
* apaññassa = 「智慧を欠いている者にとっては」
* jhānaṃ nāma = 「いわゆる『禅定』というものは」
* natthi = 「存在しない」

→「『禅定はない』とは、禅定を生起させる精進の智慧を欠いている者には、『禅定』と名づけられるものは存在しない、ということである。」

6. Paññā natthīti ajhāyantassa ‘‘samāhito bhikkhu yathābhūtaṃ jānāti passatī’’ti vuttalakkhaṇā paññā natthi.

* Paññā natthī ti = 「『智慧はない』とは」
* ajhāyantassa = 「禅定に入らないでいる者には、修禅していない者には」
  * a- + jhāyanta（jhāyati の現在分詞）+ -ssa（属格）= 「禅定していない者の（ところに）」
* ‘‘samāhito bhikkhu yathābhūtaṃ jānāti passatī’’ ti = 「『心を定めた比丘は、ありのままに知り、ありのままに見る』と説かれている」
  * samāhito = 「よく定められた（心）」
  * bhikkhu = 「比丘」
  * yathābhūtaṃ = 「如実に、ありのままに」
  * jānāti = 「知る」
  * passatī = 「見る」
* vutta-lakkhaṇā paññā = 「そのように特徴づけられている智慧、かく説かれている性質を持つ智慧」
  * vutta = 「説かれた、言われた」
  * lakkhaṇa = 「特徴、性質」
* natthi = 「存在しない」

→「『智慧はない』とは、禅定を修しない者には、『心を定めた比丘は、ありのままに知り、ありのままに見る』と説かれているような性質の智慧は存在しない、ということである。」

7. Yamhi jhānañca paññā cāti yamhi puggale idaṃ ubhayampi atthi, so nibbānassa santike ṭhitoyevāti attho.

* Yamhi jhānañca paññā cā ti = 「『どの人において禅定も智慧も』とは」
* yamhi puggale = 「どのような人（対象者）のうちに」
  * puggala = 「人、人格」
* idaṃ ubhayaṃ pi = 「この二つのもの（禅定と智慧）の両方が」
  * idaṃ = 「この」
  * ubhayaṃ = 「両者、二つとも」
  * pi = 「～もまた」
* atthi = 「存在する（備わっている）」
* so = 「その人は」
* nibbānassa santike = 「涅槃の近くに」
  * nibbāna = 「涅槃」
  * santika = 「近く」
* ṭhito yeva = 「まさに立っている、まさしく位置している」
  * ṭhita = 「立っている、位置している」
  * yeva = 強意辞「こそ、まさに」
* ti attho = 「という意味である」

→「『どの人において禅定も智慧も』とは、どの人のうちにこの二つがともに備わっているなら、その人はまさしく涅槃の近くに立っているのである、という意味である。」

---

## 意訳

「智慧を欠いている者には、禅定は成り立たない。
また、禅定を修しない者には、智慧も成り立たない。
しかし、禅定と智慧の両方がそなわっている人は、
まさに涅槃のすぐ近くにいるのである。

ここで『禅定はない』とは、禅定を生み出すための精進の智慧を欠いている者には、禅定と呼ばれるものは存在しない、という意味である。
また『智慧はない』とは、禅定を修していない者には、『心を定めた比丘は、ありのままに知り、ありのままに見る』と説かれているような智慧は存在しない、ということである。
そして、『どの人において禅定も智慧もある』とは、その両方を具えている人は、まさしく涅槃の近くに位置している、という意味である。」

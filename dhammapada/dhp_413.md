413.

‘‘Candaṃva vimalaṃ suddhaṃ, vippasannamanāvilaṃ;

Nandībhavaparikkhīṇaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha vimalanti abbhādimalarahitaṃ. Suddhanti nirupakkilesaṃ. Vippasannanti pasannacittaṃ. Anāvilanti kilesāvilattarahitaṃ. Nandībhavaparikkhīṇanti tīsu bhavesu parikkhīṇataṇhaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Candaṃva vimalaṃ suddhaṃ,

* candaṃ va = 「月のように」
  * canda = 「月」
  * va = 「～のように（比況）」
* vimalaṃ = 「汚れがない、瑕疵のない」
* suddhaṃ = 「清らかな、純粋な」

→「汚れなく清らかな、月のように」

2. vippasannam anāvilaṃ;

* vippasannaṃ = 「明澄な、澄みきった、晴れ渡った」
* anāvilaṃ = 「濁りのない、曇りのない」

→「澄みきって、いささかの曇りもなく」

3. Nandībhavaparikkhīṇaṃ,

* nandī-bhava-parikkhīṇaṃ = 「愛着と有（存在）とが、完全に尽きた者」
  * nandī = 「喜楽への愛着・歓喜執着」
  * bhava = 「有・存在・生存（欲界・色界・無色界の諸有）」  
  * parikkhīṇaṃ = 「徹底的に尽きた・断滅した」

→「愛着と諸々の有（存在）をすっかり断じ尽くした者を」

4. tam ahaṃ brūmi brāhmaṇa’’nti.

* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ／言う」
* brāhmaṇaṃ = 「バラモン（真の聖者）」
* ti = （引用の終わりを示す）
* nti = 「～と（かく）」

→「私はその人を、バラモン（真の聖者）と呼ぶのである。」

---

5. Tattha vimalanti abbhādimalarahitaṃ.

* tattha = 「ここで、この句において」
* vimalaṃ ti = 「『vimalaṃ（清浄なる）』というのは」
* abbh-ādi-mala-rahitaṃ = 「雲などの汚れを離れたもの」
  * abbhā = 「雲」
  * ādi = 「など」
  * mala = 「汚れ、垢」
  * rahitaṃ = 「～を離れた、～のない」

→「ここで『清浄なる（vimalaṃ）』とは、雲などの汚れを離れたものをいう。」

6. Suddhanti nirupakkilesaṃ.

* suddhaṃ ti = 「『suddhaṃ（清らかな）』とは」
* nirupakkilesaṃ = 「いかなる汚損（煩悩）もないもの」
  * niru- = 「全く～ない」
  * upakkilesa = 「汚損、煩悩による穢れ」

→「『清らかな』とは、一切の汚損をまったく持たないことをいう。」

7. Vippasannanti pasannacittaṃ.

* vippasannaṃ ti = 「『vippasannaṃ（澄みきった）』とは」
* pasanna-cittaṃ = 「信悦し澄みきった心、澄みきった心」

→「『澄みきった』とは、信悦して澄みきった心をいう。」

8. Anāvilanti kilesāvilattarahitaṃ.

* anāvilaṃ ti = 「『anāvilaṃ（曇りなき）』とは」
* kilesa-āvilatta-rahitaṃ = 「煩悩による濁りを離れたもの」
  * kilesa = 「煩悩」
  * āvilatta = 「濁っていること、汚濁」
  * rahitaṃ = 「～を離れた、ない」

→「『曇りなき』とは、煩悩による濁りを離れたものをいう。」

9. Nandībhavaparikkhīṇanti tīsu bhavesu parikkhīṇataṇhaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

* nandī-bhava-parikkhīṇaṃ ti = 「『nandībhavaparikkhīṇaṃ』とは」
* tīsu bhavesu = 「三つの有（存在）において」
  * tīsu = 「三つの中で」
  * bhavesu = 「諸有の中に（複・処格）」
  * 三有＝欲界有（kāmabhava）・色界有（rūpabhava）・無色界有（arūpabhava）
* parikkhīṇa-taṇhaṃ = 「渇愛を完全に尽くし終えた者」
  * taṇhā = 「渇愛」
  * parikkhīṇa = 「すっかり滅し尽くした」
* taṃ ahaṃ brāhmaṇaṃ vadāmi = 「その者を、私はバラモンと言う」
  * taṃ = 「その者を」
  * vadāmi = 「私は言う／呼ぶ」
* ti attho = 「という意味である」

→「『nandībhavaparikkhīṇaṃ』とは、三つの有において渇愛を完全に滅し尽くした者を、私はバラモンと言う、という意味である。」

---

## 意訳

「雲ひとつない満月のように、汚れなく清らかで、心が澄みきり、いささかの曇りもない者。  
三つの世界における、あらゆる愛着と存在への渇望をすっかり断ち切ったその人を、  
私は真のバラモン（聖なる人）と呼ぶ。」

381.

‘‘Pāmojjabahulo bhikkhu, pasanno buddhasāsane;

Adhigacche padaṃ santaṃ, saṅkhārūpasamaṃ sukha’’nti.

Tassattho – pakatiyāpi pāmojjabahulo bhikkhu buddhasāsane pasādaṃ roceti, so evaṃ pasanno buddhasāsane santaṃ padaṃ saṅkhārūpasamaṃ sukhanti laddhanāmaṃ nibbānaṃ adhigaccheyyāti. 

## 対訳

1. ‘‘Pāmojjabahulo bhikkhu, pasanno buddhasāsane;

* pāmojja-bahulo = 「歓喜（pāmojja）が多い・満ちている」→「大いなる喜びに満ちた」
* bhikkhu = 「比丘、出家修行者」
* pasanno = 「清信を得た、信じ喜んでいる」
* buddha-sāsane = 「ブッダの教え（教導・教法）の中で／に対して」

→ 「大いなる喜びに満ちた比丘で、ブッダの教えに対して清らかな信を得ている者は」

2. Adhigacche padaṃ santaṃ, saṅkhārūpasamaṃ sukha’’nti.

* adhigacche = 「到達すべきである、悟るであろう」（adhigacchati の願望法・未来的ニュアンス）
* padaṃ = 「境地、位、状態」
* santaṃ = 「寂静なる、安らかな」
  → padaṃ santaṃ = 「寂静なる境地」
* saṅkhāra = 「諸行、諸形成（条件づけられたあらゆる構成作用）」
* upasamaṃ = 「鎮静、しずまり、止滅」
  → saṅkhāra-upasamaṃ = 「諸行の鎮静・止滅」
* sukhaṃ = 「安楽、楽・至福」
  → saṅkhārūpasamaṃ sukhaṃ = 「諸行の鎮静という安楽」
* ti = 引用の終わり
* nti = sandhi（「ti」と次の語との結合形）

→ 「諸行の鎮静という安楽、すなわち『寂静なる境地』に到達するであろう、という意味である。」

3. Tassattho –

* tassa = 「その（偈の）」
* attho = 「意味は」
→ 「その意味は、次のようである。」

4. pakatiyāpi pāmojjabahulo bhikkhu buddhasāsane pasādaṃ roceti,

* pakatiyāpi = 「本来からしても、生来すでに、もともと」（pakatiyā = 「本性として、普通に」＋pi = 「～でさえ」）
* pāmojja-bahulo = 「歓喜に満ちた、多くの喜びを持つ」
* bhikkhu = 「比丘は」
* buddha-sāsane = 「ブッダの教えにおいて／ブッダの教えに対して」
* pasādaṃ = 「清浄な信（pasāda＝澄んだ信心・信解）」を
* roceti = 「喜ぶ、好む、向ける、育む」

→ 「本来からして歓喜に満ちた比丘は、ブッダの教えに対して清らかな信心を喜び、育むのであり、」

5. so evaṃ pasanno buddhasāsane

* so = 「その（比丘）は」
* evaṃ = 「このように」
* pasanno = 「信を得た、確信し喜んでいる」
* buddha-sāsane = 「ブッダの教えにおいて」

→ 「そのようにブッダの教えにおいて深い信を得たその比丘は」

6. santaṃ padaṃ saṅkhārūpasamaṃ sukhanti laddhanāmaṃ nibbānaṃ

* santaṃ = 「寂静なる」
* padaṃ = 「境地」
  → santaṃ padaṃ = 「寂静なる境地」
* saṅkhāra-upasamaṃ = 「諸行の鎮静・止滅」
* sukhaṃ = 「安楽、至福」
* ti = 「～と」（引用・名称を示す）
* laddha-nāmaṃ = 「その名を得た、～と呼ばれる」
* nibbānaṃ = 「ニッバーナ（涅槃）」

→ 「『寂静なる境地』『諸行の鎮静という安楽』という名を得ているニッバーナを」

7. adhigaccheyyāti.

* adhigaccheyya = 「到達すべきである、悟るであろう」（adhigacchati の願望法）
* ti = 「～ということである」（説明の終わり）

→ 「悟るであろう・到達すべきである、ということである。」

---

## 意訳

「本来から喜びに満ちた比丘であって、ブッダの教えに深い信を抱く者は、  
その信のおかげで、諸行のすべてが静まりきった安らぎ、  
すなわち『寂静なる境地』『諸行の鎮静という安楽』と呼ばれるニッバーナを  
悟るに至るのである。」

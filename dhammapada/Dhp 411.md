411.

‘‘Yassālayā na vijjanti, aññāya akathaṃkathī;

Amatogadhamanuppattaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha ālayāti taṇhā. Aññāya akathaṃkathīti aṭṭha vatthūni yathābhūtaṃ jānitvā aṭṭhavatthukāya vicikicchāya nibbicikiccho. Amatogadhamanuppattanti amataṃ nibbānaṃ ogāhetvā anuppattaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. Yassālayā na vijjanti,

* yassa = 「その者には／その人においては」（関係代名詞「～である人には」）
* ālayā = 「よりどころ・住処・執着の基盤」→ここでは註により「渇愛（taṇhā）」のこと
* na = 「…ない」
* vijjanti = 「存在する、見いだされる」（複数形）

→ 「その人においては、（渇愛という）執着のよりどころが存在しない者であり」

---

2. aññāya akathaṃkathī;

* aññāya = 「智によって、悟りによって」（aññā＝「悟り・知慧」の具格）
* akathaṃkathī = 「疑惑のない者、問い惑いのない者」
  * a- = 「ない、無～」
  * kathaṃkathā = 「どうなのかという迷い・疑惑」

→ 「悟りの智によって、一切の疑いのない者であり」

---

3. Amatogadhamanuppattaṃ,

* amatogadham = amata + ogadhaṃ
  * amata = 「不死のもの、死なないもの」→涅槃の別名
  * ogadhaṃ = 「没入したもの、浸されたもの、中に入り込んだもの」
* anuppattaṃ = 「到達した者、到達している者」

→ 「不死なるもの（涅槃）に深く入り込み、そこに到達した者を」

（註に従えば、「不死の涅槃に没入して到達した者」の意）

---

4. tamahaṃ brūmi brāhmaṇa’’nti.

* tam = 「その者を」
* ahaṃ = 「私は」
* brūmi = 「言う、呼ぶ」
* brāhmaṇaṃ = 「バラモン（真の聖者）」
* ’nti = 「ti（と）」の前の音変化（引用終わりを示す）

→ 「その人を、私は『バラモン（真の聖者）である』と呼ぶのである。」

---

5. Tattha ālayāti taṇhā.

* tattha = 「そこで、この句においては」
* ālayā ti = 「『ālayā』とは」
* taṇhā = 「渇愛」

→ 「ここで『ālayā（よりどころ・執着）』とは、渇愛のことである。」

---

6. Aññāya akathaṃkathīti aṭṭha vatthūni yathābhūtaṃ jānitvā aṭṭhavatthukāya vicikicchāya nibbicikiccho.

6-1. Aññāya akathaṃkathī ti

* aññāya = 「悟りの智によって」
* akathaṃkathī ti = 「『akathaṃkathī』とは」

→ 「『aññāya akathaṃkathī』とは、」

6-2. aṭṭha vatthūni yathābhūtaṃ jānitvā

* aṭṭha = 「八つの」
* vatthūni = 「対象・事柄」
* yathābhūtaṃ = 「あるがままに、如実に」
* jānitvā = 「知って、理解して」

→ 「八つの対象（八事）を、あるがままに知ったうえで」

6-3. aṭṭhavatthukāya vicikicchāya nibbicikiccho.

* aṭṭhavatthukāya = 「八つの対象をもつ」
  * aṭṭha = 「八つの」
  * vatthukāya (vatthuka) = 「対象をもつ」
* vicikicchāya = 「疑い・躊躇・疑惑」
* nibbicikiccho = 「疑いを離れた者、いっさい疑いのない者」
  * ni-（= nis-）+ vicikicchā → 「疑いが除かれた」

→ 「八つの対象をもつ疑いから解き放たれて、いささかの疑念もない者のことである。」

（すなわち、「八つの事柄を如実に知ることによって、八事を対象とする疑いをまったく持たない人」を指す。）

---

7. Amatogadhamanuppattanti amataṃ nibbānaṃ ogāhetvā anuppattaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

7-1. Amatogadhamanuppattan ti

* amatogadham anuppattaṃ ti = 「『amatogadham anuppattaṃ』とは」

→ 「『amatogadham anuppattaṃ』とは、」

7-2. amataṃ nibbānaṃ ogāhetvā anuppattaṃ

* amataṃ = 「不死なるもの」
* nibbānaṃ = 「涅槃」
* ogāhetvā = 「没入させて、深く入り込んで」
  * ogāhati = 「飛び込む・浸す・入り込む」の絶対分詞
* anuppattaṃ = 「到達した者」

→ 「不死の涅槃に没入して、それに到達した者を（意味し）」

7-3. tamahaṃ brāhmaṇaṃ vadāmi ti attho.

* tam = 「その者を」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「バラモン（聖者）」
* vadāmi = 「と言う／と呼ぶ」
* ti attho = 「という意味である」

→ 「その人を私は『バラモン（真の聖者）』と呼ぶ、という意味である。」

---

## 意訳

「その人には、渇愛という一切の執着のよりどころがまったく存在せず、  
悟りの智によって、いかなる疑いも残していない。  
不死なる涅槃に深く入り込み、そこへと完全に到達したその人を、  
私は真の『バラモン（聖なる人）』と呼ぶのである。

ここで『よりどころ（ālayā）』とは渇愛を意味し、  
『悟りによって疑いなき者（aññāya akathaṃkathī）』とは、  
八つの対象を如実に知ることによって、  
その八つを対象とするあらゆる疑いを断ち切り、いささかの疑念も残さない者を指す。  
『不死へ没入して到達した者（amatogadham anuppattaṃ）』とは、  
不死の涅槃に身心ともに没入し、そこに真に到達した者である、ということである。」

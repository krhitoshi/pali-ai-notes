原文

356.

‘‘Tiṇadosāni khettāni, rāgadosā ayaṃ pajā;
Tasmā hi vītarāgesu, dinnaṃ hoti mahapphalaṃ.

357.

‘‘Tiṇadosāni khettāni, dosadosā ayaṃ pajā;
Tasmā hi vītadosesu, dinnaṃ hoti mahapphalaṃ.

358.

‘‘Tiṇadosāni khettāni, mohadosā ayaṃ pajā;
Tasmā hi vītamohesu, dinnaṃ hoti mahapphalaṃ.

359.

‘‘Tiṇadosāni khettāni, icchādosā ayaṃ pajā;
Tasmā hi vigaticchesu, dinnaṃ hoti mahapphala’’nti.

Tattha tiṇadosānīti sāmākādīni tiṇāni uṭṭhahantāni pubbaṇṇāparaṇṇāni khettāni dūsenti, tena tāni na bahuphalāni honti.
Evaṃ sattānampi anto rāgo uppajjanto satte dūseti, tena tesu dinnaṃ mahapphalaṃ na hoti.
Khīṇāsavesu dinnaṃ pana mahapphalaṃ hoti. Tena vuttaṃ –

‘‘Tiṇadosāni khettāni, rāgadosā ayaṃ pajā;
Tasmā hi vītarāgesu, dinnaṃ hoti mahapphala’’nti. –

Sesagāthāsupi eseva nayo.


────────────────────────────
偈文（356〜359）の語句分解と文法
────────────────────────────

■ 共通構造（4つの偈すべて）
A) Tiṇadosāni khettāni,
B) X-dosā ayaṃ pajā;
C) Tasmā hi Y-...-esu, dinnaṃ hoti mahapphalaṃ.

- tiṇa-dosāni = 「雑草（tiṇa）という दोष (dosa, 欠点/害) をもつ」
  ※複合語：tiṇa + dosa
- khettāni = 「田畑・耕地」（khetta の複数）
- X-dosā = 「X を欠点（害）として持つ」
- ayaṃ = 「この」
- pajā = 「衆生・人々・この世の人々」
- tasmā hi = 「それゆえに（まさに）」
- Y-...-esu = 「Y を離れた人々（複数）において／彼らのもとに」
  ※locative 複数で「〜に対して、〜のところへ（施しが向けられる対象）」
- dinnaṃ = 「与えられたもの（布施）」
  ※√dā（与える）の過去分詞。中性単数で「施与（dāna）の行為/品」を指す言い方。
- hoti = 「である、〜となる」
- mahā-pphalaṃ = 「大いなる果（大きな功徳/結果）」
  ※mahā + phala の複合（連声で -pph-）

この偈の文の骨格は、
「（雑草が）田畑を悪くして収穫を減らすように、（煩悩が）人々を汚す。
だから、煩悩を離れた人々に与えられた布施は大果となる」
という比喩の対応です。


■ 356
Tiṇadosāni khettāni, rāgadosā ayaṃ pajā;
Tasmā hi vītarāgesu, dinnaṃ hoti mahapphalaṃ.

- rāga-dosā = 「貪欲（rāga）を害として持つ」
- vīta-rāgesu = 「貪欲を離れた人々において」
  * vīta = 「過ぎ去った、離れた」（√i の派生・過去分詞的）
  * rāga = 「貪・欲・染着」
  * vītarāga = 「離貪の人」→ loc.pl. vītarāgesu

→ 「雑草が欠点である田畑のように、貪が欠点であるこの人々。
　それゆえ、離貪の人々に施された布施は大果となる。」


■ 357
Tiṇadosāni khettāni, dosa-dosā ayaṃ pajā;
Tasmā hi vīta-dosesu, dinnaṃ hoti mahapphalaṃ.

- dosa-dosā = 「瞋り（dosa）を害として持つ」（語が同形になるが意味は明確）
- vīta-dosesu = 「瞋りを離れた人々において」

→ 「雑草が欠点である田畑のように、瞋りが欠点であるこの人々。
　それゆえ、離瞋の人々に施された布施は大果となる。」


■ 358
Tiṇadosāni khettāni, moha-dosā ayaṃ pajā;
Tasmā hi vīta-mohesu, dinnaṃ hoti mahapphalaṃ.

- moha-dosā = 「痴（moha）を害として持つ」
- vīta-mohesu = 「痴を離れた人々において」

→ 「雑草が欠点である田畑のように、痴が欠点であるこの人々。
　それゆえ、離痴の人々に施された布施は大果となる。」


■ 359
Tiṇadosāni khettāni, icchā-dosā ayaṃ pajā;
Tasmā hi vigati-cchesu, dinnaṃ hoti mahapphalaṃ.

- icchā-dosā = 「欲求・願望（icchā）を害として持つ」
  * icchā = 「欲すること、願うこと、求めること」
- vigati-cchesu = 「欲求が去った人々において」
  * vigata = 「去った、なくなった」（v + gata）
  * icchā → 連声で -icchesu（ここでは複合して vigaticcha-）
  * vigaticcha = 「欲求の去った者」→ loc.pl. vigaticchesu

→ 「雑草が欠点である田畑のように、欲求が欠点であるこの人々。
　それゆえ、欲求が去った人々に施された布施は大果となる。」


────────────────────────────
註釈（Atthakathā）の語句分解と解説
────────────────────────────

Tattha tiṇadosānīti ...
- tattha = 「そこで／この文脈では」
- tiṇadosānīti = 「『tiṇadosāni（雑草を害とする）』とは」
  ※-ti による引用提示 + iti の省略形のように働く説明スタイル

sāmākādīni tiṇāni uṭṭhahantāni pubbaṇṇāparaṇṇāni khettāni dūsenti,
- sāmāka-ādīni = 「サーマーカ等（の）」
  * sāmāka = 穀物畑に混じる雑草類（文脈上「雑草」）
  * ādīni = 「など」
- tiṇāni = 「草（雑草）」
- uṭṭhahantāni = 「生えてくる、立ち上がる」（uṭṭhahati 現在分詞・中性複数）
- pubbaṇṇa-aparaṇṇāni = 「前作・後作の穀物（のある畑）」
  * pubbaṇṇa = 前の作物（早稲・前作などの意）
  * aparaṇṇa = 後の作物（晩稲・後作などの意）
  ※「年に二期作する田畑」くらいの含みで、要は“作物のある畑”。
- khettāni = 「田畑」
- dūsenti = 「害する、汚す、だめにする」（dūseti）

→ 「サーマーカ等の雑草が生えてくると、前作後作の田畑を害する。」

tena tāni na bahuphalāni honti.
- tena = 「それによって」
- tāni = 「それら（田畑）は」
- na bahu-phalāni = 「多くの実り（収穫）を持たない」
- honti = 「〜となる」

→ 「それゆえ、それら（雑草にやられた畑）は多収にならない。」


Evaṃ sattānampi anto rāgo uppajjanto satte dūseti,
- evaṃ = 「このように」
- sattānaṃ pi = 「衆生たちにおいてもまた」
- anto = 「内に、内部に」
- rāgo = 「貪欲」
- uppajjanto = 「生起する」（uppajjati 現在分詞）
- satte = 「衆生たちを」（satta 複数対格）
- dūseti = 「汚す、だめにする」

→ 「同様に、衆生の内に貪欲が起こると、衆生を汚してしまう。」


tena tesu dinnaṃ mahapphalaṃ na hoti.
- tena = 「それゆえ（そのため）」
- tesu = 「それら（貪に汚れた者たち）に」
- dinnaṃ = 「施された布施は」
- mahapphalaṃ = 「大果（大きな結果）」
- na hoti = 「〜ではない」

→ 「だから、そのような者たちに与えられた布施は、大果にならない。」


Khīṇāsavesu dinnaṃ pana mahapphalaṃ hoti.
- khīṇa-āsavesu = 「漏（āsava）を尽くした人々において」
  * khīṇa = 「尽きた」
  * āsava = 「漏・煩悩の漏（酒の醗酵のように滲み出る汚れ）」
  ※一般に阿羅漢を指す語。
- pana = 「しかし一方で」
- dinnaṃ = 「施された布施は」
- mahapphalaṃ hoti = 「大果となる」

→ 「しかし、漏を尽くした人々（阿羅漢）に与えられた布施は大果となる。」


Tena vuttaṃ –（それゆえ、こう言われた）
→ ここで註釈は 356 偈を再引用し、「今の説明はこの偈の意味だ」とまとめています。


Sesagāthāsupi eseva nayo.
- sesa-gāthāsu pi = 「残りの偈においてもまた」
- esa eva nayo = 「これと同じ道理（同じ説明パターン）」

→ 「他の偈（357〜359）も、まったく同様の解釈である。」


────────────────────────────
逐語訳
────────────────────────────

356-359（共通イメージ）
「雑草を欠点とする田畑があるように、
（貪・瞋・痴・欲求）を欠点とするこの人々がいる。
それゆえ、（それらを）離れた人々に与えられた布施は、大きな果をもつ。」


────────────────────────────
自然な日本語訳
────────────────────────────

「畑は雑草が生えると台無しになり、収穫が減る。
同じように、人々も貪りや怒り、無知、欲望が起これば心が汚れてしまう。
だからこそ、貪り・怒り・無知・欲望を離れた人々に布施するなら、
その施しは大きな功徳（大きな結果）をもたらす、ということである。」

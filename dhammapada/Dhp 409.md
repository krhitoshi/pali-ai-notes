409.

‘‘Yodha dīghaṃ va rassaṃ vā, aṇuṃ thūlaṃ subhāsubhaṃ;

Loke adinnaṃ nādiyati, tamahaṃ brūmi brāhmaṇa’’nti.

Tassattho – sāṭakābharaṇādīsu dīghaṃ vā rassaṃ vā maṇimuttādīsu aṇuṃ vā thūlaṃ vā mahagghaappagghavasena subhaṃ vā asubhaṃ vā yo puggalo imasmiṃ loke parapariggahitaṃ nādiyati, tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Yodha dīghaṃ va rassaṃ vā, aṇuṃ thūlaṃ subhāsubhaṃ;

* yo = 「…である者は」  
* dha = 「まさに、そのような」（強意の語）  
* dīghaṃ = 「長いものを」  
* va = 「あるいは」  
* rassaṃ = 「短いものを」  
* vā = 「または」  
* aṇuṃ = 「小さいものを（微細なものを）」  
* thūlaṃ = 「大きいものを（粗大なものを）」  
* subhaṃ = 「美しいものを、好ましいものを」  
* asubhaṃ = 「醜いものを、好ましくないものを」

→「まさに、長いものや短いもの、小さいものや大きいもの、美しいものや醜いものを（対象として）」


2. Loke adinnaṃ nādiyati, tamahaṃ brūmi brāhmaṇa’’nti.

* loke = 「この世において」  
* adinnaṃ = 「与えられていないものを（盗品・不与取の対象を）」  
* na ādiyati = 「取らない、受け取らない」  
  * na = 「…しない」  
  * ādiyati = 「取る、受け取る、掴み取る」  
* taṃ = 「そのような者を」  
* ahaṃ = 「私は」  
* brūmi = 「説く、呼ぶ、言う」  
* brāhmaṇaṃ = 「バラモンであると（真の聖者と）」  
* ’ti = 「…と」  
* nti = 「（引用の終わり）」  

→「この世において、与えられていないものを取らない者を、私は『バラモン（真の聖者）』と呼ぶのである。」


3. Tassattho –

* tassa attho = 「その意味は（次のようである）」  

→「その意味は次のとおりである。」


4. sāṭakābharaṇādīsu dīghaṃ vā rassaṃ vā

* sāṭaka-ābharaṇa-ādīsu = 「衣や装身具などにおいて」  
  * sāṭaka = 「衣、布」  
  * ābharaṇa = 「装身具、飾り」  
  * -ādīsu = 「…などにおいて」  
* dīghaṃ vā = 「長いものでも」  
* rassaṃ vā = 「短いものでも」

→「衣や装身具などにおいて、長いものであっても短いものであっても、」


5. maṇimuttādīsu aṇuṃ vā thūlaṃ vā

* maṇi-muttā-dīsu = 「宝石や真珠などにおいて」  
  * maṇi = 「宝石」  
  * mutta = 「真珠」  
  * -ādīsu = 「…などにおいて」  
* aṇuṃ vā = 「小さいものでも」  
* thūlaṃ vā = 「大きいものでも」

→「宝石や真珠などにおいて、小さいものであっても大きいものであっても、」


6. mahagghaappagghavasena subhaṃ vā asubhaṃ vā

* mahaggha-appaggha-vasena = 「高価であるか安価であるかという観点から」  
  * mahaggha = 「高価な」  
  * appaggha = 「安価な」  
  * vasena = 「…という点から、…の基準において」  
* subhaṃ vā = 「美しいものであっても」  
* asubhaṃ vā = 「美しくないものであっても」

→「高価か安価かという点から見て、美しいものであっても、美しくないものであっても、」


7. yo puggalo imasmiṃ loke parapariggahitaṃ nādiyati,

* yo puggalo = 「いかなる人であれ、その人が」  
  * yo = 「…する者は」  
  * puggalo = 「人、人物」  
* imasmiṃ loke = 「この世において」  
* parapariggahitaṃ = 「他人によって所有されているものを」  
  * para = 「他人」  
  * pariggahita = 「取得された、所有された」  
* na ādiyati = 「取らない、手にしない」

→「この世において、他人の所有しているものを取らない人を」


8. tamahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「その者を」  
* ahaṃ = 「私は」  
* brāhmaṇaṃ = 「バラモン（真の聖者）であると」  
* vadāmi = 「言う、呼ぶ」  
* ti attho = 「という意味である」

→「その者を私は『バラモン（真の聖者）』と呼ぶのである、という意味である。」


## 意訳

「この世にある長いものや短いもの、小さいものや大きいもの、美しいものや醜いもの──  
それが衣や装身具であれ、宝石や真珠であれ、高価な物であれ安価な物であれ──  
他人の所有している、与えられていない物はいっさい取らない人、  
その人こそ、私は真のバラモン（聖者）と呼ぶのである。」

365.

‘‘Salābhaṃ nātimaññeyya, nāññesaṃ pihayaṃ care;

Aññesaṃ pihayaṃ bhikkhu, samādhiṃ nādhigacchati.

Tattha salābhanti attano uppajjanakalābhaṃ. Sapadānacārañhi parivajjetvā anesanāya jīvikaṃ kappento salābhaṃ atimaññati hīḷeti jigucchati nāma. Tasmā evaṃ akaraṇena salābhaṃ nātimaññeyya. Aññesaṃ pihayanti aññesaṃ lābhaṃ patthento na careyyāti attho. Samādhiṃ nādhigacchatīti aññesañhi lābhaṃ pihayanto tesaṃ cīvarādikaraṇe ussukkaṃ āpanno bhikkhu appanāsamādhiṃ vā upacārasamādhiṃ vā nādhigacchati. 

## 対訳

1. ‘‘Salābhaṃ nātimaññeyya, nāññesaṃ pihayaṃ care;

* salābhaṃ = 「自分に生じる利益・供養（自らの得分）」
  * sa- + lābha の語ではなく、註で「attano uppajjanaka-lābhaṃ（自分に生ずる利益）」と説明される
* na atimaññeyya = 「軽んじてはならない」
  * na = 「～してはならない」
  * atimaññeyya = 「（opt.）軽蔑すべきだと見なす、見下す」
* na aññesaṃ pihayaṃ care = 「他人のものを慕い求めながら歩み回ってはならない」
  * na = 「～してはならない」
  * aññesaṃ = 「他人の」
  * pihayaṃ = 「慕い求めつつ、欲しがりつつ（pihayati の現在分詞）」
  * care = 「行動すべきだ、歩み回るべきだ（opt.）」

→「自らの得た利益（供養）を軽んじてはならないし、  
　他人の得分を慕い求めながら立ち回ってはならない。」

---

2. Aññesaṃ pihayaṃ bhikkhu, samādhiṃ nādhigacchati.

* aññesaṃ pihayaṃ = 「他人のものを慕い求めつつ」
  * aññesaṃ = 「他人の」
  * pihayaṃ = 「慕い求めつつ」
* bhikkhu = 「比丘は」
* samādhiṃ = 「三昧を、定を」
* na adhigacchati = 「得ることがない、到達しない」
  * na = 「～しない」
  * adhigacchati = 「到達する、得る」

→「他人の得分を慕い求めている比丘は、  
　三昧（精神統一）を得ることがない。」

---

3. Tattha salābhaṃti attano uppajjanakalābhaṃ.

* tattha = 「ここで（この偈において）」
* salābhaṃ ti = 「『salābhaṃ』とは」
* attano = 「自分自身に属する、自分の」
* uppajjanaka-lābhaṃ = 「生じくる利益・供養」
  * uppajjanaka = 「生じてくる、起こる性質の」
  * lābhaṃ = 「利益・供養・得分」

→「ここで『salābhaṃ』とは、自分に生じてくる利益（供養）をいう。」

---

4. Sapadānacārañhi parivajjetvā anesanāya jīvikaṃ kappento salābhaṃ atimaññati hīḷeti jigucchati nāma.

* sapadānacāraṃ hi = 「というのも、『サパダーナ行』＝順次托鉢行を」
  * sapadānacāraṃ = 「家々を順に回る正しい托鉢行」
  * hi = 「というのも、実に」
* parivajjetvā = 「避け捨てて、退けて（parivajjeti の絶対分詞）」
* anesanāya = 「不正な追求によって、邪な求め方によって」
  * anesanā = 「不正な求め、邪な物色（ほのめかし、へつらい等による物資獲得）」
* jīvikaṃ kappento = 「生活（生計）を成り立たせている者は」
  * jīvikaṃ = 「生計、生活手段」
  * kappento = 「整える、営む（karoti/kappeti の現在分詞）」
* salābhaṃ atimaññati = 「自分の得分を軽蔑している」
  * salābhaṃ = 「自分に生じた利益・供養」
  * atimaññati = 「見下す、軽んじる」
* hīḷeti = 「卑しめる」
* jigucchati = 「嫌悪する、忌み嫌う」
* nāma = 「と名づけられる、～という者である」

→「実に、順次托鉢の行を避けて、不正な求め方によって生計を立てている者は、  
　自分の生じた得分を軽んじ、卑しめ、忌み嫌う者と呼ばれる。」

---

5. Tasmā evaṃ akaraṇena salābhaṃ nātimaññeyya.

* tasmā = 「それゆえ」
* evaṃ akaraṇena = 「このようなこと（＝sapadānacāraを避けて不正な求めをすること）をせずに」
  * evaṃ = 「このように、かく」
  * akaraṇena = 「行わないことによって」
* salābhaṃ = 「自分に生じた得分を」
* na atimaññeyya = 「軽んじてはならない」

→「だから、このようなことをしないで、自分の得分を軽んじてはならない。」

---

6. Aññesaṃ pihayanti aññesaṃ lābhaṃ patthento na careyyāti attho.

* aññesaṃ pihayan ti = 「『aññesaṃ pihayaṃ』とは」
* aññesaṃ lābhaṃ = 「他人の利益・供養を」
* patthento = 「望み求めつつ」
* na careyya = 「行動してはならない、立ち回ってはならない」
* ti attho = 「という意味である」

→「『他人のものを慕い求めつつ』とは、  
　他人の得分を望みながら立ち回ってはならない、という意味である。」

---

7. Samādhiṃ nādhigacchatīti aññesañhi lābhaṃ pihayanto tesaṃ cīvarādikaraṇe ussukkaṃ āpanno bhikkhu appanāsamādhiṃ vā upacārasamādhiṃ vā nādhigacchati.

* samādhiṃ nādhigacchatīti = 「『三昧を得ない』とは」
  * samādhiṃ = 「三昧、定」
  * na adhigacchati = 「到達しない、得ない」
* aññesañ hi lābhaṃ pihayanto = 「というのは、他人の得分を慕い求めているがゆえに」
  * aññesañ hi = 「他人たちの（aññesaṃ の属格複数）＋ hi（なぜなら）」
  * lābhaṃ = 「利益・供養」
  * pihayanto = 「慕い求めている者は」
* tesaṃ cīvarādi-karaṇe = 「彼らの衣などの作成・調達において」
  * tesaṃ = 「彼ら（施主・在家など）の／あるいは、それら（他人の得分）の」
  * cīvara-ādi-karaṇe = 「衣などの〔種々の〕事をすることにおいて」
    * cīvara = 「衣」
    * ādi = 「など」
    * karaṇe = 「作ること・整えること・取り計らうこと」
* ussukkaṃ āpanno = 「熱心になり、忙しくしている比丘は」
  * ussukkaṃ = 「熱心さ、あくせくすること」
  * āpanno = 「陥った、そうなっている」
* bhikkhu = 「比丘は」
* appanā-samādhiṃ vā = 「安止三昧（本定）を、あるいは」
  * appanā-samādhiṃ = 「安止定、吸収的な三昧」
  * vā = 「あるいは」
* upacāra-samādhiṃ vā = 「近行定を、あるいは」
  * upacāra-samādhiṃ = 「近行定」
* na adhigacchati = 「得ることがない、到達しない」

→「『三昧を得ない』とは、  
　他人の得分を慕い求めるがゆえに、  
　その人たちの衣などを作ること・取り計らうことに熱心になってしまった比丘は、  
　安止定であれ近行定であれ、どちらの三昧にも到達しない、という意味である。」

---

## 意訳

「自分に与えられたささやかな供養であっても、決して軽んじてはならない。  
また、他人の得た供養を羨んで、そうしたものを欲しがりながら立ち回ってはならない。

正しい順次托鉢の行を避け、ほのめかしやへつらいなどの不正な求めによって生計を立てる者は、  
自分に生じた得分を軽んじ、卑しめ、忌み嫌っている者と呼ばれる。  
ゆえに、そのような行為をせず、自分の得分を見下してはならない。

他人の得分を望み求めながら立ち回ってはならない、というのが  
『他人のものを慕い求めつつ歩き回るな』という句の意味である。

そして、他人の得分を羨み求め、その人たちの衣などを作ったり取り計らったりすることに  
あくせくしている比丘は、安止定であれ近行定であれ、  
いかなる三昧にも到達することができないのである。」

366.

‘‘Appalābhopi ce bhikkhu, salābhaṃ nātimaññati;

Taṃ ve devā pasaṃsanti, suddhājīviṃ atandita’’nti.

 Salābhaṃ nātimaññatīti appalābhopi samāno uccanīcakule paṭipāṭiyā sapadānaṃ caranto bhikkhu salābhaṃ nātimaññati nāma. Taṃ veti taṃ evarūpaṃ bhikkhuṃ sārajīvitatāya suddhājīviṃ jaṅghabalaṃ nissāya jīvitakappanena akusītatāya atanditaṃ devā pasaṃsanti thomentīti attho.

## 対訳

1. ‘‘Appalābhopi ce bhikkhu, salābhaṃ nātimaññati;

* appalābho pi = 「たとえ得るところが少なくても」
  * appa = 「少ない」
  * lābho = 「利益・供養・得分」
  * appalābho = 「わずかな得分」
  * pi = 「～であっても」
* ce = 「もし」
* bhikkhu = 「比丘が」
* salābhaṃ = 「自分に生じた得分（先の註によれば ‘attano uppajjanakalābhaṃ’）を」
* na atimaññati = 「軽んじない、見下さない」
  * na = 「～しない」
  * atimaññati = 「見下す、軽蔑する」

→「たとえ得るところが少なくても、もし比丘が自分の得分を軽んじないならば、」

---

2. Taṃ ve devā pasaṃsanti, suddhājīviṃ atandita’’nti.

* taṃ ve = 「その人をまさに」
  * taṃ = 「その（比丘）を」
  * ve = 「まさに、確かに」
* devā = 「諸天は」
* pasaṃsanti = 「称賛する」
* suddhā-jīviṃ = 「清浄に生きている人、清浄な生活をしている者として」
  * suddha = 「清浄な」
  * jīviṃ (jīvin) = 「生活している者」
* atanditaṃ = 「怠らない者、精進している者として」
  * a-tandita = 「怠けない、たゆまぬ」

→「その人を、諸天は、清浄に生きる怠りなき者として称賛するのである。」

---

3. Salābhaṃ nātimaññatīti appalābhopi samāno uccanīcakule paṭipāṭiyā sapadānaṃ caranto bhikkhu salābhaṃ nātimaññati nāma.

* salābhaṃ nātimaññatī ti = 「『salābhaṃ nātimaññati』とは」
  * salābhaṃ = 「自分に生じた得分を」
  * na atimaññati = 「軽んじない」
* appalābho pi samāno = 「たとえ得るところが少ない者であっても」
  * appalābho = 「わずかな得分のある者」
  * pi = 「であっても」
  * samāno = 「～である状態で」
* uccanīca-kule = 「高い家柄・低い家柄（すなわち貴賤）に〔かかわらず〕」
  * ucca = 「高い」
  * nīca = 「低い」
  * kula = 「家、家柄」
  * uccanīcakule = 「高貴な家・卑しい家（どちらの）にも」
* paṭipāṭiyā = 「順序正しく、順々に」
* sapadānaṃ caranto = 「家々を順に托鉢して歩く（正しい托鉢行をしている）」
  * sapadānaṃ = 「家々を順に」
  * caranto = 「歩き回る、行ずる（現在分詞）」
* bhikkhu = 「比丘は」
* salābhaṃ nātimaññati nāma = 「『自分の得分を軽んじない者』といわれる」
  * nāma = 「と名づけられる」

→「ここで『自分の得分を軽んじない』とは、  
　たとえ得るところが少なくても、高い家柄であろうと低い家柄であろうと、  
　家々を順々に托鉢して歩く比丘を、『自分の得分を軽んじない者』というのである。」

---

4. Taṃ veti taṃ evarūpaṃ bhikkhuṃ sārajīvitatāya suddhājīviṃ jaṅghabalaṃ nissāya jīvitakappanena akusītatāya atanditaṃ devā pasaṃsanti thomentīti attho.

* taṃ ve ti = 「『taṃ ve』とは」
  * taṃ = 「その（比丘のことを）」
  * ve = 「まことに、まさに」
* taṃ evarūpaṃ bhikkhuṃ = 「そのような比丘を」
  * evarūpaṃ = 「かくのごとき、そのような」
* sāra-jīvitatāya = 「本質的な（まっとうな）生き方であるがゆえに」
  * sāra = 「要（かなめ）、本質、真価」
  * jīvita = 「生活・生き方」
  * -tāya = 「～であるという性質によって」
* suddhā-jīviṃ = 「清浄に生きる者として」
  * suddha = 「清浄な」
  * jīviṃ = 「生活している者」
* jaṅgha-balaṃ nissāya jīvitakappanena = 「脚の力にもっぱら依って、生活を成り立たせていることによって」
  * jaṅgha = 「脚」
  * balaṃ = 「力」
  * nissāya = 「頼りとして、依って」
  * jīvita-kappanena = 「生計を営むことによって、生を維持していることによって」
* akusītatāya = 「怠惰でないという性質によって」
  * a-kusīta = 「怠惰でない、なまけていない」
  * -tāya = 「～であるという性質によって」
* atanditaṃ = 「たゆまぬ者、精勤な者として」
  * atandita = 「怠らない者」
* devā pasaṃsanti thomenti = 「諸天は称賛し、ほめたたえる」
  * pasaṃsanti = 「ほめる、称賛する」
  * thomenti = 「賞揚する、ほめたたえる」
* ti attho = 「という意味である」

→「『その人をまさに諸天は称賛する』とは、  
　そのような比丘を、要を得た正しい生活ぶりによって『清浄に生きる者』とし、  
　脚の力に頼って（すなわち自ら歩いて托鉢することによって）生計を立て、  
　怠惰でないがゆえに『たゆまぬ者』として、諸天は称賛し、ほめたたえる、  
　という意味である。」

---

## 意訳

「たとえ得る供養がわずかであっても、自分に与えられた分を軽んじない比丘を、  
諸天は、清浄な生活を送り、怠ることのない者として称賛する。

ここで『自分の得分を軽んじない』とは、  
たとえ得るものが少なくても、貴賤を問わず家々を順に托鉢して歩く比丘のことである。  

そのような比丘は、要を得た正しい生き方をしているがゆえに清浄な生活者であり、  
自らの脚の力に頼って（自分で歩いて托鉢して）生計を立て、  
怠惰ではなく精進している者であるから、  
諸天はその比丘をほめたたえるのである。」

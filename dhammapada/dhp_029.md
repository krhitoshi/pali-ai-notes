‘‘Appamatto pamattesu, suttesu bahujāgaro;

## 対訳

1. ‘‘Appamatto pamattesu, suttesu bahujāgaro;

* appamatto = 「不放逸な者は」
* pamattesu = 「放逸な（怠惰な）者たちの中で」
* suttesu = 「眠っている者たちの中で」
* bahujāgaro = 「大いに目覚めている者は」

→ 「放逸な者たちの中で不放逸であり、眠っている者たちの中で大いに目覚めている者は、」

2. Abalassaṃva sīghasso, hitvā yāti sumedhaso’’ti.

* abalassaṃva (abala-assaṃ + iva) = 「弱い馬を～のように」
* sīghasso (sīgha-asso) = 「駿馬（速い馬）が」
* hitvā = 「捨て置いて（追い越して）」
* yāti = 「進んで行く」
* sumedhaso = 「賢者は」
* ti = 「～と（引用の終わり）」

→ 「『駿馬が弱い馬を置き去りにしていくように、賢者は（彼らを）置いて進んで行く。』」

3. Tattha appamattoti sativepullappattatāya appamādasampanne khīṇāsavo.

* tattha = 「そこで（この偈において）」
* appamatto ti = 「『appamatto（不放逸な者）』とは」
* sati-vepulla-ppattatāya = 「念（気づき）の広大さに達していることによって」
* appamādasampanne = 「不放逸を具足した（※appamādasampanno の誤記と推測される）」
* khīṇāsavo = 「漏尽者（煩悩を滅尽した阿羅漢）」

→ 「そこで『不放逸な者』とは、念の広大さに達していることによって不放逸を具足した漏尽者（阿羅漢）のことである。」

4. Pamattesūti sativosagge ṭhitesu sattesu.

* pamattesu ti = 「『pamattesu（放逸な者たちの中で）』とは」
* sati-vosagge = 「念を放棄することに」
* ṭhitesu = 「留まっている」
* sattesu = 「衆生たちの中で」

→ 「『放逸な者たちの中で』とは、念を放棄した状態に留まっている衆生たちの中で（という意味である）。」

5. Suttesūti satijāgariyābhāvena sabbiriyāpathesu niddāyantesu.

* suttesu ti = 「『suttesu（眠っている者たちの中で）』とは」
* sati-jāgariya-abhāvena = 「念の目覚めがないことによって」
* sabba-iriyāpathesu = 「すべての威儀（歩住坐臥の姿勢・行動）において」
* niddāyantesu = 「眠っている者たちの中で」

→ 「『眠っている者たちの中で』とは、念の目覚めがないために、すべての威儀において眠っている（も同然の）者たちの中で（という意味である）。」

6. Bahujāgaroti mahante sativepulle jāgariye ṭhito.

* bahujāgaro ti = 「『bahujāgaro（大いに目覚めている者は）』とは」
* mahante = 「偉大な（大きな）」
* sati-vepulle = 「念の広大さに」
* jāgariye = 「目覚めていることに」
* ṭhito = 「留まっている者（のことである）」

→ 「『大いに目覚めている者は』とは、大きな念の広大さと目覚めの中に留まっている者（のことである）。」

7. Abalassaṃvāti kuṇṭhapādaṃ chinnajavaṃ dubbalassaṃ sīghajavo sindhavājānīyo viya.

→ 「『弱い馬を～のように』とは、足が不自由で勢いのない弱い馬を、足の速いシンドゥ産の駿馬が（追い越していく）ように（という意味である）。」

8. Sumedhasoti uttamapañño.

## 意訳

「放逸な者たちの中で不放逸であり、眠っている者たちの中で大いに目覚めている賢者は、駿馬が足の弱い馬を置き去りにしていくように、彼らを置いて進んでいく。」

（註釈）
ここで「不放逸な者」とは、念（気づき）の広大さに達し、不放逸を完全に備えた漏尽者（阿羅漢）のことである。
「放逸な者たちの中で」とは、念を放棄した状態に留まっている衆生たちの中で、という意味である。
「眠っている者たちの中で」とは、念の目覚めがないために、すべての日常の行動において眠っているも同然の者たちの中で、という意味である。
「大いに目覚めている者」とは、広大に念じ、目覚めた状態に留まっている者のことである。
「弱い馬を～のように」とは、足が不自由で勢いのない弱い馬を、足の速いシンドゥ産の駿馬が追い越していくように、という意味である。
「賢者」とは、最高の智慧を持つ者のことである。

賢者はそのような愚かで放逸な人を、教えの学習においても、悟りの証得においても置き去りにして進んでいく。
実に、智慧の劣った者が一つの経典を覚えようと努力している間に、賢者は一つの篇（章）全体を覚えてしまう。このようにして、まず学習において愚者を置き去りにして進む。
また、智慧の劣った者が自分の昼と夜の住処を作ろうと努力し、瞑想の対象（業処）を学んで復唱している間に、賢者は他人がすでに作った住処に入り、瞑想対象を観察し、すべての煩悩を滅ぼして九つの出世間法（四道・四果・涅槃）を手に入れる。このようにして、悟りの証得においても愚者を置き去りにして進む。
そしてまた、輪廻の中に彼らを見捨てて、自らは輪廻から脱出していくのである。

422.

‘‘Usabhaṃ pavaraṃ vīraṃ, mahesiṃ vijitāvinaṃ;

Anejaṃ nhātakaṃ buddhaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tassattho – acchambhitaṭṭhena usabhasadisatāya usabhaṃ uttamaṭṭhena pavaraṃ vīriyasampattiyā vīraṃ mahantānaṃ sīlakkhandhādīnaṃ esitattā mahesiṃ tiṇṇaṃ mārānaṃ vijitattā vijitāvinaṃ nhātakilesatāya nhātakaṃ catusaccabuddhatāya buddhaṃ taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Usabhaṃ pavaraṃ vīraṃ, mahesiṃ vijitāvinaṃ;

* usabhaṃ = 「牡牛、群れの先頭に立つ牡牛」→転じて「最上の者、雄々しい者」
* pavaraṃ = 「最上の、すぐれた、えり抜きの」
* vīraṃ = 「勇者、英雄、精進力に満ちた者」
* mahesiṃ = 「大いなる聖者、大いなる仙人（＝大いなる聖賢）」
* vijitāvinaṃ = 「征服した者、勝利を得た者」

→「群れの先頭に立つ牡牛のような最上の勇者であり、大いなる聖者であって、征服を成し遂げた者を」

2. Anejaṃ nhātakaṃ buddhaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* anejaṃ = 「動揺しない者、心が揺らがない者、不動の者」
* nhātakaṃ = 「洗い浄められた者（汚れを洗い落とした者）」
* buddhaṃ = 「悟った者、目覚めた者」
* taṃ = 「そのような者を」
* ahaṃ = 「私は」
* brūmi = 「言う、呼ぶ」
* brāhmaṇaṃ = 「バラモン（真に聖なる人、真のバラモン）」

→「動揺せず、（汚れを）洗い浄められた悟った者——そのような者を、私は『バラモン』と呼ぶのである、と。」

3. Tassattho –

* tassa = 「その（偈の）」
* attho = 「意味は」
→「その意味は、次のようである。」

4. acchambhitaṭṭhena usabhasadisatāya usabhaṃ

* acchambhita-ṭṭhena = 「恐れないという性質において」
  * acchambhita = 「おじけづかない、ひるまない」
  * ṭṭhena（= ṭhena, ṭhānena）= 「ありさま・性質において」
* usabha-sadisatāya = 「牡牛のようであることによって」
  * usabha = 「牡牛、群れの先頭に立つ牡牛」
  * sadisatā = 「～に似ていること」
* usabhaṃ = 「『usabha（牡牛）』と（言う）」

→「恐れないという性質によって、群れの先頭に立つ牡牛のようであるがゆえに、『usabha（牡牛のような最上の者）』と言うのである。」

5. uttamaṭṭhena pavaraṃ

* uttama-ṭṭhena = 「最上であるという点から」
  * uttama = 「最高の、最上の」
* pavaraṃ = 「『pavara（最上の者）』と（言う）」

→「最上であるという点から、『pavara（最上の者）』と言い。」

6. vīriyasampattiyā vīraṃ

* vīriya-sampattiyā = 「精進・勇気が具わっていることによって」
  * vīriya = 「精進、勇気、努力」
  * sampatti = 「具足、充実」
* vīraṃ = 「『vīra（勇者）』と（言う）」

→「精進の徳が具わっていることによって、『vīra（勇者）』と言い。」

7. mahantānaṃ sīlakkhandhādīnaṃ esitattā mahesiṃ

* mahantānaṃ = 「偉大な（多くの、重大な）」  
* sīla-kkhandh-ādīnaṃ = 「戒の集まり等の（大いなる徳の集まり）の」
  * sīlakkhandha = 「戒の集まり、戒の大集積」
  * -ādīnaṃ = 「などの」
* esitattā = 「求められたがゆえに／獲得したがゆえに」
  * esita (esati「求める、探し求める」の過去分詞) = 「求められた、得られた」
* mahesiṃ = 「『mahesi（大いなる聖者）』と（言う）」

→「偉大な戒の集まりなどの（大いなる徳の集積）を求めて得ているがゆえに、『mahesi（大いなる聖者）』と言い。」

8. tiṇṇaṃ mārānaṃ vijitattā vijitāvinaṃ

* tiṇṇaṃ mārānaṃ = 「三種のマーラ（悪魔）を」
  * tiṇṇaṃ = 「三つの」
  * mārānaṃ = 「マーラたちの」
    * 三種のマーラ＝khandhamāra（蘊の魔）・kilesamāra（煩悩の魔）・maccumara（死の魔）等
* vijitattā = 「征服したがゆえに」
  * vijita = 「征服された」
* vijitāvinaṃ = 「『vijitāvin（征服を成就した者）』と（言う）」

→「三種のマーラを征服したがゆえに、『vijitāvin（勝利者・征服者）』と言い。」

9. nhātakilesatāya nhātakaṃ

* nhāta-kilesatāya = 「煩悩を洗い清めた状態であることによって」
  * nhāta = 「洗われた」
  * kilesa = 「煩悩」
* nhātakaṃ = 「『nhātaka（洗い浄められた者）』と（言う）」

→「煩悩を洗い清めた者であることによって、『nhātaka（清められた者）』と言い。」

10. catusaccabuddhatāya buddhaṃ

* catu-sacca-buddhatāya = 「四つの聖なる真理を悟っていることによって」
  * catu-sacca = 「四つの真理（四聖諦）」
  * buddhatā = 「覚っているというありさま」
* buddhaṃ = 「『buddha（悟れる者）』と（言う）」

→「四つの聖なる真理を悟っているがゆえに、『buddha（覚者）』と言い。」

11. taṃ evarūpaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「そのような（人を）」
* evarūpaṃ = 「このような性質をそなえた」
* ahaṃ = 「私は」
* brāhmaṇaṃ = 「『バラモン』と」
* vadāmi = 「言う、呼ぶ」
* ti attho = 「という意味である」

→「このような性質をそなえたその人を、私は『バラモン』と呼ぶのである、という意味である。」

---

## 意訳

「群れの先頭に立つ牡牛のように恐れなく、最上であり、
精進に満ちた勇者で、大いなる戒などの徳を備えた大聖者であり、
三種のマーラを打ち破った勝利者であって、
煩悩を洗い清められ、四つの聖なる真理を悟った覚者――

そのような動揺することなき、清らかに洗い浄められた仏陀こそを、
私は『真のバラモン』と呼ぶのである。」

17.

‘‘Idha tappati pecca tappati, pāpakārī ubhayattha tappati;

Pāpaṃ me katanti tappati, bhiyyo tappati duggatiṃ gato’’ti.

Tattha idha tappatīti idha kammatappanena domanassamattena tappati. Peccāti paraloke pana vipākatappanena atidāruṇena apāyadukkhena tappati. Pāpakārīti nānappakārassa pāpassa kattā. Ubhayatthāti iminā vuttappakārena tappanena ubhayattha tappati nāma. Pāpaṃ meti so hi kammatappanena kappanto ‘‘pāpaṃ me kata’’nti tappati. Taṃ appamattakaṃ tappanaṃ, vipākatappanena pana tappanto bhiyyo tappati duggatiṃ gato atipharusena tappanena ativiya tappatīti.

## 対訳

1. ‘‘Idha tappati pecca tappati, pāpakārī ubhayattha tappati;

* idha = 「ここで、現世で」
* tappati = 「苦悩する、後悔の念に燃える」
* pecca = 「死んでから、来世で」
* tappati = 「苦悩する」
* pāpakārī = 「悪をなす者は」
* ubhayattha = 「両方で（現世と来世の両方で）」
* tappati = 「苦悩する」

→ 「悪をなす者は、現世で苦悩し、来世で苦悩し、両方で苦悩する。」

2. Pāpaṃ me katanti tappati, bhiyyo tappati duggatiṃ gato’’ti.

* pāpaṃ = 「悪が」
* me = 「私によって」
* kataṃ = 「なされた」
* ti = 「と（思って）」
* tappati = 「苦悩する」
* bhiyyo = 「さらに多く、よりいっそう」
* tappati = 「苦悩する」
* duggatiṃ = 「悪趣（悪い境涯）に」
* gato = 「赴いて」
* ti = 「（という偈文である）」

→ 「『私は悪をなした』と言って苦悩し、悪趣に赴いてよりいっそう苦悩する。」

3. Tattha idha tappatīti idha kammatappanena domanassamattena tappati.

* tattha = 「そこで（この偈文において）」
* idha tappatīti = 「『idha tappati（現世で苦悩する）』とは」
* idha = 「現世で」
* kammatappanena = 「業による苦悩によって」
* domanassamattena = 「憂いという程度で」
* tappati = 「苦悩する（という意味である）」

→ 「そこで『現世で苦悩する』とは、現世で業による苦悩、すなわち憂いによって苦悩するということである。」

4. Peccāti paraloke pana vipākatappanena atidāruṇena apāyadukkhena tappati.

* peccāti = 「『pecca（死んでから、来世で）』とは」
* paraloke = 「来世で」
* pana = 「一方」
* vipākatappanena = 「異熟（果報）による苦悩によって」
* atidāruṇena = 「極めて恐ろしい」
* apāyadukkhena = 「悪趣の苦しみによって」
* tappati = 「苦悩する（という意味である）」

→ 「『死んでから』とは、一方で来世において異熟による苦悩、すなわち極めて恐ろしい悪趣の苦しみによって苦悩するということである。」

5. Pāpakārīti nānappakārassa pāpassa kattā.

* pāpakārīti = 「『pāpakārī（悪をなす者）』とは」
* nānappakārassa = 「さまざまな種類の」
* pāpassa = 「悪の」
* kattā = 「行為者（という意味である）」

→ 「『悪をなす者』とは、さまざまな悪を行う者のことである。」

6. Ubhayatthāti iminā vuttappakārena tappanena ubhayattha tappati nāma.

* ubhayatthāti = 「『ubhayattha（両方で）』とは」
* iminā = 「この」
* vuttappakārena = 「説かれた方法（種類）の」
* tappanena = 「苦悩によって」
* ubhayattha = 「両方（現世と来世）で」
* tappati nāma = 「苦悩するということである」

→ 「『両方で』とは、この説かれた種類の苦悩によって、両方で苦悩するということである。」

7. Pāpaṃ meti so hi kammatappanena kappanto ‘‘pāpaṃ me kata’’nti tappati.

* pāpaṃ meti = 「『pāpaṃ me（私に悪が）』とは」
* so hi = 「彼は実に」
* kammatappanena = 「業による苦悩によって」
* kappanto = 「苦しみながら（※文脈および他版から tappanto の意）」
* pāpaṃ = 「悪が」
* me = 「私によって」
* kataṃ ti = 「なされた、と」
* tappati = 「苦悩する（という意味である）」

→ 「『私に悪が』とは、彼は実に業による苦悩に苛まれながら、『私は悪をなした』と言って苦悩するということである。」

8. Taṃ appamattakaṃ tappanaṃ, vipākatappanena pana tappanto bhiyyo tappati duggatiṃ gato atipharusena tappanena ativiya tappatīti.

* taṃ = 「それは」
* appamattakaṃ = 「わずかな」
* tappanaṃ = 「苦悩である」
* vipākatappanena = 「異熟による苦悩によって」
* pana = 「しかし」
* tappanto = 「苦悩する者は」
* bhiyyo tappati duggatiṃ gato = 「悪趣に赴いて、よりいっそう苦悩する（とは）」
* atipharusena = 「非常に過酷な」
* tappanena = 「苦悩によって」
* ativiya = 「極度に」
* tappatīti = 「苦悩する、という意味である」

→ 「その（現世での）苦悩はわずかなものである。しかし、異熟による苦悩によって苦しむ者は『悪趣に赴いてよりいっそう苦悩する』、すなわち非常に過酷な苦悩によって極度に苦悩する、という意味である。」

## 意訳

「悪をなす者は、現世で苦悩し、来世でも苦悩し、両方の世で苦悩する。
『私は悪をなした』と言って苦悩し、悪趣に赴いてさらにいっそう苦悩する。」

【註釈】
ここで「現世で苦悩する」とは、現世で自らの業を思って憂い、苦悩することである。
「来世で（死んでから）」とは、来世で悪業の果報として、極めて恐ろしい悪趣の苦しみを受けて苦悩することである。
「悪をなす者」とは、さまざまな悪を行う者のことである。
「両方で」とは、これまでに述べたような苦悩によって、現世と来世の両方で苦悩するということである。
「『私に悪が』とは」とは、彼はまさに自らの業の苦悩に苛まれながら、「私は悪をなした」と苦悩するということである。
その現世での苦痛はわずかなものであるが、悪業の果報によって苦しむ者は「悪趣に赴いてさらにいっそう苦悩する」、すなわち極めて過酷な苦痛によって激しく苦悩する、という意味である。

417.

‘‘Hitvā mānusakaṃ yogaṃ, dibbaṃ yogaṃ upaccagā;

Sabbayogavisaṃyuttaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha mānusakaṃ yoganti mānusakaṃ āyuñceva pañca kāmaguṇe ca. Dibbayogepi eseva nayo. Upaccagāti yo mānusakaṃ yogaṃ hitvā dibbaṃ yogaṃ atikkanto, taṃ sabbehi catūhipi yogehi visaṃyuttaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

1. ‘‘Hitvā mānusakaṃ yogaṃ, dibbaṃ yogaṃ upaccagā;

* hitvā = 「捨て去って、離れ捨てて」
* mānusakaṃ yogaṃ = 「人間のヨーガ（束縛）」  
  * mānusakaṃ = 「人間の」
  * yoga = 「くびき、束縛・執着の対象」
→「人間の束縛を捨て去り」

* dibbaṃ yogaṃ = 「天のヨーガ（束縛）」  
  * dibbaṃ = 「天上（の）、神々の」
→「天の束縛をも」

* upaccagā = 「超え去った、乗り越えた、通り越した（av + aticaram の過去形）」
→「人間の束縛を捨て、天の束縛をも乗り越え」

---

2. Sabbayogavisaṃyuttaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

* sabba-yoga-visaṃyuttaṃ = 「一切のヨーガから離れた者」  
  * sabba = 「すべての」
  * yoga = 「束縛、くびき」
  * visaṃyutta = 「離れた、解き放たれた」
→「一切の束縛から解き放たれている者を」

* taṃ = 「その者を」
* ahaṃ brūmi = 「私は～と呼ぶ（言う）」
* brāhmaṇaṃ = 「バラモン（真の意味での聖者）」
→「その者を、私はバラモンと呼ぶのだ」と。

---

3. Tattha mānusakaṃ yoganti mānusakaṃ āyuñceva pañca kāmaguṇe ca.

* tattha = 「ここで、この（偈の）箇所において」
* mānusakaṃ yogaṃ ti = 「『人間のヨーガ（束縛）』とは」
* mānusakaṃ āyuṃ = 「人間の寿命」
  * āyu = 「寿命、生命の期間」
* ca eva = 「と、まさに」
* pañca kāmaguṇe ca = 「さらに五つの欲の対象（五欲）」
  * pañca = 「五つの」
  * kāmaguṇā = 「欲の要素・欲の対象（色・声・香・味・触）」
→「ここで『人間の束縛』とは、人間としての寿命と、五つの欲の対象のことである。」

---

4. Dibbayogepi eseva nayo.

* dibba-yoge pi = 「『天のヨーガ（束縛）』についても」
  * dibba-yoga = 「天の束縛（天上の寿命と天上の五欲）」
  * pi = 「～もまた」
* eseva nayo = 「同じ道理である、解釈は同様である」
→「『天の束縛』についても、意味するところは同様である。」

---

5. Upaccagāti yo mānusakaṃ yogaṃ hitvā dibbaṃ yogaṃ atikkanto,

* upaccagā ti = 「『upaccagā』とは」
* yo = 「～である者は」
* mānusakaṃ yogaṃ hitvā = 「人間の束縛を捨て去り」
  * hitvā = 「捨てて、離れて」
* dibbaṃ yogaṃ atikkanto = 「天の束縛をも超え去った」
  * atikkanto = 「超えた、越えた」
→「『upaccagā』とは、人間の束縛を捨て、天の束縛をも超え去った者であり、」

---

6. taṃ sabbehi catūhipi yogehi visaṃyuttaṃ ahaṃ brāhmaṇaṃ vadāmīti attho.

* taṃ = 「その者を」
* sabbehi catūhipi yogehi = 「四つのすべてのヨーガ（束縛）から」  
  （一般には「欲・有・見・無明」という四つのヨーガを指す）
  * sabbehi = 「すべての」
  * catūhi = 「四つの」
  * yogehi = 「ヨーガから」
* visaṃyuttaṃ = 「離れた、解き放たれた」
* ahaṃ brāhmaṇaṃ vadāmi = 「私はバラモン（真の聖者）と言う」
* ti attho = 「という意味である」
→「その者を、四つの一切の束縛から解き放たれた者として、私はバラモンと言うのである、という意味である。」

---

## 意訳

「人としての寿命と五欲から成る人間の束縛を捨て去り、  
さらに天上の寿命と天上の五欲から成る天の束縛さえも乗り越え、  
欲・有・見・無明というあらゆる束縛から完全に解き放たれている者、  
その人をこそ、私は真のバラモンと呼ぶのである。」

403.

‘‘Gambhīrapaññaṃ medhāviṃ, maggāmaggassa kovidaṃ;

Uttamatthamanuppattaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha gabbhīrapaññanti gambhīresu khandhādīsu pavattāya paññāya samannāgataṃ dhammojapaññāya samannāgataṃ medhāviṃ ‘‘ayaṃ duggatiyā maggo, ayaṃ sugatiyā maggo, ayaṃ nibbānassa maggo, ayaṃ amaggo’’ti evaṃ magge ca amagge ca chekatāya maggāmaggassa kovidaṃ arahattasaṅkhātaṃ uttamatthaṃ anuppattaṃ tamahaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

### 偈頌部分

1. Gambhīrapaññaṃ medhāviṃ,

- gambhīra = 「深い」
- paññaṃ = 「智慧・洞察」（対格）
- gambhīra‑paññaṃ = 「深い智慧をもつ者を」
- medhāviṃ = 「利口な者・聡明な者・才知ある者を」（対格）

→「深い智慧をそなえた、聡明な者を」

2. maggāmaggassa kovidaṃ;

- magga = 「道」
- amagga = 「非道、道でないもの」
- maggāmaggassa = 「道と非道とのことに通じた」（属格複合）
- kovidaṃ = 「熟達した者、よく知る者」（対格）

→「道と非道とをよく知りつくし、熟達した者を」

3. Uttamatthamanuppattaṃ,

- uttama = 「最上の、最高の」
- atthaṃ = 「目的・利益・到達すべきもの」
- uttama‑atthaṃ = 「最上の目的・最高の到達」
- anuppattaṃ = 「到達した者」（anu‑patta〔到達する〕の過去分詞・対格）

→「最高の目的に到達した者を」

4. tamahaṃ brūmi brāhmaṇa’’nti.

- taṃ = 「そのような者を」
- ahaṃ = 「私は」
- brūmi = 「呼ぶ、言う」
- brāhmaṇaṃ = 「バラモン（まことの聖者）と」
- ’nti = 「～と（と言われた）」という引用の終わり

→「そのような者を、私は『バラモン（真の聖者）である』と言う。」

---

### 註釈部分

1. Tattha gabbhīrapaññanti

- tattha = 「そこにおいて、この語について」
- gambhīra‑paññanti = 「『gambhīrapaññaṃ（深い智慧をもつ者）』というのは」

→「ここで、『gambhīrapaññaṃ』とは」

2. gambhīresu khandhādīsu pavattāya paññāya samannāgataṃ

- gambhīresu = 「深遠なものの中において」
- khandha‑ādīsu = 「諸蘊などにおいて」  
  （khandha「五蘊」＋ādi「など」＋loc.複）
- pavattāya = 「生起する・はたらく」
- paññāya = 「智慧によって／智慧が」
- samannāgataṃ = 「そなわった者」

→「五蘊などという深遠な諸法において生起する智慧をそなえた者のことであり」

3. dhammojapaññāya samannāgataṃ medhāviṃ

- dhammoja‑paññāya = 「法から生じた智慧／法に由来する智慧（dhammoja‑paññā）」を
- samannāgataṃ = 「そなえた」
- medhāviṃ = 「聡明な者」

→「法から生じた（法に裏づけられた）智慧をそなえた、聡明な者のことである。」

4. ‘‘ayaṃ duggatiyā maggo, ayaṃ sugatiyā maggo, ayaṃ nibbānassa maggo, ayaṃ amaggo’’ti

- ayaṃ = 「これは」
- duggatiyā = 「悪趣への（悪い行き先への）」
- maggo = 「道である」
  → ayaṃ duggatiyā maggo = 「これは悪趣への道である」
- ayaṃ sugatiyā maggo = 「これは善趣への道である」
- ayaṃ nibbānassa maggo = 「これは涅槃への道である」
- ayaṃ amaggo = 「これは道ではない（非道である）」
- ’ti = 「～と」

→「『これは悪趣への道である。これは善趣への道である。これは涅槃への道である。これは道ではない』と」

5. evaṃ magge ca amagge ca chekatāya maggāmaggassa kovidaṃ

- evaṃ = 「このように」
- magge ca amagge ca = 「道においても非道においても」
- chekatāya = 「巧みさ・堪能さによって」
- maggāmaggassa kovidaṃ = 「道と非道とについてよく知る者、熟達した者」

→「このように、道にも非道にも巧みであって、道と非道とをよく知りつくした者を」

6. arahattasaṅkhātaṃ uttamatthaṃ anuppattaṃ

- arahatta‑saṅkhātaṃ = 「阿羅漢果と名づけられる」
- uttama‑atthaṃ = 「最高の目的」
- anuppattaṃ = 「到達した者」

→「阿羅漢果と名づけられる最高の目的に到達した者を」

7. tamahaṃ brāhmaṇaṃ vadāmīti attho.

- taṃ = 「そのような者を」
- ahaṃ = 「私は」
- brāhmaṇaṃ = 「バラモン（真の聖者）と」
- vadāmi = 「呼ぶ、言う」
- ti attho = 「という意味である」

→「そのような者を私は『バラモン（真の聖者）である』と言う、という意味である。」

---

## 意訳

深い智慧をそなえた聡明な人であって、  
道と非道とを見分けることに熟達し、  
阿羅漢果という最高の目的に到達した者──  
その人こそを、私はバラモン（まことの聖者）と呼ぶ。

すなわち、五蘊などの深遠な諸法において起こる、  
法に裏づけられた深い智慧を身につけ、  
「これは悪趣への道、これは善趣への道、これは涅槃への道、これは道ではない」と  
道と非道を正しく識別することに巧みであり、  
かつ阿羅漢という最高の境地に達した者を、  
ここではバラモンと呼ぶのである。

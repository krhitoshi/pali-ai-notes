原文

353.

‘‘Sabbābhibhū sabbavidūhamasmi,
Sabbesu dhammesu anūpalitto;
Sabbañjaho taṇhakkhaye vimutto,
Sayaṃ abhiññāya kamuddiseyya’’nti.

Tattha sabbābhibhūti sabbesaṃ tebhūmakadhammānaṃ abhibhavanato sabbābhibhū.
Sabbavidūti viditasabbacatubhūmakadhammo.
Sabbesu dhammesūti sabbesupi tebhūmakadhammesu taṇhādiṭṭhīhi anūpalitto.
Sabbañjahoti sabbe tebhūmakadhamme jahitvā ṭhito.
Taṇhakkhaye vimuttoti taṇhakkhayante uppādite taṇhakkhayasaṅkhāte arahatte asekhāya vimuttiyā vimutto.
Sayaṃ abhiññāyāti abhiññeyyādibhede dhamme sayameva jānitvā.
Kamuddiseyyanti ‘‘ayaṃ me upajjhāyo vā ācariyo vā’’ti kaṃ nāma uddiseyyanti.

────────────────────────
偈文（詩句）の語句分解と文法

1) Sabbābhibhū sabbavidūhamasmi,
- sabbābhibhū = sabba + abhibhū
  * sabba = 「すべて」
  * abhibhū (abhibhavati の語根) = 「征服した、凌駕した」
  → 「一切に勝った者」
- sabbavidū = sabba + vidū
  * vidū = 「知者、知っている者」
  → 「一切を知る者」
- aham asmi = 「私は…である」
→「私は一切に勝ち、一切を知る者である。」

2) Sabbesu dhammesu anūpalitto;
- sabbesu dhammesu = 「あらゆる法（事象・対象）において」
  * sabbesu = sabba の loc.pl.
  * dhammesu = dhamma の loc.pl.
- anūpalitto = an- + ūpalitta（upālitta の形）
  * upalitta / upālitta = 「塗りつけられた、汚された、付着した」
  * an- = 否定
  → 「付着しない、染まらない」
→「あらゆる法において、（私は）染まらない。」

3) Sabbañjaho taṇhakkhaye vimutto,
- sabbañjaho = sabbaṃ + jaho（jahati「捨てる」系）
  → 「一切を捨て去った者」
- taṇhakkhaye = taṇhā + khaya（「渇愛の滅」） loc.sg.（〜において／〜によって）
- vimutto = 「解脱した、 मुक्त（過去分詞）」
→「一切を捨て、渇愛の滅において（渇愛の滅によって）解脱した。」

4) Sayaṃ abhiññāya kamuddiseyya’’nti.
- sayaṃ = 「自ら」
- abhiññāya = abhiññāya（abhiññā「現観智・直接知」/ あるいは absolutive 的に「現に知って」）
  → 「自ら現に知って（自証して）」
- kaṃ uddiseyya = 「誰を指して言うべきか／誰を名指すべきか」
  * kaṃ = 「誰を」
  * uddiseyya = uddisati（指名する、挙げる、〜に言及する）optative「〜すべきだろうか」
→「自ら現に知ったうえで、いったい誰を（師として）名指すべきだろうか。」

────────────────────────
註釈（解説文）の逐語解説

① Tattha sabbābhibhūti sabbesaṃ tebhūmakadhammānaṃ abhibhavanato sabbābhibhū.
- tattha = 「ここで（この偈では）」
- sabbābhibhū ti = 「『sabbābhibhū』とは」
- sabbesaṃ tebhūmakadhammānaṃ = 「三界に属する一切の諸法の」
  * te-bhūmaka = 「三つの地（欲界・色界・無色界）に属する」
- abhibhavanato = 「凌駕することによって／凌駕するがゆえに」
→「ここで『一切に勝つ者』とは、三界に属する一切の諸法を凌駕するがゆえに『一切に勝つ者』である。」

② Sabbavidūti viditasabbacatubhūmakadhammo.
- sabbavidū ti = 「『sabbavidū』とは」
- vidita = 「知られた（＝知った）」(pp)
- sabba-catu-bhūmaka-dhammo = 「四地に属する一切の法を（について）」
  * catu-bhūmaka = 「四つの地」：註釈の定義では“三界（欲・色・無色）＋出世間（lokuttara）”を含めた枠組みを指す文脈が多い
→「『一切を知る者』とは、一切の四地（四領域）の諸法を知った者である。」

③ Sabbesu dhammesūti sabbesupi tebhūmakadhammesu taṇhādiṭṭhīhi anūpalitto.
- sabbesu dhammesu ti = 「『あらゆる法において』とは」
- sabbesupi tebhūmakadhammesu = 「三界に属するいかなる諸法においても」
- taṇhā-diṭṭhīhi = 「渇愛と見解（など）によって」
- anūpalitto = 「染まらない／付着しない」
→「『あらゆる法において』とは、三界に属するどの諸法に対しても、渇愛や見解によって染まらない（ということ）である。」

④ Sabbañjahoti sabbe tebhūmakadhamme jahitvā ṭhito.
- sabbañjaho ti = 「『一切を捨てた者』とは」
- sabbe tebhūmakadhamme = 「三界に属する一切の諸法を」
- jahitvā = 「捨てて（捨て去って）」(absolutive)
- ṭhito = 「立つ／とどまる」→「（それらを捨てて）住している」
→「『一切を捨てた者』とは、三界に属する一切の諸法を捨て去って（それらに依らずに）住している者である。」

⑤ Taṇhakkhaye vimuttoti taṇhakkhayante uppādite taṇhakkhayasaṅkhāte arahatte asekhāya vimuttiyā vimutto.
- taṇhakkhaye vimutto ti = 「『渇愛の滅により解脱した』とは」
- taṇhakkhaya-ante uppādite = 「渇愛の滅という“終点（完成）”が生起したとき」
- taṇhakkhaya-saṅkhāte arahatte = 「渇愛の滅と名づけられる阿羅漢果において」
- asekhāya vimuttiyā = 「無学（asekha）の解脱によって」
- vimutto = 「解脱した」
→「『渇愛の滅により解脱した』とは、渇愛の滅が完成して生起したとき、渇愛の滅と呼ばれる阿羅漢果において、無学の解脱によって解脱した、という意味である。」

⑥ Sayaṃ abhiññāyāti abhiññeyyādibhede dhamme sayameva jānitvā.
- sayaṃ abhiññāya ti = 「『自ら現に知って』とは」
- abhiññeyya-ādi-bhede dhamme = 「現観されるべきもの等（abhiññeyya 等）の区分に属する諸法を」
- sayam eva jānitvā = 「まさに自分自身で知って」
→「『自ら現に知って』とは、現観されるべき諸法（などの区分に属する諸法）を、他によらず自分自身で知って、ということ。」

⑦ Kamuddiseyyanti ‘‘ayaṃ me upajjhāyo vā ācariyo vā’’ti kaṃ nāma uddiseyyanti.
- kaṃ uddiseyya ti = 「誰を名指すべきだろうか」
- ayaṃ me upajjhāyo vā ācariyo vā ti = 「この人が私の和尚（upajjhāya）である、あるいは師（ācariya）である、と」
- kaṃ nāma = 「いったい誰を」
- uddiseyya = 「指名するだろうか／挙げるだろうか」
→「『誰を名指すべきだろうか』とは、『この人が私の和尚だ、この人が私の師だ』と、いったい誰を名指すことがあり得るのか、という意味である。」

────────────────────────
逐語訳（偈文）

「私は一切に勝ち、一切を知る者である。
あらゆる法において、渇愛や見解によって染まらない。
一切を捨て、渇愛の滅において解脱した。
自ら現に知った者として、いったい誰を（師として）名指すべきだろうか。」

────────────────────────
自然な日本語訳（偈文）

「私はすべてを超え、すべてを知っている。
どんな現象にも、渇愛や見解で汚されることはない。
一切を捨て去り、渇愛の滅によって解脱した。
自ら悟った私が、いったい誰を師として挙げる必要があるだろうか。」

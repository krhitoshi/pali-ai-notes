419.

‘‘Cutiṃ yo vedi sattānaṃ, upapattiñca sabbaso;

Asattaṃ sugataṃ buddhaṃ, tamahaṃ brūmi brāhmaṇaṃ.

420.

‘‘Yassa gatiṃ na jānanti, devā gandhabbamānusā;

Khīṇāsavaṃ arahantaṃ, tamahaṃ brūmi brāhmaṇa’’nti.

Tattha yo vedīti yo sattānaṃ sabbākārena cutiñca paṭisandhiñca pākaṭaṃ katvā jānāti, tamahaṃ alaggatāya asattaṃ, paṭipattiyā suṭṭhu gatattā sugataṃ, catunnaṃ saccānaṃ buddhatāya buddhaṃ brāhmaṇaṃ vadāmīti attho. Yassāti yassete devādayo gatiṃ na jānanti, tamahaṃ āsavānaṃ khīṇatāya khīṇāsavaṃ, kilesehi ārakattā arahantaṃ brāhmaṇaṃ vadāmīti attho.

## 対訳

### 419. ‘‘Cutiṃ yo vedi sattānaṃ, upapattiñca sabbaso;

1. Cutiṃ yo vedi sattānaṃ

* cutiṃ = 「死，命終」
* yo = 「～する者は」
* vedi = 「知る（vedeti の三人称単数）」
* sattānaṃ = 「有情たちの」

→ 「有情たちの死を知る者は」

2. upapattiñca sabbaso;

* upapattiṃ = 「再生，生起，次の生まれ」
* ca = 「そして」
* sabbaso = 「一切において，あらゆる点から，完全に」

→ 「そして（有情たちの）あらゆる再生を（完全に）知る者は」

---

### 419. ‘‘Asattaṃ sugataṃ buddhaṃ, tamahaṃ brūmi brāhmaṇaṃ.

3. Asattaṃ sugataṃ buddhaṃ

* asattaṃ = 「執着していない者，どこにも付着していない者」
* sugataṃ = 「善く行き着いた者，よく到達した者（涅槃に達した者）」
* buddhaṃ = 「覚った者，悟った者」

→ 「執着なき者，善く行き着いた者，覚りを得た者を」

4. tamahaṃ brūmi brāhmaṇaṃ.

* taṃ = 「その者を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ，言う」
* brāhmaṇaṃ = 「バラモン（真の聖者）と」

→ 「その者を私は『バラモン（真の聖者）』と呼ぶ。」

---

### 420. ‘‘Yassa gatiṃ na jānanti, devā gandhabbamānusā;

5. Yassa gatiṃ na jānanti

* yassa = 「その者の（～である人の）」
* gatiṃ = 「行き先，趣くところ，運命・行路（次の行き先・進み行く境地）」
* na jānanti = 「知らない」

→ 「その者の行き先を（どこへ向かっていくかを）知らない」

6. devā gandhabbamānusā;

* devā = 「天の神々」
* gandhabba = 「乾闥婆，天の音楽神」
* mānusā = 「人間たち」
* devā gandhabba-mānusā = 「神々も乾闥婆たちも人間たちも」

→ 「神々も乾闥婆たちも人間たちも（その行き先を知らない）」

---

### 420. ‘‘Khīṇāsavaṃ arahantaṃ, tamahaṃ brūmi brāhmaṇanti.

7. Khīṇāsavaṃ arahantaṃ

* khīṇāsavaṃ = 「漏（煩悩）が尽きた者」
  * khīṇa = 「滅した，尽きた」
  * āsava = 「漏（煩悩漏・汚れとなる流出）」
* arahantaṃ = 「阿羅漢，煩悩を断じた尊者」

→ 「漏が尽きた者，阿羅漢を」

8. tamahaṃ brūmi brāhmaṇanti.

* taṃ = 「その者を」
* ahaṃ = 「私は」
* brūmi = 「呼ぶ，言う」
* brāhmaṇaṃ = 「バラモン，真の聖者と」
* ti = （引用の終わりを示す語）
* nti = 「と（言った）」

→ 「その者を私は『バラモン（真の聖者）』と呼ぶのである、と（世尊は）説かれた。」

### 註釈文

#### Tattha yo vedīti yo sattānaṃ sabbākārena cutiñca paṭisandhiñca pākaṭaṃ katvā jānāti,

1. Tattha yo vedīti

* tattha = 「ここにおいて，この偈において」
* yo vedi ti = 「『yo vedi（知る者は）』とは」

→ 「ここで『yo vedi（知る者は）』とは、」

2. yo sattānaṃ sabbākārena cutiñca paṭisandhiñca pākaṭaṃ katvā jānāti,

* yo = 「～する者は」
* sattānaṃ = 「諸有情の」
* sabbākārena = 「あらゆる相において，すべての様態において」
* cutiṃ = 「死，命終を」
* ca = 「と」
* paṭisandhiṃ = 「再結生，受胎・再生を」
* pākaṭaṃ katvā = 「明らかにして，顕わにして」
* jānāti = 「知る」

→ 「諸有情の死と再結生とを、あらゆる様態において明らかにして知る者を（いう）。」

---

#### tamahaṃ alaggatāya asattaṃ,

3. tamahaṃ alaggatāya asattaṃ,

* taṃ = 「その者を」
* ahaṃ = 「私は」
* alaggatāya = 「とらわれが無いという性質によって」
  * alagga = 「掴まれていない・執着していない」
  * -tā = 「～であること，性質」
* asattaṃ = 「執着していない者，無執着の者」

→ 「その者を、執着の無いことによって『執着なき者』と」

---

#### paṭipattiyā suṭṭhu gatattā sugataṃ,

4. paṭipattiyā suṭṭhu gatattā sugataṃ,

* paṭipattiyā = 「修行・実践によって」
* suṭṭhu = 「よく，完全に」
* gatattā = 「到達しているがゆえに」
  * gata = 「行った，至った」
  * -ttā = 「～であることによって」
* sugataṃ = 「善く行き着いた者，善逝」

→ 「修行によってよく到達しているがゆえに、『善く行き着いた者（善逝）』と」

---

#### catunnaṃ saccānaṃ buddhatāya buddhaṃ brāhmaṇaṃ vadāmīti attho.

5. catunnaṃ saccānaṃ buddhatāya buddhaṃ brāhmaṇaṃ vadāmīti attho.

* catunnaṃ saccānaṃ = 「四つの〔聖〕諦（四聖諦）の」
* buddhatāya = 「覚知しているということによって」
  * buddha = 「覚った」
  * -tāya = 「その～であることによって」
* buddhaṃ = 「覚った者，ブッダ（悟者）と」
* brāhmaṇaṃ = 「バラモン（真の聖者）と」
* vadāmi = 「私は言う，呼ぶ」
* ti attho = 「という意味である」

→ 「四つの聖諦を覚知していることによって『覚った者，バラモン』と私は言うのである、という意味である。」

---

#### Yassāti yassete devādayo gatiṃ na jānanti,

6. Yassāti yassete devādayo gatiṃ na jānanti,

* yassā ti = 「『yassa（その者の）』とは」
* yassa = 「その者の」
* ete = 「これらの」
* devādayo = 「神々等は」
* gatiṃ = 「行き先，境地，行路を」
* na jānanti = 「知らない」

→ 「『yassa（その者の）』とは、これらの神々などがその者の行き先を知らない（ような者の、ということである）。」

---

#### tamahaṃ āsavānaṃ khīṇatāya khīṇāsavaṃ,

7. tamahaṃ āsavānaṃ khīṇatāya khīṇāsavaṃ,

* taṃ = 「その者を」
* ahaṃ = 「私は」
* āsavānaṃ = 「漏（諸煩悩漏）の」
* khīṇatāya = 「尽きているということによって」
  * khīṇa = 「滅した，尽きた」
  * -tāya = 「その～であることにより」
* khīṇāsavaṃ = 「漏が尽きた者」

→ 「その者を，諸漏が尽きていることによって『漏尽の者』と」

---

#### kilesehi ārakattā arahantaṃ brāhmaṇaṃ vadāmīti attho.

8. kilesehi ārakattā arahantaṃ brāhmaṇaṃ vadāmīti attho.

* kilesehi = 「煩悩から」
* ārakattā = 「遠く離れているがゆえに」
  * āraka = 「遠ざかっている，離れている」
  * -ttā = 「その～であることによって」
* arahantaṃ = 「阿羅漢，煩悩を断じた者を」
* brāhmaṇaṃ = 「バラモン（真の聖者）と」
* vadāmi = 「私は言う，呼ぶ」
* ti attho = 「という意味である」

→ 「煩悩から遠く離れているがゆえに『阿羅漢，バラモン』と私は言うのである、という意味である。」

---

## 意訳

「有情たちがどのように死に、どのように再び生まれゆくかを、  
あらゆる側面から完全に見通して知る者。  
執着するところなく、修行を通してよく到達し、  
四つの聖なる真理をさとりきった悟者。  
そのような人を、私は真のバラモンと呼ぶ。

さらに、その人がどこへ向かい、どのような境地に至るかを、  
天の神々も、乾闥婆たちも、人間たちも知りえない。  
漏れるがごとき煩悩がことごとく尽きはて、  
煩悩から遠く離れきった阿羅漢。  
そのような人をこそ、私は真のバラモンと呼ぶのである。」

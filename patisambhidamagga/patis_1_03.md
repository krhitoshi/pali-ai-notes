### 対訳

1. Satokāriñāṇaniddeso

* sato = 「気づきある、念をそなえた」（sati の派生形容詞、主格単数）

* kārī = 「行ずる者、為す者」（√kar +ṇin、行為者名詞）

  * satokārī = 持業釈/有財釈「気づきをもって行ずる者」

* ñāṇa = 「智」（中性、ここでは複合語前分・複数を含意）

* niddesa = 「解説、詳説」（ni-√dis、男性・主格単数）

→「気づきをもって行ずる者の智の解説（章）」

2. 163. Katamāni bāttiṃsa satokārissa ñāṇāni?

* katamāni = 「いずれの、何という（ものか）」（疑問代名詞、中性・主格複数）

* bāttiṃsa = 「三十二（の）」（数詞）

* satokārissa = 「気づきをもって行ずる者の」（属格単数）

* ñāṇāni = 「諸々の智」（中性・主格複数）

→「気づきをもって行ずる者の三十二の智とは何か。」

3. Idha bhikkhu araññagato vā rukkhamūlagato vā suññāgāragato vā nisīdati pallaṅkaṃ ābhujitvā ujuṃ kāyaṃ paṇidhāya parimukhaṃ satiṃ upaṭṭhapetvā.

* idha = 「ここに（=この教えにおいて）」

* bhikkhu = 「比丘」（男性・主格単数）

* araññagato = 「林に赴いた（者）」（arañña「林」+ gata「赴いた、過去分詞」、主格単数）

* vā … vā … vā = 「あるいは…あるいは…あるいは」

* rukkhamūlagato = 「樹下に赴いた（者）」（rukkha「樹」+ mūla「根・下」+ gata）

* suññāgāragato = 「空閑処に赴いた（者）」（suñña「空なる」+ agāra「家・処」+ gata）

* nisīdati = 「坐す」（3単・現在）

* pallaṅkaṃ ābhujitvā = 「結跏（趺坐）を組んで」

  * pallaṅkaṃ = 「跏趺坐」（男性・対格単数）

  * ābhujitvā = 「組んで、曲げて」（ā-√bhuj の連続体/絶対分詞）

* ujuṃ kāyaṃ paṇidhāya = 「身をまっすぐに保って」

  * ujuṃ = 「まっすぐに」（副詞的対格）

  * kāyaṃ = 「身を」（男性・対格単数）

  * paṇidhāya = 「向け定めて、整えて」（pa-ni-√dhā の連続体）

* parimukhaṃ satiṃ upaṭṭhapetvā = 「念を現前に確立して」

  * parimukhaṃ = 「面前に、現前に」（副詞的対格）

  * satiṃ = 「念を」（女性・対格単数）

  * upaṭṭhapetvā = 「確立して、現起させて」（upa-√ṭhā 使役の連続体）

→「ここに比丘は、林に赴き、あるいは樹下に赴き、あるいは空閑処に赴いて、結跏趺坐を組み、身をまっすぐに保ち、念を現前に確立して坐す。」

4. So satova assasati sato passasati.

* so = 「彼は」（男性・主格単数）

* sato va = 「まさに念ある者として」（sato「念ある」主格単数 + eva 強調）

* assasati = 「出息する（息を吐く）」（3単・現在）
  〔註：ヴィナヤ註・清浄道論の伝統では assāsa＝外に出る息＝出息、passāsa＝内に入る息＝入息。経註では方向が逆転する点に注意〕

* sato = 「念ある者として」（主格単数）

* passasati = 「入息する（息を吸う）」（3単・現在）

→「彼はまさに念をそなえて出息し、念をそなえて入息する。」

5. Dīghaṃ vā assasanto ‘‘dīghaṃ assasāmī’’ti pajānāti. Dīghaṃ vā passasanto ‘‘dīghaṃ passasāmī’’ti pajānāti.

* dīghaṃ = 「長く」（副詞的対格）

* vā = 「あるいは（=～する場合には）」（列挙の連辞）

* assasanto = 「出息しつつ」（現在分詞、男性・主格単数）

* ‘‘dīghaṃ assasāmī’’ti = 「『私は長く出息する』と」

  * assasāmi = 「私は出息する」（1単・現在）

* pajānāti = 「了知する、はっきりと知る」（pa-√ñā、3単・現在）

* passasanto = 「入息しつつ」（現在分詞、主格単数）

  * passasāmi = 「私は入息する」（1単・現在）

→「長く出息しつつ『私は長く出息する』と了知し、長く入息しつつ『私は長く入息する』と了知する。」

6. Rassaṃ vā assasanto ‘‘rassaṃ assasāmī’’ti pajānāti. Rassaṃ vā passasanto ‘‘rassaṃ passasāmī’’ti pajānāti.

* rassaṃ = 「短く」（副詞的対格）

* assasanto / passasanto = 「出息しつつ／入息しつつ」（現在分詞、主格単数）

* rassaṃ assasāmi / rassaṃ passasāmi = 「私は短く出息する／私は短く入息する」（1単・現在）

* pajānāti = 「了知する」（3単・現在）

→「短く出息しつつ『私は短く出息する』と了知し、短く入息しつつ『私は短く入息する』と了知する。」

7. ‘‘Sabbakāyapaṭisaṃvedī assasissāmī’’ti sikkhati. ‘‘Sabbakāyapaṭisaṃvedī passasissāmī’’ti sikkhati.

* sabbakāyapaṭisaṃvedī = 「全身を感知する者として」（有財釈、主格単数）

  * sabba = 「一切の、全」

  * kāya = 「身」（ここでは息の全過程＝初・中・後の全体を含意）

  * paṭisaṃvedī = 「感知する者、感受する者」（paṭi-saṃ-√vid +ṇin）

* assasissāmi / passasissāmi = 「私は出息しよう／私は入息しよう」（1単・未来）

* ti sikkhati = 「と学ぶ、と修学する」（√sikkh、3単・現在）

→「『全身を感知しながら出息しよう』と学び、『全身を感知しながら入息しよう』と学ぶ。」

8. ‘‘Passambhayaṃ kāyasaṅkhāraṃ assasissāmī’’ti sikkhati. ‘‘Passambhayaṃ kāyasaṅkhāraṃ passasissāmī’’ti sikkhati.

* passambhayaṃ = 「静めつつ、軽安ならしめつつ」（pa-√sambh 使役の現在分詞、主格単数）

* kāyasaṅkhāraṃ = 「身行を」（対格単数）

  * kāya = 「身」

  * saṅkhāra = 「行（形成作用）」＝ここでは入出息そのもの

* assasissāmi / passasissāmi = 「私は出息しよう／私は入息しよう」（1単・未来）

* ti sikkhati = 「と学ぶ」（3単・現在）

→「『身行を静めながら出息しよう』と学び、『身行を静めながら入息しよう』と学ぶ。」

9. Pītipaṭisaṃvedī…pe… sukhapaṭisaṃvedī… cittasaṅkhārapaṭisaṃvedī… passambhayaṃ cittasaṅkhāraṃ… cittapaṭisaṃvedī… abhippamodayaṃ cittaṃ… samādahaṃ cittaṃ… vimocayaṃ cittaṃ… aniccānupassī… virāgānupassī… nirodhānupassī… ‘‘paṭinissaggānupassī assasissāmī’’ti sikkhati. ‘‘Paṭinissaggānupassī passasissāmī’’ti sikkhati.

* …pe… = peyyāla（中略。「…と学ぶ。出息につき・入息につき」の反復句が省略される）

* pītipaṭisaṃvedī = 「喜を感知する者として」（pīti「喜」+ paṭisaṃvedī）〔第二の四種法・第五段〕

* sukhapaṭisaṃvedī = 「楽を感知する者として」（sukha「楽」）〔第六段〕

* cittasaṅkhārapaṭisaṃvedī = 「心行を感知する者として」（cittasaṅkhāra「心行」＝受・想）〔第七段〕

* passambhayaṃ cittasaṅkhāraṃ = 「心行を静めつつ」（現在分詞+対格単数）〔第八段〕

* cittapaṭisaṃvedī = 「心を感知する者として」（citta「心」）〔第三の四種法・第九段〕

* abhippamodayaṃ cittaṃ = 「心を歓喜させつつ」（abhi-pa-√mud 使役の現在分詞＋対格）〔第十段〕

* samādahaṃ cittaṃ = 「心を等持（定）せしめつつ」（saṃ-ā-√dhā の現在分詞＋対格）〔第十一段〕

* vimocayaṃ cittaṃ = 「心を解脱せしめつつ」（vi-√muc 使役の現在分詞＋対格）〔第十二段〕

* aniccānupassī = 「無常を随観する者として」（anicca「無常」+ anupassī「随観する者」）〔第四の四種法・第十三段〕

* virāgānupassī = 「離貪を随観する者として」（virāga「離貪・消滅」）〔第十四段〕

* nirodhānupassī = 「滅尽を随観する者として」（nirodha「滅」）〔第十五段〕

* paṭinissaggānupassī = 「捨遣（しゃけん）を随観する者として」（paṭinissagga「捨て遣ること・捨離」+ anupassī）〔第十六段〕

  * assasissāmi / passasissāmi = 「私は出息しよう／私は入息しよう」（1単・未来）

  * ti sikkhati = 「と学ぶ」（3単・現在）

→「『喜を感知しながら（出息・入息しよう）』……『楽を感知しながら……』『心行を感知しながら……』『心行を静めながら……』『心を感知しながら……』『心を歓喜させながら……』『心を等持しながら……』『心を解脱せしめながら……』『無常を随観しながら……』『離貪を随観しながら……』『滅尽を随観しながら……』〔と学び〕、『捨遣を随観しながら出息しよう』と学び、『捨遣を随観しながら入息しよう』と学ぶ。」

## 164

164.Idhāti imissā diṭṭhiyā, imissā khantiyā, imissā ruciyā, imasmiṃ ādāye, imasmiṃ dhamme, imasmiṃ vinaye, imasmiṃ dhammavinaye, imasmiṃ pāvacane, imasmiṃ brahmacariye, imasmiṃ satthusāsane. Tena vuccati – ‘‘idhā’’ti. Bhikkhūti puthujjanakalyāṇako vā hoti bhikkhu sekkho vā arahā vā akuppadhammo. Araññanti nikkhamitvā bahi indakhīlā sabbametaṃ araññaṃ. Rukkhamūlanti yattha bhikkhuno āsanaṃ paññattaṃ hoti mañco vā pīṭhaṃ vā bhisi vā taṭṭikā vā cammakhaṇḍo vā tiṇasantharo [tiṇasaṇṭharo (syā.)] vā paṇṇasantharo vā palālasantharo [palāsasaṇṭharo (syā.)] vā, tattha bhikkhu caṅkamati vā tiṭṭhati vā nisīdati vā seyyaṃ vā kappeti. Suññanti kenaci anākiṇṇaṃ hoti gahaṭṭhehi vā pabbajitehi vā. Agāranti [āgāranti (syā.)] vihāro aḍḍhayogo pāsādo hammiyaṃ guhā. Nisīdati pallaṅkaṃ ābhujitvāti nisinno hoti pallaṅkaṃ ābhujitvā . Ujuṃ kāyaṃ paṇidhāyāti ujuko hoti kāyo ṭhito [panidhito (syā.)] supaṇihito. Parimukhaṃ satiṃ upaṭṭhapetvāti. Parīti pariggahaṭṭho. Mukhanti niyyānaṭṭho. Satīti upaṭṭhānaṭṭho. Tena vuccati – ‘‘parimukhaṃ satiṃ upaṭṭhapetvā’’ti.

### 対訳

1. Idhā ti

  * idha =「ここに、この（教え・場において）」（副詞）
  * ti =「～と（は）」（引用・標示の辞、註釈される語を示す）
  * imissā diṭṭhiyā = imissā〔ayaṃ 女性・単数・処格〕「この」＋ diṭṭhiyā〔diṭṭhi 女性・単数・処格〕「見解において」
  * imissā khantiyā = khanti〔処格〕「忍・信受（受け入れ）において」
  * imissā ruciyā = ruci〔処格〕「楽欲・好み（傾倒）において」
  * imasmiṃ ādāye = imasmiṃ〔中/男性・単数・処格〕「この」＋ ādāya〔処格〕「受持（教えの取り受け）において」
  * imasmiṃ dhamme = dhamma〔処格〕「法において」
  * imasmiṃ vinaye = vinaya〔処格〕「律において」
  * imasmiṃ dhammavinaye = dhamma-vinaya〔並列複合・処格〕「法と律（法律）において」
  * imasmiṃ pāvacane = pāvacana〔処格〕「聖教（仏の教説）において」
  * imasmiṃ brahmacariye = brahmacariya〔処格〕「梵行において」
  * imasmiṃ satthusāsane = satthu-sāsana〔属格依主複合・処格〕「師の教説（教え）において」
  * Tena vuccati – ‘‘idhā’’ti = tena〔具格〕「それ故に」＋ vuccati〔現在・受動〕「言われる」
    → 「それ故に『idha（ここに）』と説かれる」

  →「『idha（ここに）』とは、この見解において、この忍（信受）において、この楽欲において、この受持において、この法において、この律において、この法と律において、この聖教において、この梵行において、この師の教えにおいて、という意味である。それ故に『ここに』と説かれるのである。」

2. Bhikkhū ti

  * bhikkhu =「比丘」（註釈される語）
  * puthujjana-kalyāṇako vā = puthujjana「凡夫」＋ kalyāṇaka「善き者」→「善凡夫（賢善なる凡夫）であるか」
  * hoti bhikkhu =「である比丘」
  * sekkho vā = sekkha「有学（学びの途上にある者）であるか」
  * arahā vā = arahant「阿羅漢であるか」
  * akuppadhammo = a-kuppa「動じない」＋ dhamma「性質」→「不動法者（解脱が揺るがぬ性質の阿羅漢）」

  →「『比丘』とは、善凡夫（賢善なる凡夫）である比丘か、あるいは有学か、あるいは阿羅漢である不動法者をいう。」

3. Araññan ti

  * araññaṃ =「阿蘭若・森」（註釈される語）
  * nikkhamitvā = nikkhamati の連続体「（村を）出て」
  * bahi indakhīlā = bahi「外に」＋ indakhīla〔奪格〕「インダキーラ（村境の門柱・境界標）より」→「インダキーラの外に」
  * sabbam etaṃ araññaṃ = sabba「すべて」＋ etaṃ「これ（その範囲）」＋ araññaṃ「森である」
    ※村境の標柱の外側を一切「阿蘭若」とする広義（律・対法的）の定義。

  →「『阿蘭若（森）』とは、（村を）出てインダキーラ（境界の門柱）の外、それらすべてが阿蘭若である。」

4. Rukkhamūlan ti

  * rukkhamūla = rukkha「樹」＋ mūla「根・もと」→「樹下（樹の根もと）」
  * yattha =「～するところの（場所）」（関係副詞）
  * bhikkhuno āsanaṃ paññattaṃ hoti = bhikkhuno〔属格〕「比丘のために」＋ āsanaṃ「座が」＋ paññatta〔過去分詞〕「設けられて」＋ hoti「いる」
  * mañco vā =「寝台か」
  * pīṭhaṃ vā =「椅子（腰掛け）か」
  * bhisi vā =「敷物（褥・クッション）か」
  * taṭṭikā vā =「編んだ敷物（ござ）か」
  * cammakhaṇḍo vā = camma「皮」＋ khaṇḍa「片」→「皮の切れ端か」
  * tiṇasantharo vā = tiṇa「草」＋ santhara「敷物」→「草の敷物か」
  * paṇṇasantharo vā = paṇṇa「葉」＋ santhara →「葉の敷物か」
  * palālasantharo vā = palāla「藁」＋ santhara →「藁の敷物か」
  * tattha =「そこで」
  * caṅkamati vā =「経行（歩行）するか」
  * tiṭṭhati vā =「立つか」
  * nisīdati vā =「座るか」
  * seyyaṃ vā kappeti = seyya「臥処」＋ kappeti「設える」→「臥処を設ける（横臥する）」

  →「『樹下』とは、比丘のために座が設けられているところ――寝台か、椅子か、敷物か、ござか、皮の切れ端か、草の敷物か、葉の敷物か、藁の敷物か――そこで比丘が経行し、あるいは立ち、あるいは座り、あるいは臥処を設ける（横たわる）、その場所をいう。」

5. Suññan ti

  * suññaṃ =「空（くう）なる・人気のない」（註釈される語）
  * kenaci anākiṇṇaṃ hoti = kenaci〔具格・不定〕「誰によっても」＋ an-ākiṇṇa「混み合っていない（ākiṇṇa の否定）」＋ hoti「である」
  * gahaṭṭhehi vā = gahaṭṭha〔具格・複数〕「在家者によってか」
  * pabbajitehi vā = pabbajita〔具格・複数〕「出家者によってか」

  →「『空なる（人気のない）』とは、在家者によっても出家者によっても、誰によっても混み合っていない（空閑である）ことをいう。」

6. Agāran ti

  * agāra =「住処・建物」（註釈される語）
  * vihāro =「精舎」
  * aḍḍhayogo = aḍḍha「半」＋ yoga「（屋根の）構え」→「片屋根の家屋」
  * pāsādo =「高楼（重層の楼閣）」
  * hammiyaṃ =「平屋根の家（上に居室を備えた建物）」
  * guhā =「洞窟」

  →「『住処（建物）』とは、精舎、片屋根の家屋、高楼、平屋根家、洞窟をいう。」

7. Nisīdati pallaṅkaṃ ābhujitvā ti

  * nisīdati =「座る」（註釈される語）
  * pallaṅkaṃ = pallaṅka〔対格〕「結跏趺坐（組んだ脚）を」
  * ābhujitvā = ābhujati の連続体「曲げて・組んで」
  * nisinno hoti = nisinna〔過去分詞〕「座して」＋ hoti「いる」

  →「『結跏趺坐を組んで座る』とは、結跏趺坐を組んで座っている、ということである。」

8. Ujuṃ kāyaṃ paṇidhāya ti

  * ujuṃ =「真っ直ぐに」（副詞的対格）
  * kāyaṃ = kāya〔対格〕「身を」
  * paṇidhāya = paṇidahati の連続体「据えて・向けて」
  * ujuko hoti kāyo = ujuka「真っ直ぐな」＋ kāya「身が」＋ hoti「である」
  * ṭhito = ṭhita〔過去分詞〕「安定して立てられた」
  * supaṇihito = su「善く」＋ paṇihita〔過去分詞〕「据えられた」→「善く据えられた」

* gocarañ ca pajānāti = 「行境をも了知する」（標挙句、gocaraṃ + ca + pajānāti）
* ti = 「～とは」（引用標識 iti）
* yaṃ = 「およそ～であるもの」（関係代名詞・中性主格）
* tassa = 「その者にとって」（属格）
* ārammaṇaṃ = 「所縁である（もの）」（主格）
* taṃ = 「それが」（指示代名詞・中性主格）
* tassa gocaraṃ = 「その者にとっての行境である」

→「『行境をも了知する』とは——その者にとって所縁であるもの、それがその者にとっての行境である。」

2. Yaṃ tassa gocaraṃ, taṃ tassa ārammaṇaṃ pajānātīti puggalo.

* yaṃ tassa gocaraṃ = 「その者にとって行境であるもの」
* taṃ tassa ārammaṇaṃ = 「それがその者にとっての所縁である」
* pajānāti = 「了知する」（ここから新たな標挙語）
* ti = 「～とは」（引用標識）
* puggalo = 「人（補特伽羅）である」（主格、了知の動作主を指す）

※構文上、"taṃ tassa ārammaṇaṃ" で前文の対句が完結し、続く "pajānātīti puggalo" は別個の標挙釈。

→「その者にとって行境であるもの、それがその者にとっての所縁である。『了知する（pajānāti）』とは、人〔補特伽羅〕である。」

3. Pajānanā paññā.

* pajānanā = 「了知すること（了知の作用）」（主格・女性）
* paññā = 「慧である」（主格・女性）

→「了知すること（pajānanā）は、慧である。」

4. Samanti ārammaṇassa upaṭṭhānaṃ samaṃ, cittassa avikkhepo samaṃ, cittassa adhiṭṭhānaṃ samaṃ, cittassa vodānaṃ samaṃ.

* samaṃ ti = 「『平等（sama）』とは」（標挙句）
* ārammaṇassa upaṭṭhānaṃ = 「所縁の現起（現前）が」（属格 + 主格）
  * samaṃ = 「平等である」（述語・中性主格）
* cittassa avikkhepo = 「心の不散乱が」
  * samaṃ = 「平等である」
* cittassa adhiṭṭhānaṃ = 「心の確立（住立・専注）が」
  * samaṃ = 「平等である」
* cittassa vodānaṃ = 「心の清白（浄化）が」
  * samaṃ = 「平等である」

→「『平等（sama）』とは——所縁の現起が平等であり、心の不散乱が平等であり、心の確立が平等であり、心の清白が平等である〔という意味である〕。」

5. Atthoti anavajjaṭṭho nikkilesaṭṭho vodānaṭṭho paramaṭṭho.

* attho ti = 「『意義（attha）』とは」（標挙句）
* anavajja-ṭṭho = 「無罪（無過）という意味」（an-avajja「罪・過なき」+ aṭṭha「意義」）
* nikkilesa-ṭṭho = 「無煩悩という意味」（nis-kilesa「煩悩を離れた」）
* vodāna-ṭṭho = 「清白（浄化）という意味」
* parama-ṭṭho = 「最勝（最高）という意味」

→「『意義（attha）』とは——無罪という意味、無煩悩という意味、清白という意味、最勝という意味である。」

6. Paṭivijjhatīti ārammaṇassa upaṭṭhānaṭṭhaṃ paṭivijjhati, cittassa avikkhepaṭṭhaṃ paṭivijjhati, cittassa adhiṭṭhānaṭṭhaṃ paṭivijjhati, cittassa vodānaṭṭhaṃ paṭivijjhati.

* paṭivijjhati ti = 「『洞察する（paṭivijjhati）』とは」（標挙句）
* ārammaṇassa upaṭṭhāna-ṭṭhaṃ = 「所縁の現起という意味を」（対格）
  * paṭivijjhati = 「洞察する、通達する、貫く」（paṭi-√vyadh）
* cittassa avikkhepa-ṭṭhaṃ = 「心の不散乱という意味を」
* cittassa adhiṭṭhāna-ṭṭhaṃ = 「心の確立という意味を」
* cittassa vodāna-ṭṭhaṃ = 「心の清白という意味を」

→「『洞察する』とは——所縁の現起という意味を洞察し、心の不散乱という意味を洞察し、心の確立という意味を洞察し、心の清白という意味を洞察する〔ことである〕。」

7. Tena vuccati – ‘‘samatthañca paṭivijjhatī’’ti.

* tena = 「それゆえに」（具格・理由）
* vuccati = 「説かれる、言われる」（受動・三人称単数、√vac）
* samatthañ ca paṭivijjhati = 「平等〔の意義〕をも洞察する」
* ti = 「と」（引用標識）

→「それゆえに『平等〔の意義〕をも洞察する』と説かれるのである。」

  * samatthaṃ = 「意義、道理、対象（artha）」（対格）／ca = 「また」
  * paṭivijjhati = 「通達する、洞察する、貫き徹る」（paṭi + vijjhati）

→「それゆえに『また〔その〕意義を通達する』と言われる。」

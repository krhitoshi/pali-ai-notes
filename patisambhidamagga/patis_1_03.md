
## 166

Paṭhamacatukkaniddeso

166. Kathaṃ dīghaṃ assasanto ‘‘dīghaṃ assasāmī’’ti pajānāti, dīghaṃ passasanto ‘‘dīghaṃ passasāmī’’ti pajānāti? Dīghaṃ assāsaṃ addhānasaṅkhāte assasati, dīghaṃ passāsaṃ addhānasaṅkhāte passasati, dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatipi passasatipi. Dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatopi passasatopi chando uppajjati. Chandavasena tato sukhumataraṃ dīghaṃ assāsaṃ addhānasaṅkhāte assasati, chandavasena tato sukhumataraṃ dīghaṃ passāsaṃ addhānasaṅkhāte passasati, chandavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatipi passasatipi. Chandavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatopi passasatopi pāmojjaṃ uppajjati. Pāmojjavasena tato sukhumataraṃ dīghaṃ assāsaṃ addhānasaṅkhāte assasati, pāmojjavasena tato sukhumataraṃ dīghaṃ passāsaṃ addhānasaṅkhāte passasati, pāmojjavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatipi passasatipi. Pāmojjavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatopi passasatopi dīghaṃ assāsapassāsāpi cittaṃ vivattati, upekkhā saṇṭhāti. Imehi navahākārehi dīghaṃ assāsapassāsā kāyo. Upaṭṭhānaṃ sati. Anupassanā ñāṇaṃ. Kāyo upaṭṭhānaṃ, no sati; sati upaṭṭhānañceva sati ca. Tāya satiyā tena ñāṇena taṃ kāyaṃ anupassati. Tena vuccati – ‘‘kāye kāyānupassanāsatipaṭṭhānabhāvanā’’ti.

### 対訳

1. Paṭhamacatukkaniddeso

* paṭhama = 「第一の」
* catukka = 「四法（四つの組）」
* niddeso = 「解説、説明」

→ 「第一の四法の解説」

2. 166. Kathaṃ dīghaṃ assasanto ‘‘dīghaṃ assasāmī’’ti pajānāti, dīghaṃ passasanto ‘‘dīghaṃ passasāmī’’ti pajānāti?

* kathaṃ = 「どのように」
* dīghaṃ = 「長く」
* assasanto = 「入息しながら（息を吸いながら）」
* dīghaṃ assasāmi iti = 「『私は長く入息する』と」
* pajānāti = 「はっきりと知る」
* dīghaṃ passasanto = 「長く出息しながら（息を吐きながら）」
* dīghaṃ passasāmi iti = 「『私は長く出息する』と」
* pajānāti = 「はっきりと知る」

→ 「166. どのようにして、長く入息しながら『私は長く入息する』と知り、長く出息しながら『私は長く出息する』と知るのか？」

3. Dīghaṃ assāsaṃ addhānasaṅkhāte assasati, dīghaṃ passāsaṃ addhānasaṅkhāte passasati, dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatipi passasatipi.

* dīghaṃ assāsaṃ = 「長い入息を」
* addhāna-saṅkhāte = 「期間とみなして（時間の長さとしてはかって）」
* assasati = 「入息する」
* dīghaṃ passāsaṃ = 「長い出息を」
* addhāna-saṅkhāte = 「期間とみなして」
* passasati = 「出息する」
* dīghaṃ assāsa-passāsaṃ = 「長い入出息を」
* addhāna-saṅkhāte = 「期間とみなして」
* assasati pi passasati pi = 「入息も出息もする」

→ 「長い入息を期間としてはかって入息し、長い出息を期間としてはかって出息し、長い入出息を期間としてはかって入息も出息もする。」

4. Dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatopi passasatopi chando uppajjati.

* dīghaṃ assāsa-passāsaṃ = 「長い入出息を」
* addhāna-saṅkhāte = 「期間とみなして」
* assasato pi passasato pi = 「入息し出息している者に（～していると）」
* chando = 「意欲（熱意）が」
* uppajjati = 「生じる」

→ 「長い入出息を期間としてはかって入息し出息していると、意欲が生じる。」

5. Chandavasena tato sukhumataraṃ dīghaṃ assāsaṃ addhānasaṅkhāte assasati, chandavasena tato sukhumataraṃ dīghaṃ passāsaṃ addhānasaṅkhāte passasati, chandavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatipi passasatipi.

* chanda-vasena = 「意欲の力（支配）によって」
* tato = 「それよりも」
* sukhumataraṃ = 「さらに微細な」
* dīghaṃ assāsaṃ addhāna-saṅkhāte assasati = 「長い入息を期間としてはかって入息する」
* dīghaṃ passāsaṃ addhāna-saṅkhāte passasati = 「長い出息を期間としてはかって出息する」
* dīghaṃ assāsa-passāsaṃ addhāna-saṅkhāte assasati pi passasati pi = 「長い入出息を期間としてはかって入息も出息もする」

→ 「意欲の力によって、それよりもさらに微細な長い入息を期間としてはかって入息し、意欲の力によって、それよりもさらに微細な長い出息を期間としてはかって出息し、意欲の力によって、それよりもさらに微細な長い入出息を期間としてはかって入息も出息もする。」

6. Chandavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatopi passasatopi pāmojjaṃ uppajjati.

* chanda-vasena = 「意欲の力によって」
* tato sukhumataraṃ = 「それよりもさらに微細な」
* dīghaṃ assāsa-passāsaṃ addhāna-saṅkhāte assasato pi passasato pi = 「長い入出息を期間としてはかって入息し出息していると」
* pāmojjaṃ = 「歓喜が」
* uppajjati = 「生じる」

→ 「意欲の力によって、それよりもさらに微細な長い入出息を期間としてはかって入息し出息していると、歓喜が生じる。」

7. Pāmojjavasena tato sukhumataraṃ dīghaṃ assāsaṃ addhānasaṅkhāte assasati, pāmojjavasena tato sukhumataraṃ dīghaṃ passāsaṃ addhānasaṅkhāte passasati, pāmojjavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatipi passasatipi.

* pāmojja-vasena = 「歓喜の力によって」
* tato sukhumataraṃ dīghaṃ assāsaṃ addhāna-saṅkhāte assasati = 「それよりもさらに微細な長い入息を期間としてはかって入息する」
* tato sukhumataraṃ dīghaṃ passāsaṃ addhāna-saṅkhāte passasati = 「それよりもさらに微細な長い出息を期間としてはかって出息する」
* tato sukhumataraṃ dīghaṃ assāsa-passāsaṃ addhāna-saṅkhāte assasati pi passasati pi = 「それよりもさらに微細な長い入出息を期間としてはかって入息も出息もする」

→ 「歓喜の力によって、それよりもさらに微細な長い入息を期間としてはかって入息し、歓喜の力によって、それよりもさらに微細な長い出息を期間としてはかって出息し、歓喜の力によって、それよりもさらに微細な長い入出息を期間としてはかって入息も出息もする。」

8. Pāmojjavasena tato sukhumataraṃ dīghaṃ assāsapassāsaṃ addhānasaṅkhāte assasatopi passasatopi dīghaṃ assāsapassāsāpi cittaṃ vivattati, upekkhā saṇṭhāti.

* pāmojja-vasena = 「歓喜の力によって」
* tato sukhumataraṃ = 「それよりもさらに微細な」
* dīghaṃ assāsa-passāsaṃ addhāna-saṅkhāte assasato pi passasato pi = 「長い入出息を期間としてはかって入息し出息していると」
* dīghaṃ assāsa-passāsā pi = 「長い入出息からも」
* cittaṃ = 「心が」
* vivattati = 「離れる（背を向ける）」
* upekkhā = 「捨（平静さ）が」
* saṇṭhāti = 「確立する」

→ 「歓喜の力によって、それよりもさらに微細な長い入出息を期間としてはかって入息し出息していると、長い入出息からも心が離れ、捨（平静さ）が確立する。」

9. Imehi navahākārehi dīghaṃ assāsapassāsā kāyo.

* imehi = 「これらの」
* navahi ākārehi = 「九つの相（プロセス）によって」
* dīghaṃ assāsa-passāsā = 「長い入出息は」
* kāyo = 「身（対象となるもの）である」

→ 「これらの九つの相によって、長い入出息は身である。」

10. Upaṭṭhānaṃ sati. Anupassanā ñāṇaṃ.

* upaṭṭhānaṃ = 「現前（対象が心に立ち現れること）は」
* sati = 「念（気づき）である」
* anupassanā = 「随観（観察）は」
* ñāṇaṃ = 「智（知識）である」

→ 「現前は念であり、随観は智である。」

11. Kāyo upaṭṭhānaṃ, no sati; sati upaṭṭhānañceva sati ca.

* kāyo = 「身（呼吸）は」
* upaṭṭhānaṃ = 「現前（現れ出た対象）であるが」
* no sati = 「念ではない」
* sati = 「念は」
* upaṭṭhānañ ceva = 「現前そのものであり」
* sati ca = 「念でもある」

→ 「身は現前であるが念ではない。念は現前であり、かつ念である。」

12. Tāya satiyā tena ñāṇena taṃ kāyaṃ anupassati.

* tāya satiyā = 「その念によって」
* tena ñāṇena = 「その智によって」
* taṃ kāyaṃ = 「その身を」
* anupassati = 「随観する」

→ 「その念によって、その智によって、その身を随観する。」

13. Tena vuccati – ‘‘kāye kāyānupassanāsatipaṭṭhānabhāvanā’’ti.

* tena = 「それゆえに」
* vuccati = 「（～と）言われる」
* kāye = 「身において」
* kāyānupassanā-sati-paṭṭhāna-bhāvanā iti = 「身随観の念処の修習である、と」

→ 「それゆえに、『身において身を随観する念処の修習である』と言われる。」

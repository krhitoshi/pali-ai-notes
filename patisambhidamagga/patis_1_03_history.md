### 2026/06/28 ānāpāna 訳語の統一 (訳語バッチ)

- 内容: ānāpāna 由来の訳語を「出入息」系に統一
  - 安般念 / 安般 (音写) → 出入息念 / 出入息 (安般念論→出入息念論, 安般論→出入息論 も連動)
  - 入出息念 (ānāpānassati) → 出入息念
  - 入出息(章) (ānāpāna 章名) → 出入息(章)
  - 訳語と音写の併記 2 箇所を解消 (出入息念・安般念→出入息念, 出入息念（安般念）→出入息念)
- 結果: 出入息念 50 件. 安般・入出息念 は 0 件
- 保持 (今回対象外): assāsapassāsa 由来の「入出息」(例: assāsapassāsānaṃ=入出息, 長い入出息) と方向注記
- 方針: vism_08 の ānāpāna 訳語統一と整合. assāsa/passāsa の方向 (現状 assāsa=入息) には触れない
- 過渡的不整合: 方向バッチ未実施のため, 同一文に「入出息」(assāsapassāsa) と「出入息念」(ānāpāna) が同居する箇所が残る (§169 §183 等). 方向バッチで解消予定
- 校訂: Claude Opus 4.8
